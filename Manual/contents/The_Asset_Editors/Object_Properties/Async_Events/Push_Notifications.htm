Notificações Push

O Evento de Notificação Push é acionado pelo retorno da chamada de notificações push no SO do dispositivo, seja de uma fonte local ou de uma fonte remota. Ele gera um DS Map que é exclusivo para este evento e é armazenado na variável especial async_load.

Este evento é usado exclusivamente por extensions (como o Firebase Cloud Messaging extension), portanto, consulte a documentação de sua extensão para obter instruções sobre o uso deste evento.
