Ladrilhos Animados

Os ladrilhos são geralmente considerados células estáticas dentro do jogo room, porém com o GameMaker é possível animá-los, assim como você faria com um sprite. Por que não usar apenas sprites, você pergunta? Bem, sprites tem uma certa sobrecarga de processamento devido a eles estarem associados a uma caixa de delimitação e à forma como são renderizados, mas tiles têm uma sobrecarga de processamento muito menor e, portanto, são mais rápidos de renderizar. No entanto, isto tem um preço, já que com tile set animações você está limitado a criando animações que devem ser uma potência de dois comprimentos - ou seja: de 2, 4, 8, 16, etc... quadros - e obviamente devem ser criadas dentro da grade tile, e todas elas devem animar à mesma velocidade (na base de um conjunto por telha). Entretanto, mesmo Com estas limitações, tiles animado é uma característica poderosa que pode ser usada para acrescentar vida e cor a ambientes e fundos estáticos de outra forma.

Para criar uma animação tile você deve primeiro ter um tile set que tenha o necessário tiles tudo dentro da imagem sprite (mas não como sprite frames, então um único grande sprite frame com todas as imagens de animação dentro dele), e então no Editor de Tile Set você clica no botão Animação de Tile para trazer o Editor de Animação:

A imagem de exemplo acima é uma imagem única com todos os quadros de animação contidos nela, mas deve ser notado que a imagem poderia também contém outras não animações tiles também, já que você pode usar um único massivo tile set e depois "escolher a cereja" as partes da animação a partir das quais você quer criar a animação. Por exemplo, você poderia ter uma grande paisagem tile set com árvores, terra e água, e dentro do tile set têm múltiplos tiles a serem animados para fazer a ondulação da água.

Para criar uma animação tile você precisa primeiro adicionar uma animação à biblioteca de animação clicando no botão Add Animation , que irá adicionar uma tira de animação vazia à biblioteca e solicitar que você selecione um número de quadros a serem usados:

Você será então apresentado com uma visão de cada frame de animação vazio, com o frame inicial destacado para você:

Se você voltar para a imagem tile set à esquerda do editor, você pode clicar em qualquer tile para adicioná-la à animação e o frame avançará para o próximo. Clicando em em sucessivos tiles você preencherá o animação, que você pode então visualizar clicando no botão "play" ao lado dos quadros:

Uma vez que você tenha feito isso, sua animação se tornará parte da biblioteca de animação e quando você entrar no Editor de Sala você poderá selecioná-la e colocá-la dentro do room. Note que se você selecionar qualquer um dos tiles de uma animação para colocar em um room, que tile irá animar a partir do tile selecionado, assim você pode "dessincronizar" uma animação colocando separadamente dentro do room e todos eles irão animar. A única ressalva é que a velocidade de animação será sempre a mesma, já que é definida para todo o tile set e não para animações individuais dentro da biblioteca.

Algumas observações sobre tiles que estão sendo usadas em animações:

  • Se o tile tem sua própria animação especificada (então há uma animação onde ela é definida como a primeira tile), então ele usará essa animação.
  • Se o tile não tem sua própria animação, mas é usado em apenas uma outra animação, então ele usará essa animação começando na primeira posição que tile está presente na animação.
  • Se o tile não tem sua própria animação e é usado em várias outras animações, então ele não animará de forma alguma. Você precisará especificar explicitamente uma animação com ela como a primeira tile.
