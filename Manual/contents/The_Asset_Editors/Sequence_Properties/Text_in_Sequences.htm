Texto em Sequências

As faixas de texto podem ser adicionadas a um Sequence através do editor. Isto é feito selecionando o botão Add a Text Track da Canvas Toolbox, e então clicando (e opcionalmente arrastando) na Canvas para criar uma faixa de texto.

Esta página cobre os seguintes tópicos:

  1. Criação de texto
  2. Modificação de texto
  3. Tipos de letra Ativos
  4. Parâmetros
  5. Inspetor

Criação de texto

Há duas maneiras de criar uma faixa de texto. Após selecionar a ferramenta de texto a partir da Caixa de ferramentas, você pode:

  • Clique uma vez na tela: Isto cria uma faixa de texto "desembrulhada". Isto significa que o texto pode ser o tempo que você quiser e ficará em uma linha. O embrulho pode ser facilmente ativado mais tarde através do inspetor.
  • Clique e arraste na lona: Isto cria uma faixa de texto "embrulhada", ou seja, o texto só é exibido dentro de uma moldura. O tamanho deste quadro é definido por arrastar enquanto se cria a trilha de texto. Seu tamanho também pode ser alterado posteriormente através de faixas de parâmetros ou do Inspetor. É recomendável usar faixas de texto embrulhadas, pois facilita muito o alinhamento de seu texto e a obtenção de resultados consistentes enquanto anima.

Modificação de texto

Quando você cria uma nova faixa de texto, GameMaker abre o Inspetor e seleciona automaticamente o campo Texto, permitindo que você comece a escrever seu texto.

Para editar o texto de uma faixa a qualquer momento, basta selecionar essa faixa, ir para o Inspetor e modificar novamente o campo Texto.

Comportamento de redimensionamento

O redimensionamento de sua trilha de texto na tela funcionará de forma diferente dependendo se sua trilha de texto estiver embrulhada ou desembrulhada.

Se desembrulhado, sua trilha de texto será escalonada, semelhante à escala de sprites e objects. No entanto, o redimensionamento de uma trilha de texto embrulhada modifica seu tamanho de quadro ao invés de modificar seus valores de escala.
Para redimensionar uma trilha de texto embrulhada, use Gizmos no lugar, ou edite os valores na trilha do parâmetro Escala manualmente.

Tipos de letra Ativos

As faixas de texto são vinculadas aos ativos da fonte. Quando você criar uma nova trilha de texto, ela procurará um font asset dentro de seu projeto, e usará o que foi adicionado pela última vez. Se seu projeto não tiver fonts, ele criará automaticamente um novo font asset.

Quando você exclui um font, todas as faixas de texto que utilizam font também são excluídas, portanto, certifique-se de não excluir acidentalmente font assets que são utilizadas em Sequences.

Parâmetros

Você pode modificar e animar todos os parâmetros habituais que são fornecidos para uma faixa sprite, como Posição, Rotação, Escala, Origem e Multiplicação de Cores. Usando estes parâmetros você pode mover e girar seu texto, torná-lo maior ou menor, e mudar sua cor e alfa.

Há também alguns parâmetros específicos de texto fornecidos, que também podem ser animados:

  • Espaçamento de caracteres: Este é o espaço entre cada caractere (em pixels).
  • Espaçamento entre linhas: Este é o espaço entre cada linha de texto (em pixels).
  • Escala de parágrafo: Este é o espaço entre cada parágrafo do texto (em pixels). Um novo parágrafo começa quando você adiciona uma quebra de linha manualmente, pressionando .
  • Tamanho da moldura: Este é o tamanho da moldura do texto, e só é usado quando o Wrap está habilitado no Inspetor (veja sua seção abaixo para mais informações).

Inspetor

Há algumas propriedades que só podem ser alteradas através do Inspetor. Tais propriedades não podem ser animadas, pois não são faixas de parâmetros, mas opções simples que se aplicam a toda a faixa.

O Inspetor mostra as seguintes propriedades quando uma faixa de texto é selecionada:

O Inspetor deve abrir automaticamente quando você cria ou seleciona uma faixa de texto, mas se seu Inspetor estiver fechado, você pode abri-la manualmente a partir do menu do Windows, na parte superior.

Algumas das faixas de parâmetros (como cor, espaçamento de caracteres, espaçamento de linhas, etc.) estão incluídas aqui, e mudá-las criará um keyframe no playhead se As mudanças de registros automáticos estiverem ativadas, ou modificar o keyframe selecionado. O restante das opções são "estáticas" (pois não podem ser animadas) e são as seguintes:

  • Na parte superior mostra o font asset que é selecionado. Você pode usar esta opção para selecionar um já existente font asset, ou criar um novo e atribuí-lo à faixa de texto selecionada.
  • Texto: Este é o texto que é exibido em sua trilha de texto. Você pode inserir várias linhas aqui, porém linhas vazias não serão exibidas.
  • Alinhamento: Aqui você pode mudar como seu texto é alinhado dentro do quadro horizontalmente (esquerda, centro, direita, justificada) e verticalmente (fundo, topo, meio). Isto só se aplica quando a opção Wrap estiver ativada.
  • Embrulho: Isto controla se o texto deve ser embrulhado automaticamente dentro do quadro definido. Quando isto for ativado, seu texto passará automaticamente para a próxima linha quando atingir o final do quadro, e será sempre cortado se o texto for mais longo do que a altura do quadro. Quando isto for desativado, todo o seu texto será exibido em uma única linha, e o quadro não será mais exibido na tela.
