Package Manager

IMAGE

The Package Manager lets you download packages and add them to your project. This is currently used for packages required for GMRT (GameMaker Runtime).

The list on the left will show you all packages from the selected source, and clicking on a package will show its information on the right, where you can add it to your project.

In the top-left corner you can search for a package, and in the top-right corner you can change the source for the packages from a drop-down menu. You can click on  to edit the list, which will show the following menu:

IMAGE

This shows all the current package sources and you can click on any to edit its details or remove it. You can click on the  button to add a new source.

For each source you can define its name in the list, its URL, its scopes, a password and the install subdirectory.
