O Micro Servidor Web

A principal função desta pequena ferramenta é conectar e servir arquivos a algumas das plataformas alvo disponíveis para o GameMaker (como o alvo HTML5). Ela também serve arquivos para o manual do GameMaker quando usado offline.

Quando você executa um jogo em um alvo que requer o Micro Web Server, você verá uma nova janela aberta conforme o jogo é servido para o dispositivo escolhido:

NOTA A primeira vez que você executar um jogo em um alvo que requer o Micro Web Server, você poderá receber um alerta de firewall. Você deve permitir que a web server se comunique através do firewall, caso isso aconteça.

Quando a Micro Web Server estiver funcionando, você poderá acessar seu jogo de qualquer dispositivo local (e até remotamente, se o encaminhamento de portas estiver devidamente configurado). Basta acessar o endereço IP local de seu computador através da porta mostrada nesta janela, e você poderá jogar seu jogo de outro dispositivo, como um telefone celular, ou outro computador.

Por exemplo, digamos que o endereço IP local de seu computador é 192.168.0.110(leia isto para saber como encontrá-lo), e a porta mostrada na janela Micro Web Server é 51264. Depois de executar o jogo em seu computador, você se conectaria ao 192.168.0.110:51264 a partir de um navegador em seu celular, e jogaria seu jogo.

A Web Server mostrará uma certa quantidade de informações de depuração relacionadas com os arquivos que foram servidos, e tem um botão para limpar isto no fundo. Ao lado do botão "Clear" está o path para o arquivo JS raiz do projeto, enquanto que abaixo dele você tem a opção Network Speed. Esta está definida como "No Limit" por padrão, mas clicando nela, será aberta uma lista de diferentes opções de rede:

A mudança desta configuração também mudará a opção de tamanho da embalagem para a direita. Você também pode editar isto manualmente, neste caso a velocidade da rede será definida para "Personalizado". Definindo-o para qualquer outra coisa além do valor máximo, a velocidade de transferência da rede será efetivamente acelerada e permitirá que você teste jogos em um ambiente de baixa largura de banda.

Você também pode ver a lista de IPs permitidos junto com o número da porta que está sendo usada, mas estes não podem ser editados na Micro Web Server. Em vez disso, eles devem ser definidos nas configurações do Gerenciador de Dispositivos para o alvo do dispositivo em questão, ou nas Preferências HTML5 ao usar o dispositivo "Padrão".
