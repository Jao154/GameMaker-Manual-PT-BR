Preferências de espaço de trabalho

As preferências aqui afetarão a forma como você se movimenta dentro dos diferentes GameMaker espaços de trabalho. As opções são:

  • Escala de graus de navegação do teclado do espaço de trabalho: Dentro de um workspace você pode ter várias janelas abertas para diferentes objects e recursos, e assim para se mover rapidamente entre elas você pode usar o atalho de teclado / + + <Arrow Keys> para se mover entre elas. Dado que as janelas dentro de um workspace não estão alinhadas em grade, o IDE tem que escolher qual janela você quer dizer quando você pressiona qualquer uma das 8 direções possíveis, e esta opção permite definir o "ângulo de visão" que é usado. O valor padrão é 40°.
  • Padding de coluna de espaço de trabalho (px): Workspace os elementos são frequentemente "encadeados" uns aos outros (por exemplo, o Editor de Código é encadeado ao Editor de Eventos é encadeado ao Editor Object ), e esta opção permite que você defina a distância horizontal entre quaisquer duas janelas encadeadas. O valor padrão é 30px.
  • Forro de corrente do espaço de trabalho (px): Como mencionado acima, workspace elementos são frequentemente "encadeados" um ao outro e esta opção permite definir a distância vertical entre quaisquer duas janelas encadeadas. O valor padrão é 20px.
  • Contagem do segmento de renderização da cadeia de espaço de trabalho: Aqui você pode definir o número máximo de segmentos para renderização de cadeias. A redução deste valor pode ajudar no desempenho de máquinas de menor porte. O valor padrão é 20.
  • Cor do elo de cadeia do espaço de trabalho: Esta é a cor que será usada para exibir correntes entre os elementos workspace. A cor padrão é ARGB $99DD00.
  • A navegação do teclado do espaço de trabalho repõe o zoom: Como mencionado acima, você pode pular de janela em janela dentro do workspace usando atalhos de teclado. Você também pode aumentar e diminuir o workspace usando o / + a roda do mouse . Ajustando esta opção, se você tiver o workspace ampliado e usar os atalhos para pular para outra janela, então o workspace será retornado para uma visualização 1:1 e centralizado na janela selecionada. Esta opção está desligada por padrão.
  • Cadeias de espaços de trabalho podem se sobrepor: Quando ativada, esta opção permitirá que workspace correntes se cruzem e se sobreponham. Esta opção está desligada por padrão.
