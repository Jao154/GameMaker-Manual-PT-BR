Preferências de ajuda

O manual de ajuda é projetado para operar como uma página web e mesmo que os arquivos possam ser armazenados localmente (dependendo de suas configurações), ele requer o uso de um navegador externo para visualizá-los. As seguintes configurações relacionadas ao manual de ajuda podem ser alteradas aqui:

  • Linguagem Manual: Isto define a linguagem do manual. O padrão é "inglês".
  • Use o manual on-line: Habilite-o a usar o manual online sempre que o manual for aberto, ou desabilite-o para executar o manual localmente. Isto é desativado por padrão.
  • Porta manual de ajuda: Isto define o número da porta para a micro web manual server a ser usada, se "Usar manual on-line" estiver desativado. O valor padrão é 51290.
