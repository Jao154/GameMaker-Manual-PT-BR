Preferências do editor da sala

As Preferências do Editor de Sala são usadas para definir como o Editor de Sala irá se parecer e se comportar. As configurações gerais são as seguintes:

  • Adistância inicial da corrente desvanece-se (px): Define a distância a partir da janela principal para que o elo da corrente se desvaneça. O valor padrão é 10px.
  • Adistância de parada da corrente desvanece-se (px): Define a distância da janela principal para o elo da corrente até o fim do desbotamento. O valor padrão é 100px.
  • Corrente mínima alfa: Define o valor alfa final após o ponto de queda para o ponto de queda da cadeia alfa. O valor padrão é 0,3.
  • Espaçamento de profundidade da camada padrão: Quando você cria várias camadas no editor room é automaticamente atribuída uma profundidade com base em sua posição dentro da lista de camadas. Esta profundidade é incrementada pela quantidade dada aqui. O valor padrão é 100, portanto cada camada será camada anterior + 100 de profundidade.
  • Largura padrão room : Define a largura inicial que será selecionada como padrão para um novo room. O padrão é 1024px.
  • Padrão room altura: Define a altura inicial que será selecionada como padrão para um novo room. O padrão é 768px.
  • Adicionar camadas padrão room: Quando você criar qualquer novo recurso room, o GameMaker irá gerar um par de camadas "base" para você - uma camada de fundo e uma camada de instância. Você pode switch esse comportamento aqui para que o novo rooms não tenha camadas pré-fabricadas. Isto está ligado por padrão.
  • Mostrar room visualizações: Se você tiver vistas ativas dentro de um room, isto é mostrado por um esboço onde os limites da vista estariam dentro do room, e a visibilidade desta área pode ser alternada usando o botão dedicado no próprio editor. Entretanto, você pode escolher ter os limites de visualização invisíveis ao abrir um room, desmarcando esta opção (o botão ainda pode ser usado para mostrá-los novamente). O valor padrão está ativado, tornando os limites de visualização visíveis quando você abre um room.
  • Cor de seleção do inspetor: A cor utilizada para destacar o item selecionado na lista Inspetor Multi-Select.
  • Outras opções de cores: Estas estão relacionadas às diferentes cores que são usadas em toda a interface do editor room, e se você clicar duas vezes em qualquer uma das amostras, você pode editar a cor para aquela configuração.
  • Ajuste do volume mestre: A configuração do volume mestre para reprodução prévia no Editor Room, um valor entre 0 e 1. O valor padrão é 1.

Confirmar Diálogos

As preferências do Confirm Dialogs são usadas para alterar as mensagens que são mostradas no editor room quando você executa certas ações. As mensagens que você pode alterar são:

  • Resposta automática à atualização das profundidades das camadas: quando você muda as profundidades das camadas, uma mensagem é mostrada para avisá-lo de que ao fazer isso, você também atualizará automaticamente todas as outras profundidades das camadas. Com esta opção você pode escolher que a mensagem seja mostrada, que ela atualize as camadas como se você tivesse clicado "Ok", ou que ela não atualize nada como se você tivesse clicado "Não".
  • Resposta automática ao aviso de bloqueio por camadas: Se você tentar excluir uma camada no room e ela por acaso estiver bloqueada, será exibida uma mensagem por padrão. No entanto, ao definir esta opção para "Ok", a mensagem não será mostrada e simplesmente não será permitido excluir a camada.
  • Resposta automática ao aviso de bloqueio do item: Se você tentar apagar um item que por acaso esteja bloqueado, será exibida uma mensagem por padrão. No entanto, ao definir esta opção para "Ok", a mensagem não será mostrada e simplesmente não permitirá que você exclua o item.
  • Resposta automática ao aviso de bloqueio da camada do proprietário do item: Se você tentar apagar um item em uma camada e a camada estiver bloqueada (mesmo que o item não esteja), será exibida uma mensagem por padrão para informá-lo de que não pode ser feito. No entanto, ao definir esta opção para "Ok", a mensagem não será mostrada e simplesmente não será permitido apagar o item.
  • Criar nova camada asset ao arrastar seqüências/prites para camadas inadequadas: Se você tentar adicionar uma sprite (ou uma sequence) a uma camada que não seja uma asset, então lhe será mostrada uma mensagem por padrão para informá-lo que isso não pode ser feito e perguntar se deseja criar uma nova camada asset para o item sendo arrastado. No entanto, ao definir esta opção para "Ok" não será mostrada a mensagem e automaticamente será criada uma nova camada para sprites (e sequences).
  • Resposta automática à eliminação de itens room dos pais: Aqui você pode escolher o que acontece quando você exclui um item que faz parte de um dos pais room (e assim presente em todos os filhos rooms). O padrão é mostrar uma mensagem de aviso e deixar que você decida o que fazer, mas você pode configurá-la para "Deletar tudo" para suprimir a mensagem e prosseguir com a exclusão do item do pai room e todos os seus filhos, ou você pode selecionar "Manter tudo" para suprimir a mensagem abortar a exclusão.
  • Resposta automática à eliminação da camada room dos pais: Aqui você pode escolher o que acontece quando você exclui uma camada que faz parte de um pai room (e assim presente em toda criança rooms). O padrão é mostrar uma mensagem de aviso e permitir que você decida o que fazer, mas você pode configurá-la para "Delete All" para suprimir a mensagem e prosseguir com a exclusão da camada do pai room e todos os seus filhos, ou você pode selecionar "Keep All" para suprimir a mensagem abortar a exclusão.
  • Resposta automática a erro de arrasto inválido asset: Se você tentar adicionar um asset a uma camada que não esteja designada para aquele tipo asset (como adicionar um path a uma camada de instância), então lhe será mostrada uma mensagem por padrão para informá-lo de que não pode ser feito. No entanto, ao definir esta opção para "Ok", a mensagem não será exibida e simplesmente não será permitido adicionar o asset. Note que esta opção não se aplica a instâncias, sprites ou sequences.
  • Criar nova camada de instância ao arrastar objects para camadas impróprias: Se você tentar adicionar uma instância object a uma camada que não seja de instância, então lhe será mostrada uma mensagem por padrão para informá-lo de que não pode ser feita e perguntar se deseja criar uma nova camada de instância para o item sendo arrastado. No entanto, ao definir esta opção para "Ok", a mensagem não será mostrada e automaticamente será criada uma nova camada para a instância.
  • Resposta automática ao erro de arrasto de seleção de camada inválida: Aqui você pode escolher o que acontece quando você arrasta um recurso para o editor room quando você selecionou várias camadas no editor de camadas (ou não selecionou nenhuma camada). O padrão é mostrar uma mensagem de aviso, mas você pode definir esta opção como "OK" para suprimir a mensagem e continuar, sem adicionar nenhum recurso.
  • Resposta automática a erro de arraste de camada bloqueada ou invisível: Aqui você pode escolher o que acontece quando você arrasta um recurso para uma camada que foi bloqueada ou que está marcada como invisível. O padrão é mostrar uma mensagem de aviso, mas você pode definir esta opção como "OK" para suprimir a mensagem e continuar, sem adicionar nenhum recurso.

Configurações da grade

As preferências dos Ajustes da Grade são usadas para mudar a forma como a sobreposição da grade funciona, com as seguintes opções disponíveis:

  • Espaçamento horizontal da grade (px): Isto irá definir o espaço horizontal para as células da grade room em pixels. O valor padrão é 32px.
  • Espaçamento vertical da grade (px): Isto irá definir o espaço vertical para as células da grade room em pixels. O valor padrão é 32px.
  • Grade de exibição: Esta opção irá ligar ou desligar a grade room quando você abrir um room pela primeira vez. Ela está ligada por padrão, mas mesmo que esteja desligada, você pode ativá-la novamente usando o botão apropriado no próprio editor room.
  • Pronto para a rede: Com esta opção você pode definir se um room terá assets snap to the room grid por padrão ou não quando um room for aberto inicialmente. Ele está ligado por padrão, mas mesmo que esteja desligado, você pode ativá-lo novamente usando o botão apropriado no próprio editor room.
  • Cor da linha da grade: Esta opção permite definir a cor (incluindo o componente alfa) das linhas da grade room. O valor padrão é ( ARGB) #30FFFFFFFF.


Guias inteligentes

Esta seção detalha as Preferências dos Guias Inteligentes para o Editor Room. As opções disponíveis são:

  • Habilitar guias inteligentes: Isto é usado para ativar/desativar o uso de guias inteligentes no editor Room. O padrão está habilitado.
  • Habilitar as Réguas: Esta opção habilitará/desabilitará as réguas laterais no Editor Room. A opção padrão está ativada.
  • Réguas de corte para bordas: Quando ativada, esta opção irá quebrar guias arrastadas para o editor Room a partir das réguas para qualquer assets colocado no room. O padrão está ativado.
  • Distância instantânea: Esta configuração diz ao editor Room a que distância um guia tem que estar de um asset antes de "estalar" para o lado do asset. O valor padrão é 6px.
