Preferências do editor do conjunto de azulejos

O Tile Set Editor Preferences tem a seguinte opção para o Auto Tiling:

Auto tile cor de fundo: Esta opção define a cor de fundo dentro do editor tile set na seção de criação do Auto Tiles. A cor padrão é (rgba) #808080FF.

O editor de preferências Tile Set também tem uma subseção Confirm Dialogs (Confirmar Diálogos) :

A opção disponível aqui controla a mensagem que aparece quando você muda a célula do tile set e isso afetará a forma como as camadas do mapa de azulejos room são renderizadas (resultando em camadas recortadas). Por padrão, você será solicitado a continuar ou cancelar a operação, e esta opção permite suprimir esta mensagem, selecionando "Ok". que irá em frente com a mudança de célula tile set independentemente do impacto nas camadas do mapa de azulejos room. O valor padrão é "Mostrar mensagem".
