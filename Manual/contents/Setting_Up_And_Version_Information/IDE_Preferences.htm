Preferências IDE

A Janela de Preferências é uma parte essencial do GameMaker, pois é onde você pode configurar e personalizar sua experiência usando o IDE, bem como configurar qualquer plataforma específica SDK's etc... Nesta seção nós cobrimos as áreas específicas IDE da janela de preferências - para as plataformas alvo individuais, consulte a seção de Preferências da Plataforma Alvo. Note que após alterar quaisquer preferências desta janela você precisa clicar no botão Aplicar para que elas tenham efeito.

Se, após mudar as opções aqui para personalizar o GameMaker ao seu próprio gosto, você desejar respaldar as preferências que você definiu, você precisa ir para esta pasta no Windows:

%AppData%/GameMaker/<your_account_name>/

Ou esta pasta no Mac:

~/.config/GameMaker

E depois copiar os dois arquivos a seguir para um local seguro:

  • local_settings.json
  • cloud_settings.json

No entanto, se você deseja redefinir as preferências que você mudou, então você precisa clicar no botão Restaurar Padrões na parte superior da janela. Note que isto só restaura os padrões para a seção que está sendo editada atualmente.
