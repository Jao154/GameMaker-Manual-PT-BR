Preferências da Ópera GX

As preferências da Opera GX têm as seguintes opções:

  • Localização Emscripten SDK: Esta é a localização do Emscripten SDK, necessário para as construções da YYC feitas para a Opera GX. Leia Configurando para Opera GX para obter informações sobre como configurar o SDK.
  • Porta padrão da Web Server : Isto permite mudar a porta padrão para a web Opera GX server. O valor padrão é 51264.
  • Endereço padrão da Web Server: Aqui você pode alterar o endereço padrão da web server para a web Opera GX server. O valor padrão é 127,0,0,0,1.
  • Web padrão Server IPs permitidos: Aqui você pode adicionar qualquer IP permitido para a web server e eles serão definidos como padrão para todos os dispositivos adicionados através do Gerenciador de destino.
