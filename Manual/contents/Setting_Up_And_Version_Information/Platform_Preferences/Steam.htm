Preferências de vapor

O Steam SDK é usado para Windows, macOS e Ubuntu (Linux) construções que devem ser distribuídas através da plataforma Steam. Esta seção contém as opções de configuração necessárias para que ela funcione dentro de seus projetos:

  • Caminho para Steam SDK: O sistema path para onde você instalou o Steam SDK. Uma vez que você tenha configurado este path para o SDK, você precisará então ativar Steam nas Opções de Jogo e então para cada plataforma adicionar a ID do aplicativo.
