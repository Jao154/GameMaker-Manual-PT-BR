tvOS Preferências

As preferências tvOS têm as seguintes opções:

  • Resposta automática a tvOS aviso de limitação de armazenamento persistente: tvOS jogos têm armazenamento local limitado para arquivos, e quando você compila para este alvo um aviso será mostrado por padrão para lembrá-lo deste fato. Você pode mudar o ajuste aqui para suprimir esse aviso, selecionando "OK". A configuração padrão é "Mostrar mensagem".
  • Suprimir a construção e o funcionamento: Por padrão quando você cria um projeto tvOS e depois constrói um pacote executável final, o GameMaker constrói o pacote e depois tenta executá-lo em qualquer dispositivo conectado. Ao marcar esta opção, o GameMaker suprimir a construção do aplicativo e sua execução no dispositivo e simplesmente criar um projeto XCode no Mac host.
  • MacOS Install Path: A instalação path para os arquivos do projeto no Mac do build.
  • Identificador da equipe padrão: Aqui você pode adicionar seu Identificador de Equipe padrão, como lhe foi designado pela Apple. Este Identificador de Equipe será usado quando seus arquivos de jogo forem enviados para Xcode para construir o aplicativo, e permitirá que Xcode gere os certificados de assinatura necessários. Note que esta configuração será aplicada por padrão a todos os jogos construídos para tvOS, mas pode ser superada com base em cada projeto do tvOS Geral Opções de jogo.
