Preferências de plataforma

Esta seção do manual cobre as diferentes Preferências específicas da plataforma que precisam ser configuradas antes que você possa compilar seus jogos para determinados objetivos. Estas geralmente só precisam ser configuradas uma vez para começar (embora futuras atualizações dos tempos de execução possam exigir que você altere certas coisas, mas estas serão mencionadas nas Notas de Lançamento) e são usadas para apontar para quaisquer SDKsque sejam necessários e definir opções básicas de compilação para como você deseja que o GameMaker compile seus projetos de jogo para qualquer plataforma. Note que as plataformas disponíveis dependerão da Licença que você possui e nem todas elas podem estar disponíveis.

Abaixo está uma lista de todas as plataformas alvo disponíveis para licenças padrão:

  • Android
  • HTML5
  • Ópera GX
  • iOS
  • tvOS
  • macOS
  • Windows
  • Windows UWP

Observe que, além de configurar estas preferências, você também precisará configurar os dispositivos que algumas destas plataformas visam, caso contrário, você poderá não ser capaz de compilar ou testar seus projetos. Consulte a seção sobre Compilação e a seção sobre o Gerenciador de Dispositivos para obter mais informações.

O manual não mostra as diversas preferências do Console (Xbox, PlayStation e Nintendo Switch) devido às diferentes restrições legais mantidas pelas diferentes empresas. Se você tem uma Licença de Console, então você pode obter estas informações no Helpdesk YoYo Games, mas primeiro você deve contatar YoYo Games com seu e-mail de identificação do console e solicitar permissão para visualizar as diferentes seções.
