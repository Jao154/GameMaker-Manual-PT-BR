Opções de jogos tvOS

Esta seção descreve as diferentes opções disponíveis para você que controlam como seus projetos de jogo tvOS serão compilados. As diferentes seções são:

Geral

Aqui você pode preencher as Informações do produto sobre seu jogo, incluindo seu Nome de exibição, seu ID do pacote e as Informações da versão . Note que o Bundle ID deve estar no formato correto de URL reversa para que a construção final do jogo funcione corretamente, EG: com.[Company].[GameName]. Você também pode definir o local de saída do aplicativo no Mac do build, e você pode dar ao Team Identifier que deseja usar para assinar o aplicativo final que o GameMaker cria para você. Defini-lo aqui substituirá o ID da equipe que você forneceu nas Preferências do tvOS.

Aqui você também tem a opção de gerar um Podfile e um Podfile.lock. Estes arquivos são necessários se você estiver adicionando CocoaPods a uma extensão tvOS usando o Editor de Extensão. Os arquivos contêm macros específicas exigidas pelo GameMaker para configurar qualquer CocaoPods ao construir seu jogo. Clicando nos botões criará os arquivos com as macros pré-geradas, e também abrirá um explorador de arquivos na localização dos arquivos, caso você deseje editá-los manualmente. Há também um botão que o levará para o seguinte guia:

  • iOS e tvOS: usando cápsulas de cacau

Gráficos

Aqui você pode alterar os seguintes detalhes relacionados a como seu jogo será exibido, com as seguintes opções disponíveis:

  • Interpolar as cores entre os pixels: Ativa a interpolação linear, que basicamente "suaviza" os pixels. Para gráficos de pixels nítidos, deve ser desligado, mas se você tiver misturas alfa agradáveis e gráficos alisados, é melhor deixá-lo ligado. Isto está desativado por padrão.
  • Escala: Aqui você pode selecionar a escala da tela de desenho mantendo a relação de aspecto dentro da tela do dispositivo - adicionando "acolchoamento" ao redor das bordas para caber na tela - ou você pode selecionar ter a tela de desenho esticada para caber na tela do dispositivo.

A seguir, há a opção de definir o tamanho da página de textura. O tamanho padrão (e o mais compatível) é 2048x2048, mas você pode escolher entre 256x256 até 8192x8192. Há também um botão marcado Preview que irá gerar as páginas texture para esta plataforma e depois abrir uma janela para que você possa ver como elas ficam. Isto pode ser muito útil se você desejar ver como as páginas texture estão estruturadas e para evitar ter páginas texture maiores (ou menores) do que o necessário. Para mais informações sobre as páginas texture, por favor, veja aqui.

NOTA: Esteja ciente de que quanto maior o tamanho da página texture, menos compatível seu jogo será com diferentes navegadores e dispositivos.

Finalmente, você escolhe Mostrar um cursor ou não. Se marcado, o SO mostrará um cursor sempre que o mouse for movido (na posição do mouse), e o cursor desaparecerá quando o mouse não for movido por um período de tempo. Se desmarcado, então nenhum cursor será mostrado.

Ícones

Esta seção permite adicionar os vários ícones diferentes exigidos pelos vários dispositivos e as diferentes partes da App Store. Estes ícones devem ser de autoria de imagens *.png 24bit no tamanho especificado para cada uma.

Vale notar que o GameMaker possui uma ferramenta Gerador de Imagens de Projeto que pode ser usada para criar automaticamente todas as imagens necessárias para todas as diferentes plataformas alvo para as quais seu jogo está sendo compilado. Se você usar esta ferramenta, você deve revisar as imagens criadas para garantir que elas sejam o que você precisa, especialmente para esta plataforma, pois os ícones necessários são retangulares e não quadrados. Esta ferramenta também gerará imagens (explicadas abaixo), bem como ícones.

Imagens

Esta seção permite adicionar gráficos separados como imagens "Top Shelf" e telas de respingo para cada um dos diferentes tamanhos de alvos tvOS. Estas imagens devem ser autoradas como imagens individuais 24bit *.png sem transparências e no tamanho correto. Você também pode definir um tempo mínimo para mostrar as telas splash a partir do menu suspenso na parte inferior (de 0 a 10 segundos).

Como mencionado na seção sobre Ícones, você pode usar o Gerador de Imagens do Projeto para criar estas imagens.
