Opções de jogos Opera GX

As Opções de Jogo Opera GX permitem que você modifique as propriedades de seu jogo. As seguintes seções são fornecidas:

Geral

Esta janela mostra seu status de login na Opera GX. Você pode clicar no ícone Refresh para fazer o login usando sua conta Opera.

Você então tem as seguintes opções:

  • Selecione o jogo: A partir desta lista, você pode selecionar um jogo existente para atualizar, ou escolher carregar um novo jogo.
  • Nome do jogo: Este é o nome de seu jogo (título) no GX.games.

    IMPORTANTE Para que o upload no GX.games tenha sucesso, é importante que você só utilize personagens permitidos no nome do jogo (que, por padrão, está definido com o nome do seu projeto GameMaker, portanto, certifique-se de escolher um nome de projeto válido ou mudar o nome do jogo manualmente depois).

    Se você carregar um jogo com um título que tenha personagens proibidos, você receberá este erro: Dados Incorretos. Verifique a saída

    Os caracteres permitidos são:
    • a-z, A-Z (com o apoio de esquemas locais como å)
    • 0-9
    • espaço em branco
    • : - ! ? . , ' & ( )
  • Versão: Esta é a versão do jogo que foi carregada pela última vez. Você só pode mudar este campo antes que seu jogo tenha sido carregado no GX.games.
  • Próxima versão: Este é o número da versão utilizada para seu próximo upload do GX.games. Você só pode mudar isto depois que seu jogo tiver sido carregado no GX.games. Você não precisa modificar isto manualmente para cada upload, pois GameMaker aumentará automaticamente o último dígito do número da versão para você.
  • Registre-se no GX.games: Clique neste botão para registrar uma nova entrada no GX.games. Isto só é válido quando "Selecionar jogo" estiver definido como "Novo jogo".
  • Selecione o grupo: Antes de carregar um novo jogo, você pode selecionar o estúdio ao qual o jogo pertencerá.
  • Editar Jogo na Ópera: Isto abre o GX.games DevCloud e o leva à página "Detalhes do jogo" de seu jogo. Isto só é clicável quando você estiver logado e seu jogo for carregado.
  • URL de compartilhamento interno: Isto abre a versão privada de seu jogo, somente quando tiver sido ativado no GX.games DevCloud.
  • URL de compartilhamento público: Isto abre a versão pública de seu jogo, somente quando tiver sido habilitado no GX.games DevCloud.

    Ambos os URLs podem ser compartilhados com outros para permitir que eles joguem seu jogo.

Gráficos

Aqui você pode alterar os seguintes detalhes relacionados a como seu jogo será exibido:

  • Exibir cursor: Controla se o cursor do mouse está visível quando seu jogo começa. Útil para jogos que não requerem a entrada do mouse ou usam um sprite personalizado do mouse.
  • Interpolar as cores entre os pixels: Ativa a interpolação linear, que basicamente "suaviza" os pixels. Para gráficos de pixels nítidos, deve ser desligado, mas se você tiver misturas alfa agradáveis e gráficos alisados, é melhor deixá-lo ligado. Isto está ligado por padrão.
  • Escala: Seu jogo pode ser configurado para escalar a tela de sorteio automaticamente para manter a relação de aspecto dentro do navegador, ou você pode selecionar para que seja executado em escala completa. A opção de escala completa não fará a tela completa do jogo no navegador, mas sim esticará o que é desenhado para caber no tamanho da tela, como definido pelo primeiro room do jogo. Isto é definido para manter a relação de aspecto por padrão.

Finalmente, há a opção de definir o tamanho da página de textura. O tamanho padrão (e o mais compatível) é 2048x2048, mas você pode escolher entre 256x256 até 8192x8192. Há também um botão marcado Pré-visualização que vai gerar as páginas texture para esta plataforma e depois abrir uma janela para que você possa ver como elas são. Isto pode ser muito útil se você desejar ver como as páginas texture estão estruturadas e para evitar ter páginas texture maiores (ou menores) do que o necessário.
