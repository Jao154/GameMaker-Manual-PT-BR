A página inicial

Quando você abrir o GameMaker pela primeira vez, você será solicitado a entrar em sua conta YoYo ou Opera Account (ou se registrar para uma se ainda não o fez - veja aqui). Uma vez registrado e logado, a IDE principal do GameMaker será aberta na página inicial:

Quando você inicia o GameMaker pela primeira vez, você poderá ver uma descrição para cada elemento da tela inicial, pairando sobre ele.

Aqui, você pode clicar no botão "SKIP to Setup Wizard" para começar a criar seu primeiro projeto com a ajuda do Setup Wizard, que o levará através dos passos necessários para criar seu primeiro jogo no GameMaker.

Você também pode clicar em "Fechar" para fechar o assistente e continuar usando o GameMaker manualmente, e opcionalmente ativar a caixa de seleção "Não me mostrar novamente" se você não quiser ver o assistente quando iniciar o GameMaker novamente.

Elementos da página inicial

A página inicial contém os seguintes elementos:

  • Barra de menu IDE: Isto mostra os menus que você pode utilizar em toda a IDE; mais informações são dadas em uma seção abaixo.
  • Opções de projeto: A seção Projetos é onde você pode criar, abrir ou importar projetos. Isto é explicado com mais detalhes mais abaixo nesta página.
  • Projetos recentes: Aqui você pode ver uma lista de projetos anteriores que você pode abrir. Você também pode ver o nome completo do projeto e path, e clicando no botão esquerdo do mouse abrirá o projeto. Você também pode switch a visualização desta seção entre o modo tile e o modo lista, pressionando os botões no canto superior direito.
  • Detalhes da versão: Esta seção fornece detalhes sobre a versão atual IDE que está sendo utilizada, assim como a atual versão Runtime. Você também será notificado de quaisquer mudanças disponíveis no IDE ou no Runtime nesta seção.
  • Detalhes do usuário: Isto mostra o nome de usuário de sua conta YoYo e a última vez que você entrou no site.
  • Recursos úteis: Isto mostra vários tiles em que você pode clicar para acessar os recursos oficiais do tutorial do GameMaker.

Menus IDE

Na parte superior você pode encontrar os menus gerais IDE que são explicados nas seções seguintes do manual:

  • O Menu Arquivo
  • O menu de edição
  • O Menu Construir
  • O menu Windows
  • O Menu de Ferramentas
  • O menu Marketplace
  • O Menu de Layouts
  • O Menu Ajuda

Observe que também aparecerão opções de menu específicas do contexto na barra de menu superior, dependendo da janela em que você se concentrou atualmente. Por exemplo, se você tiver a janela do Editor de Imagens em foco, então você terá itens extras de menu aqui para "Imagens", "Ver" e "Efeitos". Estes itens extras de menu são explicados nas seções relevantes do manual para o workspace ou janela que os gerou.

Iniciando um novo projeto

Você pode clicar no botão Novo para criar um novo projeto, no botão Abrir para abrir um projeto existente ou no botão Importar para abrir um arquivo de projeto comprimido YYZ GameMaker ou um GameMaker legado: Estúdio 1.4 GMX arquivo de projeto. Tanto Open como Import abrirá o explorador de arquivos para que você navegue até o arquivo de projeto desejado.

NOTA: O GameMaker não é completamente compatível com o GameMaker ao contrário: Os projetos Studio 1.4 mas os projetos 1.4 importados ainda devem ser executados, pois a funcionalidade obsoleta foi recriada para você automaticamente usando scripts de compatibilidade. Para detalhes completos sobre os possíveis problemas e as mudanças feitas em GML, consulte o artigo do Help Center Porting A GMS 1.4 Project To GameMaker, bem como a seção do manual sobre Funções Obsoletas.

Para criar um novo projeto basta clicar no botão Novo, que abrirá o menu Template:

Selecione um modelo (ou começando com um projeto em branco), escolha um nome e salve o local para seu projeto, e depois clique em "Let's Go!
