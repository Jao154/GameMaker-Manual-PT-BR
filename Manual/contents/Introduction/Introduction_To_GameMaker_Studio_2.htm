Introdução ao GameMaker

GameMaker é uma ferramenta projetada para capacitar você e sua equipe a fazer jogos novos e inovadores, bem como idéias de protótipos da maneira mais rápida e intuitiva possível através de múltiplas plataformas alvo. Destina-se principalmente como uma ferramenta para fazer jogos 2D - embora jogos 3D sejam perfeitamente realizáveis - e vem com uma série de ferramentas e editores para ajudá-lo a realizar seus sonhos e idéias, com seu projeto final sendo portado através de múltiplas plataformas a partir dos mesmos recursos básicos iniciais.

Para aqueles que são novos no mundo da programação ou que nunca usaram nenhuma ferramenta de criação de jogos antes, o GameMaker oferece uma interface visual de ações intuitiva e fácil de usar do GML que permite que você comece a criar seus próprios jogos muito rapidamente usando o script visual:

Se você é um programador mais experiente - ou menos visual - então você pode usar a linguagem proprietária de scripts GML para criar seus jogos:

Você pode até mesmo combinar ambas as funções GML e misturá-las com GML Ações visuais, dando-lhe uma grande flexibilidade quando se trata de como você escreve as coisas. Tudo isso de forma intuitiva, com diferentes aspectos de seu jogo "encadeado" juntos na IDE para lhe dar uma representação única e muito visual do fluxo de trabalho.

O GameMaker vem com uma série de ferramentas poderosas, como um Editor de Imagem completo para criar seus próprios sprites e a capacidade de usar conjuntos de azulejos e você também pode importar imagens que você tenha feito em qualquer editor externo, incluindo Spine animações de esqueleto e arquivos vetoriais em formato SWF. Uma vez que você tenha suas imagens é simplesmente uma questão de atribuí-las ao objects no Editor de Objetos, adicionando algum código ou GML Ações visuais, e depois colocando instâncias destas objects em um jogo room no Editor de Sala. Há também ferramentas adicionais para criar Caminhos, Seqüências, Shaders e muito mais. Você verá que o IDE também é incrivelmente flexível, e pode ser personalizado de acordo com suas necessidades, caso a configuração padrão IDE não cubra algo que você precise, usando as diferentes Preferências.

Este arquivo de ajuda lhe dirá tudo o que você precisa saber sobre o GameMaker e como você pode criar seus próprios jogos com ele. Entretanto, por favor, perceba que mesmo com um programa como o GameMaker, projetar jogos de computador não é completamente sem esforço - portanto, nada de "faça de mim um MMORPG" aqui! Esteja ciente também de que existem muitos aspectos para fazer jogos e que todos eles são igualmente importantes: jogabilidade, gráficos, sons, interação do usuário, etc... Mas, o aspecto mais importante de tudo, é a paciência. Embora o GameMaker seja uma ferramenta poderosa, ele não pode fazer tudo por você ao toque de um botão, então comece pequeno com exemplos simples e você verá que não só está criando jogos muito divertidos, mas que aprender a usar o GameMaker para fazer isso também é muito gratificante!

Para mais informações, exemplos, tutoriais e links para fóruns úteis e outros grandes recursos, visite o site GameMaker.
