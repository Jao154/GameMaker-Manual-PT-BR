novo

O operador new pode ser usado para criar um struct com uma função que é chamada com os argumentos dados. A função pode então preencher o struct com variáveis a partir dos argumentos - muito parecido com o evento de criação de uma instância definirá as variáveis iniciais para a instância - e então o operador new retornará o struct. Antes de usar este operador é importante notar que a função dada deve ser marcada como uma função constructor , caso contrário o operador new não criará o struct (o exemplo de código abaixo mostra isto, e para mais informações, por favor veja a página em Structs).

OBSERVAÇÃO Você pode verificar um struct para descobrir qual função foi usada para criá-lo usando a função runtime instanceof().

Este operador tem a seguinte sintaxe:

<variable> = new <function> (<argument>, <argument>, ...);

Quando você usa o novo operador junto com uma função, ele retornará uma referência struct que é armazenada na variável dada, permitindo que você acesse o struct e quaisquer variáveis que foram criadas dentro dele pela função.

O exemplo a seguir mostra uma função que foi definida para ser usada como construtor:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Esta função pode então ser usada junto com o operador new para criar um struct e preenchê-lo com as variáveis definidas para os valores dos argumentos usados na função, como este:

mystruct = new init_struct(10, 100, "Hello World");

É importante notar que a chamada de novas funções que não tenham sido marcadas como construtor causará uma exceção de tempo de execução.
