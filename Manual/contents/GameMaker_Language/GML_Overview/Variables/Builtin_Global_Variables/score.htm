pontuação

Esta variável é global no escopo e é usada para manter um valor numérico que normalmente é usado para a pontuação do jogador. Esta variável foi projetada apenas para suportar projetos legados de versões anteriores do GameMaker e deve não ser utilizado em novos projetos como pode ser depreciado no futuro.

Sintaxe:

score;

Devoluções:

Real (single precision floating point value)

Exemplo

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

O código acima verifica o índice object da instância que executa o código usando um switche depois adiciona valores diferentes à variável score, dependendo do que for object.
