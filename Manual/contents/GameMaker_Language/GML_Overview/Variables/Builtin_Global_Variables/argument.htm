argumento

Esta variável contém uma matriz que é usada junto com a variável somente leitura argument_count em funções ou métodos de roteiro. Cada posição da matriz possui um valor de entrada para a função e é especificamente para uso com funções de argumento variável .

Note que há também uma série de variáveis independentes de escopo global que também podem ser usadas em funções definidas pelo usuário para referenciar os diferentes argumentos de entrada: argument0, argument1, argument2, etc... até argument15.

NOTA: Isto não é mais estritamente necessário e é fornecido, mais do que qualquer outra coisa, para o apoio ao legado. Todas as funções definidas pelo usuário aceitarão argumentos variáveis, e qualquer argumento passado para uma função que não faça parte das entradas nomeadas será inicializada para indefinida por padrão, que pode ser verificada usando a função is_undefined().

Sintaxe:

argument[n]
argument1
argument2
...
argument15

Devoluções:

Variable (can be of any data type supplied to the function)

Exemplo:

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

O código acima usa a variável argument_count junto com a matriz argument para inicializar variáveis, e é um exemplo de como permitir que uma função definida pelo usuário aceite um número de argumentos de variável.
