clickable_change_ext

Com esta função você pode alterar o sprite e a posição de um ícone clicável previamente criado com clickable_add(). Tenha em mente que a posição é baseada na janela, não na tela, (0,0) posição e que o sprite deve ser referenciado diretamente da página texture (veja: sprite_get_tpe()). Esta função também permite alterar a escala do sprite utilizada (como multiplicador de modo que 1 é o padrão, 0,5 seria metade e 2 seria o dobro) e o valor alfa de 0 (totalmente transparente) para 1 (totalmente opaco) para o ícone final na tela.

Sintaxe:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Argumento Tipo Descrição
index Identificação clicável The index of the clickable icon to change.
tpe Entrada de página de textura The texture page entry for the sprite to be used
x Real The new x position within the window.
y Real The new y position within the window.
scale Real The scale of the icon (default 1).
alpha Real The image alpha of the icon (default 1).

Devoluções:

N/A

Exemplo:

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

O código acima mudará o índice de imagem e a posição do ícone clicável indexado na variável "global.Help", dependendo do atual room, mudando também a escala do ícone e o alfa.
