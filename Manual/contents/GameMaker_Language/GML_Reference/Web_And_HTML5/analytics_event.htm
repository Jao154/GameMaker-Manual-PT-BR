evento_analítico

Esta função enviará o texto especificado para o provedor analítico que você configurou através das Opções de Jogo HTML5. Esta função pode ser usada para criar uma análise personalizada para rastrear algo fora do escopo do provedor que está sendo usado.

Sintaxe:

analytics_event(string);

ArgumentoTipo Descrição
string A string to send to the provider

Devoluções:

Exemplo:

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

O código acima verificará uma variável para ver se ela é true e se é então um evento analítico especial será acionado com o string especificado.
