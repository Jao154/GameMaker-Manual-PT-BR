browser_input_capture

Com esta função você pode definir se a janela do navegador deve capturar todas as entradas (defina para false) ou se o jogo deve capturar as entradas (defina para true). Note que esta função é apenas para uso com o módulo HTML5.

Sintaxe:

browser_input_capture(enable);

Argumento Tipo Descrição
enable Booleano Enable or disable game input capture

Devoluções:

N/A

Exemplo:

browser_input_capture(true);

O código acima define o jogo para capturar todas as entradas do navegador na plataforma alvo HTML5.
