display_get_frequency

Esta função retorna a freqüência (ou taxa de atualização) do visor em que o jogo está sendo jogado. Ela retornará um valor real como quadros por segundo, assim, por exemplo, se seu monitor for 60hz você receberá 60, se estiver funcionando a 144hz então você receberá 144, e assim por diante.

NOTA: Esta função não é suportada em HTML5.

Sintaxe:

display_get_frequency();

Devoluções:

Exemplo:

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

O código acima obtém a freqüência do visor, e funciona em algumas condições para definir o jogo em 240, 120, 60 ou 30 fps. Isto significa que se seu display for 90hz o jogo funcionará a 60 fps, se for 144hz então funcionará a 120 fps, etc. É claro, você pode passar a freqüência do visor diretamente para game_set_speed() para que seja usado como a taxa de quadros do jogo.
