view_enabled

Esta variável controla se qualquer porta de visualização visível dentro do room está habilitada ou não. Se você tiver portas de visualização definidas para visível e depois desabilitar esta opção, todo o room será arrastado para a tela escalonado para o tamanho da janela, em vez das diferentes câmeras sendo desenhadas através das portas de visualização.

Sintaxe:

view_enabled;

Devoluções:

(true: enabled, false: disabled)

Exemplo:

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

O código acima verifica se as visualizações estão habilitadas para o room, e se não estiverem, garante que a porta de visualização[0] esteja visível e então habilita as visualizações para o room.
