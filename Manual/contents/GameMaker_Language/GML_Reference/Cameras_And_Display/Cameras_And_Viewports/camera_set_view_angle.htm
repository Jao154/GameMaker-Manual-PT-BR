câmera_set_view_angle

Você pode usar esta função para atualizar o ângulo da câmera de visão dentro do room. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e então dê o ângulo que você deseja ajustar a visão da câmera. O valor padrão é 0° com valores positivos girando a câmera no sentido anti-horário, ou seja: ajustando o valor para 90° girará a câmera 90° para a esquerda.

Sintaxe:

camera_set_view_angle(camera_id, angle)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera
angle The new angle to set the camera view to

Devoluções:

Exemplo:

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y));

O código acima definirá o ângulo de visão da câmera designada para a porta de visão[0].
