câmera_set_view_view_speed

Você pode usar esta função para atualizar a velocidade da câmera de visão dentro do room. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e, em seguida, dar as funções x e y (horizontal e vertical) que deve se movimentar quando definido para seguir uma determinada instância. A velocidade é calculada em pixels por passo e pode ser ajustada para "-1" para fazer a câmera se mover instantaneamente, mas se a câmera não estiver ajustada para seguir qualquer instância, então a Os valores aqui estabelecidos não terão efeito visível.

Sintaxe:

camera_set_view_speed(camera_id, xspeed, yspeed)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.
xspeed The speed (number of pixels per game frame) that the view should move on the horizontal (x) axis
yspeed The speed (number of pixels per game frame) that the view should move on the vertical (y) axis

Devoluções:

Exemplo:

camera_set_view_speed(view_camera[0], 5, 5);

O código acima definirá a velocidade de visualização da câmera designada para a porta de visualização[0].
