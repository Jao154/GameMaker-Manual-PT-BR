view_set_visible

Esta função pode ser usada para definir a visibilidade do porto de vista dado. A função toma o índice da porta de visualização (de 0 a 7) e um booleano true se estiver visível e false se não estiver.

Sintaxe:

view_set_visible(view_port, visible)

ArgumentoTipo Descrição
view_port The view port to target (0 - 7)
visible Visibility toggle (true é visível e false é invisível)

Devoluções:

N/A

Exemplo:

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

O código acima verificará se a porta de visualização[0] está visível e se não está definida para visível.
