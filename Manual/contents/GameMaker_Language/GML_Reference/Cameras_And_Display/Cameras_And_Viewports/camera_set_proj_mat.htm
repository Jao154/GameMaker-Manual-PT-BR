camera_set_proj_mat

Esta função definirá a matriz de projeção para uma determinada câmera. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e uma matriz de projeção a ser aplicada. Você pode saber mais sobre a criação de matrizes de projeção na seção Funções de Matriz, especificamente matrix_build_projection_perspective() e matrix_build_projection_ortho().

Há algumas coisas que você precisa ter em mente ao usar esta função:

  • Se sua câmera faz monitoramento automático object - ou seja: ela foi criada usando camera_create_view() com um índice object / ID de instância que não é -1, ou você está configurando uma câmera definida no editor Room, ou você está configurando a câmera padrão - então esta matriz será sobregravada no próximo quadro do jogo.
  • Se você usar esta função para switch para uma projeção em perspectiva (ao contrário da projeção ortográfica padrão), o desenho automático de azulejos sprites deixará de funcionar e precisará ser manipulado manualmente. Isto inclui quaisquer camadas de fundo que usam o azulejo horizontal ou vertical e as funções draw_sprite_tiled() e draw_sprite_tiled_ext(), que só desenharão o sprite uma vez em vez de o azulejar se estiver sendo usada uma projeção em perspectiva.

Sintaxe:

camera_set_proj_mat(camera_id, matrix)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Devoluções:

Exemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

O código acima cria uma nova câmera e a atribui para visualizar a porta[0]. Esta câmera então tem suas matrizes de visão e projeção definidas.
