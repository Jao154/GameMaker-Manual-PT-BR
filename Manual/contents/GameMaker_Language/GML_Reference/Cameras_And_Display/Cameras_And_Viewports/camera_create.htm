camera_create

Esta função criará uma câmera "vazia", ou seja: uma câmera sem nenhuma configuração de visão ou projeção definida. Uma vez criada a câmera, pode-se então usar as diferentes funções de matriz para definir as matrizes de visão e projeção. Você também pode usar esta função para criar a câmera e então usar as diferentes funções camera_set_* para configurar a visualização, entretanto note que você deve fornecer pelo menos uma posição e tamanho de visualização, caso contrário a câmera resultante não será válida e dará resultados estranhos se aplicada a uma porta de visualização (se você não configurar o object seguinte, bordas ou velocidade de movimentação estas serão definidas para os valores padrão para uma câmera de visualização no Editor Room ).

Note que as câmeras são como todos os recursos dinâmicos e ocuparão a memória quando criadas. Isto significa que você precisa tomar cuidado para não criar câmeras desnecessariamente e destruir aquelas que você não precisa chamando a função camera_destroy(). Isto também significa que as câmeras persistirão de room a room se atribuídas a uma variável global (elas persistirão quando atribuídas a variáveis locais ou de instância também, mas você não poderá acessá-las após alterar room, daí o vazamento de memória), para que você possa - por exemplo - criar uma nova câmera, atribuí-la a uma variável global, e então, no início de cada room definir as matrizes de visão e projeção necessárias e a porta de visão que você precisa usar.

Sintaxe:

camera_create();

Devoluções:

Camera ID

Exemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

O código acima cria uma nova câmera e atribui sua identificação para visualização[0]. As matrizes de visão e projeção são então criadas e armazenadas em variáveis locais antes de serem usadas para configurar a câmera.
