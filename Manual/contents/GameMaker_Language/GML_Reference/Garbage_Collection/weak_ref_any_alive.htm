Ref_ref_uma_alive_baixa

Com esta função você pode verificar a fraca referência a várias estruturas para ver se elas ainda estão "vivas" ou não. Você fornece uma série de referências fracas para verificar (cada referência fraca deveria ter sido criada usando a função weak_ref_create()), e a função retornará true se qualquer das estruturas ainda estão sendo referenciadas em algum lugar ou false se não estiverem e tiverem sido coletadas de lixo. Note que se você fornecer uma matriz onde qualquer um dos valores não seja uma referência fraca, a função retornará undefined.

Note que a função também leva dois argumentos opcionais, onde o primeiro permite fornecer um índice inicial na matriz para verificar, bem como um argumento para definir o número de posições (comprimento) a partir desse índice para verificar. Se especificado, somente os índices de matriz dentro desses parâmetros serão verificados.

Sintaxe:

weak_ref_any_alive(array, [index], [length]);

Argumento Tipo Descrição
array Fracas referências de estruturas Array containing weak references to the structs that you want to check.
[index] Real OPTIONAL O índice na matriz para começar a verificar a partir de
[length] Real OPTIONAL O número de posições, a partir do índice dado, para verificar se

Devoluções:

Boolean (or undefined)

Exemplo:

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

O código acima verifica uma série de referências fracas e, se alguma ainda estiver viva, uma instância é destruída.
