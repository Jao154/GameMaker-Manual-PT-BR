fraco_ref_alive

Com esta função você pode verificar a fraca referência a um struct para ver se ele ainda está "vivo" ou não. Você fornece a referência fraca para verificar (como retornado pela função weak_ref_create()), e a função retornará true se o struct ainda estiver sendo referenciado em algum lugar ou false se não estiver e tiver sido coletado lixo. Note que se você fornecer um valor que não seja uma referência fraca, a função retornará undefined.

Sintaxe:

weak_ref_alive(weak_ref);

Argumento Tipo Descrição
weak_ref Referência Fraca de Estruturas The weak reference to the struct you want to check.

Devoluções:

Boolean (or undefined)

Exemplo:

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

O código acima verifica uma referência fraca a um struct e, se ainda estiver vivo, a variável de referência é definida como -1, desreferenciando o struct e permitindo que seja coletado o lixo.
