video_set_volume

Esta função altera o volume de áudio do vídeo que é carregado atualmente. Ela toma um valor entre 0 e 1, onde 0 é silencioso e 1 é o volume máximo.

Certifique-se de que isto só seja chamado após uma chamada video_open(), mas antes de uma chamada video_close(), caso contrário não fará nada (pois não haverá um vídeo carregado).

Em algumas plataformas, esta função só funcionará após o vídeo ter sido completamente carregado, então use o Async Callbacks para descobrir quando o vídeo estiver pronto e depois mude o volume.

Sintaxe:

video_set_volume(value);

Argumento Tipo Descrição
value Real Volume value between 0 and 1

Devoluções:

N/A

Exemplo:

my_video = video_open("splash.mp4");
video_set_volume(0.5);

O código acima carrega splash.mp4 dos Arquivos Incluídos do jogo, e muda seu volume para 0,5 (50% do seu volume máximo).
