video_get_volume

Esta função retorna o volume do vídeo carregado atualmente, que é um valor entre 0 e 1. Você pode alterar o volume do vídeo carregado usando video_set_volume().

Sintaxe:

video_get_volume();

Devoluções:

Real

Exemplo:

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

O código acima seria executado em um evento Step. Ele recebe o volume do vídeo que está sendo exibido atualmente, e se for inferior a 1, ele aumentará o volume em 0,02, aumentando gradualmente o volume até chegar ao nível máximo.
