vídeo_abrir

Esta função abre um arquivo de vídeo usando o path especificado. O path pode se referir a um arquivo nos Arquivos Incluídos de seu projeto, um arquivo presente localmente no dispositivo do usuário, ou uma URL para carregar um arquivo da Internet (que pode não funcionar em todas as plataformas).

Como você só pode carregar um vídeo de cada vez, esta função carrega o vídeo especificado no único "slot de vídeo" disponível, que é então manipulado através das outras funções de Vídeo.

Há ligações assíncronas associadas à reprodução de vídeo. Consulte a Async Callbacks para mais informações.

Sintaxe:

video_open(path);

Argumento Tipo Descrição
path Cordão Path to the video file

Devoluções:

N/A

Exemplo:

my_video = video_open("splash.mp4");

O código acima carrega o splash.mp4 dos Arquivos Incluídos do jogo.
