video_seek_to

Esta função permite que você procure a posição dada no vídeo carregado atualmente. Você especifica o valor de tempo a ser procurado, que é em milissegundos.

Esta função funciona melhor quando o vídeo é pausado, e a distância da posição atual até a nova posição de busca não é muito pequena.

Sintaxe:

video_seek_to(time);

Argumento Tipo Descrição
time Real The time to seek to (in milliseconds).

Devoluções:

N/A

Exemplo:

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

O código acima move o vídeo em reprodução 2 segundos à frente, quando a seta para a direita é pressionada, e a nova posição é menor que a duração do vídeo. Entretanto, se não estiver dentro da duração, o vídeo é fechado.

Neste exemplo, video_seek_to() é usado para buscar o vídeo para a nova posição.
