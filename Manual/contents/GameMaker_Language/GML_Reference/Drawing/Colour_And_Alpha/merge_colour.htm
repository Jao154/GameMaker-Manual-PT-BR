merge_colour

Com esta função você pode pegar duas cores e depois fundi-las para fazer uma nova cor. A quantidade de cada uma das cores componentes pode ser definida alterando o argumento "quantidade", onde um valor 0 retornará a primeira cor (col1), um valor 1 retornará a segunda cor (col2) e um valor no meio retornará a mistura correspondente. Por exemplo, um valor de 0,5 irá misturar as duas cores igualmente. A imagem a seguir ilustra como isto funciona, fundindo as cores vermelho e azul:

Sintaxe:

merge_colour(col1, col2, amount);

ArgumentoTipo Descrição
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

Devoluções:

Colour Constant

Exemplo:

col = merge_colour(c_lime, c_orange, 0.5);

O código acima usa a função para criar uma cor fundindo cal e laranja 50/50 e depois armazena seu valor na variável "col" para uso posterior.
