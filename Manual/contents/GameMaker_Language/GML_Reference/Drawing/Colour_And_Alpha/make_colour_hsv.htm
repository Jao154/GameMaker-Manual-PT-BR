make_colour_hsv

O GameMaker fornece esta função (assim como outras) para permitir que o usuário faça suas próprias cores. Esta função particular requer três partes componentes, a tonalidade, a saturação e o valor (também conhecido como "luminosidade") para criar a cor desejada. Estes valores são tomados como estando entre 0 e 255 para que o usuário possa fazer 16.777.216 (256*256*256) cores com isto! Abaixo você pode ver uma imagem de como estes componentes ficam quando separados:

A imagem à esquerda é uma decomposição dos componentes individuais da função, e depois à direita é uma ilustração de como a mudança desses componentes afeta a cor final. Como você pode ver, mesmo sem mudar a tonalidade (ela permanece em 0, vermelho), podemos mudar drasticamente a cor com os outros dois ajustes, com a saturação fazendo o tom de cor passar de cinza (0) para um tom puro (255), e com o valor fazendo-a passar de mais escuro (0) para mais claro (255).

Sintaxe:

make_colour_hsv(hue, sat, val);

ArgumentoTipo Descrição
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

Devoluções:

Colour Constant

Exemplo:

col = make_colour_hsv(100, 145, 255);

O código acima usa a função para criar uma cor e armazenar seu valor na variável "col" para uso posterior.
