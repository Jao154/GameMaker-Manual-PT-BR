gpu_set_fog

Esta função pode ser usada para ativar ou desativar o desenho com neblina. O nevoeiro pode ser usado em jogos 3D para fazer com que as instâncias à distância pareçam desbotadas ou mesmo invisíveis, o que ajuda a criar atmosfera, além de mascarar o fato de que você não está desenhando instâncias que estão longe.

Você define se está habilitado (true) ou desabilitado (false), a cor que o nevoeiro deve usar para a mistura, assim como as distâncias de início e fim do traçado. Os valores das distâncias são essencialmente valores de profundidade em pixels, e são relativos à posição da câmera.

Por exemplo, uma posição de 0 é a mais próxima da câmera, e a cada aumento ela fica mais distante. Por padrão, uma câmera de visão é colocada a -16000 de profundidade, portanto, se você quisesse que a neblina começasse na profundidade 0 e terminasse na profundidade 1000, você definiria a distância inicial para 16000 e a distância final para 17000.

A função pode aceitar quatro argumentos individuais (dados acima), ou uma única matriz com a seguinte estrutura (o código de exemplo abaixo mostra este método):

  • [0] = alternância ativada (um booleano, ou true ou false), padrão false
  • [1] = cor (real), padrão c_black
  • [2] = distância inicial (real), padrão 0
  • [3] = distância final (real), padrão 1

Sintaxe:

gpu_set_fog(enable, colour, start, end);

Argumento Tipo Descrição
enable Booleano Enable or disable fog
colour Cor The fog colour
start Real The distance to start applying fog from (relative to the camera)
end Real The distance to end the fog (relative to the camera)

Devoluções:

N/A

Exemplo:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

O código acima obtém as configurações atuais de nevoeiro e então define o elemento de cor da matriz para c_red antes de definir o nevoeiro novamente usando a matriz alterada.
