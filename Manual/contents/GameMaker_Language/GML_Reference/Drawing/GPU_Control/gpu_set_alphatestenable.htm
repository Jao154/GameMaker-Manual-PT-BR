gpu_set_alphatestenable

Esta função ativará ou desativará o teste alfa para seu jogo (por padrão, isto está desativado). Ao ativar o teste alfa, você poderá então usar a função de companheiro gpu_set_alphatestref() para definir o valor "cut-off" no qual todos os valores alfa serão definidos como 0. A imagem abaixo mostra a diferença que pode ser vista quando o teste alfa é ligado ou desligado:

NOTA: Esta função pode afetar negativamente o desempenho em iOS e dispositivos Android.

Sintaxe:

gpu_set_alphatestenable(enable);

ArgumentoTipo Descrição
enable Enable or disable alpha testing (true / false)

Devoluções:

Exemplo:

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

O código acima verificará se o teste alfa está habilitado e, caso contrário, switch no teste alfa e definirá o limite de teste para 128 (apenas pixels com um alfa acima de 0,5 serão desenhados).
