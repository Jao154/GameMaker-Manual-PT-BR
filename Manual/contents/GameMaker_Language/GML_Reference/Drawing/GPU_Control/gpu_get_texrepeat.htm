gpu_get_texrepeat

Com esta função você pode verificar se a repetição texture está habilitada (retorna true) ou não (retorna false). Para mais informações sobre texture repetindo, veja a função gpu_set_texrepeat().

Sintaxe:

gpu_get_texrepeat();

Devoluções:

Exemplo:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

O código acima verifica se texture está ligado ou desligado e o liga e desliga de acordo.
