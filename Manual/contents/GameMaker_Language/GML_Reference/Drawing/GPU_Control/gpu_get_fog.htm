gpu_get_fog

Esta função pode ser usada para recuperar os ajustes de nevoeiro. A função retorna uma matriz de 4 elementos 1D com as seguintes informações:

  • [0] = alternância ativada (um Booleano, true ou false), padrão false
  • [1] = Constante de cor, padrão c_black
  • [2] = distância inicial(Real), padrão 0
  • [3] = distância final(Real), padrão 1

Observe que você pode mudar esses valores e passar a matriz completa para o gpu_set_fog() funcionam como um único argumento.

Sintaxe:

gpu_get_fog();

Devoluções:

Array (4 elements only; see above for details)

Exemplo:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

O código acima obtém as configurações atuais de nevoeiro e então define o elemento de cor da matriz para c_red antes de definir o nevoeiro novamente usando a matriz alterada.
