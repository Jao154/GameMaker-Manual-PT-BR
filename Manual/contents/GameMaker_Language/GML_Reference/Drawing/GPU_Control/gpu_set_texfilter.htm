gpu_set_texfilter

Esta função pode ser usada para definir a filtragem texture (interpolação linear) de todas as imagens desenhadas na tela do jogo. Quando isto for habilitado (true) todos os textures serão suavizados quando desenhados (isto inclui sprites, pois eles também são considerados textures), significando que quando escalados ou movidos se não houver uma proporção de 1:1 pixel, então haverá uma "mancha" através de vários pixels que pode fazer as imagens parecerem borradas, dependendo do estilo de arte utilizado. Se isto for desativado (false), então as imagens serão desenhadas com base no pixel mais próximo quando escalonadas ou movidas, o que pode levar a imagens "em bloco". O valor padrão é false, e isto também pode ser alterado nas Opções Globais de Jogo para plataformas alvo individuais.

NOTA: Esta configuração substituirá qualquer conjunto de interpolação de palco texture para shaders usando a função gpu_set_texfilter_ext().

NOTA: No alvo HTML5, esta função só funcionará com WebGL habilitado.

Sintaxe:

gpu_set_texfilter(enable);

ArgumentoTipo Descrição
enable Enable or disable texture filtering (true / false)

Devoluções:

Exemplo:

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

O código acima verifica se a filtragem texture está ligada ou desligada e a liga de acordo.
