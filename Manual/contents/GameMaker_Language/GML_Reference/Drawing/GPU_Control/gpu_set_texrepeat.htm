gpu_set_texrepeat

Esta função pode ser usada para indicar ao GameMaker se textures deve repetir (true) ou não (false) quando usada para modelos e primitivos. Normalmente texture larguras e alturas estão na faixa de 0 a 1, mas se você especificar um valor maior que 1 por padrão o restante não é sorteado (o texture inteiro será sorteado com uma seção "vazia" correspondente à porcentagem acima de 1 que é usada. Portanto, um valor de 2 para uma largura ou altura texture desenharia o texture na metade do espaço e deixaria a outra metade vazia). Entretanto, ao usar esta função para definir a repetição verdadeira, o texture é repetido automaticamente sobre todos os modelos e primitivos. O caso provável de uso para estas funções é para repetir um texture em 3D, mas para que ele funcione e não puxar imagens do resto da página texture, o sprite usado precisará ser marcado como estando em uma "Separate Texture Page" no Editor Sprite.

NOTA: Sprites e fundos são sempre desenhados sem repetir, portanto, uma vez desenhado um sprite ou fundo, este valor é redefinido para false. Este comportamento padrão não afetará as configurações do estágio texture (ou seja: chamar a função irá, mas a configuração interna para falso não irá).

OBSERVAÇÃO: Esta configuração irá sobrepor-se a qualquer texture cenário repetido para shaders ao utilizar a função gpu_set_texrepeat_ext().

Sintaxe:

gpu_set_texrepeat(enable);

ArgumentoTipo Descrição
enable Enable or disable texture filtering (true / false)

Devoluções:

Exemplo:

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

O código acima verifica se texture está ligado ou desligado e o liga e desliga de acordo.
