draw_sprite_ext

Esta função irá desenhar o dado sprite como na função draw_sprite() mas com opções adicionais para alterar a escala, mistura, rotação e alfa do sprite a ser desenhado. A mudança destes valores não altera o recurso de forma alguma (apenas como ele é desenhado), e você pode usar qualquer uma das variáveis sprite disponíveis em vez de valores diretos para todos os argumentos da função. A imagem abaixo ilustra como diferentes valores afetam o desenho do duende:

NOTA: A mistura de cores só é recomendada para o alvo HTML5 quando a WebGL está habilitada, embora você ainda possa definir a cor da mistura se ela não estiver habilitada e ela irá misturar o sprite como normal. Entretanto, toda mistura desta forma cria um sprite duplicado que é então armazenado no cache e usado quando necessário. Isto está longe de ser ideal e se você usar várias mudanças de cor, isso irá diminuir o desempenho de seus jogos a menos que você ative o WebGL. se você não quiser usar o WebGL, então você pode definir o tamanho font cache para tentar limitar isto caso seja necessário usando a função sprite_set_cache_size().

NOTA: Esta função pode não funcionar como esperado ao usar animação esqueleto sprites, e você pode descobrir que a função apenas desenha o primeiro quadro da pose padrão. Você deve usar as funções draw_skeleton_* em seu lugar.

Sintaxe:

draw_sprite_ext( sprite, subimg, x, y, xscale, yscale, rot, colour, alpha );

ArgumentoTipo Descrição
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x The x coordinate of where to draw the sprite.
y The y coordinate of where to draw the sprite.
xscale The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale The vertical scaling of the sprite as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot The rotation of the sprite. 0=right way up, 90=rotated 90 degrees counter-clockwise etc...
colour The colour with which to blend the sprite. c_white é exibi-la normalmente.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devoluções:

Exemplo:

draw_sprite_ext(sprite_index, image_index, x, y, image_xscale, image_yscale, image_angle, image_blend, image_alpha);

Isto atrairá as instâncias designadas sprite com todos os seus valores padrão (essencialmente o mesmo que usar draw_self).
