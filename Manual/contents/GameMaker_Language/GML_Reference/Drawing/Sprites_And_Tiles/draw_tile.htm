draw_tile

Com esta função você pode extrair qualquer dado tile de um tile set - competir com rotações, inversões e espelhos - definindo o quadro (se animado) e a posição dentro do room. O valor tile set ID é o índice tile set como definido no IDE e os dados tile são os dados "blob" que contém todas as informações sobre o índice tile e as transformações que foram aplicadas. Você pode usar a função tilemap_get() para obter os dados tile que podem então ser manipulados usando as funções de azulejo, ou você mesmo pode construí-los usando as constantes tile a seguir, juntamente com o valor do índice do tile para criar:

Constante Descrição
tile_rotate Usado para definir/escondicionar o bit de rotação de uma bolha de dados tile.
tile_mirror Usado para definir/esconder o bocado de espelho de uma bolha de dados tile.
tile_flip Usado para definir/escondicionar o flip bit de uma bolha de dados tile.
tile_index_mask Uma constante especial que é para "e" -ing com o tile data blob para extrair o índice tile.


Você "ou" estas constantes com o valor do índice tile obteria os tileados finais:

var tiledata = 8 | tile_rotate | tile_flip;

O código acima simplesmente cria uma variável local para armazenar os mosaicos para o tile com o índice 8 no tile set, e é girado e invertido. Isto pode então ser usado para desenhar o tile naquele índice do tile set que você especificar. isto não está ligado a nenhuma camada do mapa de azulejos e o tile será desenhado na profundidade da camada da instância que faz o desenho, na posição x/y que você especificar no room. Note que você também pode definir o quadro para desenhar. Se o indexado tile fizer parte de um conjunto de animação, então você pode definir aqui o número do quadro (de 0 ao número de quadros - 1) e o indexado tile será simplesmente o indicador para o qual o tile set animado está sendo usado enquanto será o quadro dado que será desenhado. Se o tile não faz parte de um conjunto animado, então o frame deve ser ajustado para 0.

Sintaxe:

draw_tile(tileset, tiledata, frame, x, y)

Argumento Tipo Descrição
tileset Conjunto de azulejos Ativo The ID of the tile set to use
tiledata Dados sobre os azulejos The tile data to use
frame Real The frame number for the animated tile to use (default 0 for non-animated tiles)
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Devoluções:

Real

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

O código acima obterá os dados em um tile a partir do mapa de azulejos dado, depois definir o "flip" flag e usar esses dados para desenhar o tile virado na posição da instância.
