draw_tilemap

Esta função pode ser usada para desenhar um determinado mapa de azulejos em qualquer lugar no room. Você dá o ID do elemento do mapa de azulejos (que você obtém ao criar um elemento do mapa de azulejos usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), depois dar a posição x/y para desenho (no site room). Note que isto simplesmente desenhará o mapa do azulejo no ponto especificado, usando a profundidade da camada da instância que está chamando a função. Ele não move o mapa de azulejos - nem o altera de forma alguma - e não importa se o mapa de azulejos está sinalizado como visível ou não.

Sintaxe:

draw_tilemap(tilemap_element_id, x, y);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tilemap element to draw
x Real The x position within the room to draw at
y Real The y position within the room to draw at

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

O código acima obtém o ID da camada e depois o usa para obter um ID específico do mapa de azulejos que, por sua vez, é usado para desenhar o mapa de azulejos na posição do mouse.
