part_system_draw_order

Com esta função você pode definir a forma como as partículas são desenhadas quando criadas na tela. O sistema padrão usa uma aparência antiga > nova (a função está definida para true), onde as partículas antigas são desenhadas a uma profundidade maior do que as novas e assim aparecem "abaixo" das novas partículas, mas ao definir esta função para false você pode inverter esta ordem e ter as novas partículas desenhadas mais alto e assim aparecem "abaixo" das mais antigas. As imagens abaixo ilustram isto, sendo a imagem à esquerda o valor padrão de true e a imagem à direita a false:

NOTA: Quando as partículas estão sendo desenhadas com um modo de mistura aditiva, o efeito desta função pode nem sempre ser óbvio.

Sintaxe:

part_system_draw_order(ind, oldtonew);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to change.
oldtonew Booleano Whether old particles should be drawn behind newer ones (true) or vice versa (false).

Devoluções:

N/A

Exemplo:

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

Isto criará um novo sistema de partículas com o índice "mysystem", e então define partículas para extrair partículas mais novas sobre as mais antigas.
