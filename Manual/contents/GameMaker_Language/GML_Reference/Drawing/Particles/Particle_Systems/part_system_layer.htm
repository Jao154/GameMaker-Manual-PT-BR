part_system_layer

Esta função pode ser usada para switch um sistema de partículas de sua camada atual para um novo sistema. Você fornece o valor de identificação do sistema de partícula (como retornado pela função part_system_create_layer()) e, em seguida, o ID da camada única (como retornado pela função layer_create() ou o nome da camada a ser usada como um string - por exemplo "instance_layer" - como definido no editor room ), e o sistema será movido para a nova camada.

Sintaxe:

part_system_layer(ps, layer);

Argumento Tipo Descrição
ps Identificação do Sistema de Partículas The particle system ID
layer Identificaçãoporcorda ou camada The new layer ID

Devoluções:

N/A

Exemplo:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

O código acima verificará um sistema de partículas para ver em que camada está e, se não estiver na camada dada, será alterado.
