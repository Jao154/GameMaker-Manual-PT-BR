effect_create_below

Com esta função você pode criar um efeito simples abaixo de todas as instâncias de seu room (ele é realmente criado a uma profundidade de 100000). Se o efeito for qualquer outro ef_rain ou ef_snow então você pode definir uma posição x/y para criar o efeito, e o tamanho pode ser um valor de 0, 1 ou 2, onde 0 é pequeno, 1 é médio e 2 é grande.

É importante notar que estes efeitos podem ter seu desenho ligado e desligado, bem como ter seu desenho pausado, usando as funções part_system_automatic_draw() e part_system_automatic_update() com o valor apropriado para o índice do sistema de partículas (onde 0 é para efeitos abaixo e 1 é para efeitos acima).

As constantes disponíveis para os diferentes tipos de partícula são:

Constante Exemplo Descrição
ef_cloud An effect that creates random cloud particles of varying sizes
ef_ellipse An effect that creates expanding ellipses
ef_explosion An effect that creates expanding fading explosions
ef_firework An effect that creates multiple small particles to generate a firework explosion
ef_flare An effect that generates a brilliant point that flares up and fades out
ef_rain An effect that generates rain particles coming down from the top of the screen
ef_ring An effect that generates expanding and fading circles
ef_smoke An effect that generates little puffs of smoke
ef_smokeup An effect that creates a smoke plume that rises up the screen
ef_snow An effect that generates multiple snow particles falling down the screen
ef_spark An effect that generates a small spark
ef_star An effect that generates star particles

Sintaxe:

effect_create_below(kind, x, y, size, colour);

Argumento Tipo Descrição
kind Tipo de Efeito Constante The kind of effect (use one of the constants listed above).
x Real The x positioning of the effect if relevant.
y Real The y positioning of the effect if relevant.
size Real The size of the effect.
colour Cor The colour of the effect.

Devoluções:

N/A

Exemplo:

if speed > 0
{
    effect_create_below(ef_smoke, x, y, 0, c_gray);
}

O código acima criará um pequeno sopro de fumaça cinza a cada passo que a velocidade da instância for maior que 0 nas coordenadas da instância x,y.
