part_emitter_existe

Com esta função, é possível verificar se o emissor de partículas em questão existe ou não no sistema em questão. Note que se a variável a ser verificada for uma variável não-inicializada (que de outra forma um emissor de partícula teria seu índice atribuído), isto lançará um erro.

Sintaxe:

part_emitter_exists(ps, ind);

Argumento Tipo Descrição
ps Identificação do Sistema de Partículas The particle system to check for an emitter.
ind Identificação do Emissor de Partículas The index of the emitter to search for.

Devoluções:

Boolean

Exemplo:

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

O código acima verificará o emissor indexado na variável "permissão" e, se existir, rebentará 30 partículas da mesma.
