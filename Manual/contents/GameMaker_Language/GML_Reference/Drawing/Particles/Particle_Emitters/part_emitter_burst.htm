part_emitter_burst

Esta função permite definir um emissor para rebentar um tipo específico de partícula e é tipicamente usada em alarmes e eventos de destruição, pois é um código único que cria o número de partículas especificadas de uma só vez, seguindo a distribuição, forma e posição definida pela função part_emitter_region(). Caso você precise que as partículas apareçam a cada passo, você deve estar usando a função part_emitter_stream() em vez de chamar esta função a cada passo.

Sintaxe:

part_emitter_burst(ps, ind, parttype, number);

Argumento Tipo Descrição
ps Identificação do Sistema de Partículas The particle system that the emitter is in.
ind Identificação do Emissor de Partículas The index of the emitter to burst from.
parttype Identificação do tipo de partícula The index (type) of the particles to be created.
number Real The number of particles to create.

Devoluções:

N/A

Exemplo:

part_emitter_burst(global. Sname, p_emit1, p1, 30 + irandom(30));

O código acima irá estourar um número aleatório de partículas entre 30 e 60 do emissor escolhido.
