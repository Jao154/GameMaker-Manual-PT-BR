vértice_color

Se o seu formato vertex definido tiver um valor de cor, você pode usar esta função para adicionar esses dados ao vertex que está sendo definido para o primitivo atual. A função precisa de um buffer para armazenar os dados e terá um valor de cor constante ou hexadecimal (usando o formato padrão do GameMaker BGR, por exemplo: $FF0000 para azul), bem como um valor alfa de 0 (transparente) a 1 (totalmente opaco).

Sintaxe:

vertex_colour(buffer, colour, alpha);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
colour Cor The colour for this vertex (can be a constant or a hex value).
alpha Real The alpha value for the vertex (from 0 to 1).

Devoluções:

N/A

Exemplo:

vertex_colour(b, c_white, 1);

O código acima definirá o vertex como sendo branco com um valor alfa de 1.
