vertex_get_buffer_size

Com esta função você pode obter o tamanho do dado vertex buffer em bytes.

Sintaxe:

vertex_get_buffer_size(buffer);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to get the size of.

Devoluções:

Real

Exemplo:

bufferBytes = vertex_get_buffer_size(buff);

O código acima obterá o número de bytes utilizados pelo vertex buffer dado e armazenará o valor na variável "bufferBytes".
