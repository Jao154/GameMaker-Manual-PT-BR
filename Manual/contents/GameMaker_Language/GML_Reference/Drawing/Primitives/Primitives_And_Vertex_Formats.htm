Primitivos e Vertex Formatos

Um primitivo é um tipo de desenho feito usando pontos para definir o que é mostrado na tela. Eles podem ser compostos de pontos únicos, linhas ou triângulos e podem ser coloridos, ter uma mistura alfa e até receber um texture, o que significa que eles podem ser usados para criar alguns efeitos muito excitantes em um jogo 2D! Por exemplo, você pode criar um retângulo usando múltiplos triângulos primitivos, depois texture com um flag sprite e no evento de sorteio de uma instância ter que flag "acenar" ao vento e distorcer o sprite que você usou simplesmente modificando os pontos que formam o primitivo.

ADVERTÊNCIA! Estas funções não funcionam com o módulo HTML5 a menos que você tenha ativado o OpenGL nas Opções de Jogo.

IMPORTANTE! Os primitivos que você constrói devem seguir o formato que você definiu usando as funções do formato Vertex, portanto, se você definiu um formato vertex com apenas dados de posicionamento, não adianta construir seu primitivo com dados de cor. Você deve observar que a ordem na qual você adiciona propriedades ao primitivo que está construindo é definida pela ordem na qual você adicionou estas propriedades ao criar o formato vertex, então se você definiu o formato vetorial com a posição de ordem, cor e coordenada texture, você deve adicionar estas propriedades ao primitivo que está construindo na mesma ordem, caso contrário você receberá erros.
