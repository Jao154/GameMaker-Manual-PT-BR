vértice_float1

Esta função adicionará um valor de ponto flutuante aos dados vertex. O vertex deve ter sido formatado corretamente para aceitar isto usando o vertex_format_add_custom() função.

Sintaxe:

vertex_float1(buffer, float);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
float Real The input value.

Devoluções:

N/A

Exemplo:

vertex_float1(buff, 0.05);

O código acima adicionará um valor de ponto flutuante aos dados vertex que estão sendo definidos.
