vertex_format_add_texcoord

Diga ao GameMaker para aceitar texture dados de posição (u e v) como parte do novo formato vertex que está sendo criado.

Sintaxe:

vertex_format_add_texcoord();

Devoluções:

Exemplo:

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

O código acima criará um novo formato vertex com apenas texture e valores de posição e então armazenará o id de formato na variável "my_format".
