vertex_format_end_end

Esta função deve ser chamada após a definição de qualquer novo formato vertex. Ela retorna o novo formato "handle" (índice) que deve ser usado em todas as outras funções vertex que se referem a este novo formato.

Sintaxe:

vertex_format_end();

Devoluções:

Vertex Format ID

Exemplo:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

O código acima criará um novo formato vertex com apenas valores de cor e posição e então armazenará o id de formato na variável "my_format".
