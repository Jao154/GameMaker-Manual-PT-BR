vértice_float3

Esta função adicionará três valores de ponto flutuante aos dados vertex. O vertex deve ter sido formatado corretamente para aceitar isto usando o vertex_format_add_custom() função.

Sintaxe:

vertex_float3(buffer, float, float, float);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
float Real The first input value.
float Real The second input value.
float Real The third input value.

Devoluções:

N/A

Exemplo:

vertex_float3(buff, 0.05, 0.01, room_width / x);

O código acima adicionará três valores de ponto flutuante aos dados vertex que estão sendo definidos.
