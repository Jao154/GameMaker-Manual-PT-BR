vértice_get_number

Com esta função você pode descobrir o número de vértices individuais definidos em qualquer vertex buffer.

Sintaxe:

vertex_get_number(buffer);

Argumento Tipo Descrição
buffer Vertex Buffer ID The vertex buffer to check.

Devoluções:

Real

Exemplo:

v_num = vertex_get_number(v_buffer);

O código acima armazenará o número de vértices armazenados no dado vertex buffer a uma variável.
