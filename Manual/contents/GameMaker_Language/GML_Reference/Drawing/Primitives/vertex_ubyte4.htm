vértice_ubyte4

Esta função adicionará quatro valores de bytes não assinados (0 - 255) aos dados vertex. O vertex deve ter sido formatado corretamente para aceitar isto usando a função vertex_format_add_custom() função.

Sintaxe:

vertex_ubyte4(buffer, byte, byte, byte, byte);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
byte Real The first input value.
byte Real The second input value.
byte Real The third input value.
byte Real The fourth input value.

Devoluções:

N/A

Exemplo:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

O código acima acrescentará quatro valores aos dados vertex que estão sendo definidos.
