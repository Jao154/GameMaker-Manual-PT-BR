vertex_create_buffer_ext_ext

Como com a função vertex_create_buffer()esta função irá criar uma nova vertex buffer. Este é um crescimento especial buffer criado pelo GameMaker que é pré-formatado para uso na construção de primitivos para uso com shaders. Você pode especificar um tamanho inicial inicial para o buffer (em bytes) e ele retornará um índice para o buffer que deverá então ser usado em todas as chamadas posteriores para o buffer.

Sintaxe:

vertex_create_buffer_ext(size);

ArgumentoTipo Descrição
size The initial size of the buffer (in bytes).

Devoluções:

Vertex Buffer ID

Exemplo:

v_buff = vertex_create_buffer_ext(1024 * 1024);

O código acima criará um novo vertex buffer, inicialmente de 1MB de tamanho, e armazenará seu cabo na variável "v_buff".
