vértice_argb

Esta função definirá os valores ARGB para o vertex que está sendo definido atualmente para os primitivos personalizados. Você fornece o buffer para escrever os dados assim como os valores vermelho, verde, azul e alfa que você deseja usar como um único inteiro de 32 bits sem assinatura - amostra alfa nos 8 bits mais altos, seguido pela amostra vermelha, amostra verde e finalmente a amostra azul nos 8 bits mais baixos. Você pode usar a notação hexadecimal ($AARRGGBB) um número real ou usar qualquer uma das funções make_colour_*() para definir o valor da cor.

Sintaxe:

vertex_argb(buffer, argb);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to write the information to.
argb Cor The ARGB value to set.

Devoluções:

N/A

Exemplo:

vertex_argb(buff, $FFFFFFFF);

O código acima definirá os valores ARGB do vertex sendo definido como branco.
