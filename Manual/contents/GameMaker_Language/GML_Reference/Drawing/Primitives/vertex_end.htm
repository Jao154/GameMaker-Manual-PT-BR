vertex_end

Com esta função, você termina a construção do primitivo personalizado. Uma vez que você chama este comando, o primitivo pode ser usado no vertex_submit() para uso em um shader ou você pode congelar o buffer (tornando o vertex buffer usado somente para leitura e muito mais rápido) usando vertex_freeze().

Sintaxe:

vertex_end(buffer);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to end writing to.

Devoluções:

N/A

Exemplo:

vertex_end(buff);

O código acima terminará a definição de um primitivo personalizado.
