vértice_formato_delete

Esta função deve ser chamada sempre que você terminar de usar qualquer formato vetorial criado. Você fornece o valor de identificação do formato (como retornado pela função vector_format_end()), e esta função libertará a memória associada a ela. Note que se você tentar usar este formato novamente após chamar esta função, você receberá um erro.

Sintaxe:

vertex_format_delete(formatID);

Argumento Tipo Descrição
formatID Identificação em formato de vértice The vertex format to delete

Devoluções:

N/A

Exemplo:

vertex_format_delete(myFormat);

O código acima removerá da memória o formato vertex criado na variável "myFormat".
