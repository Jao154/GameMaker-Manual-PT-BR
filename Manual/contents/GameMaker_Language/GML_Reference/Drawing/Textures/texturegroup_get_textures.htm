texturas_grupo_get_textures

Esta função pode ser usada para recuperar os IDs das páginas de textura das páginas individuais que compõem um grupo texture. Você fornece o ID do grupo texture string (conforme definido no Editor do grupo Texture ), e a função retornará um array 1D, onde cada entrada no array é um único ID de página texture. Se a função falhar - ou seja: um grupo inválido é dado, ou o grupo não tem texture atribuído a ele - então o array estará vazio (0 comprimento).

Sintaxe:

texturegroup_get_textures(tex_id);

Argumento Tipo Descrição
tex_id Cordão The name of the texture group to check (a string)

Devoluções:

Array of Texture Pointers

Exemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

O código acima recuperará os IDs de página texture para o grupo texture "MainMenu", então verifique se eles estão desembalados, e se eles são eles são colocados na VRAM.
