escala_global_textura

Esta função permite que você controle o escalonamento das páginas texture em carga a partir do arquivo WAD que faz parte de seu executável final do jogo. O valor de entrada deve ser um poder de dois valores e funcionará de tal forma que:

  • Para definir as páginas texture a serem usadas 1:1 texture_global_scale(1)
  • Para a metade textures o tempo todo use texture_global_scale(2)
  • Para aquartelá-los usar texture_global_scale(4), etc...

Desta forma, você pode controlar melhor o uso da memória da página texture em plataformas com problemas de pouca memória. Note que as páginas textures são criadas sob demanda do WAD, e assim você pode chamar isso no início do jogo para carregar todas as páginas textures em escala, ou você pode chamá-lo em horários específicos para carregar textures específico, caso em que você provavelmente gostaria de ligar draw_texture_flush() antes de chamar esta função, então use as funções de pré-viagem para obter o correto texture páginas na memória.

Sintaxe:

texture_global_scale(pow2integer);

ArgumentoTipo Descrição
pow2integer The scale factor to use (1, no scale, 2, half scale, 4, quarter scale, etc...)

Devoluções:

Exemplo:

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

O código acima irá limpar todo textures da memória, depois definir a escala de texture para 2 (de modo que as páginas texture são metade do tamanho) e, finalmente, pré-passar a página texture que contém o sprite "spr_Trees".
