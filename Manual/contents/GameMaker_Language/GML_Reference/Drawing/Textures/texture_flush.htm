textura_flush

Esta função pode ser usada para "enxaguar" uma página texture ou um grupo de páginas texture, ou seja: removê-los da VRAM quando não forem mais necessários. Você fornece o ID da página de textura única (como retornado pelas funções texturegroup_* ) para lavar uma única página, ou você pode fornecer um nome de grupo de textura (como definido no Editor de Grupo de Textura) para lavar todas as páginas texture do grupo.

Sintaxe:

texture_flush(tex_id);

Argumento Tipo Descrição
tex_id Ponteiro de Textura ou Cordão The texture page pointer or a texture group name (a string)

Devoluções:

N/A

Exemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_flush(_tex_array[i]);
}

O código acima irá lavar todas as páginas texture sob o grupo texture "MainMenu".
