textura_set_set_stage

Esta função irá definir o "slot" do estágio em questão um texture a ser usado. O número de "slots" de estágio disponíveis dependerá da plataforma para a qual você está compilando, com um máximo de 8 disponíveis para Windows, Mac e Linux, mas em dispositivos inferiores Android (por exemplo) este número pode ser tão baixo quanto 2. Também vale a pena notar que o primeiro slot de estágio (1) é sempre usado automaticamente pelo GameMaker.

OBSERVAÇÃO: Esta função não fará nada fora do contexto de uma shader em execução! Veja Shaders para mais informações.

Sintaxe:

texture_set_stage(stage, tex);

Argumento Tipo Descrição
stage Real The texture "slot" to use.
tex Ponteiro de Textura The texture to use.

Devoluções:

N/A

Exemplo:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

O código acima obterá o cabo para o amostrador dentro do shader indexado como "GlassShader" e então definirá que shader constante para o dado sprite texture.
