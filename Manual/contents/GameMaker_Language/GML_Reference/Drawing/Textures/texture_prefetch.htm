textura_prefetch

Esta função pode ser usada para "pré-fetar" uma página texture ou um grupo de páginas texture, ou seja: carregá-las na VRAM quando necessário. Você fornece o ID da página de textura única (como retornado pelas funções texturegroup_* ) para préfetar uma única página, ou pode fornecer um nome de grupo de textura (como definido no Editor do Grupo de Textura) para préfetar todas as páginas texture do grupo.

Sintaxe:

texture_prefetch(tex_id);

Argumento Tipo Descrição
tex_id Ponteiro de Textura ou Cordão The texture page pointer or a texture group name (a string)

Devoluções:

N/A

Exemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_prefetch(_tex_array[i]);
}

O código acima prefetará todas as páginas texture sob o grupo texture "MainMenu".
