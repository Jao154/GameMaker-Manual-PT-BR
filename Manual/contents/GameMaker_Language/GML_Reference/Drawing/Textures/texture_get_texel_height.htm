textura_get_texel_height

Um texel, ou elemento de textura, é a unidade fundamental do texture espaço usado na computação gráfica. Textures são representados por matrizes de textos, assim como as imagens são representadas por matrizes de pixels, e esta função retorna a altura de um único texel da página texture da imagem asset usada.

Sintaxe:

texture_get_texel_height(tex);

Argumento Tipo Descrição
tex Ponteiro de Textura The texture page asset pointer to use

Devoluções:

Real

Exemplo:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

O código acima terá a largura e a altura do texel texture retirado de um sprite asset.
