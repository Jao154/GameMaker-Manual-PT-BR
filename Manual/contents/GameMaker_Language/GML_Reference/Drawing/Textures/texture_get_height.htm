Textura_esquadre_altura

Retorna a altura do texture com o id dado, que é sempre um valor dentro da faixa 0 - 1. Isto pode então ser usado no mapeamento textures para modelos ou primitivos.

Sintaxe:

texture_get_height(tex);

Argumento Tipo Descrição
tex Ponteiro de Textura The texture page asset pointer to use

Devoluções:

Real

Exemplo:

tex_h = texture_get_height(surface_get_texture(global.Surf));

O código acima terá a altura do texture retirado de uma superfície previamente criada.
