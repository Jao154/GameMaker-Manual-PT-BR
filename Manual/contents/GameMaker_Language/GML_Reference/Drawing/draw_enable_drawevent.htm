draw_enable_drawevent

Com esta função você pode optar por ativar (true) ou desativar (false) o evento de sorteio para todas as instâncias do jogo, dando-lhe assim controle sobre como e quando as coisas são desenhadas, o que é útil se você desejar implementar uma técnica de "frame skip". Note que isto não apenas impede que as instâncias sejam desenhadas na tela, mas também suprime completamente o evento de sorteio, o que significa que é preciso ter cuidado, pois qualquer lógica de jogo que esteja presente naquele evento também não será executada.

Uma coisa importante a entender sobre esta função é que se você a chamar logo no início do jogo, antes que o frame inicial seja renderizado (ou seja: o Create Event do primeiro object no primeiro room do jogo), então a janela do jogo não será renderizada. Isto pode ser útil para projetos multiplayer que requerem um server dedicado que não precisa renderizar nada, porém tenha em mente que isto não o torna sem cabeça, pois GameMaker não suporta isto -- assim você não poderá executá-lo em um server que não tenha uma interface gráfica.

Sintaxe:

draw_enable_drawevent(enable);

ArgumentoTipo Descrição
Enable Set to true ou false.

Devoluções:

Exemplo:

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

O código acima verifica uma variável e se ela retornar true, ela permite o evento de sorteio, caso contrário o evento de sorteio é desativado.
