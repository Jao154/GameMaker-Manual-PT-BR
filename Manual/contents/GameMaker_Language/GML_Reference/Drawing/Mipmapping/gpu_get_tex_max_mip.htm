gpu_get_tex_max_mip

Com esta função você pode obter o nível máximo de mipmap atualmente definido que deve ser usado, onde um valor de 0 é a maior resolução, 1 é usar o primeiro mipmap, 2 é o segundo etc... Note que isto pode ser bastante útil para evitar artefatos de sangramento ao renderizar textures, por exemplo, ajustando a borda da página texture para 8px e depois ajustando o nível máximo de mipmap para 3, garantirá que você não terá nenhum problema de sangramento a distâncias maiores de renderização.

Sintaxe:

gpu_get_tex_max_mip();

Devoluções:

(default: 16)

Exemplo:

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

O código acima verificará o nível máximo atual do mipmap e, se não for 4, está ajustado para 4.
