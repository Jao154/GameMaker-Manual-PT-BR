gpu_get_tex_mip_bias

Com esta função você pode recuperar o valor do viés do mipmap, onde um valor de 0 é igual a nenhum viés, 1 é igual ao primeiro mipmap, 2 é igual ao segundo mipmap etc... Isto controla a taxa na qual o mapa mip é trocado e geralmente fará com que textures fique mais borrado quanto maior o valor e maior a "distância" sendo visualizada. Note que isto também pode retornar valores negativos, caso em que a renderização textures será mais nítida em uma distância maior quanto menor for o valor.

Sintaxe:

gpu_get_tex_mip_bias();

Devoluções:

(default: 0)

Exemplo:

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

O código acima verificará o viés atual do mipmap e, se não for 0, será ajustado para 0.
