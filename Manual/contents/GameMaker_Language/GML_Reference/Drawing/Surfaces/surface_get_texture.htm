superfície_get_texture

Esta função retorna um ponteiro especial para a superfície texture. Este valor pode então ser usado em outras funções de desenho, particularmente em geral 3D e em algumas das funções primitivas 2D.

NOTA Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória texture. Você deve SEMPRE verificar se uma superfície existe usando surface_exists() antes de referenciá-los diretamente.

Sintaxe:

surface_get_texture(surface_id);

Argumento Tipo Descrição
surface_id Identificação da superfície The ID of the surface to get the width of.

Devoluções:

Texture

Exemplo:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

O código acima desenhará uma faixa triangular 4 vertex texturizada com o texture mantido na variável "tex", que por sua vez é retirada de uma superfície previamente criada.
