superfície_get_target_ext_ext

Esta função recuperará o ID de superfície atribuído a um dos 4 alvos de renderização disponíveis para as superfícies. Você fornece o índice do alvo de renderização para verificar, e a função retornará -1 se nenhuma superfície for atribuída, ou um valor inteiro >= 0, representando o ID da superfície atribuída (como retornado pela função surface_create()).

Sintaxe:

surface_get_target_ext(index);

ArgumentoTipo Descrição
index The render target index to check (from 0 to 3).

Devoluções:

Surface ID or -1 (if no target surface is set)

Exemplo:

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

O código acima irá primeiro verificar e ver se o shader renderiza o alvo 0 foi definido para uma superfície e, se não, então um é atribuído.
