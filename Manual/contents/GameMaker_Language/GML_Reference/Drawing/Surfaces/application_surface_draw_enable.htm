application_surface_draw_enable

Você pode usar esta função para ativar ou desativar o desenho automático da superfície da aplicação. Por padrão isto está habilitado, mas em muitos casos você vai querer assumir quando e como a superfície é desenhada (ao utilizar shaders por exemplo), em cujo caso você usaria esta função para defini-la como falsa, para que você mesmo a possa desenhar quando e como você precisar. É importante observar que quando você switch desenha automaticamente e desenha a superfície da aplicação você mesmo, que você pode ver certos problemas com o componente alfa de sprites e a própria superfície. Isto porque o GameMaker desenhará a superfície de aplicação sem mistura alfa quando o desenho automático estiver ligado. Se você switch estiver fora do desenho automático, então você precisa para lidar com isso você mesmo, usando o seguinte código (por exemplo):

gpu_set_blendenable(false);
draw_surface_ext(application_surface, 0, 0, 1, 1, 0, c_white, 1);
gpu_set_blendenable(true);

Sintaxe:

application_surface_draw_enable(flag);

ArgumentoTipo Descrição
flag The flag will be either true (ativado, o valor padrão) ou false (desativado).

Devoluções:

Exemplo:

application_surface_draw_enable(false);

O código acima será switch do desenho automático da superfície da aplicação.
