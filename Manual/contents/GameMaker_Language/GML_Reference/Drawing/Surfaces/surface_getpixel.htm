surface_getpixel

Esta função pode ser usada para obter a cor de um pixel específico de uma superfície, usando as coordenadas locais da superfície onde (0,0) é o canto superior esquerdo. Esta função não deve ser usada com muita freqüência, pois é extremamente lenta e pode causar uma pausa em seu jogo.

NOTA: Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória texture. Você deve SEMPRE verificar se uma superfície existe usando surface_exists() antes de referenciá-los diretamente.

Sintaxe:

surface_getpixel(surface_id, x, y);

ArgumentoTipo Descrição
surface_id The ID of the surface.
x The x position on the surface from which to get the pixel.
y The y position on the surface from which to get the pixel.

Devoluções:

Colour Constant

Exemplo:

col = surface_getpixel(surf, 56, 78 );

Isto retornará a cor do pixel nas coordenadas (56,78) da superfície indexada na variável "surf".
