cópia_da_superfície

Esta função simplesmente tira a imagem de uma superfície e a copia para outra na posição local especificada dentro daquela superfície (onde a posição (0,0) é o canto superior esquerdo da superfície de destino). Se a superfície de destino já tiver informações, estas serão sobrescritas pela cópia, e a função não altera de forma alguma a superfície de origem.

NOTA: Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória texture. Você deve SEMPRE verificar se uma superfície existe usando surface_exists() antes de referenciá-los diretamente.

Sintaxe:

surface_copy(destination, x, y, source);

Argumento Tipo Descrição
destination Identificação da superfície The ID of the surface to copy the other surface to.
x Real The x position to copy to.
y Real The y position to copy to.
source Identificação da superfície The ID of surface to be copied.

Devoluções:

N/A

Exemplo:

if view_current == 0
{
    surface_copy(surf, 0, 0, temp_surf);
}
else
{
    draw_surface(surf, 0, 0);
}

O código acima verificará a vista atual sendo desenhada e se é vista[0] copia a superfície indexada na variável "temp_surf" para a superfície indexada na variável "surf". Se a vista atual for outra coisa além da vista[0], a superfície "surf" é desenhada para a tela.
