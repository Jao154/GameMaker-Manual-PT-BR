largura_da_superfície

Esta função simplesmente retorna a largura, em pixels, da superfície indexada. Deve-se notar que se você chamar isso para verificar o application_surface imediatamente após ter alterado seu tamanho usando surface_resize() não devolverá o novo valor, pois a mudança precisa de um ou dois passos para ser totalmente processada. Depois de esperar uma etapa, deverá devolver o novo valor corretamente.

NOTA: Quando se trabalha com superfícies há a possibilidade de que elas possam deixar de existir a qualquer momento devido ao fato de serem armazenadas na memória texture. Você deve SEMPRE verificar se uma superfície existe usando surface_exists() antes de referenciá-los diretamente.

Sintaxe:

surface_get_width(surface_id);

ArgumentoTipo Descrição
surface_id The ID of the surface to get the width of.

Devoluções:

Exemplo:

sw = surface_get_width(surf);

O código acima irá armazenar a largura da superfície indexada na variável "surf" na variável "sw".
