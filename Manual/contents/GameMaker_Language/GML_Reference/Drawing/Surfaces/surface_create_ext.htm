superfície_criar_texto

Esta função permite anexar uma superfície a um elemento de tela que já existe em sua página web, o que significa que você pode efetivamente dividir porções de seu jogo a serem desenhadas em vários lugares diferentes dentro da página. Para isso, você deve ter definido corretamente o elemento de tela dentro da página *.html de seu jogo usando os tamanhos e nomes corretos que correspondem às superfícies que você deseja criar. Assim, você teria sua tela "principal" e, em seguida, seus elementos secundários de tela de superfície, que serão designados usando esta função para as superfícies corretas. A imagem a seguir é um exemplo de como uma página com três elementos de tela seria configurada:

O layout da página pode ser complicado, especialmente se você desejar que todos os elementos se alinhem corretamente, mas uma vez concluída a difícil tarefa de criar o layout, você pode então adicionar este arquivo html como o arquivo de página padrão para o jogo usando a guia HTML5 das Opções do Jogo. A próxima coisa que você deve fazer é configurar seu room e vistas, pois cada superfície precisará ser associada a uma vista específica para "capturar" as imagens do jogo (veja a variável de vista view_surface_id[0...7]). A imagem abaixo mostra como o jogo room para o exemplo de lona acima seria apresentado:

Finalmente, você usaria esta função para criar as superfícies, sendo o nome igual ao usado para os elementos da tela e o tamanho correspondente ao tamanho daquela mesma tela. A função retornará o índice da superfície que deverá ser armazenada em uma variável para futuras chamadas de função. Quando a superfície é criada pela primeira vez, ela pode conter "ruído", pois basicamente é apenas uma área de memória que é posta de lado para o propósito (e que a memória ainda pode conter informações), portanto você pode querer limpar a superfície antes de usar com uma função como draw_clear_alpha().

NOTA: Esta função só está disponível para uso com o módulo HTML5.

Sintaxe:

surface_create_ext(name, w, h);

ArgumentoTipo Descrição
name The name of the canvas element to link the surface to.
w The width of the surface to be created.
h The height of the surface to be created.

Devoluções:

Surface ID

Exemplo:

s1 = surface_create_ext("surface1", 192, 550);
s2 = surface_create_ext("surface2", 608, 186);
view_surface_id[1] = s1;
view_surface_id[2] = s2;

O código acima cria duas superfícies de tamanhos diferentes, atribuindo cada uma a um elemento de tela diferente, e então essas superfícies são atribuídas a duas vistas para que a parte correta do room seja capturada.
