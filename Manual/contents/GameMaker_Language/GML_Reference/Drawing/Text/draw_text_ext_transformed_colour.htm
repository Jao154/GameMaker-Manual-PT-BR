draw_text_ext_transformed_colour

Esta função é uma combinação da base draw_text() função com o draw_text_ext(), draw_text_transformed() e draw_text_colour() funções. Como tal, esta é a mais geral de todas as funções para desenhar texto e dá uma grande flexibilidade ao desenhar. Note que a "largura O argumento é baseado em uma escala de 1, portanto, se a escala for diferente, este valor deve ser alterado proporcionalmente. Por exemplo, se a largura de base para uma quebra de linha for 300 e você definir a escala para 2, então o texto parecerá errado, ultrapassando o valor dado largura. Em vez disso, você deveria ter ajustado a largura para 150 para compensar a escalada.

NOTA: A mistura gradual não está disponível para o alvo HTML5 a menos que a WebGL esteja habilitada, embora você ainda possa definir as cores da mistura e ela irá misturar o font com a primeira cor dada. Entretanto, toda mistura desta forma cria uma duplicata font que é então armazenado no cache e usado quando necessário, o que está longe de ser o ideal e se você usar mudanças múltiplas de cor irá diminuir o desempenho de seus jogos. Você pode definir o tamanho font cache para tentar limitá-lo, caso seja necessário. usando a função font_set_cache_size().

Sintaxe:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentoTipo Descrição
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Devoluções:

Exemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Isto desenhará o texto dado com um gradiente de azul a azul escuro para baixo, na posição (200.200) no room, girado para estar de cabeça para baixo e na metade do tamanho normal do ponto font. Haverá 20 pixels entre cada linha caso haja mais de uma linha desenhada e cada linha terá uma largura máxima de 200 pixels.
