draw_text_ext_colour

Esta função é uma combinação da base draw_text() função com o draw_text_ext() e draw_text_colour() permitindo definir cores gradientes para o texto, assim como o espaçamento entre linhas e a largura máxima por linha, tudo junto.

NOTA A mistura gradual não está disponível para o alvo HTML5 a menos que a WebGL esteja habilitada, embora você ainda possa definir as cores da mistura e ela irá misturar o font com a primeira cor dada. Entretanto, toda mistura desta forma cria um font duplicado que é então armazenado no cache e usado quando necessário, o que está longe de ser o ideal e se você usar mudanças múltiplas de cor irá diminuir o desempenho de seus jogos. Você pode definir o tamanho font cache para tentar limitar isso caso seja necessário, usando a função font_set_cache_size().

Sintaxe:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Argumento Tipo Descrição
x Real The x coordinate of the drawn string.
y Real The y coordinate of the drawn string.
string Cordão The string to draw.
sep Real The distance in pixels between lines of text.
w Real The maximum width in pixels of the string before a line break.
c1 Cor The colour for the top left of the drawn text.
c2 Cor The colour for the top right of the drawn text.
c3 Cor The colour for the bottom right of the drawn text.
c4 Cor The colour for the bottom left of the drawn text.
alpha Real The alpha for the text.

Devoluções:

N/A

Exemplo:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

O código acima desenhará qualquer texto que o usuário digitar no teclado, dividindo-o em novas linhas toda vez que o comprimento string para aquela linha exceder 300 pixels. o código também manterá uma separação de 5 pixels entre linhas caso isso ocorra. Cada linha do texto será colorida usando um gradiente azul, com azul claro na parte superior e azul escuro na parte inferior.
