flexpanel_node_get_data

This function returns the data struct of a node.

If the node was given a data member in its initial struct or JSON when it was created, this will return that.

If no data member was set when creating the node, it will have been initialised with an empty data struct anyway, and the function will return that so you can populate it with members.

The returned struct is not a copy of the node's data struct and is a reference to that struct directly, so modifying the struct will affect the data returned in future calls to this function on the same node.

 

Syntax:

flexpanel_node_get_data(root);

Argument Type Description
root Flex Panel Node The node to get the data struct of

 

Returns:

Struct
