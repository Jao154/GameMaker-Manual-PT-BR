Units

There are some properties that can take either an absolute pixel value or a percentage value. In a setter function, like flexpanel_node_style_set_width, you can specify the unit along with the value, using one of the following enum members:

Constant Description
flexpanel_unit.point Interprets the value as number of pixels
flexpanel_unit.percent Interprets the value as a percentage
flexpanel_unit.auto Uses automatic sizing for properties that support it

Getter functions for such properties, such as flexpanel_node_style_get_width, will return a struct with two members:

Member Data Type Description
unit Flex Panel Unit Constant The unit of the value, one of the enum members in the table above
value Real The value itself
