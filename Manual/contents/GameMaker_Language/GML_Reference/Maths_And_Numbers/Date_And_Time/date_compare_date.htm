data_compare_date

Com esta função você pode verificar duas datas para ver qual é a anterior ou a posterior à outra. A função retorna -1 se a data1 for anterior, 0 se ambas as datas forem as mesmas, e 1 se a data1 for posterior.

Sintaxe:

date_compare_date( date1, date2 );

Argumento Tipo Descrição
date1 Data/hora The first date.
date2 Data/hora The date to compare it to.

Devoluções:

Real

Exemplo:

d = date_compare_date(date_create_datetime(2011, 9, 15, 11, 4, 0), date_current_datetime());

Isto definiria "d" para o valor correspondente, dependendo de qual das datas fosse a mais antiga, provavelmente 1, já que a data atual estaria mais adiantada do que 15 de setembro de 2011.
