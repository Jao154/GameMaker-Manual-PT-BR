data_set_timezone

Define o fuso horário base a ser usado para todas as demais funções de data e hora. Este fuso horário pode ser local (conforme definido pelo sistema) ou UTC, e você usaria uma das seguintes constantes para definir qual está sendo usada (por padrão, esta é a hora local):

Constante Descrição
timezone_local usar o fuso horário local conforme definido pelo sistema
timezone_utc uso Tempo Universal Coordenado

Sintaxe:

date_set_timezone(timezone);

Argumento Tipo Descrição
timezone Fuso Horário Constante The time zone to use for the base time.

Devoluções:

N/A

Exemplo:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

Este código verifica a configuração do fuso horário base do jogo e, se não for UTC, ele o altera.
