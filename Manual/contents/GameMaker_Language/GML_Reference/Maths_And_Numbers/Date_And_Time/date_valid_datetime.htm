data_valid_datetime

Com esta função você pode verificar um valor de data/hora para ver se é válido (retorna true) ou não (retorna false). Note que esta função só considerará uma data de hora válida após 1/1/1970 e qualquer coisa antes disso retornará false, portanto, o mais cedo que você puder verificar será o mais cedo possível:

date_valid_datetime(1970, 01, 01, 0, 0, 0);

Sintaxe:

date_valid_datetime(year, month, day, hour, minute, second);

ArgumentoTipo Descrição
year The year to check.
month The month to check.
day The day to check.
hour The hour to check.
minute The minute to check.
second The second to check.

Devoluções:

Boolean

Exemplo:

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
    mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

Isto definirá "mydatetime" para 15 de setembro de 2011, 10:03.30, se for um valor válido (que é).
