data_apoio_ano

Esta função retornará true se o componente de ano do valor da data é um ano bissexto ou false caso contrário. Esta pode ser uma função útil para coisas como Ovos de Páscoa em seus jogos, ou para desbloquear conteúdo especial.

Sintaxe:

date_leap_year(date);

ArgumentoTipo Descrição
date The datetime to use.

Devoluções:

Exemplo:

if date_leap_year(date_current_datetime())
{
    if (!global.ExtraContent)
    {
        global.ExtraContent = true;
    }
}

O código acima verificará a data atual para ver se o ano é um ano bissexto ou não. Se for, definirá uma variável global.
