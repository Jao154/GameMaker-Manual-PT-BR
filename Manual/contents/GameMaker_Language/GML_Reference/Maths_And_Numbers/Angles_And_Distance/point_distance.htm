point_distance

Esta função retorna o comprimento de um vetor formado pelos componentes especificados [x1,y1] e [x2,y2]. Por exemplo, na imagem abaixo, se quisermos obter a distância entre a posição do navio do jogador e a posição do inimigo para que possamos calcular se o inimigo está suficientemente próximo para atirar, então usaríamos esta função (o código exato está no exemplo abaixo da imagem):

Sintaxe:

point_distance(x1, y1, x2, y2);

ArgumentoTipo Descrição
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector

Devoluções:

Exemplo:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if (point_distance(x, y, ex, ey) < 200)
{
    instance_create_layer(x, y, "Bullets", obj_Missile)
}

O código acima obterá as coordenadas x e y do inimigo mais próximo e depois as utilizará para verificar a distância (comprimento) do vetor formado por elas e as coordenadas do jogador. Se o valor for inferior a 200, o jogador object irá criar uma instância de "obj_Missile".
