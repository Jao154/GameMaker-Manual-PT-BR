distância_a_objeto

Esta função calcula a distância da borda da caixa de delimitação da instância de chamada até a borda mais próxima da instância mais próxima da object especificada. O object pode ser um índice object ou uma identificação específica da instância, bem como o palavra-chave outros, e a distância é devolvida em pixels. Note que se qualquer um dos objects não tem sprite ou não tem máscara definida, os resultados serão incorretos.

Sintaxe:

distance_to_object(obj);

Argumento Tipo Descrição
obj Objeto Ativo ou ID da instância The object to check for.

Devoluções:

Real

Exemplo:

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

O código acima verificará a distância até o jogador object e se for menor do que o valor armazenado na variável "intervalo", a variável "canhoot" é definida como verdadeira.
