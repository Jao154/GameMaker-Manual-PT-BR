point_direction

Esta função retorna a direção de um vetor formado pelos componentes especificados [x1,y1] e [x2,y2] em relação às coordenadas fixas x/y do room. Por exemplo, na imagem abaixo, se quisermos obter a direção da posição do navio jogador para a posição do inimigo, para que possamos disparar um míssil contra o inimigo, usaríamos esta função (o código exato está no exemplo abaixo da imagem):

Sintaxe:

point_direction(x1, y1, x2, y2)

ArgumentoTipo Descrição
x1 The x coordinate of the first component of the vector
y1 The y coordinate of the first component of the vector
x2 The x coordinate of the second component of the vector
y2 The y coordinate of the second component of the vector

Devoluções:

Exemplo:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
{
    direction = point_direction(x, y, ex, ey);
}

O código acima obterá as coordenadas x e y do inimigo mais próximo e depois as passará para uma bala object para usar na função point_direction para definir corretamente sua direção de viagem.
