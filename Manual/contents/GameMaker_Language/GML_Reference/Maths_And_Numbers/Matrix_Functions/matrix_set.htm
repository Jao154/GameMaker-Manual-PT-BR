matriz_set

Esta função toma uma matriz de 16 valores, correspondentes a um determinado tipo de matriz 4x4, onde os elementos [0 - 3] seriam a linha 1, os elementos [4 -7] seriam a linha 2, etc... (veja a imagem na página principal). Você pode criar tal matriz usando a matrix_build() ou matrix_get() funções ou simplesmente construindo você mesmo a matriz e passando-a para a função. Os tipos de matriz disponíveis são visão, projeção e mundo, para os quais você usaria uma das constantes a seguir:

Constante Descrição
matrix_view A matriz de visão atual
matrix_projection A matriz de projeção atual
matrix_world A matriz mundial atual

Sintaxe:

matrix_set(type, matrix);

Argumento Tipo Descrição
type Matriz Tipo Constante The type of matrix to get the values of (see the constants listed above)
matrix Matriz de matriz The matrix data as an array

Devoluções:

N/A

Exemplo:

matrix_set(matrix_world, m_array);

O código acima definirá os valores da matriz mundial atual para aqueles armazenados na matriz de matriz "m_array".
