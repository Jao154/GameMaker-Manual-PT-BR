matrix_stack_top

Esta função retorna a matriz superior atual da pilha, mas não a remove (para esse uso matrix_stack_pop()).

Sintaxe:

matrix_stack_top();

Devoluções:

Matrix Array

Exemplo:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

O código acima construirá duas novas matrizes e depois as empurrará para a pilha de matrizes. A matriz do mundo é então colocada no topo da pilha, um sprite é desenhado e as transformações são então retiradas da pilha.
