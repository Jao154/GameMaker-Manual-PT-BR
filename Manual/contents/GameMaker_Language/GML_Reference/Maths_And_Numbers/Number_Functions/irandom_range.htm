irandom_range

Esta função é semelhante a random_range() somente com valores inteiros como entrada. Você fornece o valor baixo para a faixa, bem como o valor alto, e a função retornará um valor inteiro aleatório dentro de (e incluindo) a faixa dada. Por exemplo, irandom_range(10, 35) devolverá um número inteiro entre 10 e 35 inclusive.

Como com o irandom() podem ser usados números reais, neste caso eles serão arredondados para o EG inteiro mais próximo: irandom_range(6.2,9.9) dará um valor entre 6 e 9.

NOTA Esta função retornará o mesmo valor toda vez que o jogo for executado de novo devido ao fato de que o GameMaker: Studio gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo.

Sintaxe:

irandom_range(n1, n2);

ArgumentoTipo Descrição
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

Devoluções:

Exemplo:

score += irandom_range(500, 600);

Isto adicionará um valor inteiro entre 500 e 600 (inclusive) à pontuação total.
