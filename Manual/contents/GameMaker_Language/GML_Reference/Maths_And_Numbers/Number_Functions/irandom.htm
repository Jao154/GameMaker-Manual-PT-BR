irracional

Esta função muito útil retorna apenas números inteiros (números inteiros). Assim, por exemplo, para obter um número aleatório de 0 a 9, você pode usar irandom(9) e ele retornará um número de 0 a 9 inclusive. Note que flutuadores também podem ser usados, mas o valor superior após o ponto será excluído, portanto irandom(9.7) retornará um valor de 0 a 9 apenas. A função tem um limite superior de $7fffffffffffffffLL, portanto, deve-se ter cuidado se forem usados números muito grandes.

NOTA Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo.

Sintaxe:

irandom(n);

ArgumentoTipo Descrição
n The upper range from which the random number will be selected.

Devoluções:

Exemplo:

if (irandom(9) == 1)
{
    score += 100;
}

Isto produzirá uma chance de um em dez (já que 0 está incluído) de adicionar 100 à pontuação.
