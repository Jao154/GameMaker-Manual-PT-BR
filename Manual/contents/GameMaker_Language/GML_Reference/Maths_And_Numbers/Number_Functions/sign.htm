assine

Esta função retorna se um número é positivo, negativo ou nenhum e retorna 1, -1, 0 respectivamente. Por exemplo - sign(458) retorna 1, sign(-5) retorna -1 e sign(0) retorna 0.

Sintaxe:

sign(n);

ArgumentoTipo Descrição
n The number to get the sign of.

Devoluções:

Exemplo:

y += sign(y - mouse_y);

O código acima adicionará 1, -1 ou 0 em y, dependendo do resultado de y - mouse_y.
