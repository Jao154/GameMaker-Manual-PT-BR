aleatorise

Esta função define a semente para um valor aleatório. Caso você precise manter um valor consistente em várias corridas de um jogo, você deve estar usando random_set_seed(). Por favor, observe que ao utilizar o número aleatório no GameMaker, a semente inicial é sempre a mesma, pois isso facilita muito mais o rastreamento de erros e debugging. Caso você deseje testar com verdadeiro aleatório, você deve chamar esta função no início de seu jogo. A função irá devolver o novo valor da semente randomizada (um inteiro não assinado de 32bit).

Sintaxe:

randomise();
// or
randomize();

Devoluções:

(unsigned 32 bit value)

Exemplo:

randomise();

O código acima irá aleatorizar a semente.
