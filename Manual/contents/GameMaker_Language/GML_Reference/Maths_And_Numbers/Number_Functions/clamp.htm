Com esta função você pode manter um valor de entrada entre uma faixa especificada.

Sintaxe:

clamp(val, min, max)

ArgumentoTipo Descrição
val The value to clamp.
min The minimum value to clamp between.
max The maximum value to clamp between.

Devoluções:

Exemplo:

speed = clamp(speed, 1, 10);

O código acima fixará a velocidade para que ela nunca caia abaixo de 1 ou passe de 10.
