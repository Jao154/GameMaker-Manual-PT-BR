aleatório_set_seed

Para gerar um número aleatório, o GameMaker começa com um número de semente aleatório. Com esta função você pode definir essa semente para um valor conhecido e assim "forçar" o resultado de todos os eventos aleatórios depois a ser o mesmo toda vez que o programa for executado. Por exemplo, esta função pode ser usada em conjunto com random_get_seed() para criar conteúdo gerado processualmente e salvar os resultados sem ter grandes jogos de salvação (você salva apenas a semente, sem necessidade de mais nada). Caso você precise de resultados verdadeiramente aleatórios para tudo, você deve estar usando o randomise() função.

OBSERVAÇÃO Embora esta semente dê resultados consistentes em cada plataforma alvo, os resultados podem variar entre plataformas devido à forma diferente de funcionamento de cada alvo.

Sintaxe:

random_set_seed(val);

ArgumentoTipo Descrição
val The seed to set.

Devoluções:

N/A

Exemplo:

if debug
{
    random_set_seed(1);
}

O código acima define a semente aleatória como 1 somente se a variável "debug" for verdadeira.
