gama_randômica

Esta função retorna um número aleatório entre o intervalo especificado, e este valor de retorno não precisa ser um número inteiro. Por exemplo, random_range(20,50) retorna um número aleatório de 20 a 50, mas o valor pode ser um número real como 38,65265. Números reais também podem ser usados como argumentos de entrada.

NOTA: Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo.

Sintaxe:

random_range(n1, n2);

ArgumentoTipo Descrição
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

Devoluções:

Exemplo:

score += random_range(500, 600);

Isto adicionará entre 500 e 600, ao total da pontuação.
