point_in_circle

Ao utilizar esta função, você define uma área circular e o GameMaker irá determinar se o ponto em questão está ou não dentro de seus limites. Se o ponto estiver dentro do círculo definido, a função retornará true caso contrário, a função retornará false.

Sintaxe:

point_in_circle(px, py, x1, y1, rad);

ArgumentoTipo Descrição
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the circle centre.
y1 The y coordinate of the circle centre.
rad The radius of the circle.

Devoluções:

Exemplo:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

O código acima usa a função point_in_circle para verificar se a posição do mouse está dentro da área circular definida, definindo a variável "over" para true se estiver, ou false caso contrário.
