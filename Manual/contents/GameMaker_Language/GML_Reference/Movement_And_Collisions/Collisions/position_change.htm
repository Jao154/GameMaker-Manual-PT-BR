mudança_de_posição

Esta função verificará uma posição para uma colisão com qualquer instância no ponto determinado, e se houver uma, mudará todas as instâncias em colisão para serem instâncias do escolhido object. Você pode definir o argumento "perf" para true que forçará o GameMaker a realizar os eventos Destroy and Clean Up da instância encontrada, bem como o evento Create da nova instância, ou false para não realizar nenhum desses eventos. Observe que se você optar por não realizar os eventos Destroy, Clean Up e Create, qualquer instância criada que utilize uma variável normalmente definida no evento de criação irá travar o jogo, pois essa variável não existirá mais.

Sintaxe:

position_change(x, y, obj, perf);

Argumento Tipo Descrição
x Real The x coordinate of where to change colliding instances.
y Real The y coordinate of where to change colliding instances.
obj Objeto Ativo The new object the calling object will change into.
perf Booleano Whether to perform that new object's Create event (true) or not (false).

Devoluções:

N/A

Exemplo:

position_change(200, 200, obj_Bird, true);

Isto transformará todas as instâncias que colidem no ponto (200.200) em uma instância de "obj_Bird", realizando "obj_Bird "s Create event for each of them in the process.
