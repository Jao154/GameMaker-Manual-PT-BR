lista_de_pontos_de_colisão

Esta função é a mesma que a collision_point() apenas, em vez de apenas detectar uma instância em colisão de cada vez, ele detectará várias instâncias. Você fornece a posição x/y do ponto a ser verificado e pode definir a verificação para ser precisa (caso em que todas as instâncias a serem verificadas precisarão ter máscaras de colisão precisas ) e se a verificação deve ou não incluir a instância de chamada.

Você também fornece uma lista DS, portanto o id valores de quaisquer instâncias que estejam colidindo com a instância de chamada serão adicionados ao final da lista dada. Você pode limpar a lista antes de chamar esta função para que ela contenha apenas os resultados desta chamada de função. Você também tem a opção de ordenar as instâncias com base em suas distâncias desde o ponto dado até sua origem. A função retorna o número de instâncias encontradas, ou 0 se nenhuma for encontrada.

Observe que ao invés de um índice object você pode fornecer a palavra-chave da instância all, para verificar todas as instâncias no atual room, que pode incluir a instância executando o código (dependendo do argumento notme ).

Sintaxe:

collision_point_list(x, y, obj, prec, notme, list, ordered);

Argumento Tipo Descrição
x Real The x coordinate of the point to check.
y Real The y coordinate of the point to check.
obj Objeto Ativo ou ID da instância The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (trueque é mais lento) ou sua caixa de delimitação em geral (false, mais rápido).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) ou não (false).
list DS List ID The DS list to use to store the IDs of colliding instances.
ordered Booleano Whether the list should be ordered by distance (true) ou não (false).

Devoluções:

Real (The number of instances found to be in collision)

Exemplo:

var _list = ds_list_create();
var _num = collision_point_list(x, y, obj_Enemy, false, true, _list, false);
if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

O código acima verificará a posição da instância de chamada em caso de colisões com instâncias de "obj_Enemy". Se houver alguma colisão, então a lista pré-criada é passada em loop e cada instância que estava na colisão é destruída.
