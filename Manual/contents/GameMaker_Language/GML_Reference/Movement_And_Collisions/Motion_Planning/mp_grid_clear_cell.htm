mp_grid_clear_cell

Esta função pode ser usada para limpar uma "célula" específica de uma grade MP. As células não são calculadas como coordenadas room, mas sim como coordenadas da grade, onde (0,0) é o canto superior deixado da grade. isto significa que para limpar uma célula em uma posição específica no room, devemos mudar as coordenadas x e y em coordenadas de célula dividindo-as pela resolução da grade MP. O exemplo de código abaixo mostra como isto funciona.

Sintaxe:

mp_grid_clear_cell(id, h, v);

Argumento Tipo Descrição
id ID da grade MP Index of the mp_grid that is to be used
h Real Horizontal position of the cell to clear
v Real vertical position of the cell to clear

Devoluções:

Boolean

Exemplo:

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

O código acima fará com que todos os "obj_Box" se destruam e os fará marcar como livres as células que ocupavam na "mp_grid indexada na variável "grid". Neste exemplo, encontramos a célula apropriada tomando a coordenada x/y do object e dividindo-as pela resolução da grade (usando floor para manter os valores como números inteiros).
