place_snapped

Com esta função você pode verificar e ver se a origem de uma instância (sua posição x e y) está alinhada a uma grade com os valores hsnap e vsnap especificados por você.

Sintaxe:

place_snapped(hsnap, vsnap);

ArgumentoTipo Descrição
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

Devoluções:

Exemplo:

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

O código acima verifica todas as instâncias de "obj_Pieces" para ver se elas estão encaixadas em uma grade de 32x32 pixels, e se elas não estão encaixadas na posição mais próxima daquela grade.
