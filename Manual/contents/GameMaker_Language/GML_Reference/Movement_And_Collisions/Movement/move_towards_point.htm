move_towards_point

Esta função simplesmente diz à instância que está executando o código para se mover em direção a um ponto definido a uma velocidade definida. Tenha em mente que cabe a você dizer-lhe o que fazer quando chegar lá, pois não vai parar por si só. Isto significa que você pode obter uma instância "vibrante" enquanto ela se move (por exemplo) 3 pixels para frente, ultrapassa a posição alvo x/y em 2, depois move 3 pixels para trás novamente, ultrapassando o ponto alvo em um e assim por diante... Há várias maneiras de evitar isso, por exemplo, usando distance_to_point() para ver quão longe do local especificado a instância está antes de se mudar.

NOTA: esta função define a velocidade da instância, então mesmo que você pare de usar este código, a instância continuará se movendo na direção anterior, então é necessário definir a instância speed a 0 se você desejar que pare. Veja o exemplo de código abaixo.

Sintaxe:

move_towards_point( x, y, sp );

Argumento Tipo Descrição
x Real The x position of the point to move towards.
y Real The y position of the point to move towards.
sp Real The speed to move at in pixels per step.

Devoluções:

N/A

Exemplo:

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

O código acima moverá a instância em direção à posição da instância indexada na variável "alvo" a uma velocidade de 5 pixels por etapa. Uma vez alcançada essa posição, a velocidade é ajustada para 0.
