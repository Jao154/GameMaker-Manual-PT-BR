motion_set

Esta função define uma nova direção de movimento e uma nova velocidade para a instância que executa o código. Note que isto não acrescenta às instâncias velocidade e direção atuais (para isso você usaria motion_add()) mas, ao contrário, obriga-o aos novos ambientes.

Sintaxe:

motion_set(dir, speed);

ArgumentoTipo Descrição
dir The new direction.
speed The new speed.`

Devoluções:

Exemplo:

if irandom(9) = 1
{
    motion_set(random(360), 1 + random(3));
}

Este código acima fará com que a instância mude de velocidade e direção em intervalos aleatórios.
