move_random

Com esta função, você pode definir a instância para uma posição em qualquer lugar no room, mas alinhada a uma grade "invisível". Assim, um valor de 32 para o hsnap e vsnap irá definir a instância para uma posição aleatória que está alinhada a uma grade de 32x32 quadrados. você pode definir estes valores como 1 para obter uma posição em qualquer lugar no room.

OBSERVAÇÃO: Esta função se encaixará nas mesmas posições toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo. Isto só é verdade ao testar e debugging o jogo, pois o pacote executável final não mostrará este comportamento e será aleatório a cada jogada.

Sintaxe:

move_random(hsnap, vsnap);

ArgumentoTipo Descrição
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

Devoluções:

Exemplo:

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

Isto moverá a instância para uma posição aleatória em qualquer lugar no room quando a tecla de espaço for pressionada.
