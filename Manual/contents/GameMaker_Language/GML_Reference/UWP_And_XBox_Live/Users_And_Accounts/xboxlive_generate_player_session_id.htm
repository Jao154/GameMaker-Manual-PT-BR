xboxlive_generate_player_id_session_id

Esta função gerará um exclusivo string (contendo um GUID) para a sessão atual do jogador. Isto pode então ser usado com a função xboxlive_fire_event().

Sintaxe:

xboxlive_generate_player_session_id();

Devoluções:

Exemplo:

if !xboxlive_user_is_signed_in()
{
    if !xboxlive_user_is_signing_in()
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

O código acima verifica se um usuário está conectado ao XBox Live e se ele está conectado, o código irá obter o gamertag para o usuário e armazená-lo em uma variável global para uso futuro. Se eles não estiverem logados, então o código verifica se estão em processo de log-in, caso em que nada mais acontecerá, e se não estiverem, então ele abrirá o selecionador de conta, levando-os a logar-se.
