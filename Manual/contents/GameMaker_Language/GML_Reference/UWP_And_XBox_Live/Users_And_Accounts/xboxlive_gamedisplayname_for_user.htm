xboxlive_gamedisplayname_para_utilizador

Com esta função você pode recuperar o nome do display - como um string - a partir do ponteiro de identificação do usuário dado. Note que se o usuário for local, então o valor retornado é simplesmente um string do nome de sua tela, mas se for um usuário remoto, então o string será sua gamertag que poderá então ser usada, por exemplo, para muting/unmuting no bate-papo de voz.

Sintaxe:

xboxlive_gamedisplayname_for_user(user_id);

ArgumentoTipo Descrição
user_id The ID pointer of the user to check.

Devoluções:

Exemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

O código acima obtém o ponteiro de identificação do usuário para cada conta de usuário disponível e então armazena o ponteiro junto com o nome do display em duas arrays.
