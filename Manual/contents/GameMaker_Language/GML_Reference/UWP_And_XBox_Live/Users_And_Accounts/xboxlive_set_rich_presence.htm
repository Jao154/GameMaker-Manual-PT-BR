xboxlive_set_rich_presence

Esta função irá definir a presença rica string para o usuário em questão. Uma presença rica string mostra a atividade dentro do jogo de um usuário após o nome do jogo que o usuário está jogando, separado por um hífen. Este string é exibido sob a Gamertag de um jogador na lista "Friends & Clubs", bem como no perfil do usuário do Xbox Live do jogador.

Ao utilizar esta função você precisa fornecer o ponteiro de identificação do usuário para o usuário, e então você pode flag o usuário como atualmente ativo no jogo ou não (utilizando true/false). Se configurado para false, então a rica presença string será anexada com "/afk" ou algo apropriado. O próximo argumento é a rica presença string ID para mostrar, e então finalmente você pode (opcionalmente) fornecer um service_config_id string. Este argumento é opcional, e quando não fornecido, ele usa o SCID especificado nas Opções de Jogo do Xbox.

Sintaxe:

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

Argumento Tipo Descrição
user_id ID do usuário do Xbox The ID pointer of the user to check.
is_user_active Booleano Flag the user as active or not.
rich_presence_string Cordão The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id Cordão OPTIONAL Este é o config_id string para o jogo.

Devoluções:

N/A

Exemplo:

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

O código acima recebe o ponteiro de identificação do usuário atribuído ao slot do gamepad [0], e então define a presença rica string para esse usuário.
