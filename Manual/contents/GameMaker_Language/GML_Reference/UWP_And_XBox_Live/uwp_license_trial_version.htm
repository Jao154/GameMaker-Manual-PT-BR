uwp_license_trial_version

Esta função pode ser usada para verificar se o jogo está ou não sob uma licença de teste. Se for esta a função retornará true, ou false caso contrário.

Sintaxe:

uwp_license_trial_version();

Devoluções:

Exemplo:

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

O código acima verifica se o aplicativo está sob uma licença de teste e, em caso afirmativo, define uma variável global.
