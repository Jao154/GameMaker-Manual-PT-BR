uwp_is_suspending

Esta função retornará true para um único quadro (e false caso contrário) de maneira semelhante a os_is_paused(). O jogo então tem um segundo para salvar o estado do jogo antes que ele seja potencialmente terminado pelo sistema. Uma vez que o jogo tenha terminado de salvar os dados, ele deve chamar uwp_suspend() para suspender limpos. O jogo provavelmente também deveria entrar em um estado de pausa para que, se for retomado pelo sistema (se não tiver sido totalmente terminado), o jogador não seja jogado diretamente de volta para uma situação de jogo.

Sintaxe:

uwp_is_suspending();

Devoluções:

Exemplo:

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

O código acima verifica se o aplicativo está entrando em suspensão e se está chamando um script para salvar os dados do jogo antes de suspender o jogo.
