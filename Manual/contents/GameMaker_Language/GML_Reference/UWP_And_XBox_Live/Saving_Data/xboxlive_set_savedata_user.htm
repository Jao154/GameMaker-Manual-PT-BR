xboxlive_set_set_savedata_user

Esta função especifica que as futuras operações de arquivo que operam na área de salvar jogo (ou seja, todos os arquivos escritos, e lidos de arquivos que não estão na área do pacote) serão associados ao ponteiro de identificação do usuário especificado. Isto pode ser chamado quantas vezes for necessário para redirecionar os dados salvos para o usuário apropriado, ou você pode usar a constante pointer_null para salvar na área genérica de armazenamento da máquina.

Sintaxe:

xboxlive_set_savedata_user(user_id);

Argumento Tipo Descrição
user_id Xbox ID de usuário ou pointer_null The user ID (a pointer) to set for saving, or pointer_null

Devoluções:

N/A

Exemplo:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

O código acima verifica se um usuário está atualmente designado como alvo a ser salvo, e se não estiver, então ele é designado.
