uwp_was_terminated

Com esta função você pode descobrir se o jogo foi desligado anteriormente pelo sistema (ou seja, não é a primeira execução do jogo). Se foi, então a função retornará true, caso contrário retornará false. Um exemplo de uso seria verificar se o jogo foi anteriormente encerrado pelo sistema, e depois (se retornar true) carregar a etapa anterior em que o jogador estava no jogo. Se o objetivo for o XBox One, então o comportamento necessário dos jogos, caso seja detectado, será diferente entre as diferentes versões dos documentos de certificação, então vale a pena verificar a última versão para ver o que o jogo deve fazer.

Sintaxe:

uwp_was_terminated();

Devoluções:

Exemplo:

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

O código acima verifica se o aplicativo foi encerrado anteriormente pelo sistema e, em caso afirmativo, carrega um buffer e define uma variável global.
