xboxlive_stats_set_stat_string

Esta função pode ser usada para definir uma estatística para um string para o ID de usuário dado. O usuário fornece o ID de usuário conforme retornado pela (por exemplo) função xboxlive_get_user()Em seguida, o estatuto string para definir(se o estatuto string ainda não existir, então um novo estatuto será criado e definido para o valor dado) e depois o string para definir o estatuto. Note que o nome do stat deve ser alfanumérico apenas, sem símbolos ou espaços, mas o valor string que você está definindo pode ter espaços e caracteres especiais.

Ao definir o estatuto string, qualquer string anterior será anulado, portanto cabe a você determinar se o estatuto string deve ou não ser atualizado, comparando-o com o atual string com o novo antes de defini-lo.

A função retornará -1 se houver um erro ou a identificação do usuário for inválida, ou qualquer outro valor se a função foi chamada com sucesso.

Sintaxe:

xboxlive_stats_set_stat_string(user_id, stat, value);

Argumento Tipo Descrição
user_id ID do usuário do Xbox The user ID pointer to set the stat for
stat Cordão The statistic to set (a string)
value Cordão The value to set the stat to (a string)

Devoluções:

Real

Exemplo:

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

O código acima define a estatística "LevelName" para o string armazenado na variável global em questão.
