xboxlive_stats_set_stat_real

Esta função pode ser usada para definir o valor de uma estatística para o ID do usuário dado. O usuário fornece o ID de usuário conforme retornado pela (por exemplo) função xboxlive_get_user()então o estatuto string para definir(se o estatuto string ainda não existir, então um novo estatuto será criado e definido para o valor dado) e um valor (um real) para definir o estatuto. Note que o nome do stat deve ser apenas alfanumérico, sem símbolos ou espaços.

Ao definir o valor stat, qualquer valor anterior será substituído, portanto cabe a você determinar se o valor stat deve ser atualizado ou não (ou seja, verificar se a pontuação mais alta é realmente a mais alta), comparando o valor stat atual com o novo antes de defini-lo.

A função retornará -1 se houver um erro ou a identificação do usuário for inválida, ou qualquer outro valor se a função foi chamada com sucesso.

Sintaxe:

xboxlive_stats_set_stat_real(user_id, stat, value);

ArgumentoTipo Descrição
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

Devoluções:

Exemplo:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

O código acima verifica uma variável e se for true, então verificará o valor da estatística "PercentDone". Se este valor for inferior a 100, um valor é então gerado e o stat é definido para este valor.
