xboxlive_stats_get_stat

Esta função pode ser usada para recuperar um único valor stat do gerente stat para um determinado usuário. O usuário fornece o ID do usuário conforme retornado pela (por exemplo) função xboxlive_get_user()e, em seguida, o stat string conforme definido quando você o criou utilizando uma das funções do xboxlive_stats_set_stat_*. O valor de retorno pode ser ou um string ou um real (dependendo do stat sendo verificado) ou a constante GML undefined se o stat não existir ou se tiver havido um erro.

Sintaxe:

xboxlive_stats_get_stat(user_id, stat);

ArgumentoTipo Descrição
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

Devoluções:

Real, String or undefined

Exemplo:

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

O código acima verifica uma variável e se for true, então verificará o valor da estatística "PercentDone". Se este valor for inferior a 100, um valor é então gerado e o stat é definido para este valor.
