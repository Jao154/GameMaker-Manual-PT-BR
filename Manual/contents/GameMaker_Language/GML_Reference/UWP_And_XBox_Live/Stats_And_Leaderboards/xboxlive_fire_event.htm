xboxlive_fire_event

Esta função pode ser usada para disparar um evento estatutário. O argumento "event_name" é o nome do evento a ser disparado conforme definido no console XDP para seu jogo, e os seguintes parâmetros adicionais também dependerão do que você tiver configurado para o stat. A função retornará 0 se o evento foi enviado (e deve ser recebido/processado pelo server) ou -1 se houve um erro (ou seja: seu evento não foi configurado, pois o arquivo de manifesto de evento incluído no projeto diz outro número).

Sintaxe:

xboxlive_fire_event(event_name, [stat_parameters...]);

Argumento Tipo Descrição
event_name Cordão The name of the event to fire.
stat_parameters Variável OPTIONAL Parâmetros adicionais a serem acrescentados ao evento.

Devoluções:

Real

Exemplo:

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

O código acima obtém algumas informações sobre o usuário e, em seguida, dispara um evento estatutário junto com estes (e outros) dados.
