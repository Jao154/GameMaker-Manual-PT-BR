xboxlive_achievement_load_friends

Esta função enviará um pedido ao server para obter informações sobre todos os amigos usuários logados e acionará um Evento Assíncrono Social de retorno que contém o async_load mapa povoado com os pares chave/valor relevantes. A chave de id deste DS Map é usada para identificar o retorno de chamada correto (pode haver mais de uma função de acionamento para qualquer evento assíncrono), e será emparelhada com a constante xboxlive_achievement_friends_info, bem como um número de outros pares de chaves/valores para cada amigo. O conteúdo exato do mapa é mostrado abaixo:

  • "id" - Para esta função deve ser xboxlive_achievement_friends_info
  • "FriendN" - O nome do amigo, onde "N" é um valor inteiro correspondente à sua posição dentro da lista de amigos.
  • "FriendidN" - A identificação única do usuário do amigo, "N".

Sintaxe:

xboxlive_achievement_load_friends();

Devoluções:

N/A

Exemplo ampliado:

O seguinte código provavelmente seria chamado depois que o jogador entrasse em sua conta de jogo para obter uma lista de todos os amigos dos usuários:

xboxlive_achievement_load_friends();

Isto enviará um pedido de informações sobre os amigos usuários e gerará uma chamada assíncrona de retorno com o mapa especial async_load DS contendo os seguintes dados:

var ident = ds_map_find_value(async_load, "id");
if ident == xboxlive_achievement_friends_info
{
    var numfriends = ds_map_find_value(async_load, "numfriends");
    global.numfriends = numfriends;
    for(var i=0; i < numfriends; i++;)
    {
        global.friendname[i] = ds_map_find_value(async_load, "Friend" + string(i));
        global.friendid[i] = ds_map_find_value(async_load, "Friendid" + string(i));
    }
}

O código acima verifica o retorno DS Map no Evento Assíncrono Social e se seu "id" corresponde à constante sendo verificada, ele então loops através do mapa que armazena todos os diferentes valores em uma série de arrays.
