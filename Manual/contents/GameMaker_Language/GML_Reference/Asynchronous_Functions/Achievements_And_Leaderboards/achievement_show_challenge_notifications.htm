notificações_de_conquista_de_conquista

Com esta função você pode mostrar, ou suprimir, as várias notificações pop-up "brinde" diferentes relacionadas a desafios. Quando os argumentos forem definidos como verdadeiros, estas mensagens aparecerão, informando o jogador sobre quaisquer desafios locais ou remotos. recebidas, bem como aqueles desafios que foram concluídos, ao mesmo tempo em que os coloca em falso, suprimirá estas notificações.

Sintaxe:

achievement_show_challenge_notifications(receive, local, remote)

Argumento Descrição
receive Mostrar os desafios recebidos pop-up(verdadeiro) ou não(falso).
local Mostrar os desafios locais completados(verdadeiros) ou não(falsos).
remote Mostrar os desafios remotos completados pop-up(verdadeiro) ou não(falso).

Devoluções:

N/A

Exemplo:

achievement_show_challenge_notifications(true, false, false);

O código acima estabelecerá que as notificações de desafio serão mostradas apenas para os desafios recebidos.
