push_cancelal_local_notification

Esta função cancela a notificação push dada. Você pode obter a identificação para que a notificação seja cancelada usando as funções push_get_first_local_notification() e push_get_next_local_notification() e a função retornará true em caso de sucesso ou false caso contrário (por exemplo, se a notificação não existir ou já tiver sido acionada).

NOTA: Esta função é limitada aos módulos alvo iOS e Android.

Sintaxe:

push_cancel_local_notification(ID);

Argumento Tipo Descrição
ID Identificação da Notificação Push The ID of the notification to cancel.

Devoluções:

Boolean

Exemplo :

var map = ds_map_create();
var ntf = push_get_first_local_notification(map);
while(ntf >= 0)
{
    var data = ds_map_find_value(map, "data");
    if data == "Daily_Reward"
    {
        push_cancel_local_notification(ntf);
    }
    ntf = push_get_next_local_notification(map);
}
ds_map_destroy(map);

O código acima cria um DS Map e depois solicita dados para a primeira notificação disponível no queue. Se houver uma notificação, o mapa é preenchido e o código então verifica a chave "data" para um string específico. Se o string for encontrado, a notificação é cancelada e passa a verificar a próxima notificação até que tudo no queue tenha sido verificado.
