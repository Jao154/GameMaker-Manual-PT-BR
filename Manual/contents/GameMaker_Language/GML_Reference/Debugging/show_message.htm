show_message

Esta função cria uma caixa de mensagem pop-up que exibe o dado string e um botão marcado "Ok" para fechá-la.

NOTA Esta função é para uso de depuração apenas no alvo Windows, mas é depreciada em todos os outros alvos.

Sintaxe:

show_message(str);

Argumento Tipo Descrição
str Cordão The string to show in the pop-up message.

Devoluções:

N/A

Exemplo:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

O código acima irá loop através dos valores armazenados na matriz "inv" e adicioná-los à variável "tot" antes de mostrar uma mensagem com o total.
