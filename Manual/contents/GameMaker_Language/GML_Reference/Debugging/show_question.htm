show_question

Esta função cria uma caixa de mensagens pop-up com dois botões para "Sim" e "Não". Ela retorna true ou false, dependendo de qual dos dois botões o usuário pressiona.

NOTA Esta função é para uso de depuração apenas no alvo Windows, mas é depreciada em todos os outros alvos.

Sintaxe:

show_question(str);

Argumento Tipo Descrição
str Cordão The string to show in the pop-up question.

Devoluções:

Boolean

Exemplo:

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

O código acima verificará a pontuação e se for mais de 500, perguntará ao usuário se ele deseja continuar ou não e se o botão "sim" for clicado irá para outro room, mas se o botão "não" for selecionado, terminará o jogo.
