show_error

Esta função mostrará uma mensagem de erro personalizada string.

IMPORTANTE O segundo argumento só existe para compatibilidade retroativa com projetos mais antigos e não terá efeito no diálogo de erros mostrado.

NOTA Esta função é apenas para uso de depuração.

Sintaxe:

show_error(str, abort);

Argumento Tipo Descrição
str Cordão The string to show in the pop-up message.
abort Booleano UNUSED Se o erro deve abortar o jogo (verdadeiro) ou permitir que o jogador o ignore (falso).

Devoluções:

N/A

Exemplo:

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

O código acima verificará se o atual room é rm_Dungeon, e se não for, mostrará uma mensagem de erro.
