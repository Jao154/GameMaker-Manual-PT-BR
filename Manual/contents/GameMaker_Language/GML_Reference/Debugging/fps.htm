fps

No GameMaker há duas maneiras principais que podem ser usadas para saber a velocidade em que seu jogo funciona. O game speed (conforme especificado nas Opções de Jogo) e no fps (quadros por segundo). Estes valores são freqüentemente confusos, mas basicamente um é o número de passos de jogo que o GameMaker deve estar completando em um segundo, enquanto o outro (o fps) é o número de passos da CPU que o GameMaker está realmente completando em um segundo até um valor máximo da própria velocidade room . Para obter o verdadeiro fps, ou seja, o número real de passos da CPU por passo do jogo, use o fps_real variável.

Esta variável somente de leitura retorna o fps atual como um valor inteiro. Observe que a função só será atualizada uma vez a cada passo de seu jogo e assim pode parecer "pular" de um valor para outro, mas isto é bastante normal.

Sintaxe:

fps

Devoluções:

Exemplo:

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

O código acima verificará se o jogo está no modo de depuração e se é ele que exibirá os fps atuais na tela.
