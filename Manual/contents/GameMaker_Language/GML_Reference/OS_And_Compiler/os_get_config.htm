os_get_config

Esta função retorna o nome (como um string) da configuração atualmente selecionada para seu jogo. Para mais informações sobre as configurações, consulte a seção Configurações.

Sintaxe:

os_get_config()

Devoluções:

Exemplo:

if os_get_config() = "Free_Version"
{
    global.Ads = true;
}
else global.Ads = false;

O código acima irá verificar qual configuração está sendo usada e se for a chamada "Free_Version", os anúncios serão habilitados no jogo.
