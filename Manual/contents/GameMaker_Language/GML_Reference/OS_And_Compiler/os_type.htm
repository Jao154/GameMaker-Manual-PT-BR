os_type

Esta variável somente de leitura contém uma das várias constantes GameMaker tem que lhe dizer para qual sistema operacional o jogo foi criado. Note que isto não é necessariamente o mesmo que o SO do dispositivo que o executa, já que - por exemplo - seu jogo poderia estar rodando em um SO Amazon Fire, mas terá sido construído para a plataforma Android (nesse caso os_type será os_android).

As seguintes constantes podem ser devolvidas:

Constante tipo de SO
Constante Descrição
os_windows SO Windows
os_uwp Plataforma Windows 10 Universal Windows
os_operagx Ópera GX
os_linux Linux
os_macosx macOS X
os_ios iOS (iPhone, iPad, iPod Touch)
os_tvos TvOS de maçã
os_android Android
os_ps4 Sony PlayStation 4
os_ps5 Sony PlayStation 5
os_xboxone Microsoft Xbox One
os_xboxseriesxs Microsoft Xbox Série X/S
os_switch Interruptor Nintendo
os_unknown SO desconhecido

Sintaxe:

os_type;

Devoluções:

OS Type Constant

Exemplo:

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

O código acima verifica o sistema operacional que está rodando o jogo e define uma variável global de acordo.
