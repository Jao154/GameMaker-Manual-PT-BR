agendador_resolução_get

Esta função é utilizada para recuperar a resolução do agendador de fios Windows em milissegundos. Se a resolução do agendador for definida para o valor padrão (como definido por Windows), a função retornará -1.

Para obter informações sobre o agendador de fios, consulte a página para scheduler_resolution_set().

NOTA: Esta função é apenas para Windows .

Sintaxe:

scheduler_resolution_get();

Devoluções:

(or -1 for default)

Exemplo:

scheduler_resolution = scheduler_resolution_get();

Este exemplo recupera a resolução do agendador de fios Windows, e a armazena na variável "scheduler_resolution".
