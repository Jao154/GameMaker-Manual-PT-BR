array_length_2d DEPRECATED

ADVERTÊNCIA! Esta função é depreciada (e substituída por array_length()), pois as matrizes não estão mais limitadas a apenas 1 ou 2 dimensões, e como tal, esta função é fornecida apenas para apoio a projetos legados. Todos os novos projetos devem usar a forma atual de criar e acessar matrizes multidimensionais (veja aqui para mais informações).

Com esta função você pode obter o comprimento (número de entradas) de uma segunda dimensão de uma matriz. Você fornece o número de entradas para a primeira dimensão e a função retornará o número de entradas da segunda dimensão que a matriz tem (para encontrar o comprimento da primeira dimensão, use a função array_height_2D()). A função retornará 0 se a variável dada não for um array ou 1 se a variável for um array 1D (já que ainda há 1 linha).

Sintaxe:

array_length_2d(array_index, n);

Argumento Tipo Descrição
array_index Array The index of the array to check.
n Real The entry of the array to get the length of.

Devoluções:

Real

Exemplo:

for (var i = 0; i < array_height_2d(a); ++i;)
{
    for (var j = 0; j < array_length_2d(a, i); ++j;)
    {
        a[i, j] = -1;
    }
}

O código acima irá loop através de uma matriz 2D e definirá cada entrada para -1.
