array_push

Com esta função você pode empurrar um valor (ou valores) para o final de uma matriz sem ter que saber o comprimento da matriz. A função exige que você forneça uma variável que contenha o array, bem como pelo menos um valor para empurrar, embora você possa opcionalmente fornecer outros argumentos e todos eles serão empurrados para o array em ordem consecutiva.

Sintaxe:

array_push(variable, value, [value], [value], [etc...]);

Argumento Tipo Descrição
variable Array The variable that holds the array.
value Variável The value to push onto the end of the array
[value], [value], [etc...] Variável OPTIONAL Outros valores a serem empurrados para a matriz

Devoluções:

N/A

Exemplo:

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

O código acima irá empurrar quatro valores para o final da matriz dada.
