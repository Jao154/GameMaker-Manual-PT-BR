is_struct

Esta função verifica se o valor fornecido é um struct. Retorna true se for, caso contrário retorna false.

Note que as variáveis do método também retornarão true, e as instâncias do objeto retornarão false.

Sintaxe:

is_struct(val);

Argumento Tipo Descrição
val Variável The value to check.

Devoluções:

Boolean

Exemplo:

if is_struct(a)
{
    delete(a);
}

O código acima verifica uma variável para ver se é uma struct, e se a função retorna true, o struct é deletado.
