array_pop

Esta função removerá o último elemento da matriz dada, e retornará seu valor. Se o array estiver vazio, undefined é retornado.

Sintaxe:

array_pop(array);

ArgumentoTipo Descrição
array The array to remove the last value from.

Devoluções:

Variable or undefined

Exemplo:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

O código acima irá estourar o último valor da matriz dada, e então o desenhará junto com algum texto para a tela. A entrada recuperada também é removida da matriz de pontuação.
