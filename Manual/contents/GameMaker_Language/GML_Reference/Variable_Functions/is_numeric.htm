is_numérico

Esta função retorna se uma determinada variável é um valor numérico (real, int32, int64 ou booleano) ou não. Em alguns casos você deseja verificar e ver se uma variável no GameMaker possui algum valor numérico, e é quando você usaria esta função. A função retornará true se a entrada dada for numérica, e false caso contrário.

Sintaxe:

is_numeric(n);

Argumento Tipo Descrição
n Variável The input to check.

Devoluções:

Boolean

Exemplo:

if is_numeric(val)
{
    current_val += val;
}

O código acima verifica a variável "val" para ver se é um valor numérico e se é um valor numérico e, em seguida, adiciona-o à variável dada.
