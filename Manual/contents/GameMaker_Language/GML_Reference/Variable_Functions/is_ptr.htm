is_ptr

Esta função retorna se uma determinada variável é ou não um ponteiro. Em alguns casos, você quer verificar e ver que tipo de dado uma variável contém no GameMaker e é quando você usaria esta função. Ela retorna true ou false, dependendo se o valor é ou não um ponteiro.

Sintaxe:

is_ptr(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if !is_ptr(val)
{
    show_debug_message("Not a valid texture!");
}

O código acima verifica a variável "val" para ver se ela contém um ponteiro e se não contém, então mostra uma mensagem no console de depuração.
