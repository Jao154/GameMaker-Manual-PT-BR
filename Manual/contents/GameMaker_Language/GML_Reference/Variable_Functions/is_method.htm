is_method

Esta função pode ser usada para verificar e ver se uma variável é uma variável de método (retornará true) ou não (nesse caso retornará false).

Sintaxe:

is_method(n);

Argumento Tipo Descrição
n Variável The variable to check.

Devoluções:

Boolean

Exemplo:

if is_method(get_vec)
{
    show_debug_message("Method variable!");
}

O código acima verifica uma variável para ver se é um método, e se a função retorna true, então uma mensagem de debug é emitida para o console.
