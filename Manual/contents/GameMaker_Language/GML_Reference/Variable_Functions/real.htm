real

Esta função pode ser usada para transformar um determinado string em um número real. Ao utilizar esta função, são levados em conta números, sinais de menos, pontos decimais e partes exponenciais no string, enquanto outros caracteres (tais como letras) causarão um erro a ser jogado. Se você sabe, ou suspeita, que um string pode ter outros caracteres, então você pode usar string_digits() para remover todos os caracteres não numéricos, antes de usar esta função para girar a cadeia resultante em um número real.

Sintaxe:

real(string);

ArgumentoTipo Descrição
string The string to be converted to a real value.

Devoluções:

Exemplo:

var t_str = string_digits(input_str);
age = real(t_str);

O código acima pegará a entrada string, desnudará todos os caracteres que não sejam números e então definirá a variável "idade" para manter o valor real do número do retorno string.
