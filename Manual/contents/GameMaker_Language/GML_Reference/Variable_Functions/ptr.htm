ptr

Esta função tentará converter um determinado valor em um tipo de dado ponteiro, onde o valor deve ser ou um real, um string, um int64, um int32, ou um ptr. Qualquer outra coisa fará com que o jogo trave com uma mensagem de erro.

Sintaxe:

ptr(n);

Argumento Tipo Descrição
n Real, String ou Pointer The value to convert.

Devoluções:

Pointer

Exemplo:

if !is_ptr(val)
{
    val = ptr(application_surface);
}

O código acima verifica a variável "val" para ver se ela contém um ponteiro e se não contém, então um é atribuído a ela.
