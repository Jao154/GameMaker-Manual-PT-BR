array_length_1d DEPRECATED

ADVERTÊNCIA! Esta função é depreciada (e substituída por array_length()), pois as matrizes não estão mais limitadas a apenas 1 ou 2 dimensões, e como tal, esta função é fornecida apenas para apoio a projetos legados. Todos os novos projetos devem usar a forma atual de criar e acessar matrizes multidimensionais (veja aqui para mais informações).

Com esta função você pode obter o comprimento (número de entradas) de uma matriz 1D. Para arrays 2D você deve estar usando a array_height_2d e array_length_2d funções.

IMPORTANTE! Se a matriz tiver mais de 32.000 entradas, esta função retornará um valor errado e não deve ser usada.

Sintaxe:

array_length_1d(array_index);

Argumento Tipo Descrição
array_index Array The index of the array to check.

Devoluções:

Real

Exemplo:

for (var i = array_length_1d(a) - 1; i > -1; i--;)
{
    a[i] = -1;
}

O código acima irá loop através de uma matriz e definirá cada entrada para -1.
