variável_global_get

Com esta função, você pode obter o valor de uma determinada variável global nomeada. Você fornece o nome da variável global para obter o valor de uma string (veja o código de exemplo abaixo) e a função retornará o valor mantido pela variável global ou undefined se a variável não existir.

IMPORTANTE! Se a variável global que você está recebendo não existir, então a função retornará a palavra-chave undefined e você poderá receber erros que impedirão o funcionamento do jogo, portanto, em caso de dúvida, use a função variable_global_exists primeiro.

Sintaxe:

variable_global_get(name);

ArgumentoTipo Descrição
name The name of the global variable to get (as a string)

Devoluções:

Variable (any data type) or undefined (if the named variable does not exist)

Exemplo:

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

O código acima verificará se existe uma variável global e se ela existe, então ela é emitida para o console.
