Funções variáveis

As funções desta página são projetadas para lidar com todas as diferentes variáveis e tipos de variáveis ao utilizar o GameMaker Language em seus jogos. Para mais informações sobre variáveis, consulte a página sobre Variáveis e Escopo de Variáveis.

As seguintes funções variáveis estão disponíveis:

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • variable_struct_exists
  • variable_struct_get
  • variable_struct_set
  • variável_struct_remove
  • variable_struct_get_names
  • variable_struct_names_count
  • instanceof

Há algumas funções associadas ao uso de Arrays também. Elas são projetadas para lhe dar flexibilidade em seu código e lhe permitirão criar arrays mais funcionais e dinâmicos ao fazer seus jogos. Estas funções são:

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(depreciado)
  • array_length_2d(depreciado)
  • array_height_2d(depreciado)

Estas próximas funções estão todas relacionadas ao uso de Variáveis de Método:

  • method
  • method_get_self
  • method_get_index

Finalmente, você também tem as seguintes funções para determinar o tipo de variável (para mais informações sobre tipos de dados, veja aqui, e veja aqui as Tabelas de Tipo):

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • ptr
  • int64
