método

Com esta função você pode vincular uma função (ou método) existente à instância dada ou struct, criando uma nova variável de método que pode ser usada posteriormente.

Você fornece a identificação da instância a usar (uma instância que está ativa no room, e não um índice object ) ou uma referência struct, assim como a identificação da função (ou referência do método) que você deseja vincular. A função retornará um novo método que pode ser chamado a partir da variável à qual está atribuída (veja o exemplo de código abaixo).

O método retornado será "vinculado" à instância dada ou struct, o que significa que será sempre executado no escopo daquela instância/estrutura.

Você pode ligar funções incorporadas, bem como funções/métodos definidos pelo usuário, e você também pode fornecer undefined como o argumento instância/estrutura, o que significa que o atual self será utilizado para a encadernação.

Sintaxe:

method(struct_ref_or_instance_id, func);

Argumento Tipo Descrição
struct_ref_or_instance_id Estrutura ou identificação da instância The unique reference or ID value of the struct or instance to use (can be self ou undefined)
func Função ou Métododo Script The ID of the function (or method reference) to use

Devoluções:

Method

Exemplo:

var _inst = instance_position(mouse_x, mouse_y, obj_Enemy);
if instance_exists(_inst)
{
    enemy_func = method(_inst, enemy_ai);
}

O código acima verificará se existe uma instância inimiga na posição do mouse. Se existir, então o método enemy_ai é vinculado à instância inimiga e retornado como uma nova variável do método "enemy_func".
