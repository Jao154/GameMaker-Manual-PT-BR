is_infinity

Esta função retorna se uma determinada variável é infinity (um número infinito) ou não, retornando true se for, e false se não for.

Sintaxe:

is_infinity(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

O código acima verifica o "valor" da variável global para ver se ela é infinita ou não e mostra uma mensagem de debug se ela é.
