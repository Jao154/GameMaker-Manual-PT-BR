variável_instância_existe

Com esta função você pode verificar se uma variável de escopo de instância existe ou não. Você fornece o valor único de identificação da instância (que pode ser encontrado nas propriedades da instância no editor room, ou é retornado quando você chama a função instance_create_layer()) bem como o nome da variável a ser verificada como uma string (ver código de exemplo abaixo). A função retornará true se existir uma variável com o nome dado para a instância e false caso contrário.

Sintaxe:

variable_instance_exists(instance_id, name);

Argumento Tipo Descrição
instance_id Identificação da instância The unique ID value of the instance to check
name Cordão The name of the variable to check for

Devoluções:

Boolean

Exemplo:

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

O código acima verificará se a variável chamada "shields" existe na instância que executa o código e se não existe, então ela é criada e inicializada a 0.
