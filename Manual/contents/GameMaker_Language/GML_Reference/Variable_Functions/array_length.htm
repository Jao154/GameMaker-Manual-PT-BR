array_length

Com esta função você pode obter o comprimento (número de entradas) de uma dimensão de matriz. Você fornece o valor do índice da matriz e a função retornará um valor inteiro representando o número de entradas que a matriz contém. Esta função também pode ser usada para matrizes multidimensionais, desde que você especifique qual dimensão você deseja obter o comprimento de quando você fornece o índice da matriz, seguindo este padrão:

// Returns the first dimension of the array
val = array_length(my_array);

// Returns the second dimension of the array
val = array_length(my_array[0]);

// Returns the third dimension of the array
val = array_length(my_array[0][0]);

// etc...

Sintaxe:

array_length(array_index);

ArgumentoTipo Descrição
array_index The index of the array to check.

Devoluções:

Exemplo:

for (var i = 0; i < array_length(a); ++i)
{
    a[i] = -1;
}

O código acima irá loop através de uma matriz e definirá cada entrada para -1.
