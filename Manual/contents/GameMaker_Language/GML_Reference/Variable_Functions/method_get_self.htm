método_get_self

Com esta função você pode recuperar o ID da instância ou a referência struct que é o self contexto usado quando o método é chamado. Se a variável não for um método, então a função retornará undefined. Observe que a função também pode retornar a constante pointer_null, neste caso o eu atual está sendo usado no momento da chamada.

Sintaxe:

method_get_self(method);

ArgumentoTipo Descrição
method The method variable to check.

Devoluções:

Instance ID, Struct, undefined, or pointer_null

Exemplo:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

O código acima obtém o contexto próprio para a variável do método dado e o envia para o console.
