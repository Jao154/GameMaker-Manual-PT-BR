tipo de

Esta função retorna o tipo de dados de qualquer variável como um string. Os possíveis valores de retorno estão listados na tabela abaixo:

Cordão Descrição
"number" A variável contém um número real (ponto flutuante) - isso pode incluir NaN e infinity
"string" A variável contém um fio
"array" A variável faz referência a uma matriz
"bool" A variável contém um booleano (true / false)
"int32" A variável contém um número inteiro de 32 bits
"int64" A variável contém um inteiro de 64 bits
"ptr" A variável contém um ponteiro
"undefined" A variável é indefinida
"null" A variável tem um valor nulo (isto não deve ser visto normalmente)
"vec3" A variável possui um vetor de 3 valores
"vec4" A variável possui um vetor de 4 valores
"method" A variável contém uma referência de função
"struct" A variável contém uma referência struct
"ref" A variável contém uma referência de instância
"unknown" O valor é desconhecido. Isto nunca deve ser visto e significa que algo deu errado no nível mais básico, como uma sobregravação de memória.

Favor observar que há casos em que esta função pode não retornar o valor correto para um método. Considere as duas definições de função a seguir:

a = function()
{
    // something
}

function b()
{
    // Something
}

Tecnicamente, estes dois métodos são considerados métodos, pois ligam uma função a uma variável de escopo de instância, entretanto, ligar para typeof() na função b retornará "número" e não "método", enquanto que ligar para a retornará "método". Isto se deve ao fato de que aos métodos criados como o de b são atribuídos índices script (que são valores inteiros), já que esta é a forma pela qual o compilador reconhece funções de script, mesmo que a função não tenha sido definida em um script.

Sintaxe:

typeof(variable);

Argumento Tipo Descrição
variable Qualquer The variable to get the data type of.

Devoluções:

String (see table above)

Exemplo:

var _str = typeof(global.ExtensionInput);
show_debug_message(" global.ExtensionInput is a " + _str);

O código acima recebe o tipo de dados mantido pela variável global dada e retorna o string para uma variável local que é então usada para emitir uma mensagem para o console.
