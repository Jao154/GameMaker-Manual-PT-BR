conjunto_de_instância_variável

Com esta função você pode definir o valor de uma determinada variável em uma instância. Você fornece o valor único de identificação da instância (que pode ser encontrado nas propriedades da instância no editor room, ou é retornado quando você chama a função instance_create_layer()), bem como o nome da variável para definir o valor de uma string (ver código de exemplo abaixo), e finalmente o valor a ser definido (pode ser qualquer tipo de dado válido). Se a variável não existir já na instância, ela será criada e então lhe será atribuído o valor.

Sintaxe:

variable_instance_set(instance_id, name, val);

Argumento Tipo Descrição
instance_id Identificação da instância The unique ID value of the instance to use
name Cordão The name of the variable to set (as a string)
val Variável The value to set the variable to

Devoluções:

N/A

Exemplo:

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

O código acima verificará se uma variável de instância existe na instância de chamada e se não existe, então ela é criada e definida como 0.
