exemplo de

Esta função pode ser usada para obter o nome da função do construtor que foi usada para criar um struct. O próprio struct deveria ter sido criado usando o new operador junto com o próprio construtor. Você fornece a variável com a referência struct para verificação e a função retornará ou um string com o nome do construtor ou undefined.

Note que se você passar a função um struct literal (ou seja: um struct que foi criado sem usar uma função construtora), então ele simplesmente retornará a "estrutura" string .

Esta função também pode ser usada para verificar se uma referência struct é uma referência fraca ou não, nesse caso a função retornará o string "weakref" em vez do nome da função que criou o struct. Para mais informações, veja a função weak_ref_create().

O nome do construtor retornado incluirá texto adicional se a função do construtor não foi criada em um script. Se foi criada dentro de um object, seu nome incluirá o object e o evento onde foi criada. Por exemplo, um construtor chamado test_constructor criado no evento Create of Object1 será retornado como "test_constructor_gml_Object_Object1_Create_0". Para evitar isto, crie seus construtores em um script.

Sintaxe:

instanceof(struct_id);

Argumento Tipo Descrição
struct Estrutura The struct reference to use.

Devoluções:

String or undefined

Exemplo ampliado:

Neste exemplo, devemos primeiro definir a função que será usada como construtor para nosso struct. A seguinte função é definida em um ativo script:

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

Esta função pode então ser usada junto com o operador new para criar um struct e preenchê-lo com as variáveis definidas com os valores dos argumentos usados na função:

mystruct = new init_struct(10, 100, "Hello World");

Podemos então obter o nome da função do construtor que foi usada desta forma:

var _name = instanceof(mystruct);
if is_string(_name)
{
    show_debug_message(_name);
}

Isto imprimirá o string "init_struct" para o log de saída, que é o nome da função do construtor, como definido em seu script.
