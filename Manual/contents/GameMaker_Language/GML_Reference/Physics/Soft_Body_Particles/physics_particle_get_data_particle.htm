fisica_partícula_get_data_particle

Esta função retorna várias informações sobre uma única partícula na simulação física usando o dado flags verificado. O índice de partículas (seu ID) é o que foi retornado pela função physics_particle_create()e o buffer utilizado deve ter sido criado anteriormente usando a função buffer_create(). Deve ser do tipo "crescer", com o tamanho aproximado do dos dados de retorno esperados. Os próprios flags são definidos usando as constantes dadas abaixo, e você usaria o bitwise ou "|" para criar um único valor flag para obter as informações desejadas.

Constante Descrição Tipo de dados
phy_particle_data_flag_typeflags O valor flags para a partícula. buffer_u32
phy_particle_data_flag_position A posição x e y da partícula. 2 x buffer_f32
phy_particle_data_flag_velocity A velocidade horizontal e vertical. 2 x buffer_f32
phy_particle_data_flag_colour A cor e o valor alfa (hexadecimal). buffer_f32
phy_particle_data_flag_category A categoria da partícula (como definida quando você criou a partícula). buffer_u32

Sintaxe:

physics_particle_get_data_particle(ind, buffer, flags)

Argumento Tipo Descrição
ind Identificação de Partículas Físicas The index (ID) of the particle to get the data from.
buffer Identificação do tampão The (previously created) buffer to use to store the data.
flags Dados de Partículas Físicas Flag Constante(s) The flags to use to extract data about specific particle types.

Devoluções:

N/A

Exemplo:

var part = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)
var flags = phy_particle_data_flag_position | phy_particle_data_flag_velocity;
var buffer = buffer_create(16, buffer_grow, 4);
physics_particle_get_data_particle(part, buffer, flags);
px = buffer_read(buffer, buffer_f32);
py = buffer_read(buffer, buffer_f32);
pvelx = buffer_read(buffer, buffer_f32);
pvely = buffer_read(buffer, buffer_f32);
buffer_delete(buffer);

O código acima obtém a posição e velocidade da partícula indexada pela variável "parte" e armazena os dados em uma série de variáveis.
