fisica_particle_get_max_count

Com esta função você pode descobrir qual é o valor atual do tampão nas partículas permitidas na simulação física (você pode definir este valor usando physics_particle_set_max_count()).

Sintaxe:

physics_particle_get_max_count()

Devoluções:

Exemplo:

if (physics_particle_count() < physics_particle_get_max_count())
{
    physics_particle_create(0, x, y, 0, 0, c_white, 1, 1)
}

O código acima verificará se há menos do que o número máximo de partículas permitidas no room, e se for o caso, criará mais uma.
