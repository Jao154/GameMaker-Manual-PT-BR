escala_física_particular_agravidade_agravidade

Com esta função você pode descobrir qual é o fator de escala de gravidade atual para partículas na simulação física (você pode definir este valor usando physics_particle_set_gravity_scale()).

Sintaxe:

physics_particle_get_gravity_scale()

Devoluções:

Exemplo:

physics_particle_set_gravity_scale(physics_particle_get_gravity_scale() + 0.1);

O código acima estabelecerá a escala de gravidade.
