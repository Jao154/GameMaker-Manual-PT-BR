física_particle_group_begin

Com esta função você pode criar um grupo de partículas em um room. Um grupo de partículas é aquele em que as partículas necessárias para criar uma forma específica são criadas todas de uma só vez como um grupo, permitindo-lhe criar corpos moles simulados com várias propriedades definidas pelo flags utilizado. As "bandeiras" são o valor de retorno de uma combinação das constantes das partículas (veja a função physics_particle_create() para detalhes destes), enquanto as "groupFlags" são uma combinação das seguintes constantes de grupo (ou 0 para no flags):

Grupo de Partículas Físicas Flag Constante
Constante Descrição
phy_particle_group_flag_solid Um grupo de partículas sólidas impede que outros dispositivos se alojam dentro dele. Se algo penetrar nele, o grupo de partículas sólidas empurra o dispositivo ofensivo de volta para sua superfície, fazendo com que um grupo de partículas sólidas possua uma força repulsiva especialmente forte.
phy_particle_group_flag_rigid Os grupos rígidos de partículas são aqueles cuja forma não muda, mesmo quando colidem com outros dispositivos.

Este grupo flags utiliza a bit-masking para criar um valor de saída final que é então verificado para definir as diferentes propriedades básicas do grupo, além daquelas definidas pela partícula flags. Você pode usar nenhuma, uma ou ambas estas constantes usando o bitwise ou "|" para mascarar os bits apropriados (como mostrado no exemplo abaixo).

Além do flags, você também pode definir a posição no room para criar o grupo de partículas (a posição dada será o centro de massa do grupo acabado), a velocidade inicial horizontal e vertical, bem como a velocidade angular do grupo, permitindo a criação de grupos móveis e rotativos.

Você também pode definir a cor inicial e alfa de todas as partículas do grupo, com os valores sendo usados ao desenhar as partículas usando a função physics_particle_draw(). Depois disso, você estabelece a força coesiva das partículas no grupo, que é outra forma de dizer o quanto elas "se colam" umas às outras. A força pode ser um valor entre 0 e 1, onde 0 não é coesão, e 1 é coesão máxima.

Finalmente, você tem que dar a categoria para seu grupo de partículas. Este valor é um valor inteiro arbitrário que você dá a cada grupo de partículas que você deseja que tenha propriedades similares, e pode ser usado para definir mais tarde flags para aquela categoria específica, bem como uma série de outras coisas.

NOTA: Você pode usar qualquer valor inteiro para a categoria , exceto 0 que é reservado pela simulação e é usado para selecionar todas as categorias em outras funções (como para desenho).

É importante notar que chamar esta função não cria o grupo na Física room. Esta função simplesmente inicia o processo e define as propriedades da partícula base para o grupo. Para realmente criar o grupo de partículas é necessário dar-lhe uma forma (veja o exemplo abaixo) e então chamar a função physics_particle_group_end()e, em seguida, será criado.

NOTA: A partícula do tipo flags que você definir também influenciará a coesão do grupo de partículas, de modo que se você flag as partículas como sendo do tipo água (por exemplo), quando o grupo colide com outra fixação ele se separará, enquanto que se você flag a partícula como um tipo mola, então ele manterá sua coesão na colisão.

Sintaxe:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

Argumento Tipo Descrição
flags Partícula Física Flag Constante(s) The flags to set on the particle.
groupflags Grupo de Partículas Físicas Flag Constante(s) The group flags to set on the particle group (see table at the top).
x Real The x position to create the particle group at.
y Real The y position to create the particle group at.
ang Real The angle (in degrees) to rotate the group to on create.
xv Real The initial horizontal velocity.
yv Real The initial vertical velocity.
ang_velocity Real The initial angular velocity.
col Cor The base colour to use for the particle group.
alpha Real The base alpha to use for the particle group.
strength Real Set the cohesion strength between particles in the group.
category Real The user defined category to which the group belongs.

Devoluções:

N/A

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

O código acima armazena o flags para o tipo de partícula e as propriedades do grupo de partículas em variáveis e depois as utiliza para criar um grupo de partículas circular com um raio de 100px na posição do mouse.
