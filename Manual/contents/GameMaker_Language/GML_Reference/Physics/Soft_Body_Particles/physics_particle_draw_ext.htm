physics_particle_draw_ext

Esta função irá desenhar um sprite na posição de todas as partículas que compartilham o mesmo valor de categoria definido pelo usuário, e que têm flags que coincidem com as definidas para a máscara. O valor da máscara é definido da mesma forma que se definiria o flags ao criar as partículas (veja as funções physics_particle_create() por exemplo), ou seja: usando o bitwise ou "|" para mascarar os bits apropriados para cada flag.

NOTA: Esta função atrairá todas as partículas na simulação física com flags que coincidam com as da máscara, sejam elas partículas individuais ou grupos, se a categoria do usuário for a mesma.

A partícula desenhada ignorará a cor e os valores alfa definidos quando você a criou, e será desenhada usando os parâmetros de escala, ângulo, cor e alfa dados nesta função. Note que isto é puramente para desenhar um sprite na posição da partícula e de nenhuma maneira afeta o comportamento da partícula.

Sintaxe:

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

Argumento Tipo Descrição
typemask Partícula Física Flag Constante(s) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Ativo Sprite The sprite to use for each particle.
subimg Real The image_index do duende a sacar.
xscale Real The scale factor to apply to the sprite along the x axis.
yscale Real The scale factor to apply to the sprite along the y axis.
ang Real The angle (in degrees) to draw the sprite at.
col Cor The colour to blend with the sprite (default is c_white).
alpha Real The alpha value to use when drawing (0 - 1).

Devoluções:

N/A

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

O código acima desenhará um sprite para todas as partículas da categoria "2" definida pelo usuário que têm o mesmo flags da máscara dada, escalada por 2 ao longo de ambos os eixos e com um alfa de 0,5.
