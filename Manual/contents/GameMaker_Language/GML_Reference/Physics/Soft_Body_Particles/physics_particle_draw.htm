fisica_particle_draw

Esta função irá desenhar um sprite na posição de todas as partículas que compartilham o mesmo valor de categoria definido pelo usuário, e que têm flags que coincidem com as definidas para a máscara. O valor da máscara é definido da mesma forma que se definiria o flags ao criar a partícula (veja a função physics_particle_create() por exemplo), ou seja: usando o bitwise ou "|" para mascarar os bits apropriados para cada flag.

NOTA: Esta função atrairá todas as partículas na simulação física com flags que coincidam com as da máscara, sejam elas partículas individuais ou grupos, se a categoria do usuário for a mesma.

A partícula desenhada usará a cor e os valores alfa definidos quando você a criou, e girará com o momento angular da partícula. Entretanto, se você desejar escalar o sprite ou alterar qualquer outra de suas propriedades desenhadas, você deve usar a função physics_particle_draw_ext().

Sintaxe:

physics_particle_draw(typemask, category, sprite, subimg)

Argumento Tipo Descrição
typemask Partícula Física Flag Constante(s) The mask to compare the particles flags with.
category Real The user defined category to draw (use 0 to draw all categories).
sprite Ativo Sprite The sprite to use for each particle.
subimg Real The image_index do duende a sacar.

Devoluções:

N/A

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

O código acima desenhará um sprite para todas as partículas da categoria "2" definida pelo usuário que têm o mesmo flags da máscara dada.
