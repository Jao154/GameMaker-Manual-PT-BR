física_particle_group_add_ponto_add

Esta função estabelece um ponto no room para definir a forma de um polígono que será usado para criar um grupo de partículas macias do corpo. Você deve ter sinalizado previamente ao GameMaker que vai definir a forma de um polígono usando a função physics_particle_group_polygon( ) e depois use esta função para definir os pontos individuais do polígono. Você deve dar pelo menos três pontos ao definir a forma do polígono, mas pode dar até oito, e a função permitirá a definição de polígonos côncavos. Entretanto, se você gerar um polígono com quaisquer cavidades, os pontos dentro dele serão ignorados e uma forma convexa será criada para o grupo de partículas.

Sintaxe:

physics_particle_group_add_point(x, y)

ArgumentoTipo Descrição
x The x position in the room for the point.
y The y position in the room for the point.

Devoluções:

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

O código acima armazena o flags para o tipo de partícula e as propriedades do grupo de partículas em variáveis e depois as utiliza para criar um grupo de partículas poligonais de três lados na posição do mouse.
