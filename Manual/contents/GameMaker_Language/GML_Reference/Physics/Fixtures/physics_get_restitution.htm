physics_get_restitution

Quando você amarra um acessório a uma instância usando physics_fixture_bind() isto retorna um "id" para a fixação do limite. Você pode usar este id para obter a restituição (a propriedade "bounciness") valor da fixação vinculada, não a fixação "base", a qualquer momento usando esta função.

Sintaxe:

physics_get_restitution(fixture)

ArgumentoTipo Descrição
fixture the id of the bound fixture

Devoluções:

Exemplo:

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

O código acima obtém o valor de restituição atual para as propriedades físicas vinculadas da instância e depois as define para um valor diferente.
