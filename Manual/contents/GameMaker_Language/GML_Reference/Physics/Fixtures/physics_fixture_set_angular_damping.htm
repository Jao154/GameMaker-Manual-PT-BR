fisica_fixture_set_angular_damping

Se você pensar em qualquer rotação object no "mundo real", a menos que ele tenha um motor ou esteja no espaço, ele abranda com o tempo devido à influência de forças externas (como o atrito com o ar ao seu redor). Podemos usar a função physics_fixture_set_angular_damping() no GameMaker para simular este efeito e reduzir a velocidade de rotação de instâncias no mundo físico, pois, sem ela, qualquer instância rotativa continuaria a girar infinitamente. Os parâmetros de amortecimento devem estar entre 0 e infinito, com 0 significando sem amortecimento, e infinito significando amortecimento total. Normalmente você usará um valor de amortecimento entre 0 e 1, mas você pode usar qualquer valor não-negativo se necessário.

Sintaxe:

physics_fixture_set_angular_damping(fixture, damping)

ArgumentoTipo Descrição
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

Devoluções:

Exemplo:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

O código acima definirá o amortecimento angular da fixação indexada em "fix_ball" para 0,1.
