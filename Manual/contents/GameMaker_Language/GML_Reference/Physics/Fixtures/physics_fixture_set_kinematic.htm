fisica_fixture_set_kinematic

Haverá certas instâncias em um jogo baseado na física que você quer mover mas não deseja ser agido por forças como a gravidade, nem forças incorridas por colisões com dinâmicas objects (pense em mover plataformas em um jogo de plataforma, pois exemplo). Para tal objects, a simples definição da densidade de uma fixação a 0 significará que a física assumirá que o object pretende ser estático. Entretanto, caso você defina a phy_speed_x, phy_speed_y ou phy_angular_velocity variáveis para uma instância que tenha tido tal fixação, ela será convertida em um cinemático object e iniciar o movimento. Mudar a instância de estática para cinemática de simulação média pode ser caro em termos de processamento (dependendo do estado das instâncias no mundo da física) e, portanto, é recomendável definir o fixação para ser cinemático usando esta função antes de ligá-lo a uma instância, para que não seja afetado por colisões e forças (como a gravidade), mas possa se mover com velocidade linear e/ou angular constante.

Sintaxe:

physics_fixture_set_kinematic(fixture)

ArgumentoTipo Descrição
fixture the index of the fixture

Devoluções:

Exemplo:

physics_fixture_set_kinematic(fix_Cloud);

O código acima indicará que a fixação indexada em "fix_Cloud" deve ser marcada como cinemática.
