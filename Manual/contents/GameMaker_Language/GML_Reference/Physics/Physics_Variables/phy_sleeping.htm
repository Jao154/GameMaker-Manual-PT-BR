f_sleeping

Esta variável somente leitura retorna se a instância está "dormindo" (true) ou não (false), Uma instância "dormindo" é aquela que não está ativamente envolvida em nenhuma simulação física. O GameMaker colocará objects em repouso para salvar ciclos de simulação quando uma instância estiver em repouso e não em colisão com outra instância.

Sintaxe:

phy_sleeping;

Devoluções:

(or undefined if the instance is not physics enabled)

Exemplo:

if phy_sleeping
{
    instance_destroy();
}

O código acima verifica se o object está sendo ativamente simulado ou não e se não está sendo destruído.
