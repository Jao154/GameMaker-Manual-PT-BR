phy_active

Esta variável controla se a instância está ou não atualmente "ativa". Configurá-la para false impedirá que a instância participe do mundo físico, e configurá-la para true fará com que ela participe novamente. Observe que isto não é o mesmo que desativar a instância, pois a instância ainda é visível na tela e ainda pode ser alterada através de código, ao contrário, esta função apenas a impede de participar da simulação física.

Sintaxe:

phy_active;

Devoluções:

(or undefined if the instance is not physics enabled)

Exemplo:

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

O código acima detectará uma tecla pressionada da letra "P" e então alternará a variável global "Pause" de true para false e vice-versa. Esta variável é então usada para definir se a física está ativa ou não nas instâncias crianças do object indexado na variável "obj_Parent".
