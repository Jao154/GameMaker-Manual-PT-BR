phy_fixed_rotation

Esta variável pode ser usada para definir se a instância pode ou não ser afetada por forças rotacionais (o padrão é false). Se esta for definida para true, nenhuma força externa (seja de impulsos ou forças codificadas, ou de colisões) afetará o valor de rotação da instância e isso teria de ser definido manualmente usando o phy_rotation variável

Sintaxe:

phy_fixed_rotation;

Devoluções:

(or undefined if the instance is not physics enabled)

Exemplo:

phy_rotation = 0;
phy_fixed_rotation = true;

O código acima irá switch a instância para ter uma rotação fixa, depois definir o ângulo de rotação.
