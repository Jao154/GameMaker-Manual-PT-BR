phy_speed

Esta variável somente de leitura retorna a velocidade atual da instância física habilitada, definida em pixels por etapa. Caso você precise alterar este valor, deve fazê-lo alterando os vetores x e y usando as variáveis phy_speed_x e phy_speed_y.

Sintaxe:

phy_speed;

Devoluções:

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemplo:

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

O código acima verifica a velocidade da instância física habilitada e depois altera o amortecimento linear com base no valor retornado.
