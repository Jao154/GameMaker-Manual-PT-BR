physics_pause_enable

Normalmente para que uma simulação física funcione, ela deve ser contínua e não pode ser interrompida e iniciada, ou ter instâncias subitamente movidas de um lugar para outro no room. No entanto, há momentos em que você precisa pausar a simulação como (por exemplo) o dispositivo os pausou, e assim você usaria esta função. Se o flag estiver configurado para true, a simulação será pausada e nenhum outro cálculo físico será feito até que o flag seja configurado novamente para false.

NOTA: Isto é de particular utilidade caso você deseje desativar todas as instâncias em um room, pois mesmo quando desativado, um corpo físico continuará sendo calculado e simulado no mundo da física.

Sintaxe:

physics_pause_enable(flag)

ArgumentoTipo Descrição
flag This can be set to true para interromper a simulação, ou false para recomeçá-la.

Devoluções:

Exemplo:

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_PauseMenu);
}

O código acima verifica se o sistema operativo foi pausado e se então faz uma pausa no mundo da física, desativa tudo, exceto a si mesmo, e então cria uma instância de menu de pausa.
