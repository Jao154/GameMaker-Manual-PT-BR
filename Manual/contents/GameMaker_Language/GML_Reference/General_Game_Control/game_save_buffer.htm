jogo_save_buffer

Esta é uma variante da função game_save(), portanto, leia primeiro sua página, pois contém informações importantes relacionadas ao seu uso e também a esta função.

Com esta função você pode salvar o estado atual do jogo para um "crescer" previamente criado buffer (ver Buffers) que pode então ser carregado novamente a qualquer momento usando game_load_buffer() função. Esta função foi projetada para uso em um único room de cada vez (ou seja: não é aconselhável fazer um buffer salvo em um room, depois tente carregar o buffer de outro) e idealmente deve ser usada apenas para pontos de controle ou reinícios de nível.

NOTA: Esta função é muito limitada e foi projetada para que o iniciante tenha um sistema de ponto de verificação funcionando rapidamente, mas os usuários mais avançados podem preferir codificar seu próprio sistema usando as funções de Arquivo, devido ao fato de que o jogo não salvará nenhum dos recursos dinâmicos como estruturas de dados, superfícies, adicionou sprites etc.

Sintaxe:

game_save_buffer(buffer);

ArgumentoTipo Descrição
buffer The buffer id to save to.

Devoluções:

Exemplo:

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

O código acima definirá uma variável global para true e então salvará o estado atual do jogo no buffer indexado na variável "save_buff" quando a tecla "S" for pressionada.
