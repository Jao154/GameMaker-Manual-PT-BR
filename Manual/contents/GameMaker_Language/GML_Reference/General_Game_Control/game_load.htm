jogo_load

Com esta função você pode carregar um jogo que tenha sido salvo anteriormente usando game_save(). Note que ele irá restaurar a versão do jogo que foi usada para criar o save, portanto, quaisquer atualizações feitas após ele não serão visíveis. Para mais informações, leia a página em game_save().

Sintaxe:

game_load(filename);

Argumento Tipo Descrição
filename Cordão The name of the file to load from.

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(ord("L"))
{
    if (global.Save) game_load("Save.dat");
}

O código acima carregará um jogo previamente salvo se uma variável global for true quando o jogador apertar a tecla "L".
