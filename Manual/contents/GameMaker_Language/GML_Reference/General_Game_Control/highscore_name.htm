highscore_name

Com esta função, você pode recuperar o nome string que foi armazenado na lista de pontuação alta na posição determinada. Se nenhum nome tiver sido inserido, o string "Desconhecido" será devolvido.

Sintaxe:

highscore_name(place);

ArgumentoTipo Descrição
place The place on the table (1-10).

Devoluções:

Exemplo:

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

O código acima irá loop através da lista de pontuação alta e armazenará todos os nomes em uma matriz.
