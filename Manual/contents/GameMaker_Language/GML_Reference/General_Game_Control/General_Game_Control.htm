Controle geral do jogo

Estas funções e variáveis estão todas relacionadas à funcionalidade específica do jogo, como carregamento/reserva, reinício e assim por diante.

As seguintes variáveis de escopo global integradas são úteis para debugging seu jogo - elas podem ser usadas como saída para um arquivo de log no caso de problemas, por exemplo - assim como para mostrar o ID de construção atual, etc... para o usuário:

  • game_id
  • game_save_id
  • game_display_name
  • game_project_name

As seguintes funções são utilizadas para o controle geral do jogo:

  • game_end
  • game_restart
  • game_load
  • game_load_buffer
  • game_save
  • game_save_buffer
  • game_get_speed
  • game_set_speed

Para ajudá-lo a criar e manter pontuações altas para seus jogos, o GameMaker cria uma matriz global de pontuação alta de 10 lugares que você pode acessar, adicionar e alterar para criar suas próprias listas personalizadas de pontuação alta sem muito alarde. Esta funcionalidade é local para o jogo e funciona em todas as plataformas, assim você pode facilmente armazenar e manter uma tabela básica de alta pontuação usando estas funções junto com as Funções de Manuseio de Arquivos.

Existem as seguintes funções que lidam com a lista interna de pontuação alta.

  • highscore_add
  • highscore_name
  • highscore_value
  • highscore_clear

O GameMaker também inclui como lista uma função para desenhar as pontuações altas na tela, porém é muito básico e é recomendável que você use isso para debugging mais do que qualquer outra coisa, pois fazer sua própria tabela de pontuação alta é fácil e lhe dará muito mais controle sobre como ela é exibida:

  • draw_highscore

Finalmente, temos uma função que lhe permite atribuir um sprite ao cursor do mouse:

  • cursor_sprite
