jogo_save

Esta é uma função herdada que pode ser usada para salvar o estado atual do jogo, e não é mais recomendada para uso. Use as funções File para criar um sistema de salvamento personalizado onde você só salva e carrega dados específicos do jogo.

Esta função salvará o estado do jogo como ele é, porém não salvará nenhum recurso dinâmico sendo usado naquele momento, como estruturas de dados, superfícies, assets adicionado em runtime, etc. Se tal arquivo de gravação for carregado, qualquer atualização do jogo que tenha sido feita após a gravação não será visível, pois restaurará a versão do jogo que foi usada para criar a gravação.

Note que salvar arquivos criados usando esta função pode não ser suportado por versões atualizadas em Runtime, portanto, substituir esta função por um novo sistema de salvamento é essencial para garantir a compatibilidade com futuras atualizações GameMaker, e pelas razões listadas acima, atualizações feitas em seu próprio jogo.

Sintaxe:

game_save(filename);

ArgumentoTipo Descrição
filename The name of the file to save the game to.

Devoluções:

Exemplo:

if keyboard_check_pressed(ord("S"))
{
    global.Saved = true;
    game_save("Save.dat");
}

O código acima definirá uma variável global para true e então salvará o jogo no arquivo "Save.dat" quando a tecla "S" for pressionada.
