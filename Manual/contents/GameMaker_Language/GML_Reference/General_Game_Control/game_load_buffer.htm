jogo_load_buffer

Com esta função você pode carregar um estado de jogo que já tenha sido salvo anteriormente. O jogo é carregado a partir de um "grow" buffer previamente criado (ver Buffers) e o buffer deve ter tido um estado de jogo salvo a ele usando game_save_buffer() função.

Sintaxe:

game_load_buffer(buffer);

ArgumentoTipo Descrição
buffer The buffer id to load from.

Devoluções:

Exemplo:

if keyboard_check_pressed(ord("L"))
{
    if (global.Checkpoint) game_load_buffer(save_buff);
}

O código acima carregará um estado de jogo previamente salvo do buffer indexado na variável "save_buff", somente se a variável global for true, quando o jogador pressiona a tecla "L".
