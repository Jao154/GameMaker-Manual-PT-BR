cartas_de_filtro

Com esta função você pode remover todos os caracteres que não são classificados como letras. Isto é útil para evitar que as pessoas digitem caracteres indesejados em uma entrada de texto (como digitar "#" para forçar uma nova linha).

NOTA Esta função detecta e retorna apenas o alfabeto inglês de 26 letras de A a Z (maiúsculas ou minúsculas).

Sintaxe:

string_letters(string);

ArgumentoTipo Descrição
string The string to convert to letters.

Devoluções:

Exemplo:

username = string_letters(username);

O código acima definirá a variável "username" para conter apenas os caracteres das letras do original string (id o original string era "MacSw33ny" o retorno string seria "MacSwny").
