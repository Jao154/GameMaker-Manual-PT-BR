string_lower

Com esta função você pode forçar um string a conter apenas caracteres minúsculos.

NOTA Esta função detecta apenas o alfabeto inglês de 26 letras de A a Z.

Sintaxe:

string_lower(string);

ArgumentoTipo Descrição
string The string to convert to lowercase.

Devoluções:

Exemplo:

str1 = "Hello World";
str2 = string_lower(str1);

O código acima irá definir str2 para "olá mundo".
