dígitos_de_filtro

Você pode usar esta função para analisar um determinado string e obter qualquer número dele. Por exemplo, digamos que você tenha este texto - "Eu tenho 81 anos". Com esta função, você obteria um retorno string de "81".

Sintaxe:

string_digits(string);

ArgumentoTipo Descrição
string The string to get the digits from.

Devoluções:

Exemplo:

var t_str = string_digits(input_str);
age = real(t_str);

O código acima pegará a entrada string, desnudará todos os caracteres que não sejam números e então definirá a variável age para manter o valor real do número do retorno string (assim, se a entrada string foi - por exemplo - "I am 18", o função retornaria "18").
