comprimento_de_filtro

Esta função retorna o número de caracteres que compõem um determinado string. Ela pode ser útil para coisas como trabalhar quando se deve limitar o comprimento de um texto personalizado (por exemplo: limitar o nome de um jogador a 10 caracteres). Lembre-se de que isto é diferente para string_width() na medida em que mede o número de caracteres no string, e não sua largura como desenhada na tela em pixels.

Sintaxe:

string_length(string);

Argumento Tipo Descrição
string Cordão The string to measure the number of characters of.

Devoluções:

Real

Exemplo:

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

Isto verificará se o string do nome é maior que dez caracteres e, se for, copiará e utilizará apenas os primeiros dez caracteres.
