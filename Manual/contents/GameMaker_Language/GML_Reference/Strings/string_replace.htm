string_replace

Você pode usar esta função para analisar um string procurando uma peça específica, que pode então ser substituída pelo novo string que você especificou.

Sintaxe:

string_replace(str, substr, newstr);

ArgumentoTipo Descrição
str The string to be copied.
substr The substring within the string to be replaced.
newstr The new substring to replace the previous one.

Devoluções:

Exemplo:

str1 = "Hello Earth";
str2 = string_replace(str1, "Earth", "World");

Isto colocará str2 em str1, mas com seu exemplo de "Terra" substituído por "Mundo", resultando em str2 como "Olá Mundo".
