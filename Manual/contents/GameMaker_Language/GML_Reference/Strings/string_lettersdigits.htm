string_lettersdigits

Esta função devolverá uma cópia de um dado string com tudo menos suas letras e dígitos removidos, o que significa que ela pode ser usada para remover quaisquer caracteres indesejados (como "#" ou "?") de, por exemplo, um nome de login ou uma senha.

NOTA Esta função só retorna o alfabeto inglês de 26 letras de A a Z (maiúsculas ou minúsculas) e os números 0 a 9.

Sintaxe:

string_lettersdigits(string);

ArgumentoTipo Descrição
string The string to convert to letters and digits.

Devoluções:

Exemplo:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
    draw_text(32,32, "Invalid Password! Only Letters and numbers please!");
}

O código acima verificará o comprimento de um string em relação ao comprimento do mesmo string, mas com todos os símbolos removidos. Se eles não forem os mesmos, uma mensagem é desenhada na tela.
