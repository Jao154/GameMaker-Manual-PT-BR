ansi_char

Esta função retorna um string contendo o caractere com o conjunto de valores brutos BYTE. Isto não será, e não deve ser exibido, mas será salvo corretamente no disco para uso na codificação.

Sintaxe:

ansi_char(val);

ArgumentoTipo Descrição
val The raw byte value.

Devoluções:

(Single character)

Exemplo:

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

O código acima cria um string a partir de dados de bytes brutos e o grava em um arquivo (previamente aberto).
