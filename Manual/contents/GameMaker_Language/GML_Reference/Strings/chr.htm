chr

Esta função retorna um string contendo o caractere que está relacionado ao código Unicode de entrada para exibição. Este caractere depende do desenho atual fonts página de código de conjunto de caracteres e se não for definido font, ele usará a página de código padrão para a máquina.

Sintaxe:

chr(val);

ArgumentoTipo Descrição
val The Unicode code value to get the string from

Devoluções:

Exemplo:

mystring = chr(53) + chr(48);

Isto colocaria mystring a "50" (a string, não um inteiro), pois chr(53) é "5" e chr(48) é "0".
