string_upper

Com esta função, você pode forçar um string a conter apenas caracteres maiúsculos.

NOTA Esta função detecta apenas o alfabeto inglês de 26 letras de A a Z.

Sintaxe:

string_upper(string);

ArgumentoTipo Descrição
string The string to convert to uppercase.

Devoluções:

Exemplo:

str1 = "Hello World";
str2 = string_upper(str1);

O código acima definirá a variável str2 como "HELLO WORLD".
