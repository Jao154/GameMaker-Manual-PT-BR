string_last_pos_ext_ext

Esta função retornará a posição de caráter de uma instância de uma sub-corda dentro de um string, pesquisando para trás através do string a partir da posição dada como posição inicial. A função retornará 0 se a busca string não for encontrada, ou a posição do primeiro caractere da busca string, se for. Tenha em mente que para suporte antigo strings são indexados a partir de 1, portanto 1 é a primeira posição no string, e não 0 como você pode esperar.

Sintaxe:

string_last_pos_ext(substr, str, start_pos);

ArgumentoTipo Descrição
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

Devoluções:

Exemplo:

if (string_last_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_last_pos_ext(",", text, 20));
}

O código acima procura no string armazenado na variável "texto" por uma vírgula antes do 20º caractere, e se encontrar uma, insere o "nome" do substrato nessa posição.
