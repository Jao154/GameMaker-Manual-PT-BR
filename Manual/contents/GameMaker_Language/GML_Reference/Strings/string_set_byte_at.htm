string_set_byte_at

Esta função define um byte diretamente em um string (baseado no formato UTF8) e retorna uma cópia do string com as mudanças.

NOTA Esta função é incrivelmente lenta, portanto, considere cuidadosamente se é necessária e onde você a utiliza.

Sintaxe:

string_set_byte_at(str, pos, byte);

ArgumentoTipo Descrição
str The string to change the byte of.
pos The position within the string (starting at 1) to change the byte of.
byte The new byte value.

Devoluções:

Exemplo:

str = string_set_byte_at("hello", 2, 97);

O código acima mudaria o valor do byte da segunda letra no string, e assim definir a variável str para manter "hallo".
