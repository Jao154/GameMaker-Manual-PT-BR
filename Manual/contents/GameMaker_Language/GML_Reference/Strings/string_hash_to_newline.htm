string_hash_to_newline

Esta função retorna um string onde o símbolo "#" foi convertido em uma nova linha. Assim, um string formatado como:

"Hello#World"

Seria impresso para a tela como:

Hello
World

IMPORTANTE! Esta função é fornecida para compatibilidade de importação entre GameMaker e versões anteriores do software e, como tal, esta função não deve ser usada, exceto quando necessário para replicar funcionalidades obsoletas. Em vez disso, você deve estar usando os caracteres de fuga para marcar uma nova linha. Para mais informações sobre compatibilidade de importação, veja aqui.

Sintaxe:

string_hash_to_newline(string);

ArgumentoTipo Descrição
string The string to convert over multiple lines

Devoluções:

Exemplo:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

O código acima converte o string com o símbolo hash em um string dividido em duas linhas e depois o imprime para a tela.
