altura_de_filtro

Esta função retornará a altura (em pixels) da entrada string, levando em conta a separação de linhas e quaisquer quebras de linha que o texto possa ter. É muito útil para calcular as distâncias entre os elementos do texto com base na mais alta das letras que compõem o string como seria elaborado com draw_text() utilizando o atualmente definido font.

Sintaxe:

string_height(string);

ArgumentoTipo Descrição
string The string to measure the height of.

Devoluções:

Exemplo:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

O código acima terá a altura do dado string e então desenhará duas linhas de texto, usando a altura retornada string como separador.
