string_ord_at

Você pode usar esta função para retornar um código de caractere específico em uma posição específica dentro de um string, com o índice começando em 1 para o primeiro caractere. Se nenhum caractere for encontrado ou se o string for menor que o valor dado ao índice, -1 é retornado.

Sintaxe:

string_ord_at(str, index);

ArgumentoTipo Descrição
str The string to check.
index The position to get the character code from.

Devoluções:

Exemplo:

str = "Hello World";
char_code = string_ord_at(str, 7);

Isto irá obter o código do sétimo caractere (onde "H" conta como o primeiro) em string "str" e armazená-lo na variável "char_code".
