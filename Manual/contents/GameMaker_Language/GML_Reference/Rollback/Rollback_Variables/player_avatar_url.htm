player_avatar_url

Esta variável armazena a URL para o avatar do jogador, que pode ser carregada usando sprite_add().

Você pode usar player_avatar_sprite que já tem o sprite carregado.

Esta variável só existe em instâncias de jogadores criados por rollback_define_player()e é diferente para cada jogador.

Os jogadores convidados exibirão um espaço reservado sprite.

Sintaxe:

player_avatar_url;

Devoluções:

String

Voltar: Variáveis Rollback
