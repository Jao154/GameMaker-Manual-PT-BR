tipo player_type

Esta variável informa se a instância do jogador é um usuário conectado ao GX.games, ou um convidado. Ela pode retornar qualquer uma das seguintes cordas:

Cordão Descrição
"Guest" O jogador não está inscrito no GX.games
"User" O jogador é inscrito no GX.games

Os usuários convidados recebem avatares e nomes de titulares de lugar.

Esta variável só existe em instâncias de jogadores criados por rollback_define_player()e é diferente para cada jogador.

Sintaxe:

player_type;

Devoluções:

String
