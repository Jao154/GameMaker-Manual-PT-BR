rollback_game_running

Esta variável global indica se o jogo de rollback está correndo atualmente. Se isto for verdade, significa que o jogo de rollback já começou e está atualmente em execução. Em todos os outros momentos, será falso.

Sintaxe:

rollback_game_running

Devoluções:

Boolean

Exemplo:

if (!rollback_game_running) return;

x += move_x;
y += move_y;

O código acima seria executado no evento Step de um jogador object. Quando o jogo de rollback não estiver em andamento, ele sairia do evento, portanto o jogador não pode ser controlado até que o jogo de rollback tenha começado.
