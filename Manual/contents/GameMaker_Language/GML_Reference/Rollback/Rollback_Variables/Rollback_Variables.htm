Variáveis de Rollback

  • Variáveis de instância
    • player_id
    • player_local
    • player_name
    • player_avatar_sprite
    • player_avatar_url
    • tipo player_type
    • player_user_id
    • play_prefs
    • gerenciado
  • Variáveis globais
    • rollback_game_running
    • rollback_current_frame
    • rollback_confirmed_frame
    • rollback_api_server
