rollback_use_player_prefs

Esta função permite que você defina as preferências padrão de todos os jogadores.

As preferências padrão podem ser de qualquer tipo. Se você não especificar as preferências padrão, então todos os jogadores precisam ter preferências definidas antes que o jogo possa começar.

Chamando esta função irá desativar a partida automática, então você tem que iniciar o jogo manualmente. Veja: Manual vs. Auto Start

Se utilizada, esta função deve ser executada antes da função de junção/criação.

Por favor, leia: Preferências do jogador

Sintaxe:

rollback_use_player_prefs([default]);

Argumento Tipo Descrição
default Qualquer OPTIONAL As preferências padrão a serem definidas para todos os jogadores.

Devoluções:

N/A

Exemplo:

rollback_use_player_prefs({
caráter: 0,
cor: c_white
});

O código acima permite as preferências do jogador, e define as preferências padrão em um struct.
