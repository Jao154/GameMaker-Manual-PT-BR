rollback_set_player_prefs

Após habilitar as preferências, esta função é usada para definir as preferências para o jogador local. Isto significa que a função deve depender da entrada local e não da entrada Rollback.

As preferências podem ser de qualquer tipo. Se as preferências padrão forem definidas, certifique-se de usar o mesmo tipo e estrutura de dados.

Quando qualquer jogador define suas preferências, todos no jogo recebem um Evento de Rollback.

Se utilizada, esta função deve ser executada antes da função de junção/criação.

Por favor, leia: Preferências do jogador

Sintaxe:

rollback_set_player_prefs(preferences);

Argumento Tipo Descrição
preferences Qualquer The preferences to set for the local player.

Devoluções:

N/A

Exemplo:

rollback_set_player_prefs({
character: obj_customizer.character_selected,
cor: obj_customizer.color_selected
});

O código acima estabelece as preferências do jogador, lendo valores de um "customizador" object, que não seria gerenciado (local).
