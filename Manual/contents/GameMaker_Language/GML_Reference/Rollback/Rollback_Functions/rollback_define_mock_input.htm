rollback_define_mock_input

Esta função pode ser usada durante o Sync Test para atribuir uma entrada "mock" temporária ao jogador especificado. Isto substitui os controles definidos para o jogador e qualquer entrada aleatória do Sync Test.

Para informações sobre seu uso, veja Mock Input.

Sintaxe:

rollback_define_mock_input(player_id, input_struct);

Argumento Tipo Descrição
player_id Real The ID of the player to assign mock input to.
input_struct Estrutura A struct containing input names as variables, and their assigned inputs as values.

Devoluções:

N/A

Exemplo:

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: ord("A"),
    right: ord("D")
});

rollback_define_mock_input(1, {
    fire: vk_control,
    interact: vk_shift,
    left: ord("J"),
    right: ord("L")
});

O código acima define primeiro as entradas para os jogadores. Em seguida, ele define a entrada simulada para o jogador 1, mantendo os mesmos nomes de entrada, mas atribuindo entradas diferentes. Desta forma, duas pessoas poderiam jogar o jogo localmente usando o teclado.
