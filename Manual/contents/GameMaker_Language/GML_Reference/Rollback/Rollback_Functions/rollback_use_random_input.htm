rollback_use_random_input

Quando você joga um jogo usando o Sync Test, todos os jogadores "remotos" recebem valores aleatórios para suas entradas definidas, como uma forma básica de teste. Você pode desativar este comportamento chamando esta função com falso como argumento, e habilitá-la novamente especificando verdadeiro.

Sintaxe:

rollback_use_random_input(enabled);

Argumento Tipo Descrição
enabled Booleano Specifies whether random input should be enabled (true) ou desativado(falso). Por padrão, ele está habilitado.

Devoluções:

N/A

Exemplo:

rollback_use_random_input(false);

O código acima desativa a entrada aleatória durante o Sync Test.
