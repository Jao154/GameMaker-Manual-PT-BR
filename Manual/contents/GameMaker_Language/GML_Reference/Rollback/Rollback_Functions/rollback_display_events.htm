rollback_display_events

Por padrão, quando você executa o jogo, você vê mensagens de debug no canto superior esquerdo de seu jogo, exibindo eventos multiplayer à medida que eles acontecem. Você pode desativar essas mensagens chamando essa função com argumentos falsos, e habilitá-las novamente especificando verdadeiro.

Sintaxe:

rollback_display_events(enabled);

Argumento Tipo Descrição
enabled Booleano Specifies whether event display should be enabled (true) ou desativado(falso). Por padrão, ele está habilitado.

Devoluções:

N/A

Exemplo:

rollback_display_events(false);

O código acima desabilitou a exibição dos eventos.
