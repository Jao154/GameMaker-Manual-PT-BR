rollback_start_game

Esta função força o jogo multijogador atual a começar, antes que todos os jogadores tenham se juntado. Primeiro é necessário criar um jogo multiplayer usando rollback_create_game().

O sistema normalmente espera que todos os jogadores adiram antes de iniciar automaticamente o jogo. Esta função pode ser usada para iniciar o jogo mais cedo, antes que todos os jogadores tenham aderido.

Você pode chamar rollback_use_manual_start() para fazer Rollback esperar por esta função, mesmo depois que todos os jogadores tenham aderido.

A chamada desta função desencadeará o evento Rollback Start.

Sintaxe:

rollback_start_game();

Devoluções:

N/A

Exemplo:

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

Se o jogo Rollback não estiver funcionando, e Enter for pressionado, isto iniciará o jogo.

Isto permite que o jogador inicie o jogo manualmente antes de todos os jogadores aderirem.
