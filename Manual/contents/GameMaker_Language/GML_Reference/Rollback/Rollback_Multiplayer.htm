Rollback Multiplayer

O Rollback multiplayer em GameMaker permite configurar e criar rapidamente um jogo multiplayer em rede para o alvo Opera GX.

Esta funcionalidade usa servidores GX para conectar os jogadores e sincronizar as entradas, portanto não é necessária nenhuma configuração server pelo desenvolvedor para iniciar um novo jogo online.

Enquanto a funcionalidade online só pode ser usada com o alvo Opera GX, você pode usar o "Sync Test" para testar seu jogo multiplayer localmente, com entrada aleatória ou falsa sendo aplicada aos outros jogadores. O Sync Test é funcional nos alvos Windows e macOS, além do Opera GX.
