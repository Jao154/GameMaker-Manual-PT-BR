network_connect_async

Com esta função você pode enviar um pedido para conectar a um server. A função leva a identificação do soquete para conectar (veja network_create_socket()) e exige que você forneça o endereço IP para se conectar para (a string), bem como a porta de passagem, e se a conexão falhar, um valor inferior a 0 será devolvido. A conexão usa um protocolo especial que garante que somente os jogos GameMaker se conectem entre si, no entanto, se você precisar para se conectar a um jogo server que não é um jogo GameMaker, você pode usar network_connect_async_raw(). Note que esta função é assíncrona, gerando um evento de Rede Assíncrona do tipo network_type_non_blocking_connect.

Sintaxe:

network_connect_async(socket, url, port);

ArgumentoTipo Descrição
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Devoluções:

Exemplo:

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

O código acima criará um novo soquete TCP e então tentará fazer uma conexão assíncrona através dele com o endereço IP dado na porta 6510.
