network_connect_raw_async

Com esta função você pode enviar um pedido para conectar a um server. A função leva a identificação do soquete para conectar (veja network_create_socket()) e exige que você forneça o endereço IP para se conectar para (a string), bem como a porta de passagem, e se a conexão falhar, um valor inferior a 0 será devolvido. A diferença entre esta função e network_connect_async()é que este pode se conectar a qualquer server e não faz nada com os dados brutos, o que significa que você mesmo tem que implementar os protocolos no final do server. Note que esta função é assíncrona, gerando um evento de rede assíncrona do tipo network_type_non_blocking_connect.

Sintaxe:

network_connect_raw_async(socket, url, port);

ArgumentoTipo Descrição
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

Devoluções:

Exemplo:

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

O código acima criará um novo soquete TCP e então tentará criar uma conexão assíncrona "bruta" através dela para a URL dada na porta 6510.
