A API Steam suporta quadros de liderança persistentes com entradas ordenadas automaticamente. Estes quadros de líderes podem ser usados para exibir quadros de líderes globais e de amigos em seu jogo e na página web da comunidade para seu jogo. Cada jogo pode ter até 10.000 leaderboards, e cada leaderboard pode ser recuperado imediatamente após a pontuação de um jogador ter sido inserida nele, mas observe que para cada leaderboard, um jogador pode ter apenas uma entrada, embora não haja limite no número de jogadores por leaderboard.

Cada entrada do leaderboard contém um nome, uma pontuação e uma classificação para o leaderboard, e estes dados serão substituídos quando uma nova entrada do leaderboard for criada para o usuário, e as seguintes funções podem ser usadas para adicionar e recuperar estes dados do leaderboard para seu jogo:

  • steam_create_leaderboard
  • steam_upload_score
  • steam_upload_score_buffer
  • steam_download_scores
  • steam_download_scores_around_user
  • steam_download_friends_scores
