vapor_download_friends_scores

Com esta função você pode recuperar somente as pontuações no quadro de líderes que pertencem àquelas pessoas que estão marcadas como "amigos" no cliente Steam. Portanto, se seu quadro de líderes tem 200 entradas, e 50 delas são seus amigos, esta função recuperará apenas aqueles 50 resultados. O nome do quadro de líderes é um string que foi definido quando você criou o quadro de líderes usando a função steam_create_leaderboard()e a função retornará um valor que pode então ser usado para identificar a chamada de retorno no Evento Steam Async, ou retornará -1 se tiver falhado.

Sintaxe:

steam_download_friends_scores(lb_name);

Argumento Descrição
lb_name O nome do quadro de líderes do qual você está baixando as pontuações (a string).

Devoluções:

Real

Exemplo:

score_get = steam_download_friends_scores("Game Scores");

Isto enviará uma solicitação para o Steam Server para os amigos usuários, para as pontuações do quadro de líderes dado e armazenará a identificação assimétrica da solicitação na variável "score_get". Isto será então tratado no Evento Steam Async, como mostrado no Exemplo Ampliado para vapor_download_scores().
