vapor_disponível_línguas

Esta função pode ser usada para recuperar uma lista de todos os idiomas disponíveis para Steam. O valor retornado é simplesmente uma lista de idiomas separada por vírgulas.

Sintaxe:

steam_available_languages();

Devoluções:

String

Exemplo:

language = steam_available_languages();

Os idiomas disponíveis no site Steam são os acima mencionados como um string e o armazena em uma variável.
