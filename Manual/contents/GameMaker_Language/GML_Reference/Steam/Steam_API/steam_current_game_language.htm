linguagem_de_jogo_corrente_de_vapor

Esta função é usada para recuperar o idioma atual que Steam está usando (como um string), por exemplo "english".

Sintaxe:

steam_current_game_language();

Devoluções:

String

Exemplo:

language = steam_current_game_language();

O código acima obtém o idioma utilizado para o jogo atual.
