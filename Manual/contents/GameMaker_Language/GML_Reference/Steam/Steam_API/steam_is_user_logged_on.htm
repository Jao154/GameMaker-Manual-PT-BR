steam_is_user_logged_on

Esta função voltará a ser verdadeira se o cliente Steam tiver atualmente uma conexão ao vivo com o Steam servers. Se ela retornar falsa, significa que não há conexão ativa devido a um problema de rede na máquina local, ou o Steam server estar em baixo ou ocupado.

Sintaxe:

steam_is_user_logged_on();

Devoluções:

String

Exemplo:

if steam_is_user_logged_on()
{
    global.user_id = steam_get_user_account_id();
}

O código acima verificará se o usuário está logado no Steam server e se armazena o ID do usuário em uma variável global.
