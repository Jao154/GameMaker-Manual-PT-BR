Geral Steam API

O conjunto de funções a seguir são todas para verificar a disponibilidade de certos aspectos do cliente Steam ou server API, e como tal retornam verdadeiro ou falso dependendo de fatores como o server estar ocupado ou o usuário estar logado etc... Isso significa que essas funções devem ser usadas antes de qualquer outra chamada de função da API Steam para garantir que a configuração do cliente/servidor esteja correta e se comunique com seu jogo:

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
