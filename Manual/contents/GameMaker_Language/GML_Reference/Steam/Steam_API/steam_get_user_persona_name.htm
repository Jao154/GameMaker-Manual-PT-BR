steam_get_user_persona_name

Esta função pode ser usada para recuperar o nome do usuário (nome de tela) para qualquer ID de usuário específico. Você dá o ID do usuário e a função retorna um valor único de ID assimétrico que pode ser usado para verificar os detalhes corretamente no evento Steam Async. Este evento terá a função async_load Mapa DS que será povoado com os seguintes pares de chaves/valores:

  • "id" - A identificação assimétrica retornada pela função que acionou o evento
  • "event_type" - O string "user_persona_name"
  • "steamid"" - O valor de identificação Steam usado para obter o nome do usuário.
  • "persona_name" - O nome do usuário para o dado Steam ID

Sintaxe:

steam_get_user_persona_name(steamID);

Argumento Descrição
steamID O ID único Steam para um usuário.

Devoluções:

Real

Exemplo:

request = steam_get_user_persona_name(global.UGC_UserID);

O código acima solicitará o nome do usuário do ID do usuário armazenado na variável global "UGC_UserID", armazenando o valor retornado em uma variável para análise no Evento Async.
