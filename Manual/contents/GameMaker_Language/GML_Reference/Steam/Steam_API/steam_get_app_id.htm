steam_get_app_id

Esta função é utilizada para recuperar o ID exclusivo do aplicativo que Steam designa para seu jogo, que é necessário para usar algumas das funções de Conteúdo Gerado pelo Usuário.

Sintaxe:

steam_get_app_id();

Devoluções:

Real

Exemplo:

global.app_id = steam_get_app_id();

O código acima obtém o ID exclusivo do aplicativo para seu jogo em Steam e o armazena em uma variável global.
