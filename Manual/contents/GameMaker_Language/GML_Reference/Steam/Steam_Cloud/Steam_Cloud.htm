Nuvem de vapor

A nuvem Steam fornece um sistema de armazenamento remoto de arquivos fácil e transparente para seu jogo. Todos os arquivos escritos em disco usando as funções da nuvem GameMaker Studio 2 (listadas abaixo) serão replicados para o Steam servers após a saída do jogo. Se o usuário então mudar de computador, os arquivos serão então baixados para o novo computador antes do lançamento do jogo, significando que o jogo poderá então acessar os arquivos lendo-os usando as funções apropriadas Steam. O cliente Steam faz o trabalho de garantir que os arquivos sejam mantidos sincronizados em todos os computadores que o usuário possa estar acessando.

NOTA Por padrão, a Nuvem não está habilitada para um jogo no Steamworks. ela deve ser habilitada anteriormente na guia "Nuvem" do administrador do jogo Steamworks, onde você deve definir o byte e a cota de arquivo. Na próxima vez que você publicar sua configuração de jogos Steamworks, o armazenamento em Nuvem estará pronto para uso.

As seguintes funções podem ser usadas para acessar a nuvem Steam de dentro do GameMaker Studio 2

  • steam_is_cloud_enabled_for_app
  • steam_is_cloud_enabled_for_account
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • steam_file_size
  • steam_file_persisted
  • steam_file_write
  • steam_file_write_file
  • steam_file_read
  • steam_file_share
  • steam_file_delete
