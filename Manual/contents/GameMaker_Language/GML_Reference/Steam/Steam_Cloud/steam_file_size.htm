tamanho_do_arquivo_de_vapor

Com esta função, você pode verificar o tamanho de um arquivo armazenado na nuvem Steam. O número real retornado é o tamanho, em bytes, do arquivo.

Sintaxe:

steam_file_size(filename);

Argumento Descrição
filename O nome do arquivo para verificar o tamanho do mesmo.

Devoluções:

Real

Exemplo:

file_bytes = steam_file_size("Save.txt");

O código acima armazena o tamanho de um arquivo da nuvem Steam na variável "file_bytes".
