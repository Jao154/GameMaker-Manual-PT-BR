steam_file_share

Com esta função, você pode forçar seu jogo a sincronizar o arquivo dado com a nuvem Steam. Isto não é normalmente necessário devido ao fato de que o jogo sincronizará automaticamente no final da sessão do jogador, nem é recomendado pelo Steam, mas pode ser útil para garantir que as informações sensíveis sejam sincronizadas imediatamente. A função retornará um valor 0 se falhar por qualquer razão e um valor maior que 0 se for bem sucedida.

Sintaxe:

steam_file_share(filename);

Argumento Descrição
filename O nome do arquivo é sincronizado.

Devoluções:

Real

Exemplo:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

O código acima verificará se um arquivo foi armazenado na nuvem Steam, e se não foi, então o sincronizará.
