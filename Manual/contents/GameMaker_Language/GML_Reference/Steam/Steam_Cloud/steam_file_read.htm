steam_file_read

Esta função lerá o conteúdo do arquivo dado em um string que poderá ser analisado posteriormente em seu jogo.

Sintaxe:

steam_file_read(filename);

Argumento Descrição
filename O nome do arquivo a ser lido.

Devoluções:

String

Exemplo:

if steam_file_exists("Save.txt")
{
    save_str = steam_file_read("Save.txt");
}

O código acima verifica se um arquivo existe na nuvem Steam e se existe, ele o abre e lê seu conteúdo na variável "save_str".
