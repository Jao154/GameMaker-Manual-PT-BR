vapor_file_persisted

Com esta função, você pode verificar o arquivo dado para ver se ele foi sincronizado com a nuvem Steam. Um valor de retorno de verdadeiro significa que é, enquanto falso significa que não é.

Sintaxe:

steam_file_persisted(filename);

Argumento Descrição
filename O nome do arquivo a ser verificado.

Devoluções:

Boolean

Exemplo:

if !steam_file_persisted("Save.txt")
{
    steam_file_share("Save.txt");
}

O código acima verificará se um arquivo foi armazenado na nuvem Steam, e se não foi, então o sincronizará.
