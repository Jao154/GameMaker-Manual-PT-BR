steam_file_delete

Esta função excluirá o arquivo em questão da nuvem Steam. A função retornará um valor 0 se falhar por qualquer razão e um valor maior que 0 se for bem sucedida.

Sintaxe:

steam_file_delete(filename);

Argumento Descrição
filename O nome do arquivo excluir.

Devoluções:

Real

Exemplo:

if steam_file_exists("Save.txt")
{
    steam_file_delete("Save.txt");
}

O código acima verificará se um arquivo existe, e se existe, ele apaga o arquivo da nuvem Steam.
