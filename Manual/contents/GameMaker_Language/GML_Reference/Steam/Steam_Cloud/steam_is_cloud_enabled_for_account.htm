vapor_é_activado_para_conta

Com esta função você pode verificar se o serviço Steam Cloud está habilitado pelo usuário em suas configurações do cliente Steam. Ele retornará verdadeiro se for e falso caso contrário. Note que isto não significa automaticamente que você pode armazenar dados na Nuvem, pois também terá que ter sido habilitado para seu jogo (você pode verificar isto usando a função steam_is_cloud_enabled_for_app()). Se a nuvem Steam estiver habilitada para seu jogo, mas o usuário a tiver desligado localmente, você ainda poderá usar as funções da nuvem para armazenar e recuperar dados de uma cópia local de todos os arquivos, ela simplesmente não irá carregá-los para a nuvem no final do jogo, nem sincronizá-los no início do jogo.

Sintaxe:

steam_is_cloud_enabled_for_account();

Devoluções:

Boolean

Exemplo:

if steam_is_cloud_enabled_for_account
{
    steam_file_share("Save.txt");
}

O código acima verifica se o usuário tem a nuvem Steam habilitada e se ela retornar verdadeiro, então sincronizará o arquivo dado.
