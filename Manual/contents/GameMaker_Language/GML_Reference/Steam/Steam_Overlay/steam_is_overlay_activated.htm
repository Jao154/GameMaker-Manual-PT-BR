vapor_é_activado_sobre_camada

Esta função pode ser usada para descobrir se o usuário tem ou não a sobreposição Steam ativa. Se a sobreposição estiver ativa e visível para o usuário, a função retornará verdadeira, e se não estiver, então retornará falsa. Um exemplo do que esta função pode ser usada seria para pesquisar a API Steam para a sobreposição, para que você possa pausar seu jogo enquanto a sobreposição está sendo mostrada.

Sintaxe:

steam_is_overlay_activated();

Devoluções:

Boolean

Exemplo:

if steam_is_overlay_activated()
{
    global.Pause = true;
}

O código acima verificará se a sobreposição do Steam está ativa e se é ela que definirá a variável global "Pausa" como verdadeira.
