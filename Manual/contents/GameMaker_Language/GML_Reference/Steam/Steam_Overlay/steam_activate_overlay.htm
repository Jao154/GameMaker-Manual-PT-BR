vapor_ativar_sobreposição

A sobreposição Steam é uma peça da interface do usuário Steam que pode ser ativada por cima de quase todos os jogos lançados através de Steam. Ela permite que o usuário acesse sua lista de amigos, navegador da web, bate-papo e compra de DLC dentro do jogo. A chave padrão para um usuário acessar a sobreposição enquanto estiver em um jogo é SHIFT+TAB, mas você também pode trazer à tona qualquer página da sobreposição usando esta função. É necessária uma das seis constantes que estão listadas abaixo:

Constante Descrição
ov_friends A página de amigos para o usuário atual
ov_community A página da comunidade para seu jogo
ov_players A página mostrando outros que estão jogando o mesmo jogo ou que você jogou recentemente com
ov_settings As configurações de sobreposição de clientes Steam
ov_gamegroup Abre o Steam Navegador da comunidade para o grupo oficial de jogos deste jogo
ov_achievements A página de conquistas para seu jogo

Sintaxe:

steam_activate_overlay(overlayindex);

Argumento Descrição
overlayindex O índice da página do Steam API UI para mostrar (veja as constantes listadas abaixo).

Devoluções:

N/A

Exemplo:

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay(ov_friends); break;
    case vk_f2: steam_activate_overlay(ov_community); break;
    case vk_f3: steam_activate_overlay(ov_players); break;
    case vk_f4: steam_activate_overlay(ov_settings); break;
    case vk_f5: steam_activate_overlay(ov_gamegroup); break;
    case vk_f6: steam_activate_overlay(ov_achievements); break;
}

O código acima pesquisa a última tecla do teclado pressionada e se for alguma das teclas de função de 1 a 6 abrirá a página correspondente da sobreposição do Steam.
