vapor_é_apto_a_sobre_camada

Ao utilizar a API Steam, esta função pode ser chamada para verificar se a API do cliente Steam tem a funcionalidade de sobreposição ativada.

Sintaxe:

steam_is_overlay_enabled();

Devoluções:

Boolean

Exemplo:

global.steam_api = false;
if steam_initialised()
{
    if steam_stats_ready() && steam_is_overlay_enabled()
    {
        global.steamapi = true;
    }
}

O código acima definirá uma variável global como verdadeira se a API do cliente Steam for inicializada corretamente, juntamente com a estatística Steam e a funcionalidade de sobreposição, ou definirá a variável como falsa de outra forma.
