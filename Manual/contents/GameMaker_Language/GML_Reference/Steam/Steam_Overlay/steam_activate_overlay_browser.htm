vapor_ativar_ativar_sobre_browser

Com esta função você pode abrir o jogo Steam overlay para seu navegador da web e depois fazer com que ele carregue a URL especificada. você precisa usar a URL completa como um http://www.steamgames.com/ para que isso seja resolvido corretamente, por exemplo: " string ".

Sintaxe:

steam_activate_overlay(url);

Argumento Descrição
url A URL (completa) para a sobreposição a ser aberta.

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_browser("http://www.steamgames.com/");
}

O código acima pesquisa o teclado para a tecla F1 e se for então Steam overlay será aberto e resolvido para a URL dada.
