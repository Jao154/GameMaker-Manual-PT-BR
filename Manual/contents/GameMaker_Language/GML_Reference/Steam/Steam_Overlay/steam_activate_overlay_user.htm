vapor_ativar_overlay_user

Esta função abrirá a sobreposição de Steam para um dos diálogos escolhidos em relação à identificação do usuário dado. Os diálogos possíveis strings disponíveis são:

  • "steamid" - abre o navegador da comunidade Steam para a página do usuário ou grupo
  • "chat" - abre uma janela de chat para o usuário especificado

Note que Steam IDs podem ser grandes números e, portanto, você pode precisar lançar seu valor de ID como um int64() antes de fornecê-lo para a função.

Sintaxe:

steam_activate_overlay_user(dialog_name, steamid);

Argumento Descrição
dialog_name O diálogo para abrir a sobreposição (ver abaixo).
steamid O Steam ID de usuário ou ID de grupo a ser usado.

Devoluções:

N/A

Exemplo:

var key = keyboard_lastkey;
switch (key)
{
    case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
    case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

O código acima pesquisa a última tecla do teclado pressionada e se for a tecla de função 1 ou a tecla de função 2, abrirá a sobreposição de Steam para ver o grupo Steam armazenado na variável global "GamegroupID", ou abrirá a janela de chat para conversar com o usuário armazenado na variável global "FriendID".
