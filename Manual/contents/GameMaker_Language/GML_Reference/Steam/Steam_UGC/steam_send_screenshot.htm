steam_send_screenshot

Com esta função você pode carregar uma captura de tela para a página do perfil da comunidade Steam do usuário atualmente conectado. O nome do arquivo que você fornece é o nome do arquivo local que foi criado quando você tirou a captura de tela usando as funções GameMaker Studio 2. screen_save() ou screen_save_part(). A largura e a altura definem o tamanho da imagem, e a função retornará um valor 0 se falhar por qualquer motivo e um valor maior que 0 se for bem sucedida.

Sintaxe:

steam_send_screenshot(filename, width, height);

Argumento Descrição
filename O nome do arquivo de imagem a ser carregado (a string).
width A largura da imagem (um número inteiro).
height A altura da imagem (um número inteiro).

Devoluções:

Real

Exemplo:

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

O código acima pesquisará a API Steam para uma solicitação de captura de tela e, se tiver sido, será gerado um nome único para o arquivo de imagem, será tirada uma captura de tela e o arquivo será enviado para a página da comunidade Steam para o usuário.
