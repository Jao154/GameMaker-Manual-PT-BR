steam_ugc_set_item_title

Esta função definirá o título a ser usado para o item em questão. O título deve ser um string e a alça de atualização é o valor retornado quando você chama a função steam_ugc_start_item_update(). A função retornará verdadeira se a API foi acessada com sucesso e falsa se houvesse um problema.

Sintaxe:

steam_ugc_set_item_title(ugc_update_handle, title);

Argumento Descrição
ugc_update_handle O cabo único para a UGC a ser atualizado.
title O título (a string - máximo 128 caracteres) a ser usado para o item.

Devoluções:

Boolean

Exemplo:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

O código acima obtém o ID do jogo, então usa-o junto com um ID de arquivo publicado anteriormente para gerar uma alça de atualização para o item. Este cabo é então usado para atualizar várias informações antes que a atualização seja empurrada para o Workshop. servers .
