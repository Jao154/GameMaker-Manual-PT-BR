steam_ugc_query_set_cloud_filename_filtro

Esta função pode ser usada para filtrar ainda mais qualquer consulta UGC, especificamente para verificar e ver se um nome de arquivo de item de Workshop deve ou não corresponder. O tratamento da consulta é o valor retornado quando você criou a consulta (usando, por exemplo, steam_ugc_create_query_user()) e o segundo argumento ou é verdadeiro ou falso, dependendo se você necessita que os nomes dos arquivos coincidam. A função retornará verdadeiro se o filtro de consulta foi configurado corretamente, ou falso caso contrário.

Sintaxe:

steam_ugc_query_set_cloud_filename_filter(ugc_query_handle , match_cloud_filename);

Argumento Descrição
ugc_query_handle O cabo de consulta a ser usado.
match_cloud_filename Define se o nome do arquivo do item UGC deve ou não corresponder.

Devoluções:

Boolean

Exemplo:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_set_cloud_filename_filter(query_handle, true);
steam_ugc_query_add_excluded_tag(query_handle, "nasty chips");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

O código acima cria um pedido de consulta e armazena seu tratamento em uma variável local para uso futuro nas demais funções que definem melhor o pedido de consulta antes de enviar a consulta.
