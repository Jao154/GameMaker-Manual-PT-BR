steam_ugc_get_item_install_info

Esta função pode ser usada para recuperar informações sobre qualquer item de arquivo publicado que tenha sido subscrito e baixado para a área de armazenamento local Steam para seu jogo. Você dá o ID do item e fornece o índice a um mapa DS vazio que será então preenchido com os seguintes pares de chaves/valores, se o item existir:

  • "size_on_disk" - O tamanho do arquivo em disco (em bytes)
  • "legacy_item" - Será verdadeiro ou falso, dependendo se é ou não um arquivo legado.
  • "pasta" - Esta é a pasta completa path para o conteúdo instalado (consulte "Item Installation" em Steam Os documentos SDK, como itens "legados" carregados com o método antigo, são tratados de forma diferente).

Se o item existir, então a função retornará verdadeiro e povoará o mapa, caso contrário, retornará falso e o mapa permanecerá vazio.

Sintaxe:

steam_ugc_get_item_install_info(published_file_id, info_map);

Argumento Descrição
published_file_id O cabo único para a UGC a ser atualizado.
info_map Um índice do mapa DS (anteriormente criado).

Devoluções:

Boolean

Exemplo:

var item_map = ds_map_create();
steam_ugc_get_item_install_info(global.fileID, item_map);

O código acima irá consultar o status de instalação do item indexado na variável global "fileID", usando um DS Map para armazenar as informações.
