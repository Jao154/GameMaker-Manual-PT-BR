steam_is_screenshot_requested

Esta função vai pesquisar a API Steam para ver se a tecla para tirar uma captura de tela do jogo foi pressionada. A função só retornará verdadeiro para um passo (tique do jogo) quando a tecla for pressionada, e retornará falso em todos os outros momentos.

Observe que se a tecla de captura de tela for pressionada, esta função só retornará true uma vez para cada etapa que for pressionada, e retornará false para quaisquer chamadas subseqüentes dentro da mesma etapa. Por exemplo, se uma captura de tela for solicitada no quadro atual e você ligar para esta função no evento Step para descobrir isso, você receberá true; no entanto, se você ligar novamente no GUI Draw para verificar se uma captura de tela foi solicitada, você receberá false, pois a função já tinha sido "usada" no evento Step. Para usar o valor de retorno da função várias vezes dentro do mesmo quadro, é recomendável armazená-la em uma variável e ler isso ao invés de chamar a função novamente.

OBSERVAÇÃO: Esta função não tira uma foto de tela para você. Isto só sinaliza que a tecla foi pressionada e você deve usar as funções GameMaker Studio 2. screen_save() ou screen_save_part() para salvar uma cópia local do arquivo a ser carregado.

Sintaxe:

steam_is_screenshot_requested();

Devoluções:

Boolean

Exemplo:

if steam_is_screenshot_requested()
{
    var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
    screen_save(file)
    steam_send_screenshot(file, window_get_width(), window_get_height());
    global.scrn_num += 1;
}

O código acima pesquisará a API Steam para uma solicitação de captura de tela e, se tiver sido, será gerado um nome único para o arquivo de imagem, será tirada uma captura de tela e o arquivo será enviado para a página da comunidade Steam para o usuário.
