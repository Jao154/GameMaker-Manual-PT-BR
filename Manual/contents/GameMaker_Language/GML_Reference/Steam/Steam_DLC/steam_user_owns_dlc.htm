steam_user_owns_dlc

Se seu jogo tem DLC criado para ele, você pode usar esta função para verificar se o usuário o comprou antes de acessar qualquer arquivo associado a ele. A função retornará verdadeiro (1) se o jogador possuir o conteúdo, falso (0) se não possuir o DLC ou se o DLC ID dado for inválido, ou -1 se não estiver logado em Steam.

Note que mesmo que o usuário seja proprietário do DLC, ele pode não ter o DLC instalado, portanto você deve usar adicionalmente a função steam_user_installed_dlc() para garantir que ele seja instalado antes de utilizá-lo. Note que Steam IDs podem ser grandes números e, portanto, você pode precisar lançar seu valor de ID como um int64() antes de fornecê-lo para a função.

Sintaxe:

steam_user_owns_dlc(dlc_id);

Argumento Descrição
dlc_id O identificador único para o DLC a ser verificado (um número inteiro int64).

Devoluções:

Integer

Exemplo:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
    if steam_user_installed_dlc(10354)
    {
        global.Level_max = 200;
    }
}

O código acima verificará se o usuário comprou, e instalou, o DLC com o id 10354, e se assim for, definirá uma variável global para um valor diferente.
