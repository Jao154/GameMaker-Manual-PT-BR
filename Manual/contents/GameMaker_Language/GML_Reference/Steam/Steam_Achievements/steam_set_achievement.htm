vapor_set_achievement

Com esta função você pode dizer à API Steam para conceder ("set") uma conquista para o jogador. Estas conquistas deveriam ter sido definidas na página de contas do painel de controle Steamworks para seu jogo e o string que é passado para a função deve corresponder ao que é usado como o nome da API no painel de controle. A Steam Game Overlay exibirá um painel de notificação ao usuário informando sobre a realização que recebeu, a menos que a realização já tenha sido concedida, caso em que nada acontecerá.

Sintaxe:

steam_set_achievement(ach_name);

Argumento Descrição
ach_name O nome da realização a ser definida (string).

Devoluções:

N/A

Exemplo:

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

O código acima recompensará o jogador uma conquista se a variável global "Mortes" for igual a 10 e se a conquista ainda não tiver sido concedida.
