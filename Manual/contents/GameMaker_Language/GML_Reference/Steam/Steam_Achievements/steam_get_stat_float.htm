steam_get_stat_float

Com esta função, você pode obter o valor de uma estatística específica de ponto flutuante. A estatística deve ter sido previamente definida na página de contas do painel de controle Steamworks para seu jogo e o string que é passado para a função deve corresponder ao que é usado como o Nome API no painel de controle.

Sintaxe:

steam_get_stat_float(stat_name);

Argumento Descrição
stat_name O nome da estatística a obter (a string).

Devoluções:

Real

Exemplo:

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
{
    steam_set_stat_int("Travelled", dist_pc);
}

O código acima calcula uma porcentagem com base na variável distância percorrida "dist" e a distância máxima que você pode percorrer "dist_max". Em seguida, ele pesquisa o valor atual para a estatística "Travelled" e, se for menor que o valor calculado, ele define novamente a estatística.
