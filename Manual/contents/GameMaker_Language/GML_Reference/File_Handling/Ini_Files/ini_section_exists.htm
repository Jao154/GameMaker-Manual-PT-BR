ini_section_existe

Esta função verifica se existe uma seção no ini atualmente aberto e retornará true se existir ou false caso contrário. Esta não é uma verificação necessária para evitar erros, pois, quando uma seção não existe, a leitura de uma seção inexistente apenas retornará um valor padrão, porém pode ser útil para ver se um arquivo ini salvou dados específicos.

Sintaxe:

ini_section_exists(section);

ArgumentoTipo Descrição
section The section in the open .ini file to check for.

Devoluções:

Exemplo:

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

Isto definirá a variável global.savegame para "1" se "save1" NÃO for encontrado. Com um para loop, isto poderia ser usado como uma forma básica de calcular quantos jogos salvos existem para um jogo (ou seja: criando um sistema para vários jogos salvos simultaneamente para um jogo).
