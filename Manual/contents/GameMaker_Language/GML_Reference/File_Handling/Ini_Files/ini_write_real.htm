ini_write_real

Você pode usar esta função para escrever um valor (numérico) em um arquivo de dados ini. Os arquivos Ini são divididos em seções e, em seguida, cada seção é dividida em pares de chaves - valores. Assim, um arquivo ini típico se pareceria com algo assim:

Sintaxe:

ini_write_real(section, key, value);

ArgumentoTipo Descrição
section The section of the .ini to write to.
key The key within the relevant section of the .ini to write to.
value The real value to write to the relevant destination.

Devoluções:

Exemplo:

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

Isto definirá a pontuação para 1000, depois abra "savedata.ini" e escreva este valor para "save1" > "Score". Em seguida, definirá a pontuação2 para o valor real em "save1" > "score" nele, 1000, e fechará o arquivo .ini novamente. Se não tivesse encontrado este valor, a pontuação 2 teria acabado fixada em 0.
