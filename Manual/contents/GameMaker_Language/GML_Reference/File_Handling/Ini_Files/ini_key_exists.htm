ini_key_existe

Esta função verifica se existe uma chave no ini atualmente aberto e retornará true se existir ou false caso contrário. Esta não é uma verificação necessária para evitar erros, pois, quando uma chave não existe, a leitura de uma chave inexistente apenas retornará um valor padrão. Pode ser útil ver se um arquivo ini salvou dados específicos e algumas outras coisas, no entanto.

Sintaxe:

ini_key_exists(section, key);

ArgumentoTipo Descrição
section The section in the open .ini file to check a key in.
key The key to check for.

Devoluções:

Exemplo:

ini_open("savedata.ini");
if !ini_key_exists("save1", "name")
{
    global.name = "Player1";
}
ini_close();

Isto definirá a variável global.name como "Player1" se não existir uma chave como "name".
