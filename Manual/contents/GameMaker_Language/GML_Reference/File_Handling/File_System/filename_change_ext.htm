nome_de_arquivo_mudar_extra

Esta função retorna o nome do arquivo indicado com o extension (incluindo o ponto) alterado para o novo extension. Usando um string vazio como o novo extension você pode remover a parte extension todos juntos.

Sintaxe:

filename_change_ext(fname, newext);

ArgumentoTipo Descrição
fname The file to use.
newext The new extension to use.

Devoluções:

Exemplo:

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

O código acima recebe o nome do arquivo (como um string) com a parte extension removida.
