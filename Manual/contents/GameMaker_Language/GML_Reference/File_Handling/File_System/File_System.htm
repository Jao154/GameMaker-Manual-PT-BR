Sistema de arquivo

Existe no GameMaker uma série de funções que permitem a manipulação de arquivos de todos os tipos de diferentes maneiras:

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

As seguintes funções podem ser usadas para mudar os nomes dos arquivos:

NOTA Estas funções não funcionam com os arquivos reais! Elas lidam apenas com o strings relacionado aos diferentes elementos path e devem ser usadas em conjunto com as outras funções dos arquivos.

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

Finalmente há algumas funções especiais para obter nomes de arquivos que podem ser usados para carregar e salvar arquivos com filtros do tipo arquivo. Estas funções abrirão uma janela exploradora e, portanto, farão com que o jogo perca o foco. Isto significa que qualquer entrada que o jogo estiver dependendo pode não ser corretamente registrada, e como tal, recomendamos que (se usar estas funções em um evento de teclado) você use o evento de tecla para cima e não os eventos pressionados ou para baixo.

NOTA Estas funções são apenas Windows e macOS.

AVISO Estas funções podem não funcionar como você espera devido ao GameMaker estar sendo lixado! Consulte a seção sobre O Sistema de Arquivo para maiores informações.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
