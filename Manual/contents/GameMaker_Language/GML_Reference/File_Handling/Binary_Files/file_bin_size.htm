file_bin_size

Esta função devolverá o tamanho (em bytes) de um arquivo que foi aberto para leitura e/ou escrita. O ID do arquivo é o valor retornado pela função file_bin_open().

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Sintaxe:

file_bin_size(binfile);

ArgumentoTipo Descrição
binfile The ID of the file to get the size of.

Devoluções:

Exemplo:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_close(file);

Isto abriria um arquivo do diretório local do jogo, e atribuiria seu índice à variável "arquivo". Em seguida, ele obteria o tamanho desse arquivo e o fecharia novamente.
