diretório_existe

Esta função retornará true se o diretório indicado existir ou false se não existir. O nome especificado deve incluir o path completo, não um path relativo e, por padrão, não é possível acessar nenhum diretório de fora - com o pacote de jogos, pois todos os jogos são sandboxed (veja a seção sobre o Sistema de Arquivos para mais informações).

Sintaxe:

directory_exists(dname)

ArgumentoTipo Descrição
dname The name of the directory to look for.

Devoluções:

Exemplo:

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Isto verificará se o diretório especificado existe, então, se existir, vá lá e devolva o primeiro arquivo "read only" doc encontrado.
