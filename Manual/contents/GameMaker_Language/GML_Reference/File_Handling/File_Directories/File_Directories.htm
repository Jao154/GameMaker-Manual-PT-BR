Diretórios de arquivos

Ao trabalhar com arquivos, haverá momentos em que você precisará consultar o diretório no qual os arquivos são armazenados e o GameMaker tem algumas funções e variáveis especiais relacionadas aos diferentes diretórios possíveis que você possa precisar:

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory
  • game_save_id (do Controle Geral de Jogos)

NOTA Você não pode criar ou destruir diretórios em HTML5.
