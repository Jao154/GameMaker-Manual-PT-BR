file_text_write_string

Com esta função você pode escrever um string em um arquivo de texto previamente aberto. Se o arquivo já contém informações, estas informações serão apagadas e o string será escrito no início do arquivo, a menos que você tenha aberto o arquivo com o file_text_open_append(). Você também pode evitar isso usando o file_text_readln() função junto com o file_text_eof() para loop através do conteúdo do arquivo até chegar ao final e depois começar a escrever.

Sintaxe:

file_text_write_string(fileid, str);

ArgumentoTipo Descrição
fileid The id of the file to edit.
str The string to write to the file.

Devoluções:

Exemplo:

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

O código acima abre um arquivo para escrita e depois loops através de duas matrizes, escrevendo cada valor da matriz em uma nova linha do arquivo. O arquivo é então fechado quando o loop tiver terminado.
