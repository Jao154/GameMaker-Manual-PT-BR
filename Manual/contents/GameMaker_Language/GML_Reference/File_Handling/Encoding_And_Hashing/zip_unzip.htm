zip_unzip

Esta função abrirá um arquivo zip armazenado e extrairá seu conteúdo para o diretório dado. Note que se você não fornecer um path completo para o diretório ZIP, então a raiz do drive atual será usada, e se você quiser colocá-lo em um path relativo ao diretório de trabalho do pacote de jogos, então você deve usar a função working_directory como parte da variável path (relativo paths usando "." ou "..." não funcionará como esperado, portanto deve ser evitado). Observe também que o zip deve fazer parte do pacote do jogo (ou seja: um arquivo incluído) ou ter sido baixado para a área de armazenamento usando http_get_file().

A função retornará um valor indicando o número de arquivos extraídos, ou retornará 0 ou menos se a extração tiver falhado.

Sintaxe:

zip_unzip(zip_file, target_directory)

ArgumentoTipo Descrição
zip_file The zip file to open
target_directory The target directory to extract the files to

Devoluções:

Exemplo:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
{
    show_debug_message("Extraction Failed!");
}

O código acima abrirá o arquivo zip armazenado no diretório "downloads" e extrairá seu conteúdo para o diretório "extraído" (criando esse diretório se ele ainda não existir) e então verificará se a extração foi correta, mostrando uma mensagem de debug caso falhe.
