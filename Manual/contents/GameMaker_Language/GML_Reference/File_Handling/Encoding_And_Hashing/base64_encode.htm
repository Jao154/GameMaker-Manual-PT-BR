base64_código

Esta função irá converter um string em um formato base64 codificado string. Este é um esquema de codificação comumente utilizado para qualquer mídia que precise ser armazenada ou transferida pela Internet como texto, e torna a saída ilegível para o olho humano.

Sintaxe:

base64_encode(string)

ArgumentoTipo Descrição
string The string to encode.

Devoluções:

Exemplo:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

O código acima cobrirá o string armazenado em "game_data" em uma base64 codificada string que é então armazenada em um arquivo de texto externo.
