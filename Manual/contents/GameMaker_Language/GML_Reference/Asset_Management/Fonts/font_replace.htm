font_replace

Esta função pode ser usada para substituir um font em seu jogo daqueles fonts que estão instalados no sistema em que ele está rodando. Note que o font sendo substituído não pode ser um font do navegador Asset, e deve ser um font que foi adicionado anteriormente usando a função font_add().

Ao substituir um font desta forma, você pode definir o tamanho do font (em pontos), assim como se o font deve ser em negrito ou itálico, e você também pode definir o intervalo de caracteres a incluir. Se a função falhar, ela falhará silenciosamente e o dado font não será substituído.

Para o módulo de destino HTML5, esta função só pode ser usada para substituir Fontes Web, e só funcionará se WebGL não estiver habilitada - se WebGL estiver habilitada, então você deve usar um recurso font geral (ou seja: usando draw_set_font() juntamente com um font do Browser Asset ) ou bitmap fonts (ver font_add_sprite()).

IMPORTANTE Ao visar HTML5, o texto em itálico e negrito só é suportado se o próprio font asset ou web font tiver estes caracteres incluídos. Eles não serão sintetizados a partir da base font.

Em todos os outros alvos (não Web), você pode usar esta função para substituir um font de um arquivo. O arquivo deve ser incluído no pacote de jogos usando a funcionalidade Included Files do GameMaker, e deve ser um arquivo no formato *.ttf ou *.otf font , útil para adicionar arquivos não-padrão fonts como os asiáticos ou árabes.

DICA Se você incluir um arquivo font com um jogo, certifique-se de que ele esteja devidamente licenciado para distribuição com o jogo.

Ao carregar um font de um arquivo desta forma o tamanho do font está em pixels e o primeiro e último valores são ignorados, significando que todos os glifos no font serão adicionados. Fonts adicionados desta forma são atribuídos a sua própria página texture, portanto, deve-se ter cuidado ao usar esta função, pois ela aumentará o número de swaps texture ao desenhar. Vale notar também que fonts pode parecer ligeiramente maior quando desenhado, uma vez que os glifos podem ter partes que são desenhadas fora da caixa delimitadora definida para o font. Você também deve estar ciente de que há uma limitação de cerca de 200 glifos que podem ser renderizados em um único quadro para um determinado font.

NOTA Quando você carrega um font no GameMaker, você deve se lembrar de removê-lo novamente (com font_delete()) quando não for mais necessário, caso contrário há o risco de um vazamento de memória que irá retardar e, eventualmente, quebrar seu jogo.

Sintaxe:

font_replace(ind, name, size, bold, italic, first, last);

Argumento Tipo Descrição
ind Fonte Ativo The index of the (previously added) font to be replaced.
name Cordão The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf ou *.otf arquivo.
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Booleano Whether the font is bold (true) or not (false).
italics Booleano Whether the font is italic (true) or not (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Devoluções:

N/A

Exemplo:

font_replace(myfont, "Arial", 24, true, true, 32, 128);

Isto substituirá o font indexado na variável "myfont" por um novo font que é 24pt em tamanho, usa "Arial" e que é ousado e itálico. A faixa font inclui letras maiúsculas e minúsculas, números e todas as pontuações comuns.
