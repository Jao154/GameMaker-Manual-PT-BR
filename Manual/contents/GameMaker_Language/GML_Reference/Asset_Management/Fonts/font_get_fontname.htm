font_get_fontname

Com esta função você pode obter o nome real do sistema do dado font asset. Esta função retorna uma string e não um índice, e o nome retornado dependerá do font a ser utilizado. Por exemplo, você pode ter um font asset chamado "fnt_Main" no navegador Asset, e o próprio font pode ser o sistema Windows font Arial. Neste caso, a função retornará "Arial", pois é o nome do sistema font. Se você precisar do nome que aparece no Asset Browser, use font_get_name().

O comportamento descrito acima só se aplica a font assets adicionado através do IDE. Quando usado em um font carregado em runtime (usando font_add()), a função devolverá o path completo a esse arquivo font ao invés de seu nome font.

Sintaxe:

font_get_fontname(ind);

ArgumentoTipo Descrição
ind Index of the font to check.

Devoluções:

String

Exemplo:

fnt_Name = font_get_fontname(font0);

O código acima receberá o nome do sistema de um recurso font e o armazenará como um string na variável "fnt_Name".
