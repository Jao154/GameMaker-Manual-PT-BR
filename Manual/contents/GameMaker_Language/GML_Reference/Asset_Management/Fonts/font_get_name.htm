font_get_name

Esta função pode ser usada para obter o nome (como um string) que foi dado ao font quando foi adicionado como um asset ao GameMaker Asset Browser. Observe que este é apenas um string e não pode ser usado para fazer referência direta ao font - para isso você precisaria do índice de fontes. Você pode, entretanto, usar este string para obter o índice de fontes usando o retorno string junto com o função asset_get_index().

Sintaxe:

font_get_name(ind);

ArgumentoTipo Descrição
ind Index of the font to check.

Devoluções:

Exemplo:

fnt_Name = font_get_name(font0);

O código acima receberá o nome de um recurso font como aparece no Asset Browser e o armazenará como um string na variável "fnt_Name".
