font_replace_sprite_ext

Esta função substituirá um sprite font que você criou anteriormente usando a função font_add_sprite_ext(). Ela usa uma "faixa de duende", onde cada subimagem é um símbolo ou letra individual, e as imagens são mapeadas para letras com base no argumento "string_map" da função. Este argumento é um string que você pode usar para dizer ao GameMaker qual é a ordem das subimagens do sprite font e ele as mapeará automaticamente ao escrever o texto. Assim, por exemplo, se você tiver um "string-map" de "AaBbCcDdEeFfGgHh", seu sprite font deve ter as subimagens ordenadas desta forma.

Você também pode definir o espaçamento para que o font seja proporcional (true) ou não (false), onde um font proporcional é espaçado com base na largura real das letras (assim a letra "i" leva menos room do que a letra "w", por exemplo), enquanto um font não proporcional será espaçado com base na largura da subimagem, como um monospaced font. Finalmente, você pode definir o espaço a ser deixado entre cada letra ao escrever, e este pode ser qualquer valor inteiro, sendo que 0 não tem espaço (as letras "tocarão" se forem proporcionais). A função retorna um valor de índice que deve ser armazenado em uma variável, pois isso será necessário em todos os outros códigos que se referem a este font.

Se a função falhar, então ela falhará silenciosamente sem substituir o dado sprite font.

NOTA Esta função requer que você tenha adicionado previamente um sprite font usando font_add_sprite()pois irá substituir isso e não pode ser usado em fonts que estão incluídos como parte dos recursos do jogo.

Sintaxe:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Argumento Tipo Descrição
font Fonte Ativo The font to replace.
spr Ativo Sprite The sprite to add a font based on.
string_map Cordão String from which sprite sub-image order is taken.
prop Booleano Set as proportional font or not.
sep Real The space to leave between each letter.

Devoluções:

N/A

Exemplo:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

O código acima substituirá um sprite font asset que está indexado na variável "global.Font" por imagens sprite indexadas na variável "spr_CalcFont".
