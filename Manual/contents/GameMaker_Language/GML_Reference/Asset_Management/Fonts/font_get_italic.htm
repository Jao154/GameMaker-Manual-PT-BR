font_get_italic

Com esta função você pode verificar qualquer font asset para ver se ele tem ou não o itálico flag. Se ela fizer a função retornará true, caso contrário retornará false.

Sintaxe:

font_get_italic(ind);

ArgumentoTipo Descrição
ind Index of the font to check.

Devoluções:

Exemplo:

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Isto irá definir o desenho ativo font para fnt_Main se for definido como itálico em suas propriedades font.
