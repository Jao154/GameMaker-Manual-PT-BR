font_get_size_size

Com esta função você pode obter o tamanho de qualquer recurso font, que é o valor em pontos mostrado pelo diálogo de recursos font.

Sintaxe:

font_get_size(ind);

ArgumentoTipo Descrição
ind Index of the font to get the size of.

Devoluções:

Exemplo:

sz = font_get_size(font0);

Isto terá o tamanho do font indexado pela variável "font0" e o armazenará na variável "sz".
