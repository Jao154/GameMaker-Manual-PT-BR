font_replace_sprite

Com esta função, você pode usar uma "faixa sprite" para substituir uma sprite font criada anteriormente asset, onde cada subimagem seria um símbolo ou letra individual. O índice é o número do mapa UTF8 para o primeiro caractere a ser usado e as faixas a serem usadas são as mesmas que você usaria para um normal font, começando com "!", mas lembre-se que você só deve incluir o que você precisa e que você deve estar usando o ord() para obter o valor UTF8 correto para a letra inicial.

Você também pode definir o espaçamento para que o font seja proporcional (true) ou não (false), onde um font proporcional é espaçado com base na largura real das letras (assim a letra "i" leva menos room do que a letra "w", por exemplo), enquanto um font não proporcional será espaçado com base na largura da subimagem, como um monospaced font. Finalmente, você pode definir o espaço a ser deixado entre cada letra ao escrever, e este pode ser qualquer valor inteiro, sendo que 0 não tem espaço (as letras "tocarão" se forem proporcionais).

Se a função falhar, então ela falhará silenciosamente sem substituir o dado sprite font.

t também vale a pena notar que não pode haver espaços no mapa de caracteres, de modo que se você, por exemplo, usar '.' e '0' mas não '/', você precisa incluir '/' de qualquer forma ou deixar uma subimagem em branco entre os dois caracteres usados. Sintaxe:

font_replace_sprite(ind, spr, first, prop, sep);

Argumento Tipo Descrição
ind Fonte Ativo The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr Ativo Sprite The sprite to add a font based on.
first Real The index of the first available character.
prop Booleano Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep Real The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

Devoluções:

N/A

Exemplo:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

O código acima substituirá o font asset indexado na variável global "Font" por um novo font asset a partir de um arquivo de imagem externo.
