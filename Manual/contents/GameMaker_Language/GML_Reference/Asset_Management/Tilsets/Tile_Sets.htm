Conjuntos de azulejos

As seguintes funções estão disponíveis quando se trabalha com tile set assets, conforme criado no Navegador de Ativos. Estas funções são apenas para lidar com o próprio asset, e não para trabalhar com mapas tiles ou tile quando eles estão presentes em seu jogo (para isso veja as funções de camada apropriadas na seção sobre Salas).

As funções disponíveis são:

  • tileset_get_name
  • tileset_get_texture
  • tileset_get_uvs
