tileset_get_name

Esta função devolverá o nome como uma seqüência do especificado tile set asset. Este nome é o que foi especificado para o tile set no Asset Browser da janela principal do GameMaker. Note que este é apenas um string e não pode ser usado para fazer referência direta ao tile set - para isso você precisaria do índice do conjunto de azulejos. Você pode, entretanto, usar este string para obter o índice do conjunto de azulejos usando o string retornado junto com a função asset_get_index().

Sintaxe:

tileset_get_name(index);

Argumento Tipo Descrição
index Conjunto de azulejos Ativo The index of the tile set to get the name of.

Devoluções

String

Exemplo:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

O código acima receberá o nome do índice tile set para a camada dada, armazenando o retorno string na variável tileset_name.
