tileset_get_uvs

Esta função retorna uma matriz com as coordenadas UV e outros dados para o texture do dado tile set na página texture. A função retorna um array com os 8 elementos a seguir:

  • [0] = esquerda
  • [1] = topo
  • [2] = direito
  • [3] = fundo
  • [4] = quantidade de pixels que o compilador asset aparou a partir do lado esquerdo tile set
  • [5] = quantidade de pixels que o compilador asset aparou do lado de cima do tile set
  • [6] = porcentagem normalizada de dados de pixels da largura original tile set que foi salva na página texture
  • [7] = porcentagem normalizada de dados de pixels da altura original tile set que foi salva na página texture

Esta matriz pode então ser usada em outras funções de desenho, particularmente quando se trabalha em 3D ou usando as funções primitivas 2D, bem como quando se trabalha com as funções Shader.

Sintaxe:

tileset_get_uvs(tileset);

ArgumentoTipo Descrição
tileset The index of the tile set to use.

Devoluções

(1D, 8 elements)

Exemplo:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

O código acima armazenará as coordenadas UV para o dado tile set em uma matriz local e então atribuirá os valores às variáveis de instância.
