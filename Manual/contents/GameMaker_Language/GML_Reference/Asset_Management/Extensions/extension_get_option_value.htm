extensão_get_opção_valor_valor

Esta função retorna o valor da opção dada no extension com o nome dado.

A função retornará indefinida se o nome extension fornecido for inválido.

Sintaxe:

extension_get_option_value(extension_name, option_name);

Argumento Tipo Descrição
extension_name Cordão The name of your extension asset as a string
option_name Cordão The name of the option to read

Devoluções:

Any

Exemplo:

var _enabled = extension_get_option_value("MyExtension", "enabled");

Este código recebe o valor da opção ativada no MyExtension extension.
