extensão_get_options

Esta função retorna um struct contendo todas as opções, e seus valores, para o extension com o nome dado.

A função retornará indefinida se o nome extension fornecido for inválido.

Sintaxe:

extension_get_options(extension_name);

Argumento Tipo Descrição
extension_name Cordão The name of your extension asset as a string

Devoluções:

Struct

Exemplo:

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

Este código recebe as opções struct para um extension e lê um valor a partir dele.
