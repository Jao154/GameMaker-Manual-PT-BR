linha do tempo_executada

Esta variável mantém o estado atual da linha de tempo designada e retornará true se estiver em funcionamento e falsa se não estiver. Você também pode definir esta variável para true ou false para iniciar e parar a linha de tempo a qualquer momento. deve-se observar que uma linha de tempo parada não é reinicializada e, portanto, iniciá-la novamente em um momento posterior irá iniciá-la a partir do momento exato em que foi parada.

Sintaxe:

timeline_running;

Devoluções:

Exemplo:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

O código acima verificará se a instância está executando uma linha de tempo, e se não estiver, então ela repõe a linha de tempo atribuída para começar no primeiro momento e depois a inicia.
