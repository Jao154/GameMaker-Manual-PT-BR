linha do tempo_existe

Com esta função você pode verificar e ver se existe uma linha de tempo (retorna true) ou não (retorna false). Isto é particularmente útil ao criar Timelines dinamicamente usando o timeline_add() mas você deve observar que se você procurar por uma variável não inicializada (que de outra forma seria atribuída a um índice de uma linha de tempo), um erro será lançado.

Sintaxe:

timeline_exists(ind);

ArgumentoTipo Descrição
ind The index of the time line to check for.

Devoluções:

Exemplo:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

O código acima verifica se existe uma linha de tempo e se ela está indexada na variável global "tl" e, em caso afirmativo, apaga essa linha de tempo.
