cronograma_nome_get_name

Esta função pode ser usada para obter o nome de uma linha de tempo como um string. se a linha de tempo tiver sido criada dinamicamente usando o timeline_add() função, o nome devolvido terá o formato "__newtimelineN" onde "N" é o número da linha do tempo (a partir de 0). Observe que este é apenas um string e não pode ser usado para fazer referência direta ao timeline - para que você precisaria do índice da linha do tempo. Você pode, entretanto, usar este string para obter o índice da linha do tempo usando o retorno string junto com a função asset_get_index().

Sintaxe:

timeline_get_name(ind);

ArgumentoTipo Descrição
ind The index of the time line to check the name of.

Devoluções:

Exemplo:

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

O código acima recebe o nome do código atualmente atribuído timeline e o envia para o console.
