posição_da_linha do tempo

Esta variável mantém a posição atual (momento) em que uma linha de tempo está atualmente. Você pode alterar este valor para pular partes da linha de tempo, ou para repetir partes ou para iniciar a linha de tempo novamente desde o início.

Sintaxe:

timeline_position;

Devoluções:

(single precision floating point value)

Exemplo:

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

O código acima verificará se a instância está executando uma linha de tempo, e se não estiver, então ela repõe a linha de tempo atribuída para começar no primeiro momento e depois a inicia.
