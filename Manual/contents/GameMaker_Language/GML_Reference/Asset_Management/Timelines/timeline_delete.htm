linha do tempo_delete

Com esta função você pode apagar qualquer linha de tempo de seu jogo. Se esta linha de tempo foi criada no GameMaker Asset Browser, note que ela é removida completamente do jogo e tentar chamar a linha de tempo indexada novamente após usar esta função causará um erro. Se a linha de tempo foi criada dinamicamente usando a função timeline_add() Esta função deve ser usada no momento em que a linha de tempo não for mais necessária para evitar qualquer vazamento de memória que possa retardar (e eventualmente travar) seu jogo.

Sintaxe:

timeline_delete(ind);

ArgumentoTipo Descrição
ind The index of the time line to delete.

Devoluções:

Exemplo:

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

O código acima verifica se existe uma linha de tempo e se ela está indexada na variável global "tl" e, em caso afirmativo, apaga essa linha de tempo.
