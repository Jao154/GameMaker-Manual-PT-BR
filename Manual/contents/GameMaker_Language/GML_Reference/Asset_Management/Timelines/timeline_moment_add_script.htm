timeline_moment_add_script

Com esta função você pode adicionar dinamicamente uma função de script a Timelines em qualquer "momento" dentro daquela linha de tempo, onde um "momento" é o equivalente a um tique (ou passo) de jogo. Desta forma, você pode criar uma nova linha de tempo usando a função timeline_add() e adicionar diferentes comportamentos em qualquer ponto, ou simplesmente modificar um recurso de linha de tempo previamente criado com novos comportamentos. Note que a função não pode exigir nenhum argumento adicional ao usar esta função, e se você usá-la para modificar um Timeline asset presente no navegador Asset, então todas as instâncias que usam este timeline serão afetadas, e a mudança durará até o final do jogo (chamando game_restart() também não irá redefinir a mudança).

Sintaxe:

timeline_moment_add_script(ind, step, script);

Argumento Tipo Descrição
ind Linha do tempo Ativo The index of the timeline to add a moment to.
step Real The moment (step) to add to.
script Função do Script The index of the script function to add into the moment.

Devoluções:

N/A

Exemplo:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

O código acima criará uma nova linha de tempo e armazenará seu índice na variável "global.tl". Ele então adicionará três scripts à linha de tempo, escolhida aleatoriamente, em intervalos de um minuto.
