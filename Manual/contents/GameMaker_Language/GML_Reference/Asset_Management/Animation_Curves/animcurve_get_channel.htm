animcurve_get_channel

Esta função retornará a estrutura contendo os dados do canal especificado em animation curve asset. Você fornece o ID animation curve ou struct, bem como o nome do canal ou índice, e a função retornará um struct com o seguinte formato:

Nome da variável Tipo de dados Descrição
name corda Este é o nome do canal.
type Interpolação da Curva de Animação Tipo Constante Esta será uma das constantes animcurvetype_linear para interpolação linear entre pontos, animcurvetype_catmullrom para interpolação "suave" entre os pontos usando interpolação catmull-rom, ou animcurvetype_bezier para interpolação Bezier.
iterations inteiro Se o canal estiver usando interpolação catmull-rom ("suave"), isso detém quantos pontos foram gerados para cada segmento da curva (observe que esses pontos extras são internos à função e usados apenas para os cálculos runtime ). Se o canal estiver usando interpolação linear, este valor ainda existirá, mas pode ser ignorado, pois não tem influência sobre como a curva é interpolada.
points ponteiro da matriz Esta é uma matriz, onde cada item da matriz é um ponto struct.


O ID animation curve é o nome do animation curve como foi definido no Browser Asset, ou o ponteiro struct como retornado pela função animcurve_create(). O nome do canal é um string que se refere ao canal como foi definido na Curva de Animação asset, ou você pode fornecer um valor de índice, que é de 0 a n, onde n é o último canal na curva asset (por exemplo: se um animation curve tem 4 canais, eles serão indexados de 0 a 3). Note que passar um valor de índice exigirá menos processamento do que passar no nome de um canal. Se a função falhar (ou seja: não existe nenhum canal com o nome dado ou índice), então você receberá um erro.

Os pontos em um único canal são armazenados como structs em uma matriz, onde cada item da matriz é um único ponto struct. O ponto struct tem as seguintes variáveis:

Nome da variável Tipo de dados Descrição
posx real A posição no tempo (normalizada de 0 a 1) do ponto.
value real O valor do ponto.

Sintaxe:

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

Argumento Tipo Descrição
curve_struct_or_id Estruturas Curvas de Animação The ID or struct pointer of the animation curve to target
channel_name_or_index Cordão ou Real The channel name (a string) or the channel index (an integer).

Devoluções:

Animation Curve Channel Struct

Exemplo:

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

O código acima recupera o canal struct para o canal 0 na curva asset "ac_ButtonTween", depois loops através dos pontos na curva do canal e adiciona um ao seu valor.
