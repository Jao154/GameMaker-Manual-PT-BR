animcurve_existe

Esta função voltará a ser verdadeira se o dado Curva de Animação Ativo ou Estruturas Curvas de Animação existe e é uma Curva de Animação válida, ou falsa, se não for.

Sintaxe:

animcurve_exists(curve_struct_or_id);

Argumento Tipo Descrição
curve_struct_or_id Estruturas Curvas de Animação ou Curva de Animação Ativo The Animation Curve that will be checked.

Devoluções:

Boolean

Exemplo:

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

O código acima verifica se a Curva de Animação armazenada na variável personalizada Spring_curve existe. Se existir, ele executa um método personalizado para usar essa Curva de Animação.
