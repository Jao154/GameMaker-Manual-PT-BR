animcurve_get

Esta função devolverá uma estrutura contendo todos os dados para o dado animation curve. Você fornece o ID animation curve asset (conforme definido no Browser Asset ), e a função retornará um struct com as seguintes variáveis:

Estruturas Curvas de Animação
Nome da variável Tipo de dados Descrição
name Cordão Este é o nome do site animation curve.
channels Array de Curva de Animação Estruturas de Canals Esta é uma matriz, onde cada item da matriz é um canal struct.


A variável channels no struct é um array onde cada item do array é um Curva de Animação Estruturas de Canal com dados relativos a um canal dentro da curva. O canal struct é explicado nesta página.

Assim como nos canais, os pontos em um único canal são armazenados como structs em uma matriz, onde cada item da matriz é um único ponto struct. O ponto struct é explicado nesta página.

Observe que se a função falhar (por exemplo, a Curva de Animação dada asset não existe), então a função retornará -1.

Sintaxe:

animcurve_get(curve_id);

Argumento Tipo Descrição
curve_id Curva de Animação Ativo The asset browser ID (index) of the animation curve to get.

Devoluções:

Animation Curve Struct or -1

Exemplo:

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if (_channel.type != animcurvetype_linear)
{
    _channel.type = animcurvetype_linear;
}

O código acima recupera o animation curve struct para a curva asset "ac_ButtonTween", depois verifica o tipo de curva para o primeiro canal, e se não for linear, define-o como linear.
