animcurve_point_new

Esta função pode ser usada para criar uma nova estrutura de pontos a ser adicionada a um canal animation curve. Ao criar um ponto struct está vazio e você precisa definir as seguintes variáveis para gerar os dados do ponto:

Estruturas de pontos curvos de animação
Nome da variável Tipo de dados Descrição
posx real A posição no tempo (normalizada de 0 a 1) do ponto.
value real O valor do ponto.

Cada ponto struct que você criar deve ser adicionado a um array e este é o array que é adicionado ao canal animation curve struct "canais" variável (como mostrado no exemplo abaixo).

Sintaxe:

animcurve_point_new();

Devoluções:

Animation Curve Point Struct

Exemplo:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

O código acima cria um novo animation curve struct e o armazena na variável "my_curve". Este struct é então preenchido com um nome e uma matriz de canais. A matriz de canais contém um único canal com três pontos.
