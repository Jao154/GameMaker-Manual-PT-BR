A estrutura do Keyframe

Cada pista (conforme definido na página que cobre as estruturas da pista) terá uma ou mais estruturas keyframe atribuídas a ela (que você pode obter na propriedade keyframes da pista struct), e cada uma delas terá as seguintes propriedades:

Sequência Keyframe Estruturas
Variável Tipo Descrição
frame Real A posição (em quadros) ao longo do timeline para o keyframe. O valor padrão é 0.
length Real O comprimento do keyframe. O valor padrão é 1, e quando definido para valores maiores, então a propriedade da pista a que se refere o keyframe será mantida no valor inicial durante o período de duração dado. Note que a propriedade stretch irá sobrepor-se a isso se estiver configurada para true.
stretch Booleano Se esta propriedade estiver definida para true então o keyframe se estende para o próximo keyframe para a pista ou para o final da pista se for o último keyframe. Você pode obter ou definir este valor, e o valor padrão é false.
channels Array de Sequência Keyframe Estruturas de dadoss Esta propriedade permite o acesso à lista de estruturas de dados do quadro-chave para os canais da pista. Ao obter esta propriedade, um array de keyframe data structs é retornado, e ao definir esta propriedade, um array de keyframe data structs deve ser especificado.
