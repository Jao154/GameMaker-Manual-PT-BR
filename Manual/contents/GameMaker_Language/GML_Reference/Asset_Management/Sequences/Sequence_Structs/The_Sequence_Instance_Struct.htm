A estrutura da instância Sequence

Um sequence, quando colocado em um room, é colocado como um elemento sobre a camada, e este elemento contém uma instância de seqüência. Esta instância sequence terá as seguintes propriedades em sua estrutura:

Estruturas de instâncias seqüenciais
Variável Tipo Descrição
sequence Sequência Object Estruturas Esta é a identificação do objeto de seqüência struct do qual a instância sequence foi criada (veja mais detalhes aqui). Você pode obter ou definir este struct, e assim alterar a base sequence que a instância está referenciando.
headPosition Real Esta é a atual posição de cabeça de jogo (em quadros) para a instância sequence. Você pode obter ou definir este valor, mas note que defini-lo para um valor maior ou menor que a região de reprodução atualmente definida terá efeitos diferentes, dependendo do tipo de reprodução definida para o sequence. Veja layer_sequence_headpos() para mais informações.
headDirection Constante de direção sequencial Esta é a direção atual de reprodução para a instância sequence. Você pode obter ou definir este valor usando as constantes dadas na tabela seguinte (Note que você pode usar as funções de camada layer_sequence_get_headdir() e layer_sequence_headdir() para obter ou definir este valor sem ter que acessar o struct diretamente).
speedScale Real Esta propriedade pode ser usada para obter ou definir a escala de velocidade de reprodução. A escala de velocidade é um multiplicador, onde 1 é a velocidade de reprodução padrão e valores inferiores a 1 irão diminuir a velocidade de reprodução e valores superiores a 1 irão acelerá-la, por exemplo: um valor de 0,5 seria metade da velocidade de reprodução, enquanto um valor de 2 seria o dobro da velocidade de reprodução. Observe que você pode usar as funções de camada layer_sequence_get_speedscale() e layer_sequence_speedscale() para obter ou definir este valor sem ter que acessar o struct diretamente.
paused Booleano Você pode verificar esta propriedade para ver se um sequence foi pausado ou não, e será true se foi, ou false caso contrário. Esta é uma propriedade somente de leitura, mas você pode usar a função de camada layer_sequence_pause() para pausar a reprodução, e layer_sequence_play() para retomá-la novamente, se necessário. Você também pode verificar esta propriedade usando a função de camada layer_sequence_is_paused() em vez de verificar a propriedade no struct diretamente.
finished Booleano Você pode verificar esta propriedade para ver se um sequence terminou ou não de jogar, retornando true se terminou de jogar, e false caso contrário. Esta é uma propriedade somente para leitura, e só retornará true para faixas que não estejam definidas para loop ou ping-pong. Você também pode verificar esta propriedade usando a função de camada layer_sequence_is_finished() em vez de verificar o imóvel no struct diretamente.
elementID Identificação do Elemento de Sequência Esta propriedade possui a identificação do elemento sequence. Este ID é um valor de identificação simples que está associado a uma camada no editor room, e pode então ser usado com as funções de camada apropriadas para encontrar a camada à qual o sequence foi atribuído ou para alterar certas propriedades sequence sem ter que lidar com o struct.
activeTracks Array de Seqüência de estruturas ativas de vias Esta propriedade terá um conjunto de "avaliação" structs contendo informações sobre o estado atual de cada faixa de ativo no sequence (gráficos, sequence, áudio etc...). Estas são as faixas de nível superior, e a matriz retornada é somente leitura. Para o conteúdo da faixa asset struct retornada em cada entrada da matriz, por favor, veja a seção abaixo.

Constante de direção sequencial
Constante Descrição Valor
seqdir_right O sequence jogará quadros em uma ordem incremental da esquerda para a direita 1
seqdir_left O sequence jogará quadros em uma ordem decrescente da direita para a esquerda -1

Estrutura ActiveTracks

Embora esta propriedade já exista na instância sequence struct no evento Create Sequence, ela só é preenchida com faixas após o primeiro evento Sequence Begin Step. Portanto, a primeira vez que você encontrará valores neste struct é durante a primeira iteração do evento Sequence Step Event.

A lista a seguir contém as propriedades que podem estar disponíveis para você ao acessar uma faixa ativa struct (como incluído na matriz activeTracks de um Sequence Instance struct, explicado acima). Cada faixa asset struct pode ter qualquer uma das seguintes propriedades, dependendo do tipo de asset que a faixa usa (note que você pode modificar todas estas propriedades, exceto onde especificado como somente leitura), e todos os valores retornados são para a posição atual de playhead:

Seqüência de estruturas ativas de via
Variável Tipo Descrição
matrix Matriz de matriz Esta é a matriz de transformação utilizada para desenhar a pista (sprite, exemplo e sequence pistas).
posx Real A posição do asset no sequence ao longo do eixo X para a pista na posição atual de playhead (todos assets).
posy Real A posição do asset no sequence ao longo do eixo Y para a pista na posição atual de playhead (todos assets).
scalex Real A escala do asset no sequence ao longo do eixo X para a pista na posição atual de playhead (sprite, exemplo e sequence pistas).
scaley Real A escala do asset no sequence ao longo do eixo Y para a faixa na posição atual de playhead (sprite, exemplo e sequence faixas).
xorigin Real A origem X do asset para a pista (todos assets).
yorigin Real A origem em Y do asset para a pista (todos assets).
width Real A largura (em pixels) do asset para a pista (sprite, exemplo e sequence pistas).
height Real A altura (em pixels) do asset para a pista (sprite, exemplo e sequence pistas).
imageindex Real O índice de imagem para o asset na pista no sequence na posição atual de playhead (sprite e pistas de instância).
imagespeed Real A velocidade da imagem para o asset na pista no sequence na posição atual de playhead (sprite e pistas de instância).
colourMultiply Cor O valor da multiplicação de cores para o asset na pista no sequence na posição atual de playhead (sprite, exemplo e sequence pistas). Este valor será um conjunto de quatro valores ARGB com o formato [A, R, G, B]. Observe que os valores para cada componente são expressos entre 0 e 1, onde 0 corresponde ao valor HEX #00 e 1 corresponde ao valor HEX #FF (0 - 255 como mostrado no seletor de cores para faixas de multiplicação de cores no Editor Sequences ).
emitterIndex ID do emissor de áudio O índice do emissor de áudio pertencente a esta faixa (faixas de áudio).
gain Real O ganho (volume) da faixa na posição atual de playhead (faixas de áudio).
pitch Real O tom da faixa na posição atual de playhead (faixas de áudio).
frameSizeX Real O tamanho horizontal do quadro de texto (faixas de texto).
frameSizeY Real O tamanho vertical do quadro de texto (faixas de texto).
characterSpacing Real O valor do espaçamento de caracteres (faixas de texto).
lineSpacing Real O valor do espaçamento entre linhas (faixas de texto).
paragraphSpacing Real O valor do espaçamento de parágrafos (faixas de texto).
track Estruturas de trilhos seqüenciais A estrutura de trilhos em que esta trilha se baseia (todos assets, um valor só de leitura ).
parent Estruturas de instâncias seqüenciais A ID de instância sequence da pista (todos assets, um valor só de leitura ).
activeTracks Array de Estruturas de trilhos seqüenciaiss Esta é uma série de avaliações structs para cada faixa de parâmetros que a faixa asset contém (todas assets, um valor somente leitura ). O conteúdo de cada struct na matriz está listado na seção Estrutura da trilha do parâmetro.
