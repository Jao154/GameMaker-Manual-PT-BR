A estrutura de dados do Keyframe

Quando você define uma estrutura de quadro-chave para uma pista, você também precisa definir os dados que serão associados a ela. Isto é composto de canais diferentes, onde cada canal é simplesmente um Keyframe Data Struct. Um canal pode receber um tipo específico de dados, dependendo do tipo de faixa para a qual você está definindo os dados keyframe.

A Keyframe Data Struct contém as seguintes variáveis:

Sequência Keyframe Estruturas de dados
Variável Tipo Descrição
channel Real Este é o canal a que os dados keyframe devem ser aplicados. É um valor inteiro positivo começando em 0, e vale a pena notar que ao criar faixas de parâmetros para "posição" ou "escala" keyframes, então você precisa usar valores de canal muito específicos. Estes são: canal 0 é a posição X ou a escala X, canal 1 é a posição Y ou a escala Y.
spriteIndex Ativo Sprite O índice do sprite asset a ser utilizado para a pista. Esta propriedade está disponível apenas para pistas do tipo seqtracktype_graphic e você pode obtê-lo ou configurá-lo.
soundIndex Bens sólidos O índice do áudio asset a ser utilizado para a faixa. Esta propriedade está disponível apenas para faixas do tipo seqtracktype_audio e você pode obtê-lo ou configurá-lo.
particleSystemIndex Particle System Asset The particle system asset used for the track. This property is only available for tracks of the type seqtracktype_particlesystem and you can get or set it.
playbackMode Chave de seqüência de áudio constante O modo de reprodução para o som. Esta propriedade está disponível apenas para faixas do tipo seqtracktype_audio e você pode obtê-la ou configurá-la. A propriedade deve ser uma das constantes dadas na tabela abaixo desta.
curve Estruturas Curvas de Animação Esta propriedade requer um animation curve struct (veja aqui para mais informações) e só está disponível para faixas do tipo seqtracktype_real. Se nenhuma curva struct for utilizada, então o valor para esta propriedade será -1.
value Real Esta propriedade é simplesmente um valor associado ao canal de dados keyframe, e só está disponível para faixas do tipo seqtracktype_real quando não é fornecido curve struct. Esta pode ser, por exemplo, a posição X ou Y da pista se colocada dentro de uma pista com parâmetros de "posição".
colour Array de Reals Esta propriedade retorna (ou requer, se for definida) um array para o valor da cor do keyframe com o formato [A, R, G, B]. Isto só está disponível para faixas do tipo seqtracktype_colour. Observe que os valores para cada componente devem ser expressos entre 0 e 1, onde 0 corresponde ao valor HEX #00 e 1 corresponde ao valor HEX #FF (0 - 255 como mostrado no seletor de cores para trilhas coloridas no Editor Sequences ).
sequence Sequência Object Estruturas Esta propriedade retornará (ou exigirá, se for definida) uma estrutura object e só está disponível para faixas do tipo seqtracktype_sequence.
objectIndex Objeto Ativo Esta propriedade retornará (ou exigirá, se for definida) um índice object e só está disponível para faixas do tipo seqtracktype_instance.
events Array de Cordãos Esta propriedade permite o acesso aos eventos e mensagens transmitidas associadas aos dados keyframe struct . Você pode obter ou definir esta propriedade, e ao obtê-la é devolvido um array de strings, e ao defini-la deve ser especificado um array de strings. Para mais informações sobre eventos, consulte a seção Eventos e Momentos Sequenciais. Esta propriedade está disponível apenas para faixas do tipo seqtracktype_message.
event Método Esta propriedade retornará (ou pode ser definida para) o method associado com os dados keyframe struct . Se nenhum método foi especificado ou se você deseja remover o método, então a propriedade deve ser -1. Esta propriedade está disponível apenas para faixas do tipo seqtracktype_moment.
Qualquer variável adicional listada na(s) tabela(s) abaixo

A variável playbackMode pode ser uma das constantes a seguir:

Chave de seqüência de áudio constante
Constante Descrição Valor
seqaudiokey_loop O som será loop quando tocado. 0
seqaudiokey_oneshot O som só tocará uma vez e depois parará. 1

Dados de rastreamento de texto

O struct conterá as seguintes variáveis adicionais, se atribuídas a uma faixa de texto (seqtracktype_text):

Sequência Keyframe Estruturas de dados
Variável Tipo Descrição
text Cordão Este é o texto string que é desenhado na pista.
wrap Booleano Este é um booleano que indica se o texto deve ser embrulhado (true) ou não (false).
alignmentH Constante de alinhamento de texto sequencial Este é o alinhamento horizontal do texto, e será um dos Constante de alinhamento de texto sequencials dadas abaixo.
alignmentV Constante de alinhamento de texto sequencial Este é o alinhamento vertical do texto, e será um dos Constante de alinhamento de texto sequencials dadas abaixo.
fontIndex Fonte Ativo Este é o Fonte Ativo usado pela faixa de texto.

alignmentH pode ser qualquer uma das seguintes constantes:

Constante de alinhamento de texto sequencial (alignmentH)
Constante Descrição
seqtextkey_left O texto será alinhado à esquerda do quadro
seqtextkey_right O texto será alinhado à direita do quadro
seqtextkey_center O texto será centralizado horizontalmente dentro do quadro
seqtextkey_justify O texto será justificado dentro do quadro

alignmentV pode ser qualquer uma das seguintes constantes:

Constante de alinhamento de texto sequencial (alignmentV)
Constante Descrição
seqtextkey_top O texto será alinhado à parte superior do quadro
seqtextkey_bottom O texto será alinhado ao fundo do quadro
seqtextkey_middle O texto será centralizado verticalmente dentro do quadro
