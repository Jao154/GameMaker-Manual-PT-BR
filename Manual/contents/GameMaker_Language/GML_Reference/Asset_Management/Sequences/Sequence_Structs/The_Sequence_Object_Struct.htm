Sequência Object Estruturas

A Sequência Object Estruturas é o nome dado ao struct recuperado de um Sequence asset. Todos os dados de um Sequence são armazenados neste struct em runtime, e são referenciados por quaisquer elementos do Sequence que são criados em um room. Isto significa que se qualquer variável desta estrutura (ou dentro de qualquer um de seus aninhados structs) for modificada, o original Sequence será alterado (até que o jogo seja fechado e reaberto) e qualquer um de seus elementos Sequence existentes será afetado.

Esta estrutura terá as seguintes propriedades que podem ser alteradas:

Sequência Object Estruturas
Variável Tipo Descrição
name Cordão Este é o nome do sequence como um string e você pode obter ou definir este valor conforme necessário. Note que sequences foi criado usando a função sequence_create() não terá um nome e este será simplesmente um string vazio "".
loopmode Constante do Modo de Jogo de Sequência Este é o modo de reprodução do sequence object e pode ser obtido ou configurado. Esta pode ser qualquer uma das constantes mostradas na tabela abaixo desta.
playbackSpeed Real Isto especifica a velocidade de reprodução do sequence, que é interpretado como frames por segundo ou frames por game frame, dependendo do playbackSpeedType (veja abaixo). Você pode obter ou definir este valor.
playbackSpeedType Constante de velocidade do duende Isto especifica como o playbackSpeed deve ser interpretado e você pode obter ou definir este valor.
length Real O comprimento do sequence em armações. Você pode obter ou definir este valor, mas note que fazer um sequence mais curto pode causar problemas se uma instância sequence referenciando este sequence tiver seu playhead definido para além do novo comprimento.
volume Real Este é um valor escalar de 0 a 1 que é usado para escalar o volume de todas as faixas de áudio no sequence. Você pode obter ou definir este valor e ele modificará a saída de áudio global para todas as faixas - por exemplo, se você tiver uma faixa de áudio com um volume de 0,8 e depois definir a propriedade de volume sequence para 0,5, a faixa de áudio terá um volume final de 0,4.
xorigin Real Esta é a origem do sequence ao longo do eixo X.
yorigin Real Esta é a origem do sequence ao longo do eixo Y.
messageEventKeyframes Array de Sequência Keyframe Estruturass Isto permite o acesso ao evento de mensagem keyframes para o sequence. Você pode obter ou definir estes eventos de mensagem, e ao obter este imóvel é devolvido um array de keyframe structs, e para definir o imóvel você deve fornecer um array de keyframe structs. Para mais informações, consulte a página sobre Eventos e Momentos Seqüenciais.
momentKeyframes Array de Sequência Keyframe Estruturass Isto permite o acesso ao evento de momento keyframes para o sequence. Você pode obter ou definir estes eventos de momento, e ao obter este imóvel um array de keyframe structs é devolvido, e para definir o imóvel você deve fornecer um array de keyframe structs. Para mais informações, consulte a página sobre Eventos e Momentos Seqüenciais.
tracks Array de Estruturas de trilhos seqüenciaiss Isto permite o acesso à lista de faixas de bens no nível superior do sequence. Você pode obter ou definir esta propriedade, e ao obter esta propriedade uma matriz de faixas structs é devolvido, e para definir a propriedade você deve fornecer uma matriz de faixas structs. Para mais informações, consulte a seção sobre Estruturas de trilhos.

loopMode pode ser qualquer uma das seguintes constantes:

Constante do Modo de Jogo de Sequência
Constante Descrição Valor
seqplay_oneshot O sequence tocará uma vez e depois parará quando terminar. 0
seqplay_loop O sequence irá loop, com o playhead voltando ao início quando chegar ao final da região de reprodução. 1
seqplay_pingpong O sequence irá loop, com o playhead invertendo a direção quando chegar ao final da região de reprodução. 2


Observações

Note que se você quiser acessar as propriedades de um sequence que foi criado no navegador asset, você deve primeiro chamar a função sequence_get() no índice asset para recuperar o sequence object struct . Note também que quaisquer mudanças feitas neste sequence struct significarão que todas as outras instâncias deste sequence asset também terão estas mudanças, e elas serão mantidas enquanto o jogo estiver funcionando, mesmo se você chamar a função game_restart().
