sequence_destroy

Com esta função, você pode destruir um sequence object que foi criado dinamicamente. Você fornece ou o sequence object struct (como retornado pela função sequence_create()) ou o ID sequence (como retornado pela função layer_sequence_get_sequence() ou da instância sequence struct propriedade sequence). Esta função deve ser usada sempre que uma sequence criada dinamicamente não for mais necessária para liberar a memória associada a ela.

Sintaxe:

sequence_destroy(sequence_struct_or_id);

Argumento Tipo Descrição
sequence_struct_or_id Sequência Ativo ou Sequência Object Estruturas The sequence object struct or ID to destroy

Devoluções:

N/A

Exemplo:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

O código acima verifica se o dado sequence object existe e se o faz é destruído.
