sequence_instance_override_object

Com esta função, você pode anular (substituir) todas as instâncias de um object usado em um sequence por outro. Você fornece o ID da instância sequence struct (como retornado quando a instância sequence foi criada no room ou usando uma das funções da camada room - veja aqui), bem como o índice object (como definido no navegador asset ) para o object que você deseja sobrepor. Finalmente, você dá um índice object ou um ID de instância para usar como o object que vai sobrepor o sequence (fornecendo um ID de instância simplesmente usará o object do qual a instância foi criada como sobreposição). Note que isto só pode ser feito em instâncias sequence (não sequence objects) e deve ser feito antes que o sequence comece a funcionar, caso contrário não vai funcionar.

Sintaxe:

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

Argumento Tipo Descrição
sequence_instance_struct Estruturas de instâncias seqüenciais The sequence instance struct to modify.
object_id Objeto Ativo The object index of the object within the sequence to override.
instance_or_object_id Objeto Ativo ou ID da instância The object index or instance ID to use to override the sequence objects.

Devoluções:

N/A

Exemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

O código acima cria uma nova instância sequence na camada dada e depois a modifica para que todas as instâncias do object "obj_Trees_Winter" sejam substituídas por instâncias do object "obj_Trees_Summer".
