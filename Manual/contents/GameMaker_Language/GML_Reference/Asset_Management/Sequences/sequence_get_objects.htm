Sequência_get_objects

Com esta função você pode recuperar um conjunto de todos os índices object que têm instâncias sendo criadas dentro do dado sequence. Você pode fornecer ou o sequence object struct (como retornado pela função sequence_create() ou sequence_get()) ou o ID sequence (como retornado pela função layer_sequence_get_sequence() ou da instância sequence struct propriedade sequence) e a função retornará uma matriz, onde cada item da matriz é um object_index para os diferentes objects que estão sendo usados pelo sequence para criar instâncias.

Sintaxe:

sequence_get_objects(sequence_struct_or_id);

Argumento Tipo Descrição
sequence_struct_or_id Sequência Ativo ou Sequência Object Estruturas The sequence object struct or ID to get the objects from.

Devoluções:

Array

Exemplo:

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

O código acima recebe o struct para um sequence object e depois recupera o objects que usa e armazena o array em uma variável.
