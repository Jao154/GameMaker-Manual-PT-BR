room_instance_add

Com esta função você pode adicionar uma instância em qualquer room que não seja a atual e em qualquer posição dentro dessa room. A função retorna o exclusivo id da instância que pode então ser usada em outras chamadas de função para definir propriedades etc... dessa instância, mas somente depois que o jogo tiver entrado na sala especificada. Se você deseja criar uma instância no atual room, você deve estar usando a função instance_create_layer(). Note que chamar esta função em um room asset criado no navegador Asset adicionará permanentemente a instância à sala, e mesmo chamando game_restart() não retornará o room ao seu estado original (somente terminando o jogo e abrindo-o novamente começará com o room em seu estado original novamente).

Sintaxe:

room_instance_add(index, x, y, obj);

ArgumentoTipo Descrição
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

Devoluções:

Instance ID

Exemplo:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

O código acima adicionará um novo room ao jogo e então copiará o conteúdo do room indexado como "rm_Base" nele. Ele então adicionará uma instância do object "obj_player" na posição 100.100 deste novo room.
