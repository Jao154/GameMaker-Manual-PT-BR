room_exists

Com esta função, você pode verificar e ver se o room que você especifica existe ou não. Esta função toma o índice room (um número real) e não o nome room (um string). Esta função é mais útil quando você está criando rooms dinamicamente usando a função room_add()mas você também pode usar as variáveis somente de leitura room_first e room_last ou as funções room_next() e room_previous() para obter um índice específico room, ou fornecer uma variável que tenha armazenado o índice de qualquer outro room em seu jogo.

Sintaxe:

room_exists(index);

ArgumentoTipo Descrição
index The index of the room to check.

Devoluções:

Exemplo:

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

O código acima verifica se o room indexado na matriz "global.rm[]" existe e se o faz, então vai para room.
