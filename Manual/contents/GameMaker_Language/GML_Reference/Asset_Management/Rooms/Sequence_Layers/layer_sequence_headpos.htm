layer_sequence_headpos

Com esta função você pode definir a posição de cabeça de jogo de um elemento sequence para uma moldura específica. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções dos elementos da camada junto com a nova posição a ser definida. Note que a posição está em quadros e se você definir um valor maior que o número total de quadros (ou menor que 0) então a posição final real do playhead dependerá do tipo de reprodução sequence que foi selecionada, seguindo estas regras:

  • Sem looping: O playhead se fixará ao final do loop (ou ao início, se o valor dado for negativo)
  • Looping: O playhead voltará ao início do sequence mais os quadros extras, por exemplo: se você colocar o playhead em um quadro 30 sequence para uma posição de quadro 45, a posição real do playhead acabará sendo 15. Se você der um valor negativo, então o playhead sequence será embrulhado na direção inversa.
  • Laço de ping-pong: O playhead avançará até o final e então voltará o número extra de quadros que estão sobre o comprimento total do quadro, invertendo a direção da reprodução, por exemplo: Se o sequence for 30 quadros e você definir o playhead para 40, a posição final do playhead será 20 e a direção será invertida (descendo para 0). Observe que você nunca deve dar um valor negativo para a reprodução do ping-pong de um sequence, pois ele pode ter resultados em undefined.

Estabelecer a posição da cabeça desta maneira não interromperá a reprodução e o sequence simplesmente continuará a partir da nova posição de cabeça de reprodução, a menos que seja pausado.

Sintaxe:

layer_sequence_headpos(sequence_element_id, position)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target
position Real The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

Devoluções:

N/A

Exemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

O código acima obtém as identificações de todos os elementos atribuídos à camada da instância que está executando o código. O código então verifica se algum dos elementos retornados é sequence assets e se a posição de cabeça de jogo deles está definida como 0.
