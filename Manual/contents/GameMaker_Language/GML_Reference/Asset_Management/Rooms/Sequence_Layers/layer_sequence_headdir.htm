layer_sequence_headdir

Com esta função, você pode definir a direção para o dado sequence playhead. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções do elemento camada, e então dar a direção do jogo, que deve ser uma das constantes seguintes:

Constante Descrição Valor
seq_dir_right O sequence jogará quadros em uma ordem incremental da esquerda para a direita 1
seq_dir_left O sequence jogará quadros em uma ordem decrescente da direita para a esquerda -1

Sintaxe:

layer_sequence_headdir(sequence_element_id, direction)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target
direction Constante de direção sequencial The playhead direction, a constant, listed above

Devoluções:

N/A

Exemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

O código acima cria um novo sequence e armazena sua identificação em uma variável local. Esta identificação é então usada para definir a direção da cabeça de reprodução para diminuir os quadros (da direita para a esquerda).
