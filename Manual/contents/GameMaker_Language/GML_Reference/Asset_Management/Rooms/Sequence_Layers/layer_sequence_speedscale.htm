escala_de_série_de_camadas

Com esta função você pode alterar a velocidade de reprodução do dado sequence. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções dos elementos da camada e depois um valor de escala de velocidade. Este valor é um multiplicador, onde 1 é a velocidade de reprodução padrão e valores inferiores a 1 irão diminuir a velocidade de reprodução e valores superiores a 1 irão acelerá-la, por exemplo: um valor de 0,5 seria metade da velocidade de reprodução, enquanto um valor de 2 seria o dobro da velocidade de reprodução.

Sintaxe:

layer_sequence_speedscale(sequence_element_id, speedscale)

Argumento Tipo Descrição
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target
speedscale Real The speed scale to use (a multiplier), default is 1

Devoluções:

N/A

Exemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

O código acima cria um novo sequence na camada da instância de chamada e então define seu valor de escala de velocidade para 0,75 (três quartos de velocidade de reprodução padrão).
