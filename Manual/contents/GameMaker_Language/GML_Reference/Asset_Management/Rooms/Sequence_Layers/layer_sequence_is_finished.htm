layer_sequence_is_is_finished

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e verificará se o sequence terminou ou não de tocar, retornando true se for, e false se não for. Note que isto só é aplicável quando o sequence não está configurado para loop ou ping-pong no modo de reprodução.

Sintaxe:

layer_sequence_is_finished(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Exemplo:

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

O código acima verifica se o ID do elemento sequence armazenado na variável "title_sequence" está finalizado, e se é ele que começa a tocar novamente e estabelece um alarme.
