layer_sequence_get_length

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e retornará o comprimento do sequence. Este é o número de quadros que o sequence irá executar.

Sintaxe:

layer_sequence_get_length(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Exemplo:

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

O código acima recupera o número de quadros que um sequence irá executar, então usa este valor para definir um alarme.
