Sequência de camadas

Nesta seção temos as funções que são usadas para adicionar, remover e editar como as Sequências se comportam em um jogo room em relação à camada em que se encontram. Deve-se notar que sequences é semelhante a objects, na medida em que quando se cria em uma camada em room, é considerado uma instância do objeto principal sequence do Navegador de Ativos. Assim, quando se adiciona um sequence a uma camada em um room, está-se criando uma instância da base sequence object, e esta instância sequence é adicionada como um elemento dentro de uma camada room. A instância sequence é o que controla coisas como velocidade e direção de reprodução, e ela mesma conterá outra estrutura de dados sequence, que é o que contém os dados reais da pista sequence. Isto é importante, pois as funções listadas abaixo farão referência ao ID do elemento sequence (o ID do sequence na camada) assim como o ID da instância sequence (o ID real do sequence que está sendo referenciado pelo elemento) e a estrutura de dados sequence (que contém todos os dados sequence ).
