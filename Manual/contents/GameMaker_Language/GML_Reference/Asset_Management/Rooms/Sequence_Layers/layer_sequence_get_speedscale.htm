escala_de_série_de_série_de_camadas

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos de camada - e retornará a escala de velocidade de reprodução atual. Este é o valor multiplicador usado para desacelerar ou acelerar a velocidade de reprodução. Um valor de 1 é o valor padrão, e valores inferiores a 1 significam que a reprodução é diminuída e valores superiores a 1 significam que a reprodução é acelerada.

Sintaxe:

layer_sequence_get_speedscale(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Exemplo:

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

O código acima verifica a escala de velocidade atual da cabeça de jogo do elemento ID sequence armazenado na variável "title_sequence", e se não estiver definido para 1, está definido para este valor.
