camada_background_visível

Usando esta função, você pode alternar a visibilidade dos elementos de fundo. Você fornece a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e então definir o argumento visível para true ou false. Quando ajustado para true o elemento será exibido, e quando ajustado para false, o elemento não será exibido. Note que isto depende da visibilidade da camada, e mesmo que o elemento de fundo esteja marcado como visível, ele não será desenhado se a camada em que estiver marcado como não visível.

Sintaxe:

layer_background_visible(background_element_id, visible)

Argumento Tipo Descrição
background_element_id Identificação do elemento de fundo The unique ID value of the background element to change
visible Booleano The visibility toggle, which can be true ou false

Devoluções:

N/A

Exemplo:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_visible(back))
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

O código acima verifica se há um botão do mouse pressionado e, se for detectado, alternará a visibilidade de fundo do elemento de fundo atribuído à camada em que a instância que está executando o código está.
