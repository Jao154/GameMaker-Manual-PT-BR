fundo_de_camada_esticar

Usando esta função, você pode alternar um elemento de fundo sprite para esticar para caber no room ou permanecer a 1:1 com a resolução. Você fornece o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e então definir o argumento de esticar para true ou false. Quando ajustado para true o elemento sprite será esticado para caber no room (ou escalado para cima ou para baixo dependendo dos tamanhos do sprite e do room), e quando ajustado para false, o elemento sprite será exibido em sua resolução nativa.

Sintaxe:

layer_background_stretch(background_element_id, stretch)

Argumento Tipo Descrição
background_element_id Identificação do elemento de fundo The unique ID value of the background element to change
stretch Booleano The stretch toggle, which can be true ou false

Devoluções:

N/A

Exemplo:

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

O código acima verifica se há um botão do mouse pressionado e, se for detectado, alternará o alongamento do elemento de fundo sprite atribuído à camada em que a instância que está executando o código está.
