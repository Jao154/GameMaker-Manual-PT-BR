fundo_camada_cortado

Usando esta função, você pode mudar se o elemento de fundo dado em uma camada deve ser ladrilhado horizontalmente ou não. Você dá a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e depois definir o valor do azulejo. Se ajustado para true, então o elemento será exibido em mosaico horizontal no room e quando ajustado para false, o elemento não será mosaico. Esta função é apenas para projetos 2D, e não funcionará corretamente quando uma projeção de câmera 3D for usada.

Sintaxe:

layer_background_htiled(background_element_id, tiled);

Argumento Tipo Descrição
background_element_id Identificação do elemento de fundo The unique ID value of the background element to change
tiled Booleano The horizontal tiling toggle, which can be true ou false

Devoluções:

N/A

Exemplo:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

O código acima verificará o elemento de fundo atribuído à camada em que a instância que está executando o código está e, se não estiver ladrilhada em nenhuma das direções, ela o colocará em tile na vertical e na horizontal.
