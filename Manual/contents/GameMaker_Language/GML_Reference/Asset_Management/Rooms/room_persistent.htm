room_persistent

Esta variável pode ser usada para obter e definir o persistente flag para o atual room. Se definido para true o room é considerado persistente, neste caso cada vez que você deixar o room e voltar novamente o estado das instâncias dentro daquele room terá sido mantido. Entretanto, se for assinalado como false, cada vez que você retornar ao room, ele será reposto ao seu estado inicial. Você deve observar que um persistente room usa consideravelmente mais memória do que um room normal e não é recomendado ter muitos deles em seu jogo.

Sintaxe:

room_persistent;

Devoluções:

Exemplo:

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

O código acima verifica a variável "vidas" e, se for inferior a 1, definirá a persistência de room para false e depois mudará para rooms.
