mapa do azulejo_y

Esta função controla a posição ao longo do eixo y do room do elemento do mapa asset tile na camada. Você dá o ID do elemento de mapa tile (que você obtém quando cria um elemento de mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e então definir o valor y a ser usado (com base nas coordenadas room ).

Sintaxe:

tilemap_y(tilemap_element_id, y);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to change
y Real The y position for the tile map

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

tile O código acima obtém o valor de ID do mapa asset atribuído à camada "Asset_sky" e então define sua posição y para um valor aleatório entre 0 e a altura do room.
