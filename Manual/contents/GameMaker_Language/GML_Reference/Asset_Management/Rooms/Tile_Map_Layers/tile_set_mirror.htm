espelho_de_série

Esta função pode ser usada para definir um determinado conjunto de dados de azulejos para espelhar o tile ou não. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e, em seguida, defina o argumento do espelho para true se você quiser o tile espelhado, ou false se você quiser que o tile esteja em seu estado padrão, sem espelhamento. A função retornará um conjunto modificado de dados de azulejos que pode então ser aplicado usando o tilemap_set() função.

Sintaxe:

tile_set_mirror(tiledata, mirror)

ArgumentoTipo Descrição
tiledata the tile-data to set
mirror Set to true para espelhar e false para deixar as coisas como estão

Devoluções:

Tile Data (Real)

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

O código acima obtém o ID do mapa tile da camada dada e depois obtém a posição das células x e y para o tile sob o mouse. Esta posição é então usada para obter os dados dos azulejos que são espelhados e depois usados para definir novamente o tile.
