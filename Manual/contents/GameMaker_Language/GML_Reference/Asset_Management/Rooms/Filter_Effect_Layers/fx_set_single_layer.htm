fx_set_single_layer

Esta função é usada para ativar ou desativar o modo "Camada única" para um filtro/efeito. Por padrão, um filtro/efeito é aplicado à camada à qual é atribuído e a todas as camadas abaixo dessa camada, porém é possível habilitar o modo "Single Layer" em um FX struct para garantir que ele seja aplicado apenas à camada à qual é atribuído.

Você especifica um FX struct (como retornado de fx_create() ou layer_get_fx()) e um valor booleano para habilitar (true) ou desabilitar (false) modo Camada única.

O seguinte visual mostra um filtro sendo usado com o modo Camada Única desabilitado (que é o comportamento padrão para todas as camadas FX) e o mesmo filtro com o modo Camada Única habilitado:

OBSERVAÇÃO Ao ativar o modo Camada Única para um FX, certifique-se de aplicá-lo em uma camada que não seja uma camada FX, pois uma camada FX por si só não contém nada e, portanto, nenhum filtro/efeito será visível.

Sintaxe:

fx_set_single_layer(filter_or_effect, enable);

ArgumentoTipo Descrição
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) ou desabilitar (false) Modo Camada Única

Devoluções:

Exemplo:

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

O código acima cria um novo screenshake FX, permite o modo Single Layer sobre ele e depois o aplica a uma camada room. Isto significa que o filtro de agitação de tela só será aplicado à camada "ShakeyThings ".
