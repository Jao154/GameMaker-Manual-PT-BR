fx_set_parametros

Esta função é utilizada para alterar os parâmetros de um filtro/efeito. Você especifica um FX struct (como retornado de fx_create() ou layer_get_fx()) e um struct contendo seus parâmetros (como retornado de fx_get_parameters()). Isto tornará suas mudanças visíveis na camada à qual o FX struct é atribuído.

Seu parâmetro struct não é obrigado a conter todos os parâmetros aplicáveis ao filtro/efeito, e somente as variáveis que estão presentes no dado struct serão atualizadas para o filtro/efeito.

Sintaxe:

fx_set_parameters(filter_or_effect, parameter_struct);

ArgumentoTipo Descrição
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

Devoluções:

Exemplo:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

O código acima obtém o FX struct para uma camada que se supõe ter o filtro "Color Tint" aplicado a ele, e recupera seu parâmetro struct chamando fx_get_parameters(). Depois disso, ele cria um valor oscilante usando sin() e current_time, que é então usado para os valores RGBA para o efeito de tonalidade, de modo que ele continua animando. O array RGBA é atribuído à variável g_TintCol no parâmetro struct, e o struct é então aplicado de volta ao FX struct chamando fx_set_parameters().
