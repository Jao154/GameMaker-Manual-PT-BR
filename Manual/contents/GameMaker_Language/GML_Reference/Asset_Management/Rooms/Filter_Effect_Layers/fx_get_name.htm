fx_get_name

Esta função é utilizada para recuperar o nome de um filtro/efeito a partir de seu struct. Você especifica o FX struct para ler (como retornado de fx_create() ou layer_get_fx()) e a função retorna seu nome como um string. Este nome pode então ser passado para fx_create() para criar um novo FX struct usando o mesmo tipo de filtro/efeito.

Sintaxe:

fx_get_name(filter_or_effect);

ArgumentoTipo Descrição
filter_or_effect The FX struct to get the name of

Devoluções:

Exemplo:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

O código acima executa um for percorre todas as camadas presentes no room, e verifica cada camada em busca de um FX struct. Se uma camada tem um FX struct, ele verifica o nome desse filtro/efeito chamando fx_get_name(); se for igual a "_filter_tintfilter" significando que é um filtro "Colour Tint", ele muda sua cor de tonalidade para azul.
