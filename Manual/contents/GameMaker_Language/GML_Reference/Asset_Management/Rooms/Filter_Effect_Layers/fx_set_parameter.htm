fx_set_parameter

Esta função é utilizada para alterar o valor de um parâmetro no FX struct. Você especifica um FX struct (como retornado de fx_create() ou layer_get_fx()), um nome de parâmetro como um string, e seu novo valor. Isto tornará suas mudanças visíveis na camada à qual o FX struct é atribuído.

Você é capaz de passar em múltiplos argumentos para o parâmetro caso ele seja um vetor, porém você também pode simplesmente especificar um array como seu único argumento que conteria seus múltiplos valores.

Encontre os nomes dos parâmetros para um filtro/efeito nesta página: Tipos e parâmetros de FX.

Sintaxe:

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

Argumento Tipo Descrição
filter_or_effect Estruturas FX The FX struct to modify
parameter_name Cordão The name of the parameter to change the value of (as a string)
val... Real ou Array The new value(s) for that parameter as a real or array

Devoluções:

N/A

Exemplo:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

O código acima cria um novo FX struct a partir do efeito "_filter_tintfilter ", que é o filtro "Colour Tint" encontrado no editor Room. Ele atribui um valor a seu parâmetro "g_TintCol " que é a cor da tonalidade, e como é um vec4 internamente, ele toma uma matriz contendo quatro valores (correspondentes a seus valores vermelho, verde, azul e alfa). O FX struct para a tonalidade é então aplicado a uma camada FX.

Observe que você também é capaz de passar valores para o vetor como argumentos separados, em vez de uma matriz:

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
