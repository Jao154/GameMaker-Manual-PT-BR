fx_get_parameter_names

Esta função é utilizada para recuperar os nomes de todos os parâmetros presentes em um filtro/efeito. Você especifica o FX struct para ler (como retornado de fx_create() ou layer_get_fx()) e a função retorna um array contendo os nomes dos parâmetros do filtro como strings.

Sintaxe:

fx_get_parameter_names(filter_or_effect);

ArgumentoTipo Descrição
filter_or_effect The FX struct to get the parameter names of

Devoluções:

(of Strings)

Exemplo:

var _fx_struct = layer_get_fx("EffectLayer");
var _param_names = fx_get_parameter_names(_fx_struct);

for (var i = 0; i < array_length(_param_names); i ++) {
    show_debug_message("Parameter " + string(i) + ": " + _param_names[i]);
}

O código acima recupera o FX struct para uma camada e recebe seus nomes de parâmetros em uma matriz. Em seguida, ele executa um for loop nessa matriz e imprime cada parâmetro para o log de saída.
