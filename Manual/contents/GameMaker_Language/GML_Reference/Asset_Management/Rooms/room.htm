Esta variável contém o índice room para o atual room que seu jogo está rodando. Esta não é uma variável somente de leitura, porém mudando isto não mudará o valor do índice para o atual room, mas sim mudará o room para corresponder ao índice para o qual você definiu a variável. Deve-se ter cuidado ao fazer isto como se o índice para o qual você muda a variável não fosse válido, o jogo irá lançar um erro e fechar. Em geral, é uma prática muito melhor usar room_goto() para mudar rooms.

Nota: Room Os IDs não são baseados em seu pedido no Asset Browser ou no Room Manager, e assim você deve evitar fornecer um valor numérico diretamente. Em vez disso, use a constante de sala para o asset que você deseja consultar (que é simplesmente seu nome) ou recupere através de uma função.

Sintaxe:

room;

Devoluções:

Room Asset

Exemplo:

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

O código acima verificará se o atual room é o primeiro nível e, nesse caso, toca a música desse nível.
