layer_get_depth

Você pode usar esta função para obter o valor de profundidade associado a uma determinada camada. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função retornará essa profundidade de camadas como um número real. Note que a profundidade é definida como sendo maior a "mais distante" da câmera e menor a "mais próxima" da câmera, assim, se você tiver três camadas com profundidades -100, 0, 100, as camadas serão desenhadas na ordem 100, 0, -100, de modo que a camada "superior" (ou seja, a mais próxima da visão da câmera e assim desenhada sobre tudo mais) será a camada com a profundidade -100.

Observe que se você fornecer um valor de identificação de camada inválido, você receberá um erro.

Sintaxe:

layer_get_depth(layer_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to get the depth of (or the layer name as a string)

Devoluções:

Real

Exemplo:

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

O código acima verifica a profundidade de um ID de camada armazenado em uma variável global e se não for -100, destrói a camada e a recria na profundidade de -100.
