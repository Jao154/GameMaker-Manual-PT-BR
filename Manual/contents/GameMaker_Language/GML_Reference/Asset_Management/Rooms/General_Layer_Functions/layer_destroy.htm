layer_destroy

Esta função destruirá a camada dada. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isto terá um impacto no desempenho) e isto irá removê-lo do atual room. Se a camada for uma que foi projetada no editor room, então na próxima vez que você deixar o room e depois retornar, a camada será recriada novamente com o conteúdo original, porém se o room for persistente, a camada será removida, a menos que room persistência seja desligada novamente. Ao destruir uma camada desta forma, todo o seu conteúdo também será removido, portanto, quaisquer IDs de referência para fundos ou mapas tile, etc... não serão mais válidos e quaisquer instâncias atribuídas à camada serão destruídas (realizando seu Evento de Destruição ao mesmo tempo, se tiverem um, assim como o Evento de Limpeza).

Sintaxe:

layer_destroy(layer_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to destroy (or the layer name as a string)

Devoluções:

N/A

Exemplo:

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

O código acima verificará se existe alguma instância do object "obj_Bullet_Parent", e se não existir, destruirá a camada com o ID armazenado na variável global.
