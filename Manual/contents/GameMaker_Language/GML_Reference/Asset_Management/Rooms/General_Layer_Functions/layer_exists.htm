camada_existe

Esta função pode ser usada para verificar se a camada em questão existe. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função retornará um valor booleano de true se existir ou de false se não existir.

NOTA: Esta função funciona dentro do escopo do alvo atual room - por padrão o room no qual a função é chamada - que pode ser definido usando a função layer_set_target_room().

Sintaxe:

layer_exists(layer_name)

Argumento Tipo Descrição
layer_name Identificaçãoporcorda ou camada The name of the layer (a string or ID value)

Devoluções:

Boolean

Exemplo:

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

O código acima verificará se a camada armazenada na variável global realmente existe, e se não existe, então ela é criada.
