layer_get_element_layer

Você pode usar esta função para obter o Layer ID que o elemento dado está ligado. Você fornece o valor único de identificação do elemento (por exemplo, como retornado pela função que criou o elemento ou pelo editor room ) e a função retornará a identificação única da camada em que o elemento se encontra. Se o ID do elemento dado não for válido, então a função retornará -1.

Sintaxe:

layer_get_element_layer(element_id)

Argumento Tipo Descrição
element_id Identificação do elemento da camada The unique ID value of the element to get the layer of

Devoluções:

Layer ID or -1 (if the element is invalid)

Exemplo:

element = layer_get_element_layer(asset_1);

O código acima obtém o ID da camada do elemento com o ID armazenado na variável "asset_1" e o armazena em uma variável.
