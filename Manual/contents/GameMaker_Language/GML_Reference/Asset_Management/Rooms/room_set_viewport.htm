room_set_viewport

Com esta função, você pode definir as propriedades da porta de visualização para qualquer room em seu jogo, exceto o atual. Você fornece o índice room para definir a porta de visualização, o índice da porta de visualização (de 0 a 7) e então se a porta de visualização deve ser visível no início do room ou não (defina para true para tornar a porta visível, e false caso contrário). Depois disso, você define as coordenadas x e y (correspondentes à posição superior esquerda da porta de visualização) e depois a largura e a altura em pixels.

Sintaxe:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

Argumento Tipo Descrição
rm Ativos da sala The index of the room to set
vind Real The index of the view port to set
vis Booleano The visibility of the view port (true é visível, false é invisível)
xport Real The x position for the view port in the room
yport Real The y position of the view port in the room
wport Real The width (in pixels) of the view port
hport Real The height (in pixels) of the view port

Devoluções:

N/A

Exemplo:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Isto irá definir as propriedades da porta de visualização [0] no room indexado na variável "global.myroom".
