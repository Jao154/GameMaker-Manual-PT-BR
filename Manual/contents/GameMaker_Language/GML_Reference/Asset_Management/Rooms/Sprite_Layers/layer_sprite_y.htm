layer_sprite_y

Esta função controla a posição ao longo do eixo y do elemento room do elemento asset sprite na camada. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o valor y a ser usado (com base nas coordenadas room ).

Sintaxe:

layer_sprite_y(sprite_element_id, y);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to change
y The y position for the sprite

Devoluções:

Exemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

O código acima obtém o valor de ID do sprite asset "Clouds" atribuído à camada "Asset_sky" e então define sua posição y para um valor aleatório entre 50 e a altura do room menos 200.
