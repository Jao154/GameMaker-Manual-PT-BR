layer_sprite_get_id

Esta função pode ser usada para recuperar o valor de identificação único de um elemento sprite em uma camada. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create() ou quando você usa o nome da camada junto com layer_get_id()) e o nome do elemento sprite, conforme definido no editor Room. A função retornará o valor de identificação associado a esse elemento sprite na camada.

Note que esta função foi especificamente projetada para uso com sprites que foram adicionados no IDE em uma camada asset, e se você adicionou um sprite a uma camada através de código usando a função layer_sprite_create()então o ID retornado por essa função deve ser usado para todas as referências futuras (já que o elemento sprite não terá nome a ser passado para esta função).

Se a camada especificada não existir, ou se o elemento sprite dado não puder ser encontrado, a função retornará -1.

Sintaxe:

layer_sprite_get_id(layer_id, sprite_element_name)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target
sprite_element_name Cordão The unique name of the sprite element on the layer as defined in the Room Editor

Devoluções:

Sprite Element ID

Exemplo:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

O código acima obterá o ID da camada chamada "Assets_trees" e então o utilizará para recuperar o ID do elemento sprite "graphic_254367CB" nessa camada. O ID do elemento sprite recuperado é então usado para alterar o ID do elemento sprite.
