layer_sprite_angle

Usando esta função você pode mudar o ângulo para o elemento sprite dado em uma camada. Você dá o ID do elemento sprite (que você obtém ao criar um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o valor do ângulo, de 0 a 359 no sentido anti-horário, onde 0 é para a direita, 90 é para cima, 180 é para a esquerda e 270 é para baixo. Se você definir um valor maior que 360, isto será feito em loop para colocá-lo dentro da faixa de 0 a 359.

Sintaxe:

layer_sprite_angle(sprite_element_id, angle);

Argumento Tipo Descrição
sprite_element_id Identificação do Elemento Sprite The unique ID value of the sprite element to change
angle Real The angle of the sprite (default is 0)

Devoluções:

N/A

Exemplo:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if (layer_sprite_get_angle(asset_sprite) != 0)
{
    layer_sprite_angle(asset_sprite, 0);
}

O código acima verificará o elemento sprite atribuído à camada em que a instância que executa o código está e, se seu ângulo não for 0, ele será ajustado para 0.
