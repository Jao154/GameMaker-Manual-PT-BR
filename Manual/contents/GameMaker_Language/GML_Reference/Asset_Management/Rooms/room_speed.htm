room_speed (depreciado)

Esta variável mantém a velocidade de funcionamento de todos rooms (e do jogo) em quadros de jogo por segundo. Note que isto NÃO é o FPS (frames por segundo), mas sim o número de etapas do jogo que o GameMaker tentará manter em cada segundo.

IMPORTANTE! Esta variável é mantida apenas para o Legacy Support, e não deve ser usada, pois não define mais a velocidade para um único room, mas para TODOS rooms no jogo. Para mudar a velocidade do jogo você deve, ao invés disso, estar usando a função game_set_speed().

Sintaxe:

room_speed;

Devoluções:

Exemplo:

alarm[0] = room_speed * 10;

O código acima usa room_speed para programar um alarme para funcionar por 10 segundos. O uso da variável desta forma garantirá que o alarme funcione por 10 segundos, não importando a velocidade definida para o room.
