altura_do_quarto

Esta variável mantém a altura do atual room em pixels. Você pode mudar esta variável para mudar a altura do room a qualquer momento, e as mudanças serão aplicadas na parte inferior do room, pois a origem é considerada como o canto superior esquerdo. Assim, por exemplo, se o room tiver uma altura de 480px e você o definir para 640px, o room será expandido para baixo com um acréscimo de 180px a mais na parte inferior.

Sintaxe:

room_height;

Devoluções:

Exemplo:

if bbox_bottom > room_height
{
    y += room_height - bbox_bottom;
}

O código acima verifica se a caixa de delimitação sprite da instância atual é maior do que a altura do room, e se é ela que move a instância para cima.
