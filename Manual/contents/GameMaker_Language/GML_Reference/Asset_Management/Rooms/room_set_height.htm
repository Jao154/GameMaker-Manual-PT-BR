altura_do_sala

Com esta função você pode alterar (ou definir) a altura de qualquer room em seu jogo, exceto o atual.

Sintaxe:

room_set_height(index, h);

Argumento Tipo Descrição
index Ativos da sala The index of the room to set the height of.
h Real The new height of the room in pixels.

Devoluções:

N/A

Exemplo:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Isto criará um novo room e armazenará seu índice na variável "global.myroom". Em seguida, fixará sua largura para 640 pixels, sua altura para 480 pixels, sua legenda para "Game Room" e sua persistência para "falso".
