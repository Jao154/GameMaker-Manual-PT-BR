object_index

Esta variável lida retorna apenas o índice do object a partir do qual a instância foi criada. Este não é o mesmo nome do object, que é um string e pode ser encontrado usando object_get_name()Esta função retorna o número de índice, que é um valor único que o GameMaker atribui a cada object no momento da criação.

Sintaxe:

object_index;

Devoluções:

Object Asset

Exemplo:

obj_name = object_get_name(object_index);

O código acima utilizará o object_index para encontrar o nome do object a partir do qual a instância atual foi criada.
