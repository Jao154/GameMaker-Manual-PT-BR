object_set_solid

Com esta função você pode flag um object específico como sendo sólido ou não (para mais informações sobre o "sólido" flag veja a seção sobre o Editor de Objetos). Isto significa que todas as instâncias deste object que são criadas após a mudança de sólido serão criadas com este novo estado, enquanto as instâncias que já estão no room podem não ser afetadas.

Por favor, note que esta não é uma função de instância! Você pode definir o sólido flag de instâncias individuais usando a variável solid e assim ter dez instâncias todas marcadas como sólidas mesmo que o object em questão tenha o flag marcado como falso no IDE, e mesmo se você mudar o sólido flag do object para verdadeiro usando esta função, todas as instâncias que atualmente no room permanecerão como estavam, e somente as instâncias criadas após chamar a função começarão com o sólido marcado como verdadeiro.

Sintaxe:

object_set_solid( index, solid );

Argumento Tipo Descrição
index Objeto Ativo The index of the object to change.
solid Booleano The new solidity of the object (true=sólido, false=não sólido).

Devoluções:

N/A

Exemplo:

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

O código acima verifica o sólido flag do object "obj_Block" e se for falso, então ele é alterado para ser sinalizado como verdadeiro.
