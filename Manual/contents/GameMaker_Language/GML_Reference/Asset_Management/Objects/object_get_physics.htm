object_get_physics

Esta função lhe dirá se o object que você está verificando foi assinalado como "physics enabled" - nesse caso retornará true, - ou não - nesse caso retornará false.

Sintaxe:

object_get_physics(obj);

ArgumentoTipo Descrição
obj The index of the object to check.

Devoluções:

Exemplo:

if object_get_physics(object_index)
{
    phy_active = true;
}

O código acima verificará a instância que a executa para ver se o object a partir do qual foi criada está habilitada para a física, e se é ele que ativa a simulação física da instância.
