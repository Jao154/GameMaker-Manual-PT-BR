evento_ação DEPRECADO

Esta variável somente de leitura retorna o índice da ação que está sendo executada atualmente, começando em 0 nas versões anteriores de GameMaker. No entanto, esta é agora uma variável obsoleta no GameMaker. Ela foi deixada apenas para suporte legado, e sempre retornará 0, já que todas as ações são concatenadas juntas para melhorar a velocidade de execução.

Sintaxe:

event_action;

Devoluções:

Real

Exemplo:

num = event_action;

O código acima armazena o número da ação atual na variável "num".
