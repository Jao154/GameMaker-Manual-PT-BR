shader_enable_corner_id

Com esta função você pode definir um estado global para todos shaders sendo usado onde, quando ativado, o shader "rouba" 2 bits dos valores de cor de entrada. O primeiro é a partir do bit inferior do valor da cor vermelha, e o segundo é a partir do bit inferior do valor da cor vermelha. um pouco do valor da cor azul. Estes valores podem ser recuperados no site shader para se saber com qual vertex você está lidando (ou seja: qual canto).

Sintaxe:

shader_enable_corner_id(enable);

ArgumentoTipo Descrição
enable Enable (true) ou desabilitar (false) esta função.

Devoluções:

Exemplo:

shader_enable_corner_id(true);

O código acima permitirá o uso de bits de cor para a identificação do canto para todos shaders.
