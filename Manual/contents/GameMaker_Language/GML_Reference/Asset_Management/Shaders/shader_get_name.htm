shader_get_name

Com esta função, você pode recuperar o nome de um recurso shader. Você fornece o valor de identificação único para o shader para obter o nome do recurso e a função retornará o nome do recurso como um string.

Sintaxe:

shader_get_name(shader);

ArgumentoTipo Descrição
shader The index of the shader to get the name of.

Devoluções:

Exemplo:

var _shader = shader_current();
var _name = shader_get_name(_shader);
draw_text(32, 32, "Debug - Currently Rendering = " + _name);

O código acima receberá o nome do dado shader e o desenhará para a tela.
