instance_activate_layer

Com esta função você pode ativar uma camada que tenha sido desativada anteriormente. Você precisa fornecer o ID da camada, que pode ser o nome da camada como escrito no editor de código (como um string) ou o valor real do ID da camada como retornado pelo layer_create() e todas as instâncias desativadas nessa camada serão ativadas mais uma vez. Note que se você desativou uma camada que tem instâncias persistentes, você precisará reativar a camada novamente com esta função antes de mudar room, caso contrário quaisquer instâncias persistentes não serão transportadas e serão descartadas. Note também que a ativação não é instantânea, e uma instância que tenha sido ativada desta forma não será considerada ativa até o final do evento em que a função foi chamada.

Sintaxe:

instance_activate_layer(layer_id);

Argumento Tipo Descrição
layer_id Identificação da camada ou String The layer name string (or ID value) to be used

Devoluções:

N/A

Exemplo:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

O código acima ativa todas as instâncias dentro do room e depois desativa aquelas que estão fora dos limites da visão atual da câmera, exceto as instâncias na camada "Player_Layer", que são reativadas novamente no final.
