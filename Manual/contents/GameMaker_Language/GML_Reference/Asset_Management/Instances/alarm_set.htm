alarme_set

Esta função pode ser usada para definir um alarme. Você fornece o número do alarme de 0 a 11, e depois o valor para ajustar o alarme. O valor deve ser um valor inteiro, e você pode defini-lo como -1 para parar o alarme (valores não inteiros serão arredondados para o número inteiro mais próximo). Este é um método alternativo para definir diretamente a matriz do alarme.

Sintaxe:

alarm_set(index, value);

Argumento Tipo Descrição
index Real The alarm index to set, from 0 to 11.
value Real The value (an integer) to set the alarm to.

Devoluções:

N/A

Exemplo:

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

O código acima verifica todos os alarmes na instância de chamada e, se forem maiores que 0, os coloca em -1, impedindo-os de continuar a contagem regressiva.
