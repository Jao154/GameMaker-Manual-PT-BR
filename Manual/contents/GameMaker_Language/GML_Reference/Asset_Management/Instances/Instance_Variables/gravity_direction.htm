direção_da_gravidade

gravity_direction é uma das propriedades "embutidas" que todas as instâncias têm e podem ser usadas para definir a direção do movimento quando as instâncias gravity Note que as direções no GameMaker são normalmente calculadas como 0° sendo a direita, 90° sendo para cima, 180° sendo para a esquerda e 270° sendo para baixo.

Sintaxe:

gravity_direction;

Devoluções:

(single precision floating point value)

Exemplo:

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

O código acima mudará a direção da gravidade se o jogador object encontrar uma instância de "obj_switch".
