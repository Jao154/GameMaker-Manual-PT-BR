fricção

Friction é uma das propriedades "embutidas" que as instâncias podem ter e podem ser usadas para retardar a instância ao longo do tempo, quando o speed é diferente de zero. Funciona simplesmente subtraindo uma quantidade da velocidade de cada passo até que o object tenha uma velocidade 0, portanto, se o atrito for ajustado, por exemplo, para 0,1 e a velocidade da instância for 1 (1 pixel por passo), ele irá diminuir a velocidade e parar após 10 passos terem passado. Observe também que o atrito é aplicado a velocidades positivas e negativas igualmente, sendo o resultado líquido sempre que o object tem uma velocidade 0 após um determinado tempo.

Sintaxe:

friction;

Devoluções:

(single precision floating point value)

Exemplo:

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

O código acima só aplicará o atrito se a velocidade absoluta da instância estiver acima de 0.
