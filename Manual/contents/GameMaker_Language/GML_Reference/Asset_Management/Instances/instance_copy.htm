exemplo_copia

Com esta função, você pode "clonar" uma instância, pois isto criará uma nova versão da instância executando o código em sua mesma posição. O argumento "perf" é usado para instruir esta nova instância a realizar ou não o evento de criação. Esta função retorna a função id da nova instância, que pode então ser armazenada em uma variável ou usada para acessar essa instância.

NOTA: Se você optar por não realizar o evento de criação, poderá encontrar erros se a instância depender de qualquer variável rubricada neste evento.

Sintaxe:

instance_copy(perf);

Argumento Tipo Descrição
perf Booleano Whether to perform the new instance's Create event (true) or not (false).

Devoluções:

Instance ID

Exemplo:

var inst = instance_number(object_index);
if (inst < 10)
{
    instance_copy(true);
}

O código acima cria uma variável local e a utiliza para armazenar o número de instâncias do object rodando o código no room. Se o número for inferior a 10, a instância então faz uma cópia de si mesma.
