número_da_classe

Com esta função você pode descobrir quantas instâncias ativas do object especificado existe no room. Ao verificar usando esta função, se o object é um dos pais, então todos os filhos objects também serão incluídos no valor de retorno, e também observar que aquelas instâncias que foram desativadas com a instância funções desativadas não serão incluídas nesta verificação.

Sintaxe:

instance_number(obj);

ArgumentoTipo Descrição
obj The object to total the number of instances of.

Devoluções:

Exemplo:

if instance_number(object_index) < 50
{
    instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
}

O código acima verificará o número de instâncias criadas no mesmo object que a instância atual e, se houver menos de 50, criará outra em uma posição aleatória dentro do room.
