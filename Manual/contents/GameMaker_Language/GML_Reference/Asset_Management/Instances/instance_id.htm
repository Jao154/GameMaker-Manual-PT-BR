exemplo_id

Esta matriz de leitura contém todos os id s de cada instância ativa dentro do room. Isto significa que se você tiver usado qualquer uma das funções Desativar Instância, as instâncias que foram desativadas não serão incluídas neste array (se você tiver usado um valor deste array anteriormente, ele agora retornará a palavra-chave noone).

Sintaxe:

instance_id[num];

Devoluções:

Instance ID

Exemplo:

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

O código acima será loop através de todas as instâncias dentro do room e adicionará 0,1 à sua velocidade.
