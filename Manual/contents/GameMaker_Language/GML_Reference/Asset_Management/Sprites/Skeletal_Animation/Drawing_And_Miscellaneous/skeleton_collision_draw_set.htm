esqueleto_colisão_desenho_de_squadrilho

Com esta função, você pode ativar (true) ou desativar (false) desenhando os dados de colisão para a animação do esqueleto atual sprite que está sendo utilizada pela instância. Se esta for ligada, a caixa de delimitação e a máscara de colisão precisa serão desenhadas como contornos ao redor do sprite.

Esta função só funciona quando o desenho do esqueleto sprite está sendo tratado pelo object. Para desenhar a caixa delimitadora de colisão ao desenhar um esqueleto sprite manualmente, use draw_skeleton_collision().

Sintaxe:

skeleton_collision_draw_set(flag);

Argumento Tipo Descrição
flag Booleano Set to true para ligar o desenho, e false para desligá-lo.

Devoluções:

N/A

Exemplo:

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

O código acima verifica se o jogo está sendo executado no modo de depuração e, se estiver, alterna os dados de colisão do esqueleto para a instância a ser mostrada.
