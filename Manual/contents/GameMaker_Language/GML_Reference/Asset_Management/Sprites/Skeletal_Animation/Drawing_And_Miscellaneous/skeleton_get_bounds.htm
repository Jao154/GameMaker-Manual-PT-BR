esqueleto_get_bounds

Esta função retornará uma série de valores associados a qualquer caixa de delimitação para a animação do esqueleto atualmente atribuída sprite. Você fornece o número de índice para a caixa de delimitação para obter os detalhes de (você pode recuperar o número total de caixas de delimitação para o sprite usando a função skeleton_get_num_bounds()) e a função retornará uma matriz com os seguintes elementos:

  • [0] - o número de pontos na caixa delimitadora (um real)
  • [1] - o nome da caixa de delimitação (a string)
  • [2] - a posição x do primeiro ponto
  • [3] - a posição y do primeiro ponto
  • [etc...] - mais dados de posição x/y até o número de pontos na caixa delimitadora

Sintaxe:

skeleton_get_bounds(index);

Argumento Tipo Descrição
index Real The bounding box index to get the values of.

Devoluções:

Array (minimum 2 elements: numPoints, name [, xPos, yPos, etc...])

Exemplo:

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

O código acima irá loop através de cada uma das caixas de delimitação associadas ao sprite atualmente atribuído e então desenhar esses dados como um string para a tela.
