draw_get_enable_enable_blendmodes

Esta função retorna se os modos de mistura por ranhura para esqueleto sprites estão habilitados(verdadeiro) ou desabilitados(falso).

Sintaxe:

draw_get_enable_skeleton_blendmodes()

Devoluções:

Boolean

Exemplo:

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

O código acima permite os modos de mistura por ranhura para esqueleto sprites, se estiverem desabilitados.
