draw_skeleton_instance

Esta função é apenas para uso com instâncias que têm sprites atribuídas a elas criadas usando o programa de animação esquelética Spine. Enquanto você pode desenhar estas sprites usando as funções normais de desenho (como draw_self()), há momentos em que você pode querer desenhar um único quadro ou mostrar uma mudança de pele texture sem realmente mudar as propriedades sprite por exemplo. Nestes casos você pode desenhar o sprite que foi atribuído à instância com esta função, onde você dá a identificação única da instância e depois o nome da animação para obter o quadro (um string, como definido dentro do programa utilizado para fazer o sprite). Em seguida, você dá o nome da pele para usar (novamente, como definido quando o sprite foi feito), e um valor do frame. O valor do frame é para dizer ao GameMaker qual parte da animação deve mostrar e é um valor entre 0 e o image_number. O resto dos argumentos são os mesmos usados para o desenho normal sprite e mudarão a escala, ângulo, cor da mistura e alfa do quadro de animação final a ser desenhado.

ADVERTÊNCIA! Devido à forma como as animações esqueléticas são interpoladas entre vários quadros "chave", há uma sobrecarga adicional associada a esta chamada e, portanto, é recomendável evitar desenhar o esqueleto baseado em sprites usando este método, a menos que seja absolutamente necessário.

Sintaxe:

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Argumento Tipo Descrição
instance Identificação da instância The index of the instance to draw the sprite from.
animname Cordão The name of the animation to get the frame from (a string).
skinname Cordão The name of the skin to use (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Cor The colour with which to blend the sprite.
alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devoluções:

N/A

Exemplo:

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

O código acima desenhará o primeiro frame da animação esquelética sprite atribuído à instância dada, usando o conjunto de animação "salto" e a skin definida como "skin1".
