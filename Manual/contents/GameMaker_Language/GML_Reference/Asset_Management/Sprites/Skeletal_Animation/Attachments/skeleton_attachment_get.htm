esqueleto_aparelhagem_esqueleto

Uma animação esquelética sprite pode ter outros sprites adicionados como anexos, com estes sprites sendo adicionados a um slot nomeado (o nome é dado quando você cria o slot anexo em seu programa de animação) e eles serão desenhados junto com a animação do atual sprite. Com essa função você pode obter o nome (como um string) do anexo para o slot dado do atualmente atribuído sprite. Note que o anexo sprites é referenciado através de sua cadeia de nomes como atribuído em Spine, ou quando você chama skeleton_attachment_create().

Sintaxe:

skeleton_attachment_get(slot);

Argumento Tipo Descrição
slot Cordão The slot name (a string) to get the attachment of.

Devoluções:

String

Exemplo:

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

O código acima verificaria o nome do anexo atualmente atribuído para o slot chamado "slot_leftHand" e se um string vazio for devolvido, um novo sprite será anexado.
