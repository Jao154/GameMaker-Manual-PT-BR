esqueleto_animação_set_ext_sext

Quando você define um conjunto de animação para sua animação esquelética sprite, é atribuída uma faixa 0 internamente. Entretanto, você pode atribuir outros conjuntos a outras pistas (1 e acima) e, desta forma, fazer com que seu sprite toque duas (ou mais) animações ao mesmo tempo. Note que o sprite deveria ter sido projetado em seu programa de animação com isto em mente, de modo que a animação "base" na faixa 0 só tem keyframes para certos ossos, enquanto as próximas faixas têm keyframes para as outras. Por exemplo, se você tiver uma animação "walk" e seu keyframes só afeta as pernas do sprite, você pode então ter um conjunto de animações de cabeça e de braço que você pode definir nas pistas 1 e 2 respectivamente, permitindo que você tenha um movimento de base e se adapte nele em tempo real enquanto o jogo está sendo jogado.
A animação é feita em loop, que é o padrão.

Sintaxe:

skeleton_animation_set_ext(animname, track, [loop]);

Argumento Tipo Descrição
animname Cordão The name (a string) of the animation set to use.
track Real The track for the animation set to use.
loop Booleano Se a animação deve ou não ser loop (o padrão é true)

Devoluções:

N/A

Exemplo:

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

O código acima mudará o conjunto de animação sendo usado pela faixa 1 para o conjunto "bodyfight" ou o conjunto "bodywalk" quando o botão esquerdo do mouse for pressionado.
