esqueleto_posição_set_animação

Esta função definirá a animação atribuída à faixa dada para a posição que você fornecer. A posição deve ser um valor real entre 0 e 1. Se você fornecer um valor fora deste intervalo, então a posição da pista será definida para a parte fracionária desse valor. Suponha que você use um valor de 0,4, um valor de 1,4, um valor de 2,4, ... Todos eles definirão a posição para 0,4.

Sintaxe:

skeleton_animation_set_position(track, position);

Argumento Tipo Descrição
track Real The animation track to set the frame of.
position Real The new position for the animation frame.

Devoluções:

N/A

Exemplo:

skeleton_animation_set_position(1, 0.25);

O código acima definirá a posição para a faixa 1 da animação do esqueleto para 0,25 ou, colocado de forma diferente, a 25% desde o início da animação.
