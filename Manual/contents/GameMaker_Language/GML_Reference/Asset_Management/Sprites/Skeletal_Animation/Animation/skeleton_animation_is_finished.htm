esqueleto_animação_esqueleto_é_acabado

Com esta função você pode verificar se a atual animação esquelética não inclinada terminou de tocar na pista dada.

Sintaxe:

skeleton_animation_is_finished(track);

Argumento Tipo de dados Descrição
faixa Real O índice da faixa de animação

Devoluções:

Boolean

Exemplo:

var _finished = skeleton_animation_is_finished(1);
if _finished
{
    show_debug_message("The animation has finished playing");
}

O código acima verifica primeiro se a animação na pista 1 terminou de tocar e armazena o resultado em uma variável local _finished. Em seguida, ele mostra uma mensagem de debug caso a animação tenha terminado de tocar.
