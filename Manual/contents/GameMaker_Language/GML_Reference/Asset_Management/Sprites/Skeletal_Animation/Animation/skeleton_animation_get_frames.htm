esqueleto_animação_get_frames

Esta função pode ser usada para recuperar o número de quadros que qualquer animação de esqueleto possui. Você fornece o nome da animação do esqueleto (como um string, como definido no programa usado para fazer a animação, ou como retornado usando a função skeleton_animation_get()) e a função retorna os frames que ela tem como um valor inteiro. A função retornará 0 se a animação especificada não existir.

Sintaxe:

skeleton_animation_get_frames(anim_name);

Argumento Tipo Descrição
anim_name Cordão The animation name to get the frames of.

Devoluções:

Real

Exemplo:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

O código acima obterá o número de quadros na animação e então definirá o sprite para o último quadro e deixará de animar.
