esqueleto_de_squadra_esqueleto

Com esta função você pode encontrar quais slots estão em uma posição específica de espaço de sala no Spine sprite associado à instância atual. Você cria uma lista DS e fornece sua identificação juntamente com uma posição x/y para verificação e a lista será preenchida com o nome string para cada um dos slots de anexo disponíveis para o sprite (incluindo quaisquer modificações de anexo). Observe que a lista é sempre ordenada em ordem decrescente a partir do slot mais alto.

Sintaxe:

skeleton_find_slot(x, y, list);

Argumento Tipo Descrição
x Real The x position in the room to check.
y Real The y position in the room to check.
list DS List ID The ID of the DS list to populate with the DS maps.

Devoluções:

N/A

Exemplo:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

O código acima cria uma lista DS e depois a preenche com os nomes dos slots encontrados na posição do mouse no site room.
