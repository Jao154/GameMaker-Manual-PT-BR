lista_esqueleto_de_squema

Com esta função você pode preencher uma lista DS (pré-criada) com todos os nomes dos slots criados como parte da animação do esqueleto sprite. Os nomes serão strings e poderão ser usados nas outras funções de slots de animação esqueletizada para estes tipos de sprite.

Sintaxe:

skeleton_slot_list(sprite, list);

Argumento Tipo Descrição
sprite Ativo Sprite The sprite index of the Spine skeletal animation to get the list from.
list DS List ID The ID of the DS list to populate with the bone names.

Devoluções:

N/A

Exemplo:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

O código acima cria uma lista DS e depois a preenche com os nomes dos slots para uso posterior.
