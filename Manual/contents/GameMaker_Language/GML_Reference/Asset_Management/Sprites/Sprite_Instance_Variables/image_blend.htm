image_blend

Esta variável controla a "coloração" da instância sprite e o valor padrão é -1 (mas também pode ser c_white). Qualquer outro valor (incluindo constantes de cor internas como c_red, ou c_aqua) irá misturar a cor especificada com a original sprite. Observe que para que as mudanças nesta variável sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker irá desenhar por padrão o sprite) ou ser desenhada usando uma das funções de desenho estendido como draw_self() ou draw_sprite_ext().

Observe que você deve tentar limitar a mistura na plataforma HTML5 (a menos que utilize WebGL), pois cada mistura sprite tem de ser colocada em cache separadamente e, portanto, ter muitas misturas sprites pode afetar negativamente o desempenho (você também pode definir o tamanho cache utilizando a função sprite_set_cache_size()).

Sintaxe:

image_blend;

Devoluções:

Colour Constant

Exemplo:

image_blend = make_colour_hsv(255, 255, random(255));

O código acima definirá a mistura_de_imagem do sprite definido para a instância a uma cor aleatória.
