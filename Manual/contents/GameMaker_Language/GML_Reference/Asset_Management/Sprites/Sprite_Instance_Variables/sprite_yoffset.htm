sprite_yoffset

Esta variável lida apenas retorna o yoffset local (o componente y da origem como definido no editor sprite ) do sprite que foi atribuído à instância. Este yoffset é retornado em pixels e será dependente do image_yscalede modo que (por exemplo) se um quadrado 32x32 sprite for definido como tendo a origem y em 16, quando esse sprite for escalado por 2 esta variável retornará 32. Se você precisar do yoffset não escalonado, você deve usar sprite_get_yoffset().

Sintaxe:

sprite_yoffset;

Devoluções:

Real

Exemplo:

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

O código acima verifica se a borda superior do sprite da instância está fora do room e se ela move a instância de modo que todo o sprite seja visível.
