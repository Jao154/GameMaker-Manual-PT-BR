máscara_index

Quando você define um object no GameMaker você pode atribuir-lhe uma máscara a ser usada para colisões em vez daquela que corresponde ao definido sprite. Esta variável pode ser usada para encontrar a sprite_index dessa máscara (ou retornará -1 se não tiver sido atribuído o endereço sprite ) ou para definir a máscara por exemplo para o escolhido sprite. Ajustar o índice da máscara significa que você pode ter, por exemplo, um sprite para a instância com uma forma irregular, mas dar-lhe uma máscara de colisão circular que é obtida a partir de um sprite diferente.

Sintaxe:

mask_index;

Devoluções:

Sprite Asset

Exemplo:

mask_index = spr_Round;

O código acima define a máscara da instância como a do sprite "spr_Round".
