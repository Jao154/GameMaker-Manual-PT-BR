sprite_width

Esta variável lida retorna apenas a largura do sprite que foi atribuído à instância. Esta largura é retornada em pixels e será dependente do image_xscale. Se você precisar da largura não escalonada, você deve usar sprite_get_width().

Sintaxe:

sprite_width;

Devoluções:

Real

Exemplo:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

O código acima verifica a largura do sprite atribuído à instância executando o código contra a largura do recurso sprite e, se não for o mesmo, redefine o image_xscale para 1,
