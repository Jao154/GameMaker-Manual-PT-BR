sprite_xoffset

Esta variável lida apenas retorna o xoffset local (o componente x da origem como definido no editor sprite ) do sprite que foi atribuído à instância. Este xoffset é retornado em pixels e será dependente do image_xscalede modo que (por exemplo) se um quadrado 32x32 sprite for definido como tendo a origem x em 16, quando esse sprite for escalado por 2, essa variável retornará 32. Se você precisar do xoffset não escalonado, você deve usar sprite_get_xoffset().

Sintaxe:

sprite_xoffset;

Devoluções:

Real

Exemplo:

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

O código acima verifica se a borda esquerda do sprite da instância está fora do room e se ela move a instância de modo que todo o sprite seja visível.
