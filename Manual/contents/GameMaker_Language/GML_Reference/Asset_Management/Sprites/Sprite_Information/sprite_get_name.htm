sprite_get_name

Esta função devolverá o nome como uma seqüência do especificado sprite. Este nome é o que foi especificado para o sprite na árvore de recursos da janela principal do GameMaker. Note que este é apenas um string e não pode ser usado para fazer referência direta ao sprite - para isso você precisaria do índice sprite. Você pode, entretanto, usar este string para obter o índice sprite usando o string retornado junto com a função asset_get_index().

Sintaxe:

sprite_get_name(index);

Argumento Tipo Descrição
index Ativo The index of the sprite to get the name of.

Devoluções

String

Exemplo:

str = sprite_get_name(sprite_index);

O código acima receberá o nome do índice sprite para a instância que executa o código e armazenará o valor de retorno na variável "str".
