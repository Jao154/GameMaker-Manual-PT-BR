sprite_get_yoffset

Quando você define um sprite no editor sprite, você precisa definir a origem para isso sprite. Este é o ponto em que o sprite será "anexado" ou "desenhado" quando usado com uma instância. Esta função retorna o offset relativo para o eixo y da origem baseado no canto superior esquerdo sendo a posição 0,0, com um +y sendo para baixo e um -y sendo para cima a partir disto. A imagem a seguir ilustra isto:

Sintaxe:

sprite_get_yoffset(index);

ArgumentoTipo Descrição
index The index of the sprite to find the yoffset of.

Devoluções

Exemplo:

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = sprite_get_yoffset(sprite_index);
}

O código acima garantirá que uma instância seja mantida dentro do room com base no sprite (portanto, o sprite é sempre visível).
