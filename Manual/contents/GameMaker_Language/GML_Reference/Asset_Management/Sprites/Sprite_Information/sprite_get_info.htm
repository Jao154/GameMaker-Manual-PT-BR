sprite_get_info

Esta função é utilizada para recuperar informações para o dado sprite. Você fornece um índice sprite (que pode ser um asset adicionado através do Navegador de Ativos ou um sprite adicionado em runtime) e a função retorna uma estrutura com as seguintes variáveis:

Estruturas de Informação Sprite
Variável Tipo Descrição
largura real A largura do duende (em pixels)
altura real A altura do duende (em pixels)
xoffset real Offset/origem em X do duende (em pixels)
yoffset real Offset em Y/origem do duende (em pixels)
transparente booleano true se o sprite estiver marcado como transparente, caso contrário false
(Isto só pode ser especificado através de sprite_add() ou funções similares, e será false para sprites criado no IDE)
smooth booleano true se o sprite estiver marcado como liso, caso contrário false
(Isto só pode ser especificado através de sprite_add() ou funções similares, e será false para sprites criado no IDE)
tipo real O tipo de duende:
0 - Bitmap (Regular sprites)
1 - SWF
2 - Lombada
bbox_left real Posição da borda esquerda da caixa delimitadora (em pixels)
bbox_top real Posição da borda superior da caixa delimitadora (em pixels)
bbox_right real Posição da borda direita da caixa delimitadora (em pixels)
bbox_bottom real Posição da borda inferior da caixa delimitadora (em pixels)
nome corda O nome do duende
num_subimages real O número de sub-imagens (ou quadros) no duende
use_mask booleano true se este sprite usa uma máscara de colisão, caso contrário false
num_masks real O número de máscaras neste sprite (será maior que 1 se o sprite usar máscaras por estrutura)
nineslice estrutura A estrutura de nove fatias para este sprite, ou undefined se não tiver nove dados de fatias
mensagens matriz Conjunto de mensagens transmitidas para este sprite, onde cada mensagem transmitida é um struct contendo informações sobre a mensagem (mais informações em "General Sprite Data")
frame_info matriz Conjunto de quadros para este sprite, onde cada quadro é um struct contendo informações sobre seu tempo (mais informações em "General Sprite Data")

Esta variável adicional está disponível apenas para sprites Bitmap (regulares):

Estruturas de Informação Sprite
Variável Tipo Descrição
quadros matriz Conjunto de quadros para este sprite, onde cada quadro é um struct contendo informações sobre sua textura (mais informações em "Bitmap Sprite Data")

Estas variáveis adicionais estão disponíveis apenas para Spine sprites:

Estruturas de Informação Sprite
Variável Tipo Descrição
num_atlas real O número de atlas textures utilizado
atlas_textura matriz Matriz de texture IDs utilizados para o atlas
pré-multiplicado booleano true se este sprite estiver marcado como pré-multiplicado, caso contrário false
nomes_de_animação matriz Array contendo os nomes de cada animação neste sprite
skin_names matriz Array contendo os nomes de cada pele deste duende
ossos matriz Array contendo structs para cada osso deste duende (mais informações em "Spine Sprite Data")
vagas matriz Array contendo structs para cada slot neste sprite (mais informações em "Spine Sprite Data")

A função retornará undefined se o dado sprite não existir. Observe também que as informações retornadas neste struct devem ser consideradas somente leitura, pois a modificação de qualquer uma destas variáveis não afetará o sprite.

As seções abaixo contêm informações sobre as arrays e structs incluídas no retorno struct com base no tipo sprite:

Geral Sprite Dados

Esta seção contém informações sobre as variáveis incluídas no struct para todos os tipos de sprites.

A variável messages é uma matriz que contém informações sobre as mensagens transmitidas que existem no dado sprite. Cada mensagem difundida nesta matriz é uma struct contendo as seguintes variáveis:

Nome da variável Tipo de dados Descrição
quadro real O momento da transmissão desta mensagem desde o início da animação (em frames)
mensagem corda A cadeia de mensagens transmitidas

A variável frame_info é uma matriz que contém informações sobre os horários das armações do duende. Cada frame desta matriz é um struct que contém as seguintes variáveis:

Nome da variável Tipo de dados Descrição
quadro real O tempo para o início deste quadro (em quadros)
duração real A duração deste quadro (em quadros)
image_index real O índice de imagem deste quadro

Bitmap Sprite Dados

Esta seção contém informações sobre variáveis incluídas no struct para Bitmap sprites (sprites que não são Spine ou SWF tipo sprites).

A variável frames é uma matriz que contém informações no textures dos quadros da sprite. Cada frame desta matriz é um struct que contém as seguintes variáveis:

Nome da variável Tipo de dados Descrição
x real A posição X deste quadro em sua página texture (em pixels)
y real A posição em Y deste quadro em sua página texture (em pixels)
w real A largura lógica da moldura (em pixels) utilizada internamente
h real A altura lógica do quadro (em pixels) utilizado internamente
textura real O ID da página texture para este quadro
largura_ original real A largura original da moldura (em pixels)
altura_ original real A altura original da moldura (em pixels)
largura_da_culturas real A largura real da moldura na página texture após o corte e a escala (uma vez que o GameMaker corta automaticamente o espaço vazio ao redor de uma imagem, e também a escala para baixo se não couber)
altura_da_culturas real A altura real da moldura na página texture após o corte e a escamação
x_offset real O deslocamento X da margem esquerda da imagem original para a margem esquerda da seção recortada
y_offset real O desvio em Y da borda superior da imagem original para a borda superior da seção recortada

Lombada Sprite Dados

Esta seção contém informações sobre variáveis incluídas no struct para Spine sprites.

A variável bones é uma matriz que contém informações sobre todos os ossos no dado Spine sprite. Cada osso desta matriz é um struct contendo as seguintes variáveis:

Nome da variável Tipo de dados Descrição
pai corda O nome do osso pai, ou undefined se este osso não tiver um osso pai
nome corda O nome deste osso
índice real O índice deste osso
comprimento real O comprimento deste osso
x real A posição X deste osso
y real A posição em Y deste osso
rotação real A rotação deste osso
escala_x real (Interno para Spine) Valor da escala em X
escala_y real (Interno para Spine) Valor da escala em Y
shear_x real (Interno para Spine) Valor de cisalhamento em X
shear_y real (Interno para Spine) Valor de cisalhamento em Y
transform_mode real (Interno para Spine) O modo de transformação

NOTA: Favor consultar a documentação da Coluna vertebral para maiores informações sobre as variáveis internas da coluna vertebral.

A variável slots é uma matriz que contém informações sobre todos os slots no dado Spine sprite. Cada slot desta matriz é um struct contendo as seguintes variáveis:

Nome da variável Tipo de dados Descrição
nome corda O nome do espaço
índice real O índice do slot
osso corda O nome do osso do slot, ou "(none)" se não houver osso para este slot
anexo corda Nome do anexo
vermelho real Componente vermelho da cor do slot (0-1)
verde real Componente verde da cor da ranhura (0-1)
azul real Componente azul da cor da ranhura (0-1)
alfa real Componente alfa da cor do slot (0-1)
blendMode real (Interno para Spine) Modo de mistura para o slot
vermelho_escuro* real Componente vermelho da cor escura do slot (0-1)
verde_escuro* real Componente verde do slot de cor escura (0-1)
azul_escuro* real Componente azul da ranhura de cor escura (0-1)
alfa_escuro* corda Componente alfa da cor escura do slot (0-1)

* NOTA: As variáveis de cor escura só estão disponíveis se o slot tiver o Tint Black habilitado.

Sintaxe:

sprite_get_info(index);

Argumento Tipo Descrição
index Ativo Sprite The index of the sprite to get the information for.

Devoluções

Sprite Info Struct (or undefined)

Exemplo:

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

O código acima recebe a informação struct para a instância sprite, e então verifica se não é undefined e se seu tipo é 0 (o que significa que é um bitmap sprite). Nesse caso, ele recebe a matriz de mensagens de difusão para esse sprite e então executa um para loop para imprimir cada mensagem de difusão (junto com seu frame) para o log de saída.
