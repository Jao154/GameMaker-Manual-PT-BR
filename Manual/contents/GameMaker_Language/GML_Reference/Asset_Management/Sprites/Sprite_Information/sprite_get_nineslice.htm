sprite_get_nineslice

Esta função retorna a estrutura Nine Slice para um sprite, contendo todas as suas propriedades Nine Slice como definido no Editor Sprite ou usando sprite_set_nineslice(). O conteúdo deste struct está detalhado nesta página.

Se o sprite fornecido não tiver uma Nove Fatias struct atribuída a ele, uma nova struct com propriedades padrão de Nove Fatias será criada para o sprite e devolvida.

A alteração de quaisquer valores neste struct modificará as propriedades das Nove Fatias do original sprite, afetando quaisquer futuras chamadas de sorteio feitas com aquele sprite.

Sintaxe:

sprite_get_nineslice(ind);

ArgumentoTipo Descrição
ind The index of the sprite from which the Nine Slice struct will be retrieved

Devoluções:

Nine Slice Struct (or -1 if the sprite doesn't exist)

Exemplo:

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

O código acima recupera a Nove Fatias struct de um sprite, habilita a Nove Fatias para ela e define os offsets do guia.
