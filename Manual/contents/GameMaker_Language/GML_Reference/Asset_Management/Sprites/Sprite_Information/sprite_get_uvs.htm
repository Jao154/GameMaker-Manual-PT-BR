sprite_get_uvs

Esta função retorna uma matriz com as coordenadas UV e outros dados para o texture da subimagem sprite na página texture. A função retorna um array com os 8 elementos a seguir:

  • [0] = esquerda
  • [1] = topo
  • [2] = direito
  • [3] = fundo
  • [4] = quantidade de pixels que o compilador asset aparou a partir do lado esquerdo sprites
  • [5] = quantidade de pixels que o compilador asset aparou do lado de cima do sprites
  • [6] = porcentagem normalizada de dados de pixels da largura original sprites que foi salva na página texture
  • [7] = porcentagem normalizada de dados de pixels da altura original sprites que foi salva na página texture

Esta matriz pode então ser usada em outras funções de desenho, particularmente quando se trabalha em 3D ou usando as funções primitivas 2D, bem como quando se trabalha com as funções Shader.

NOTA: Esta função não funcionará com o vetor sprites ou animação do esqueleto sprites.

Sintaxe:

sprite_get_uvs(sprite, subimage);

ArgumentoTipo Descrição
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

Devoluções

(1D, 8 elements)

Exemplo:

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

O código acima armazenará as coordenadas UV para o dado sprite em uma matriz local e então atribuirá os valores às variáveis de instância.
