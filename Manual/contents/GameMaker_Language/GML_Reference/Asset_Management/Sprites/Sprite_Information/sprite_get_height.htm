sprite_get_height

Com esta função você pode encontrar a altura da base sprite asset, sem transformações, em pixels.

Sintaxe:

sprite_get_height(index);

ArgumentoTipo Descrição
index The index of the sprite to find the height of.

Devoluções

Exemplo:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

O código acima verifica a altura do sprite como está na instância atual e se houver uma diferença entre isso e a base original sprite, ele redefine a escala do eixo y.
