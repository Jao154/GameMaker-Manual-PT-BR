sprite_set_offset

Esta função pode ser usada para definir a origem x e y de um sprite, e toma valores relativos com base na posição (0,0) sendo o canto superior esquerdo do sprite. A imagem seguinte ilustra isto:

NOTA Esta função afeta o ativo sprite para que todas as outras instâncias com este sprite tenham a mesma compensação.

Sintaxe:

sprite_set_offset(ind, xoff, yoff);

ArgumentoTipo Descrição
ind The index of the sprite to change the offset of.
xoff The x position of the origin.
yoff The y position of the origin.

Devoluções

Exemplo:

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

O código acima atribui o sprite indexado em "spr_Base" ao sprite indexado em "spr_Custom" e então usa os valores de compensação x e y de "spr_Base" para definir a origem do novo duende.
