sprite_flush

Com esta função, você pode remover a página texture dada para o dado sprite da memória texture. Você fornece o índice sprite (como definido ao criar o recurso sprite ) e a página texture para a qual ele é designado será removida da memória texture. Note que se a página texture for usada em outro lugar no room (por outra instância sprite ou por um fundo, etc...) você poderá ter um pequeno acerto de desempenho quando a página for imediatamente recarregada, portanto, deve-se ter cuidado ao usar esta função. Note que a função retornará -1 se a descarga não for suportada para o recurso escolhido, ou retornará 0 se tudo funcionou corretamente.

Sintaxe:

sprite_flush(ind)

ArgumentoTipo Descrição
ind The index (resource name) of the sprite asset to flush

Devoluções:

(-1 or 0)

Exemplo:

sprite_flush(spr_Player_Aura);

O código acima descarrega o sprite "spr_Player_Aura" da memória.
