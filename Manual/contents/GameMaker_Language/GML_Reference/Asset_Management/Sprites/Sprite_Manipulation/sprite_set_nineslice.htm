sprite_set_nineslice

Esta função é usada para aplicar uma estrutura de Nove Fatias a uma sprite. Você fornece o índice sprite e a Nove Fatias struct para aplicar, que pode ser recuperada usando sprite_nineslice_create() ou sprite_get_nineslice().

NOTA Esta função afeta o ativo sprite, portanto qualquer mudança que você fizer com esta função afetará todas as instâncias que estão usando este sprite e todas as instâncias futuras também.

Sintaxe:

sprite_set_nineslice(ind, nineslice);

Argumento Tipo Descrição
ind Ativo Sprite The index of the sprite to modify
nineslice Estrutura de nove fatias The Nine Slice struct to apply

Devoluções:

N/A

Exemplo:

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

O código acima cria uma nova Nine Slice struct, habilita-a e define os offsets para os guias. O struct é então aplicado a um sprite, alterando suas propriedades da Nove Fatias.
