path_get_length

Você pode usar esta função para obter o comprimento exato de um path em pixels. este não é um comprimento aproximado de ponto a ponto, mas sim um comprimento exato ao longo da forma do path, mesmo quando o path é liso com uma alta precisão curvada.

Sintaxe:

path_get_length(index);

ArgumentoTipo Descrição
index The index of the path to measure.

Devoluções:

Exemplo:

path_len = path_get_length(pth_AI);

Isto definirá "path_len" para o comprimento total do path indexado em "pth_AI" em pixels.
