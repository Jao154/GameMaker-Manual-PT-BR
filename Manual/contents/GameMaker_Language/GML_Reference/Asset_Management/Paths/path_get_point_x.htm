caminho_get_point_x

Esta função retornará a posição x (nas coordenadas room ) do ponto que você inserir para o path que você indexar. Se o ponto estiver fora do intervalo do path (ou seja: um path tem 8 pontos e você pede a posição x do ponto 10), então um valor de 0 será retornado.

Sintaxe:

path_get_point_x(index, n);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to check.
n Real The point number to check.

Devoluções:

Real

Exemplo:

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

O código acima verificará se uma instância está no final de um path. Se estiver, então escolherá um ponto aleatório no path e moverá a instância para esse ponto.
