caminho_get_kind

Os percursos podem ser lisos ou retos (um suave path calcula uma curva path em torno dos pontos definidores, enquanto que um reto path vai direto de um ponto a outro). Este pode ser usada para descobrir se a função path é suave (true) ou não (false).

Sintaxe:

path_get_kind(index);

ArgumentoTipo Descrição
index The index of the path to check.

Devoluções:

Exemplo:

path_kind = path_get_kind(pth_Patrol);

Isto definirá a variável "path_kind" para true ou false, dependendo do tipo de path o índice path dado é.
