path_get_x

Com esta função você pode obter a coordenada x de uma posição em qualquer path. 0 é o início do path, 1 é o fim do path, e qualquer coisa entre eles equivale a tanto através do path. Este não precisa ser um ponto de definição do path, pode estar em qualquer lugar nele.

Sintaxe:

path_get_x(ind, pos);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end).

Devoluções:

Real

Exemplo:

x = path_get_x(pth_WallWalk, 0.5);

Isto colocará o x da instância de chamada na coordenada x do ponto exatamente na metade do dado path.
