Caminhos

O GameMaker tem uma extensa coleção de funções e variáveis embutidas para usar paths dentro de seu jogo. Esta seção trata das básicas que são usadas quando se lida com paths que foram criadas anteriormente no Path Editor, mas existem outras funções que permitem criar seu próprio paths usando código, e manipulá-las de várias maneiras.

Existem as seguintes funções básicas que tratam dos caminhos:

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

Para ajudar com debugging paths, você tem uma função para atraí-lo também para a tela:

  • draw_path

Cada instância de um object também tem uma série de variáveis embutidas projetadas para facilitar o uso do paths, e estas variáveis estão listadas na seção seguinte:

  • Variáveis de caminho

Finalmente, é possível manipular o conteúdo de um path, bem como criar um novo paths quando o jogo estiver em execução. As funções que permitem estas coisas e muito mais estão disponíveis na seção seguinte:

  • Manipulação do caminho
