orientação_pista

Esta variável mantém a orientação atual do path que foi atribuído à instância quando a função path_start() foi chamada. Quando um path é criado, sua orientação é o padrão 0 graus, mas você pode definir este valor para qualquer coisa que desejar usando isto. Lembre-se de que no GameMaker (a menos que você esteja usando física) os ângulos são calculados no sentido anti-horário, portanto definindo a orientação path a 90° giraria o path para a esquerda.

Sintaxe:

path_orientation;

Devoluções:

Exemplo:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

O código acima duplica um código aleatório, pré-fabricado path asset, na variável "mypath". Este novo path é então iniciado e girado a 90°.
