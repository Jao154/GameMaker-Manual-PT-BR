Variáveis de caminho

As seguintes variáveis são incorporadas a todas as instâncias quando criadas e podem ser usadas para obter informações sobre um path, ou para alterar certas propriedades:

  • path_index
  • path_position
  • path_positionprevious
  • path_speed
  • path_scale
  • path_orientation
  • path_endaction

Observe que você pode encontrar mais informações sobre as variáveis de instância aqui: Variáveis de instância.
