path_append

Com este código você pode anexar um path a outro, unindo-os efetivamente, e para que isto funcione, ambos paths devem existir previamente. Em geral você gostaria de usar esta função em paths criada usando path_add(), pois se você a usar em um path asset, ela afetará permanentemente o path para instâncias no jogo desde o momento em que a função é usada até o final do jogo.

OBSERVAÇÃO Isto não removerá o path que está sendo anexado do jogo. Ele ainda está lá, só que agora não tem pontos nele e se você não quiser mais usá-lo, deve removê-lo com a função path_delete().

Sintaxe:

path_append(index, path);

ArgumentoTipo Descrição
index The new index of the path.
path The index of the path that will be appended to 'index'.

Devoluções:

N/A

Exemplo:

path_append(mypath, mppath);
path_delete(mppath);

O código acima anexa os dados path indexados na variável "mppath" ao path indexado na variável "mypath" e depois apaga o "mppath" da memória.
