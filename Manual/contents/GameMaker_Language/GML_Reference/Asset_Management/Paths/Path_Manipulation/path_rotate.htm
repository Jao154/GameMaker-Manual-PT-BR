caminho_rota

Você pode usar esta função para girar um determinado path ao redor de seu centro. Lembre-se de que no GameMaker (a menos que você esteja usando física) os ângulos são calculados no sentido anti-horário, portanto girar o path em 90 graus giraria o path para a esquerda. Esta função muda o recurso real path , e assim afetará permanentemente como o path é usado por todas as instâncias do jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função na duplicata asset (não esqueça de ligar para path_delete() no asset quando não for mais necessário).

OBSERVAÇÃO: Uma vez que um path tenha sido designado para uma instância usando o path_start() você pode usar a variável path_orientation para alterar também a rotação path.

Sintaxe:

path_rotate(index, angle);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to flip.
angle Real The angle of rotation degrees.

Devoluções:

N/A

Exemplo:

path_rotate(pth_Patrol, 90);

Isto giraria os 90 graus dados no sentido anti-horário.
