caminho_de_série_fechada

Esta função pode ser usada para flag um dado path como sendo aberta (falsa) ou fechada (verdadeira). Um path fechado tem seu ponto inicial conectado a seu ponto final, formando um loop, e um path aberto tem um início e fim definitivo, não conectado. Esta função muda o ativo real path , e assim afetará permanentemente como o path é usado por todas as instâncias do jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função no asset duplicado (não esqueça de ligar para path_delete() no asset quando não for mais necessário).

Sintaxe:

path_set_closed(index, closed);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to change.
closed Booleano Whether the path is closed (true) or not (false).

Devoluções:

N/A

Exemplo:

path_set_closed(pth_Patrol, true);

Isto fará com que o dado path seja um fechado path.
