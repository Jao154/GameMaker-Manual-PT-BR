caminho_tipo_de_série

Esta função pode ser usada para definir o tipo de path que você deseja que o recurso especificado seja. Esta pode ser uma linha reta path (set to 0) ou uma linha reta path (set to 1), neste caso a precisão path também deve ser levada em conta (a precisão pode ser ajustado também usando path_set_precision()). Esta função muda o recurso real path, e assim afetará permanentemente como o path é usado por todas as instâncias do jogo a partir do momento a função é usada até o final do jogo. Se isso não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então chame essa função no duplicado asset (não esqueça de ligar para path_delete() no asset quando não for mais necessário).

Sintaxe:

path_set_kind(index, val);

ArgumentoTipo Descrição
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

Devoluções:

Exemplo:

path_set_kind(pth_Patrol, true);

Isto fará com que o dado path seja um "suave" path.
