caminho_duplicar_duplicar

Esta função toma um path e o copia em um novo path. O novo path é criado no processo, e seu índice é devolvido para ser usado em todas as chamadas posteriores para usar este novo path.

Sintaxe:

path_duplicate(index);

ArgumentoTipo Descrição
index The index of the existing path to duplicate.

Devoluções:

Path Asset

Exemplo:

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

O código acima escolhe um dos quatro recursos path e o duplica, armazenando o índice do novo path na variável "mypath".
