caminho_ponto_de_apagamento

Com esta função você pode remover um ponto do especificado path. Path pontos são numerados a partir de 0, portanto o primeiro ponto é sempre o 0º ponto e o último path ponto será path_get_number() - Se você chamar esta função em path asset, então a remoção do ponto será permanente e todas as instâncias designadas a path no futuro não terão o ponto que foi removido. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função no asset duplicado (não esqueça de ligar para path_delete() no asset quando não for mais necessário).

Sintaxe:

path_delete_point(ind, n);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to delete a point from.
n Real The defining point to delete.

Devoluções:

N/A

Exemplo:

var _col = true;
while (_col)
{
    var _num = path_get_number(mypath) - 1;
    if (_num > 2)
    {
        var _x1 = path_get_point_x(my_path, _num);
        var _y1 = path_get_point_y(my_path, _num);
        var _x2 = path_get_point_x(my_path, _num -1);
        var _y2 = path_get_point_y(my_path, _num -1);
        if (collision_line(_x1, _y1, _x2, _y2, obj_Wall, true, false))
        {
            path_delete_point(mypath, _num);
            _num -= 1;
        }
        else _col = false;
    }
    else _col = false;
}

O código acima gera um loop que verifica os pontos de trás para frente do final para uma colisão com o dado object. Se for encontrada uma colisão, um ponto path é excluído, até que não haja colisão, ou o path tem apenas 2 pontos.
