caminho_ponto_de_inserção

Com esta função você pode inserir um novo ponto em um path (o path pode ter sido criado no editor path ou através de código usando path_add()). O ponto será adicionado no path antes do ponto "n" que é especificado na função.

Sintaxe:

path_insert_point(index, n, x, y, speed);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to insert the point into.
n Real The defining point to insert the new point BEFORE.
x Real The x coordinate (relative to the path) of the new point.
y Real The y coordinate (relative to the path) of the new point.
speed Real The speed factor of the point.

Devoluções:

N/A

Exemplo:

path_insert_point(mypath, 0, 50, 50, 100);

Isto inserirá um ponto logo no início do path indexado na variável "mypath", a (50,50), com um fator de velocidade de 100%.
