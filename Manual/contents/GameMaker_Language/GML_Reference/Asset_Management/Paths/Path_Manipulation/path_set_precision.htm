caminho_set_precisão

Com esta função você pode mudar a "suavidade" de um path. Este valor deve estar entre 1 e 8, com um valor baixo criando bordas mais retas com curvas mais nítidas entre os pontos, enquanto um valor mais alto arredondará os pontos e tornará o path muito mais "curvado". Note que esta função não terá efeito visível se o path não tiver sido configurado para suavizar no editor do path ou usando a função path_set_kind(). Esta função muda o ativo real path , e assim afetará permanentemente como o path é usado por todas as instâncias no jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função no asset duplicado (não se esqueça de ligar para path_delete() no asset quando não for mais necessário).

Sintaxe:

path_set_precision(index, prec);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to change.
prec Real The precision of the path. Must be an integer between 1 and 8.

Devoluções:

N/A

Exemplo:

path_set_precision(pth_Patrol, 2);

Isto determinará a precisão do caminho dado a 2.
