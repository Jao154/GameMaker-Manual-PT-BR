path_shift

Com esta função você pode deslocar um path ao longo do eixo horizontal e vertical. Note que isto não terá nenhum efeito visível no jogo se a instância que segue o path sendo deslocada o segue relativamente (em vez de absolutamente) como o caminho A posição é então desconsiderada. Esta função muda o ativo real path , e assim afetará permanentemente como o path é usado por todas as instâncias no jogo desde o momento em que a função é usada até o final do jogo. Se isto for não o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, depois ligue para esta função no duplicado asset (não se esqueça de ligar para path_delete() no site asset quando não for mais necessário).

Sintaxe:

path_shift(index, xshift, yshift);

ArgumentoTipo Descrição
index The index of the path to change.
xshift The number of horizontal pixels to shift the path. Negative=left, positive=right.
yshift The number of vertical pixels to shift the path. Negative=up, positive=down.

Devoluções:

Exemplo:

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

O código acima duplica um path asset e armazena seu valor de identificação em uma variável. Esta variável é então usada para mudar a posição path duplicada e então a instância inicia o path.
