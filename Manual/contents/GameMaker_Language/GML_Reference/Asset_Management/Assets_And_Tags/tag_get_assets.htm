tag_get_assets

Com esta função você pode recuperar os nomes de todos os assets que receberam a tag ou tags. Você fornece uma única tag string ou uma matriz, onde cada item da matriz é uma tag string. A função irá retornar um array onde cada entrada é o nome (como um string) do asset com a tag dada. Se você precisar do índice único para o asset, então você pode usar a função asset_get_index() junto com o nome devolvido. Se não houver assets com a(s) etiqueta(s) dada(s), ou se houver um erro (por exemplo: as etiquetas dadas não existem), então uma matriz vazia será devolvida.

Sintaxe:

tag_get_assets(tags);

Argumento Tipo Descrição
tags Cordão ou matriz de cordas A single asset tag string or an array with various asset tags.

Devoluções:

Array

Exemplo:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

O código acima cria uma lista, depois recupera os nomes de todos os assets com a etiqueta "background". Ele loops através da matriz de nomes retornados, verificando se algum deles é sprite assets, e se eles são então o valor de índice único para o asset é adicionado à lista para uso futuro.
