asset_get_tags

Com esta função você pode recuperar todas as tags atribuídas a um asset a partir do navegador Asset. Você fornece ou o nome asset (como um string) ou o índice asset, e a função retornará um conjunto de tags para isso asset. Se nenhuma tag for encontrada ou se houver um erro (ou seja: o nome string dado não existe), então a matriz retornada estará vazia. Se você fornecer um valor de índice asset, então você precisará fornecer o argumento de tipo opcional asset (uma constante), pois assets de diferentes tipos pode ter o mesmo índice, mesmo que eles não possam ter o mesmo nome. Os tipos disponíveis asset estão listados na tabela abaixo:

Constante Descrição
asset_object O nome dado refere-se a um objeto.
asset_sprite O nome dado refere-se a um duende.
asset_sound O nome dado refere-se a um som.
asset_room O nome dado refere-se a um room.
asset_tiles O nome dado refere-se a um conjunto de azulejos.
asset_path O nome dado refere-se a um caminho.
asset_script O nome dado refere-se a um script.
asset_font O nome dado refere-se a uma fonte.
asset_timeline O nome dado refere-se a uma linha de tempo.
asset_shader O nome dado refere-se a um sombreador.
asset_animationcurve O nome dado refere-se a uma Curva de Animação.
asset_sequence O nome dado refere-se a uma sequência.

Sintaxe:

asset_get_tags(name_or_index, [asset_type]);

Argumento Tipo Descrição
name_or_index Cordão ou Ativo The name of the asset (a string) or its index.
[asset_type] Tipo de ativo Constante OPTIONAL! The type of asset to get the tags for, only used when supplying an index for the first argument.

Devoluções:

Array

Exemplo:

my_tags = asset_get_tags(object_get_name(object_index));

O código acima recuperará todas as tags atribuídas ao object a partir do qual a instância que executa o código foi criada.
