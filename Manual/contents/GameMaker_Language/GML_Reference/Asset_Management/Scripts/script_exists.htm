script_existe

Esta função retornará true ou false dependendo se a função script ou script com o índice dado existe. Note que este não é um string, mas sim o nome asset que contém o índice único para cada script (como apareceria no IDE) ou a função script nomeada, como definida dentro do script asset (note que isto não funcionará para as variáveis do método). Para maiores informações sobre scripts, consulte o editor do Script .

Sintaxe:

script_exists(scr);

Argumento Tipo Descrição
scr Ativo Script The script index that you want to check.

Devoluções:

Boolean

Exemplo:

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

O exemplo acima acrescenta duas funções script e um valor em uma matriz, depois procede para obter um número aleatório e usa-o para escolher uma função script a ser executada, a menos que o -1 seja escolhido, caso em que nada acontecerá.
