script_execute_ext

Esta função funciona de forma semelhante à função script_execute() somente você pode fornecer uma matriz que contém os argumentos necessários para a função/script que está sendo chamada. Você também pode fornecer dois argumentos opcionais à função para especificar um offset no array para obter os argumentos, assim como o número de argumentos a serem usados do array (este deve ser um máximo de comprimento do array - offset).

Sintaxe:

script_execute_ext(scr, array_args, [offset], [num_args]);

Argumento Tipo Descrição
scr Ativo Script ou Função Script The function/script that you want to call
array_args Array The array containing the arguments for the function/script
[offset] Real OPTIONAL A compensação na matriz de argumentos
[num_args] Real OPTIONAL O número de argumentos a usar (a partir da compensação)

Devoluções:

Variable (Will depend on the return value from the function/script being called)

Exemplo:

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

O código de exemplo acima usará script_execute_ext para chamar a função dada, fornecendo uma série de quatro argumentos, mas usando apenas um deles de forma aleatória.

NOTA: Esta função não pode ser usada com funções embutidas.
