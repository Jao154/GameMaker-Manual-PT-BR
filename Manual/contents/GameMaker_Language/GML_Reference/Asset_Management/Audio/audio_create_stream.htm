audio_create_stream

Com esta função você pode criar um novo índice de som que pode então ser usado nas funções de áudio regulares para transmitir áudio diretamente de uma fonte de arquivo OGG externa. A função requer o nome do arquivo (que pode ser um arquivo incluído, por exemplo) e retornará o novo índice de som para uso. Note que depois que você não precisar mais do som, você deve chamar a função audio_destroy_stream() com o índice de som para removê-lo da memória, caso contrário, você poderá ter um vazamento de memória que irá diminuir a velocidade e, eventualmente, quebrar seu jogo.

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_create_stream(filename);

Argumento Tipo Descrição
filename Cordão Path to the file (OGG only) to stream the audio from.

Devoluções:

Sound Asset

Exemplo:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

O código acima cria um novo índice de som na variável "snd" do arquivo dado, depois toca este som.
