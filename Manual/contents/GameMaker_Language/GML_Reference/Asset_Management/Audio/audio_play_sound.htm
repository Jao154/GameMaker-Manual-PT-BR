audio_play_sound

Com esta função, você pode jogar qualquer recurso sonoro em seu jogo. Você fornece o índice de som e lhe atribui uma prioridade, que é então usada para determinar como os sons são tratados quando o número de sons jogados está acima do limite estabelecido pela função audio_channel_num(). Sons de prioridade mais baixa serão interrompidos em favor de sons de prioridade mais alta, e o valor de prioridade pode ser qualquer número real (o valor real é arbitrário, e pode ser de 0 a 1 ou 0 a 100, pois o GameMaker os priorizará da mesma forma). Observe que ao lidar com prioridade, quanto maior o número, maior será a prioridade, de modo que um som com prioridade 100 será favorecido em relação a um som com prioridade 1. O terceiro argumento é para fazer o som loop e configurá-lo para true fará o som loop até que seja parado e configurá-lo para false tocará o som apenas uma vez.

Esta função também retornará um número de índice único para o som que está sendo reproduzido, que pode então ser armazenado em uma variável para que você possa pausá-lo ou pará-lo com as funções apropriadas. Isto significa que se você tiver múltiplas instâncias do mesmo som sendo reproduzido a qualquer momento, você pode direcionar apenas uma instância desse som para lidar com o uso das funções de áudio.

Sintaxe:

audio_play_sound(index, priority, loop, [gain], [offset], [pitch], [listener_mask]);

Argumento Tipo Descrição
index Bens sólidos The index of the sound to play.
priority Real Set the channel priority for the sound.
loop Booleano Sets the sound to loop or not.
gain Real OPTIONAL Valor para o ganho.
offset Real OPTIONAL O tempo (em segundos) para definir o ponto de partida. Valores maiores do que o comprimento do som dado são ignorados.
pitch Real OPTIONAL O multiplicador de passo (padrão 1).
listener_mask Real OPTIONAL Os dados de bitmask a serem definidos para o som. No alvo HTML5 isto não terá efeito, pois o alvo não suporta mais que um ouvinte.

Devoluções:

Sound Instance ID

Exemplo 1:

if health <= 0
{
    lives -= 1;
    audio_play_sound(snd_PlayerDead, 10, false);
}

O código acima verifica a variável global "saúde" e se ela for menor ou igual a 0, removerá 1 da variável global "vidas" e tocará um som.

Exemplo 2:

if bbox_left > room_width
{
    audio_play_sound(snd_Goodbye, 10, false, 1.1, 0, 2);
}

O código acima verifica se a instância está além do limite direito do room e toca um som com um ganho de 1,1 e um tom de 2 (o dobro do tom normal).
