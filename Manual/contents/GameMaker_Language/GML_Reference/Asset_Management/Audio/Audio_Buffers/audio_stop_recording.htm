audio_stop_recording

Esta função interromperá a gravação no canal do gravador em questão (o índice do canal é retornado quando você chama a função audio_start_recording()). Quando você parar de gravar, nenhum outro evento assíncrono de gravação de áudio será acionado para o canal do gravador em questão, portanto, você normalmente usaria esta função no evento assíncrono real para garantir que você tenha capturado todos os dados.

NOTA A maioria das plataformas suporta a gravação de áudio de alguma forma, mas isso não significa que todos os dispositivos o permitam, mesmo que a plataforma o permita, então você deve sempre verificar se o audio_get_recorder_count() retorna um valor maior que 0 para verificar se os dispositivos de gravação estão disponíveis antes de utilizar o restante das funções de gravação.

Sintaxe:

audio_stop_recording(channel_index);

Argumento Tipo Descrição
channel_index Real The index of the recorder channel to stop.

Devoluções:

N/A

Exemplo:

audio_stop_recording(audio_channel);

O código acima diz ao GameMaker para parar de gravar no índice de canais de áudio dado.
