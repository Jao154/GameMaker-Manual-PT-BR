audio_queue_sound

Esta função adicionará os dados de um buffer no áudio queue que você criou anteriormente usando a função audio_create_play_queue(). Você especifica o índice queue para adicionar, e o ID buffer para obter os dados, assim como a posição (offset) dentro do buffer e o número de bytes para adicionar. Uma vez adicionado o áudio de um buffer a um queue, o buffer não pode ser excluído até que você tenha liberado primeiro o queue usando a função audio_free_play_queue()e as propriedades buffer devem corresponder às do queue ao qual você está adicionando o som.

Sintaxe:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Argumento Tipo Descrição
queueIndex Áudio Queue ID The index of the queue to add to.
bufferId Identificação do tampão The buffer to add to the queue.
bufferOffset Real The offset within the source buffer to start from.
bufferLength Real The length of the buffer (the number of the bytes in the buffer).

Devoluções:

N/A

Exemplo:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

O código acima seria chamado no evento assíncrono de gravação de áudio e atribui alguns dados gravados a um buffer, que é então adicionado a um áudio queue. Isto é então reproduzido.
