audio_resume_sound

Com esta função você pode retomar qualquer som que esteja em pausa no momento (depois de usar a função audio_pause_sound()). O som pode ser uma única instância de um som (o índice para sons individuais que estão sendo reproduzidos pode ser armazenado em uma variável quando se usa o audio_play_sound() ou audio_play_sound_at() funções) ou um som asset, em cujo caso todas as instâncias do som dado serão reiniciadas.

Sintaxe:

audio_resume_sound(index);

Argumento Tipo Descrição
index Bens sólidos ou identificação de instância sonora The index of the sound to resume.

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

O código acima verifica se a tecla do teclado "P" está pressionada e se detecta uma, define a variável global "Pause" para true ou falsa e então ou pausa o som indexado na variável "snd_Waterfall" ou retoma o som de seu estado de pausa.
