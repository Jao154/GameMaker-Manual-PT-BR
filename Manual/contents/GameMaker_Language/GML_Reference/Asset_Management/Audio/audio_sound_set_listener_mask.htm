audio_sound_set_listener_mask

Esta função pode ser usada para definir a máscara de bits para um som, de modo que ele seja tocado apenas pelos ouvintes especificados. Você pode criar uma bit-máscara usando a função audio_get_listener_info() e depois usando o bitwise or ("|") para criar uma máscara personalizada com aqueles ouvintes dos quais você requer o som para tocar, e depois aplicar esta máscara personalizada ao som depois de tê-la tocado. Esta máscara sobreporá a máscara padrão ou aquela que você possa ter definido usando a função audio_set_listener_mask().

Esta função só pode ser usada em um som após ter sido ajustada para tocar e requer a identificação do som como retornado pela função usada para tocar o som.

Sintaxe:

audio_sound_set_listener_mask(soundID, mask);

Argumento Tipo Descrição
soundID Identificação da instância sonora The unique ID of the sound to set the mask of
mask Real The bitmask data to set for the sound

Devoluções:

N/A

Exemplo:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

O código acima toca um som e, em seguida, define o(s) ouvinte(s) para tocar a partir do uso dos dados da máscara armazenados em uma variável global.
