audio_set_master_gain

Com esta função você pode definir o valor absoluto para o volume global de todos os sons e músicas para um ouvinte específico. O índice padrão do ouvinte é 0, mas você pode usar a função audio_get_listener_info() para obter os diferentes índices disponíveis para a plataforma alvo. O valor do ganho é baseado em uma escala linear de 0 (silêncio) a 1 (volume completo) e afetará o volume relativo de todos os sons e músicas reproduzidos de dentro de seu jogo através daquele ouvinte.

Sintaxe:

audio_set_master_gain(listenerIndex, gain);

Argumento Tipo Descrição
listenerIndex ID do ouvintereal ou de áudio The index of the listener to set the gain on.
gain Real Value for the global volume (0 to 1).

Devoluções:

N/A

Exemplo:

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

O código acima loops através dos ouvintes disponíveis e, em seguida, estabelece seu ganho principal para 0,75.
