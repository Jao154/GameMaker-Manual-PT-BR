audio_stop_all

Esta função interromperá todos os sons que estão tocando atualmente.

Sintaxe:

audio_stop_all();

Devoluções:

Exemplo:

if !global.SFX
{
    audio_stop_all();
}

O código acima verifica a variável global "SFX" e se ela retornar false, ela parará todos os sons que estão tocando atualmente.
