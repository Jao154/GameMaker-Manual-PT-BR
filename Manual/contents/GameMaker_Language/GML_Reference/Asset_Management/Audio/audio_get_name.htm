audio_get_name

Esta função retornará o nome de um determinado recurso de áudio como um string. O valor "índice" pode ser o do próprio recurso (como visto no navegador Asset ) ou o valor de identificação único que é dado quando você toca o som usando, por exemplo, audio_play_sound(). Observe que o string retornado não é o mesmo que o ID do recurso e não pode ser usado para acessar o recurso em si, portanto, só deve ser usado para exibição ou verificação de erros.

Sintaxe:

audio_get_name(index);

ArgumentoTipo Descrição
index The index of the sound to check.

Devoluções:

Exemplo:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

O código acima reproduz um som aleatório escolhido a partir de três recursos sonoros diferentes e depois mostra uma mensagem de debug com seu nome.
