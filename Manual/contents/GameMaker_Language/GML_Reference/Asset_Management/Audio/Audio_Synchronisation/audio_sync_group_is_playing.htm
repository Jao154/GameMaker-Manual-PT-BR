audio_sync_group_is_playing

Esta função pode ser usada para verificar se algum áudio em um grupo sincronizado está sendo reproduzido. É necessário fornecer o ID do grupo de sincronização conforme retornado pela função audio_create_sync_group().

Esta funcionalidade não está disponível para a plataforma alvo HTML5.

Sintaxe:

audio_sync_group_is_playing(group_index);

Argumento Tipo Descrição
group_index ID do Grupo de Sincronização de Áudio The group index to check.

Devoluções:

Boolean

Exemplo:

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

O código acima verifica se o grupo de sincronização group_one está tocando no momento e se estiver, o grupo está parado.
