audio_channel_num

Com esta função você pode definir quantos canais de áudio estão disponíveis para a reprodução de áudio no GameMaker. Isto basicamente significa que você dá o número de sons simultâneos que podem ser reproduzidos a qualquer momento, e se o número exceder a quantidade, aqueles sons com prioridade menor são interrompidos para liberar um canal para os sons com prioridade maior. Você pode usar esta função para otimizar seu jogo para dispositivos como quanto menor o número de canais para áudio melhor o desempenho, mas tenha em mente que isto também pode causar um corte notável de certos sons se muitos forem reproduzidos ao mesmo tempo.

Sintaxe:

audio_channel_num(num);

ArgumentoTipo Descrição
num Number of available audio channels (default is 128).

Devoluções:

Exemplo:

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

O código acima verifica a plataforma em que o jogo está sendo executado e muda o número de canais de som disponíveis para aumentar o desempenho.
