audio_get_listener_info

Esta função criará um mapa DS e o preencherá com informações para o ouvinte em questão.

NOTA Você é responsável pela destruição do retorno DS Map usando a função apropriada.

O mapa DS conterá as seguintes chaves:

  • "name" - O nome do ouvinte, como um string, sendo "default" o nome padrão do ouvinte na maioria das plataformas alvo
  • "máscara" - A máscara de bits para o ouvinte
  • "index" - O valor único do índice do ouvinte

O valor da máscara pode ser usado para definir um som ou emissor para tocar de múltiplos ouvintes ao mesmo tempo, simplesmente usando o bitwise or "|" para gerar uma máscara para o som (veja o código de exemplo abaixo), enquanto o índice é usado para definir as propriedades como posição ou velocidade para um determinado ouvinte usando funções como audio_listener_set_position().

Sintaxe:

audio_get_listener_info(num);

Argumento Tipo Descrição
num ID do ouvintereal ou de áudio The listener number to get the data for.

Devoluções:

DS Map ID

Exemplo:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

O código acima verifica o número de ouvintes disponíveis e então loops através deles recebe seus bits de máscara, que são então combinados para criar uma única máscara de bit que é aplicada ao ouvinte global.
