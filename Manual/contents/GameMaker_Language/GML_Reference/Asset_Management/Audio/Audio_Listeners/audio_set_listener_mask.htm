audio_set_listener_mask

Ao utilizar vários ouvintes em um sistema, você pode definir a máscara de bits para um som e fazer com que ele seja ouvido apenas pelo ouvinte sinalizado. Entretanto, você também pode definir a máscara global usando esta função e todos os sons reproduzidos normalmente serão ouvidos pelos ouvintes sinalizados por esta máscara, sem a necessidade de definir a máscara para cada som individualmente.

Sintaxe:

audio_set_listener_mask(mask);

Argumento Tipo Descrição
mask Real The bit-mask data to set for the listeners

Devoluções:

N/A

Exemplo:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

O código acima verifica o número de ouvintes disponíveis e então loops através deles recebe seus bits de máscara, que são então combinados para criar uma única máscara de bit que é aplicada ao ouvinte global.
