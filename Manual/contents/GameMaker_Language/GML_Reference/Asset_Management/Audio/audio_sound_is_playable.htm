audio_sound_is_playable

Esta função pode ser usada para verificar se o índice de som dado pode ser reproduzido atualmente. Isto é necessário devido às diferentes formas de reprodução de som streamed e unstreamed na plataforma de destino HTML5, e retornará true se o som puder ser reproduzido e false se não puder. Note que em todas as outras plataformas além do HTML5, a função sempre retornará true.

Sintaxe:

audio_sound_is_playable(index);

Argumento Tipo Descrição
index Bens sólidos The index of the sound to check.

Devoluções:

Boolean

Exemplo:

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

O código acima tocará o som "snd_Shot" se ele for tocável.
