ds_grid_multiply_region

Com esta função é possível especificar uma região da grade na qual se pode multiplicar cada valor de célula por um determinado valor.

NOTA Esta função só funcionará com números reais, não strings.

Sintaxe:

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

Argumento Tipo Descrição
index DS Grid ID The index of the grid.
x1 Real The x position of the left of the region in the grid.
y1 Real The y position of the top of the region in the grid.
x2 Real The x position of the right of the region in the grid.
y2 Real The y position of the bottom of the region in the grid.
val Real The value to multiply with the region cells.

Devoluções:

N/A

Exemplo:

ds_grid_multiply_region(mygrid, 5, 5, 10, 10, 2);

O código acima tomará todos os valores encontrados dentro da área de grade retangular definida e multiplicará cada um deles por 2.
