ds_grid_get_disk_min

Esta função pode ser usada para encontrar o valor mínimo para todas as células encontradas dentro da área definida do disco de uma grade, como mostrado na imagem abaixo:

Sintaxe:

ds_grid_get_disk_min(index, xm, ym, r);

ArgumentoTipo Descrição
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.

Devoluções:

Real or String

Exemplo:

val = ds_grid_get_disk_min(grid, 5, 5, 2)

O código acima definirá a variável "val" para o valor mínimo contido dentro do disco dado da grade DS indexada na variável "grid".
