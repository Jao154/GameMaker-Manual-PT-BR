ds_map_secure_save_buffer

Esta função salvará um mapa DS previamente criado em um buffer. Você fornece o valor de identificação do mapa DS (como retornado pela função ds_map_create()) e o ID do buffer para escrever para (como retornado pela função buffer_create()). Observe que se o mapa DS que está sendo salvo contém uma matriz, esta será convertida em uma lista DS quando for salvo.

IMPORTANTE! O arquivo seguro de mapa DS salvo só pode ser carregado no dispositivo que o criou, e se você tentar carregar um arquivo que foi salvo em um dispositivo diferente, então ele não funcionará.

NOTA: Esta função não é suportada em HTML5 e UWP.

Sintaxe:

ds_map_secure_save_buffer(filename);

Argumento Tipo Descrição
id DS Map ID The DS map ID value.
buffer Identificação do tampão The buffer to save to.

Devoluções:

N/A

Exemplo:

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

O código acima criará um buffer e um mapa DS, depois povoará o mapa com alguns valores e o escreverá no buffer antes de deletar o mapa.
