ds_map_secure_load_buffer

Esta função irá carregar um mapa DS salvo em segurança a partir de um buffer. Você deve ter carregado previamente o buffer na memória (usando buffer_load()) e, em seguida, passando isso para esta função, retornará um mapa DS preenchido com o conteúdo do buffer. Note que o buffer deve ter sido criado usando a função ds_map_secure_save_buffer() para que isto funcione corretamente, e também note que se o mapa DS sendo carregado contivesse uma matriz, esta seria convertida em uma lista DS em vez de ser carregada.

IMPORTANTE! Uma das características de um arquivo seguro salvo é que ele está travado no dispositivo em que foi criado, de modo que você não pode carregar um arquivo salvo em um dispositivo em um projeto em execução em outro dispositivo.

NOTA: Esta função não é suportada em HTML5 e UWP.

Sintaxe:

ds_map_secure_load_buffer(buffer);

Argumento Tipo Descrição
buffer Identificação do tampão The buffer ID of the buffer to load the map data from.

Devoluções:

DS Map ID

Exemplo:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

O código acima carregará um mapa DS salvo com segurança a partir de um buffer e armazenará seu valor de índice em uma variável para uso futuro.
