ds_list_find_index

Com esta função você pode verificar a lista dada para um valor e a posição dentro da lista para esse valor será devolvida. Note que se houver mais de uma entrada na lista com o mesmo valor, a posição de qualquer uma delas poderá ser retornada, e que se o valor não existir, então -1 será retornado.

Sintaxe:

ds_list_find_index(id, val);

Argumento Tipo Descrição
id DS List ID The id of the list to use.
val Variável The value to find.

Devoluções:

Real

Exemplo:

pos = ds_list_find_index(list, "Player1");

O código acima verifica a lista indexada na variável "lista" para o valor "Player1" e armazena a posição retornada na variável "pos".
