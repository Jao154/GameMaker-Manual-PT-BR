ds_existe

Com esta função você pode verificar se existe uma estrutura de dados do tipo em questão. Você fornece o valor "índice" (como mantido em uma variável) e o "tipo" de DS, que pode ser qualquer uma das constantes listadas abaixo, e a função retornará true se a estrutura de dados existir e false caso contrário.

Constante tipo DS
Constante Descrição
ds_type_map Uma estrutura de dados cartográficos
ds_type_list Uma estrutura de dados da lista
ds_type_stack Uma estrutura de dados em pilha
ds_type_grid Uma estrutura de dados em grade
ds_type_queue Uma estrutura de dados em fila
ds_type_priority Uma estrutura de dados prioritária

NOTA Não é possível usar esta função para verificar o tipo de estrutura de dados, pois o mesmo número de índice pode ser usado por múltiplas estruturas de dados de diferentes tipos.

Sintaxe:

ds_exists(ind, type);

Argumento Tipo Descrição
ind Qualquer DS ID The variable index to check for the data structure
type Constante tipo DS The type of data structure to check for (see the list of constants above)

Devoluções:

Boolean

Exemplo:

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

O código acima verifica a variável (previamente rubricada) "ai_grid" para ver se ela indexa uma estrutura de dados do tipo DS grid, e se não indexa, então cria uma e armazena seu índice na variável.
