gxc_stop_movie_recording

Esta função é usada para parar uma gravação GXC que foi previamente iniciada com gxc_start_movie_recording(). Também submete a gravação ao URL especificado nos argumentos, porém o comportamento é diferente, dependendo do tipo de string especificado:

  • Se você especificar um nome de arquivo, a gravação será salva com esse nome em sua pasta Downloads
  • Se você especificar um URL, a gravação será carregada para esse URL

O método URL deve ser usado quando a gravação precisar ser carregada no GXC, e o arquivo deve ser salvo localmente apenas para verificar se a gravação foi bem sucedida.

Sintaxe:

gxc_stop_movie_recording(filename);

Argumento Tipo Descrição
filename Cordão The file name or URL where the recording is to be saved or uploaded

Devoluções:

N/A

Exemplo:

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

O código acima usa a biblioteca API do GXC para carregar uma pontuação para o desafio atualmente ativo, e quando uma resposta é recebida em seu método de retorno de chamada, ele recupera a URL de carregamento para o vídeo e carrega a gravação para ele.
