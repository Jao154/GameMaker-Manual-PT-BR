buffer_base64_decode_ext

Com esta função você pode decodificar uma base64 codificada string (criada usando o buffer_base64_encode() buffer Ao contrário da função buffer_base64_decode(), isto não criará um buffer para você, mas sim você já deveria ter criado o buffer (veja buffer_create()), cuja identificação você usaria então com esta função. O "offset" é o posição dentro do buffer para decodificar o dado string (em bytes).

Sintaxe:

buffer_base64_decode_ext(buffer, string, offset);

ArgumentoTipo Descrição
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

Devoluções:

Exemplo:

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

O código acima criará um buffer e armazenará a identificação única para ele na variável "buff", depois abrirá um arquivo ini e lerá um string a partir dele na variável local "str". Este string é então decodificado no recém-criado buffer antes fechando novamente o arquivo ini.
