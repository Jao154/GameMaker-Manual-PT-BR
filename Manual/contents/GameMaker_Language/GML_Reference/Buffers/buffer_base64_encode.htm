base64_código_de_tampão

Esta função converterá os dados do dado buffer em um formato base64 codificado string. Este é um esquema de codificação comumente utilizado para qualquer mídia que precise ser armazenada ou transferida pela Internet como texto, e torna o produzem ilegível para o olho humano. Para utilizar isto, é necessário especificar um já criado buffer, o valor de offset (que é o ponto dentro do buffer no qual se deseja iniciar a codificação), bem como o tamanho, em bytes, da memória buffer a ser codificada.

Sintaxe:

buffer_base64_encode(buffer, offset, size);

ArgumentoTipo Descrição
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

Devoluções:

Exemplo:

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

O código acima criará a codificação dos dados completos armazenados no buffer indexado pela variável "buff", e armazenará o retorno string na variável local "b_str".
