buffer_load_ext_ext

Esta função irá carregar os dados buffer que foram salvos anteriormente usando o buffer_save() funciona em um buffer existente. Você dá o ID do buffer criado anteriormente para carregar, depois o arquivo buffer salvo para carregar, e finalmente o offset desde o início do buffer (em bytes) para o qual você deseja carregar os dados.

Por favor, leia a página buffer_load() para notas específicas da plataforma.

Sintaxe:

buffer_load_ext(buffer, filename, offset);

ArgumentoTipo Descrição
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

Devoluções:

Exemplo:

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

O código acima obterá primeiro a posição do final do buffer indexado na variável "player_buffer" e depois carregará os dados do dado para aquela posição (note que este exemplo só funcionará com os tipos "grow" ou "wrap" buffer ).
