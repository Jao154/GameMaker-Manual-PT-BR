superfície_superfície_de_tampão

Com esta função, você pode escrever informações de uma buffer para uma determinada superfície. Tanto o buffer que está sendo lido quanto a superfície que está sendo escrita devem ter sido criados anteriormente, e você pode fornecer um offset no buffer para começar a ler a partir dele. Note que a leitura sempre começará no início do buffer mais o valor do offset e não na posição atual de busca mais o valor do offset.

NOTA: Esta função foi alterada a partir da atualização do GameMaker 2.3.1 em diante. As versões anteriores utilizavam a constante buffer_surface_copy, que agora está obsoleta.

Sintaxe:

buffer_set_surface(buffer, surface, offset);

ArgumentoTipo Descrição
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

Devoluções:

Exemplo:

buffer_set_surface(buff, application_surface, 0);

O código acima copiará todos os dados armazenados no buffer indexado na variável "buff" para a superfície da aplicação sem compensação.
