buffer_existe

Esta função pode ser usada para verificar uma variável para ver se ela possui ou não um valor de ID buffer válido. Se ela fizer a função retornará true caso contrário retornará false.

Sintaxe:

buffer_exists(buffer)

ArgumentoTipo Descrição
buffer The index of the buffer to check.

Devoluções:

Exemplo:

if buffer_exists(buff)
{
    buffer_delete(buff);
}

O código acima verifica se a variável "buff" possui um ID buffer e se possui, o buffer é excluído.
