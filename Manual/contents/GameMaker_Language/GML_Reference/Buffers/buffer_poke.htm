ponto_de_tampão

Com o buffer_write() você pode escrever os dados para a função buffer na posição "procurar" atual, com cada dado avançando esta posição pelos bytes sendo escritos ou lidos. Entretanto, pode ser necessário alterar um dado dado sem querer alterar a posição atual de "busca", e é quando você usaria esta função. Você simplesmente fornece a função com um índice buffer, e então a posição de offset do início buffer (em bytes) dentro daquele buffer para escrever, assim como o tipo de dados e o valor a ser escrito.

Sintaxe:

buffer_poke(buffer, offset, type, value);

Argumento Tipo Descrição
buffer Identificação do tampão The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to write the given data to.
type Constante do tipo de dados tampão The type of data that is to be written to the buffer (see the list of constants here).
value Variável The data to add to the buffer, in accordance with the type specified.

Devoluções:

N/A

Exemplo:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

O código acima adicionará o valor do componente azul da cor utilizada para a mistura da imagem no buffer indexado na variável "buff", na terceira posição no buffer e como um valor de 8bit não assinado.
