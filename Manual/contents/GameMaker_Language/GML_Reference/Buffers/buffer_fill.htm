buffer_fill

Esta função pode ser usada para preencher um buffer previamente criado com um determinado tipo e valor de dados. Os dados com os quais você preenche o buffer devem estar de acordo com o argumento "tipo" desta função, o que significa que você não pode tentar preencher com um string e usar o tipo inteiro de 16 bits não assinado, por exemplo. As constantes do tipo são as mesmas que as utilizadas pelo buffer_read() e buffer_write() funções. O "tamanho". é o tamanho do buffer (em bytes) que você deseja preencher, enquanto o offset é o valor do offset (também em bytes) desde o início do buffer para iniciar o preenchimento a partir dele.

Sintaxe:

buffer_fill(buffer, offset, type, value, size);

ArgumentoTipo Descrição
buffer The index of the buffer to fill.
offset The data offset value (in bytes).
type The type of data that is to be written to the buffer (see the list of constants here).
value The data to write.
size The size of the buffer (in bytes) that you wish to fill.

Devoluções:

Exemplo:

map_buffer = buffer_create(16384, buffer_fixed, 0);
buffer_fill(map_buffer, 0, buffer_u16, 0, 16384);

O código acima encontra o início do buffer com o id armazenado na variável "buff" eles escrevem uma série de valores inteiros assinados de 16 bits para ele.
