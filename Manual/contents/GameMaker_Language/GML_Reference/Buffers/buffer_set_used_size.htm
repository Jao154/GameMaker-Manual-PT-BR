tamanho_utilizado_do_soltampão

Esta função é principalmente para uso dentro de extensions, e permite definir o tamanho "usado" do dado buffer, que é o número de bytes que lhe foram escritos.

Quando você escreve dados para um buffer a partir de um extension, GameMaker não sabe quanto do buffer foi preenchido pelo código extension e não é capaz de ler esses dados. Esta função pode ser chamada pelo extension para dizer ao motor quantos bytes de dados foi escrito ao buffer, para que o motor possa ler esses dados.

Sintaxe:

buffer_set_used_size(buffer, size);

ArgumentoTipo Descrição
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

Devoluções:

Exemplo:

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

As funções acima seriam chamadas em um extension para escrever alguns dados em um GameMaker buffer (através de seu endereço de memória), e então definir o número de bytes que lhe foram escritos para que o motor possa ler esses dados.
