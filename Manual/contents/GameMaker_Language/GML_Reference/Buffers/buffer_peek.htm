tampão_peek

Com o buffer_read() você pode ler os dados da função buffer na posição "procurar" atual, com cada dado sendo lido avançando esta posição pelos bytes que estão sendo lidos ou escritos. Entretanto, pode ser necessário que você leia um dado dado sem querer mudar a posição atual de busca, e é quando você usaria esta função. Você simplesmente fornece a função com um id buffer, e depois uma posição de offset (a partir do início de buffer ) dentro daquele buffer para ler, assim como o tipo de dados que você está querendo ler.

NOTA: O uso do tipo de dados incorretos para os dados que estão sendo lidos resultará em valores errôneos!

Se a função tiver sucesso, então um valor (Real, string, booleano, etc...) será retornado dependendo do tipo de dado, enquanto uma falha retornará simplesmente 0.

Sintaxe:

buffer_peek(buffer, offset, type);

Argumento Tipo Descrição
buffer Identificação do tampão The index of the buffer to use.
offset Real The offset position (in bytes) within the buffer to read the given data from.
type Constante do tipo de dados tampão The type of data that is to be read from the buffer (see the list of constants here).

Devoluções:

Real, Boolean, String or 0 (if it fails)

Exemplo:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

O código acima obterá três valores de três posições diferentes dentro do buffer indexado na variável "buff" e então usará esses valores para definir a mistura de imagem da instância.
