buffer_copy

Esta função pode ser usada para copiar um segmento (ou todos) dos dados armazenados em um buffer para outro. Ao usar dois buffers e copiar de um para o outro, ambos buffers devem ter sido previamente criados usando o buffer_create() (por exemplo), e você pode especificar um offset de dados (em bytes) para o ponto inicial dos dados a serem copiados da fonte buffer relativo ao início do buffer, assim como outro offset de dados para definir a posição para copiar os dados para no destino buffer.

NOTA: Você não pode copiar para o mesmo buffer.

Sintaxe:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

ArgumentoTipo Descrição
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

Devoluções:

Exemplo:

buffer_copy(buff1, 0, 2048, buff2, 2048);

O código acima copiará os dados armazenados no buffer indexado na variável "buff1", e depois os colará no buffer indexado na variável "buff2", mas compensado por 2048 bytes.
