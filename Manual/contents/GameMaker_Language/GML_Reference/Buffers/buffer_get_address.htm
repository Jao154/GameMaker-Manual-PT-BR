endereço_de_suporte

Com esta função você pode obter o ponteiro para o endereço bruto, alinhado buffer. Isto é principalmente para uso com extensions, pois você pode passar este valor para eles, permitindo-lhes acessar os dados buffer. Note que há algumas coisas você não pode fazer com o buffer ao acessar a partir de uma extensão:

  • Não se pode liberar a memória.
  • Não se pode redimensionar a memória.
  • Você não pode escrever além do endereço buffer mais o tamanho buffer (você pode usar o buffer_get_size() para isto) ou você receberá um erro fora dos limites.

Sintaxe:

buffer_get_address(buffer);

ArgumentoTipo Descrição
buffer The index of the buffer to use.

Devoluções:

Exemplo:

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

O código acima obterá o endereço de memória do buffer indexado na variável "buff_model" e então obterá o endereço de memória para o final do buffer usando a função buffer_get_size(), armazenando ambos os valores em variáveis locais para utilização posterior.
