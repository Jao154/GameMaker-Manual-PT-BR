Unidades de fonte de tempo

Unidade Fonte de Tempo Constante
Constante Descrição Valor
time_source_units_seconds Use segundos para o período de Fonte de Tempo (independente do quadro) 0
time_source_units_frames Use quadros para o período de Fonte de Tempo (dependente do quadro) 1

Estas constantes são usadas em time_source_create() e time_source_reconfigure() para definir as unidades utilizadas para o período da Fonte de Tempo, e são devolvidas por time_source_get_units().

Se você usar segundos, sua Fonte de Tempo funcionará independentemente da estrutura do jogo.

Se você usar armações, sua fonte de tempo dependerá da armação do jogo.

Você também pode usar BPM (batidas por minuto) como uma unidade chamando time_bpm_to_seconds() para converter seu valor BPM para um período em segundos, e depois usando time_source_units_seconds como unidade.
