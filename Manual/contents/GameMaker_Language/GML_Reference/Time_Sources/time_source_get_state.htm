time_source_get_state

Esta função retorna o estado da Fonte de Tempo dada.

Sintaxe:

time_source_get_state(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para obter o estado de

Devoluções:

Time Source State Constant

Exemplo:

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_start(time_source);
    }
}

Quando a tecla de espaço é pressionada, este código obterá o estado de uma Fonte de Tempo.

Quando o estado está ativo, ele fará uma pausa na Fonte de Tempo, e quando for pausado, ele o retomará.
