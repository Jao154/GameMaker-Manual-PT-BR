Fontes de tempo

Uma "Fonte de Tempo" é um temporizador personalizado que você cria. Ele funciona por um determinado período de tempo e, no final, ele expira.

Uma Fonte de Tempo (TS) tem a capacidade de chamar um método personalizado ("callback") ao expirar.

Também é capaz de repetir um certo número de vezes, ou para sempre. Cada repetição de uma Fonte de Tempo é conhecida como um "representante".

Como utilizar as fontes de tempo?

Criar e iniciar uma Fonte de Tempo que chame um método personalizado:

  • Ligue para time_source_create() / Criar uma Fonte de Tempo para criar uma Fonte de Tempo
    • Especifique se este inherits da Global Time Source, da Game Time Source ou de uma fonte de tempo personalizada
    • Especifique o período de tempo de duração, e as unidades utilizadas (segundos ou quadros)
    • Especifique um método personalizado ou função script, com uma matriz de argumentos para passar para esse método
    • Opcionalmente, especifique o número de vezes que ele deve repetir
      • Por padrão, ele funciona apenas uma vez
  • Ligue para time_source_start() / Start Time Source para iniciar essa Fonte de Tempo imediatamente, ou chamá-la mais tarde quando você precisar dela para começar

Ao fazer isso, a Fonte de Tempo funcionará e, após o período especificado, ela expirará e chamará seu método especificado.

Fontes de tempo incorporadas

Uma Fonte de Tempo pode inherit a partir de qualquer uma das duas Fontes de Tempoembutidas, ou a partir de uma Fonte de Tempo personalizada.

Temporizadores simples

Embora a configuração de uma fonte de tempo não requeira muitas linhas de código, pode ser conveniente configurar e gerenciar rapidamente um temporizador usando uma única função.

As duas funções a seguir fornecem uma forma mais simples e rápida de criar uma Fonte de Tempo e pará-la:

  • call_later
  • call_cancel

NOTA Não é possível usar as funções Time Source com os IDs de Time Source usados por estas funções. Veja call_later para mais informações.

Lista de funções

As seguintes funções são utilizadas para criar e modificar as Fontes de Tempo:

  • time_source_create
  • time_source_destroy
  • time_source_start
  • time_source_stop
  • time_source_pause
  • tempo_curso_de_fonte
  • tempo_fonte_reconfigurar
  • time_source_reset
  • time_source_get_children
  • time_source_get_parent
  • tempo_source_get_period
  • time_source_get_reps_completed
  • time_source_get_reps_reps_remaining
  • time_source_get_state
  • time_source_get_time_remaining
  • unidades_de_recurso_de_tempo
  • time_source_exists

Cada função get_* retornará indefinida se a Fonte de Tempo especificada não existir.

As seguintes funções de auxílio são fornecidas para a conversão de períodos de tempo:

  • time_seconds_to_bpm
  • time_bpm_to_seconds
