time_source_get_children

Para a Fonte de Tempo dada, esta função retorna uma matriz contendo o Identificação da fonte de tempos de seus filhos.

Sintaxe:

time_source_get_children(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para obter os filhos de

Devoluções:

Array of Time Source ID

Exemplo:

var _children = time_source_get_children(time_source);
var _count = array_length(_children);

for (var i = 0; i < _count; i ++)
{
    time_source_destroy(_children[i]);
}

time_source_destroy(time_source);

Este código passa por todas as crianças de uma Fonte de Tempo, e as destrói uma a uma.

No final, ela destrói a Fonte de Tempo pai.
