time_source_start

Esta função inicia a fonte de tempo dada, mudando seu estado para time_source_state_active.

A Fonte de Tempo especificada pode ser uma nova Fonte de Tempo que nunca foi iniciada, foi pausada, parada ou expirada sem repetições.

Esta função "reinicializará suavemente" a Fonte de Tempo dada, redefinindo seu tempo de expiração e os representantes remanescentes para os valores que foram configurados para ela.

Sintaxe:

time_source_start(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para começar

Devoluções:

N/A

Exemplo:

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

Neste exemplo, queremos que a instância se destrua a si mesma 300 quadros depois.

O código primeiro cria um método que simplesmente chama instance_destroy().

Em seguida, ele cria uma Fonte de Tempo, herdada da Fonte de Tempo do Jogo. Ela define seu período para 300 quadros.

Finalmente, inicia a Fonte do Tempo.
