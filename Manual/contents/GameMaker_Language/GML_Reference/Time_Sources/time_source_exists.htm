time_source_exists

Esta função retorna se o ID dado é uma fonte de tempo válida.

Se a Fonte de Tempo foi destruída, isto retornará falso, porém se a Fonte de Tempo foi parada ou simplesmente expirou, ela continuará a existir, o que significa que esta função retorna verdadeiro.

Sintaxe:

time_source_exists(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para verificar

Devoluções:

Boolean

Exemplo:

if (time_source_exists(global.spawn_time_source))
{
    if (keyboard_check_pressed(vk_space))
    {
        time_source_destroy(global.spawn_time_source);
    }
}

Este código verifica se existe uma Fonte de Tempo. Se existir, ele verifica se a tecla Space foi pressionada.

Nesse caso, ela destrói a Fonte do Tempo.
