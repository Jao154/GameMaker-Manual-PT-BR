time_bpm_to_seconds

Esta função leva um valor de batidas por minuto, e retorna a duração de cada batida em segundos.

Isto pode ser usado ao criar uma Fonte de Tempo para usar um valor BPM para a Fonte de Tempo. É importante que tal Fonte de Tempo use segundos como sua unidade.

Sintaxe:

time_bpm_to_seconds(bpm);

Argumento Tipo Descrição
bpm Real The beats-per-minute value to convert into seconds-per-beat

Devoluções:

Real

Exemplo:

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

Este código converte um valor de 90 BPM em segundos, e usa isso para criar uma Fonte de Tempo que funciona indefinidamente.

Em cada batida, ele imprime a mensagem "BEAT! " para o registro de saída.
