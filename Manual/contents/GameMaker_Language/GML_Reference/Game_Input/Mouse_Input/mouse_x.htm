mouse_x

Esta variável somente leitura retorna a posição atual do eixo x do mouse dentro do room.

Sintaxe:

mouse_x;

Devoluções:

Exemplo:

x = median(64, mouse_x, room_width - 64);

O código acima manterá a instância na posição x do mouse enquanto estiver dentro dos limites de 64 pixels de cada lado do room.
