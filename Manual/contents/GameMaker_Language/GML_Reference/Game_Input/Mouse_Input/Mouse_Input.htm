Entrada do mouse

A entrada do mouse é aceita em todas as plataformas (em dispositivos móveis é aceita como um único toque na tela - se você precisar usar multi-toque, você deve estar usando as funções específicas do dispositivo) e tem algumas constantes que são usadas para especificar os botões que estão sendo pressionados. Estas constantes são mostradas na tabela a seguir:

As seguintes funções existem para os controles padrão do botão do mouse:

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

Note que também há um conjunto de funções de janela relacionadas ao uso do mouse em alvos de desktop:

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
