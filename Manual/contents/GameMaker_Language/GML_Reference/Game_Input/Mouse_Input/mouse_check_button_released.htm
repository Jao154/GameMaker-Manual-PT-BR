mouse_check_button_released

Esta função retornará true se o botão do mouse a ser verificado tiver sido liberado ou false se não tiver sido. Esta função só será acionada uma vez para qualquer botão do mouse quando ele for liberado e para acioná-lo novamente o botão precisará ter sido pressionado e liberado novamente. Você fornece o botão do mouse para verificar a partir de uma das seguintes constantes:

Sintaxe:

mouse_check_button_released(numb);

Argumento Tipo Descrição
numb Botão do Rato Constante Which mouse button constant to check for.

Devoluções:

Boolean

Exemplo:

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

O código acima verificará se o botão direito do mouse foi liberado e se ele tem a velocidade da instância para um décimo da distância entre a posição x/y atual e a posição x/y do mouse.
