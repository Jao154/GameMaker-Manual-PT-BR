gamepad_remove_mapping

Esta função pode ser usada para remover o mapeamento do dispositivo atual do índice de slots do gamepad dado. Uma vez chamado, o índice de slot precisará ser refazido usando a função gamepad_test_mapping() se você quiser ser capaz de usar as constantes do controle de jogo para detectar a entrada correta (a entrada direta sempre pode ser recuperada usando as funções gamepad_axis/button/hat_count() e gamepad_axis/button/hat_value() juntas).

Sintaxe:

gamepad_remove_mapping(index);

ArgumentoTipo Descrição
index Which gamepad index "slot" to remove the mapping from.

Devoluções:

Exemplo:

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

O código acima removerá o mapeamento do slot de índice gamepad dado, com base no valor de uma variável.
