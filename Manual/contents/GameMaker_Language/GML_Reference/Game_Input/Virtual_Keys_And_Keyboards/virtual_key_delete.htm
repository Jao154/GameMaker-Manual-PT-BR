virtual_key_delete

Se seu jogo tem diferentes rooms ou instâncias para controlar menus e jogabilidade e outras coisas, então é provável que você precise mudar a posição e os mapas de chaves de suas chaves virtuais em algum momento. Para isso, você pode usar esta função para apagar as antigas antes de criar as novas (se necessário). A função exige que você forneça a identificação da chave virtual para apagar - como retornado por virtual_key_add().

Note que quaisquer chaves virtuais serão automaticamente removidas do room quando o room for alterado, portanto esta função só é necessária quando você desejar remover manualmente as chaves antes que o próprio room tenha terminado.

Sintaxe

virtual_key_delete(index);

Argumento Tipo Descrição
index Identificação da chave virtual The index of the virtual key to delete.

Devoluções:

N/A

Exemplo:

virtual_key_delete(global.Left);

O código acima elimina a chave virtual indexada na variável global "global.Left".
