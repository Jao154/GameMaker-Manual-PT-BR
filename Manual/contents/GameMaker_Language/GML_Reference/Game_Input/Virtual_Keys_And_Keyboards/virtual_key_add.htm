virtual_key_add

Esta função permite mapear "toques" de uma área da tela para eventos do teclado. Isto significa que uma vez atribuída uma área a uma chave virtual, todos os toques nessa área acionarão o evento do teclado correspondente à chave que você mapeou para a área. Você pode atribuir cada chave virtual definida a uma variável também, que pode então ser usada nas outras funções da chave virtual para mostrá-las, ocultá-las e excluí-las. Estas teclas são atribuídas por sala e serão automaticamente removidas pelo GameMaker ao mudar rooms.

A posição real da chave virtual é baseada na posição da tela em vez da posição room e, portanto, os valores x/y são absolutos na tela. Isto significa que você não precisa se preocupar com o uso das vistas ou das coordenadas relativas room, e pode simplesmente definir as chaves no Create Event of an object (você só precisa definir uma chave virtual uma vez por sala), depois desenhe sua chave sprites no Draw GUI Event, dimensionando a camada GUI para ser a mesma que a tela.

Sintaxe

virtual_key_add(x, y, w, h, keycode);

Argumento Tipo Descrição
x Real The x coordinate (left side) of the virtual key on the screen
y Real The y coordinate (top side) of the virtual key on the screen
w Real The width of the virtual key
h Real The height of the virtual key
keycode Constante Chave Virtual (vk_*) Which keyboard key event should be triggered by touching this area

Devoluções:

Virtual Key ID

Exemplo:

global.Left = virtual_key_add(32, 32, 64, 64, vk_left);

O código acima cria uma chave virtual de 64x64 pixels quadrados, posicionada na tela em (32, 32) que acionará o evento vk_left quando tocado e atribuirá o índice desta chave virtual a uma variável global.
