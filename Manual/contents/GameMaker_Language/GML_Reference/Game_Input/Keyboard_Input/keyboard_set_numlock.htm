teclado_set_numlock

Você pode usar esta função para switch o teclado numérico de bloqueio ligado ou desligado (definido como verdadeiro para ligado, e falso para desligado).

NOTA Esta funcionalidade está disponível apenas no site Windows e não funcionará em nenhum outro dispositivo.

Sintaxe:

keyboard_set_numlock(value);

Argumento Tipo Descrição
value Booleano Set this to true for "on" and false for "off"

Devoluções:

N/A

Exemplo:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

O código do exemplo acima obterá o estado da chave de bloqueio numérico e, se estiver ligado (verdadeiro), o fará disparar (falso) e vice-versa.
