teclado_string

Esta variável contém um string contendo os últimos (no máximo) 1024 caracteres digitados no teclado. Este string conterá apenas caracteres imprimíveis digitados, mas responderá corretamente ao pressionar a tecla de backspace , apagando o último caractere. Esta variável não é somente lida e você pode modificá-la, por exemplo, para defini-la como "" (um string vazio) se você já a manipulou, e você pode usar as Funções de String para manipulá-la. Note que ao utilizar o teclado virtual na tela, somente esta variável será atualizada com a entrada do teclado.

Sintaxe:

keyboard_string;

Devoluções:

Exemplo:

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

O código acima limitará o comprimento do teclado string a 15 caracteres, removendo aqueles que estiverem acima desse limite, copiando os primeiros quinze caracteres de volta para a variável.
