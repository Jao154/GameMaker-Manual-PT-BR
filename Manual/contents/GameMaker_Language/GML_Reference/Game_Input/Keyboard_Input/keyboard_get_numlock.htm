teclado_get_numlock

Você pode usar esta função para encontrar o status do bloqueio do número do teclado com o verdadeiro sendo devolvido para ligado, e o falso sendo devolvido para desligado.

NOTA Esta funcionalidade só está disponível no Windows exe constrói e não funcionará em nenhum outro dispositivo.

Sintaxe:

keyboard_get_numlock();

Devoluções:

Exemplo:

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

O código de exemplo acima obterá o estado da chave de bloqueio numérico e, se estiver ligado (verdadeiro), o fará disparar (falso) e vice-versa.
