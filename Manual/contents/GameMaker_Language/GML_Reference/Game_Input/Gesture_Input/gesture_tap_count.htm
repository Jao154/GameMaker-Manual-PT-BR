gesto_tap_count

Esta função é usada para definir se a contagem de torneira está ativada ou desativada. Quando configurada para true, a contagem de tap está ativada, e quando configurada para false, ela está desativada, embora você possa verificar em qual estado ela está usando a função gesture_get_tap_count(). Quando ativado, cada evento de tap terá uma entrada adicional no mapa DS "tapcount" que terá registrado o número de toques, e quando ativado significa que todos os eventos de tap serão disparados, ou seja: dois toques dispararão tanto o evento de tap único como o evento de tap duplo toque, sendo a contagem de tap único toque 1 e a contagem de tap duplo toque 2. O valor da contagem de tap será reajustado para 0 após o tempo definido para uma detecção de tap duplo (veja a função gesture_double_tap_time()). Se a contagem da torneira estiver desativada, então a torneira inicial não será registrada até que o tempo de dupla torneira tenha passado e nenhuma segunda torneira tenha sido detectada. Note que isto é habilitado por padrão.

Sintaxe:

gesture_tap_count(enable);

ArgumentoTipo Descrição
enable Enable (set to true) ou desativar (ajustado para false) a contagem de torneira.

Devoluções:

Exemplo:

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

O código acima verifica se a contagem de torneira está ativada e se não está, então está ligada.
