gesto_drag_time

Esta função é utilizada para definir o tempo necessário para que um evento de Drag Gesture seja acionado por um toque ou clique. Este tempo também afetará como o Evento de Toque é acionado como um toque/clique e a liberação antes que este tempo acabe será considerada um Toque. O tempo é medido em segundos e tem um valor padrão de 0,16.

Sintaxe:

gesture_drag_time(time);

ArgumentoTipo Descrição
time The time (in seconds) to set for drag gesture event detection.

Devoluções:

Exemplo:

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

O código acima verifica se o tempo de arrastamento dos gestos está configurado para 0,16 segundos e se não está configurado para esse valor.
