dados_do_evento

Esta variável é de escopo global e é usada para manter um Mapa DS quando usada no evento apropriado (por exemplo, Eventos de Gesto e Mensagens de Transmissão), e -1 em todos os outros momentos. O conteúdo real do mapa DS dependerá do tipo de evento que o acionou, portanto, consulte as seções individuais para esses eventos.

Sintaxe:

event_data;

Devoluções:

DS Map ID

Exemplo:

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

O código acima é retirado do evento do gesto Drag End e verifica se o ventilador é um evento "flick", e se é ele que extrai os dados necessários do mapa event_data DS e o usa para definir algumas variáveis. Se um evento "flick" não for detectado, então as mesmas variáveis são definidas como 0.
