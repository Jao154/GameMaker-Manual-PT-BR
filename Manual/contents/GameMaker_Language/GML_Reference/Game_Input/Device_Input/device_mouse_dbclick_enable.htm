dispositivo_mouse_dbclick_enable

Esta função pode ser usada para configurar o dispositivo para detectar um duplo toque do mb_left (botão esquerdo do mouse) como uma entrada mb_right (botão direito do mouse).

Por padrão, isto é definido para true, o que significa que toda vez que o usuário bate duas vezes na tela do dispositivo de forma rápida e consecutiva, o valor de retorno é o mesmo como se o botão direito do mouse tivesse sido clicado. Quando isto estiver ligado, o primeiro toque será detectado como mb_left, e o segundo como mb_right, portanto, certifique-se de que qualquer código utilizado leve isto em consideração.

Sintaxe:

device_mouse_dbclick_enable(bool);

Argumento Tipo Descrição
bool Booleano Set double-click detection on (true) ou desligado (false).

Devoluções:

Boolean

Exemplo:

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

O código acima verifica se o dispositivo que executa o jogo é um Windows PC ou um Mac e, se for um deles, desabilita a função de duplo toque.
