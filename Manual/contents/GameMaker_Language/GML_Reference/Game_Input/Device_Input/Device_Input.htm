Entrada do dispositivo

Quando seu jogo estiver sendo direcionado para aquelas plataformas que permitem multi-toque, você deve estar usando estas funções. Elas funcionam da mesma forma que um PC funcionaria se tivesse vários ratos conectados e as constantes que estão sendo verificadas são as mesmas que as usadas para as funções padrão do mouse:

Constante Descrição
mb_left O botão esquerdo do mouse
mb_middle O botão central do mouse (isto pode não ser válido para todas as plataformas alvo)
mb_right O botão direito do mouse
mb_side1* Botão do lado do mouse 1
mb_side2* Botão do lado do mouse 2
mb_any Qualquer um dos botões do mouse
mb_none Sem botão do mouse

NOTA * Os botões mb_side1 e mb_side2 estão disponíveis apenas em Windows, macOS, Ubuntu e HTML5.

Todas as funções podem ter um valor de "toque" que corresponde a um toque no display do dispositivo, porém o número máximo de toques que podem ser detectados dependerá muito do dispositivo e do sistema operacional sendo executado. Por exemplo, alguns dispositivos de baixo nível Android podem ser capazes de detectar apenas 2 ou 3 toques ao mesmo tempo, enquanto um dispositivo UWP pode ser capaz de detectar até 10 toques. Observe também que nos dispositivos com tela sensível ao toque, um toque duplo é retornado como o equivalente a um evento de botão direito do mouse. Portanto, se você precisar detectar vários toques em sucessão rápida, por exemplo, você precisará verificar tanto o mb_left como o mb_right. Esta funcionalidade pode ser ativada ou desativada usando device_mouse_dbclick_enable().

Abaixo você pode encontrar uma lista de todas as funções de controle específicas do dispositivo:

  • device_mouse_check_button
  • device_mouse_check_button_pressed
  • device_mouse_check_button_released
  • device_mouse_dbclick_enable
  • device_mouse_x
  • device_mouse_y
  • device_mouse_raw_x
  • device_mouse_raw_y
  • device_mouse_x_to_gui
  • device_mouse_y_to_gui
  • device_is_keypad_open
  • device_get_tilt_x
  • device_get_tilt_y
  • device_get_tilt_z

Observe que nos dispositivos móveis (Android, iOS e Windows Phone), você pode usar o botão voltar a qualquer momento, pois ele é mapeado para o backspace do teclado que é a constante do teclado do GameMaker vk_backspace, assim você pode fazer verificações para essa tecla e executar ações nesses dispositivos como se o botão voltar tivesse sido usado.
