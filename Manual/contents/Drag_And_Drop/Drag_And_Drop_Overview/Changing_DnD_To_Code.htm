Mudança GML Visual To Code

Outra característica do GML Visual é que você pode convertê-lo em código GML, e desta forma ver o que exatamente está acontecendo nos bastidores quando você usa ações. Entretanto, antes de realmente mudar as ações em código, você pode primeiro visualizá-las usando o botão direito do mouse opção de menu Live Preview:

O que abrirá uma nova janela:

Ao adicionar, alterar ou remover GML Ações visuais, o GML Live Preview será atualizado para mostrar o código real que está sendo criado "nos bastidores". O código na visualização ao vivo não pode ser editado diretamente, mas você pode selecionar seções e copiá-las para colar em GML scripts ou Ações de código (por exemplo).

Se você decidir que deseja converter o GML Visual em código após ver a visualização, então isto pode ser feito novamente clicando com o botão direito do mouse em qualquer caso workspace com ações e selecionando Converter para GML Código.

A primeira vez que você fizer isso, receberá uma mensagem de aviso dizendo que esta é uma conversão de uma maneira, já que você pode converter ações em código, mas não pode convertê-las novamente em ações individuais mais tarde. Clicando em "ok" aqui, você fará a conversão para você.

O código resultante usará {} para delimitar ações individuais, e você pode ver claramente quais ações estão relacionadas a quais funções ou declarações de variáveis dentro do código. Se o GML Visual for mais complexo, então o código também será, mas as mesmas regras gerais se aplicam e o código será estruturas sequencialmente exatamente as mesmas que você escreveu o GML Visual. Ao aprender a programar usando GML Visual, esta pode ser uma ferramenta importante para passar a usar GML em um estágio posterior, mas não é de forma alguma obrigatório e você ainda pode fazer grandes jogos usando GML Visual! Também vale a pena notar que embora o processo de conversão seja uma forma, após converter ações em código, você pode voltar a usar GML Visual novamente usando o menu no editor de código e selecionando "Converter para GML Visual".
