Conjunto Tile Transformação de dados

Com esta ação, você pode mudar uma das três transformações tile nos dados tile. Você fornece a variável que contém os dados tile para o tile, depois seleciona a transformada a ser modificada. Clicando na caixa de seleção "Aplicar", será aplicada a transformação e desmarcando-o, ele será removido. Os dados tile são simplesmente um valor que reflete o índice do tile junto com as transformações aplicadas assim como qualquer máscara personalizada tile que você tenha usado, e esta ação permite que você altere três destes valores: Virar, espelhar e girar. Note que você normalmente desejará obter os dados de tile primeiro usando ou Obter Tile Data In Cell ou Obter Tile Data At Pixel, depois aplique o valor requerido transformar(s) e então definir o tile no mapa tile novamente usando o Set Tile Data In Cell ou Set Tile Data At Pixel, uma vez que esta ação apenas define um valor mas não o aplica ao tile do qual foi retirada. Para mais informações sobre os dados tile, consulte a seção GML sobre Funções do Mapa de Telha.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Tile Os dados de tile para a mudança do tile
Transform A transformação a aplicar/remover

Exemplo:

O código de bloco de ação acima recebe os dados tile na posição do mouse e então verifica se o tile teve uma transformação de rotação aplicada a ele. Se tiver, então a transformada é reinicializada e os dados do tile colocam o tile novamente na posição.
