Obter Tile Índice Na Célula

Com esta ação, você pode obter um índice tile em uma determinada camada. A camada é especificada a partir do nome da camada dada (um string, como usado para nomear a camada no editor Room ), e então você dá a linha e a coluna da célula dentro da camada para obter a telha índice de. Finalmente você fornece a variável alvo para armazenar o índice tile retornado, onde 0 não é tile, e um inteiro positivo é o índice do tile. Note que a variável alvo pode ser marcada como uma variável temporária (local) que pode ser usada até o final do evento.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada para obter o tile de
Column A coluna (da esquerda para a direita) da célula
Row A fila (de cima para baixo) da célula
Target A variável-alvo para o valor de retorno
