Obtenha Tile Dados na Pixel

Com esta ação você pode obter os dados tile para uma célula tile em uma camada de mapa de azulejos a partir de uma posição. Os dados tile são simplesmente um valor que reflete o índice do tile junto com os valores de inverter, espelhar e girar, assim como qualquer máscara personalizada tile que você já usou. Com esta ação você primeiro fornece a camada a ser alvo usando o nome da camada (a string) como definido no editor room, e depois os eixos x e y no room para obter os dados da telha. O valor dos dados de tile retornados coincidirá com seja qual for a célula do mapa de azulejos que esteja nesse ponto. Por exemplo, se seu conjunto de azulejos tiles for 16x16, então a camada do mapa de azulejos irá conter tiles em células de 16x16 pixels, portanto, se você usar esta ação para atingir a posição room (8, 8), você estará obtendo os dados para a célula (0, 0) do mapa do azulejo.

NOTA: A camada selecionada deve ter sido definida como um Tile Map Layer no editor room, caso contrário, você poderá receber erros.

O valor retornado será então armazenado na variável de destino que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você marcar a caixa de seleção "Temp"). Para mais informações sobre os dados tile, consulte a seção GML sobre Funções do Mapa de Telha.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada com o mapa de azulejos ao alvo
X A posição ao longo do eixo horizontal (X) para definir os dados da telha para
Y A posição ao longo do eixo vertical (Y) para definir os dados da telha

Exemplo:

O código de bloqueio de ação acima testa para um clique do mouse e quando um é recebido recupera os dados do azulejo para a célula que coincide com a posição do mouse. Estes dados são então modificados e retornados para a camada do mapa de azulejos, girando o tile naquela célula.
