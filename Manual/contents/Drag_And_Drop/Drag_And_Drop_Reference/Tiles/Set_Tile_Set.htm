Conjunto de azulejos

Com esta ação você pode definir uma camada para usar conjuntos de azulejos diferentes daqueles criados no Asset Browser. Você dá o nome da camada (um string, conforme definido no Editor Room ), e então o recurso tile set a ser utilizado, e todos tiles no room sobre essa camada serão desenhados com o novo tile set.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada sobre a qual se deve colocar o novo tile set.
Tileset O tile set a ser usado para a camada.
