Conjunto Tile Índice em Pixel

Com esta ação você pode definir um tile em uma determinada camada para um novo índice. A camada é especificada a partir do nome da camada dada (um string, como usado para nomear a camada no editor Room ), e então você dá uma posição x e y para que o tile seja definido. O x e o y a posição pode ser assinalada como sendo relativa à posição da instância de chamada, e a tile que está sendo definida será aquela em que a posição dada se enquadra no room e a tile set utilizada será a atribuída à camada na sala editor. Finalmente você fornece o índice tile para definir o tile para (tiles são indexados da esquerda para a direita de cima para baixo, com 0 na parte superior esquerda). Note que você sempre pode definir o índice tile com 0 para limpar o tile.

Argumentos:

Argumento Descrição
Layer A camada para colocar o tile em
x A posição x para verificar
y A posição y para verificar
Tile O novo índice do tile a utilizar
