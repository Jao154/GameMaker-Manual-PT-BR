Evento de chamada aos pais

Este bloco de ação executará o evento do pai object de que a instância é uma criança antes de continuar a executar as ações na instância real. Quando você atribui um pai object a uma instância, todos os eventos daquela object são "herdados". pela criança. Entretanto, se você adicionar um evento à criança object, ele substitui o evento dos pais. Usando este bloco de ação, você pode forçar uma criança object a executar tanto o evento dos pais quanto o evento na própria instância. Consulte a seção Editor de Objetos para obter mais informações sobre os pais.
