Conjunto Sprite

Este bloco de ação coloca o sprite_index da instância em outro sprite do Browser Asset. Você seleciona o sprite para mudar, e depois dá a imagem (quadro de animação) para mostrar. Note que se você não quiser que o sprite seja animado depois definindo o quadro, você precisará usar a ação Set Animation Speed e defini-lo como 0, ou se você quiser que o sprite continue animando, então defina o valor para usar a variável embutida image_index ao invés disso.

A mudança do sprite também mudará a máscara de colisão da instância, a menos que você também tenha especificado um mask_index separado no Editor de Objetos. Favor observar que para mudanças nesta ação para ser visível, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker irá, por padrão, sortear o sprite) ou ser sorteada usando a ação do Draw Self.
