Rotação da Instância

Este bloco de ação estabelece o image_angle (rotação) da instância. O ângulo é medido em graus, com a direita sendo 0°, acima sendo 90°, esquerda sendo 180° e abaixo sendo 270°. Defina esta variável como 0 para reiniciar a instância, o que significa que ela desenhar qualquer sprite atribuído como foi definido no editor sprite. Observe que para que as mudanças nesta ação sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker irá, por padrão, sortear o sprite) ou ser sorteado usando a ação do Draw Self.
