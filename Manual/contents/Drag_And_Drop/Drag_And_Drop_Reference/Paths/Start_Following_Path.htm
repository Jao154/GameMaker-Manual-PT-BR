Comece a seguir o caminho

Esta ação iniciará uma instância seguindo um path que foi criado no Browser Asset. Você então usará esta ação para dizer à sua instância qual path seguir, que velocidade seguir o path (medido em pixels por frame do jogo), como comportar-se quando chegar ao final do path, e se deve seguir a posição absoluta ou relativa path. Ao definir a velocidade você pode usar valores negativos para significar que a instância deve seguir o path ao contrário e vale a pena notar que a velocidade que você definir aqui pode ser modificada pelo modificador de velocidade definido no editor path para cada ponto path, então se você definir a velocidade path para 2 e no path um dos pontos tem um modificador de velocidade de 200%, então a velocidade real nesse ponto será 4 pixels por moldura de jogo.

Uma vez definida a velocidade inicial path você pode definir como a instância deve se comportar quando chegar ao final do path a partir das seguintes opções (note que quando uma instância chegar ao final de um path isso também desencadeará um Outro - Path Ended Event):

  • Pare: Termine o path, parando a instância no último ponto path.
  • Reiniciar: Continue o path desde o início, pulando novamente para a posição inicial se o path não estiver marcado como fechado no Editor Path.
  • Loop: Continue a partir da posição atual para executar novamente o path.
  • Reverter: Volte para trás ao longo do path novamente (conseguido invertendo o path siga a velocidade), e quando chegar ao início do path, ele irá reverter novamente, etc.

Finalmente, você pode flag o path como sendo Absoluto ou não. Por padrão, a instância começará a seguir o path a partir de sua posição atual (a posição relativa), mas se você flag a caixa de seleção Absoluto, então começará a seguir o path exatamente como você o projetou e o colocou no Editor Path (a posição absoluta). Sintaxe de ação:

Argumentos:

Argumento Descrição
Path O path asset para começar a seguir
Speed A velocidade (em pixels por frame de jogo) para seguir o caminho
On End A resposta para chegar ao fim do caminho
Absolute Se começar na posição absoluta ou relativa
