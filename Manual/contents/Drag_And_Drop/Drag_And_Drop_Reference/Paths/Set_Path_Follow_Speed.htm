Conjunto Path Siga a velocidade

Esta ação definirá a velocidade, em pixels por quadro de jogo, que uma instância deve seguir um path (note que você deve ter chamado a ação Start Following Path para atribuir um path à instância para que esta tenha qualquer efeito). Você pode usar valores negativos para significar que a instância deve seguir o path ao contrário e vale a pena notar que a velocidade que você definir aqui pode ser modificada pelo modificador de velocidade definido no editor path para cada ponto path, portanto, se você defina uma velocidade path para 2 e no path um dos pontos tem um modificador de velocidade de 200% a velocidade real por frame de jogo nesse ponto será de 4 pixels.

Argumentos:

Argumento Descrição
Speed A velocidade (em pixels por frame de jogo) que a instância deve seguir o caminho

Exemplo:

O código de bloco de ação acima inicia uma instância após um path, depois gera um entre 2 e 4 antes de usar este valor para definir a velocidade que a instância se move ao longo do path.
