Esconder Teclado Virtual

Esta ação pode ser usada para esconder o teclado virtual no dispositivo que executa o jogo. A chamada desta ação gerará um Evento Assíncrono de Sistema, no qual o mapa DS async_load  será preenchido com os seguintes pares de teclas/valores:

  • "event_type" - este será o "status do teclado virtual" quando acionado por ações do teclado virtual.
  • "screen_height" - a altura do teclado virtual (em pixels). Esta será 0 se o teclado for invisível.
  • "keyboard_status" - o status atual do teclado, retornado como uma das seguintes seqüências:
    • "escondido".
    • "oculto".
    • "mostrando"
    • "visível".
