Se o mouse for liberado

Esta ação verifica quando um botão do mouse foi solto. Ela retornará true na etapa de jogo único (frame) em que o botão do mouse é liberado e false em todos os outros momentos. Se você ativar o modificador not, esta ação será revertida, e verificará se o botão do mouse não foi liberado, retornando false na etapa do jogo (frame) em que foi liberado e true em todos os outros momentos.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão agora executadas se o "se" for avaliado para true, enquanto quaisquer ações descartadas em outro lugar serão executadas após o bloco "se".

Argumentos:

Argumento Descrição
Mouse Button O nome do botão do mouse para verificar (veja esta página para informações sobre as constantes do botão do mouse).
Not Negar o cheque (o verdadeiro torna-se falso e vice-versa)

Exemplo:

O código de bloqueio de ação acima faz com que o botão do mouse se movimente a cada passo e, se estiver sendo mantido pressionado, ele verifica se o mouse está pressionado inicialmente. Se a verificação for true na descida inicial pressione a velocidade da instância é definida, então, enquanto o botão está pressionado, a mistura sprite é mudada para vermelho. Se o mouse não estiver sendo pressionado, uma verificação é feita na liberação do mouse para redefinir a cor da mistura para branco e ajustar a velocidade para 0.
