Arquivo de Iniciativas Abertas

Com esta ação você pode abrir um arquivo da Ini para leitura e/ou escrita. Se o arquivo Ini não existir no local que você está verificando, o GameMaker pode criar um, mas somente se você escrever dados para ele. Se você só tiver lido informações de o arquivo Ini, então os valores padrão para a função lida serão devolvidos, mas o arquivo Ini não será realmente criado.

Observe que você só pode ter um arquivo Ini aberto de cada vez e lembre-se de usar Fechar arquivo Ini quando terminar de ler/escrever a partir do arquivo Ini, pois as informações não são realmente armazenadas em disco até lá (é armazenado na memória até que o arquivo seja fechado).
