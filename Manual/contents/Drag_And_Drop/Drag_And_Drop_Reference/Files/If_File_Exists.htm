Se o arquivo existir

Com esta ação, você pode verificar se existe um arquivo com o nome dado. Você seleciona o arquivo a ser verificado (como uma picada e incluindo o extension) e a função retornará true se ele existir, e false se não existir. Note que isto retornará true se o arquivo existir no pacote do jogo como um arquivo incluído ou se o arquivo tiver sido salvo enquanto o jogo estava em execução (veja a seção sobre O Sistema de Arquivo para mais informações).

O código de bloco de ação acima criará uma variável para conter dados buffer e então verificará se um arquivo de salvamento buffer já foi feito anteriormente. Se o arquivo existe, então ele é carregado na variável buffer que criamos, mas se não existir, ele é criado, escrito e então salvo pronto para ser carregado na próxima vez.
