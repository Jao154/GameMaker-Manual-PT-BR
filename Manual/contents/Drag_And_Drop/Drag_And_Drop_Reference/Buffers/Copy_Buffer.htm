Buffer de cópia

Esta ação pode ser usada para copiar todos os dados armazenados em um buffer para outro. Ao usar dois buffers e copiar de um para o outro, ambos buffers devem ter sido criados anteriormente usando a ação Criar Tampão, e você fornece o ID único buffer devolvido como o de/para argumentos.

NOTA: Você não pode copiar para o mesmo buffer.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Buffer O índice buffer (armazenado em uma variável) para copiar de
Copy To O índice buffer (armazenado em uma variável) para copiar para

Exemplo:

O código de bloco de ação acima criará um novo buffer e armazenará sua identificação em uma variável temporária. Ele então verifica se o buffer indexado na variável global em questão existe, e se existe então é copiado para o buffer temporário e a posição de leitura/gravação é reinicializada para o início, e se não existe então ele é criado.
