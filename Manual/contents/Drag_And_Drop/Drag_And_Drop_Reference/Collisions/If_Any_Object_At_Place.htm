Se houver Object No local

Esta ação é usada para verificar e ver se há uma instância de qualquer object colocada em uma posição baseada na máscara de colisão da instância de chamada. Você dá a posição, que pode ser uma posição absoluta ou uma posição relativa à instância, e a ação retornará true se houver uma colisão ou false se não houver. Você também pode verificar o "Não" flag para verificar se não há colisão na posição determinada, e a ação retornará então true se não houver colisões e false caso contrário. As colisões são calculadas com base na máscara de colisão da instância de chamada sobreposição da máscara de colisão de qualquer instância na posição (como se ela estivesse sendo "colocada" na posição).

IMPORTANTE! As colisões só serão registradas para aquelas instâncias que possuem uma máscara de colisão válida, ou seja: elas têm um sprite atribuído ao sprite_index, ou um sprite atribuído ao mask_index. Se qualquer uma das instâncias em a colisão não tem máscara de colisão então a colisão não será detectada, independentemente do que a instância estiver desenhando no momento.
