Para

O For action é usado para executar loops que só termina quando um valor atinge a condição determinada. Você dá um valor inicial como uma variável (que pode ser marcada como uma variável local temporária - geralmente é o que você quer), então você dá a condição para comparar a variável (por exemplo, se for inferior a um determinado valor) e, finalmente, a quantia para incrementar a variável a cada iteração do loop. Assim, o fluxo geral é o seguinte:

  • Você rubricar uma variável até um valor
  • Você define a condição para verificar esta variável em relação a
  • Você define o quanto a variável deve ser incrementada por cada iteração do laço
  • Você então adiciona ações no loop, e estas serão executadas para cada iteração até que a condição retorne false ou uma ação Break seja encontrada.

Ao adicionar ações a serem realizadas dentro do loop, estas são adicionadas tanto quanto você adicionaria uma ação a um "se", ou seja: deixá-la cair para o lado da ação em vez de sob ela:

Todas as ações adicionadas ao lado serão incluídas no For e serão realizadas antes do evento ou script continua com as ações acrescentadas abaixo. Note que você pode terminar qualquer loop prematuramente se você chamar a ação Break, e ao usar For loops você não precisa aumentar e pode fazer com que ela diminua também, desde que a condição que você usa é apropriado. Por exemplo, defina a variável inicial para 10, defina a condição para verificar menos de 0 e depois defina o valor de incremento para -1 e o loop irá iterar 10 vezes, contando para trás de 10 para 0.

Argumentos:

Argumento Descrição
Initialise Inicializar a variável que será usada para contar até um valor (pode ser marcada como uma variável local temporária)
Condition A condição para verificar para quebrar o laço
Increment A quantidade para incrementar (ou diminuir) a variável inicial de contagem por cada iteração do laço
