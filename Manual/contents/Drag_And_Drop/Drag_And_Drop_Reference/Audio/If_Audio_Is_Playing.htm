Se o áudio estiver tocando

Esta ação pode ser usada para verificar se uma instância de algum recurso sonoro está jogando atualmente em seu jogo. Você dá o recurso de som a ser verificado pelo explorador asset e a função retornará true if it is playing and falso se não for, embora se você verificar o "não" flag, isto é invertido e a função retornará true se o som não estiver tocando e false caso contrário.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão executadas agora se o "se" for avaliado para true, enquanto qualquer ações abandonadas em outro lugar serão realizadas após o bloco "se".
