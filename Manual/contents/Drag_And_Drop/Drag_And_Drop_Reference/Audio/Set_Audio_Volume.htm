Volume de áudio definido

Com esta ação, você pode definir o volume de um determinado som. Você seleciona o som do explorador asset e depois define o valor do volume. O volume pode estar entre 0 (silencioso) e 1 (volume completo) e a escala é linear, de modo que um valor de 0,5 seria metade do volume. Note que esta ação afetará todas as instâncias do som que estão tocando atualmente no room, e afetará todas as instâncias futuras do som tocado.

O campo "Tempo" controla quanto tempo leva para que a mudança de volume tenha efeito. Um valor de 0 resulta em uma mudança instantânea, onde um valor maior mudará gradualmente o volume ao longo de uma determinada quantidade de milissegundos.

Argumentos:

Argumento Descrição
Sound O bom recurso para definir o volume de
Volume O novo volume para o som (de 0 a 1, padrão 1)
Time A quantidade de tempo ao longo do qual o volume deve desbotar (em milissegundos).

Exemplo:

O código de bloco de ação acima verifica uma variável global para ver se ela é true ou falsa. Se for true, então o volume do som dado é recuperado e armazenado em uma variável local temporária. O valor é então verificado para ver se não é igual a 0,5, e se não for, então o volume do som é ajustado para desbotar para 0,5 em 1000 milissegundos (1 segundo). Se a variável global for avaliada para false, então o som tem seu volume ajustado para 1. Como o valor para o tempo é ajustado para 0 aqui, a mudança no volume ocorre imediatamente.
