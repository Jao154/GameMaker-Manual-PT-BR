Reproduzir áudio

Esta ação irá definir o jogo de som dado. Você seleciona o som do explorador Asset e então define se deseja ou não que o som seja tocado em loop (um som em loop será tocado repetidamente até parar) ajustando o "Loop" flag.

Além disso, você pode definir o ganho, o offset (a posição da pista para começar a tocar) e o arremesso para tocar o som.

O valor final de ganho e pitch para o som que está sendo tocado é o produto dos valores de ganho e pitch set para o som asset. O valor do offset é o valor fornecido no campo "Offset" ou o valor do offset do ativo sonoro sendo jogado quando o campo é deixado vazio (ver Exemplo 2).

Sintaxe de ação:

Argumentos:

Argumento Descrição
Sound O som asset para tocar
Loop Seja para loop o som ou não (desligado por padrão)
Target A variável-alvo para armazenar o valor de retorno
Gain Valor para o multiplicador de ganho (padrão 1). O multiplicador de ganho definido aqui é multiplicado com o multiplicador de ganho do ativo sonoro
Offset O tempo (em segundos) para definir o ponto de partida (padrão 0). Valores mais longos que o comprimento do som dado são ignorados.
O offset do ativo sonoro (posição da faixa) é usado quando este campo é deixado em branco.
Pitch O multiplicador de passo (padrão 1). O multiplicador de ganho definido aqui é multiplicado com o multiplicador de ganho do ativo sonoro

Exemplo 1:

O código de bloco de ação acima verifica se há um botão do mouse e, se for detectado, verifica uma variável global para ver se ela não é true. Se não for true, então um som é reproduzido (e feito em loop) e a variável global é definida para true, caso contrário o som é parado e a variável global é definida para false.
O som é reproduzido com as configurações padrão: um multiplicador de ganho e de pitch de 1 e um offset de 0 (o offset padrão do som asset).

Exemplo 2:

O código de bloco de ação acima define primeiro a posição da faixa de som asset "snd_MusicBoss" para 20 segundos. Em seguida, verifica se a variável de instância "skip_intro" está definida para true. Se estiver, começa a tocar "snd_MusicBoss" em 20 segundos, o valor para a posição da faixa definida anteriormente (isto porque o campo "Offset" é deixado em branco para que o offset do ativo seja usado). Os primeiros 20 segundos do áudio são pulados.
Se a variável de instância "skip_intro" estiver definida para false, ela também começará a tocar "snd_MusicBoss" mas, como o "Offset" foi definido para 0, o som começará a tocar desde o início. A posição da faixa de 20 segundos, conforme definido anteriormente para o ativo sonoro, é ignorada.
