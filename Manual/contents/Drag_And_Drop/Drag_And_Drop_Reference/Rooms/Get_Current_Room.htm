Obter Sala de Corrente

Esta ação devolverá o valor único do índice room para o atual room. Você precisa fornecer uma variável alvo para manter o valor retornado, e esta variável pode ser marcada como uma variável local temporária.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Target A variável alvo para armazenar o índice de retorno room

Exemplo:

O código de bloco de ação acima recebe o valor atual do índice room e então usa uma ação Switch para verificar o room e definir uma variável global, dependendo do resultado.
