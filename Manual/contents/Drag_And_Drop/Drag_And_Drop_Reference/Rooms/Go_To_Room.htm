Ir para o quarto

Esta ação encerrará o atual room e irá para o room especificado pelo índice de entrada. O room deve existir ou então um erro será dado, e se você usar o mesmo room que o atual room terá o mesmo efeito que usar a Sala de Reinício. Note que o room não mudará até o final do evento ou script onde a ação foi chamada, portanto qualquer ação após esta ainda será executada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Room O room para ir para

Exemplo:

O código de bloco de ação acima recupera uma variável global e depois verifica se o valor que ela contém é menor ou igual a 0. Se for, então o jogo irá para outro room.
