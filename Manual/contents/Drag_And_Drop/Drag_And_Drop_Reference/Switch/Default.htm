Predefinição

Esta ação criará um novo caso padrão dentro de um Switch. O caso padrão é aquele que será executado se o valor de entrada não corresponder a nenhuma outra constante de caso. As ações que são adicionadas a um caso por padrão devem ser caiu para o lado da ação, assim como você acrescentaria uma ação a um "se":

O código de bloco de ação acima recebe o valor armazenado em uma variável global e depois o compara com os diferentes casos possíveis. Se qualquer um dos casos for igual ao valor, então a ação sob esse caso será executada, caso contrário, o próximo caso será avaliado. Se nenhum caso corresponder ao valor de entrada, o caso padrão será executado em seu lugar.
