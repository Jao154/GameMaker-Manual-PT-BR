Obter índice do item da lista

Com esta ação, você pode recuperar a posição indexada de um item dentro da lista fornecida. Você fornece a variável que armazena o índice da lista (conforme retornado pela ação Criar Lista) e o valor que você deseja encontrar a posição do índice de dentro da lista, bem como uma variável alvo para armazenar a posição do índice retornado para o item (que pode ser marcada como uma variável local temporária a ser usada até o final do script ou evento). Observe que se o valor que você está procurando faz não existe dentro da lista, então não haverá posição de índice da lista para retornar, e assim o valor de retorno será -1.

Sintaxe de ação:

Argumentos:

Argumento Descrição
List O índice (armazenado em uma variável) da lista para remover de
Value O valor a ser verificado na lista para
Target A variável alvo para armazenar o índice retornado

Exemplo:

O código de bloco de ação acima verifica se há uma colisão na posição da instância e se for encontrado o valor de identificação único para a instância é armazenado em uma variável temporária e depois verificado para ver se ele existe dentro da estrutura de dados da lista. Se ele existe, então o item é removido da lista e a instância é destruída.
