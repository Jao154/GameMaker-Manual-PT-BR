Obter a contagem de itens da lista

Você pode usar esta ação para descobrir quantas entradas existem em uma lista. Você fornece a variável que contém o índice da lista (como retornado pela ação Criar Lista) e a variável alvo para armazenar o número retornado de itens da lista (que podem ser marcados como uma variável local temporária a ser usada até o final do script ou evento).

Sintaxe de ação:

Argumentos:

Argumento Descrição
List O índice (armazenado em uma variável) para obter a contagem do item de
Target A variável de destino para armazenar a contagem de itens devolvidos

Exemplo:

O código de bloco de ação acima obtém o tamanho da lista referenciada no e depois usa uma variável para loop a loop através dela e destrói as instâncias com as IDs armazenadas na lista. Depois que o loop é terminado, a lista é liberada da memória.
