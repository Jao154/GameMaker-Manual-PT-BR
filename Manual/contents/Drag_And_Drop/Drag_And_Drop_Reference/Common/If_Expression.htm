Se a expressão

Esta ação é usada para verificar que uma expressão como true ou false, e você pode verificar qualquer expressão lógica aqui, desde que ela avalie um ou outro valor. Por exemplo, qualquer uma das seguintes expressões pode ser usada:

x < (room_width / 2)
a == b
++val >= 100

Se você flag o argumento "Não", então as expressões acima serão negadas, por exemplo "==" torna-se "not=". Observe também que a expressão não está limitada a valores numéricos e você pode verificar strings, ponteiros, IDs de recursos ou qualquer outra coisa que uma função possa retornar ou usar (veja a seção sobre tipos de dados para mais informações), mas você não pode comparar dois tipos de dados diferentes sem obter um erro - por exemplo, se você tentar usar um string e um número inteiro na expressão, você obterá um erro. Para mais informações sobre expressões, veja aqui.

NOTA IMPORTANTE: Devido a questões de precisão de ponto flutuante, verificar se dois valores são exatamente iguais pode retornar false, já que um pode ser exatamente 1, enquanto a outra pode ser 1.00000000000001. Isto pode ser evitado usando a ação Decimal para Inteiro antes de verificar ou usando as expressões "maior que (>)" ou "menor que (<)".

Essas ações serão executadas agora se o "se" for avaliado para true, enquanto quaisquer ações descartadas em outro lugar serão executadas após o bloco "se". Note que você pode "empilhar" expressões na mesma ação, clicando no ícone "mais ao lado de a ação, e dando outra expressão para verificar. Isto verificará se todas as expressões dadas retornam true em uma única verificação e só executam o código de ação adicionado se o fizerem (veja o exemplo dado abaixo).
