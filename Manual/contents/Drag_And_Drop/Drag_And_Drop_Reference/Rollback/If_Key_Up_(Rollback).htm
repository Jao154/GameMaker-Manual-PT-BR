Se a chave está em cima (Rollback)

Esta ação verifica se a entrada dada acabou de ser liberada para o jogador atual.

Ele permite anexar ações à direita, que só funcionam quando a entrada dada é encontrada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Input O nome da entrada a ser verificada
Not Permitir que isto verifique se a entrada não foi liberada

Voltar: Ações Multiplayer Rollback
