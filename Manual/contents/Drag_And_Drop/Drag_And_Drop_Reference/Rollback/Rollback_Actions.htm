Ações Multiplayer Rollback

As ações de Rollback são usadas para fazer jogos multiplayer usando o sistema Rollback do GameMaker.

Para uma introdução, veja: Rollback Multiplayer

Para uma lista de variáveis utilizadas com o sistema Rollback, veja: Variáveis de Rollback

As seguintes ações são fornecidas para o Rollback multiplayer:

  • Criar jogo
  • Junte-se ao jogo
  • Iniciar o jogo
  • Deixar o jogo
  • Definir entrada
  • Definir o atraso da moldura de entrada
  • Definir entrada simulada
  • Definir Jogador
  • Mostrar eventos
  • Obtenha informações
  • Obter Input
  • Se a chave estiver em baixo (Rollback)
  • Se a tecla for pressionada (Rollback)
  • Se a chave está em cima (Rollback)
  • Se o evento Rollback for demitido
  • Sync On Frame
  • Usar Entrada Aleatória
  • Início do manual de uso
  • Enviar mensagem
