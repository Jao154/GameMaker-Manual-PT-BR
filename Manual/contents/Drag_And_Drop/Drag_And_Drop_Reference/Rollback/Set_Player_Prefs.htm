Pré-formas de Jogadores (Rollback)

Quando você joga um jogo usando o Sync Test, todos os jogadores "remotos" recebem valores aleatórios para suas entradas definidas, como uma forma básica de teste. Você pode desativar ou reativar este comportamento usando esta ação.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Enabled Habilitar ou desativar a entrada aleatória para jogadores remotos durante os testes

Voltar: Ações Multiplayer Rollback
