Obter informações sobre o Rollback

Esta função retorna um struct contendo informações sobre um jogador. Isto só deve ser usado quando se gerencia manualmente as instâncias de um jogador. Se você estiver usando Definir Player, então cada instância de jogador receberá automaticamente esta informação como variáveis.

Para informações sobre o conteúdo do retorno struct, veja: rollback_get_info

Sintaxe de ação:

Argumentos:

Argumento Descrição
Player ID A identificação do jogador para obter informações para
Target A variável onde a variável retornada struct deve ser armazenada

Voltar: Ações Multiplayer Rollback
