Enviar mensagem (Rollback)

Esta ação é utilizada para enviar uma mensagem string para outros jogadores.

Você especifica um string para enviar, com um tamanho máximo de 128 bytes.

Quando o segundo argumento não é especificado, a mensagem é enviada a todos os jogadores, incluindo o remetente.

Você pode especificar uma identificação do jogador no segundo argumento, que é o jogador que receberá a mensagem.

Como usar?

Esta ação destina-se a ser usada "localmente", e assim resultaria de entradas locais, e não de entradas Rollback.

Veja o exemplo de uso na parte inferior. Ouça as mensagens recebidas no evento Rollback Event.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Message A mensagem a enviar, como um string.
To OPCIONAL A identificação do jogador para o qual a mensagem será enviada. Não especificando este argumento, ou especificando -1, envia a mensagem a todos os jogadores.

Exemplo:

Se a tecla enter for pressionada localmente, a keyboard_string valor é enviado a todos os jogadores. O endereço string é então liberado para que uma nova mensagem possa ser digitada.

Isto acionará o evento Rollback Event quando a mensagem for recebida por todos os clientes (incluindo o remetente).
