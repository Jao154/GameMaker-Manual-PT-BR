Criar Jogo de Rollback

Esta ação hospeda um novo jogo multiplayer. É necessário o número de jogadores que devem estar conectados para que o jogo comece e, opcionalmente, permite desativar o Sync Test.

Quando chamado com o Sync Test desativado, o sistema espera que a quantidade dada de jogadores participe antes de iniciar o jogo. Certifique-se de não criar ou modificar nenhuma instância gerenciada até o início do jogo.

DICA Opcionalmente, você pode forçar o jogo a começar antes que todos os jogadores tenham aderido, executando o Start Game.

Atualmente, você só pode ter até 4 jogadores em um jogo.

AVISO Não é recomendado definir uma região padrão nesta ação a menos que você esteja permitindo que o jogador a selecione através de um menu.

Argumentos:

Argumento Descrição
Count of players O número de jogadores que devem estar conectados para que o jogo comece. Se for usado Definir Jogador, este é o número de instâncias que são criadas automaticamente.
Sync test Especifica se o Sync Test deve ser ativado, e é ativado por padrão. Quando desativado, a funcionalidade online é ativada.
Region Especifica a região onde o virtual room será criado. Para informações sobre como obter regiões, veja rollback_create_game().
