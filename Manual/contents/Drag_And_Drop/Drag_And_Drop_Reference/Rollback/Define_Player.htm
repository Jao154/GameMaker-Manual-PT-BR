Definir Jogador (Rollback)

Esta ação diz ao sistema Rollback o object que deve ser usado pelos jogadores. Isto é opcional, e quando usado, o sistema criará automaticamente instâncias para jogadores que se conectam, e destruirá instâncias para jogadores que se desconectam.

Você pode especificar a camada onde as instâncias do jogador serão criadas. Se uma camada não for especificada, os jogadores serão criados na camada "Instâncias ", que deve existir no site room.

Para informações sobre como criar um jogo com ou sem esta função, consulte Definindo um objeto do jogador.

Se utilizada, esta função deve ser executada antes da ação de join/start.

Cada instância criada com esta função recebe as variáveis da instância listadas nesta página.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Object O object a ser usado para instâncias de jogadores
Layer OPCIONAL O nome da camada onde as instâncias do jogador serão criadas. "Instâncias" por padrão (quando nenhum valor é especificado).
