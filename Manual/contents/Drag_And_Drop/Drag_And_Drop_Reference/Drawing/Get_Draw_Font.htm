Obter a fonte Draw Font

Com esta ação, você pode obter o valor de identificação do font atualmente designado para o desenho. O valor retornado será ou -1 para nenhum font atribuído, ou um valor >= 0 que representa uma constante font do Navegador Asset. O valor de retorno será armazenado na variável alvo que você especificar, que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você marcar a caixa de seleção "Temp").
