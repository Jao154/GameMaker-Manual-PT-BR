Biblioteca de Ação de Desenho

A biblioteca de ações de desenho é onde você pode encontrar as ações necessárias para desenhar sprites, texto ou formas, assim como definir certas propriedades de desenho. A maioria dessas ações são apenas para uso nos diversos eventos de sorteio de um object, e podem não funcionar se usadas fora do evento do sorteio. As exceções a isto são as ações Set, que podem ser adicionadas a qualquer evento e afetarão todos os sorteios para todas as instâncias posteriores.

É importante notar que se você adicionar qualquer ação no evento principal de sorteio de um object, então ele não sorteará o sprite que foi atribuído à instância a menos que você diga explicitamente ao GameMaker para sorteá-lo, usando uma ação como o Draw Self. Basicamente, o GameMaker sorteará por padrão qualquer sprite atribuído a uma instância, somente se não houver mais nada no Evento de Sorteio.
