Valor do sorteio

Com esta ação você pode desenhar uma legenda string e/ou uma variável ou valor para a tela. O argumento do título é um string que você deseja adicionar antes do valor, e pode ser deixado em branco se necessário, mas se você desejar incluir texto, ele deve ser embrulhado entre aspas, e o valor pode ser uma variável, um valor fixo, ou outro string - nesse caso também deve ser incluído entre aspas - mas não pode ser deixado em branco. Em seguida, você dá a posição em que o string deve ser desenhado, seja como uma posição absoluta ou como uma posição relativa à instância que está desenhando o texto. Observe que o aspecto do texto quando desenhado dependerá do font utilizado, da cor utilizada e do alinhamento. Todas estas coisas podem ser alteradas antes de desenhar, como mostra o exemplo abaixo.

NOTA: Esta ação é apenas para uso nos diversos eventos de sorteio, e não desenhará nada se for usada em outro lugar.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Caption A legenda a desenhar (a string), pode ser deixada em branco
Value O valor a sacar (real, variável ou string)
x A posição x para desenhar em
y A posição y para desenhar em
