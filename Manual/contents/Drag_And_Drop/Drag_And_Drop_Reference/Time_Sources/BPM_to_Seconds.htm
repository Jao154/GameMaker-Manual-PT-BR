BPM para Segundos

Esta ação leva um valor de batidas por minuto e retorna a duração de cada batida em segundos.

Isto pode ser usado ao criar uma Fonte de Tempo para usar um valor BPM para a Fonte de Tempo. É importante que tal Fonte de Tempo use segundos como sua unidade.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Value O valor das batidas por minuto
Target A variável para armazenar o valor retornado segundos por batimento
Temp Se a variável deve ser criada como uma variável temporária

Voltar: Ações de fonte de tempo
