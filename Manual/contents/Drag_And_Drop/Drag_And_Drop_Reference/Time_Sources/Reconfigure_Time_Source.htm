Reconfigurar a fonte de tempo

Esta ação permite alterar as configurações para uma Fonte de Tempo existente, especificada no campo "Fonte de Tempo".

As configurações fornecidas aqui são as mesmas da Create Time Source, exceto para a matriz.

Ele reiniciará e desativará a Fonte de Tempo, de modo que você terá que reiniciá-la.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Time Source O Identificação da fonte de tempo para reconfigurar
Period A duração do período da Fonte de Tempo, quanto tempo leva para expirar
Units As unidades em que o período é expresso (segundos ou quadros)
Callback O method para ligar quando a Fonte de Tempo expirar
Arguments OPCIONAL Uma matriz contendo os argumentos para passar para a função de retorno de chamada
Repetitions OPTATIVA Quantas vezes a Fonte de Tempo deve funcionar no total, ou -1 para repetição indefinida
Expiry Type OPCIONAL O tipo de expiração para a Fonte de Tempo

Voltar: Ações de fonte de tempo
