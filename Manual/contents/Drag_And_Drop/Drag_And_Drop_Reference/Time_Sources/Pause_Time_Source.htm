Fonte de tempo de pausa

Esta ação faz uma pausa na Fonte de Tempo dada.

Você pode retomá-lo mais tarde usando a Start Time Source.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Time Source O Identificação da fonte de tempo para fazer uma pausa

Exemplo:


Este bloco de código seria idealmente executado em um evento da Key Press.

Ele verifica se o estado de uma Fonte de Tempo está "ativo", e nesse caso, ele faz uma pausa nessa Fonte de Tempo.

Entretanto, se o estado for "pausado", ele retoma a Fonte de Tempo.
