Saltar ao ponto

Esta ação é utilizada para mover uma instância de um local para outro. A instância não se moverá suavemente para a nova posição, mas "saltará" para ser colocada lá, portanto, se os valores forem suficientemente grandes, então ela "saltará" qualquer outra instâncias no caminho. Você pode definir valores absolutos para mover a instância, ou você pode marcar o "relativo" flag para que a instância se mova em relação à posição atual. Por exemplo, se você quiser que a instância salte apenas 10 pixels para a esquerda, você definiria o valor x para -10 e o valor y para 0 e marque relativo para ambos. Você pode usar valores negativos para coordenadas absolutas e a instância será movida para fora da área da base room.

Argumentos:

Argumento Descrição
x A posição x para saltar para
y A posição y para saltar para
