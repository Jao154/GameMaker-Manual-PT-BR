O Menu Arquivo

O file menu é onde você pode escolher mudar, salvar ou importar projetos, assim como definir as preferências do GameMaker IDE. As opções disponíveis são:

  • Novo projeto - Isto é para criar um novo projeto a partir do zero no GameMaker. Ele abrirá o diálogo "Salvar" e o levará a escolher um local para salvar o novo projeto. Uma vez que você tenha dado um nome ao projeto e um local para salvá-lo, você será levado a um novo espaço de trabalho em branco para começar a trabalhar nele. Os arquivos do projeto são salvos com o arquivo *.yyp extension .
  • Open Project - Isto abrirá um explorador de arquivos para que você selecione um arquivo de projeto *.yyz previamente criado para carregamento. Observe que na importação, se o projeto foi criado utilizando uma versão inferior à 2.3.0, então você receberá a seguinte mensagem de aviso: Se você optar por prosseguir, então um explorador de arquivos será aberto e você poderá dar ao projeto um nome e localização atualizados para salvá-lo, de modo que o projeto original não será sobregravado. Uma vez que o projeto tenha sido convertido para o novo formato 2.3.0+, ele será aberto no IDE e você receberá uma mensagem dizendo que a conversão foi bem sucedida. Você deve verificar o relatório de compatibilidade que foi gerado pelo processo de conversão e ver se há algo que você precisa consertar antes de tentar executar ou compilar o projeto. Note que se você receber repetidos erros ao tentar abrir um projeto feito com uma versão anterior à 2.3.0, então você deve contatar YoYo Games (usando a opção "Reportar um erro" do menu Ajuda), fornecendo o arquivo original *.yyp que você está tentando converter.
  • Projeto de importação - Aqui você pode importar projetos que tenham sido previamente exportados como um arquivo *.yyz (veja "Projeto de exportação", abaixo), ou - se você tiver um projeto feito com o GameMaker legado: Studio 1.4 - você pode selecionar esta opção para importar o arquivo *.gmx ou o arquivo *.gmz a partir dessa linha de produtos. Como na abertura de um projeto, se o *.yyz foi criado com uma versão do GameMaker antes da atualização 2.3.0, então ele precisará ser convertido conforme o formato do arquivo do projeto mudou com essa atualização.

NOTA: Para importar um projeto anterior GameMaker para o GameMaker ele deve ter sido criado usando a versão 1.4, e mesmo assim ele pode não funcionar diretamente na importação. Muitas coisas são diferentes entre o GameMaker e o GameMaker: Studio 1.4, e por isso seus projetos mais antigos podem precisar de alguns ajustes para que eles funcionem. Consulte o artigo Portando um GM:S 1.4 GameMaker na Base de Conhecimento GameMaker e a seção do manual que cobre Funções Obsoletas e Scripts de Compatibilidade.

IMPORTANTE! Importar um projeto *.yyz que contenha recursos font pode exigir que você "regenere" o fonts, removendo-os/adicionando-os, ou mudando o font no editor font e, em seguida, colocando-o de volta novamente. Isto porque as informações font existentes são retidas pelo *.yyz quando são exportadas, e, na importação posterior ou em outra máquina, o projeto pode não ter o fonts necessário já instalado (ou pode ter um font diferente com o mesmo nome). Isto só deve ser feito se você estiver vendo problemas com a forma como o font se torna após a importação.

  • Projetos Recentes - Você pode usar esta opção para visualizar projetos usados recentemente. Selecionando-o, você reabrirá a página inicial (caso você a tenha fechado) e poderá selecionar um projeto usado recentemente a partir da lista encontrada lá.
  • Save Project - Isto salvará seu jogo no mesmo local que você definiu quando criou o projeto (ou de onde você o abriu).
  • Salvar projeto como - O mesmo que acima, somente você especifica a localização e o novo nome do projeto.
  • Projeto de exportação - Isto permite exportar seu projeto como um arquivo ou como um modelo:
    • YYZ - Isto irá exportar o projeto como um único arquivo *.yyz e é muito útil para enviar cópias para as pessoas ou para salvar backups.
    • Como modelo - Isto lhe permitirá exportar seu projeto como um modelo, que poderá ser utilizado posteriormente como base para outros projetos na página inicial. Selecionando esta opção, você verá o seguinte prompt:
      Você deve primeiro clicar em "Editar Informações do Modelo" para garantir que as informações de seu modelo estejam corretas, e então selecionar "Exportar" ao abrir este prompt uma segunda vez para salvá-lo como um modelo. Você então o verá no menu Modelo na tela inicial.
  • Preferências - Selecionando isto, você será levado para a Janela de Preferências onde você poderá configurar como o GameMaker se parece e se sente.
  • Nova IDE - Quando você selecionar esta novaIDE, uma nova instância do IDE será aberta para você usar (ela será aberta na página inicial).
  • Minha conta - Esta opção abrirá seu navegador da web padrão e o levará para sua página YoYo.
  • Logout - Selecionando isto, você sairá do GameMaker. Você precisará fazer logon novamente com a mesma licença (ou qualquer outra) para poder usar o produto mais tarde.

IMPORTANTE! Se você estiver usando uma licença gratuita para o GameMaker e depois comprar uma das outras licenças, talvez seja necessário sair do site IDE e depois reiniciar a sessão no GameMaker e voltar a entrar novamente para certificar-se de que a nova licença seja aplicada corretamente (embora isto deva acontecer automaticamente).

  • Saída - Saída do GameMaker para a área de trabalho.
