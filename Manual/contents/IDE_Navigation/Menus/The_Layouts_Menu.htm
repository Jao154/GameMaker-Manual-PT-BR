O Menu de Layouts

Este menu está relacionado a salvar e carregar diferentes layouts IDE. Você pode configurar o IDE como quiser, com certos elementos (como o Asset Browser, ou a Saída Windows) disposta de forma diferente do padrão. Isto permite que você configure o GameMaker de acordo com seu fluxo de trabalho e com as opções neste menu você pode salvar os layouts atuais do projeto como um arquivo *.xml que pode então ser carregado em qualquer projeto para configurar os layouts para serem os mesmos. As opções neste menu são as seguintes:

  • Load Layout - Isto abrirá um explorador de arquivos onde você poderá selecionar um arquivo *.xml de layout previamente salvo para carregamento.
  • Save Layout - Isto abrirá um explorador de arquivos onde você poderá nomear e salvar o atual layout IDE como um arquivo *.xml.
  • Reset Layout - Isto irá redefinir o atual layout IDE para as configurações padrão.
