O Menu Construir

O menu de construção é onde você pode escolher construir seu projeto tanto para testes, debugging ou como executável final. As opções disponíveis são:

  • Run - Execute seu projeto atual na plataforma alvo escolhida para os testes. Para as diferentes opções de compilação disponíveis ao compilar, consulte a seção sobre Compilação. Note que este comando é o mesmo que pressionar o botão Executar no IDE.
  • Debug - Execute seu projeto atual para testes com o debugger para que você possa ver informações detalhadas de desempenho e depurar quaisquer erros ou problemas que você possa ter. Para mais informações em debugging, consulte a seção Debugging. Note que este comando é o mesmo que pressionar o botão Debug no IDE.
  • Clean - GameMaker mantém um Cache de arquivos para ajudar a acelerar o tempo de compilação durante os testes. No entanto, às vezes um cache "velho" pode levar a erros estranhos e inesperados em seu jogo, em cujo caso a primeira coisa que você deve fazer é limpar o cache a partir daqui e fazer um novo teste. Também é uma boa idéia limpar o cache antes de criar um executável final para a plataforma alvo. Note que este comando é o mesmo que pressionar o botão Clean Cache no IDE.
  • Criar Executável - Isto compilará o projeto atual e criará um pacote executável (ou pacotes) para a plataforma alvo. Primeiro abrirá o explorador de arquivos para que você possa selecionar onde deseja salvar o projeto e depois compilar o jogo. Veja a seção sobre Compilação para mais informações sobre como configurar os alvos da compilação. Note que este comando é o mesmo que pressionar o botão Criar Executável no IDE.
