O menu Marketplace

Além de criar seu próprio assets e código para seus projetos, você também tem acesso ao GameMaker Marketplace onde você pode baixar tanto grátis quanto pago assets para ser adicionado a seus projetos. Você pode saber mais sobre o Marketplace em a seção sobre o GameMaker Marketplace, e as opções deste menu são dadas abaixo:

  • Ir para o Marketplace - Abra a janela Marketplace onde você pode procurar por assets.
  • Minha biblioteca - Veja sua biblioteca de conteúdo Marketplace atual.
  • Meu histórico de compras - Esta opção abrirá um navegador e o levará para a página da conta do usuário logado para que você possa ver o histórico de compras para aquela conta.
  • Update Existing Package - Esta opção abrirá o Update Package Wizard para que você possa atualizar um pacote existente Marketplace que você tenha criado anteriormente.
  • Create Package - Com esta opção você pode abrir o Create Package Wizard para criar um novo pacote asset ou tutorial para o Marketplace.
