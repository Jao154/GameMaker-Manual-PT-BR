O Menu de Ferramentas

Este menu permite que você abra as diversas ferramentas autônomas que estão disponíveis dentro do GameMaker, com as seguintes opções:

  • Grupos de Textura - Veja a janela do Grupo de Textura.
  • Grupos de Áudio - Veja a janela do Grupo de Áudio.
  • Sound Mixer - Veja a janela do Sound Mixer.
  • Gerar Imagens do Projeto - Abre a janela Gerar Imagens do Projeto.
  • Create Local Package - Abre a janela do Asset Packager, para que você possa fazer e salvar pacotes locais asset para uso futuro.
  • Importar pacote local - Abre uma janela do explorador de arquivos onde você pode então selecionar um pacote local Asset e importá-lo, seja adicionando-o ao projeto atual, ou criando um novo projeto para ele.
