Opções de Repositório

Tendo criado o executável Git e ativado Source Control para seu projeto, você agora precisa conectar o GameMaker IDE a um repositório, o que é feito de uma de duas maneiras:

  • Criar um repositório de projetos
  • Clonar um repositório existente

No primeiro caso, criando um repositório de projetos, isto criará um repositório Git diretamente onde seu projeto está. Na linguagem Git, este é um "repositório não nu", então Push/Pull não faz sentido neste caso, mas você ainda pode se comprometer e acompanhar as mudanças do projeto, bem como reverter as mudanças e revisões, se necessário. Para configurar isto, você só precisa clicar em Source Control > Create Project Repository ( Controle de Fonte > Criar Repositório de Projetos ) no menu principal.

Como uma função de conveniência, você também pode clonar um repositório externo através do IDE. Novamente, você precisará do nome de usuário e senha do repositório que você vai conectar para configurar em suas preferências de identidade, e então é apenas um caso de clicar em Controle de Fonte > Clone Repositório.
