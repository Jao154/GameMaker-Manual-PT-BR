Configurando o Plugin Git

Antes de podermos usar o plugin Git, precisamos configurar o executável Git para usar, entrando na seção Plugins - Source Control (Git) das Preferências:

O executável Git path deve ser preenchido automaticamente se for encontrado por GameMaker, porém se o campo estiver vazio, você pode clicar no botão e selecionar manualmente o arquivo git.exe incluído na pasta cmd de sua instalação Git, que normalmente está dentro de C:\Program Files\Git\.

Para o restante das opções Git, consulte a página de Preferências de Plugin.
