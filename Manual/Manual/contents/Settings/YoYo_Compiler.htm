Compilador YoYo

O compilador YoYo (YYC) é um compilador especial para criar pacotes executáveis que utilizam código de máquina em vez de código interpretado e um runner (que é o que o compilador GameMaker VM utiliza). O código compilado é mais rápido de ser executado, mas leva mais tempo para compilar o executável.

Licença

Sua licença GameMaker pode ou não incluir o Compilador YoYo. Para descobrir, veja a página do Centro de Ajuda Qual Assinatura é Adequada para Mim?

Configurando o SDK

Para que o Compilador YoYo possa compilar para a plataforma alvo, você precisará ter os SDKscorretos instalados.

Para uma visão geral sobre isto, consulte a seguinte página do Centro de Ajuda: GameMaker SDKs requeridos.

As informações sobre os SDKs necessários por plataforma também podem ser encontradas no Centro de Ajuda:

  • Ópera GX
  • Windows
  • macOS
  • Ubuntu
  • Android
  • iOS (Incluindo iPadOS)
  • tvOS
