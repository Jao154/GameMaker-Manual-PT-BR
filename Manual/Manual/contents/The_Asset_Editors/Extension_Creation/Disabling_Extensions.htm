Como desativar Extensions?

GameMaker tem extensions para algumas plataformas alvo, por exemplo, Steamworks e GDK.

Se você estiver usando estes dois extensions, você vai querer desativar o GDK extension ao construir para Steam, e vice-versa.

Este problema não se limita a estes dois extensions, e se aplica a qualquer extension que se destine a um alvo específico.

Desativação extensions seletivamente

Primeiro, estabeleça uma configuração para cada uma de suas plataformas alvo, caso ainda não o tenha feito.

Este exemplo só usa Steamworks e GDK.

Selecione a configuração para uma plataforma de destino. Vamos supor que você selecionou a configuração GDK.

Abra um extension que não deve ser usado para a GDK. Para este exemplo, abra o Steamworks extension.

Desativar todos os alvos sob Copies To. Isto assegurará que o Steamworks extension não seja executado ao utilizar a configuração GDK.

Agora, switch para a configuração de sua Steamworks. Abra o GDK extension, e desative todas as suas metas.

Feito!

Agora, sempre que você construir para GDK, use a configuração GDK; quando construir para Steam, use a configuração Steamworks, e assim por diante.

Usando configurações desta forma, é possível controlar quais extensions são utilizadas para construir quais plataformas.

Se você tiver mais extensions, dê-lhes o mesmo tratamento: desabilite suas exportações em configurações que não precisam delas.

Se você tiver mais plataformas alvo, adicione configurações para elas e modifique quais extensions são usadas para elas.
