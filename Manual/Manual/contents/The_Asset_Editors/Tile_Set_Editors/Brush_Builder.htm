O construtor da escova Tile Set

Por padrão, quando você "pinta" tiles em uma camada de mapa de azulejos no editor room, você seleciona um único tile e pinta com isso. Entretanto, tile sets é quase sempre projetado para ter seções que se encaixam de diferentes maneiras para formar áreas ou itens inteiros. Por exemplo, um RPG tile set pode ter o recurso de paisagem tiles que pode ser conectado para criar características maiores ou menores - como árvores e edifícios - dependendo do número de azulejos usado. Agora, colocar múltiplas características como esta em uma camada room exigiria que você voltasse para frente e para trás muitas vezes para mudar tile, o que não é bom para o seu fluxo de trabalho. Para resolver isso, o GameMaker permite que você crie tile Pincéis em o editor Tile Set, disponível quando você clica em no botão " Brush Builder":

No Brush Builder você tem o original tile set à esquerda e uma "lona" em branco à direita. Agora você pode selecionar qualquer tile da esquerda e pinte-o à direita para criar "pincéis" personalizados que você pode usar no editor room. Observe que você pode clicar e segurar o botão esquerdo do mouse e depois arrastar o tile set para selecionar vários tiles para pintar na tela do pincel e você também pode segurar / + para adicionar um indivíduo específico tiles ou segurar + para remover um tiles específico.

A imagem a seguir mostra nossas características paisagísticas anteriormente mencionadas como um exemplo de três escovas personalizadas feitas de um único conjunto de azulejos:

À direita você pode ver três características que fizemos (destacado com uma caixa laranja na imagem). Observe como deixamos uma lacuna de um tile entre cada característica - isto porque qualquer grupo de tiles será tratado como um único pincel no editor room, então deixamos uma lacuna de um tile para mostrar que cada conjunto é um conjunto distinto. escova que queremos criar. Ao criar seus pincéis, você pinta com o botão esquerdo do mouse e apaga com o botão direito do mouse . Você também pode ampliar a folha tile ou a tela do pincel usando / e a roda do mouse quando o mouse está sobre uma das janelas, ou você pode navegar usando + ou o botão do meio do mouse e arrastar. Observe que você também pode usar as ferramentas de zoom na Caixa de ferramentas dentro das janelas do editor para mudar os níveis de zoom de qualquer uma das janelas, bem como alternar a visibilidade e a cor da grade.

Na parte superior direita você pode ver a ferramenta selecionada atualmente, e também pode definir o tamanho do pincel com o qual você quer pintar. O tamanho padrão é 1, que é um único tile, mas se você defini-lo para valores mais altos, então você pode pintar (e apagar) com um escova maior composta do selecionado tile repetido, como mostrado na imagem abaixo:

A caixa de ferramentas é onde você pode selecionar a ferramenta a ser usada para muitas tarefas diferentes no editor Tile Set, algumas das quais dependerão se você tem qualquer coisa definida em sua biblioteca de azulejos automotivos. Um breve esboço de cada ferramenta é dado abaixo (note que quando você tiver selecionado uma camada tile no editor Room, então esta caixa de ferramentas é exibida na parte superior da sala workspace junto com ferramentas de desenho adicionais):

Lápis Esta é a ferramenta do lápis. Ela usa o tile selecionado para pintar na janela de pincéis com o botão esquerdo do mouse e você pode apagar com o botão direito do mouse .
Apagador Com a ferramenta de borracha você pode usar o botão esquerdo do mouse para apagar um dado tile na janela Pinceladas. Essencialmente, tudo isso faz com que o índice tile fique em 0, que é o reservado "vazio" tile.
Seleção Esta é a ferramenta de seleção, que pode ser usada para definir uma área da janela das escovas para trabalhar. Você pode clicar no botão esquerdo do mouse e depois arrastar o mouse para criar uma área retangular, ou você pode pressionar / + para adicionar múltiplas seleções e + para limpar uma parte da seleção. Para limpar toda a seleção, você pode pressionar . Quando você tiver uma área de uma janela de Pincéis selecionada, o restante das ferramentas (Lápis, Flip, Rotacionar, etc...) só funcionará dentro da área selecionada. Observe que você também pode selecionar tiles na janela, depois copiar ( / + "C"), cortar ( / + "X") e colar ( / + " V") eles, que então switch a ferramenta para o Lápis e permitir que você "pinte" com eles.
AutoTile Clicar nesta ferramenta permite o estilo Auto tiling paint. Quando este estiver ativo, você pode selecionar qualquer tile na aba Autotile Libraries, e então pintá-lo na janela Brushes e o GameMaker irá mudá-lo automaticamente. para combinar com os arredores tiles, desde que você tenha configurado corretamente a guia Auto Tile . Note que a seleção de um tile a partir do tile set que não faz parte das bibliotecas de autóteis irá redefinir a ferramenta de desenho para o padrão Ferramenta de lápis.
Flip Clicando na ferramenta Flip com o botão esquerdo do mouse (ou usando o atalho de teclado "X" ) virará o tile ou tiles atualmente selecionado para desenhar ao longo do eixo horizontal , sem mudar a ferramenta de desenho (se você tiver um pincel personalizado selecionado para desenhar, todo o pincel será virado). Se você não tiver nenhum tile selecionado para desenhar, mas tiver um grupo de tiles selecionado na janela do pincel, então a ferramenta de inverter inverterá o selecionado tiles.
Mirror Clicando na ferramenta Espelho com o botão esquerdo do mouse (ou usando o atalho de teclado " Y" ) espelhará a(s) telha(s) atual(is) selecionado para desenhar ao longo do eixo vertical , sem mudar a ferramenta de desenho (se você tiver um pincel personalizado selecionado para desenhar, o pincel inteiro se espelhará). Se você não tiver nenhum tile selecionado para desenhar, mas tem um grupo de tiles selecionado na janela do pincel, então a ferramenta Espelho espelhará o selecionado tiles.
Girar Clicando na ferramenta Rodar com o botão esquerdo do mouse (ou usando o atalho de teclado " Z" ), o(s) azulejo(s) atualmente selecionado para desenhar 90° no sentido horário, sem mudar a ferramenta de desenho (se você tiver um pincel personalizado selecionado para desenhar, o pincel inteiro irá girar). Se você não tem nenhum tile selecionado para desenhar, mas tem um grupo de tiles selecionado no janela de escova, então a ferramenta Rotate irá girar a ferramenta selecionada tiles.

Abaixo das ferramentas, você pode encontrar duas seções diferentes para selecionar qualquer azulejo automotivo ou azulejosanimados que foram criados usando a atual imagem tile set. Um único sprite que é usado para um tile set pode ter muitas, muitas, imagens de uma única célula nele, e estas podem ser combinadas no editor de Animação ou Autotile para criar escovas personalizadas que aparecerão nestas seções e podem ser usadas em conjunto com estático regular tiles para criar escovas (note que um animado tile animará independentemente de ter sido selecionado da biblioteca ou da base tile set).

Uma vez instalados todos os pincéis necessários, você pode então usá-los para pintar tiles em qualquer camada do mapa tile dentro do Room Editor.
