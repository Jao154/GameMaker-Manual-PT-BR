Azulejos automotivos

O recurso Auto Tile é uma ferramenta incrivelmente poderosa que pode ser usada para facilitar muito a construção de níveis, seja para um projeto de cima para baixo ou de lado para baixo. Essencialmente, você cria uma biblioteca auto tile, e depois sempre que você estabelece um tile dessa biblioteca que "conectará" com aqueles tiles que estão ao seu redor para criar uma parede ou plataforma sem costura.

Antes de começar a usar a função auto tile, é muito importante que você tenha configurado corretamente o tile set sprite e que você escolha um tipo apropriado de auto tile. Os tipos disponíveis para você são 47 tile auto tiles, ou 16 tile auto tiles, com (em geral) 16 tiles sendo usado para top down ou paisagens (já que dão transições mais agradáveis) e 47 sendo usado para platformer/side em vistas ou top down onde as paredes exigem múltiplas conexões, mas que não é de forma alguma uma regra difícil e rápida e dependerá de como você quer que seu projeto final seja.

Note que o tile set sprite que está sendo usado tem TODO o tiles que necessitamos, e também pode ter outros tiles, já que você pode escolher e escolher quais devem estar no auto tile. Agora você pode clicar no botão 47 tile na área Templates para criar uma nova entrada na biblioteca 47 tile e ela será levante o modelo a ser utilizado para selecionar o tiles que você precisa:

No modelo, a área cinza claro representa a borda externa do tile que está sendo usado e o cinza escuro é a área de "preenchimento". Para adicionar um tile no modelo, basta clicar na primeira célula de modelo disponível e depois selecionar o tile para adicionar a partir da esquerda:

Se você cometer um erro, simplesmente selecione o autotile que você quer remover e selecione o "vazio" tile de o tile set. Depois de acrescentá-los todos, você pode verificar o tiles no modelo, clicando no botão "Show Template" no e isto irá alternar a sobreposição do modelo no autotile final, como mostrado na imagem abaixo:

Se você quiser fazer um 16 tile autotile, o procedimento é exatamente o mesmo que o explicado acima, apenas em vez de usar o modelo 47 tile você usa o modelo 16 tile:

Agora temos algum auto tiles em nossa biblioteca auto tile retirada de uma tile set. Estes podem ser usados no editor room simplesmente criando um azulejo camada de mapa e depois selecionando um dos auto tiles da biblioteca (que selecionará automaticamente o pincel auto tile do topo do editor room ) e pintando em seu room. O tiles agora se conectará "auto-magicamente" para criar mapas tile devidamente sequenciados.

Favor notar que o comportamento de um autotile em torno das bordas room dependerá do botão Open Or Closed Edges . Por padrão, quando um auto tile é colocado ao longo da borda de um room, ele escolherá um tile como se o exterior do room estivesse vazio (isto é: ele dará ao room uma "borda"). Entretanto, se você clicar neste botão, o autotile irá tile o room como se a área externa estivesse preenchida com tiles e assim escolher tiles que se misturam e não têm "borda".
