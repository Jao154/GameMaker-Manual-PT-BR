Menus

O topo do GameMaker IDE é ocupado pela barra de menu. Aqui você encontra uma série de menus suspensos que podem ser usados para navegar o IDE, realizar tarefas ou abrir janelas adicionais com ferramentas para ajudá-lo a criar seus jogos. Note que listamos aqui os menus padrão que estão disponíveis, mas alguns dos editores do asset adicionarão menus adicionais dependendo do contexto. Estes menus adicionais Os menus são explicados nas páginas relevantes para o editor que os criou.

Os menus padrão disponíveis são:

  • O Menu Arquivo
  • O menu de edição
  • O Menu Construir
  • O menu Windows
  • O Menu de Ferramentas
  • O menu Marketplace
  • O Menu de Layouts
  • O Menu Ajuda
