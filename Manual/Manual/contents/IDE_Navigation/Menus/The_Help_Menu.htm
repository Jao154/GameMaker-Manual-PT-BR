O Menu Ajuda

Este menu está relacionado à obtenção de ajuda ao usar o GameMaker. Ele contém os seguintes itens de menu:

  • Manual Aberto - Abre o manual de ajuda. Note que clicar no botão de ajuda na barra de menu também abrirá o documento, assim como pressionar .
  • Base de Conhecimento - Isto o levará à Base de ConhecimentoGameMaker onde você pode encontrar artigos sobre configuração de plataformas, assuntos conhecidos, usando o GameMaker e uma série de outras coisas.
  • Report A Bug - Caso você encontre algum bug ou tenha algum problema durante o uso do GameMaker, você pode reportá-lo aqui.
  • Open Log In Explorer - Isto abre o diretório em seu sistema onde você pode encontrar o arquivo ui.log, que normalmente é necessário para enviar relatórios de bugs relacionados ao IDE.
  • GameMaker.io - Abre o site GameMaker.
  • Comunidade GameMaker - Isto abrirá os fóruns da Comunidade GameMaker em seu navegador. Note que este é um fórum de execução comunitária , portanto, se você tiver um relatório de bug ou precisar de assistência técnica, você deve ir primeiro ao Site Oficial de Suporte (veja acima).
  • Notas de Lançamento - Isto abrirá as Notas de Lançamento para GameMaker em seu navegador. Caso você precise reverter uma atualização, você pode obter instaladores de versões anteriores clicando no link do número da versão nesta página.
  • Atual Runtime Notas de Lançamento - Uma vez que o Runtime pode ser atualizado independentemente do IDE, ele tem seu próprio conjunto de Notas de Lançamento em Tempo de Execução. Selecionando esta opção, elas serão abertas em seu navegador, mas note que o último runtime pode não ser o que você está usando. Para saber em que runtime você está, ou para obter informações sobre como adicionar, remover ou ativar runtimes, consulte a seção Preferências de Tempo de Execução.
  • SDKs necessários - Devido à natureza multiplataforma do GameMaker, vários SDKs de terceiros são necessários. Clicando aqui, você encontrará um artigo de apoio que detalha o que você precisa em cada plataforma.
  • Open Project In Explorer - Isto abrirá o local de salvamento do projeto atualmente carregado no OS file explorer.
  • Refresh System Fonts - Isto refrescará o sistema fonts que o GameMaker é capaz de ver, e é útil para detectar novos fonts que você instalou enquanto o IDE estava aberto.
  • Licenças - Isto abre uma janela contendo licenças para todas as bibliotecas de terceiros utilizadas no GameMaker.
  • Sobre - Isto abre a janela "Sobre o GameMaker" onde você pode obter informações sobre a versão do produto (essencial para reportar quaisquer bugs ou problemas) e ver os créditos.
