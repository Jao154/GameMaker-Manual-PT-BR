Criação de conjuntos de azulejos

Osconjuntos de azulejos são criados a partir de sprites, mas também são classificados como um tipo separado asset, uma vez que a forma como o GameMaker lida com eles é um pouco diferente tanto para sprites como para objects. Portanto, um tile set é uma coleção de tiles criada juntos em um sprite, e um tile é simplesmente um recurso gráfico que é adicionado a um GameMaker room como parte de um mapa de azulejos. Estes tile mapas são então atraídos para o com muito pouco processamento de CPU/GPU, tornando-os ideais para projetar qualquer item estático em níveis.

Para tornar o processamento tiles tão rápido, eles têm certas restrições, sendo que a mais básica é que eles só podem ser retangulares ou quadrados (se você quiser qualquer outra coisa além disto, então você precisará criar um duende regular e usar a camada de ativos no editor room, mas discutiremos isso na seção posterior sobre salas). Outras restrições incluem o fato de que elas só podem ser giradas em 90°, ou espelhadas ou invertidas, e elas podem ser animado somente com quadros que numerem uma potência de 2 (ou seja: 2, 4, 8, 16, etc...).

Note que o primeiro tile está em branco nessa imagem. A forma como o mapa tile para o room é armazenado no GameMaker significa que o primeiro tile sempre precisará estar em branco, pois é essencialmente o "apagar" tile ou o "espaço vazio" tile (este "em branco" tile terá um valor de índice de 0 - tiles são indexados com um valor que começa em 0 e conta da esquerda para a direita, de cima para baixo). Portanto, ao criar seu tile sets, o canto superior esquerdo tile sempre precisará estar vazio desta forma. Observe também que enquanto a imagem acima mostra uma parede completa tile set com 47 tiles, você pode não precisa de tantos que você pode girar e virar tiles ao colocá-los no editor room mais tarde.

Com o sprite criado, podemos ir em frente e definir o próprio tile set, o que significa que primeiro você tem que criar um novo recurso tile set clicando com o botão direito o recurso Tile Set pasta e selecionando Criar. Isto abrirá o Editor de Conjunto de Azulejos onde você poderá então adicionar o sprite e definir suas propriedades:

Ajustamos o tamanho tile set para 64px, e deixamos o resto das opções em seus valores padrão, já que não temos necessidade de offsets ou algo parecido (os valores de offset são úteis para qualquer tile set sprite que tenha espaços ao redor de cada um dos tiles). Você vai querer nomear o tile set asset usando um prefixo apropriado como " ts_" para que você possa identificá-lo facilmente mais tarde, e note que as mesmas restrições se aplicam ao sprites, ou seja: o nome não pode começar com um número, e só pode usar o símbolo "_" e os caracteres alfanuméricos.
