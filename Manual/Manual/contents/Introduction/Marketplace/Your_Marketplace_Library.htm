Sua Marketplace Biblioteca

A janela Biblioteca lhe mostra os pacotes que você comprou e lhe dá a opção de baixá-los e/ou adicioná-los ao seu projeto.À esquerda estão os diferentes filtros de categoria que você pode usar para ver o que você tem em cada um, e depois no meio você tem a lista de assets em sua biblioteca. Cada um deles pode ser selecionado para que você possa ver uma visão geral do que faz (à direita) e todos eles têm ícones para visualizar, baixar ou instalar o pacote asset em um projeto. Note que você tem dois filtros adicionais no topo da janela principal onde você pode selecionar para ver apenas aqueles assets que foram criados especificamente para o GameMaker, bem como um filtro para mostrar apenas aqueles assets que você já baixou para sua máquina. Você também pode atualizar a lista de assets exibida a qualquer momento, clicando no botão Atualizar .

Para adicionar um pacote asset ao seu projeto atual, você deve primeiro baixá-lo clicando no botão Download e quando o download estiver concluído, você pode clicar no botão Importar para adicioná-lo ao seu projeto. Adicionar um pacote desta forma criará um extension dentro de seu projeto atual, e lhe pedirá para selecionar entre os diferentes recursos do pacote asset para escolher quais você deseja adicionar (ou você pode importar todos eles). Você também pode selecionar a opção Importar todos os recursos para um novo projeto, e em vez de adicionar o assets ao seu projeto atual, o GameMaker irá solicitar que você dê um novo nome ao projeto e salve o asset dentro desse projeto. Note que se você for diretamente à Biblioteca a partir da Página Inicial, você encontrará a maioria dos assets que não podem ser adicionados individualmente, e em vez disso você será forçado a usar o botão Add All. Isto porque você não tem um projeto para adicionar o assets e por isso o GameMaker deve criar um novo projeto para você adicionar todo o assets no pacote para (ao clicar no botão Importar, você será solicitado a salvar o novo projeto com o nome de sua escolha).Com o assets adicionado ao seu navegador Asset, você pode agora ir em frente e usá-los em seu jogo como usaria aqueles assets que você mesmo criou. Observe que os ativos baixados do Marketplace são para uso individual e não podem ser compartilhados com terceiros - embora você possa fazer backup deles e copiá-los para outros locais por segurança - a menos que o asset tenha um EULA personalizado que declare o contrário.

NOTA: As pessoas que criam os pacotes asset para o Marketplace trabalham muito para criar um grande recurso para você usar, portanto, considere avaliar o assets que você compra quando você os tiver testado ou se você os usar. Isto incentiva o desenvolvedor e é também um guia útil para outros usuários quanto à qualidade do assets disponível. Você pode avaliar facilmente seu assets baixado da janela Biblioteca.
