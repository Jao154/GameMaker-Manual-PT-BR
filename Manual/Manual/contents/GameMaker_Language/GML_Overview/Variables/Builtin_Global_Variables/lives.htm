vive

Esta variável é global no escopo e é usada para manter um valor numérico que normalmente é usado para a vida do jogador. Esta variável foi projetada apenas para suportar projetos legados de versões anteriores do GameMaker e deve não ser utilizado em novos projetos como pode ser depreciado no futuro.

Sintaxe:

lives;

Devoluções:

Real (single precision floating point value)

Exemplo

if (lives < 5)
{
    lives += 1;
}

O código acima verifica a variável lives e se ela for menor que 5, 1 é adicionado a ela.
