Visão geral doCódigo GML

Esta seção do manual contém todas as informações necessárias para compreender e usar Código GML. As funções runtime neste idioma podem ser usadas para criar seus jogos e são adicionadas em objects a partir do Editor de objetosembora também possa ser usado junto com GML Visual. Abaixo você pode ver uma imagem típica de um object com o editor de código aberto em um evento:

Cada evento tem sua própria guia no editor e você pode adicionar, editar ou remover código deles a qualquer momento (para mais informações sobre eventos, veja Eventos Objeto). O código em si deve ter uma estrutura básica e pode conter índices de recursos, variáveis, funções, expressões, palavras-chave, etc., todos os quais são explicados nas seções abaixo. Se você é um novato em programação ou fazendo o switch do GML Visual, é recomendável que você comece com a página em Estrutura básica do código e depois ler todas as outras páginas desta seção, testando o código de cada uma delas dentro GameMaker em si.

Você pode usar Feather para preenchimento inteligente de código e relatório de erros em seu GML Código scripts.
