Estrutura básica do código

O código é escrito em blocos e um bloco de código típico consiste em um conjunto de instruções, chamadas dedeclarações, que são então interpretadas pelo GameMaker e usadas para fazer algo acontecer dentro de seu jogo. Esse "algo" pode ser tão simples como adicionar 2 e 2 para obter 4, ou tão complexo como fazer um inimigo fugir quando sua saúde fica abaixo de um certo valor. A estrutura real do programa pode variar muito, dependendo das funções que ele usa, mas, decomposto ao básico, ele apenas se parece com isto:

<statement>;
<statement>;
...

As declarações devem ser separadas com um símbolo ';' para evitar erros com declarações variáveis e para manter seu código limpo e arrumado, e podem consistir em declarações variáveis,expressões e chamadas para funções específicas. Você também pode "agrupar" declarações como um bloco usando os colchetes {} para que elas funcionem juntas, como no exemplo condicional a seguir:

if (<conditional>)
{
    <statement>;
    <statement>;
    ...
}

O GameMaker Language também aceitará begin e end em vez dos colchetes {}, embora esta não seja normalmente a forma mais comum de fazê-lo:

if (<conditional>)
begin
    <statement>;
    <statement>;
    ...
end
