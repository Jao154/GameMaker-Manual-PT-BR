continuar

A declaração continue tem a seguinte sintaxe básica:

continue;

Se usado dentro de uma declaração que forma um loop (for, repeat, while ou do / until), terminará imediatamente a atual iteração e voltará ao início do loop iniciando uma nova iteração e omitindo qualquer código que venha após o continue dentro do loop. Também pode ser usado dentro do with declaração, onde fará com que o código salte para a próxima instância e seja executado novamente. Observe que se continue for utilizado fora de qualquer um destes contextos, isso dará um erro.

Abaixo está um exemplo de utilização em um laço for:

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

var file = file_text_open_read("Game_Data.txt");
