Código GML

O GameMaker Language (também chamado simplesmente GML) é a linguagem proprietária GameMaker scripting. Esta linguagem é estruturada para permitir que os usuários criem seus jogos de forma intuitiva e flexível, enquanto oferece todo o poder de qualquer outra grande linguagem de programação. Ela também é a base do GML Visual e pode ser usada em conjunto com ele, se necessário.
