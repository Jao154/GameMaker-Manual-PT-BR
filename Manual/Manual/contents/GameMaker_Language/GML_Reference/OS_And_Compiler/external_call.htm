chamada_externa

Se você criou uma chamada de função externa para um dll ou dylib usando external_define()Você pode usar esta função para então chamá-la. Você fornece o nome da função previamente definida assim como cada um dos argumentos que ela requer (cada argumento deve ser do tipo correto, seja real ou string) e a função retorna o resultado da chamada externa.

Sintaxe:

external_call(id, args[0...15]);

Argumento Tipo Descrição
id Função externa The name of the function that you want to call
args[0...10] Real ou String The different arguments that you want to pass through to the external function

Devoluções:

Variable (the type of value returned will depend on the defined function)

Exemplo:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

O código do exemplo acima chama uma função externa previamente definida e armazena o valor retornado em uma variável local.
