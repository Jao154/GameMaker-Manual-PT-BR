OS E Compilador

Ao criar jogos entre plataformas, muitas vezes é de vital importância que você obtenha os detalhes do sistema operacional que o dispositivo que está rodando seu jogo está usando. Coisas como o idioma, a versão ou o estado da rede podem ser usadas para adaptar seu jogo ao sistema que o está rodando e fazer com que a experiência do usuário final seja a melhor possível, portanto o GameMaker tem uma série de funções especiais que podem ser usadas para obter as informações necessárias.

As seguintes funções e variáveis existem para você usar para obter certos detalhes sobre o sistema operacional ou navegador que está rodando seu jogo:

  • os_browser
  • os_device
  • os_type
  • os_version
  • os_is_paused
  • os_is_network_connected
  • os_get_config
  • os_get_language
  • os_get_region
  • os_get_info
  • os_powersave_enable
  • os_lock_orientation
  • os_check_permission
  • os_request_permission

Algumas funções são pré-compiladas quando você executa seu jogo (ou seja: elas são usadas para definir como o jogo final será compilado), e algumas também são para uso quando o jogo foi compilado para obter detalhes específicos sobre o ambiente runtime. Aqui você pode encontrar a lista completa destas funções:

  • GM_build_date
  • Tipo GM_build_type
  • GM_version
  • GM_runtime_version
  • GM_projeto_nome_do_filme
  • gml_release_mode
  • gml_pragma
  • parameter_count
  • parameter_string
  • environment_get_variable

Em Windows, a resolução do agendador de linha Windows é definida para 1ms por padrão quando seu jogo for executado. As seguintes funções estão disponíveis para recuperar e alterar a resolução do agendador de linha Windows em tempo de execução:

  • agendador_resolução_get
  • conjunto_de_resolução_agendador

Finalmente, ao trabalhar com Extensões você pode precisar ser capaz de definir e chamar funções externas em runtime, e assim você usaria o seguinte:

NOTA: Estas são funções herdadas e você deve definir quaisquer extension funções dentro do próprio editor Extension.

NOTA: Estas funções são apenas para as plataformas alvo Windows e MacOS.

  • external_define
  • external_call
  • external_free
