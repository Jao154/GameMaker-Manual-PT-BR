externo_define

Esta função pode ser usada para definir uma chamada de função externa para uma dll específica (para Windows) ou dylib (para Mac). Este arquivo pode ser um arquivo incluído ou parte de uma dll extension.

Você fornece o nome (e path) do arquivo, depois o nome da função que você deseja definir. Em seguida, você precisa definir a convenção de chamada a ser utilizada (veja a lista de constantes abaixo), bem como o tipo de resultado a ser esperado (também uma constante, como listado abaixo).

Finalmente, você deve dar o número de argumentos que a função pode levar (de 0 a 15) e para cada um dos argumentos você deve especificar seu tipo também. Observe que para funções com 4 ou mais argumentos, todos eles devem ser do tipo ty_real.

NOTA: Isto é apenas para dll ou dylib que foram adicionados como "Included Files" ao GameMaker IDE. Não funcionará com aqueles arquivos adicionados como extensions, já que estes requerem que você defina as funções no próprio pacote extension.

Constantes do tipo de chamada externa
Constante Descrição
dll_cdecl Esta é a chamada padrão C, C++
dll_stdcall Esta é a chamada padrão da WinAPI (Windows dll apenas)
Constantes do tipo de dados externos
Constante Descrição
ty_real Um verdadeiro argumento numérico
ty_string Um argumento nulo string

Sintaxe:

external_define(dll, name, calltype, restype, argnumb, argtype[0], argtype[1], ...argtype[10]);

Argumento Tipo Descrição
dll Cordão The name of the DLL file (string)
name Cordão The name of the function (string)
calltype Constante de Chamada Externa Tipo (dll_*) The calling convention used
restype Tipo de dados externos Constante (ty_*) The type of the result to expect
argnumb Real The number of arguments (0 - 10)
argtype[0 ... 10] Tipo de dados externos Constante (ty_*) The types of the arguments being used

Devoluções:

External Function

Exemplo:

my_funcion = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);

O código do exemplo acima definirá uma função externa chamada "MyMin" com dois argumentos.
