os_powersave_enable

Com esta função, você pode ligar ou desligar as características de economia de energia do dispositivo. Isto é importante, pois certos jogos (por exemplo, aqueles que utilizam as funções de inclinação) podem não gerar eventos que o sistema operacional possa interpretar como sendo de entrada do usuário e, portanto, desligar a tela ou sair do jogo. Ao configurar esta função para false você pode desativar os recursos de economia de energia e garantir que a tela (e o jogo) estejam sempre funcionando.

NOTA: Isto se limita a iOS e Android alvos.

Sintaxe:

os_powersave_enable(flag)

ArgumentoTipo Descrição
flag Set to true ou false para ativar ou desativar as funções de powersave.

Devoluções:

Exemplo:

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

O código acima verifica o tipo de sistema operacional e se for Android ou um iOS, os recursos de economia de energia serão desativados.
