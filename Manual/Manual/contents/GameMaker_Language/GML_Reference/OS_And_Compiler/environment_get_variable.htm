ambiente_get_variable

Esta função retorna o valor (a string) da variável de ambiente com o nome dado (também um string). Você pode obter as variáveis de ambiente disponíveis em macOS e Ubuntu (Linux) digitando "env" no aplicativo terminal, e para informações sobre variáveis de ambiente Windows, se você estiver usando o prompt de comando digite "echo %PATH%", e usando o PowerShell é "ls env". Note que em macOS e Ubuntu (Linux) o "HOME". variável ambiental retornará o "~/" path que mapeia para "/Users/<username>" em macOS e "/home/<username>" em Ubuntu (Linux).

NOTA: Esta função é apenas para Windows, macOS e Ubuntu (Linux).

Sintaxe:

environment_get_variable(name);

ArgumentoTipo Descrição
name The name (a string) of the environment variable to check.

Devoluções:

Exemplo:

e_str = environment_get_variable("APPDATA");

O código acima retornará o path completo para a variável ambiental "%appdata%", que normalmente é "C:\Users\{username}\AppData\Roaming".
