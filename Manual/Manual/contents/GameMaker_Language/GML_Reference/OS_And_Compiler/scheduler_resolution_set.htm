conjunto_de_resolução_agendador

Esta função é utilizada para alterar a resolução do agendador de fios Windows. O valor da resolução fornecido no argumento precisa ser em milissegundos.

Por padrão, GameMaker usa uma resolução de 1 milissegundo, que é a menor resolução permitida e dá o melhor desempenho. Mudar a resolução para um valor mais alto pode afetar negativamente o desempenho, mas também pode reduzir o consumo de energia.

Você pode fornecer -1 como argumento para reverter a resolução para seu valor padrão (conforme definido por Windows). Note que mudar a resolução do agendador terá impacto em todos os processos em execução ao mesmo tempo em que seu jogo.

NOTA: Esta função é apenas para Windows .

Sintaxe:

scheduler_resolution_set(milliseconds);

ArgumentoTipo Descrição
milliseconds The new resolution value (in milliseconds) or -1 for default

Devoluções:

Exemplo:

scheduler_resolution_set(2);

Este exemplo estabelece a resolução do programador de linhas Windows para 2 milissegundos.
