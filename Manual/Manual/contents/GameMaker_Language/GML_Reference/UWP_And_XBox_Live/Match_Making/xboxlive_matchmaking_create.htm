xboxlive_matchmaking_create

Esta função criará uma sessão multiplayer usando o modelo Session Template, matchmaking hopper e Secure Device Association modelo, que você deveria ter criado antes no painel do XDP. O ID de usuário que você pode recuperar para um determinado bloco de jogos chamando a função xboxlive_user_for_pad() função e visibilidade será uma das constantes a seguir:

Constante de Visibilidade do Xbox Match
Constante Descrição
xboxlive_match_visibility_open especifica que a sessão pode ser acompanhada pelos outros.
xboxlive_match_visibility_private especifica que a sessão não é visível para os usuários que não são membros da sessão.
xboxlive_match_visibility_usetemplate especifica que o valor usado no modelo de sessão deve ser usado (isto normalmente deve ser usado porque não se pode sobrepor o valor de um modelo se este tiver sido definido no XDP).

A função retorna um valor único de Request ID, que pode então ser usado para identificar o Evento Assíncrono Social correto para esta função. Este evento será acionado quando a sessão tiver sido criada e conterá um Mapa DS na variável async_load com os seguintes pares de chaves/valores:

  • "requestid" - contém a identificação do pedido conforme retornado pela função de chamada
  • "status" - contém o string "session_created" para informá-lo sobre o tipo de evento que foi acionado.
  • "sessionid" - contém o ID da sessão, ou -1 em caso de falha
  • "erro" - contém 0 em caso de sucesso, ou -1 em caso de fracasso
  • "correlationid" - o manípulo da sessão para correlação com a função xboxlive_matchmaking_join_session()

Sintaxe:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Argumento Tipo Descrição
user_id ID do usuário do Xbox The user ID pointer to use
visibility Constante de Visibilidade do Xbox Match One of the constants listed above
template Cordão The name of the session template
hopper Cordão The name of the matchmaking hopper
sdatemplate Cordão The name of the secure device association template
matchattributes Real OPTIONAL Os atributos da partida

Devoluções:

Async Request ID

Exemplo:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

O acima mencionado recuperará o ID do usuário no gamepad dado e então iniciará uma sessão de matchmaking para eles.
