xboxlive_matchmaking_set_joinable_joinable_session

Esta função define uma sessão (previamente criada) para estar disponível para que outros usuários possam aderir através da interface do sistema. Um usuário só tem uma sessão acoplável de uma vez, e quando ele sai da sessão (ou a sessão termina) isto será liberado, porém você pode liberá-lo manualmente passando -1 para a discussão da sessão.

Sintaxe:

xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable);

ArgumentoTipo Descrição
local_user The local user ID pointer.
session_that_is_joinable The session ID to make joinable, or -1.

Devoluções:

Exemplo:

if (global.session_ID != -1)
{
    xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID);
}

O código acima verifica o valor de identificação da sessão válida e, se for detectado, define a sessão para ser juntada.
