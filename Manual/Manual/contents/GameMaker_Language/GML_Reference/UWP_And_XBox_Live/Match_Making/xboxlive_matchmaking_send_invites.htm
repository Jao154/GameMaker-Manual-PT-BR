xboxlive_matchmaking_send_invites

Esta função traz à tona o diálogo do sistema Xbox Live para permitir que um usuário envie convites para seu jogo. Você especifica o usuário local que deve controlar o diálogo (e a lista de amigos de quem será usada pelo diálogo), o valor único de identificação da sessão para a qual eles querem convidar as pessoas, e algum texto padrão para mostrar no diálogo. Você obtém o ID da sessão a partir do async_load Mapa DS gerado pela função xboxlive_matchmaking_create().

A função retornará -1 se os argumentos forem inválidos ou se o usuário não estiver conectado ao Xbox Live, ou um valor maior que 0 de outra forma.

Sintaxe:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

ArgumentoTipo Descrição
from_user The local user ID for controlling the dialogue.
session_to_invite_into The unique session ID for the session to join.
text_to_add_to_invite A short text to add to the invitation.

Devoluções:

Exemplo:

if gamepad_button_check_pressed(0, gp_face1)
{
    var _uid = xboxlive_user_for_pad(0);
    xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Come and join my game!");
}

O código acima verifica se há um botão de controle de jogo e, se for detectado, será aberto o diálogo Xbox Live para convites de sessão.
