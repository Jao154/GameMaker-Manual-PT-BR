uwp_is_constrained

Esta função retornará true se o jogo estiver rodando em um modo com recursos limitados ou false se estiver rodando normalmente. O modo de restrição de recursos pode ser ativado quando o jogo não tem foco, e neste modo o jogo reduziu os recursos da GPU. A resposta a isto normalmente deve ser pausar o jogo (já que o usuário não tem mais controle). Além disso, desativar qualquer efeito particularmente tributário pode ser uma boa idéia se a desaceleração for observada neste modo. Note que, ao contrário da uwp_is_suspending() esta função retornará true continuamente enquanto o jogo estiver em modo restrito (e não apenas uma vez quando isso acontecer pela primeira vez).

Sintaxe:

uwp_is_constrained();

Devoluções:

Exemplo:

if xboxliveis_constrained()
{
    if !instance_exists(obj_PauseMenu)
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

O código acima verifica se o aplicativo está restrito e se ele cria uma instância de um object (usado para pausar o jogo).
