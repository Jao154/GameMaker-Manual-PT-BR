uwp_show_help

Esta função lança o aplicativo de ajuda Xbox One que exibirá o arquivo de ajuda de seu jogo (que é um arquivo em formato HTML que você já carregou anteriormente no XBox Developer Portal). Note que um ID de usuário é especificado ao chamar esta função para que as informações de localização do usuário possam ser usadas ao exibi-la. Você pode recuperar o ID do usuário com as funções xboxlive_get_user(), xboxlive_get_activating_user()ou uwp_license_trial_user().

Sintaxe:

uwp_show_help(user_id);

ArgumentoTipo Descrição
user_id The User ID pointer to open the helpfile for.

Devoluções:

Exemplo:

if gamepad_button_check_pressed(0, gp_start)
{
    if uwp_license_trial_version()
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

O código acima verifica se um botão é pressionado e depois verifica o aplicativo para ver se ele está funcionando com uma licença de teste. Se estiver, então ele obtém o ID de usuário para licença e abre o arquivo de ajuda para esse usuário.
