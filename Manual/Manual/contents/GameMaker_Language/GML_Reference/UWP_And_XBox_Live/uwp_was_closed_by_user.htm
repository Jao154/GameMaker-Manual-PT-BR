uwp_was_closed_by_user

Com esta função você pode descobrir se o jogo foi anteriormente fechado pelo usuário enquanto jogava (ou seja, foi finalizado corretamente e não pelo sistema). Se tiver sido, então a função retornará true, caso contrário retornará false.

Sintaxe:

uwp_was_closed_by_user();

Devoluções:

Exemplo:

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

O código acima verifica se o aplicativo foi encerrado anteriormente pelo usuário e, em caso afirmativo, define uma variável global.
