xboxlive_get_user_count

Com esta função, você pode encontrar o número total de usuários atualmente conectados ao sistema XBox. O valor de retorno será um valor inteiro, de 0 - N, onde N é o número de usuários assinados -1.

Sintaxe:

xboxlive_get_user_count();

Devoluções:

Exemplo:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

O acima loops através de todos os usuários assinados e armazena seu ponteiro de identificação único em uma matriz.
