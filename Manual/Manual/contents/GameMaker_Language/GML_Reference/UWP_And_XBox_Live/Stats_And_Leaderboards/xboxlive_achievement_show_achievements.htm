xboxlive_achievement_show_achievements

Esta função abrirá a página de realizações para a plataforma escolhida. Observe que esta é uma função assíncrona, ou seja: seu jogo continuará a ser executado em segundo plano enquanto a página de realizações estiver sendo mostrada. Como tal, você deve ter cuidado onde usar isto e certificar-se de pausar o jogo ou apenas permitir que ele seja mostrado em áreas de seu jogo onde ele não interferirá com o jogo.

Sintaxe:

xboxlive_achievement_show_achievements()

Devoluções:

Exemplo:

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

O código acima verifica uma variável personalizada para ver se as realizações estão habilitadas e, em caso positivo, define uma variável global e cria uma instância de "pausa" do controlador antes de chamar a função para mostrar as realizações na tela.
