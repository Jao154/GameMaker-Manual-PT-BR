xboxlive_read_player_leader_leaderboard

A função retornará informações do leaderboard para o nome do usuário dado. O quadro de líderes deve ter sido criado anteriormente no painel do XDP para seu jogo, e a chamada à função desencadeará um Evento Assíncrono Social de retorno que contém o async_load mapa povoado com os pares chave/valor relevantes. A chave "id" deste mapa DS é usada para identificar o retorno de chamada correto (pode haver mais de uma função de acionamento para qualquer evento assíncrono), e a TI será emparelhada com a constante achievement_leaderboard_info, bem como um número de outros pares de chave/valor. O conteúdo exato do mapa é mostrado abaixo:

  • "id" - Para esta função deve ser achievement_leaderboard_info
  • "leaderboardid" - A identificação única para o painel de líderes, conforme definido no painel do fornecedor.
  • "numentries" - O número de entradas no quadro de líderes que você recebeu.
  • "PlayerN" - O nome do jogador, onde "N" é um valor inteiro correspondente à sua posição dentro da lista de entradas do quadro de líderes.
  • "PlayeridN" - A identificação única do usuário do jogador, "N".
  • "RankN" - A posição do jogador "N" dentro da tabela de liderança.
  • "ScoreN" - A pontuação do jogador "N".


A função exige que você dê uma das seguintes constantes para definir as propriedades do filtro:

Constante Descrição
xboxlive_achievement_filter_all_players Obtenha todas as pontuações.
xboxlive_achievement_filter_friends_only Obtenha apenas pontuações de amigos, em ordem ascendente.
xboxlive_achievement_filter_favorites_only Obtenha apenas as pontuações favoritas, em ordem ascendente
xboxlive_achievement_filter_friends_alt Obtenha apenas pontuações de amigos em ordem decrescente.
xboxlive_achievement_filter_favorites_alt Obtenha apenas as pontuações favoritas em ordem decrescente.

Sintaxe:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Argumento Tipo Descrição
leaderboard_name Cordão The name of the leaderboard to read, as set up on XDP.
user_name ID do usuário do Xbox The ID of the user to read from.
num_items Real The number of items to retrieve.
friend_filter Constante do filtro Xbox Live Achievement One of the filter constants (see the description, above).

Devoluções:

Real

Exemplo:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

O código acima obtém a identificação do usuário e depois a utiliza para solicitar todos os dados do jogador para os primeiros 10 lugares do quadro de líderes dado.
