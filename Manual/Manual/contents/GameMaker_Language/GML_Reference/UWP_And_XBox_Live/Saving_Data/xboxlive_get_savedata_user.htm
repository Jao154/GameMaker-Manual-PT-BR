xboxlive_get_savedata_user

Esta função retorna o ponteiro de identificação do usuário atualmente associado à gravação de arquivos (ou a constante pointer_null se nenhum usuário estiver sendo usado no momento). Ver xboxlive_set_savedata_user() para mais detalhes.

Sintaxe:

xboxlive_get_savedata_user();

Devoluções:

Xbox User ID or pointer_null

Exemplo:

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

O código acima verifica se um usuário está atualmente designado como o alvo a ser salvo, e se não estiver, então ele é designado.
