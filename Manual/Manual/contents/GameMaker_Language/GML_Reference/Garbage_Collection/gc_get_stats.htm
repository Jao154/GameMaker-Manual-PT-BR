gc_get_stats

Com esta função você pode recuperar informações sobre o estado atual do coletor de lixo. A função retornará uma estrutura que terá as seguintes variáveis de membros (note que "objetos" aqui se refere a qualquer coisa que possa ser coletada de lixo e não instâncias gerais object como definido no Asset Browser):

Estrutura das estatísticas da GC
Variável Tipo Descrição
objects_touched Real Este é o número de ativo objects o coletor de lixo encontrado no quadro anterior. Isto varia de acordo com a geração que foi coletada.
objects_collected Real O número de objects que o coletor de lixo determinou não estar ativo no quadro anterior, e que, portanto, poderia ser apagado.
traversal_time Real Este é o tempo em microssegundos (na linha principal) que o coletor de lixo levou para descobrir quais objects estavam ativos.
collection_time Real Este é o tempo em microssegundos (em uma linha separada) que o catador de lixo levou para limpar o objects considerado inativo.
gc_frame Real Este é um contador que é incrementado cada vez que ocorre uma passagem de coleta de lixo. Se a coleta de lixo for desativada, isto não irá aumentar.
generation_collected Real Este é o índice da geração que foi coletada por último. 0 é a geração mais jovem e 3 é atualmente a mais velha.
num_generations Real Este é o número total de gerações de coleta de lixo.
num_objects_in_generation Array de Reals Esta é uma matriz (do tamanho num_generations) contendo o número de objects em cada geração.

NOTA: No HTML5 a coleta de lixo da plataforma alvo é tratada pelo mecanismo JavaScript e, portanto, as variáveis de membro no struct acima retornarão todas 0 se esta função for usada naquela plataforma.

Ao utilizar esta função, observe que as informações mostradas para o objects só são atualizadas quando o processamento de uma geração completa é concluído, o que pode levar vários quadros, dependendo da definição do tempo de quadro (veja aqui para mais informações sobre o tempo de quadro).

Sintaxe:

gc_get_stats();

Devoluções:

GC Stats Struct

Exemplo:

if (global.debug == true)
{
    var _s = gc_get_stats();
    var _t = _s.traversal_time;
    var _c = _s.collection_time;
    show_debug_message("Traversal time = " + string(_t))
    show_debug_message("Collection time = " + string(_c))
}

O código acima verifica uma variável global e, se for true, obtém informações do garabge collector e as envia para o console como mensagens de depuração.
