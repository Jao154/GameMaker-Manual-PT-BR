Ref_ref_create fraco

Com esta função você pode criar uma referência fraca a um struct que pode então ser usado para verificar se o struct ainda está "vivo" (referenciado) ou não no jogo. Você fornece a referência ao struct que deseja rastrear, e a função retornará outro struct que é uma referência fraca a esse struct. Note que você pode verificar se uma referência é "forte" ou "fraca" usando a função instanceof(), pois uma referência forte retornará ou "struct" ou o nome da função do construtor que criou o struct, ou "weakref" se for uma referência fraca. Observe também que a referência fraca struct terá uma variável "ref" que pode ser acessada para obter a referência forte para o struct em questão, a menos que tenha sido coletado lixo, neste caso retornará undefined.

Sintaxe:

weak_ref_create(struct_to_track);

Argumento Tipo Descrição
struct_to_track Estrutura The struct that you want to create a weak reference for

Devoluções:

Struct Weak Reference

Exemplo:

inventory_ref = weak_ref_create(inventory);

O código acima cria uma referência fraca a um struct e o armazena em uma variável de instância para uso posterior.
