Estados Fontes do Tempo

Fonte de tempo Estado Constante
Constante Descrição Valor
time_source_state_initial A Fonte de Tempo ainda não foi iniciada 0
time_source_state_active A Fonte de Tempo foi iniciada e está em contagem regressiva 1
time_source_state_paused A Fonte de Tempo é paused 2
time_source_state_stopped A Fonte de Tempo foi interrompida ou expirou completamente 3

Uma Fonte de Tempo pode ter um estado, que pode ser qualquer uma das constantes acima. O estado padrão para uma Fonte de Tempo recém-criada é time_source_state_initial.

O estado pode ser mudado usando time_source_start(), time_source_stop()e time_source_pause().

O estado de uma Fonte de Tempo é devolvido usando time_source_get_state().
