tempo_fonte_reconfigurar

Esta função é utilizada para modificar as propriedades centrais de uma Fonte de Tempo, sem ter que criar uma fonte de tempo inteiramente nova.

Você especifica uma Fonte de Tempo existente, e então define as propriedades que também são definidas em time_source_create()exceto o pai. Leia essa página para obter informações detalhadas sobre essas propriedades.

A fonte de tempo especificada será redefinida e desativada, e precisará ser iniciada novamente.

Sintaxe:

time_source_reconfigure(id, period, units, callback, [args, repetitions, expiry_type]);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para reconfigurar
period Real The period that the Time Source funciona para, nas unidades dadas
units Unidade Fonte de Tempo Constante The units that the given period is in
callback Método The method to call when the Time Source expira
args Array OPTIONAL Uma matriz contendo os argumentos para passar para o método
repetitions Real OPTIONAL O número de vezes que a Fonte de Tempo deve se repetir, ou -1 para repetição indefinida
expiry_type Fonte de tempo de expiração constante OPTIONAL Se a Fonte de Tempo expira no quadro mais próximo de sua expiração, ou no próximo quadro

Devoluções:

N/A

Exemplo:

function change_spawn_delay(new_delay)
{
    time_source_reconfigure(obj_game.spawn_time_source, new_delay, time_source_units_frames, obj_game.spawn_method, [], -1);
    time_source_start(obj_game.spawn_time_source);
}

Isto cria uma nova função que muda o atraso da desova usada para os inimigos no jogo.

Assumindo que o jogo usa uma Fonte de Tempo chamada obj_game.spawn_time_source para desovar inimigos, essa Fonte de Tempo precisará ser atualizada assim que o atraso da desova mudar.

Esta função faz exatamente isso, reconfigurando a Fonte de Tempo com o novo atraso e, em seguida, reiniciando-a.
