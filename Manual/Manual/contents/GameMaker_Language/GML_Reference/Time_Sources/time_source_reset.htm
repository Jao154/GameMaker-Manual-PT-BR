time_source_reset

Esta função redefine a fonte de tempo dada, redefinindo seu temporizador e informações de repetição.

Depois de chamar esta função, você precisará iniciar novamente a Fonte de Tempo para que ela conte para baixo.

Sintaxe:

time_source_reset(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para reiniciar

Devoluções:

N/A

Exemplo:

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

Este código irá redefinir a Fonte de Tempo dada sempre que uma nova room começar, e então iniciar a Fonte de Tempo novamente.
