time_source_get_parent

Esta função retorna o Identificação da fonte de tempo do pai da Fonte de Tempo dada.

O valor retornado pode ser o ID de uma Fonte de Tempo personalizada, ou uma das Fontes de Tempo Incorporadas, dependendo de como a Fonte de Tempo em questão foi criada.

Sintaxe:

time_source_get_parent(id);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para obter o pai de

Devoluções:

Time Source ID

Exemplo:

var _parent = time_source_get_parent(time_source);

Este código recebe o pai de uma fonte de tempo e o armazena em uma variável local.
