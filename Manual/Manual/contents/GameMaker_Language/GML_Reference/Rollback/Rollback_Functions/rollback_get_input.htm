rollback_get_input

Esta função retorna um struct contendo os valores de entrada para um jogador. É necessário um argumento opcional especificando o ID do jogador para o qual serão retornados os valores de entrada.

Quando usado em uma instância de jogador que foi criada por rollback_define_player()Não é necessário especificar uma identificação de jogador, para que você possa chamar esta função sem nenhum argumento.

Quando usado em quaisquer outras instâncias, ou instâncias de jogador que foram criadas manualmente, você deve especificar uma identificação de jogador.

Para informações sobre quais entradas são devolvidas e como podem ser alteradas, consulte Definindo entradas.

Sintaxe:

rollback_get_input(player_id);

Argumento Tipo Descrição
player_id Real OPTIONAL A identificação do jogador para o qual as entradas serão devolvidas.

Devoluções:

Struct

Exemplo:

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

O código acima recebe a entrada struct em uma instância de jogador que foi criada automaticamente pelo sistema Rollback. Com base nos valores retornados, ele move a instância em 2 pixels em cada direção.

Para um exemplo ampliado, veja Criando um Jogo Multiplayer.
