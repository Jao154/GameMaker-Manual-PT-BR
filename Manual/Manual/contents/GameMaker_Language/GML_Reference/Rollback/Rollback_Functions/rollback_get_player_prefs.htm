rollback_geter_player_prefs

Esta função retorna as preferências definidas para o ID do jogador em questão.

Se um ID de jogador não for especificado, a função deve ser chamada dentro de uma instância de jogador, caso em que as preferências para esse jogador serão devolvidas.

Você também pode usar a variável de instância player_prefs.

Esta função retorna as últimas preferências que foram sincronizadas para o jogador em questão, portanto a definição das preferências não atualizará imediatamente as preferências que esta função retorna.

Por favor, leia: Preferências do jogador

Sintaxe:

rollback_get_player_prefs([player_id]);

Argumento Tipo Descrição
player_id Real OPTIONAL A identificação do jogador para o qual as preferências são devolvidas.

Devoluções:

Any

Exemplo:

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

O código acima recebe as preferências definidas para o jogador atual, e aplica uma preferência a uma variável de instância.
