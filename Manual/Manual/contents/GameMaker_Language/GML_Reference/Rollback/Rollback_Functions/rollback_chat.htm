rollback_chat

Esta função é utilizada para enviar uma mensagem string para outros jogadores.

Você especifica um string para enviar, com um tamanho máximo de 128 bytes.

Quando o segundo argumento não é especificado, a mensagem é enviada a todos os jogadores, incluindo o remetente.

Você pode especificar uma identificação do jogador no segundo argumento, que é o jogador que receberá a mensagem.

Como usar?

Esta função destina-se a ser usada "localmente", e assim resultaria de entradas locais, e não de entradas Rollback.

Veja o exemplo de uso na parte inferior. Ouça as mensagens recebidas no evento Rollback Event.

Sintaxe:

rollback_chat(message, [to]);

Argumento Tipo Descrição
message Cordão The message to send, as a string.
to Real OPTIONAL A identificação do jogador para o qual a mensagem será enviada. Não especificando este argumento, ou especificando -1, envia a mensagem a todos os jogadores.

Devoluções:

N/A

Exemplo:

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

Se a tecla enter for pressionada localmente, a keyboard_string valor é enviado a todos os jogadores. O endereço string é então liberado para que uma nova mensagem possa ser digitada.

Isto acionará o evento Rollback Event quando a mensagem for recebida por todos os clientes (incluindo o remetente).

Evento de Rollback

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
