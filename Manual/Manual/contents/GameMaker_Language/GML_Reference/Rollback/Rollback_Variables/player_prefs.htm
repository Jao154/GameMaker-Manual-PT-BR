play_prefs

Esta variável armazena as preferências definidas para o jogador atual. Ela é atribuída às instâncias do jogador criadas usando rollback_define_player().

Por favor, leia: Preferências do jogador

Sintaxe:

player_prefs;

Devoluções:

Any

Exemplo:

image_index = player_prefs.character;

O código acima lê uma preferência e estabelece uma variável de instância no jogador.
