player_name

Esta variável armazena o nome da instância do jogador atual.

Esta variável só existe em instâncias de jogadores criados por rollback_define_player()e é diferente para cada jogador.

Os jogadores convidados devolverão um nome de espaço reservado.

Sintaxe:

player_name;

Devoluções:

String

Exemplo:

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

Isto leva o nome do jogador a 100px acima de sua instância, com um alinhamento centro-médio.
