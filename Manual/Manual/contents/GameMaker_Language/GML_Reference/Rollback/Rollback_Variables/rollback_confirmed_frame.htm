rollback_confirmed_frame

Esta variável global armazena o último número de quadro onde as entradas para todos os jogadores foram sincronizadas e confirmadas.

Você pode comparar isto com rollback_current_frame para verificar se o quadro atual tem entradas confirmadas para todos os jogadores.

Você também pode usar a função rollback_sync_on_frame() para verificar se o quadro atual está sincronizado.

Sintaxe:

rollback_confirmed_frame

Devoluções:

Real

Exemplo:

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

Este código verifica se o frame atual tem entradas confirmadas de todos os jogadores, verificando se o valor do frame atual é igual ao valor do frame confirmado.

Se isso for verdade, ele executa uma função para verificar se o jogo sobre a condição foi satisfeito, para que possa terminar o jogo.

Isto é feito para evitar um estado baseado em previsões que termina o jogo, e espera por um quadro totalmente sincronizado antes de tentar verificar as condições do jogo.
