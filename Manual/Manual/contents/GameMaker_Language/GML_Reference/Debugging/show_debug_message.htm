show_debug_message

Esta função criará uma mensagem de depuração personalizada que é mostrada na janela do compilador em runtime. Note que a mensagem pode ser ou um string ou um número real, mas se você precisar de ambos, então o número terá que ser convertido para string primeiro usando o string() (veja o exemplo abaixo) e que, se o número tiver mais de duas casas decimais, então você deve usar string_format() para mostrá-los como por padrão as casas decimais são arredondadas para as duas casas decimais mais próximas (assim "1,2468" seria mostrado como "1,25" na janela de saída).

As mensagens de depuração mostradas com esta função serão mostradas na Janela de Saída do Compilador na parte inferior do IDE, bem como na Vista Gráfica do debugger ao executar o jogo no Modo de Depuração. Se você só quer ver mensagens no Modo de Depuração, então você provavelmente deve estar usando debug_event() ao invés disso.

Sintaxe:

show_debug_message(string);

ArgumentoTipo Descrição
string The custom debug message to show.

Devoluções:

Exemplo:

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

O código acima verifica se uma instância existe e se não existe, uma mensagem de debug é mostrada na janela de compilação e o jogo é encerrado.
