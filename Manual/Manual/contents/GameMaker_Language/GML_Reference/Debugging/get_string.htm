get_string

Isto cria uma janela pop-up mostrando uma mensagem padrão, com um botão chamado "Ok", que pede ao usuário para inserir um string. A função retornará a entrada string, ou o valor padrão se nada tiver sido inserido.

NOTA Esta função é para uso de depuração apenas no alvo Windows, mas é depreciada em todos os outros alvos.

Sintaxe:

get_string(str, def);

Argumento Tipo Descrição
str Cordão The string to show in the pop-up message.
def Cordão The default string in the text box.

Devoluções:

String

Exemplo:

global.test_name = get_string("Test highscore name:", "Anonymous");

O código acima solicitará que o usuário dê um nome que será então armazenado na variável global "test_name". Se nada for inserido e o usuário apenas pressionar "Ok", então o valor padrão, "Anônimo", será retornado.
