code_is_compiled

Esta função retornará true se a plataforma se compilar fora da máquina virtual, como para as plataformas YYC e JS.

Sintaxe:

code_is_compiled();

Devoluções:

Exemplo:

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

O código acima verifica se o jogo foi compilado usando o compilador YoYo e mostra uma mensagem de debug se foi.
