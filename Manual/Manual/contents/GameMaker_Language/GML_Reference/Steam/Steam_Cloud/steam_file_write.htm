steam_file_write

Você pode usar esta função para escrever dados em um arquivo, que será então sincronizado com a nuvem Steam quando o usuário sair do jogo. se o arquivo não existir, esta função irá criá-lo para você, e se já existir, ele irá sobrescrever qualquer dado que já esteja armazenado no arquivo com os novos dados string. A função retornará um valor 0 se falhar por qualquer razão e um valor maior que 0 se for bem sucedida.

Sintaxe:

steam_file_write(filename, data, size);

Argumento Descrição
filename O nome do arquivo para o qual escrever.
data Os dados para escrever (a string).
size o tamanho dos dados a serem escritos.

Devoluções:

Real

Exemplo:

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

O código acima preparará uma série de variáveis locais e depois as utilizará para escrever (ou criar) um arquivo que será então sincronizado com a nuvem Steam.
