steam_get_achievement

Com esta função, você pode verificar a API Steam para ver se uma conquista específica foi concedida. A conquista deve ter sido previamente definida na página de contas do painel de controle Steamworks para seu jogo e o string que é passado para a função deve corresponder ao que é usado como o nome da API no painel de controle.

Sintaxe:

steam_get_achievement(ach_name);

Argumento Descrição
ach_name O nome da realização a obter (string).

Devoluções:

Boolean

Exemplo:

if hp <= 0
{
    global.Deaths += 1;
    if global.Deaths == 10
    {
        if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times");
    }
}

O código acima recompensará o jogador uma realização se a variável global "Mortes" for igual a 10 e se a realização ainda não tiver sido concedida.
