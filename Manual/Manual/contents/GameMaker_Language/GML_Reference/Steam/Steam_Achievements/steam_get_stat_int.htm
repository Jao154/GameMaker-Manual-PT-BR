steam_get_stat_int

Com esta função, você pode obter o valor de uma estatística inteira específica assinada. A estatística deve ter sido previamente definida na página de contas do painel de controle Steamworks para seu jogo e o string que é passado para a função deve corresponder ao que é usado como o Nome API no painel de controle.

Sintaxe:

steam_get_stat_int(stat_name);

Argumento Descrição
stat_name O nome da estatística a obter (a string).

Devoluções:

Real

Exemplo:

xp += 100;
steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100);
if steam_get_stat_int("Total_XP") > 1000
{
    if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP");
}

O código acima estabelece uma estatística e depois verifica o valor final para que ele decida se deve ou não conceder uma conquista.
