vapor_activar_sobre_loja

Com esta função, você pode abrir a página Steam overlay na página da loja para um jogo para que os usuários possam comprar ou baixar DLC (por exemplo). Você precisa fornecer a ID do aplicativo exclusivo para o jogo ou DLC que você obteria no painel Steam ao configurá-lo.

Sintaxe:

steam_activate_overlay_store(app_id);

Argumento Descrição
app_id O único App ID para o seu jogo.

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(vk_f1)
{
    steam_activate_overlay_store(global.DLC_id);
}

O código acima pesquisa o teclado para a tecla F1 e se for então Steam overlay será aberto na página para o conteúdo do jogo usando o app ID armazenado na variável global.
