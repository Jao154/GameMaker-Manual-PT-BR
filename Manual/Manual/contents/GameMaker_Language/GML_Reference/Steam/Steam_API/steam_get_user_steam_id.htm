steam_get_user_steam_id

Você pode usar esta função para retornar a identificação de usuário única Steam do usuário atualmente conectado ao cliente Steam. Este é um valor único de 64 bits que pode ser usado para identificar usuários individuais e não seu nome de usuário na tela (isto pode ser encontrado usando a função steam_get_persona_name()). Como esta é uma int64 você pode precisar lançá-la como tal quando usá-la em funções subseqüentes usando o int64() função.

Sintaxe:

steam_get_user_steam_id();

Devoluções:

Integer (int64)

Exemplo:

if steam_initialised()
{
    global.u_id = steam_get_user_steam_id();
}

O código acima definirá uma variável global para os usuários atuais Steam ID única se a API do cliente Steam for inicializada corretamente.
