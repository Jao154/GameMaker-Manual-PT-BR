game_save_id

Esta variável somente leitura retornará o ID completo path do diretório que é usado por seu jogo para salvar arquivos. Este diretório pode ou não ser visível para outras aplicações, dependendo da plataforma.

No alvo HTML5, ele retornará um string vazio.

NOTA Esta variável existe apenas para informação e não deve ser usada para obter um arquivo path ou algo semelhante, pois não funcionará corretamente em todas as plataformas.

Sintaxe:

game_save_id

Devoluções:

String

Exemplo:

save_dir = game_save_id;

Isto irá armazenar o diretório para salvar arquivos em uma variável para uso futuro.
