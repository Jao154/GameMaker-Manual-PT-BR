game_display_name

Esta variável lida apenas retorna o nome de exibição de seu jogo para a plataforma alvo, conforme definido nas Opções de Jogo.

Sintaxe:

game_display_name

Devoluções:

Exemplo:

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

O código acima recebe o nome de exibição e o número da versão do jogo e os desenha.
