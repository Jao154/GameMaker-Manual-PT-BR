highscore_add

Com esta função, você pode adicionar um nome e uma pontuação à lista interna de pontuação máxima global. Não há necessidade de verificar o valor para ver se ele é alto o suficiente para entrar na lista de pontuação, pois o GameMaker armazenará apenas aqueles valores que são maiores do que a décima posição armazenada.

Sintaxe:

highscore_add(str, numb);

Argumento Tipo Descrição
str Cordão The string (name) to attribute the score to.
numb Real The number (score) to add.

Devoluções:

N/A

Exemplo:

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

O código acima verificará as vidas e se elas estiverem em 0, adicionará a pontuação atual e o string mantido na variável global "Name" na lista de pontuação alta, antes de finalmente definir a pontuação para 0 e enviar o jogador para o room indexado em "rm_MainMenu".
