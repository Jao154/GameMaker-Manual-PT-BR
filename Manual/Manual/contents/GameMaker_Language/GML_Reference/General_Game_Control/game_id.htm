jogo_id DEPRECADO

Esta variável lida apenas retorna o identificador único para o jogo que você criou. Você pode usar isto se precisar de um nome de arquivo único, ou qualquer outra coisa que precise de algo para identificar apenas seu jogo. Isto pode ser definido nas Opções de Jogo.

Sintaxe:

game_id

Devoluções:

Real

Exemplo:

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

O código acima executa uma criptografia muito básica na pontuação, adicionando o game_id a ele antes de salvar. Ao lê-lo de volta no jogo, você deduziria o game_id para obter o valor correto novamente.
