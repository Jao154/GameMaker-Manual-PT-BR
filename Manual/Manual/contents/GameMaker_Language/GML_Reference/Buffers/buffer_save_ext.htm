buffer_save_ext_ext

Com esta função você pode salvar parte do conteúdo de um buffer em um arquivo, pronto para ser lido de volta à memória usando o buffer_load() função. O "offset" define a posição inicial dentro do buffer para economia (em bytes), e o "tamanho" é o tamanho da área buffer a ser salva a partir dessa compensação (também em bytes).

Sintaxe:

buffer_save_ext(buffer, filename, offset, size);

ArgumentoTipo Descrição
buffer The index of the buffer to save.
filename The name of the file to save as.
offset The offset within the buffer to save from (in bytes).
size The size of the buffer area to save (in bytes).

Devoluções:

Exemplo:

buffer_save_ext(buff, "Player_Save.sav", 0, 16384);

Salva parte do conteúdo atual do buffer com a identificação armazenada na variável "buff" em um arquivo.
