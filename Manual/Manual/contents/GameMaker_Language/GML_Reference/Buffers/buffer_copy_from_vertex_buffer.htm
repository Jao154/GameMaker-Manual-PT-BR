buffer_copy_from_vertex_buffer

Esta função pode ser usada para copiar alguns (ou todos) dos dados de vertex armazenados em um vertex buffer em um buffer regular previamente criado. Ao copiar de um vertex buffer para um buffer regular com esta função, ambos buffers devem ter sido criados anteriormente (usando o vertex_create_buffer() e buffer_create() funções, por exemplo). Você pode especificar a faixa de dados vertex que deseja copiar no buffer, onde o início vertex pode estar entre 0 e o número de vértices -1, e você pode dar o número de vértices a partir desse ponto para copiar. Você pode usar a função vertex_get_number() no site vertex buffer para obter o número total de vértices armazenados. Por fim, você fornece o índice buffer para copiar os dados vertex, assim como uma compensação de dados para definir a posição para copiar os dados vertex no destino buffer.

Sintaxe:

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

Argumento Tipo Descrição
vertex_buffer Vertex Buffer ID The index of the vertex buffer to copy from.
start_vertex Real The starting vertex.
num_vertices Real The total number of vertices to use.
dest_buffer Identificação do tampão The index of the buffer to copy to.
dest_offset Real The offset position to copy the data to (in bytes).

Devoluções:

N/A

Exemplo:

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

O código acima copiará os dados de vertex armazenados no vertex buffer indexado na variável "model_buffer", e depois os colará no buffer indexado na variável "player_buffer".
