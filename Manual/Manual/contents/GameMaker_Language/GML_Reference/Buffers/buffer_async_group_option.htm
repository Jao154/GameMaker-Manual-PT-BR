opção_de_tampão_async_group_option

Com esta função você pode definir algumas opções específicas de plataforma para o grupo buffer que está sendo salvo/carregado. As opções disponíveis são as seguintes:

  • "subtítulo" ou "slottitle" - O valor para esta opção seria um string, que será mostrado ao usuário ao gerenciar seus dados salvos no sistema operacional. Isto só é importante quando se salvam dados, não quando se carregam novamente.
  • "showdialog" - O valor para esta opção é um booleano true ou false. Se ajustado para true, quando você salvar ou carregar o GameMaker mostrará a interface do sistema, caso contrário ele fará um quicksave/quickload sem a interface mostrada. Você normalmente só precisa disto se você estiver suportando vários slots salvos para permitir que o usuário escolha um slot, mas se você suportar apenas um slot por usuário, defina isto para false.
  • "savepadindex" - Para esta opção você especificaria o índice do pad do jogador que está salvando ou carregando e o sistema irá escrever e ler os dados da pasta de salvamento deste jogador.
  • "Esta opção requer que você especifique o tamanho real em bytes que deseja salvar (para que você possa fazer um buffer_seek() e buffer_tell() para conseguir isso, por exemplo). Note que não é obrigatório fornecer este valor, pois todas as economias são pré-atribuídas com um espaço mínimo, que geralmente varia de acordo com a plataforma alvo.
  • "ps_create_backup" - (somente PlayStation) - Definindo isto em true assegura que os dados salvos sejam copiados quando o diretório de salvamento não for montado (ou seja, quando a função buffer_async_group_end() for chamada). Consulte a documentação do console para obter mais informações.

IMPORTANTE! Esta função é válida atualmente apenas para os módulos alvo PSVita, PS4 e XBox One. Em todos os outros alvos, ela não fará nada.

Sintaxe:

buffer_async_group_option(option, value);

Argumento Tipo Descrição
option Cordão The option to set.
value Real ou Cordão The value to set (can be a real or string, depending on the option).

Devoluções:

N/A

Exemplo:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

O código acima inicia um grupo buffer e então define as opções do grupo antes de definir 4 arquivos para salvar de forma assíncrona. A definição do grupo é então terminada (momento em que a gravação começará).
