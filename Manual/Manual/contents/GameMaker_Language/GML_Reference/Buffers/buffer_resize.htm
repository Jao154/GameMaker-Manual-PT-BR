tamanho_tampão

Com esta função você pode redimensionar um determinado buffer para ser o tamanho (em bytes) que você especificar.

Sintaxe:

buffer_resize(buffer, newsize);

ArgumentoTipo Descrição
buffer The index of the buffer to change the size of.
newsize The new size of the buffer (in bytes).

Devoluções:

Exemplo:

if (buffer_get_size(buff) < 16384)
{
    buffer_resize(buff, 16384);
}

O código acima verificará o tamanho do buffer indexado na variável "buff" e se for menor que o valor dado, o buffer é redimensionado.
