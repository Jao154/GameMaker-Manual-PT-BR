tipo_computador

Com esta função você recupera o tipo de buffer que está sendo referenciado por um valor de identificação buffer. O valor de retorno será uma das constantes a seguir:

Constante Descrição
buffer_fixed Um buffer de tamanho fixo.
buffer_grow Um buffer que irá "crescer" dinamicamente à medida que os dados forem sendo adicionados
buffer_wrap A buffer onde os dados serão "embrulhados". Quando os dados a serem adicionados atingirem o limite do tamanho buffer, o sobrescrito será colocado de volta no início do buffer, e a partir desse ponto continuará a ser escrito.
buffer_fast Especial "despojado" buffer que é extremamente rápido de ler/escrever. Só pode ser usado com os tipos de dados buffer_u8, e deve estar alinhado a 1 byte.
buffer_vbuffer Um buffer vertex

Sintaxe:

buffer_get_type(buffer);

ArgumentoTipo Descrição
buffer The index of the buffer to check.

Devoluções:

Buffer Type Constant

Exemplo:

type = buffer_get_type(buff);

O código acima obterá o tipo de buffer a partir do valor indexado na variável "buff" e o armazenará em uma variável.
