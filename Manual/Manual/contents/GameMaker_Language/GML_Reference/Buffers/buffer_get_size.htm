tamanho_do_tampão_de_servidor

Com esta função você pode obter o tamanho do dado buffer em bytes.

Sintaxe:

buffer_get_size(buffer);

Argumento Tipo Descrição
buffer Identificação do tampão The index of the buffer to get the size of.

Devoluções:

Real

Exemplo:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

O código acima criará um novo buffer e armazenará seu índice na variável local "_temp", sendo o tamanho deste novo buffer o mesmo que o do buffer indexado na variável "player_data".
