gif_save

Com esta função, você pode salvar uma animação GIF. Você fornece o índice GIF (como retornado pela função gif_open()), bem como um nome de arquivo para salvá-lo. Note que o GameMaker não anexa automaticamente o arquivo .gif extension , portanto você deve incluir isto como parte do nome do arquivo string se desejar que o arquivo salvo seja identificado como um GIF. O GIF criado será paletizado usando a Paleta Universal 884 (veja aqui para mais informações).

Observe que se a função for bem sucedida e o GIF for salvo corretamente, então ele retornará 0, caso contrário retornará -1.

Sintaxe:

gif_save(gif_index, fname);

ArgumentoTipo Descrição
gif_index The ID of gif to save
fname The filename to use for the gif

Devoluções:

Exemplo:

if save_gif == true
{
    if count == 0
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if count < 30
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

O código acima criará um arquivo de imagem GIF com 30 quadros tirados da superfície da aplicação e depois o salvará.
