exibir_get_gui_width

Com esta função você pode obter a largura (em pixels) da GUI como utilizada no Evento Draw GUI.

Sintaxe:

display_get_gui_width();

Devoluções:

Exemplo:

ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0);

O código acima irá definir um anúncio a ser exibido na parte superior direita da tela.
