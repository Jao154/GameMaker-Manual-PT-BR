camera_set_view_mat

Esta função definirá a matriz de visualização para uma determinada câmera. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e uma matriz de visualização a ser aplicada. Você pode saber mais sobre a criação de matrizes de visão na seção Funções de Matriz, especificamente matrix_build_lookat().

Observe que se sua câmera faz o rastreamento automático object - ou seja: ela foi criada usando camera_create_view() com um índice object / ID de instância que não é -1, ou você está configurando uma câmera definida no Editor Room, ou você está configurando a câmera padrão - então esta matriz será sobregravada no próximo quadro do jogo.

Sintaxe:

camera_set_view_mat(camera_id, matrix)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

Devoluções:

Exemplo:

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

O código acima cria uma nova câmera e a atribui para visualizar a porta[0]. Esta câmera então tem suas matrizes de visão e projeção definidas.
