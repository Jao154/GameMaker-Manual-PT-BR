view_set_surface_id

Com esta variável, você pode definir o conteúdo de uma porta de visão para desenhar em uma superfície. Quando se trabalha com superfícies, muitas vezes é necessário capturar toda a região visível da tela para a superfície, e assim você a atribuiria a uma porta de visualização usando esta função. Isto significa que tudo que é mostrado na porta de visualização escolhida será agora atraído para a superfície designada e o conteúdo dessa porta de visualização não será mais exibido, o que significa que você precisará de uma das duas:

  • Permitir uma nova visão e desenhar a superfície somente nessa visão (usando view_current para verificar qual visão está sendo desenhada)
  • Desenhe a superfície no evento de Sorteio GUI ou Pós-Sorteio de uma instância, uma vez que estes eventos são independentes de pontos de vista.

Ao usar esta função você dá o índice da porta de visão (de 0 a 7) e um índice de superfície (ou o application_surface ou o valor de índice único retornado pela função surface_create()) ou, se uma superfície foi previamente atribuída e você deseja removê-la, um valor de -1. Para mais exemplos sobre como definir a porta de visualização para uma superfície, veja a variável view_surface_id.

Sintaxe:

view_set_surface_id(view_port, surf)

Argumento Tipo Descrição
view_port Real The view port to target (0 - 7)
surf Identificação da superfície The surface to apply to the view

Devoluções:

Real

Exemplo:

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

O código acima verificará se uma superfície foi atribuída à porta de visualização[0] e se não foi atribuída, então uma é atribuída.
