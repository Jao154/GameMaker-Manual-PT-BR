display_reset

Esta função repõe as configurações de exibição para as que foram definidas quando o jogo foi iniciado, mas também permite alterar o nível atual de anti-aliasing em tela cheia que está sendo usado e se deve ser usada a sincronização vertical. Os níveis de anti-aliasing disponíveis são 0,2,4 e 8, com o valor inicial padrão sendo definido como 0, e a configuração padrão v-sync é falsa (off). Ligar o v-sync pode proporcionar uma experiência de jogo mais suave, mas também precisará de mais poder de processamento e, portanto, seu impacto deve ser considerado cuidadosamente antes do uso, e o mesmo vale para o anti-aliasing onde quanto maior o número, mais processamento é necessário.

Como nem todos os dispositivos alvo são iguais, alguns podem não suportar 8x ou 4x anti-aliasing, por exemplo, e por isso há uma variável de leitura apenas disponível para obter os diferentes níveis de AA que o dispositivo executando o jogo pode exibir:

display_aa

Esta variável retornará um valor inteiro baseado na definição de bits para os diferentes níveis. Assim, para apenas 2xAA, isto reportará 2, para 2x e 4x disponibilidade, reportará 6. Para 8 e 4, ela relatará 12. Para todos os 3 níveis (2,4 e 8), ele reportará 14.

Sintaxe:

display_reset(aa, vsync);

Argumento Tipo Descrição
aa Real The level of anti-aliasing filtering (0, 2, 4 or 8).
vsync Booleano Toggle vertical synchronisation to on (true) or off (false).

Devoluções:

Real

Exemplo:

if (display_aa > 12)
{
    display_reset(8, true);
}

O código acima estabelecerá o nível de anti-aliasing para 8 se suportado e switch v-sync para on.
