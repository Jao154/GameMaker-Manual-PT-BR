screen_save

Com esta função, você pode salvar uma captura de tela do jogo como ela aparece atualmente na janela do jogo.

O arquivo será salvo no diretório de trabalho do jogo, que no windows está localizado em <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ e no Mac seria ~/Library/Application Support/[Game Name]/. A imagem capturada será a renderização final da superfície de aplicação do jogo, ou (se a superfície da aplicação tiver sido desativada) será uma imagem de toda a tela ou janela.

É importante notar que a função foi projetada para ser chamada no evento final do Draw GUI, que deve dar resultados consistentes em todas as plataformas. A sua utilização em qualquer outro evento pode não funcionar como esperado, e pode dar resultados diferentes entre alvos diferentes e até mesmo entre dispositivos.

Caso você precise de mais controle sobre o que está sendo salvo, você deve usar a função screen_save_part().

NOTA Esta função não funciona com o alvo HTML5.

Sintaxe:

screen_save(fname);

Argumento Tipo Descrição
fname Cordão The filename for the screenshot to save

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

O código acima tira uma captura de tela do jogo quando o usuário pressiona a tecla "S". Esta captura de tela é salva em um diretório específico e numerada usando uma variável que é então incrementada.
