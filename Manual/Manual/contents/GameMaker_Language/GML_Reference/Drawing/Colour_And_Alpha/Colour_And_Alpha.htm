Cor e Alfa

O GameMaker tem uma seleção abrangente de funções para manipular a cor e o alfa do que é desenhado na tela. Usando estas funções você pode separar as cores em suas partes componentes, criar novas cores e manipular o alfa do desenho para mudar a forma como as coisas diferentes em seu jogo são desenhadas juntas.

Para misturar imagens usando modos de mistura especiais e outras funções específicas da GPU, veja a seção sobre o Controle de GPU.

Cores hexadecimais

Você pode criar cores a partir de valores hexadecimais usando o símbolo $ antes, e estes valores também podem ser usados com várias funções de cor para obter valores de componentes separadamente. O valor hexadecimal para uma cor é dividido em três partes, com o formato sendo: azul/verde/vermelho(ou $BBBGGRR), assim, por exemplo, para definir uma cor azul claro/aqua que você poderia usar:

col = $FFCC11;

Aqui, o FF é o componente azul, o CC é o componente verde e o 11 é o componente vermelho.

Você também pode usar "cores CSS" que começam com um sinal de hash/libra(#). Isto usa um formato vermelho/verde/azul(ou #RRGGBB) que é mais amplamente utilizado e é compatível com várias outras aplicações.

col = #11CCFF;

Esta é a mesma cor mostrada no bloco de código anterior, apenas os valores RR e BB são trocados para combinar com o formato.

NOTA A GML também suporta o formato $AABBBGGRR (com o valor alfa hexadecimal no início), porém a maioria das funções de cor GML só poderá fazer uso da parte de cor e ignorar o alfa.

Funções

As seguintes funções podem ser usadas para obter as tonalidades dos componentes, matizes compostos, saturação e luminosidade de uma cor selecionada, assim como o alfa e outras propriedades:

  • colour_get_blue
  • colour_get_green
  • colour_get_red
  • colour_get_hue
  • colour_get_saturation
  • colour_get_value
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Estas funções podem ser usadas para criar cores a partir de valores de entrada brutos para componentes de cor:

  • make_colour_hsv
  • make_colour_rgb
  • merge_colour

As seguintes funções podem ser usadas para definir as diversas opções de desenho para a tela, incluindo alfa, cor e mistura:

  • draw_clear
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
