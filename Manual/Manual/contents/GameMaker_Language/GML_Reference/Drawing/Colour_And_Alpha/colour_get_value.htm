color_get_value

Esta função devolverá o valor (luminosidade) da cor dada. Esta é a quantidade de "luz" que é misturada na cor final e faz parte do método de tonalidade, saturação e valor para definir uma cor. A seguinte imagem ilustra como este valor corresponde à escala de cor HSV:

Sintaxe:

colour_get_value(col);

ArgumentoTipo Descrição
col The colour to check

Devoluções:

Exemplo:

col = make_colour_hsv(random(255), 255, colour_get_value(c_teal));

O código acima obtém o valor usado para fazer a cor constante "c_teal" e então usa esse valor para definir uma cor aleatória para ter a mesma luminosidade, armazenando a cor resultante na variável "col".
