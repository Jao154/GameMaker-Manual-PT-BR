aplicação_surface

Esta variável global integrada pode ser usada para acessar a superfície de aplicação, para uso em qualquer uma das funções da superfície. Esta superfície está permanentemente disponível e é onde o jogo é desenhado pelo GameMaker.

Os eventos regulares de Sorteio(Início do Sorteio, Fim do Sorteio e Fim do Sorteio) são atraídos para esta superfície por padrão. Outros eventos de Sorteio (como Pré/Pós-Sorteio e Desenho GUI) não desenham na superfície de aplicação. Veja Eventos de Sorteio para mais informações sobre como esses eventos funcionam.

Sintaxe:

application_surface;

Devoluções:

Surface ID

Exemplo:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

O código acima redimensionará a superfície de aplicação para ter uma relação de 1:1 com a camada GUI.
