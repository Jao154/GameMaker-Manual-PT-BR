superfície_save_parte

Esta função salvará uma parte de uma superfície para o disco usando o nome do arquivo dado. A superfície deve ser salva como um arquivo no formato *.png, e a posição (x,y) deve ser dada como coordenadas locais à superfície, tendo em mente que o canto superior esquerdo da superfície é sempre (0,0).

Sintaxe:

surface_save_part(surface_id, fname, x, y, width, height);

ArgumentoTipo Descrição
surface_id The ID of the surface to set as the drawing target.
fname The name of the saved image file.
x The starting x position within the surface.
y The starting y position within the surface.
width The width of the part to save.
height The height of the part to save.

Devoluções:

Exemplo:

if keyboard_check_pressed(ord("S")
{
    surface_save_part(surf, "test.png", 0, 0, 100, 100);
}

O código acima verificará se o usuário pressiona a tecla "S" no teclado e se o faz salvará uma parte da superfície indexada na variável "surf" ao disco.
