draw_surface_stretched_ext

Esta função faz exatamente o mesmo que o draw_surface_stretched() com a capacidade adicional de definir a mistura de cores e o valor alfa para a superfície quando ela é desenhada (semelhante à função draw_surface_ext()).

Sintaxe:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Argumento Tipo Descrição
id Identificação da superfície The unique ID value of the surface to draw.
x Real The x position of where to draw the surface.
y Real The y position of where to draw the surface.
w Real The width at which to draw the surface.
h Real The height at which to draw the surface.
colour Cor The colour with which to colour the surface.
alpha Real The alpha with which to blend the surface.

Devoluções:

N/A

Exemplo:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Isto desenhará a superfície dada com seu canto esquerdo na posição x/y e será esticada para ocupar uma área de 200x200 pixels sem mistura, mas com transparência parcial.
