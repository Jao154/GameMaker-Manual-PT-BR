surface_get_depth_disable

Esta função verifica se a geração automática de profundidade buffer para superfícies está habilitada. Normalmente todas as superfícies têm profundidade buffers, portanto, se você desenhar 3D objects para elas, então ele as classificará corretamente pela profundidade, porém alocando a profundidade buffers essencialmente dobra o tamanho das superfícies, o que pode ser uma sobrecarga excessiva e desnecessária, especialmente se seu jogo for muito intensivo em memória ou predominantemente 2D. Nestes casos, você pode verificar isto usando esta função e desativar a profundidade buffer para superfícies, se necessário usando a função surface_depth_disable().

Sintaxe:

surface_get_depth_disable();

Devoluções:

Exemplo:

if (!surface_exists(surf))
{
    if surface_get_depth_disable() == false
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

O código acima verificará se a superfície em questão existe, e se não existir, então verificará o estado atual da profundidade da superfície buffer e, se estiver habilitada, a desativará em seu lugar, antes de finalmente criar a superfície.
