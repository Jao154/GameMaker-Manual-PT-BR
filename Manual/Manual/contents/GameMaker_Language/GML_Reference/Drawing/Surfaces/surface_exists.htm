superfície_existe

Esta função é essencial quando se trabalha com superfícies, devido à sua natureza volátil. As superfícies são sempre mantidas na memória texture, o que significa que elas podem ser destruídas de um momento para o outro (por exemplo, quando um protetor de tela começa nas janelas, ou quando minimizado em um dispositivo Android ), portanto, deve-se sempre verificar se existe uma superfície antes de fazer qualquer coisa com ela (isto inclui desenhá-la para a tela). O código de exemplo abaixo mostra um uso típico deste comando no evento de desenho de uma instância para verificar a existência de uma superfície e recriá-la caso tenha sido removida (note que a superfície terá sido originalmente criada no evento de criação do object).

Sintaxe:

surface_exists(surface_id);

ArgumentoTipo Descrição
surface_id The ID of the surface to check.

Devoluções:

Exemplo:

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
}
else
{
    if view_current = 1
    {
        draw_surface(surf,0,0);
    }
}

O código acima verificará se existe uma superfície indexada na variável "surf", e se não existir, ela será recriada. Se ela existe, então verifica para ver qual vista está sendo desenhada no momento e se é vista[1] desenha a superfície.
