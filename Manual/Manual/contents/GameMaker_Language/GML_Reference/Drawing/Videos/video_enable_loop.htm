video_enable_loop

Esta função ativa ou desativa o looping para o vídeo atualmente carregado. Defina o argumento de habilitação como verdadeiro para habilitar o looping, e falso para desabilitá-lo.

Por padrão, o looping está desativado.

Sintaxe:

video_enable_loop(enable);

Argumento Tipo Descrição
enable Booleano Whether to enable (true) ou desabilitar(falso) o looping

Devoluções:

N/A

Exemplo:

video_open("Loading.mp4");
video_enable_loop(true);

O código acima carrega um vídeo e permite o looping sobre ele.
