vídeo_fechar

Esta função fecha o arquivo de vídeo que está sendo carregado atualmente. Certifique-se de que este só seja chamado após uma chamada video_open(), caso contrário, não fará nada.

Sintaxe:

video_close();

Devoluções:

N/A

Exemplo:

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

O código acima fecha o vídeo quando a tecla Escape é pressionada.
