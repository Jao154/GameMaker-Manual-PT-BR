video_is_looping

Esta função retorna se o vídeo atualmente carregado está configurado para loop (verdadeiro) ou não(falso). Você pode definir um vídeo para loop ligando para video_enable_loop().

Sintaxe:

video_is_looping();

Devoluções:

Boolean

Exemplo:

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

O código acima verifica se o vídeo carregado atualmente está em loop e, nesse caso, desativa os controles do player, definindo uma variável de instância personalizada como falsa.
