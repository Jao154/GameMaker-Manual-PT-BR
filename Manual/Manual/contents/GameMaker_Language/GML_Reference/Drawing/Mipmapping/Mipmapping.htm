Mipmapping

Esta seção contém todas as funções relacionadas ao uso das funções de mipmapping. Antes de usar estas funções você deve ter habilitado o mipmapping para as páginas necessárias texture no Editor do Grupo de Textura e/ou habilitado o mipmapping para aquelas sprites que foram configuradas para usar apenas uma única página texture das Opções Gerais de Jogo. A imagem abaixo mostra a diferença que o mipmapping pode fazer ao renderizar seu projeto:

Uma vez ativado o mipmapping para o projeto ou uma página texture, você pode então usar as seguintes funções para obter e definir valores que mudarão a aparência das coisas em seu projeto quando executado.

As funções listadas abaixo podem ser usadas para obter os diferentes valores associados ao mipmapping no GameMaker:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

As funções listadas abaixo podem ser usadas para definir os diferentes valores associados ao mipmapping no GameMaker:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
