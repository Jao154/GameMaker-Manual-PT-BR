gpu_get_tex_mip_filter

Com esta função, você pode obter o modo atual de filtro mip. O valor do modo retornado pela função será uma das constantes listadas abaixo.

Sintaxe:

gpu_get_tex_mip_filter();

Devoluções:

Mipmapping Filter Constant (listed below):

Filtro de Mipmapping Constante
Constante Descrição
tf_point Isto significa que a mistura entre os níveis do mipmap está desativada, o que pode causar transições visíveis texture, mas dá o melhor desempenho.
tf_linear Isto significa que a mistura entre níveis de mipmap está habilitada (também conhecida como filtragem trilinear), o que suaviza as transições texture, mas dará um pequeno golpe no desempenho.
tf_anisotropic Isto significa que a filtragem anisotrópica está habilitada, o que melhora muito a qualidade de transição texture e pode reduzir o embaçamento visível com outros modos de filtragem, mas tem o maior impacto no desempenho.

Exemplo:

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

O código acima verifica o teclado e, se a tecla especificada for pressionada, ele obterá o filtro mip atual e alternará o valor para o próximo, pedalando pelos diferentes modos.
