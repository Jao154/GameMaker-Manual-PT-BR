part_system_create_layer

Esta função criará um novo sistema de partículas em uma determinada camada. Você dá o ID da camada única como retornado pela função layer_create() ou o nome da camada a ser usada como um string - por exemplo, "instance_layer" - e depois flag o sistema como sendo persistente ou não. Se o sistema não for sinalizado como persistente, será automaticamente destruído no final do room em que foi criado (isto é o mesmo que se você tivesse chamado a função part_system_destroy() e também destruirá quaisquer emissores associados com o sistema). Entretanto, quando marcado como persistente, o sistema será levado para o próximo room quando o room for alterado, e se o seguinte room não tiver uma camada com o mesmo nome ou profundidade que o atribuído, então uma nova camada será criada para o sistema que está persistindo através do rooms, e será nomeado da mesma forma que a camada original. Ao mudar rooms, se houver outra camada no seguinte rooms com o mesmo nome, então a instância persistente será atribuída à camada com o mesmo nome, independentemente da profundidade da camada. Finalmente, se um sistema persistido se mover para um room com uma camada na mesma profundidade da camada em que o sistema foi criado, não será atribuído a esta camada, mas em vez disso será criada uma nova camada na mesma profundidade (com o mesmo nome que a camada original).

A função retornará um valor de identificação único para o sistema de partículas que deve ser usado em todas as outras chamadas de função em que você precisa dar uma identificação do sistema.

IMPORTANTE! Se você flag o sistema de partículas como persistente, então ele (e quaisquer emissores atribuídos a ele) precisará ser limpo manualmente usando as funções de destruição apropriadas quando não estiver em uso, caso contrário você corre o risco de um vazamento de memória que impactará negativamente seu jogo final.

Sintaxe:

part_system_create_layer(layer_id, persistent);

Argumento Tipo Descrição
layer Identificaçãoporcorda ou camada The layer ID (or name) to assign the particle system to (can be any layer type)
persistent Booleano Flag the particle system as persistent (set to true) ou não (set to false)

Devoluções:

Particle System ID

Exemplo:

global.p_sys = part_system_create_layer("effects_layer", true);

O código acima criará um novo sistema de partículas na camada dada e flag como persistindo sobre o subseqüente rooms. A identificação para o sistema de partículas é armazenada em uma variável de escopo global para referência futura.
