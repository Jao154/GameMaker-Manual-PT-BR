part_type_sprite

Esta função pode ser usada para definir um tipo de partícula para usar um sprite personalizado do jogo assets. Você pode selecionar o sprite e então dizer ao GameMaker para usar o sprite e animá-lo ou não, e se você escolher animado, então você também pode optar por ter as subimagens alteradas em um intervalo tal que a animação começará e terminará no início e no final da vida útil de cada partícula criada (de modo que se você tiver uma vida útil definida para um mínimo de 15 passos e um máximo de 30 passos, não importa qual valor a vida útil da animação será feita para caber, com ela correndo mais rápido para as durações mais curtas e mais lento para as mais longas). Você também pode escolher uma subimagem aleatória para que se o sprite não for animado, uma imagem diferente será escolhida aleatoriamente para criar o sprite, ou se a partícula for animada, ela iniciará a animação em um ponto aleatório.

Sintaxe:

part_type_sprite(ind, sprite, animate, stretch, random);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
sprite Ativo Sprite The index of the sprite to set the particle type to.
animate Booleano Whether to follow the sprite's animation normally (true) or not (false).
stretch Booleano Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random Booleano Whether to choose a random sub-image (true) or not (false).

Devoluções:

N/A

Exemplo:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

O código acima define o tipo de partícula indexada na variável global "p1" para a forma da sprite indexada em "spr_Coins", e depois define várias outras propriedades de partícula.
