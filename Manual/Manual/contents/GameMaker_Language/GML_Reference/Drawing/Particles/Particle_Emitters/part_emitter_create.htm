part_emitter_create

Esta função é utilizada para criar um novo emissor de partículas no sistema de partículas em questão. A função retornará o valor índice para o novo emissor que deve ser armazenado em uma variável e usado em todas as outras funções que fazem referência ao emissor, e o próprio emissor deve ser destruído quando não for mais usado para evitar vazamentos de memória (isto pode ser conseguido usando as funções específicas de destruição do emissor ou destruindo todo o sistema de partículas ao qual o emissor pertence).

NOTA Os IDs dos emissores retornados por esta função são reutilizados, por exemplo, se você criar primeiro três emissores de partículas, eles terão os IDs 0, 1 e 2 atribuídos. Se depois você destruir o segundo emissor (que tem uma identificação de 1) usando part_emitter_destroy e depois criar outro emissor, então este emissor reutilizará o ID de 1. Como o ID do emissor 2 está em uso, o próximo emissor terá um ID de 3, e assim por diante.

Sintaxe:

part_emitter_create(ps);

Argumento Tipo Descrição
ps Identificação do Sistema de Partículas The particle system to create the emitter in.

Devoluções:

Particle Emitter ID

Exemplo:

p_emit1 = part_emitter_create(Sname);

Isto criará um novo emissor de partículas e armazenará seu índice na variável "p_emit".
