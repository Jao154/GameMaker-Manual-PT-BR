part_emitter_stream

Esta função permite definir um emissor para transmitir um tipo específico de partícula e normalmente é usado o evento de criação, pois é um código único que define o emissor para gerar o número de partículas especificadas em cada etapa do jogo - depois disso. As partículas são emitidas de acordo com a distribuição, forma e posição definida pela função part_emitter_region(). Você pode definir o número de partículas sendo transmitidas a zero e chamar esta função novamente para "desligar" o fluxo de partículas, e a função também aceitará números negativos para a quantidade, caso em que o emissor produzirá partículas com base na probabilidade aleatória. Por exemplo, se você tiver o número de partículas definido para -5, há uma chance de 1:5 de que uma partícula seja produzida a cada passo. Caso você precise que as partículas apareçam todas de uma só vez ao invés de cada passo, você deve estar usando a função part_emitter_burst().

Sintaxe:

part_emitter_stream(ps, ind, parttype, number);

Argumento Tipo Descrição
ps Identificação do Sistema de Partículas The particle system that the emitter is in.
ind Identificação do Emissor de Partículas The index of the emitter to burst from.
parttype Identificação do tipo de partícula The index (type) of the particles to be created.
number Real The number of particles to create per step.

Devoluções:

N/A

Exemplo:

part_emitter_stream(global. Sname, p_emit1, p1, 1);

O código acima transmitirá 1 partícula a cada etapa do jogo até que o emissor seja destruído ou até que o fluxo seja ajustado para 0.
