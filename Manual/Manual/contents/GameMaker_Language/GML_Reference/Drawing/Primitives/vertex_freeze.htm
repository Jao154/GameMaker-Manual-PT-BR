vértice_freeze

Esta função pode ser usada para "congelar" um vertex buffer. Este buffer torna-se somente leitura, ou seja, se você precisar mudá-lo, você teria que apagar o buffer inteiro e recriá-lo. Um buffer congelado pode ser submetido ao shader mais rápido do que um buffer normal e dinâmico e é recomendado para aqueles efeitos que requerem um primitivo personalizado imutável durante a duração de um nível ou do jogo.

A função retornará 0 ao congelar com sucesso o vertex buffer especificado, porém, se falhar por qualquer razão, retornará -1.

Sintaxe:

vertex_freeze(buffer);

Argumento Tipo Descrição
buffer Vertex Buffer ID The vertex buffer to freeze.

Devoluções:

Real (0 for success, -1 for failure)

Exemplo:

// Create event
vertex_freeze(vbuff);

O código acima congelará o dado vertex buffer no evento Create, para que possa ser usado como um "estático" buffer em outros eventos para um acesso mais rápido.
