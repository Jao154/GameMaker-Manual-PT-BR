vertex_create_buffer_from_buffer_ext

Como com a função vertex_create_buffer()Esta função criará um novo vertex buffer, somente agora os dados vertex que armazena são copiados do regular buffer que é especificado como a fonte. O buffer é pré-formatado com o formato vertex para construir primitivos para uso com (por exemplo) shaders, e você também pode fornecer um offset dentro da fonte buffer para copiar de e o número de vértices que o buffer final deve ter. Observe que se o número de vértices não corresponder aos que estão sendo copiados, você poderá obter dados corrompidos em vertex.

Sintaxe:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

Argumento Tipo Descrição
buffer Identificação do tampão The buffer to create the vertex buffer from.
format Constante do tipo primitivo The primitive vertex format to use.
src_offset Real The offset within the the source buffer to copy from.
vert_num Real The number of vertices the buffer should have.

Devoluções:

Vertex Buffer ID

Exemplo:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

O código acima criará um novo formato vertex e em seguida criará um novo vertex buffer a partir de um formato regular anteriormente criado buffer, aplicando a ele o formato personalizado vertex com 0 offset. A função diz ao novo vertex buffer que ele deve esperar 512 vértices.
