vértice_delete_buffer

Esta função pode ser usada para remover um vertex buffer previamente criado (ver vertex_create_buffer()) a partir da memória do sistema. Isto também removerá todos os dados vertex de quaisquer primitivos personalizados que contivesse.

Sintaxe:

vertex_delete_buffer(buffer);

Argumento Tipo Descrição
buffer Vertex Buffer ID The vertex buffer to be removed.

Devoluções:

N/A

Exemplo:

vertex_delete_buffer(v_buff);

O código acima excluirá da memória o buffer indexado na variável "v_buff".
