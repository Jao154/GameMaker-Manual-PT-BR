vértice_begin

Com esta função você inicia a definição de um primitivo personalizado. Você atribui um buffer para escrever o primitivo, e o formato vertex para usar (previamente definido usando as funções do formato vertex ). Você então definiria os pontos necessários para cada vertex do primitivo antes de chamar vertex_end() para finalizar a criação primitiva.

Sintaxe:

vertex_begin(buffer, format);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to be written to.
format Identificação em formato de vértice The vertex format to use.

Devoluções:

N/A

Exemplo:

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

O código acima irá definir um novo formato vertex, criar um novo buffer e iniciar o processo de definição de um novo primitivo.
