draw_enable_swf_aaa

Com esta função você pode habilitar ou desabilitar o anti-aliasing (AA) para o formato vetor SWF sprites. AA simplesmente suaviza as bordas das imagens vetoriais para dar-lhes uma aparência mais agradável. A quantidade de AA usada dependerá do valor definido usando a função draw_set_swf_aa_level(). Por padrão, isto é desativado.

Sintaxe:

draw_enable_swf_aa(enable);

ArgumentoTipo Descrição
enable Enable (true) ou desativar (false) AA para todos os SWF sprites.

Devoluções:

Exemplo:

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

O código acima verificará o valor AA para o formato SWF sprites, e se for 0, habilita AA e define o valor para 0,5.
