gpu_get_state

Esta função obterá o estado atual da GPU, devolvendo-a como um Mapa DS. Isto pode então ser manipulado ou mesmo salvo, e você pode retornar este mapa para a GPU usando a função gpu_set_state().

Sintaxe:

gpu_get_state();

Devoluções:

DS Map ID

Exemplo:

gpu_map = gpu_get_state();

O código acima armazena o estado atual da GPU em uma variável.
