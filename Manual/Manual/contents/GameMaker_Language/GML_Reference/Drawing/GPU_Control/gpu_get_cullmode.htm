gpu_get_cullmode

Esta função pode ser usada para recuperar o modo de abate na face traseira. O valor retornado será uma das seguintes constantes (o valor padrão é cull_noculling):

Constante do modo de abate
Constante Descrição
cull_noculling Nenhum abate será feito
cull_clockwise Todos os triângulos no sentido horário serão abatidos
cull_counterclockwise Todos os triângulos no sentido anti-horário serão abatidos

Sintaxe:

gpu_get_cullmode();

Devoluções:

Culling Mode Constant (see above for constants)

Exemplo:

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

O código acima obtém o modo de abate atual e, se não for cull_clockwise, é ajustado para essa constante.
