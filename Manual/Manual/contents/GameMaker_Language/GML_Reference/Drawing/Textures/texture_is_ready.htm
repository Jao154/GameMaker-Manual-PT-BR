texture_is_ready

Esta função pode ser usada para verificar se uma página específica texture foi desempacotada e está pronta para uso, ou se um grupo de páginas texture foi desempacotado e está pronto para uso. Você fornece o ID da página de textura única (conforme retornado pelas funções texturegroup_* ) ou o ID do grupo texture string (conforme definido no Editor do grupo Texture ), e a função retornará true se tiverem sido desempacotadas, ou false caso contrário.

Sintaxe:

texture_is_ready(tex_id);

Argumento Tipo Descrição
tex_id Ponteiro de Textura ou Cordão The texture page pointer or a texture group name (a string)

Devoluções:

Boolean

Exemplo:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

O código acima recuperará os IDs de página texture para o grupo texture "MainMenu", então verifique se eles estão desempacotados, e se eles não estão pré-fetados na VRAM.
