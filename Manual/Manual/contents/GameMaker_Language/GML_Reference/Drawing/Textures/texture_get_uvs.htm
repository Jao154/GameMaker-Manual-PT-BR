textura_get_uvs

Esta função retorna uma matriz 1D com 4 elementos representando as coordenadas UV para o ponteiro texture dado, preenchendo a matriz com os seguintes valores:

  • [0] = esquerda
  • [1] = topo
  • [2] = direito
  • [3] = fundo
  • [4] = quantidade de pixels que o compilador asset aparou do lado esquerdo do sprites (sprite assets apenas)
  • [5] = quantidade de pixels que o compilador asset aparou do lado superior do sprites (sprite assets apenas)
  • [6] = porcentagem normalizada de dados de pixels da largura original sprites que foi salva na página texture (sprite assets apenas)
  • [7] = porcentagem normalizada de dados de pixels da altura original sprites que foi salva na página texture (sprite assets apenas)

Este valor pode então ser usado em outras funções de desenho, particularmente em geral 3D e algumas das funções primitivas 2D, assim como as funções Shader. Se você precisar do UVS para um sprite então você pode usar o sprite_get_uvs() e para um font, font_get_uvs().

Sintaxe:

texture_get_uvs(texid)

Argumento Tipo Descrição
texid Textura The texture pointer to get the UVS for

Devoluções:

Array (4 - 8 elements)

Exemplo:

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

O código acima recupera primeiro o texture para a superfície armazenada no surf_back, e depois obtém as coordenadas UV para isso texture. Em seguida, ele recupera as coordenadas UV esquerda, superior, direita e inferior da matriz retornada e as armazena em variáveis locais.
