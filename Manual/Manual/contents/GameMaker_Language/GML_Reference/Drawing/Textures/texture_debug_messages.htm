texturas_debug_messages

Esta função pode ser usada para habilitar ou desabilitar texture mensagens de depuração. Quando ativada (definida para true), informações adicionais sobre o uso da página texture serão enviadas para a janela de saída. Configurar para false para desabilitar esta saída novamente.

Sintaxe:

texture_debug_messages(enable);

ArgumentoTipo Descrição
enable Enable or disable the texture debug messages

Devoluções:

Exemplo:

if global.debug
{
    texture_debug_messages(true);
}

O código acima verificará o valor de uma variável global e se estiver configurado para true então texture mensagens de depuração estão habilitadas.
