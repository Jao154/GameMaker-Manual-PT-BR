grupo de textura_get_fonts

Com esta função, você pode recuperar o índice font de cada uma das páginas fonts atribuídas a texture dentro do grupo texture dado. Você fornece o ID do grupo texture string (conforme definido no editor do grupo texture ) e a função retornará uma matriz 1D onde cada entrada contém o índice font para um recurso font. Se a função falhar - ou seja: um grupo inválido é dado, ou o grupo não tem texture atribuído a ele - então a matriz estará vazia (0 comprimento).

Sintaxe:

texturegroup_get_fonts(tex_id);

Argumento Tipo Descrição
tex Cordão The name of the texture group to check (a string)

Devoluções:

Array

Exemplo:

var _tex_array = texturegroup_get_fonts( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
}

O código acima recuperará os índices font para o grupo texture "MainMenu", depois exibirá essas IDs na janela de saída do console.
