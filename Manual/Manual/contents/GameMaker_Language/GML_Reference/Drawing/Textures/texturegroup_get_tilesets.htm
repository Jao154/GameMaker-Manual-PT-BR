grupos de texturas_get_tilesets

Com esta função você pode recuperar o índice do tileset de cada uma das páginas tilesets atribuídas a texture dentro do grupo texture. Você fornece o ID do grupo texture string (conforme definido no editor do grupo texture ) e a função retornará uma matriz 1D onde cada entrada contém o índice do tileset para um recurso tileset. Se a função falhar - ou seja: um grupo inválido é dado, ou o grupo não tem texture atribuído a ele - então a matriz estará vazia (0 comprimento).

Sintaxe:

texturegroup_get_tilesets(tex_id);

Argumento Tipo Descrição
tex Cordão The name of the texture group to check (a string)

Devoluções:

Array

Exemplo:

var _tex_array = texturegroup_get_tilesets( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i]));
}

O código acima recuperará os índices do tileset para o grupo texture "MainMenu", depois exibirá esses IDs na janela de saída do console.
