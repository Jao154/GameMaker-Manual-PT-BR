largura_de_traço_largura_color

Com esta função você pode traçar uma linha de uma largura específica com a cor misturada entre a cor 1 no primeiro ponto e a cor 2 no segundo ponto. As configurações de cores sobreporão o conjunto de cores de base com a função draw_set_colour(). Favor observar que a linha a ser desenhada pode precisar de valores diferentes (+/-1 no x, y) para ser desenhada com as dimensões desejadas devido às diferenças entre as várias plataformas suportadas.

NOTA: Se você estiver querendo desenhar uma forma usando um shader, você deve estar ciente de que a maioria shaders espera as seguintes entradas: vertex, texture, Cor. Entretanto, ao usar esta função, apenas vertex e dados de cor estão sendo passados, e assim o shader não pode desenhar nada (ou desenhar algo, mas não corretamente). Se você precisar desenhar formas desta forma, então o shader deve ser personalizado com isto em mente.

Sintaxe:

draw_line_width_colour(x1, y1, x2, y2, w, col1, col2);

ArgumentoTipo Descrição
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
w The width in pixels of the line.
col1 The colour of the start of the line.
col2 The colour of the end of the line.

Devoluções:

Exemplo:

draw_line_width_colour(50, 50, 300, 50, 4, c_red, c_blue);

Isto desenharia uma linha horizontal de (50,50) a (300,50), quatro pixels de largura, com um gradiente suave de vermelho a azul.
