draw_text

Com esta função você pode desenhar qualquer string em qualquer posição dentro do room (para desenhar números reais você deve usar o string() função para convertê-los em texto). Para combinar strings você pode usar + (veja exemplo abaixo) e também pode usar \n dentro de um string para adicionar uma quebra de linha para que ela seja desenhada sobre várias linhas (para informações sobre como formatar corretamente um string e quais caracteres de escape você pode usar, por favor veja aqui). A cor do texto e o alfa são regidos pelos valores atuais da base alfa e da cor, conforme definido por draw_set_alpha() e draw_set_colour().

NOTA: A posição real do texto será influenciada pelos valores de alinhamento definidos por draw_set_halign() e draw_set_valign().

Sintaxe:

draw_text(x, y, string);

ArgumentoTipo Descrição
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

Devoluções:

Exemplo:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

O código acima desenhará um string na posição x/y da instância, que utilizará o string armazenado na variável global "Name" e o dividirá em duas linhas.
