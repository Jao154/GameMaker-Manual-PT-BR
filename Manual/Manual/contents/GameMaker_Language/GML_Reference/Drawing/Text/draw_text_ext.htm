texto_desenho_texto

Esta função irá desenhar texto de forma semelhante a draw_text() somente agora você pode definir o espaço entre cada linha de texto - se o texto ocupar mais de uma linha - e limitar a largura (em pixels) do string por linha, de modo que se qualquer linha exceder este valor, o GameMaker dividirá automaticamente o texto para a próxima linha no espaço branco disponível mais próximo (se o texto não tiver espaços em branco, então ele excederá este máximo valor da largura). Note que qualquer espaço branco colocado no início do string será removido antes de ser analisado para desenho por causa disso. Observe também que um valor de -1 para o argumento de separação de linha será padrão para uma separação baseada no altura do caractere "M" no escolhido font.

Sintaxe:

draw_text_ext(x, y, string, sep, w);

ArgumentoTipo Descrição
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

Devoluções:

Exemplo:

draw_text_ext(100, 50, keyboard_string, 3, 300);

O código acima desenhará qualquer texto que o usuário digitar no teclado, dividindo-o em novas linhas toda vez que o comprimento string para aquela linha exceder 300 pixels. o código também manterá uma separação de 3 pixels entre linhas caso isso ocorra.
