draw_text_transformed_colour

Esta função é uma combinação da base draw_text() função com o draw_text_transformed() e draw_text_colour() permitindo escalar e girar o texto, bem como colori-lo com um preenchimento gradiente e alterar seu valor alfa, ignorando as configurações básicas alfa e de cor para desenho.

NOTA A mistura gradual não está disponível para o alvo HTML5 a menos que a WebGL esteja habilitada, embora você ainda possa definir as cores da mistura e ela irá misturar o font com a primeira cor dada. No entanto, toda mistura desta forma cria uma duplicata font que é então armazenado no cache e usado quando necessário, o que está longe de ser o ideal e se você usar mudanças múltiplas de cor irá diminuir o desempenho de seus jogos. Você pode definir o tamanho font cache para tentar limitá-lo, caso seja necessário. usando a função font_set_cache_size().

Sintaxe:

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

ArgumentoTipo Descrição
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
xscale The horizontal scale.
yscale The vertical scale.
angle The angle of the text.
c1 The colour for the top left of the drawn text.
c2 The colour for the top right of the drawn text.
c3 The colour for the bottom right of the drawn text.
c4 The colour for the bottom left of the drawn text.
alpha The alpha for the text.

Devoluções:

Exemplo:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

O código acima desenhará o texto dado no meio do room, girando redondo e escalonado para o dobro de seu tamanho original, com um gradiente de cor passando de amarelo para vermelho, assim como um alfa de 0,5.
