Ópera GX e GXC

Esta seção delineia as funções previstas para a plataforma alvo da Opera GX. As funções listadas nesta página permitem que você grave sua tela e a carregue para o GXC junto com sua pontuação de desafio, no entanto, note que você requer o uso da biblioteca API do GXC para conseguir isso. Consulte a seção Opera GX no site YoYo Games Helpdesk para instruções sobre como implementar a biblioteca API do GXC e trabalhar com desafios.

As seguintes funções são fornecidas para o upload de gravações de vídeo com desafios:

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
