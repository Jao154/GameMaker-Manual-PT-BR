gxc_start_movie_recording

Esta função é usada para começar a registrar seu desafio no GXC. Você especifica a largura, altura e taxa de quadros da gravação, e um booleano para saber se o áudio deve ou não ser gravado.

Sintaxe:

gxc_start_movie_recording(width, height, fps, record_audio);

Argumento Tipo Descrição
width Real The width of the recording
height Real The height of the recording
fps Real The frame rate (frames-per-second) of the recording
record_audio Booleano Whether audio should be recorded (true) ou não (false)

Devoluções:

N/A

Exemplo:

gxc_start_movie_recording(320, 240, 4, false);

Isto iniciará uma gravação de vídeo com uma resolução de 320x240 a 4 quadros por segundo, sem que nenhum áudio seja gravado.
