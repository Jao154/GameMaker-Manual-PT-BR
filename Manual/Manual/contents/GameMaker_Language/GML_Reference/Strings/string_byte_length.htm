string_byte_length

Esta função retorna o número de bytes em um string. Devido a serem mantidos como UTF8, este não será igual ao número de caracteres no string.

Sintaxe:

string_byte_length(string);

Argumento Tipo Descrição
string Cordão The string to measure the number of bytes of.

Devoluções:

Real

Exemplo:

bytesize = string_byte_length("Hello World");

Isto definiria o número de bytes no "Hello World".
