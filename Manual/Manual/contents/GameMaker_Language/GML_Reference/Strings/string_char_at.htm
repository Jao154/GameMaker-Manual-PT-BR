string_char_at

Você pode usar esta função para retornar um caractere específico em uma posição específica dentro de um string, com o índice começando em 1 para o primeiro caractere. Se nenhum caractere for encontrado ou se o string for menor que o valor do índice dado, um string "" vazio é retornado, porém se o índice dado for igual ou menor que 0, então o primeiro caractere do string é retornado.

Sintaxe:

string_char_at(str, index);

ArgumentoTipo Descrição
str The string to check.
index The position to get the character from.

Devoluções:

Exemplo:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Isto colocará str2 no sétimo caractere ("H" contando como o 1º) em string str1, neste caso "W".
