string_byte_at

Retorna o valor do byte bruto como um valor real em uma determinada posição no dado string.

Sintaxe:

string_byte_at(str, index);

ArgumentoTipo Descrição
str The string to check.
index The position to get the byte from.

Devoluções:

Exemplo:

newbyte = string_byte_at("Hello World", 5);

Isto colocará newbyte no valor do byte bruto da sexta carta do "Hello World".
