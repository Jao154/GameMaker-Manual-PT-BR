string_insert

Com esta função você pode criar um novo string composto de dois strings, onde um foi inserido no outro em uma determinada posição. Pode ser útil, por exemplo, adicionar um nome de usuário em um texto pré-definido e assim fazer o jogador de seu jogo sentir mais envolvidos na ação. Tenha em mente que ao calcular a posição a ser inserida nesse strings são indexados a partir de 1, portanto o primeiro caractere em um string é 1 e não 0, como você pode esperar.

Sintaxe:

string_insert(substr, str, index);

ArgumentoTipo Descrição
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

Devoluções:

Exemplo:

str2 = string_insert(username, "Hello, , how are you?", 8);

Isto irá inserir o string na variável "username" na frase dada, com o resultado string parecendo assim: "Hello, NAME, how are you?".
