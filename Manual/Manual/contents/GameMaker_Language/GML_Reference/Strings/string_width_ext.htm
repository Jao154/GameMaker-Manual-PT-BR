largura_de_filtro_extra

Esta função retornará a largura máxima (em pixels) da entrada string, levando em conta a separação de linha e a largura de quebra de linha (que é definida como o número de pixels que o string pode ocupar antes que uma quebra de linha seja inserida). É muito útil para calcular distâncias entre elementos de texto com base na largura máxima de um string que está dividido em várias linhas, uma vez que seria desenhado com draw_text_ext() usando o atualmente definido font. A separação e a largura podem ser definidas como -1 para obter o espaçamento padrão.

Sintaxe:

string_width_ext(string, sep, w);

ArgumentoTipo Descrição
string The string to measure the width of.
sep The distance in pixels between lines of text as if the string was being drawn.
w The maximum width (in pixels) of the string before a line break as if the string was bring drawn.

Devoluções:

Exemplo:

var ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

O código acima obterá a largura do dado string, levando em conta a separação de linhas e a largura de quebra de linha, e então desenhará duas linhas de texto, usando a largura total retornada string como separador.
