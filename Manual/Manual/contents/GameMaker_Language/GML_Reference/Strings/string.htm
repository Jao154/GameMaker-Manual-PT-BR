corda

Com esta função você pode transformar qualquer número real em um string. Se o número real for um número inteiro, ele será salvo sem casas decimais, caso contrário, ele será salvo com duas casas decimais. Se você precisar de mais casas decimais, então use a função string_format(). Observe também que o uso desta função em uma variável que armazena uma matriz, uma estrutura de dados ou uma estrutura irá converter o conteúdo destas variáveis em um string que pode então ser emitido para o console ou salvo em um arquivo para debugging.

Sintaxe:

string(val);

Argumento Tipo Descrição
val Real The real number to be turned into a string.

Devoluções:

String

Exemplo:

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

O código acima usa a função string para desenhar os dois números reais e strings juntos ( o sorteio só aceitará um string ou um real, mas não ambos).
