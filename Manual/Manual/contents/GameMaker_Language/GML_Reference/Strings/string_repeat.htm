string_repeat

Esta função simplesmente retorna a mesma string repetida um determinado número de vezes sobre si mesma.

Sintaxe:

string_repeat(str, count);

ArgumentoTipo Descrição
str The string to repeat.
count The number of times to repeat the string.

Devoluções:

Exemplo:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

O código acima definirá str2 para "Hello WorldHello WorldHello World".
