string_copy

Com esta função você pode selecionar facilmente um número de caracteres de dentro de um string para ser copiado para outro. O primeiro caractere de um string é sempre indexado como 1 e não 0 como você pode esperar, assim, para copiar (por exemplo) os primeiros cinco caracteres de string você teria string_copy(str, 1, 5). Um outro exemplo é fornecido abaixo.

Sintaxe:

string_copy(str, index, count);

ArgumentoTipo Descrição
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

Devoluções:

Exemplo:

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

O código acima permitirá que o jogador introduza um string através do teclado, que é então armazenado na variável name. Esta variável é então verificada para ver se excede o comprimento máximo de 15 caracteres e se excede, o teclado_string é substituído por uma cópia de 15 caracteres do name string. Isto efetivamente limita a entrada do jogador a 15 caracteres.
