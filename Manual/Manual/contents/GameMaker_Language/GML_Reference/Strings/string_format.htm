string_format

Transforma um número real em um string usando sua própria formatação, onde você pode escolher quantas "casas" são salvas no string e quantas casas decimais também são salvas. Ambos podem ser muito úteis, alguns jogos preferem exibir uma pontuação como um número definido de dígitos, enquanto o controle sobre as casas decimais pode ser bom para uma alta precisão as duas casas decimais de string() não pode fornecer. Se o número de casas especificado for maior que o valor a ser mostrado e/ou o número mais as casas decimais que foram especificadas for menor que o total de casas, então espaços serão adicionados antes do valor para compensar a diferença (veja o exemplo abaixo). Zeros serão adicionados à direita do ponto decimal se o valor dado for menor que o total e o número de casas decimais a incluir. O formato padrão não é nenhum espaço extra à esquerda e somente duas casas decimais à direita, por exemplo "265,73".

Sintaxe:

string_format(val, tot, dec);

Argumento Tipo Descrição
val Real The real number to be turned into a string.
tot Real The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec Real The number of decimal places to be included.

Devoluções:

String

Exemplo:

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

Isto colocará str1 a " 1234", str2 a " 3.1415926535" e str3 a " 3.14159".
