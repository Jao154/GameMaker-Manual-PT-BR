string_last_pos

Esta função retornará a posição de caráter de uma instância de uma sub-corda dentro de um string, pesquisando desde o final do string até o início (portanto, o inverso de string_pos(). A função retornará 0 se a busca string não for encontrada, ou a posição do primeiro caractere da busca string se for. Tenha em mente que para o suporte antigo strings são indexados a partir de 1, portanto 1 é a primeira posição no string, e não 0 como você pode esperar.

Sintaxe:

string_last_pos(substr, str);

ArgumentoTipo Descrição
substr The substring to look for in the string.
str The string.

Devoluções:

Exemplo:

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

O código acima procura no string armazenado na variável "texto" por uma vírgula começando a busca a partir do final do string. Se encontrar uma, insere o "nome" do substrato nessa posição.
