string_count

Esta função devolverá a quantidade de vezes que o substrato dado aparece dentro de um string específico. Desta forma, você pode verificar quantas vezes uma única letra ou frase é repetida em uma seção do texto armazenado.

Sintaxe:

string_count(substr, str);

ArgumentoTipo Descrição
substr The substring to check the string for.
str The string to check.

Devoluções:

Exemplo:

str1 = "Hello World";
ocount = string_count( "o", str1 );

Isto definirá a variável ocount para o número de "o "s em str1, neste caso 2.
