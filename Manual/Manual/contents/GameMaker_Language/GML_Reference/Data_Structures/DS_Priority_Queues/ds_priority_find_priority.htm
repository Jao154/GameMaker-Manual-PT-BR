ds_priority_find_priority

Com esta função, você pode recuperar a prioridade de qualquer valor dado. Se o valor não existir na prioridade queue, então undefined será devolvido.

Sintaxe:

ds_priority_find_priority(id, val);

ArgumentoTipo Descrição
id The id of the priority queue to use.
val The value to find the priority of.

Devoluções:

Real or undefined

Exemplo:

p = ds_priority_find_priority(ai_priority, "intelligence");

O código acima armazenará a prioridade de retorno para o valor dado na variável de instância "p".
