ds_queue_enqueue

Esta função agregará um valor (real ou string) à cauda da fila DS. A função pode levar mais 14 argumentos opcionais (fazendo um total de 15 adições possíveis), permitindo que você acrescente vários valores consecutivamente à cauda do queue em uma única chamada.

Sintaxe:

ds_queue_enqueue(id, val [, val2, ... val15]);

ArgumentoTipo Descrição
id The id of the queue to add to.
val The value to add to the queue.
[val2, ... val15] Optional values to be added to the queue.

Devoluções:

Exemplo:

move_queue = ds_queue_create();
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y);
ds_queue_enqueue(move_queue, x + 200);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y + 200);
ds_queue_enqueue(move_queue, x);
ds_queue_enqueue(move_queue, y);

O código acima cria uma nova fila de DS e armazena seu índice na variável "move_queue". Em seguida, ele empurra uma série de valores para o queue para uso futuro.
