ds_stack_push

Esta função irá empurrar (adicionar) um valor, que pode ser de qualquer tipo de dado, para o topo da pilha. A função pode levar mais 14 argumentos opcionais (fazendo um total de 15 adições possíveis), permitindo que você empurre vários valores consecutivamente para a pilha em uma única chamada.

Sintaxe:

ds_stack_push(id, val [, val2, ... val15]);

ArgumentoTipo Descrição
id The id of the data structure to push onto.
val The value to push onto the stack.
[val2, ... val13] Optional values to be added to the stack.

Devoluções:

Exemplo:

move_stack = ds_stack_create();
ds_stack_push(move_stack, x, y, x, y + 200, x + 200, y + 200, x +200, y);

O código acima cria uma nova pilha DS e armazena seu índice na variável "move_stack". Ele então empurra uma série de valores para a pilha para uso futuro.
