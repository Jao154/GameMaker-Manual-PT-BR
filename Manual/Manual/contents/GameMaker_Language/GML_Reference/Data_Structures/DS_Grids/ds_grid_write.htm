ds_grid_write

Esta função pode ser usada para converter a dada ds_grid em um string, que pode então ser armazenada em um arquivo externo (por exemplo). Você pode ler o retorno string desta função de volta para uma ds_grid usando a função ds_grid_read().

NOTA O retorno string não é uma leitura humana string, mas sim um lixão do conteúdo da estrutura de dados.

Sintaxe:

ds_grid_write(index);

ArgumentoTipo Descrição
index The index of the grid to write.

Devoluções:

Exemplo:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

O código acima abrirá um arquivo ini (criando-o se ainda não existir) e então escreverá a ds_grid dada como um string para esse arquivo.
