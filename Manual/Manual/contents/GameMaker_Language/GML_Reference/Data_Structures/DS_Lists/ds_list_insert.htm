ds_list_insert

Esta função adicionará o valor dado à lista na posição dada. se a lista contiver mais valores após a posição dada, sua posição será deslocada para cima um para fazer room tornando a lista maior por um.

Sintaxe:

ds_list_insert(id, pos, val);

Argumento Tipo Descrição
id DS List ID The id of the list to add to.
pos Real The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val Variável The value to add to the list.

Devoluções:

N/A

Exemplo:

ds_list_insert(list, 9, score);

O código acima adicionará o valor armazenado na variável "pontuação" na 10ª posição da lista indexada pela variável "lista" (listas são contadas a partir de 0, portanto uma lista de dez valores é numerada de 0 a 9).
