clickable_change

Com esta função você pode alterar o sprite e a posição de um ícone clicável previamente criado com clickable_add(). Observe que a posição é baseada na janela, não na tela, (0,0) posição e que o sprite deve ser referenciado diretamente da página texture (veja: sprite_get_tpe()).

Sintaxe:

clickable_change(index, tpe, x, y)

Argumento Tipo Descrição
index Identificação clicável The index of the clickable icon to change.
tpe Entrada de página de textura The texture page entry for the sprite to be used.
x Real The new x position within the window.
y Real The new y position within the window.

Devoluções:

N/A

Exemplo:

switch (room)
{
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

O código acima mudará o índice de imagem e a posição do ícone clicável indexado na variável "global.Help", dependendo do atual room.
