altura_do_browser

Esta variável mantém a altura (em pixels) do navegador em que o jogo está sendo executado. Se nenhum navegador estiver presente, o tamanho da janela é retornado. a imagem a seguir ilustra como esta variável se relaciona com o resto das funções de posicionamento disponíveis através das Funções de Janela.

Sintaxe:

browser_height

Devoluções:

Exemplo:

if (browser_height > window_get_height()) || (browser_width > window_get_width())
{
    var xx, yy;
    if (browser_width > window_get_width())
    {
        xx = (browser_width - window_get_width()) / 2;
    }
    else
    {
        xx = 0;
    }
    if (browser_height > window_get_height())
    {
        yy = (browser_height - window_get_height()) / 2;
    }
    else
    {
        yy = 0;
    }
    window_set_position(xx, yy);
}

O código acima verifica a largura e a altura do navegador para ver se algum deles é maior do que a largura e a altura da janela atual. Se algum deles for, então calcula uma nova posição para a janela do jogo dentro do navegador.
