clickable_add_ext

Com esta função você pode criar um ícone personalizado, clicável, que atua como um "botão" fora da tela do jogo, mas dentro da própria janela do navegador. A função retornará um valor de identificação que deve ser armazenado em uma variável para ser usada em todas as outras funções de botão que referenciam a nova opção clicável. Esta função é muito útil para evitar que o navegador crie uma janela pop-up ao clicar no botão, pois tudo isso é feito através do próprio GameMaker.

Esta função funciona exatamente da mesma forma que a clickable_add() onde você deve definir a posição relativa à (0,0) da janela do navegador, dê a entrada da página texture para o ícone do botão sprite que você deseja que o botão tenha (veja: sprite_get_tpe()), especifique ou uma URL para abrir ou um script (do navegador Asset do GameMaker) a ser aberto ou executado e então defina o alvo e os parâmetros a serem usados. Entretanto, esta função também tem duas opções adicionais que permitem definir o ícone alfa de 0 (totalmente transparente) para 1 (totalmente opaco) e a escala do ícone, que é um valor multiplicador aplicado ao sprite utilizado. Portanto, um valor de 1 é a escala padrão, enquanto (por exemplo) 0,5 seria a metade da escala e 2 seria o dobro.

Também vale a pena notar que se o argumento URL que você passa é o nome de uma função de script (ou seja: não começa com " script ") E o é chamado gmcallback_* então você pode obter feeds diretos em um evento html5. Os nomes das funções de script do GameMaker não são ofuscados se começarem com gmcallback_.

Sintaxe:

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

Argumento Tipo Descrição
x Real The x position within the window.
y Real The y position within the window.
tpe Entrada de página de textura The texture page entry for the sprite to be used.
url Cordão The URL (website address) to link to, or the script to be run.
target Cordão This is the target area to open the URL in (see description).
params Cordão The various parameters for the icon (see description).
alpha Real The image alpha of the icon (default 1).
scale Real The scale of the icon (default 1).

Devoluções:

Clickable ID

Exemplo:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

O código acima cria um ícone DOM clicável na posição (32, 32) da página em que a tela do jogo está rodando. O ícone usa o sprite referido da página texture como "spr_MS_Home" e quando clicado o ícone abrirá uma nova janela para a URL especificada e com as propriedades definidas para aquela janela. o botão também será escalado para o dobro do tamanho original sprite e metade de uma imagem alfa de 1.
