url_open_ext

Isto abrirá a URL especificada no navegador do dispositivo alvo escolhido, ou, se você estiver usando o módulo HTML5, no navegador atualmente aberto. O parâmetro "target" que você especifica é o mesmo que o valor padrão do "nome" JavaScript quando você usa o método open(), e você deve estar ciente de que tudo menos "_self" pode resultar no bloqueio do navegador, ou perguntando ao usuário se ele deseja permitir isso.

Os alvos válidos são:

Meta Descrição
em branco Abre o documento vinculado em uma nova janela ou aba (isto não funcionará se os pop-ups estiverem sendo bloqueados pelo usuário, caso em que você pode usar o clickable_* funções em seu lugar).
_self Abre o documento vinculado no mesmo quadro em que foi clicado (este é o padrão).
pai Abre o documento vinculado no quadro superior.
_topo Abre o documento vinculado no corpo completo da janela.

Sintaxe:

url_open_ext(url, target);

Argumento Tipo Descrição
url Cordão The URL (website address) to link to.
target Cordão This is the target area to open the URL in.

Devoluções:

N/A

Exemplo:

url_open_ext("http://gamemaker.io", "_blank");

Isto abriria a página inicial do GameMaker em uma nova janela.
