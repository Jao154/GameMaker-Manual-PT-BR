url_open

Isto abrirá a URL especificada no navegador do dispositivo alvo escolhido, ou, se você estiver usando o módulo HTML5, no navegador atualmente aberto.

Sintaxe:

url_open(url);

Argumento Tipo Descrição
url Cordão The URL (website address) to link to.

Devoluções:

N/A

Exemplo:

url_open("http://gamemaker.io");

Isto abriria a página inicial do GameMaker na janela atual.
