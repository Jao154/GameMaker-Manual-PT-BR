botão_de_marcação_pressionado_do_mouse

Esta função retornará true se o botão do mouse que está sendo verificado tiver sido pressionado ou false se não tiver sido. Esta função só será acionada uma vez para qualquer botão do mouse quando ele for pressionado pela primeira vez e para acioná-lo novamente o botão precisará ter sido liberado e pressionado novamente. Note que ela será considerada acionada durante a duração da etapa, e para todas as instâncias que tenham qualquer evento do mouse ou que usem esta mesma função.

Você fornece o botão do mouse para verificar a partir de uma das seguintes constantes:

Sintaxe:

mouse_check_button_pressed(numb);

Argumento Tipo Descrição
numb Botão do Rato Constante Which mouse button constant to check for.

Devoluções:

Boolean

Exemplo:

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

O código acima verificará se o botão esquerdo do mouse foi pressionado e se ele tem, acrescentará 50 à pontuação.
