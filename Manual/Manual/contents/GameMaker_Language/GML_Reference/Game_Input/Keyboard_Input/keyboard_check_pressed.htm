teclado_check_pressed

Com esta função você pode verificar se uma tecla foi pressionada ou não. Ao contrário da função keyboard_check() esta função só funcionará uma vez para cada vez que a tecla for pressionada, portanto, para que ela seja acionada novamente, a tecla deve ser primeiramente liberada e depois pressionada novamente. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal de Entrada de Teclado.

Sintaxe:

keyboard_check_pressed(key);

Argumento Tipo Descrição
key Constante Chave Virtual (vk_*) The key to check the pressed state of.

Devoluções:

Boolean

Exemplo:

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

O código acima avançará para o próximo room se o jogador pressionar qualquer uma das teclas do teclado (funcionando como um prompt "Press Any Key to Continue").
