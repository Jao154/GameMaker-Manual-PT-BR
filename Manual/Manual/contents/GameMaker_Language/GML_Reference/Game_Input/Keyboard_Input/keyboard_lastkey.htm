teclado_última tecla

Esta variável se refere ao valor que keyboard_key estava no quadro anterior, devolvendo o código chave dessa chave (todas as constantes de código chave padrão são devolvidas). Esta variável não é somente lida e você pode modificá-la, por exemplo, para defini-la como -1 se você já a manipulou.

Sintaxe:

keyboard_lastkey;

Devoluções:

Virtual Key Constant

Exemplo:

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

O código acima verifica se a última variável chave não é igual a -1, e se é ela adiciona qualquer que seja a última chave como um string à variável "str", então ela redefine a variável keyboard_lastkey para aceitar novas entradas.
