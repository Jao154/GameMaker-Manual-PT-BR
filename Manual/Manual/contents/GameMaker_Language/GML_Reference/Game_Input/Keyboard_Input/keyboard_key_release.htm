teclados_chaves_liberação

Com esta função você pode simular a liberação de qualquer tecla no teclado. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal do Keyboard Input.

Sintaxe:

keyboard_key_release(key);

ArgumentoTipo Descrição
key The key to simulate a release of.

Devoluções:

Exemplo:

keyboard_key_release(vk_space);

Isto irá simular uma liberação da barra de espaço.
