teclado_check_direct

Esta função retornará true se a tecla com o código de chave particular for pressionada, ou false se não for, verificando diretamente o hardware. Ela permite mais algumas verificações, em particular, você pode usar os códigos de chave vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol e vk_ralt para verificar se a tecla esquerda ou direita, controle ou alt está pressionada. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal de Entrada de Teclado.

NOTA Esta função só está disponível para o alvo padrão Windows e o resultado é independente de qual aplicação tem foco.

Sintaxe:

keyboard_check_direct(key);

ArgumentoTipo Descrição
key The key to check the down state of.

Devoluções:

Exemplo:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

O código acima verificará se as teclas alt esquerda ou direita foram pressionadas, e se elas têm a variável "agachar" para verdadeira.
