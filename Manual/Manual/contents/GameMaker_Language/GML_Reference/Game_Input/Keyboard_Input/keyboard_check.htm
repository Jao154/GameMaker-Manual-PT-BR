teclado_check

Com esta função, você pode verificar se uma chave está ou não pressionada. Ao contrário da função keyboard_check_pressed() ou keyboard_check_released() funções que só são acionadas uma vez quando a tecla é pressionada ou liberada, esta função é acionada a cada passo em que a tecla é mantida pressionada. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal de Entrada de Teclado.

Sintaxe:

keyboard_check(key);

Argumento Tipo Descrição
key Constante Chave Virtual (vk_*) The key to check the down state of.

Devoluções:

Boolean

Exemplo:

if keyboard_check(vk_left)
{
    x -= 5;
}

O código acima verificará se a tecla de seta está sendo pressionada e moverá a instância 5 pixels deixados a cada passo que ela retorna verdadeiro.
