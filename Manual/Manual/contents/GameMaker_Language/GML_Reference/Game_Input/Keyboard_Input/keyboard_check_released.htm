teclado_check_released

Com esta função você pode verificar se uma chave foi liberada ou não. Ao contrário da função keyboard_check() esta função só funcionará uma vez para cada vez que a chave for levantada, portanto, para que ela seja acionada novamente, a chave deve ser primeiramente pressionada e depois liberada novamente. A função terá um valor de código de chave como retornado pela função ord() (somente letras maiúsculas de A-Z ou números de 0-9), ou qualquer uma das constantes vk_* listadas na página principal de Entrada de Teclado.

Sintaxe:

keyboard_check_released(key);

ArgumentoTipo Descrição
key The key to check the released state of.

Devoluções:

Exemplo:

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

O código acima verificará se a chave "P" foi liberada e, em caso afirmativo, criará uma instância de "obj_Pause".
