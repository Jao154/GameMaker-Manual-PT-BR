teclado_chave

Com esta variável você pode obter o código da tecla que está sendo pressionada atualmente e ele retornará 0 se nenhuma tecla estiver sendo pressionada quando a verificação for feita.

Sintaxe:

keyboard_key;

Devoluções:

Virtual Key Constant

Exemplo:

switch (keyboard_key)
{
    case vk_numpad1: gun = weapon[0][0]; break;
    case vk_numpad2: gun = weapon[1][0]; break;
    case vk_numpad3: gun = weapon[2][0]; break;
}

O código acima usa o valor da variável keyboard_key para definir uma variável com o mesmo valor de uma matriz.
