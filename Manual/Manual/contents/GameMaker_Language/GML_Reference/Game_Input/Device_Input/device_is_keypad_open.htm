device_is_keypad_open

Isto faz uma verificação do dispositivo para um teclado e se um estiver disponível ele retorna true caso contrário ele retorna false. Note que esta função é principalmente para uso com dispositivos Android.

Para aqueles usuários com um Sony Xperia Play, há um conjunto de botões/mapas de teclas no GameMaker, assim você pode usar as constantes de teclado vk_up, vk_down, vk_left, vk_ right para as teclas do joypad e Triangle é ord("T"), Square é ord("S"), Circle é vk_alt + vk_backspace enquanto o botão voltar é simplesmente vk_backspace, Cross é vk_space, Select é vk_return e Start é vk_rshift, o L Trigger é ord("L") e R Trigger é ord("R").

Sintaxe:

device_is_keypad_open

Devoluções:

Exemplo:

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

O código acima verifica se há um teclado e então muda uma variável global, dependendo do valor retornado.
