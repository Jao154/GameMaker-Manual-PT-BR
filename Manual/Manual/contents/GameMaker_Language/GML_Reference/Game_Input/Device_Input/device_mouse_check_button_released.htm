device_mouse_check_button_released

Esta função retorna true ou false, dependendo se o toque do dispositivo que você especificar foi liberado ou não. O argumento do dispositivo refere-se ao número de toque do dispositivo, que pode ser de 0 a n e observe que o número máximo de toques que podem ser detectados dependerá muito do dispositivo sendo usado e do sistema operacional que ele executar (a maioria dos dispositivos suportará pelo menos 4 toques simultâneos). Esta função é acionada apenas uma vez pela ação de liberação real, e as constantes listadas nesta página podem ser usadas para verificar os botões do mouse. Note que mb_right só terá sido detectado se um toque duplo for detectado e mantido no segundo toque antes do lançamento (este comportamento pode ser desativado usando a função device_mouse_dbclick_enable()).

Sintaxe:

device_mouse_check_button_released(device, button);

ArgumentoTipo Descrição
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

Devoluções:

Exemplo:

if device_mouse_check_button_released(0, mb_left)
{
    press = false;
}

O código acima verifica se o dispositivo foi pressionado e define uma variável como falsa, se for o caso.
