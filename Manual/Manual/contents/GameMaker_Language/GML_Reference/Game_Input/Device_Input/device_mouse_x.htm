dispositivo_mouse_x

Esta função retorna a posição x de um toque no dispositivo. Se você estiver executando isto nos módulos HTML5 ou PC e Mac, então este valor é atualizado constantemente, desde que o dispositivo (geralmente um mouse) esteja conectado, porém para dispositivos móveis, isto só será atualizado enquanto a tela estiver sendo tocada e note que o número máximo de toques que podem ser detectados dependerá muito do dispositivo sendo usado e do sistema operacional que ele roda.

Sintaxe:

device_mouse_x(device);

ArgumentoTipo Descrição
device The device (from 0 - n) that is being checked.

Devoluções:

Exemplo:

if device_mouse_check_button(0, mb_left)
{
    if device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

O código acima verifica se o dispositivo está sendo pressionado e, em caso afirmativo, verifica a posição x do dispositivo para ver se ele está dentro dos parâmetros. Se for, ele ajusta a variável "pressionado" para true, outros ajustes são feitos para false.
