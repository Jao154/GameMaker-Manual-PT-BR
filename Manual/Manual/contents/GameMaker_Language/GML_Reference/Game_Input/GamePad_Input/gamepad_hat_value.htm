gamepad_hat_value

Com esta função você pode obter o valor atual de um "chapéu" de jogo. Cada valor de "chapéu" é uma pequena máscara para as diferentes direções onde:

  • Para cima = 1
  • Direito = 2
  • Abaixo = 4
  • Esquerda = 8

Note que estes podem ser combinados (por exemplo, fornecer um índice de chapéu de 3 seria verificar e corrigir, ou você pode usar o | operador para combinar os vários valores) mas somente 2 bits, no máximo, podem ser pressionados de uma vez. A função retornará um valor real entre 0 e 1, onde 0 não é pressionado e 1 é totalmente pressionado (e pode haver valores intermediários dependendo se o gamepad suporta entrada analógica para chapéus).

Sintaxe:

gamepad_hat_value(device, hatindex);

Argumento Tipo Descrição
device Real Which gamepad device "slot" to check.
hatindex Real Which gamepad hat (or hats) to check for.

Devoluções:

Real

Exemplo:

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
}

O código acima armazena o estado do chapéu "0" em uma variável local, depois verifica qual é o valor de retorno e age de acordo.
