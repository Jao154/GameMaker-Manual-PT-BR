gamepad_get_guid

Com esta função você pode recuperar o GUID para o gamepad conectado a um determinado índice de slots. Você fornece o slot do gamepad para verificar (de 0 a 11), e a função retornará uma das seguintes seqüências:

  • "nenhum" se não houver um GUID disponível ou se não houver um gamepad conectado
  • "índice de dispositivo fora de alcance" se o índice de ranhura do gamepad estiver fora de alcance
  • O GUID string, algo como "050000005e040000fd020000fd020000ffff3f00", se o slot tiver um controle de jogo atribuído e puder obter o GUID para ele

Esta função seria normalmente usada em conjunto com gamepad_get_description() para gerar parte do SDL string necessário para refazer os controladores em Windows, Android e macOS.

Notas específicas da plataforma
Plataforma Nota
Ópera GX Na plataforma alvo Opera GX, esta função sempre retornará "nenhum" para dispositivos não-DirectInput

Sintaxe:

gamepad_get_guid(index);

ArgumentoTipo Descrição
index Which gamepad "slot" index to check

Devoluções:

Exemplo:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

O código acima recebe o GUID e a descrição do nome strings, depois os concatenam e armazenam o final string em uma variável global para referência futura.
