círculo_físico_grupo_de_partículas

Esta função irá definir a forma do grupo de partículas que está sendo criado. Primeiro é preciso ter começado a definição do grupo usando a função physics_particle_group_begin()e então você use esta função para definir o raio aproximado do grupo em pixels - aproximado porque o raio exato dependerá do tamanho das partículas base, conforme definido pelo physics_particle_set_radius() A simulação física tenta "encaixar" o maior número possível de partículas na forma definida. Finalmente, é preciso chamar physics_particle_group_end() para criar o grupo de partículas no site room.

Sintaxe:

physics_particle_group_circle(radius)

ArgumentoTipo Descrição
radius The radius (in pixels) of the circle to create.

Devoluções:

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

O código acima armazena o flags para o tipo de partícula e as propriedades do grupo de partículas em variáveis e depois as utiliza para criar um grupo de partículas circular com um raio de 100px na posição do mouse.
