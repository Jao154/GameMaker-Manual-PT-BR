fisica_particle_group_join

Com esta função você pode unir dois grupos de partículas, e os grupos unidos se comportarão como se ambos fizessem parte de uma única entidade. Os grupos deveriam ter sido criados com bordas sobrepostas, como se já não estivessem se tocando, não serão unidos. A função leva os IDs exclusivos dos grupos para se unir, como retornado pela função physics_particle_group_end()e você pode usar a função quantas vezes quiser para que um único grupo junte vários corpos macios.

Sintaxe:

physics_particle_group_join(to, from)

Argumento Tipo Descrição
to ID do Grupo de Partículas Físicas The first particle group to join.
from ID do Grupo de Partículas Físicas The second particle group to join.

Devoluções:

N/A

Exemplo:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x- 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g1 = physics_particle_group_end();
physics_particle_group_begin(flags, groupflags, mouse_x + 45, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(50);
var g2 = physics_particle_group_end();
physics_particle_group_join(g1, g2);

O código acima cria dois grupos de partículas circulares e os une.
