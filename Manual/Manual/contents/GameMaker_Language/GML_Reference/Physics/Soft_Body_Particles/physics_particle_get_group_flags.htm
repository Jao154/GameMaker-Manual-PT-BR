Com esta função você pode recuperar o grupo flags para um grupo de partículas. O valor group é aquele que foi retornado quando você criou o grupo de partículas usando a função physics_particle_group_end()e a função retornará um valor que é o valor combinado do conjunto atual flags.

Sintaxe:

physics_particle_get_group_flags(group)

Argumento Tipo Descrição
group ID do Grupo de Partículas Físicas The particle group to get.

Devoluções:

Real

Exemplo:

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

O código acima criará uma variável para armazenar o valor flags e depois a utilizará para verificar o flags do grupo indexado na variável "grupo". Se eles não forem os mesmos, o grupo é definido com estes flags.
