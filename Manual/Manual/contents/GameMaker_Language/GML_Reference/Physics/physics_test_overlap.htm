fisica_test_overlap

Esta função pode ser usada para verificar e ver se um corpo físico (ou seja: a fixação de uma instância) se sobrepõe, ou se irá sobrepor, quando rotacionado e colocado em uma determinada posição no room. o argumento "ângulo" é o ângulo de rotação que a instância chamadora tem (ou terá) na posição a ser verificada, e o argumento "obj" pode ser ou uma única instância id, e o índice object ou o palavras-chave todos ou outros.

Sintaxe:

physics_test_overlap(xpos, ypos, angle, obj);

Argumento Tipo Descrição
xpos Real The x position in the room to check
ypos Real The y position in the room to check
angle Real The angle to check (of the calling instance)
obj Objeto Ativo ou ID da instância The object to check for

Devoluções:

Boolean

Exemplo:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

O código acima verificará uma posição para uma sobreposição de uma fixação física e, se houver, definirá uma variável e um alarme.
