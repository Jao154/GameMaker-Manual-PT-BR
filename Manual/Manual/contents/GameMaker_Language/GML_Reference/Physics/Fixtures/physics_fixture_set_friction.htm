física_fixture_set_fricção_física

O atrito é a força que resiste ao movimento relativo dos elementos materiais que deslizam um contra o outro, o que no mundo da física GameMaker se traduz na perda de impulso causada pela colisão de duas instâncias com os dispositivos ligados a eles. Assim, quando duas instâncias colidem, seu movimento é afetado por este valor, com um alto atrito causando uma maior perda de impulso do que um valor mais baixo. Observe que o atrito geralmente é definido para um valor entre 0 e 1, mas você pode usar qualquer valor não negativo valor, se necessário.

Sintaxe:

physics_fixture_set_friction(fixture, friction)

ArgumentoTipo Descrição
fixture the index of the fixture
friction the friction of the fixture

Devoluções:

Exemplo:

physics_fixture_set_friction(fix_Brick, 0.1);

O código acima definirá o atrito do acessório indexado em "fix_brick" para 0,1.
