forma_de_série_física

Esta função define uma forma de fixação "borda". Uma forma de borda é simplesmente uma linha que irá gerar uma colisão quando outras fixações sobre ela, e pode ser muito útil para gerar (por exemplo) terreno, ou para criar bordas em torno de um room. A posição da borda é definida utilizando o espaço local, ou seja: a posição x/y da instância é considerada (0,0), portanto, isto deve ser levado em consideração ao criá-las (no exemplo de código abaixo, a instância teria sido colocada em (0,0) no room para evitar complicações).

Sintaxe:

physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2)

ArgumentoTipo Descrição
fixture the index of the fixture
local_x1 start x position for the edge
local_y1 start y position for the edge
local_x2 end x position for the edge
local_y2 end y position for the edge

Devoluções:

Exemplo:

var xx = 0;
var y1 = room_height - 100;
var y2 = room_height - 50 - irandom(100);
for (var i = 0; i < 10; i++;)
{
    var fix = physics_fixture_create();
    physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2);
    physics_fixture_bind(fix, id);
    physics_fixture_delete(fix);
    xx += 50;
    y1 = y2;
    y2 = room_height - 50 - irandom(100);
}

O código acima criará uma linha de "bordas" com uma variedade de alturas ao longo do comprimento do room.
