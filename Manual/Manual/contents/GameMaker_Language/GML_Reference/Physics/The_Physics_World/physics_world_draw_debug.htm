physics_world_draw_debug

Esta função irá desenhar uma representação do sistema mundial de física do room para ajudar com debugging quando você estiver tendo problemas para fazer o cenário da física funcionar corretamente. Às vezes o problema pode estar na parte de renderização de seu jogo, por exemplo, um sprite pode ser desenhado na posição ou rotação errada, dando a aparência de física incorreta, ou pode ser que seus cálculos não estejam corretos. De qualquer forma, esta função o ajudará a encontrar e resolver o problema desenhando o que está acontecendo em seu sistema de física. Para fazer isto, você precisa dizer qual parte do sistema deve desenhar, e isto é feito passando um valor criado a partir dos bits armazenados em várias constantes de "bandeira":

Bandeira de Depuração Física
Bandeira Descrição
phy_debug_render_aabb Isto mostra a caixa de delimitação absoluta de cada fixação em relação ao eixo room
phy_debug_render_collision_pairs Isto mostrará qualquer equipamento que esteja atualmente em colisão
phy_debug_render_coms Isto marca o centro de massa de cada luminária na sala
phy_debug_render_core_shapes Mostra as formas básicas que compõem as luminárias na sala
phy_debug_render_joints Isto desenhará cada uma das juntas de todas as luminárias da sala
phy_debug_render_obb Isto mostra a caixa de delimitação relativa para as luminárias na sala
phy_debug_render_shapes Isto mostra as formas reais que compõem as luminárias dentro da sala

Assim, para desenhar quaisquer duas dessas opções, precisaríamos fazer um pouco "ou" (no GameMaker isto é representado por | ) nas duas e armazenar a variável resultante para a função a ser utilizada. Aqui está um exemplo de como você definiria uma variável para usar com a função que desenhará apenas os centros de massa e as juntas de todas as luminárias em uma sala:

flag = phy_debug_render_coms | phy_debug_render_shapes;

O que lhe daria algo parecido com isto:

Como você pode ver com o código acima, apenas as formas e centros de massa são desenhados na tela (assim como as cores que representam seus estados - para maiores informações, favor verificar a documentação on-line da Caixa 2D). Da mesma forma, você pode exibir mais, ou mesmo todas as constantes que deseja depurar... Apenas "ou" todas elas juntas! Uma outra coisa deve ser observada - esta função deve ser chamada apenas no caso de um sorteio de uma instância que tenha uma profundidade inferior àquelas que você deseja depurar ou então você não será capaz de ver as informações de depuração.

Sintaxe:

physics_world_draw_debug(flag)

Argumento Tipo Descrição
flag Bandeira de Depuração Física(s) The flag value for showing different debug features

Devoluções:

N/A

Exemplo:

flag = phy_debug_render_aabb | phy_debug_render_collision_pairs | phy_debug_render_obb;
physics_world_draw_debug(flag);

O código acima desenhará todas as caixas de delimitação absoluta, as caixas de delimitação relativa, bem como os dispositivos em colisão dentro do sistema físico atual.
