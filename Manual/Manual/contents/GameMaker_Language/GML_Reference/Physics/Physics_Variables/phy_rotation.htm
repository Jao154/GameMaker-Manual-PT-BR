phy_rotation

Esta variável pode ser usada para obter (ou para definir) o ângulo de fixação das instâncias em graus, semelhante à definição ou obtenção do image_angle. Entretanto, observe que no mundo físico as rotações são calculadas de forma oposta ao mundo normal do GameMaker, o que significa que funções vetoriais como point_direction()devem ter seus valores de retorno modificados (simplesmente fazer positivo para negativo deve resolver isso).

Sintaxe:

phy_rotation;

Devoluções:

(single precision floating point value, or undefined if the instance is not physics enabled)

Exemplo:

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

O código acima verifica a velocidade linear e se algum dos vetores não for 0, então calcula a velocidade real e a utiliza para definir a rotação.
