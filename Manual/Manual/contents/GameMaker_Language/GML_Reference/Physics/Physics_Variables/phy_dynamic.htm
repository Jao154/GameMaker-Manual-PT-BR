fitodinâmica

Uma instância dinâmica é aquela que é totalmente simulada dentro do mundo da física e esta variável somente leitura retornará true se a instância sendo verificada for totalmente simulada ou false se não for

Sintaxe:

phy_dynamic;

Devoluções:

(or undefined if the instance is not physics enabled)

Exemplo:

if other.phy_dynamic
{
    with (other)
    {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
    }
}

O código acima cria uma nova instância e depois a define como sendo uma "bala" no mundo da física antes de dar-lhe um impulso maciço ao longo do eixo y.
