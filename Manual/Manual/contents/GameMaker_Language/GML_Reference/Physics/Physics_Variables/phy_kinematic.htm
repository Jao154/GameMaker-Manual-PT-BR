fitocinemático

Esta variável somente leitura retornará true se a instância for classificada como sendo cinemática object, ou false se não for. Uma instância cinemática é aquela que tem massa infinita (uma densidade de 0), mas que pode se mover. Assim, para fazer uma instância cinemática, você primeiro criaria uma instância estática e depois definiria uma ou mais variáveis da instância relacionadas ao movimento (ou seja, uma variável de 0): phy_speed_x, phy_speed_you phy_angular_velocity)

Sintaxe:

phy_kinematic;

Devoluções:

(or undefined if the instance is not physics enabled)

Exemplo:

if !phy_kinematic
{
    phy_speed_x = 5;
}

O código acima verifica se a instância é cinemática e, se não for, define a velocidade horizontal para 5.
