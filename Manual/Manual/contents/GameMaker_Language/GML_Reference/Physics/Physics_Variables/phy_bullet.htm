phy_bullet

Esta variável define se a instância é ou não extremamente rápida (por exemplo, uma bala). O valor padrão é false, mas se ajustado para true, isto diz ao GameMaker que a instância estará se movendo a velocidades tão altas que exigirá uma detecção de colisão mais cara para garantir que não passe por outras instâncias sem ser detectada.

Sintaxe:

phy_bullet;

Devoluções:

(or undefined if the instance is not physics enabled)

Exemplo:

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

O código acima cria uma nova instância e depois a define como sendo uma "bala" no mundo da física antes de dar-lhe um impulso maciço ao longo do eixo y.
