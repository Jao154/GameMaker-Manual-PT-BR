física_aplicar_impulso_angular

Esta função dará um impulso angular a uma instância habilitada para a física. Este impulso definirá a rotação angular pela quantidade dada, ignorando o torque atual, essencialmente definindo a quantidade de "giro" que uma fixação tem. Se você deseja aplicar uma força angular a uma instância usando o torque, então você deve estar usando a função physics_apply_torque().

Sintaxe:

physics_apply_angular_impulse(impulse)

ArgumentoTipo Descrição
impulse The impulse (in Newton metres) to be applied

Devoluções:

Exemplo:

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

O código acima dará à fixação física habilitada um impulso angular se uma tecla for pressionada.
