position_meeting

Com esta função você pode verificar uma posição para uma colisão com outra instância ou com todas as instâncias de uma object. Quando você usa esta função você está verificando um único ponto no room para uma instância ou um object. Esta verificação será feita contra a caixa de delimitação da instância ou contra a máscara da instância se essa instância tiver colisões precisas verificadas. Se você precisar obter a instância única id se o object colidir com você deve usar instance_position(). Esta função também aceita as palavras-chave especiais todas e outras.

Sintaxe:

position_meeting(x, y, obj);

Argumento Tipo Descrição
x Real The x position to check.
y Real The y position to check.
obj Objeto Ativo ou ID da instância The object (or instance id, or the keywords "all" or "other") to check for a collision with.

Devoluções:

Boolean

Exemplo:

if mouse_check_button(mb_left)
{
    if !position_meeting(mouse_x, mouse_y, all)
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
    }
}

O código acima verifica o botão esquerdo do mouse, e se for pressionado, verifica a posição x/y do mouse para uma colisão com qualquer instância. Se não houver nenhuma, então é criada uma instância de "obj_Wall".
