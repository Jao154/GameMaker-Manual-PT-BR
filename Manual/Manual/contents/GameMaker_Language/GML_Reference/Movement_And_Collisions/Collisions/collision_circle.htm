círculo_de_colisão

Collision_circle cria uma zona circular do raio dado pelo usuário em torno dos pontos x1,y1 e, em seguida, verifica essa zona para uma colisão com qualquer instância do object especificada pelo argumento "obj". Esta verificação pode ser precisa ou não, mas para que colisões precisas sejam ativadas, o object ou instância que você está verificando também deve ter colisões precisas ativadas para seu sprite. Caso contrário, a verificação padrão é baseada em caixas delimitadoras. A imagem a seguir ilustra como isto funciona:

Aqui, o exemplo no meio é usar um círculo de colisão para verificar a bola objects. Agora, os azuis não têm uma caixa de delimitação precisa e, como você pode ver, mesmo que o sprite não esteja realmente tocando o círculo, a colisão ainda pode acontecer (mesmo se você definir a opção precisa na função como verdadeira) como a caixa de delimitação daquele sprite sobrelaça a área circular definida pelo círculo de colisão. Por outro lado, as bolas verdes só serão consideradas em colisão se a própria sprite sobrelaçar com o círculo definido. Lembre-se, para que colisões precisas sejam consideradas tanto a object sprite quanto a função de colisão devem ter a marcação precisa como em. Deve-se observar também que o valor de retorno da função pode ser o id de qualquer uma das instâncias consideradas como em colisão.

Note que ao invés de um índice object você pode fornecer uma instância id para verificar uma instância específica, ou as palavras-chave da instância all, ou other (dependendo do evento e do escopo do código atual).

Sintaxe:

collision_circle(x1, y1, rad, obj, prec, notme);

Argumento Tipo Descrição
x1 Real The x coordinate of the center of the circle to check.
y1 Real The y coordinate of the center of the circle to check.
rad Real The radius (distance in pixels from its center to its edge).
obj Objeto Ativo ou ID da instância The object to check for instance collisions.
prec Booleano Whether the check is based on pixel-perfect collisions (true = slow) or its bounding box in general (false = fast).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) ou não (false).

Devoluções:

Instance ID or noone

Exemplo:

if collision_circle(x, y, 20, obj_Cursor, false, true)
{
    image_index = 1;
}
else image_index = 0;

O código acima verificará uma circular com um raio de 20pixel para uma colisão com "obj_Cursor" e, se houver uma, definirá o índice de imagem do object rodando o código para 1, mas se não houver, definirá o índice de imagem do object para 0.
