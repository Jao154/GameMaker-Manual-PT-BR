colisão_elipse

Collision_ellipse usa os primeiros quatro argumentos (x1,y1,x2,y2) para definir a largura e a altura de uma elipse dentro do atual room e depois verifica se algum object que é definido pelo argumento "obj" está em colisão com aquela área. Esta colisão pode ser verificada como precisa ou não, e você também pode optar por verificar, por exemplo, se está ou não executando o código em si. Considere esta imagem:

Aqui, o exemplo no meio é a utilização de uma elipse de colisão para verificar a bola objects. Agora, as azuis não têm uma caixa de delimitação precisa e, como você pode ver, mesmo que a sprite não esteja realmente tocando a elipse, a colisão ainda pode acontecer (mesmo se você definir a opção precisa na função como verdadeira) como a caixa de delimitação daquele sprite sobrelaça a área elíptica definida pelo círculo de colisão. Por outro lado, as bolas verdes só serão consideradas em colisão se a própria sprite sobrelaçar a elipse definida. Lembre-se, para que colisões precisas sejam consideradas tanto a object sprite quanto a função de colisão devem ter a marcação precisa como em. Também deve ser observado que o valor de retorno da função pode ser o id de qualquer uma das instâncias consideradas como em colisão.

Note que ao invés de um índice object você pode fornecer uma instância id para verificar uma instância específica, ou as palavras-chave da instância all, ou other (dependendo do evento e do escopo do código atual).

Sintaxe:

collision_ellipse(x1, y1, x2, y2, obj, prec, notme);

Argumento Tipo Descrição
x1 Real The x coordinate of the left side of the ellipse to check.
y1 Real The y coordinate of the top side of the ellipse to check.
x2 Real The x coordinate of the right side of the ellipse to check.
y2 Real The y coordinate of the bottom side of the ellipse to check.
obj Objeto Ativo ou ID da instância The object to check for instance collisions.
prec Booleano Whether the check is based on precise collisions (trueque é mais lento) ou sua caixa de delimitação em geral (false, mais rápido).
notme Booleano Whether the calling instance, if relevant, should be excluded (true) ou não (false).

Devoluções:

Instance ID or noone

Exemplo:

if collision_ellipse(50, 50, 200, 100, obj_Player, false, true)
{
    instance_create_layer(obj_Player.x, obj_Player.y, "Effects", obj_Splash);
}

Isto verificará uma zona elíptica dentro dos limites de 50x, 50y e 200x, 100y para a object "obj_Player". Se houver uma colisão com o object, então será criada uma instância de "obj_Splash" nas coordenadas x/y do obj_Player.
