move_bounce_all

Com esta função, você pode comandar uma instância para ressaltar todas as instâncias dentro do room, com a única exceção daquelas que não têm sprite ou índice de máscara atribuído a elas. Você também pode dizer a ela para usar a verificação precisa de colisão quando ativada, mas esteja ciente de que isto requer que todas as instâncias tenham máscaras precisas ativadas e irá retardar muito seu jogo quando muitas instâncias estiverem envolvidas devido à quantidade de processamento que tem que ser feita. Isto deve normalmente acontecer no evento de uma instância, mas pode ser usado seletivamente em um evento de colisão também, como ilustrado pelo exemplo de código abaixo.

Sintaxe:

move_bounce_all( adv );

ArgumentoTipo Descrição
adv Whether to enable advanced bouncing (true) or not (false).

Devoluções:

Exemplo:

if other.visible
{
    move_bounce_all(false);
}

O código acima seria colocado em um evento de colisão com outro object. Primeiro verifica se o object é visível e se é, em seguida, executa a ação move_bounce_all(). Note que, neste caso, o ressalto é seletivo e só será calculado para esta colisão, e não para cada instância, cada passo.
