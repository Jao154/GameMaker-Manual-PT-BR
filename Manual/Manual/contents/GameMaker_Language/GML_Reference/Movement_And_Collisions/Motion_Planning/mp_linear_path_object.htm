mp_objeto_de_caminho_linear

Esta função calcula uma linha reta path da posição atual da instância até o ponto especificado pelos valores "xgoal" e "ygoal". Ela utiliza o tamanho do passo indicado, o mesmo que na função mp_linear_step(). O path indicado já deve existir e será substituído pelo novo path e a função retornará se um path completo for encontrado (verdadeiro) ou não (falso). Um path completo só é encontrado se não houve colisão com o object ou instância especificada e se falso for retornado, então um path ainda é gerado, mas só irá até a posição onde o path foi bloqueado.

NOTA: Esta função não move a instância. Ela define apenas um path, e você deve usar as funções de Caminho para se movimentar.

Sintaxe:

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

Argumento Tipo Descrição
path Ativo do caminho The index of the path to be used
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Objeto Ativo ou ID da instância The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

Devoluções:

Boolean

Exemplo:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

O código acima verifica uma colisão com "obj_Wall" ao longo do path entre o object rodando o código e a posição x/y do mouse. Se não houver colisão e o path completo for gerado, então ele iniciará o object ao longo do path gerado.
