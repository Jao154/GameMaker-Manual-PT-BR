mp_potencial_objecto_de_passo

Esta função permite à instância dar um passo em direção a uma determinada posição definida por xgoal/ygoal, sempre tentando evitar obstáculos. Quando a instância se deparar com uma instância do object especificada pelo argumento "obj", ela mudará a direção do movimento para tentar evitar aquela instância e se mover em torno dela. Esta abordagem não é garantia de funcionar, mas na maioria dos casos fáceis, ela efetivamente moverá a instância em direção ao objetivo. A função retorna se o objetivo foi alcançado ou não.

Sintaxe:

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

Argumento Tipo Descrição
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Objeto Ativo ou ID da instância The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

Devoluções:

Boolean

Exemplo:

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

O código acima verifica primeiro se há alguma instância de "obj_Enemy" no room. Se houver, então encontra o mais próximo e armazena sua identificação em uma variável. Esta variável é então usada para dizer a mp_potential_step_object para mover a instância com o código para a posição x/y do object que foi encontrado a uma velocidade de 5 pixels por passo, evitando apenas instâncias da "obj_Wall" do object.
