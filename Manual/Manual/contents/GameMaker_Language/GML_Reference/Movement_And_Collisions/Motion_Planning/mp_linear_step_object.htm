mp_linear_objeto_de_passo

Com esta função, você diz a uma instância para dar um "passo" em direção a um ponto específico, especificado pelos valores xgoal e ygoal. O tamanho da etapa (que é quantos pixels a instância deve mover cada etapa) é indicado pelo tamanho da etapa, e se a instância já estiver na posição ela parará e não se moverá mais, ao contrário de outras funções mais simples, como move_towards_point(). O tamanho das etapas é também a distância "à frente" que o object verificará cada etapa para uma colisão, e você pode escolher se a instância pára em uma colisão com um object ou instância de sua escolha.

Sintaxe:

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

Argumento Tipo Descrição
xgoal Real The target x position.
ygoal Real The target y position.
stepsize Real The speed the instance moves in pixels per step.
obj Objeto Ativo ou ID da instância The object that is to block the path. Can be an object index, an instance id or the special keyword, all

Devoluções:

Boolean

Exemplo:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

O código acima fará com que o object se mova em direção ao mouse a uma velocidade de 5 pixels por passo, verificando apenas se há colisões com o object "obj_Wall". Uma vez alcançada a posição do mouse, criará um object "obj_Explosion" e se destruirá.
