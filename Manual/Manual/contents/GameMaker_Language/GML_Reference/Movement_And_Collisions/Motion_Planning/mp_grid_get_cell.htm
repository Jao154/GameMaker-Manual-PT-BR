mp_grid_get_cell

Com esta função você pode verificar qualquer célula da mp_grid para ver se ela foi marcada como ocupada ou não, dando o índice da mp_grid (previamente criada) e as coordenadas x e y da célula para verificar.

Se tiver sido ocupada ou a posição sendo verificada estiver fora dos limites da grade, a função retornará -1, caso contrário retornará 0.

Sintaxe:

mp_grid_get_cell(id, x , y);

Argumento Tipo Descrição
id ID da grade MP Index of the mp_grid that is to be used
x1 Real The x coordinate of the grid to check.
y1 Real The y coordinate of the grid to check.

Devoluções:

Real

Exemplo:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

O código acima verificará a célula mp_grid que corresponde à posição do mouse e, se estiver ocupada, define a variável image_blend para vermelho, e se não estiver ocupada, define para verde.
