file_text_eof

Esta função retorna true quando o final de um determinado arquivo de texto aberto é alcançado ou false se não for alcançado.

Sintaxe:

file_text_eof(fileid);

ArgumentoTipo Descrição
fileid The id of the file to check.

Devoluções:

Exemplo:

var num = 0;
var file = file_text_open_read(working_directory + "Game_Data.txt");
while (!file_text_eof(file))
{
    str[num++] = file_text_readln(file);
}
file_text_close(file);

O código acima abre um arquivo para escrever então loops através das linhas de texto já escritas no arquivo até chegar ao final, armazenando cada linha na matriz "str".
