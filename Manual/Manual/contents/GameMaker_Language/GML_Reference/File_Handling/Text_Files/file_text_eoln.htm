file_text_eoln

Com esta função você pode fazer com que o GameMaker verifique o arquivo atualmente aberto para ver se a linha que está sendo lida terminou. A função retorna true se o final da linha tiver sido alcançado e false caso contrário.

Sintaxe:

file_text_eoln(fileid);

ArgumentoTipo Descrição
fileid The id of the file to check.

Devoluções:

Exemplo:

var file = file_text_open_read(working_directory + "Game_Data.txt");
var num = 0; while (!file_text_eoln(file))
{
    score_array[num] = file_text_read_real(file);
    num++;
}
file_text_close(file);

O código acima abre um arquivo para leitura e depois lê os valores de uma única linha em uma matriz.
