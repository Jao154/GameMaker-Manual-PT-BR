file_bin_seek

Esta função move a posição atual de leitura dentro do arquivo para a posição indicada. Você fornece o valor de identificação do arquivo, conforme retornado pela função file_bin_open()e, para anexar um arquivo, mover a posição para a tamanho do arquivo antes de escrever.

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Sintaxe:

file_bin_seek(binfile, pos);

ArgumentoTipo Descrição
binfile The ID of the file to read from.
pos The position in the file to move to.

Devoluções:

Exemplo:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

Isto abriria um arquivo do mesmo diretório do jogo, e atribuiria seu índice à variável "arquivo", então obteria o tamanho do arquivo e definiria a próxima posição de escrita para esse tamanho.
