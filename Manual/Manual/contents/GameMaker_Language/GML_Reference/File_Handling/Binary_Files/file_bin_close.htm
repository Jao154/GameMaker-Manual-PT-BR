file_bin_close

Fecha um arquivo binário previamente aberto. Você fornece o valor de identificação do arquivo, conforme retornado pela função file_bin_open().

NOTA: Estas funções não funcionam quando o módulo alvo é HTML5.

Sintaxe:

file_bin_close(binfile);

ArgumentoTipo Descrição
binfile The ID of the file to close.

Devoluções:

Exemplo:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Isto abriria um arquivo do mesmo diretório do jogo, e atribuiria seu índice à variável "arquivo". Em seguida, ele reescreveria o arquivo (limpando-o de dados), e o fecharia novamente.
