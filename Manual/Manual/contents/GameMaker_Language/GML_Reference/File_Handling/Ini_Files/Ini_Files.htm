Arquivos de iniciativas

Os arquivos INI são arquivos pequenos, leves e compatíveis com a maioria das plataformas. São ideais para armazenar pequenas peças de informação, como preferências de interface, pontuações locais altas, dados de nível etc... e como tal, são uma ótima e fácil maneira de você começar a guardar informações de seus jogos. Se você estiver procurando por formas mais avançadas de salvar informações em arquivos, você pode querer ver algumas das outras funções de arquivos como Arquivos de Texto, Arquivos Binários e Buffers.

IMPORTANTE! Todos os projetos GameMaker têm um arquivo chamado "options.ini", então você não pode nomear nenhum arquivo com esse nome (você terá uma falha no uid quando tentar compilar seu projeto de jogo).

Observe que para jogos com localizações é muito importante que as letras acentuadas possam ser lidas a partir de arquivos externos. Isto significa que você deve criar primeiro o arquivo ini no formato UTF8 e depois adicioná-lo ao GameMaker como um arquivo incluído para que ele seja exportado durante a execução do jogo e usado no lugar do arquivo ini no formato padrão ANSI que é criado pelas funções ini quando nenhum foi fornecido anteriormente. Desta forma, você pode ler e escrever corretamente com todos os sotaques e letras não romanas sendo mantidos.

As seguintes funções existem para os arquivos ini:

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
