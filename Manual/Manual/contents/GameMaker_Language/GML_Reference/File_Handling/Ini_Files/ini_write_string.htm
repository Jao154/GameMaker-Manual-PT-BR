ini_write_string

Você pode usar esta função para escrever um string (texto) em um arquivo de dados ini. Os arquivos Ini são divididos em seções e, em seguida, cada seção é dividida em pares de chaves - valores. Assim, um arquivo ini típico ficaria algo parecido com isto:

Vale notar que o uso de aspas e caracteres de escape ao escrever ini strings pode causar problemas ao ler os dados posteriormente, pois pode ser prematuramente truncado. Por exemplo, escrever isto será um problema:

ini_write_string("test2", "section", "hello \"Fritz\"");

Sintaxe:

ini_write_string(section, key, value);

Argumento Tipo Descrição
section Cordão The section of the .ini to write to.
key Cordão The key within the relevant section of the .ini to write to.
value Cordão The string to write to the relevant destination.

Devoluções:

N/A

Exemplo:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

O código acima abre um arquivo ini para leitura e escrita, depois escreve o string armazenado na variável global "Name" na seção "Save" com a chave "Player" antes de fechar o arquivo novamente.
