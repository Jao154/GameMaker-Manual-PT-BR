ini_read_real

Você pode usar esta função para ler um número a partir de um arquivo de dados ini. Os arquivos Ini são divididos em seções e, em seguida, cada seção é dividida em pares de chaves - valores. Assim, um arquivo ini típico se pareceria com algo assim:

Sintaxe:

ini_read_real(section, key, default);

ArgumentoTipo Descrição
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

Devoluções:

Exemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

Isto abrirá "savedata.ini" e definirá a pontuação para o valor sob "save1" > "score" nele, depois fechará o .ini novamente. Se não houver valor em "save1" > "score", ou se não houver "savedata.ini". arquivo presente, a pontuação será ajustada para 0.
