diretório_criar

Esta função criará um diretório com o nome dado na área de salvamento.

NOTA: Esta função não funcionará em HTML5, pois não é possível criar ou remover diretórios no armazenamento local do navegador.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

directory_create(dname)

ArgumentoTipo Descrição
dname The name of the directory to create.

Devoluções:

Exemplo:

if !directory_exists("Games")
{
    directory_create("Games");
}

Isto verificará se o diretório especificado existe na pasta de dados local e, se não existir, o criará para você.
