diretorio_de_trabalho

Working_directory pode realmente retornar dois valores diferentes, dependendo para o que você está usando-o.

Se você estiver escrevendo um arquivo em disco, working_directory aponta para a área de armazenamento local que é reservada para seu jogo no dispositivo alvo (em Windows isto é "%LOCALAPPDATA%/gamedir/" onde o "gamedir" é o diretório com o nome de seu jogo).

Entretanto, ao ler a partir do disco, working_directory pode ser ou o armazenamento local (game_save_id) ou a área onde os Arquivos Incluídos são armazenados.

NOTA A variável working_directory retornará o path incluindo a contrabarra final.

Por exemplo, se você tiver um arquivo padrão *.txt incluído com seu jogo e lê-lo na memória usando working_directory, então esse arquivo será retirado da área onde os Arquivos Incluídos estão armazenados. Se você então optar por escrever essa informação em um arquivo de texto usando working_directory, isto irá escrever o arquivo no armazenamento local da plataforma alvo, onde ele será lido da próxima vez a menos que você instrua o GameMaker a excluir o arquivo, neste caso working_directory irá mais uma vez apontar para a área onde os Arquivos Incluídos estão armazenados.

Vale ressaltar que , em geral, você não precisa usar isto e simplesmente especificar o próprio nome do arquivo (mais qualquer informação adicional path ), o que é suficiente. Por exemplo, isto:

file_text_open_read(working_directory + "my_file.txt");

seria melhor fazer assim:

file_text_open_read("my_file.txt");

AVISO Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

working_directory

Devoluções:

String

Exemplo:

ini_open(working_directory + "temp_ini.ini");

Isto abrirá um arquivo ini do diretório de trabalho do jogo (criando-o se ele ainda não existir). Este pode ser o armazenamento local, ou a área onde os arquivos incluídos estão, dependendo se o arquivo que está sendo procurado existe em qualquer um dos lugares.
