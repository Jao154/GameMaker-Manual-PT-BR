nome do arquivo_ext

Esta função retorna a parte extension do nome do arquivo indicado, incluindo o ponto principal.

Sintaxe:

filename_ext(fname);

ArgumentoTipo Descrição
fname The file to use.

Devoluções:

Exemplo:

ext = filename_ext(file_find_first("*.*", 0));

O código acima recebe o extension (como um string) do arquivo especificado.
