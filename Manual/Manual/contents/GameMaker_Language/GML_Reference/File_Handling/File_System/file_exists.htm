file_exists

Esta função retornará true se o arquivo especificado existir e false se não existir. Note que a função só pode ser usada para verificar arquivos locais, mas não quaisquer arquivos armazenados em um arquivo remoto server.

Sintaxe:

file_exists(fname);

ArgumentoTipo Descrição
fname The name of the file to check for.

Devoluções:

Exemplo:

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

Isto verificaria a existência de um arquivo e, se existir, ele está aberto para leitura.
