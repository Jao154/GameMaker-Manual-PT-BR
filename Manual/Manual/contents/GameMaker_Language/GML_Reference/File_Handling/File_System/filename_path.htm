filename_path

Usando esta função, retorna a parte path do arquivo indicado path, incluindo a contrabarra final.

Sintaxe:

filename_path(fname);

ArgumentoTipo Descrição
fname The file name and path to use.

Devoluções:

Exemplo:

path = filename_path(working_directory + "Test.ini");

O código acima recebe o path (como um string) do arquivo especificado.
