filename_dir

Esta função retorna a parte do diretório do nome do arquivo indicado, que normalmente é o mesmo que o path, exceto pela contrabarra final.

Sintaxe:

filename_dir(fname);

ArgumentoTipo Descrição
fname The file to use.

Devoluções:

Exemplo:

dir = filename_dir("Test.ini");

O código acima recebe o diretório (como um string) do arquivo especificado.
