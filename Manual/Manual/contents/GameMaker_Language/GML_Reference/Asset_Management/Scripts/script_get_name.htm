nome_get_name_do_script

Esta função devolverá o nome como uma seqüência do especificado script. Este nome é o que foi especificado para o script no Asset Browser da janela principal do GameMaker. Para mais informações sobre scripts, consulte o editor Script .

Sintaxe:

script_get_name(scr);

Argumento Tipo Descrição
scr Ativo Script The index of the script that you want to get the name of.

Devoluções:

String

Exemplo:

scr_name = script_get_name(Help_File);

O código do exemplo acima armazenará o nome do índice script indicado na variável "scr_name".
