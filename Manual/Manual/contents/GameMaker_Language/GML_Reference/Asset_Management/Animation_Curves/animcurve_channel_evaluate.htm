animcurve_channel_evaluate

Esta função pode ser usada para obter o valor em um ponto específico no tempo a partir de uma estrutura de canal. Você fornece o ponteiro struct para o canal (como retornado pela função animcurve_get_channel()ou como retornado no animation curve struct da função animcurve_get()) e a posição "x" (tempo) ao longo do canal da curva para avaliar. Esta posição deve estar entre 0 e 1, e a função retornará o valor da curva naquela posição, ou retornará 0 se o canal struct fornecido for inválido.

Sintaxe:

animcurve_channel_evaluate(channel_struct, posx);

Argumento Tipo Descrição
channel_struct Curva de Animação Estruturas de Canal The struct pointer for the channel to evaluate.
posx Real The position in time to check (from 0 to 1).

Devoluções:

Real

Exemplo:

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

O código acima recebe o canal struct para o animation curve asset "ac_AlphaCurve". Ele então usa o valor de avaliação retornado para definir o alfa da imagem para a instância.
