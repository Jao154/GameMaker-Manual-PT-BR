animcurve_create

Esta função criará uma estrutura vazia animation curve, pronta para você preencher com os dados do canal. A função retornará um ponteiro ao struct e este será então usado para adicionar canais e outros dados ao animation curve. Você também pode fornecer isto a funções como animcurve_get() para mais tarde obter os dados da curva.

Quando você usar esta função, o struct criado não terá dados nele, e para usá-lo você deve adicionar pelo menos um canal e o canal deve ter pontos adicionados a ele. Para adicionar um canal, veja a função animcurve_channel_new()e para adicionar pontos ao canal, veja a função animcurve_point_new(). Além disso, você pode dar um nome ao animation curve struct definindo a variável "name" (como mostrado no código do exemplo, abaixo). Os canais adicionados a um novo animation curve devem ser adicionados como um array, onde cada item do array faz referência a um canal struct, e os pontos em um canal também devem ser adicionados como um array, onde cada item do array faz referência a um ponto struct.

Note que animation curves criado desta forma deve ser excluído quando não for mais necessário, pois de outra forma eles ocuparão espaço na memória, o que pode levar a um vazamento de memória, diminuindo a velocidade e, eventualmente, quebrando seu jogo. Você pode remover um animation curve quando não for mais necessário, usando a função animcurve_destroy(). Não é necessário limpar nenhum canal ou apontar dados dentro da curva, pois isso será feito pelo coletor de lixo automaticamente quando a própria curva for destruída.

Sintaxe:

animcurve_create();

Devoluções:

Animation Curve Struct

Exemplo:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

O código acima cria um novo animation curve struct e o armazena na variável "my_curve". Este struct é então preenchido com um nome e uma matriz de canais. A matriz de canais contém um único canal com três pontos.
