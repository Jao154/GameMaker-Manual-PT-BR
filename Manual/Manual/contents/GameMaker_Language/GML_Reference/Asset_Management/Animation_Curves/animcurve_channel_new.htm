animcurve_channel_new

Esta função pode ser usada para criar uma nova estrutura de canais animation curve. Um canal struct tem as seguintes variáveis:

Curva de Animação Estruturas de Canal
Nome da variável Tipo de dados Descrição
nome corda Este é o nome do canal.
tipo Interpolação da Curva de Animação Tipo Constante Esta será uma das constantes dadas na tabela abaixo.
iterações inteiro Se o canal estiver usando interpolação catmull-rom ("suave"), isso detém quantos pontos foram gerados para cada segmento da curva (observe que esses pontos extras são internos à função e usados apenas para os cálculos runtime ). Se o canal estiver usando interpolação linear, este valor ainda existirá, mas pode ser ignorado, pois não tem influência sobre como a curva é interpolada.
pontos matriz Esta é uma matriz, onde cada item da matriz é um Estruturas de pontos curvos de animação.

Interpolação da Curva de Animação Tipo Constante
Constante Descrição
animcurvetype_linear Utilizado para interpolação linear entre pontos.
animcurvetype_catmullrom Utilizado para interpolação suave entre pontos utilizando interpolação catmull-rom.
animcurvetype_bezier Utilizado para a interpolação de Bezier entre pontos.


Por padrão quando você cria um novo canal struct, a variável "nome" será uma variável vazia string, o "tipo" será animcurvetype_linear e as "iterações" serão 16. Todas estas variáveis podem ser definidas para os valores que você desejar (note que o valor "iterações" não tem efeito sobre os tipos de curvas lineares). Uma vez criada, você precisa adicionar pontos à curva, o que é feito adicionando diferentes pontos structs à matriz de"pontos". Estes pontos structs podem ser criados usando a função animcurve_point_new().

Sintaxe:

animcurve_channel_new();

Devoluções:

Animation Curve Channel Struct

Exemplo:

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

O código acima cria um novo animation curve struct e o armazena na variável "my_curve". Este struct é então preenchido com um nome e uma matriz de canais. A matriz de canais contém um único canal com três pontos.
