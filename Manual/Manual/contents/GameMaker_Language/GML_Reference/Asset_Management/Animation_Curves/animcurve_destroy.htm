animcurve_destroy

Esta função pode ser usada para destruir qualquer animation curves que tenha sido previamente criada com a função animcurve_create(). Chamando esta função removerá o animation curve da memória e limpará quaisquer canais ou pontos que ele contenha também (estes nunca precisam ser limpos manualmente). Note que você não pode destruir nenhum animation curves criado no navegador asset , somente aqueles criados em runtime.

Sintaxe:

animcurve_destroy(curve_struct);

Argumento Tipo Descrição
curve_struct Estruturas Curvas de Animação The pointer to the curve struct to destroy.

Devoluções:

N/A

Exemplo:

animcurve_destroy(my_curve);

O código acima destruirá o (anteriormente criado) animation curve struct indexado na variável "my_curve".
