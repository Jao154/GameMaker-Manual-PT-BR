Ativos e etiquetas

As funções listadas aqui são especificamente projetadas para obter informações de qualquer um dos assets definidos no Navegador de Ativos, assim como as etiquetas que lhes foram atribuídas. As tags são uma ferramenta muito poderosa, tanto no Navegador Asset quanto em seu código de jogo real, pois podem ser lidas e criadas em tempo de execução, dando-lhe maneiras adicionais de filtrar assets quando seu jogo estiver rodando e criar verificações condicionais ou sistemas do tipo "pai" fora do sistema baseado em objetos.

Para assets em geral, temos as seguintes funções:

  • asset_get_index
  • asset_get_type

Para lidar com o sistema de "tags" do navegador Asset, nós temos as seguintes funções:

  • tag_get_asset_ids
  • tag_get_assets
  • asset_get_tags
  • asset_add_tags
  • asset_remove_tags
  • asset_has_tags
  • asset_has_any_tag
  • asset_clear_tags
