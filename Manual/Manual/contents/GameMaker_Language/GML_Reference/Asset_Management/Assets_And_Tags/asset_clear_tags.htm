marcas_de_activo

Com esta função você pode limpar todas as tags presentes em qualquer asset a partir do navegador asset. Você fornece ou o nome asset (como string) ou seu índice asset, e se você fornecer um valor de índice asset, então você precisará fornecer o argumento opcional do tipo asset (uma constante), pois assets de diferentes tipos pode ter o mesmo índice, mesmo que eles não possam ter o mesmo nome. Os tipos disponíveis asset estão listados na tabela abaixo:

Constante Descrição
asset_object O nome dado refere-se a um objeto.
asset_sprite O nome dado refere-se a um duende.
asset_sound O nome dado refere-se a um som.
asset_room O nome dado refere-se a um room.
asset_tiles O nome dado refere-se a um conjunto de azulejos.
asset_path O nome dado refere-se a um caminho.
asset_script O nome dado refere-se a um script.
asset_font O nome dado refere-se a uma fonte.
asset_timeline O nome dado refere-se a uma linha de tempo.
asset_shader O nome dado refere-se a um sombreador.
asset_animationcurve O nome dado refere-se a uma Curva de Animação.
asset_sequence O nome dado refere-se a uma sequência.

Se a função conseguir limpar as etiquetas, ela retornará true caso contrário, retornará false.

Sintaxe:

asset_clear_tags(name,[asset_type]);

Argumento Tipo Descrição
name_or_index Cordão ou Ativo The name of the asset (a string) or its index value (an integer).
[asset_type] Tipo de ativo Constante OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

Devoluções:

Boolean

Exemplo:

asset_clear_tags(obj_Enemy_Parent, asset_object);

O código acima limpará todas as tags do dado object asset.
