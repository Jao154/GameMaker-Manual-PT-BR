ativo_ tem_qualquer_e_etiqueta

Com esta função você pode verificar se uma ou mais tags strings está atribuída a qualquer asset a partir do navegador asset. Você fornece ou o nome asset (como string) ou seu índice asset, assim como uma única tag string ou uma matriz onde cada item é uma única tag string. Se você fornecer um valor de índice asset, então você precisará fornecer o argumento opcional do tipo asset (uma constante), pois assets de diferentes tipos pode ter o mesmo índice, mesmo que eles não possam ter o mesmo nome. Os tipos disponíveis asset estão listados na tabela abaixo:

Constante Descrição
asset_object O nome dado refere-se a um objeto.
asset_sprite O nome dado refere-se a um duende.
asset_sound O nome dado refere-se a um som.
asset_room O nome dado refere-se a um room.
asset_tiles O nome dado refere-se a um conjunto de azulejos.
asset_path O nome dado refere-se a um caminho.
asset_script O nome dado refere-se a um script.
asset_font O nome dado refere-se a uma fonte.
asset_timeline O nome dado refere-se a uma linha de tempo.
asset_shader O nome dado refere-se a um sombreador.
asset_animationcurve O nome dado refere-se a uma Curva de Animação.
asset_sequence O nome dado refere-se a uma sequência.

Se a função for bem sucedida e uma ou mais das tags estiverem presentes no asset, ele retornará true caso contrário, retornará false. Se você precisar verificar uma correspondência exata com qualquer tag ou conjunto de tags, então use a função asset_has_tags().

Sintaxe:

asset_has_any_tag(name_or_index, tags, [asset_type]);

Argumento Tipo Descrição
name_or_index Cordão ou Ativo The name of the asset (a string) or its index value (an integer).
tags Cordão ou Array de Cordas A single asset tag string or an array with various asset tags.
[asset_type] Tipo de ativo Constante OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

Devoluções:

Boolean

Exemplo:

var _a = array_create(3);
_a[0] = "boss";
_a[1] = "main_boss";
_a[2] = "final_boss";

if asset_has_any_tag(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

O código acima criará um conjunto de tags e então verificará se alguma delas é aplicada ao dado object, e se são, criará uma instância de outra object.
