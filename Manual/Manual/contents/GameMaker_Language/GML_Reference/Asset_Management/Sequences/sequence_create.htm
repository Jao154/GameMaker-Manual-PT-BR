sequence_create

Com esta função você pode criar um novo sequence object ao qual você pode então adicionar faixas. A função retorna uma estrutura que você pode então acessar para configurar o novo sequence que você criou. O conteúdo deste struct está detalhado nesta página. O sequence object struct pode então ser usado para criar instâncias do sequence em uma camada room usando a função layer_sequence_create(). Note que ao criar sequence objects desta forma, você deve removê-los novamente chamando a função sequence_destroy() quando elas não forem mais necessárias, caso contrário você terá um vazamento de memória que pode retardar e, eventualmente, quebrar seu jogo.

Sintaxe:

sequence_create();

Devoluções:

Sequence Object Struct

Exemplo:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

O código acima cria um novo sequence object struct e define seu comprimento de reprodução e o modo loop.
