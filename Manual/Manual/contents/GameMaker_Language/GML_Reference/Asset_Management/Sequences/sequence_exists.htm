Sequência_existe

Com esta função, você pode verificar se existe ou não um sequence object. Você fornece ou o sequence object struct (como retornado pela função sequence_create() ou sequence_get()) ou o ID sequence (como retornado pela função layer_sequence_get_sequence() ou da instância sequence struct propriedade sequence, ou do índice do navegador asset ) e a função retornará true se o sequence object existir ou false se não existir.

Sintaxe:

sequence_exists(sequence_struct_or_id);

Argumento Tipo Descrição
sequence_struct_or_id Sequência Ativo ou Sequência Object Estruturas The sequence to check for, can be the asset reference itself or its Sequence object struct.

Devoluções:

Boolean

Exemplo:

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

O código acima verifica se o dado sequence object existe e se o faz é destruído.
