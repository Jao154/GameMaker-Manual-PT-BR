audio_sync_group_debug

sua função pode ser usada para exibir informações de depuração sobre qualquer grupo de sincronização (o índice do grupo é o valor retornado quando o grupo foi criado usando a função audio_create_sync_group()). Você pode chamar a função com um valor de -1 a switch fora da sobreposição.

Quando você estiver debugging os grupos de sincronização de áudio, a sobreposição aparecerá:

  • Qual grupo de sincronia é o debugging
  • Quantos canais há nele
  • Quantas vezes o grupo de sincronização falhou (isto significa que ele tentou fazer muito trabalho)
  • Uma lista dos canais
    • Vermelho significa que este canal não está fazendo nenhum trabalho no momento
    • Amarelo significa que o canal está no estado inicial (ainda não decodificando ou tocando)
    • Verde significa que a pista está atualmente procurando/descodificando
    • Roxo significa que o canal ficou sem dados
  • Uma barra de perfil que mostra o trabalho que a linha de decodificação está fazendo
    • azul significa que um arquivo está sendo lido pela primeira vez
    • vermelho significa que o fio está lendo um arquivo para descobrir de onde ler
    • verde significa que o fio está decodificando dados
    • roxo significa que o fio está dormindo até precisar trabalhar

Note que para o geral debugging de áudio, você também tem a função audio_debug().

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_sync_group_debug(group_index);

Argumento Tipo Descrição
group_index ID do Grupo de Sincronização de Áudio The group index to debug, or -1 to switch off the debugging.

Devoluções:

N/A

Exemplo:

if (debug_mode)
{
    audio_sync_group_debug(sg);
}
else
{
    audio_sync_group_debug(-1);
}

O código acima irá switch ligado ou desligado da sobreposição do grupo de sincronização de depuração de áudio mostrando o grupo de sincronização indexado na variável "sg" dependendo se o jogo é executado em modo de depuração ou não.
