audio_sync_group_get_track_pos

Esta função retorna a posição atual de jogo do grupo de sincronização em questão. O índice do grupo é o valor retornado quando você criou o grupo usando a função audio_create_sync_group()e o valor de retorno é o tempo em segundos que as faixas têm sido tocadas.

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_sync_group_get_track_pos(group_index);

Argumento Tipo Descrição
group_index ID do Grupo de Sincronização de Áudio The group index to get the position of.

Devoluções:

Real

Exemplo:

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

O código acima obtém a posição de tempo para o grupo de sincronização indexado na variável "sg", então usa este valor para desenhar o tempo reproduzido na tela.
