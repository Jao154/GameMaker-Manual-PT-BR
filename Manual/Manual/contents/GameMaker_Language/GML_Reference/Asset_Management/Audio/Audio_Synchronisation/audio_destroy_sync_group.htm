audio_destroy_sync_group

Os grupos de sincronização de áudio precisam ser destruídos quando não estão sendo utilizados para liberar a memória e os recursos sonoros associados a eles usando esta função. Ele toma o índice do grupo como retornado quando o grupo foi criado usando a função audio_create_sync_group()e libera todos os recursos utilizados pelo grupo.

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_destroy_sync_group(group_index);

Argumento Tipo Descrição
group_index ID do Grupo de Sincronização de Áudio The group index to be destroyed.

Devoluções:

N/A

Exemplo:

audio_destroy_sync_group(sg);

O código acima destrói o grupo de sincronização indexado na variável "sg", e provavelmente seria usado nos eventos destroy ou Room End.
