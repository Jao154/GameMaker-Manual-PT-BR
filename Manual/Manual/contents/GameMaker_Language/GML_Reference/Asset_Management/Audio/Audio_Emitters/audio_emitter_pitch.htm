audio_emitter_pitch

Esta função pode ser usada para alterar o tom de todos os sons emitidos pelo emissor dado. É um multiplicador de tom, na medida em que o valor de entrada multiplica o tom atual por aquela quantidade, de modo que o valor padrão de 1 não é alteração do tom, enquanto que um valor menor que 1 diminuirá o tom e maior que 1 aumentará o tom. É melhor usar pequenos incrementos para esta função, pois qualquer valor abaixo de 0 ou acima de 5 pode não ser audível de qualquer forma.

Os sons já tocados no emissor de áudio não precisam ser reiniciados para que a mudança no tom seja audível. A mudança no tom de voz é aplicada imediatamente.

Sintaxe:

audio_emitter_pitch(emitter, pitch);

Argumento Tipo Descrição
emitter ID do emissor de áudio The index of the emitter to change.
pitch Real The pitch multiplier (default 1).

Devoluções:

N/A

Exemplo:

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

O código acima mudará o tom do áudio reproduzido do emissor indexado na variável "s_emit", com base no valor da variável "gear".
