audio_emitter_existe

Esta função retorna se um emissor de áudio existe (true) ou não (false). Note que se o índice que você procura não foi inicializado anteriormente, esta função causará um erro, pois está procurando por índices inexistentes.

Sintaxe:

audio_emitter_exists(index);

Argumento Tipo Descrição
index ID do emissor de áudio The index of the emitter to check the existence of.

Devoluções:

Boolean

Exemplo:

if audio_emitter_exists(s_emit)
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

O código acima verifica se existe um emissor, indexado na variável (previamente rubricada) "s_emit". Se ele faz então um som é tocado através dele, mas se não faz, ele é criado e então o som é tocado.
