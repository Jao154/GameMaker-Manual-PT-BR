audio_emitter_velocity

Esta função pode ser usada para dar um efeito Doppler emissor e simular o movimento de áudio baseado no vetor que é resolvido a partir das posições relativas x, y e z. Se o emissor em si não vai se mover, você normalmente não precisaria definir estes valores, mas, por exemplo, se você estiver fazendo um jogo de tiro rolante onde os inimigos vêm da direita e rolam para a esquerda, você definiria isto para ter uma velocidade de (hspeed, 0, 0) no evento de criação dos inimigos (e atualizaria o emissor de cada instância no evento de passo usando audio_emitter_position()) para dar a qualquer som emitido pelas instâncias inimigas o Doppler correto ao passar pela instância do jogador (que, por sua vez, estaria usando a função audio_listener_position() para colocar o ouvinte na posição correta).

Sintaxe:

audio_emitter_velocity(emitter, vx, vy, vz);

Argumento Tipo Descrição
emitter ID do emissor de áudio The index of the emitter to change.
vx Real The x vector value (default 0).
vy Real The y vector value (default 0).
vz Real The z vector value (default 0).

Devoluções:

N/A

Exemplo:

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, room_width, 0, 0);
audio_emitter_velocity(s_emit, -5, 0, 0);

O código acima cria um emissor de áudio e atribui seu índice à variável "s_emit". Este emissor é então colocado em uma posição dentro do room e recebe uma velocidade de 5 pixels por degrau ao longo do eixo x (ele fará Doppler corretamente em relação ao ouvinte como se tivesse uma velocidade horizontal de 5 pixels por degrau).
