posição_emissor_de_áudio

Com esta função você pode mudar a posição de um emissor de áudio dentro do espaço de áudio 3D. A posição afetará o som de diferentes maneiras, dependendo de onde o ouvinte estiver posicionado dentro do espaço de áudio também (a posição padrão é (0, 0, 0). Veja audio_listener_position() para maiores informações), assim, por exemplo, se a posição do emissor estiver definida para (100, 0, 0) e o ouvinte atual estiver em (200, 0, 0) o áudio transmitido pelo emissor parecerá estar à esquerda do campo de áudio. A imagem abaixo mostra uma representação visual dos emissores e suas posições relativas para o ouvinte:

Sintaxe:

audio_emitter_position(emitter, x, y, z);

Argumento Tipo Descrição
emitter ID do emissor de áudio The index of the emitter to change.
x Real The x position.
y Real The y position.
z Real The z position.

Devoluções:

N/A

Exemplo:

if speed > 0
{
    audio_emitter_position(s_emit, x, y, 0);
}

O código acima verifica se a velocidade da instância está acima de 0 e se é ela que atualiza o emissor de áudio indexado na variável "s_emit" para a posição x/y atual.
