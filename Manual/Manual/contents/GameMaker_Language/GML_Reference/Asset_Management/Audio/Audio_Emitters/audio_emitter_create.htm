audio_emitter_create

Esta função cria um novo emissor de áudio e retorna o índice para ele. Este índice deve ser armazenado em uma variável para todas as outras funções relacionadas a este emissor, e então quando não for mais necessário, deve ser removido da memória usando a função audio_emitter_free() para evitar vazamentos de memória que podem acabar por travar seu jogo.

O emissor de áudio é criado com as seguintes configurações padrão:

  • Um ganho e uma inclinação de 1
  • Uma máscara de ouvinte definida apenas para o ouvinte padrão (1)
  • Uma posição de (0, 0, 0)
  • Uma velocidade de 0 (vx = 0, vy = 0, vz = 0)

Sintaxe:

audio_emitter_create();

Devoluções:

Audio Emitter ID

Exemplo:

s_emit = audio_emitter_create();

O código acima cria um novo emissor de áudio e atribui seu índice à variável "s_emit".
