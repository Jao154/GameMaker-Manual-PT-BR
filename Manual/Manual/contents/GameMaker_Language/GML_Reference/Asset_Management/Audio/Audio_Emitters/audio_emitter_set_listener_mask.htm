audio_emitter_set_listener_mask

Esta função pode ser usada para definir a máscara de bit para um emissor de modo que todos os sons tocados através do emissor só possam ser ouvidos pelos ouvintes especificados. Você pode criar uma bit-máscara usando a função audio_get_listener_info() e depois usando o bitwise or ("|") para criar uma máscara personalizada com aqueles ouvintes dos quais você requer o som para tocar, e depois aplicar esta máscara personalizada ao emissor. Esta máscara sobreporá a máscara padrão ou aquela que você possa ter definido usando a função audio_set_listener_mask().

Sintaxe:

audio_emitter_set_listener_mask(emitterID, mask);

Argumento Tipo Descrição
emitterID ID do emissor de áudio The unique ID of the emitter to set the mask of
mask Real The bitmask data to set for the sound

Devoluções:

N/A

Exemplo:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

O código acima cria um emissor de áudio e então define o(s) ouvinte(s) para tocar a partir do uso dos dados da máscara armazenados em uma variável global.
