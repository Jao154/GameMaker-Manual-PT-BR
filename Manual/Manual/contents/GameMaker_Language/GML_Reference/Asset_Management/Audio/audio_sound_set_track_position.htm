posição_de_som_som_som_de_audio

Esta função definirá a posição (em segundos) para a identificação sonora dada ou asset.

O "índice" fornecido pode ser ou uma única instância de som (como retornado de audio_play_sound()) ou um som asset (por exemplo, um adicionado através do Asset Browser). O comportamento desta função será diferente dependendo do tipo de índice que você especificou:

  • Se é uma identificação sonora única que você usa, que já está jogando, então sua posição mudará imediatamente para o tempo dado.
  • Se for um som asset, então todas as novas execuções do som dado asset começarão no novo momento.

Esta função só mudará a posição da faixa para o som atualmente tocado, ou sua próxima peça (no caso de ser fornecido um som asset ). Se o som for reproduzido com o loop habilitado, as reproduções subseqüentes sempre começarão do início (0,0 segundos), e não da posição da faixa definida usando esta função. Por exemplo, iniciar um som loop em 5,7 segundos tocará o primeiro som a partir desse ponto, porém depois disso continuará a repetir a faixa inteira, do início ao fim.

Sintaxe:

audio_sound_set_track_position(index, time);

Argumento Tipo Descrição
index Bens sólidos ou identificação de instância sonora The index of the sound to change.
time Real The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

Devoluções:

N/A

Exemplo:

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

O código acima toca um som e depois usa o valor de identificação retornado para definir a posição inicial para o som para 32 segundos.
