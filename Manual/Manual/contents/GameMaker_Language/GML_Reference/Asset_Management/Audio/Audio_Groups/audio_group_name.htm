nome_do_grupo_de_áudio

Esta função retornará um string contendo o nome do grupo de áudio dado para exibição ou verificação. Quando você define um grupo de áudio nas Opções de Jogo, você lhe dá um "nome" único que é realmente uma constante a ser usada como um valor deidentificação para o grupo. Tudo o que esta função faz é pegar a ID e retornar um string do nome da ID que você deu.

Sintaxe:

audio_group_name(groupID);

Argumento Tipo Descrição
groupID ID do grupo de áudio The index value constant of the audio group to check (as defined in the Audio Groups Window)

Devoluções:

String

Exemplo:

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

O código acima recupera o nome da constante do grupo de áudio dado e o exibe na tela.
