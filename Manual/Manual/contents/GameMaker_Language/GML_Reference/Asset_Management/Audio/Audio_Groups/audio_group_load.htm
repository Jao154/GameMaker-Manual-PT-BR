audio_group_load

Esta função irá carregar todos os sons que são marcados como pertencentes ao grupo de áudio em questão na memória. A função retornará true se o carregamento for iniciado e false se o ID do grupo for inválido, ou se já tiver sido sinalizado para carregamento. A função é assíncrona, portanto seu jogo continuará a rodar enquanto o áudio é carregado em segundo plano. Isto significa que também acionará um evento de Carga/Salva Assíncrona para informar que o grupo inteiro foi carregado na memória e os sons podem agora ser usados.

Sintaxe:

audio_group_load(groupID);

Argumento Tipo Descrição
groupID ID do grupo de áudio The index of the audio group to load (as defined in the Audio Groups Window)

Devoluções:

Boolean

Exemplo:

if !audio_group_is_loaded(audiogroup_level1)
{
    audio_group_load(audiogroup_level1);
}

O código acima verifica se um grupo de áudio foi carregado e se não foi, ele o carrega.
