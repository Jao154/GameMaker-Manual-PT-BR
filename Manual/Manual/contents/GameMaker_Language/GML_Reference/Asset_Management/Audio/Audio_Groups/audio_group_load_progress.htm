audio_group_load_progress

Esta função verificará o progresso de carga de um grupo de áudio e retornará um valor (aproximado) entre 0 e 100.

Sintaxe:

audio_group_load_progress(groupID);

Argumento Tipo Descrição
groupID ID do Grupo de Áudio The index of the audio group to check (as defined in the Audio Groups Window)

Devoluções:

Real

Exemplo:

if load
{
    var pc = audio_group_load_progress(audiogroup_level1);
    draw_text(32, 32, "Loading: " + string(pc));
}

O código acima verifica uma variável e se ela retornar true então algum texto será desenhado mostrando o progresso do áudio que está sendo carregado.
