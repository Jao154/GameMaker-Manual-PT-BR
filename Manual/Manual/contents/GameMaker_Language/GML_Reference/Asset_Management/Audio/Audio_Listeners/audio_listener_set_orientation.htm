audio_listener_set_orientation

Com esta função, você pode mudar a orientação do ouvinte dado dentro do espaço de áudio 3D. O índice padrão do ouvinte é 0, mas você pode usar a função audio_get_listener_info() para obter os diferentes índices disponíveis para a plataforma alvo.

O olhar vetorial e vetor ascendente são baseados nos valores que são resolvidos a partir das posições relativas x, y e z dadas, e o padrão para (0, 0, 1) para o olhar vetorial e (0, 1, 0) para o vetor ascendente, como mostrado na ilustração abaixo:

Mudar a orientação do ouvinte com esta função mudará como o som criado pelos emissores de áudio ao redor do jogo room é percebido pelo jogador de seu jogo. No exemplo abaixo, os sons criados pelo emissor quando o ouvinte está na posição padrão parecem vir de baixo e à direita do ouvinte, mas com a nova posição e orientação do ouvinte, eles agora serão percebidos como vindo de cima e à direita.

Sintaxe:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Argumento Tipo Descrição
index Real ou ID do ouvinte de áudio The listener to set the orientation of.
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Devoluções:

N/A

Exemplo:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

O código acima recupera a matriz de visualização para visualização da câmera [0] e depois a utiliza para definir a posição e orientação do ouvinte de áudio com a identificação armazenada na variável global "Player_Listener".
