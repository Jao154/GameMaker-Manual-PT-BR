audio_listener_orientation

Com esta função você pode mudar a orientação do ouvinte dentro do espaço de áudio 3D. O olhar para a direção e para cima é baseado nos vetores que são resolvidos a partir das posições relativas x, y e z, e padrão para (0, 0, 1) para o olhar para a direção e (0, 1, 0) para a direção para cima, como mostrado na ilustração abaixo:

NOTA se você tem vários ouvintes, você deve usar a função audio_listener_set_orientation().

Mudar a orientação do ouvinte com esta função mudará como o som criado pelos emissores de áudio ao redor do jogo room é percebido pelo jogador de seu jogo. No exemplo abaixo, os sons criados pelo emissor quando o ouvinte está na posição padrão parecem vir de baixo e para a direita do ouvinte, mas com a nova posição e orientação do ouvinte serão agora percebidos como vindo de cima e para a direita.

Sintaxe:

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Argumento Tipo Descrição
lookat_x Real The x look vector (default 0).
lookat_y Real The y look vector (default 0).
lookat_z Real The z look vector (default 1).
up_x Real The x up vector (default 0).
up_y Real The y up vector (default 1).
up_z Real The z up vector (default 0).

Devoluções:

N/A

Exemplo:

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

O código acima define a posição do ouvinte de áudio 3D para as variáveis x e y de uma instância e uma variável z extra. Em seguida, ele define a orientação do ouvinte de áudio para três variáveis calculadas a partir da instância direction e uma variável de direção z extra.
