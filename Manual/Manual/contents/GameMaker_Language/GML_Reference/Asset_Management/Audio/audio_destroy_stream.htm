audio_destroy_stream

Se você criou anteriormente um fluxo de áudio a partir de um arquivo usando a função audio_create_stream() e não precisa mais desse som, você pode chamar essa função para apagá-lo da memória. Qualquer outra chamada para o índice de som após ter sido destruído dará um erro. Deve-se notar que isto liberará o fluxo, mas na plataforma alvo isto pode não aparecer em um gerenciador de memória. Isto porque o GameMaker reúne os recursos de memória para evitar a alocação de memória, e assim a memória permanecerá alocada até ser necessária para algo mais ou reutilizada para um novo fluxo.

NOTA: Esta funcionalidade não está disponível para a plataforma de destino HTML5.

Sintaxe:

audio_destroy_stream(filename);

Argumento Tipo Descrição
filename Cordão Path to the audio file that was streamed.

Devoluções:

N/A

Exemplo:

audio_destroy_stream(snd);

O código acima remove da memória o som indexado na variável "snd".
