audio_existe

Esta função retorna se existe um som (true) ou não (false). Note que se o índice que você procura não foi rubricado anteriormente, esta função causará um erro, pois está procurando por índices inexistentes ( asset ). O som a ser verificado pode ser uma única instância de um som (o índice para sons individuais que estão sendo reproduzidos pode ser armazenado em uma variável ao usar o audio_play_sound() ou audio_play_sound_at() funções) ou um som asset.

Sintaxe:

audio_exists(index);

Argumento Tipo Descrição
index Ativo The index of the sound to check the existence of.

Devoluções:

Boolean

Exemplo:

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

O código acima verifica se existe um som. Se existir, ele será reproduzido em um loop.
