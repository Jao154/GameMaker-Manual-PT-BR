audio_stop_sound

Esta função interromperá o som dado se ele estiver tocando no momento. O som pode ser uma única instância de um som (o índice para sons individuais que estão sendo tocados pode ser armazenado em uma variável quando se usa o audio_play_sound() ou audio_play_sound_at() funções) ou um som asset, em cujo caso todas as instâncias do som dado serão interrompidas.

Sintaxe:

audio_stop_sound(index);

Argumento Tipo Descrição
index Bens sólidos ou identificação de instância sonora The index of the sound to stop.

Devoluções:

N/A

Exemplo:

if !global.SFX
{
    audio_stop_sound(snd_Waterfall);
}
else
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

O código acima verifica a variável global "SFX" e se retornar false, parará o som indexado na variável "snd_Waterfall" que está tocando no momento, e se retornar true, irá loop o som.
