audio_is_playing

Esta função verificará o som dado para ver se ele está tocando no momento. O som pode ser uma única instância de um som (o índice para sons individuais que estão sendo tocados pode ser armazenado em uma variável quando se usa o audio_play_sound() ou audio_play_sound_at() funções) ou um som asset, em cujo caso todas as instâncias do som dado serão verificadas e se alguma delas estiver tocando a função retornará true caso contrário retornará false. Note que esta função ainda retornará true se o som que está sendo verificado tiver sido previamente pausado usando o audio_pause_sound() função.

Sintaxe:

audio_is_playing(index);

Argumento Tipo Descrição
index Bens sólidos ou identificação de instância sonora The index of the sound to check.

Devoluções:

Boolean

Exemplo:

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

O código acima verifica se o som indexado na variável "snd_Waterfall" está tocando atualmente e se ele retornar false então o som será laçado em sua posição room, com uma distância de queda de 300 e uma baixa prioridade.
