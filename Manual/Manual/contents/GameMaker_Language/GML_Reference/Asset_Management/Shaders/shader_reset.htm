shader_reset

Esta função redefine o shader utilizado para desenho e deve ser chamada quando você não quiser mais utilizar o atual shader (conjunto utilizando shader_set()).

Sintaxe:

shader_reset();

Devoluções:

Exemplo:

shader_set(shader_Glass);
draw_self();
shader_reset();

O código acima estabelecerá um shader a ser usado para desenhar, depois desenhe o atual sprite usado para o exemplo usando-o. Finalmente, ele reinicializará o shader para retornar ao padrão do GameMaker shader.
