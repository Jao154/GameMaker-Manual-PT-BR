shader_is_compiled

Esta função verificará um shader em tempo de execução para ter certeza de que foi compilado com sucesso. Se ela tiver então a função retorna true caso contrário, retorna false. Esta função deve ser usada no início do jogo para garantir que a plataforma rodando seu jogo tenha compilado com sucesso qualquer shaders usado (particularmente em Windows onde alguns computadores podem estar usando DX9 com Shader nível 2.0 e não uma versão posterior usando shader nível 3.0).

Se seu shader NÃO tiver sido compilado e você ligar shader_set() o jogo irá cair, então vale a pena fazer algum tipo de verificação sempre que você estiver usando algo que não seja simples GLSL ES shaders.

Sintaxe:

shader_is_compiled(shader);

ArgumentoTipo Descrição
shader The shader to check.

Devoluções:

Boolean

Exemplo:

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

O código acima definirá uma variável global para false, e então se ambos os shaders verificados tiverem compilado corretamente, ele será definido para true.
