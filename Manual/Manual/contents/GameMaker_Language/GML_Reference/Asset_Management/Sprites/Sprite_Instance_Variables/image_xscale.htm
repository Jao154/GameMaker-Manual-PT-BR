image_xscale

Este valor define a escala horizontal aplicada à sprite que foi atribuída à instância atual. Uma escala de 1 indica que não há escala (1:1), valores menores serão escalados para baixo (0,5, por exemplo, será metade da largura do sprite), valores maiores serão escalados para cima e valores negativos serão invertidos para sprite e escalados a menos que o valor utilizado seja exatamente -1 (neste caso o sprite é apenas invertido sobre sua origem sem escala).

Sintaxe:

image_xscale;

Devoluções:

(single precision floating point value)

Exemplo:

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

O código acima escalona o sprite e, uma vez escalado para 5 vezes seu tamanho original, é criada uma nova instância de outra object e a instância é destruída.
