sprite_height

Esta leitura só retorna a altura da variável sprite que foi atribuída à instância. Esta altura é retornada em pixels e será dependente do image_yscale. Se você precisar da altura não escalonada, você deve usar sprite_get_height().

Sintaxe:

sprite_height;

Devoluções:

Real

Exemplo:

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

O código acima verifica a altura do sprite atribuído à instância que executa o código contra a altura do recurso sprite e, se não for o mesmo, repõe o image_yscale para 1,
