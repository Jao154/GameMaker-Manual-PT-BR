image_speed

Esta variável determina a velocidade na qual o GameMaker irá percorrer as sub-imagens para a instância atual sprite. O valor da velocidade dada é um multiplicador, sendo 1 o valor padrão, e definindo-o como 0,5 será metade da velocidade da animação - como definido no Editor Sprite ou Editor de Imagem - enquanto que definindo-o como 2 será o dobro. Se o sprite utilizado não tiver sub-imagens, esta variável não terá efeito.

Sintaxe:

image_speed;

Devoluções:

(single precision floating point value)

Exemplo:

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

O código acima cria uma instância da "obj_Explosion" object e define sua velocidade_de_imagem para 0,5 (efetivamente reduzindo pela metade a velocidade de execução).
