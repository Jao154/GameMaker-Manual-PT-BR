image_angle

Este valor define o ângulo (rotação) do sprite e é medido em graus, com a direita sendo 0º, acima sendo 90º, esquerda sendo 180º e abaixo sendo 270º. Esta variável é ajustada para 0 para reiniciar o sprite a ser desenhado como foi definido no editor sprite. Observe que para que as mudanças nesta variável sejam visíveis, a instância não deve ter nenhum evento de sorteio (e assim, o GameMaker irá, por padrão, sortear o sprite) ou ser desenhada usando uma das funções de desenho estendido como draw_self() ou draw_sprite_ext().

Sintaxe:

image_angle;

Devoluções:

Real

Exemplo:

image_angle = point_direction(x, y, mouse_x, mouse_y);

O código acima irá girar o sprite da instância para sempre apontar para a posição do mouse.
