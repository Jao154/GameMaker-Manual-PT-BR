As Nove Estruturas em Fatias

Uma estrutura de nove fatias pode ser criada usando sprite_nineslice_create()ou recuperada de um sprite usando sprite_get_nineslice(). Este struct conterá propriedades que podem ser alteradas e aplicadas a um sprite usando sprite_set_nineslice().

Se for recuperado de um sprite, a mudança de valores neste struct afetará o original sprite.

As seguintes propriedades estarão disponíveis nesta estrutura:

Estrutura de nove fatias
Variável Tipo Descrição
enabled Booleano Este booleano será verdadeiro se o Nine Slicing estiver habilitado para o sprite, e falso se estiver desabilitado. Você pode modificar isto para ativar ou desativar o Nine Slicing para o sprite.
left Real Este é o desvio para a guia esquerda no eixo x, em relação à borda esquerda do sprite.
right Real Este é o desvio para o guia direito no eixo x, relativo à borda direita do sprite.
top Real Este é o desvio para a guia superior no eixo y, em relação à borda superior do sprite.
bottom Real Este é o desvio para a guia inferior no eixo y, em relação à borda inferior do sprite.
tilemode Array de Constante do Modo Ladrilhos Isto devolve uma matriz contendo os Modos de Ladrilho para a borda e as fatias centrais.

Modos de azulejo

O Modo Tile de cada fatia pode ser lido ou modificado usando estas constantes como índice de array:

Constante de nove fatias
Constante Descrição
nineslice_left A fatia da borda esquerda
nineslice_top A fatia superior da borda
nineslice_right A fatia da borda direita
nineslice_bottom A fatia inferior da borda
nineslice_centre A fatia central

O Modo Tile para uma fatia será (ou pode ser definido para) uma das seguintes constantes:

Constante do Modo Ladrilho
Constante Descrição
nineslice_stretch A fatia será esticada
nineslice_repeat A fatia será repetida
nineslice_mirror A fatia será repetida por espelhamento
nineslice_blank A fatia não será esticada ou repetida, resultando em uma área em branco depois dela
nineslice_hide A fatia não aparecerá de forma alguma

Como o tilemode é um array, você precisa usar a sintaxe do array para mudar o modo Tile de uma fatia. O exemplo de código abaixo define o modo Tile da fatia superior como Repeat:

Exemplo:

var _nineslice = sprite_get_nineslice(spr_box);

_nineslice.tilemode[nineslice_top] = nineslice_repeat;
