sprite_set_set_speed

Esta função pode ser usada para definir a velocidade e o tipo de animação base como para o recurso sprite. Isso normalmente é definido no Editor Sprite, mas pode haver momentos em que seja necessário defini-lo manualmente para um determinado recurso sprite. Você precisa fornecer o índice sprite para definir, a velocidade da animação e o tipo de velocidade de animação a ser aplicada. O tipo de velocidade de animação pode ser qualquer uma das constantes a seguir:

Constante Descrição
spritespeed_framespersecond O sprite foi definido com animação em quadros por segundo.
spritespeed_framespergameframe O sprite foi definido com em animação em quadros por quadro de jogo.


Selecionar o tipo de velocidade a ser usado para calcular as animações é muito importante, pois há uma enorme diferença entre as duas opções. Os dois exemplos a seguir ilustram essa diferença:

  • Se você tem um sprite que desenha 1 quadro por segundo e define a velocidade da imagem para 0,5 quadros por segundo, ele desenhará a 0,5 quadros por segundo. Se sua velocidade de quadros de jogo for 60 quadros por segundo, então o sprite sorteará 1 quadro para cada 120 quadros de jogo.
  • Se você tiver um sprite que desenha 1 quadro por quadro de jogo e definir a velocidade da imagem para 0,5, ele desenhará 0,5 quadros por quadro de jogo. Se sua taxa de quadros por segundo for de 60 quadros, então o sprite sorteará 30 quadros para cada 60 quadros de jogo.

Você pode descobrir qual foi o valor da velocidade utilizada para a animação sprite usando a função sprite_get_speed()e você pode obter o tipo de velocidade de animação usando sprite_get_speed_type().

IMPORTANTE! Esta função afetará o recurso sprite, portanto qualquer mudança que você fizer com esta função afetará todas as instâncias que estiverem usando o sprite e todas as instâncias futuras também.

Sintaxe:

sprite_set_speed(index, speed, type)

Argumento Tipo Descrição
index Ativo Sprite The index of the sprite to set the animation speed of
speed Real The speed value to set
type Constante de velocidade do duende The speed type to use

Devoluções:

Constant (see above)

Exemplo:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

O código acima define a velocidade básica de animação para o sprite atribuído à instância para 10 quadros por segundo.
