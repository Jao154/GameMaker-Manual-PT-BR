sprite_assign

Esta função toma dois índices previamente criados (ou incluídos) sprite, e copia a imagem de um para o outro. Desta forma, você pode copiar (ou "clonar") um sprite em outro índice. Note que você não pode copiar para um recurso do jogo. Você tem que ter criado o sprite para ser copiado anteriormente usando o sprite_add() ou sprite_duplicate() funções.

Sintaxe:

sprite_assign(index, sprite);

ArgumentoTipo Descrição
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

Devoluções

Exemplo:

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

O código acima verifica se a variável global "Player_Sprite" contém uma sprite e se ela usa sprite_assign para mudá-la para uma que tenha sido carregada de um arquivo externo. Se não contiver um sprite, ele é carregado e seu índice é armazenado nessa variável.
