sprite_set_bbox_mode

Esta função pode ser usada para definir o modo caixa delimitadora para um sprite. Você fornece o índice sprite e o modo a ser usado, que deve ser uma das constantes a seguir:

Constante do modo caixa de delimitação
Constante Descrição
bboxmode_automatic Automático - A caixa de delimitação será calculada automaticamente, com base no ajuste de tolerância para o duende
bboxmode_fullimage Full Image - A caixa de delimitação será ajustada para usar toda a largura e altura do sprite, independentemente da tolerância e dos pixels "vazios".
bboxmode_manual Manual - A caixa de delimitação está sendo definida manualmente para valores definidos pelo usuário, o que requer o uso da função sprite_set_bbox()

NOTA Esta função afeta o ativo sprite, de modo que todas as outras instâncias com este sprite terão o mesmo modo de caixa delimitadora.

Sintaxe:

sprite_set_bbox_mode(ind, mode);

Argumento Tipo Descrição
ind Ativo Sprite The index of the sprite to change the mode of.
mode Constante do modo caixa de delimitação The mode to set (a constant).

Devoluções

N/A

Exemplo:

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

O código acima verifica o modo bbox para o atual sprite e se não for automatic, então ele é definido para esse valor.
