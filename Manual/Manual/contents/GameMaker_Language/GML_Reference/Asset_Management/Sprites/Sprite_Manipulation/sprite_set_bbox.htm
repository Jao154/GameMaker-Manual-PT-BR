sprite_set_bbox

Esta função pode ser usada para definir os valores da caixa delimitadora para um sprite. Você fornece o índice sprite para usar e depois os valores esquerdo, superior, direito e inferior para as posições da caixa delimitadora. As posições são valores absolutos, onde a posição (0, 0) corresponde ao canto superior esquerdo do sprite, independentemente do offset para o sprite, quaisquer pixels "vazios" que o sprite possa ter, ou onde ele está sendo desenhado no room.

NOTA Esta função afeta o ativo sprite, de modo que todas as outras instâncias com este sprite terão a mesma caixa de delimitação.

Sintaxe:

sprite_set_bbox(ind, left, top, right, bottom);

ArgumentoTipo Descrição
ind The index of the sprite to set the bounding box on.
left The left side of the bounding box
top The top of the bounding box.
right The right side of the bounding box
bottom The bottom of the bounding box.

Devoluções

Exemplo:

if sprite_get_bbox_mode(sprite_index) == 2
{
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
}

O código acima verificará o modo de caixa delimitadora do sprite atribuído ao sprite_index, e se estiver definido para manual, então terá sua caixa delimitadora alterada.
