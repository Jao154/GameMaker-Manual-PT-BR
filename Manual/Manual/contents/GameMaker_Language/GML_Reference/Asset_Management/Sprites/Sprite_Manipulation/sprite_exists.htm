sprite_existe

Esta função retorna se existe ou não um sprite com o índice especificado no projeto em execução.

Sintaxe:

sprite_exists(index);

ArgumentoTipo Descrição
index The index of the sprite to check.

Devoluções

Exemplo:

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

O código acima verifica um array para ver se contém um índice sprite válido e, em caso afirmativo, atribui esse sprite à instância, mas se não, atribui um sprite a partir dos recursos incluídos.
