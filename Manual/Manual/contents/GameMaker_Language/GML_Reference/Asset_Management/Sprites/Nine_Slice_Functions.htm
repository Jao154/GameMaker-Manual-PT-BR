Nove funções em fatias

Além da seção de nove fatias no editor Sprite, GameMaker fornece funções para definir as propriedades de Nove fatias para sprites em runtime. Antes de ler sobre estas funções, é recomendável ler esta página para obter um entendimento da técnica das Nove Fatias.

As seguintes funções podem ser usadas para recuperar e alterar as propriedades de Nove fatias para um sprite asset em tempo de execução:

  • sprite_nineslice_create
  • sprite_get_nineslice
  • sprite_set_nineslice

Estas funções fazem uso de uma Nove Fatias struct, e você pode ler sobre o conteúdo dessa struct nesta página.

Desenho de Nove Sprites em Fatias

Não há funções separadas para o desenho da Nove Fatias sprites, pois elas funcionam com as funções de desenho de duendes existentes (exceto para funções que desenham apenas uma parte do sprite ou o distorcem, tais como draw_sprite_part() ou draw_sprite_pos()).

Recomenda-se a utilização do draw_sprite_stretched() para desenhar uma Nove Fatias sprite, pois ela permite especificar as dimensões absolutas do sprite em pixels e será fácil de usar junto com a Nove Fatias. Você também pode usar a função draw_sprite_ext() que, em vez disso, assume valores de escala.
