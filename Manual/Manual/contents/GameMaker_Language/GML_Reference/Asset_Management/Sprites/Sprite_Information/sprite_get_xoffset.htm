sprite_get_xoffset

Quando você define um sprite no editor sprite, você precisa definir a origem para isso sprite. Este é o ponto em que o sprite será "anexado" ou "desenhado" quando usado com uma instância. Esta função retorna o offset relativo para o eixo x da origem baseado no canto superior esquerdo sendo a posição 0,0, sendo um +x a direita e um -x a esquerda desta posição. A imagem a seguir ilustra isto:

Sintaxe:

sprite_get_xoffset(index);

ArgumentoTipo Descrição
index The index of the sprite to find the xoffset of.

Devoluções

Exemplo:

if x - sprite_get_xoffset(sprite_index) < 0
{
    x = sprite_get_xoffset(sprite_index);
}

O código acima garantirá que uma instância seja mantida dentro do room com base no sprite (portanto, o sprite é sempre visível).
