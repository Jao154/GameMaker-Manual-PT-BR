sprite_get_width

Com esta função você pode encontrar a largura do recurso base sprite, sem transformações, em pixels.

Sintaxe:

sprite_get_width(index);

ArgumentoTipo Descrição
index The index of the sprite to find the width of.

Devoluções

Exemplo:

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

O código acima verifica a largura do sprite como está na instância atual e se houver uma diferença entre isso e a base original sprite, ele redefine a escala do eixo x.
