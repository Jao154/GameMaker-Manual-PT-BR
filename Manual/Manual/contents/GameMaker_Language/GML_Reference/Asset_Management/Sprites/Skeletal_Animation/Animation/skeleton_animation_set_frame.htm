esqueleto_estrutura_de_animação

Esta função definirá a animação atribuída à faixa dada ao frame que você fornece. O índice de quadros deve ser um valor real entre 0 e o número de quadros para a animação (que você pode obter usando a função skeleton_animation_get_frames). Se você fornecer um valor fora desta faixa, a animação será laçada para fazer a diferença, por exemplo, se a animação tiver 5 quadros e você definir o quadro como 8, o quadro real mostrado será 3.

Sintaxe:

skeleton_animation_set_frame(track, index);

Argumento Tipo Descrição
track Real The animation track to set the frame of.
index Real The new index for the animation frame.

Devoluções:

N/A

Exemplo:

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

O código acima definirá o índice de quadros para a animação na faixa 0 para a velocidade negativa da imagem quando a variável global "pausa" for true (essencialmente pausando a animação).
