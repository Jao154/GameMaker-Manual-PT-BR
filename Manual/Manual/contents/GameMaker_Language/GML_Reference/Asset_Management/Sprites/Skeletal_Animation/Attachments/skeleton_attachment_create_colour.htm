esqueleto_criar_côr_cores

Quando você cria sua animação esquelética sprite, você pode atribuir ranhuras de fixação e anexos para ir nelas. Estas são simplesmente imagens (sprites) que são separadas da animação, mas quando anexadas se moverão junto com o osso ao qual estão anexadas. Normalmente você atribuiria estes anexos em seu programa de animação (Spine), mas com esta função você pode criar seu próprio em tempo de execução usando um sprite asset de seu jogo. a função requer que você dê um nome ao anexo (como um string) e então defina o sprite_index e image_index para usar, assim como a origem x e y (que pode ser diferente daquela definida pelo sprite nas propriedades sprite ), e você também pode definir qualquer transformação que desejar aplicar à imagem quando anexada, assim como a cor a ser misturada com a imagem e o seu valor alfa (transparência). Se o anexo foi criado com sucesso, a função retornará 1 e se não foi (você forneceu um índice sprite inválido, ou a base sprite não é um Spine sprite), então ele retornará -1.

Vale notar que o exemplo sprite pode ter uma cor de mistura e um ajuste alfa (image_blend e image_alpha), assim como a ranhura de fixação (ver a função skeleton_slot_colour_set()), e assim a cor final e alfa que o anexo designado sprite para o slot terá será um composto de todos esses valores.

Sintaxe:

skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, colour, alpha);

Argumento Tipo Descrição
name Cordão The name (as a string) of the attachment to create.
sprite Ativo Sprite The sprite_index to get the attachment image from.
ind Real The image_index to get the attachment image from.
xorigin Real The x origin for the image being used.
yorigin Real The y origin for the image being used.
xscale Real The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the image: 0 = normal, 90 = turned 90° counter-clockwise etc.
Colour Cor The colour value to use (A constant, integer or hex value).
Alpha Real The alpha value to use (from 0 to 1).

Devoluções:

Real (1 if successful, -1 if not)

Exemplo:

skeleton_attachment_create_colour("sword", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1);
skeleton_attachment_create_colour("knife", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1);
skeleton_attachment_create_colour("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

O código acima verificaria o nome do anexo atualmente atribuído para o slot chamado "slot_leftHand" e se um string vazio for devolvido, um novo sprite será anexado.
