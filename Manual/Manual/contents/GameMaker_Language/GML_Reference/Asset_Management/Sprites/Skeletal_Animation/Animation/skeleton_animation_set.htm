conjunto_esqueleto_animação_esqueleto

Uma vez que você tenha atribuído uma animação esquelética sprite a sua instância, você pode então definir qual conjunto de animação deve ser usado a qualquer momento com esta função. Ao criar seu sprite (em Spine), você terá definido vários conjuntos de animações para diferentes estados e nomeado cada um deles apropriadamente. Esses nomes são o que você usará para dizer ao GameMaker qual conjunto de animação deve ser usado a qualquer momento em seu jogo. Por exemplo, se seu sprite é para um jogo de plataforma, então você pode ter um conjunto de "salto", um conjunto de "corrida" e um conjunto de "ociosidade", todos contidos no sprite e você pode switch entre eles em seu jogo usando esta função.
A animação é feita em loop, que é o padrão.

Note que a definição de uma nova animação desta forma irá reinicializar o image_index a 0, iniciando o novo conjunto de animação desde o início de sua animação loop. Se este não for o comportamento desejado, então você pode definir um valor "mix" usando skeleton_animation_mix() e quando você definir uma nova animação, ela será facilmente interpolada com o conjunto anterior. Você também deve certificar-se de ter um keyframe no início de cada animação para evitar que os dados sejam retidos de animações anteriores, a menos que você planeje usar trilhas de animação separadas.

Sintaxe:

skeleton_animation_set(animname, [loop]);

Argumento Tipo Descrição
animname Cordão The name (a string) of the animation set to use.
loop Booleano Se a animação deve ou não ser loop (o padrão é true)

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

O código acima mudará o conjunto de animação sendo usado para o conjunto de "salto" quando a tecla de espaço for pressionada, mas somente se o conjunto atual sendo usado ainda não estiver "pulando".
