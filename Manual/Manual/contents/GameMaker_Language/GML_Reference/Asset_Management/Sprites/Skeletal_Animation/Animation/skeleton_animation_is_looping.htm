esqueleto_animação_é_animação_de_skeleton

Com esta função você pode verificar se a animação atual na faixa indicada está configurada para loop.

Sintaxe:

skeleton_animation_is_looping(track);

Argumento Tipo Descrição
faixa Real O índice da faixa de animação

Devoluções:

Boolean

Exemplo:

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

O código acima verificará se a animação na pista 0 está definida para loop. Se este for o caso, ela mostra uma mensagem de debug.
