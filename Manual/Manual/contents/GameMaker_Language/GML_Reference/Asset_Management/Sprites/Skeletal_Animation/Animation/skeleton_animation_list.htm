lista_esqueleto_animação_esqueleto

Com esta função você pode preencher uma lista DS (pré-criada) com todos os nomes das animações incluídas como parte da animação do esqueleto sprite. Os nomes serão strings e poderão ser usados nas outras funções de animação para estes tipos de sprite.

Sintaxe:

skeleton_animation_list(sprite, list);

Argumento Tipo Descrição
sprite Ativo Sprite The sprite index of the Spine skeletal animation to get the list from.
list DS List ID The ID of the DS list to populate with the animation names.

Devoluções:

N/A

Exemplo:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

O código acima cria uma lista DS de todos os nomes de animação para o sprite que estão sendo usados pela instância. Ele então loops através destes e define o valor da mistura para todos eles com a primeira animação para 0,5.
