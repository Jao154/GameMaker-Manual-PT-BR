draw_skeleton

Esta função é apenas para uso com sprites que foram criados usando um programa de animação esquelética como Spine. Enquanto você pode desenhar estes sprites usando as funções normais de desenho (como draw_self()), há momentos em que você pode querer desenhar um único quadro ou mostrar uma mudança de pele texture sem realmente mudar as propriedades sprite. Nestes casos você pode desenhar o sprite com esta função, onde você dá o índice sprite e depois o nome da animação para obter o quadro (um string, como definido dentro do programa usado para fazer o sprite). Em seguida, você dá o nome da pele para usar (novamente, como definido quando o sprite foi feito), e um valor do frame. O valor do frame é para dizer ao GameMaker qual parte da animação deve mostrar e é um valor entre 0 e o image_number. O resto dos argumentos são os mesmos usados para o desenho normal sprite e mudarão a escala, ângulo, cor da mistura e alfa do quadro de animação final a ser desenhado.

ADVERTÊNCIA! Devido à forma como as animações esqueléticas são interpoladas entre vários quadros "chave", há uma sobrecarga adicional associada a esta chamada e, portanto, é recomendável evitar desenhar o esqueleto baseado em sprites usando este método, a menos que seja absolutamente necessário.

Sintaxe:

draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

Argumento Tipo Descrição
sprite Ativo Sprite The index of the sprite to draw.
animname Cordão The name of the animation to get the frame from (a string).
skinname Cordão The name of the skin to use (a string).
frame Real The animation frame to draw (from 0 to image_number - 1).
x Real The x coordinate of where to draw the sprite.
y Real The y coordinate of where to draw the sprite.
xscale Real The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale Real The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot Real The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour Cor The colour with which to blend the sprite.
alpha Real The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devoluções:

N/A

Exemplo:

draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0.5);

O código acima desenhará o primeiro quadro da animação esquelética sprite, usando o conjunto de animação "salto", a pele definida como "skin1" e na mesma escala e posição que as propriedades da instância definem, mas com um alfa de 0,5.
