Com a animação esquelética sprites, você pode designar separadamente textures (chamada "skins") para a animação, usando assim uma animação para várias coisas diferentes. Essa função retornará o nome da pele (como um string) que é atualmente atribuído à animação esquelética sprite que sua instância está usando. O nome retornado é o que você definiu quando criou o sprite em seu programa de animação.

Sintaxe:

skeleton_skin_get();

Devoluções:

Exemplo:

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

O código acima verificaria a pele atualmente designada para a animação e se for "skin_Enemy1", ele escolhe e define uma nova pele a partir de uma das três opções.
