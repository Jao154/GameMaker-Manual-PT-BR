sólido

Uma instância pode ser assinalada como sólida através das propriedades object no Editor de Objetos, ou alterando o valor desta variável incorporada. Se solid for definido como true, então, quando uma colisão for detectada, a instância em colisão é retornada automaticamente para a posição em que estava no passo anterior à colisão (e então qualquer código ou ações são executadas no evento da colisão). Se for ajustado para false, todo o posicionamento deve ser tratado através do evento de colisão.

Sintaxe:

solid;

Devoluções:

Exemplo:

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

O código acima verificará o sólido flag da "outra" instância em uma colisão e, se estiver configurado para true, moverá essa instância de volta para sua posição anterior.
