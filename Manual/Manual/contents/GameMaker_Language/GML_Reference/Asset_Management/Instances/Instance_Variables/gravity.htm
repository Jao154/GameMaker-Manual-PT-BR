gravidade

gravity é uma das variáveis "embutidas" que todas as instâncias têm e, quando definidas, aplicarão uma força constante no gravity_direction da instância, influenciando tanto a instância speed como direction. Note que a gravidade é uma força cumulativa e irá acelerar o object se você optar por não limitar a velocidade final, e é normal que você defina esta variável para pequenos valores decimais como 0,01. Se você definir o gravity como 0, então nenhuma gravidade será aplicada à instância (este é o valor padrão).

Sintaxe:

gravity;

Devoluções:

(single precision floating point value)

Exemplo:

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

O código acima só aplicará a gravidade se a instância não encontrar nenhuma instância de "obj_Ground" abaixo dela.
