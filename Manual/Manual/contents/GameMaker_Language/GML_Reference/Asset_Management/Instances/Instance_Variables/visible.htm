visível

Uma instância pode ser assinalada como visible ou não definindo esta variável para true (visível) ou false (invisível). Isto funciona dizendo ao GameMaker para pular o evento do sorteio desta instância, portanto, deve-se ter cuidado ao usá-la, pois isso significa que nenhum código colocado no evento do sorteio será executado enquanto a instância não estiver visível. Observe também que se o layer que a instância é designada como invisível, a definição desta variável para true não terá efeito até que a própria camada seja sinalizada como visível também.

Sintaxe:

visible;

Devoluções:

Exemplo:

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

O código acima verificará o visível flag da "outra" instância em um evento de colisão e, se estiver configurado para true, moverá essa instância de volta para sua posição anterior.
