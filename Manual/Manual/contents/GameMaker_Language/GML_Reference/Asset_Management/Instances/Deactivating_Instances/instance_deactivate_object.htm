instance_deactivate_object

Com esta função, você pode desativar uma única instância ou todas as instâncias de uma object específica de todas aquelas que foram ativadas anteriormente. Observe que se você desativou uma instância ou object que foi assinalada como Persistente, então você precisará reativá-lo novamente com a função instance_activate_object() antes de mudar room, caso contrário, não será transportado e será descartado em seu lugar. Note também que a desativação não é instantânea, e uma instância que tenha sido desativada desta forma não será considerada inativa até o final do evento em que a função foi chamada.

OBSERVAÇÃO: Se você desativar uma instância em room iniciar (ou seja: a partir do código de criação room, ou a partir de uma instância criar evento de uma instância dentro do room) todas as instâncias que forem colocadas dentro do room do editor room ainda executarão seu evento de criação antes de serem desativadas.

AVISO: A desativação de instâncias que têm a física habilitada NÃO irá impedir que seus dispositivos interajam dentro da simulação física. Para isso, você deve definir seus phy_active variável para true ou false à medida que você ativa/desativa as instâncias.

Sintaxe:

instance_deactivate_object(obj);

ArgumentoTipo Descrição
obj The object or instance to deactivate (the keyword all can also be used).

Devoluções:

Exemplo:

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

O código acima desativa todas as instâncias do object "obj_Enemy" e depois ativa uma região dentro do room.
