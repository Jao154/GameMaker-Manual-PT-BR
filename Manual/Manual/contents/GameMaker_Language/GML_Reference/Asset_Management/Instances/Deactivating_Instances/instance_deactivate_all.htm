instance_deactivate_all

Esta função desativará todas as instâncias que estão no room no momento em que o código for executado. Isto pode incluir a instância executando o código se o "notme" flag estiver configurado para false, mas normalmente você gostaria que esta instância estivesse ativa, neste caso o "notme" flag deve estar configurado para true. Note que a desativação não é instantânea, e uma instância que tenha sido desativada desta forma não será considerada inativa até o final do evento no qual a função foi chamada.

OBSERVAÇÃO: Se você desativar uma instância em room iniciar (ou seja: a partir do código de criação room, ou a partir de uma instância criar evento de uma instância dentro do room) todas as instâncias que forem colocadas dentro do room do editor room ainda executarão seu evento de criação antes de serem desativadas.

AVISO: A desativação de instâncias que têm a física habilitada NÃO irá impedir que seus dispositivos interajam dentro da simulação física. Para isso, você deve definir seus phy_active variável para true ou false à medida que você ativa/desativa as instâncias.

Sintaxe:

instance_deactivate_all(notme);

Argumento Tipo Descrição
notme Booleano Whether to keep the calling instance activated (true) or not (false).

Devoluções:

N/A

Exemplo:

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

O código acima desativa todas as instâncias, exceto a que está executando o código e depois ativa uma região dentro do room.
