instance_deactivate_layer

Com esta função você pode desativar todas as instâncias atribuídas a uma camada específica. Você precisa fornecer o ID da camada, que pode ser o nome da camada como escrito no editor de código (como um string) ou o valor real do ID da camada como retornado pelo layer_create()e observe que você só pode desativar camadas de instância com esta função. Note que se você desativou uma camada que tem instâncias de objects marcadas como Persistentes, então você precisará reativar a camada novamente com a função instance_activate_layer() antes de mudar room, caso contrário, quaisquer instâncias persistentes na camada não serão transportadas e serão descartadas. Observe também que a desativação não é instantânea, e uma instância que tenha sido desativada desta forma não será considerada inativa até o final do evento em que a função foi chamada.

OBSERVAÇÃO: Se você desativar uma instância em room iniciar (ou seja: a partir do código de criação room, ou a partir de uma instância criar evento de uma instância dentro do room) todas as instâncias que forem colocadas dentro do room do editor room ainda executarão seu evento de criação antes de serem desativadas.

AVISO: A desativação de instâncias que têm a física habilitada NÃO irá impedir que seus dispositivos interajam dentro da simulação física. Para isso, você deve definir seus phy_active variável para true ou false à medida que você ativa/desativa as instâncias.

Sintaxe:

instance_deactivate_layer(obj);

Argumento Tipo Descrição
layer Identificação da camada ou String The layer name string (or ID value) to be used

Devoluções:

N/A

Exemplo:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

O código acima desativa todas as instâncias atribuídas à camada "Enemy_Layer" e depois ativa uma região dentro da room.
