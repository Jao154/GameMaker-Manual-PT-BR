instance_exists

Esta função pode ser usada de duas maneiras, dependendo do que você deseja verificar. Você pode dar-lhe um object_index para verificar, caso em que esta função retornará true se alguma instância ativa do especificado object existir no atual room, ou você também pode fornecer-lhe uma instância id, caso em que esta função retornará true se essa instância específica existir e estiver ativa no atual room.

Observe que esta função não leva em conta as instâncias que foram desativadas usando as funções de desativação da instância.

Sintaxe:

instance_exists(obj);

Argumento Tipo Descrição
obj Identificação da instância ou do bem objeto The object or instance to check for the existence of.

Devoluções:

Boolean

Exemplo:

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

O código acima verifica se existe alguma instância do object "obj_Enemy" e, se não existir, adiciona à variável "score" e muda room.
