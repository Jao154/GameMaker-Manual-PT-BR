instance_find

Todas as instâncias têm um identificador único (id) que pode ser usada para modificá-las e manipulá-las enquanto um jogo está em execução, mas você pode nem sempre saber qual é a identificação para uma instância específica e, portanto, esta função pode ajudar, pois você pode usá-la para iterar através de todas elas para encontrar o que você precisa. Você especifica o object do qual você quer encontrar a instância e um número, e se houver uma instância nessa posição na lista de instâncias então a função retorna o id dessa instância, e se não retornar a palavra-chave especial noone. Você também pode usar a palavra-chave all para iterar todas as instâncias em um room, assim como um pai object para iterar todas as instâncias que fazem parte dessa hierarquia pai/filho, e você pode até mesmo especificar uma instância (se você tiver seu id) como uma verificação para ver se ela realmente existe no atual room. Observe que como as instâncias são classificadas de forma arbitrária, não há uma ordem específica de como as instâncias são verificadas por esta função, e qualquer instância pode estar em qualquer posição.

O valor máximo para "n" nesta função seria

  • Para a palavra-chave all: instance_count - 1
  • Para um índice object: instance_number(OBJ) - 1

Sintaxe:

instance_find(obj, n);

Argumento Tipo Descrição
obj Objeto Ativo The object to find the nth instance of
n Real The number of the instance to find.

Devoluções:

Instance ID or noone

Exemplo:

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

O código acima usará um for loop para iterar através de todas as instâncias do "obj_Enemy" e armazenar sua identificação na matriz "enemy[]".
