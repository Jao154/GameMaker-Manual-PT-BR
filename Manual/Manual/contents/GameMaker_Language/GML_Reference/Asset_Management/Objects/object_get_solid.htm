objeto_get_solid

Esta função lhe dirá se a object que você está verificando foi assinalada como "sólida" ou não. Um object sólido gera um evento de colisão especial ao utilizar o sistema de colisão tradicional (ou seja: o mundo da física está desligado). Observe que esta não é uma função de instância! Portanto, você pode ter um object sólido e uma instância normal do mesmo object e vice-versa. Você pode definir uma instância individual sólida flag usando o solid variável de instância.

Sintaxe:

object_get_solid(obj);

ArgumentoTipo Descrição
obj The index of the object to check.

Devoluções:

Exemplo:

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

O código acima verificará a instância em execução para ver se ela é sólida ou não, assim como verificará o índice object da instância para ver se ela está sinalizada como sólida ou não. Se a instância ainda não for sólida, o índice object será sinalizado como sólido, ele definirá "sólido" como verdadeiro para aquela instância.
