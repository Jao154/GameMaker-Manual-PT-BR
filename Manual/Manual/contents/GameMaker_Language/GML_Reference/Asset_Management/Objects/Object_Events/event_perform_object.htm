event_perform_object

Esta função funciona da mesma forma que event_perform() exceto que desta vez você pode especificar eventos de outro object. Há muitas opções aqui que permitem a simulação completa de todos os eventos possíveis, mas note que isto literalmente apenas executa todo o código no evento e o jogo não modificará nada para fazê-lo acionar manualmente, por exemplo, se você optar por executar um evento de apertar o teclado, o evento será acionado, mas a tecla relevante não será reconhecida como tendo sido pressionada. Ou se você executar um evento de alarme, o contador de alarme não será ajustado para -1, mas sim continuará a contagem regressiva. Você pode encontrar uma lista completa das constantes disponíveis que esta função requer na página para a função event_perform().

NOTA As ações no evento chamado com esta função são aplicadas à instância atual, e não às instâncias do dado object.

Sintaxe:

event_perform_object(obj, type, numb);

Argumento Tipo Descrição
obj Objeto Ativo The object that should have its event triggered.
type Tipo de evento Constante The type of event to perform.
numb Númeroreal ou constante de evento The specific event, if one is necessary (otherwise, just use 0).

Devoluções:

N/A

Exemplo:

event_perform_object(obj_Player, ev_keypress, ord("W"));

Isto realizaria o evento associado à verificação do teclado pressionado > W da tecla object "obj_Player" na instância atual.
