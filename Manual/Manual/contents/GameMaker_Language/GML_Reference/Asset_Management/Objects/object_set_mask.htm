object_set_mask

Com esta função você pode definir o índice de máscara de um object específico (para mais informações sobre máscaras, consulte o editor Object ). Isto significa que todas as instâncias deste object que são criadas após a máscara ter sido mudada serão criadas com esta nova máscara, enquanto as instâncias que já estão no room podem não ser afetadas. Você pode definir isto como -1 para remover uma máscara sprite e assim, por padrão, para a máscara definida para o sprite do object (ou sem máscaras se nenhuma sprite tiver sido escolhida).

Por favor, note que esta não é uma função de instância! Você pode definir o índice de máscara de instâncias individuais usando a função mask_index e, portanto, têm dez instâncias todas com uma máscara diferente da object de onde foram criadas, e mesmo se você alterar o índice de máscara do object usando esta função, todas as instâncias que estão atualmente no room permanecerão como estavam, e somente as instâncias criadas após a chamada da função começarão com a nova máscara.

Sintaxe:

object_set_mask(index, spr);

Argumento Tipo Descrição
index Objeto Ativo The index of the object to change.
spr Ativo Sprite The new sprite to use as the object's mask.

Devoluções:

N/A

Exemplo:

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

O código acima verifica o valor da variável global e depois muda o índice de máscara da "obj_Platform" object se for igual a dez.
