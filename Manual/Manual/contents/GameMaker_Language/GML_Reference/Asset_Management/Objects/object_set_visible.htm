object_set_visible

Com esta função você pode flag um object específico como sendo visível ou não (para mais informações sobre o "visível" flag veja a seção sobre o Editor de Objetos). Isto significa que todas as instâncias deste object que são criadas depois que o visível foi alterado serão criadas com este novo estado, enquanto as instâncias que já estão no room podem não ser afetadas.

Por favor, note que esta não é uma função de instância! Você pode definir o visível flag de instâncias individuais usando a função de visible e, portanto, dez instâncias todas marcadas como visíveis mesmo que o object em questão tenha o flag definido como falso no IDE, e mesmo que você mude o flag visível do object para verdadeiro usando esta função, todas as instâncias que atualmente no room permanecerão como estavam, e somente as instâncias criadas após a chamada da função começarão com o visível marcadas como verdadeiras.

Sintaxe:

object_set_visible( index, vis );

Argumento Tipo Descrição
index Objeto Ativo The index of the object to change.
vis Booleano The new visibility of the object (true=visível, false=invisível).

Devoluções:

N/A

Exemplo:

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

O código acima verifica o visível flag do object "obj_Block" e se for falso, então é alterado para ser sinalizado como verdadeiro.
