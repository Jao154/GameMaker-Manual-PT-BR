objeto_existe

Esta função retorna se o valor especificado é um object asset (true) ou não (false). Note que isto verifica se um object está presente no navegador Asset, e não no jogo room. Para isso você deve usar a função instance_exists().

Sintaxe:

object_exists(obj);

ArgumentoTipo Descrição
obj The index of the object to check.

Devoluções:

Exemplo:

var object_id = buffer_read(net_buffer, buffer_u16);
var pos_x = buffer_read(net_buffer, buffer_u16);
var pos_y = buffer_read(net_buffer, buffer_u16);

if (object_exists(object_id))
{
    instance_create_layer(pos_x, pos_y, "Instances", object_id);
}

O código acima lê dados de um buffer que foram recebidos via rede. Como ele contém um ID object (usado para casos de desova), precisamos primeiro garantir que o valor não seja inválido e que o object realmente exista. Para isso, chamamos object_exists() em uma condição, e se isso for verdade, criamos uma instância desse object na posição recebida.
