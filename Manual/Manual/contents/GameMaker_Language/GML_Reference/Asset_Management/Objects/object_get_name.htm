object_get_name

Esta função devolverá o nome como uma seqüência do especificado object. Este nome é o que foi especificado para o object no Asset Browser da janela principal do GameMaker. Note que isto é apenas uma cadeia de caracteres. e não pode ser usado para fazer referência direta ao object - para isso você precisaria do índice de objetos. Você pode, entretanto, usar este string para obter o índice de objetos usando o string retornado junto com a função asset_get_index().

Sintaxe:

object_get_name(obj);

ArgumentoTipo Descrição
obj The index of the object to check.

Devoluções:

Exemplo:

str = object_get_name(object_index);

O código acima receberá o nome do índice object para a instância que executa o código e armazenará o valor de retorno na variável "str".
