object_set_sprite

Com esta função você pode definir o índice sprite de um object específico. Isto significa que todas as instâncias deste object que forem criadas depois que o sprite_index for alterado serão criadas com este novo sprite_index, enquanto as instâncias que já estiverem no room podem não ser afetadas.

Por favor, note que esta não é uma função de instância! Você pode definir o índice sprite de instâncias individuais usando a função sprite_index e, portanto, têm dez instâncias, todas com um sprite diferente do object de onde foram criadas, e mesmo se você mudar o índice sprite do object usando esta função, todas as instâncias que estão atualmente no room permanecerão como estavam, e somente as instâncias criadas após a chamada da função começarão com o novo sprite.

Sintaxe:

object_set_sprite( index, spr );

Argumento Tipo Descrição
index Objeto Ativo The index of the object to change.
spr Ativo Sprite The sprite to assign to the object.

Devoluções:

N/A

Exemplo:

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

O código acima definirá o índice object "obj_Player" sprite para valores diferentes, dependendo do room em que a instância executando o código está atualmente. Em seguida, ele cria uma instância de "obj_Player".
