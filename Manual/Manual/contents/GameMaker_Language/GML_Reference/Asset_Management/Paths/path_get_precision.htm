caminho_esquer_precisão

Esta função retorna com que precisão o dado path foi "suavizado", e será um valor inteiro de 1 a 8. Embora você possa obter (e definir) este valor para uma linha reta path, ele não terá influência sobre como uma instância usa o path, pois só é relevante quando o tipo path é definido como "alisado".

Sintaxe:

path_get_precision(index);

ArgumentoTipo Descrição
index The index of the path to check.

Devoluções:

Exemplo:

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

O código acima verifica o path indexado em "pth_Patrol" para ver de que tipo é e se é liso, então verifica o quão preciso é o alisamento. Se for inferior a 8, é então definido para um valor de 8.
