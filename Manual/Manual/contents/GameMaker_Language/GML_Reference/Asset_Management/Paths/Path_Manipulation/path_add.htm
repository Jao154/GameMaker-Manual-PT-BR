caminho_add

Com esta função, você pode criar um path no GameMaker sem utilizar o editor path. Esta função retornará o índice do path que deverá ser armazenado em uma variável e usado como referência para aquele path a partir de então. Observe que o path criado está vazio, ou seja, não tem pontos definidos, então você terá que usar as outras funções disponíveis para adicionar pontos ao path ou estar usando grades MP para gerar o path. Uma vez que você tenha terminado de usar o path, ou desejar criar um novo e armazenar seu índice na mesma variável, você deve primeiro apagar o antigo path com path_delete para evitar vazamentos de memória que podem acabar por travar seu jogo.

Sintaxe:

path_add();

Devoluções:

Path Asset

Exemplo:

global.newpath = path_add();

Isto criará um novo path e atribuirá seu índice ao global.newpath.
