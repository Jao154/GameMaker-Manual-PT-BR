path_delete

Você pode usar este código para remover um path da memória. Se este path tiver sido criado dinamicamente usando path_add()a variável que contém o índice path não será mais válida para acessar o path por não existir mais, e se o path foi criado usando o Editor de Caminhos que path não pode mais ser acessado em todo o jogo, pois você está apagando permanentemente.

Sintaxe:

path_delete(index);

ArgumentoTipo Descrição
index The index of the path to delete.

Devoluções:

Exemplo:

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

O código acima irá criar um path e armazenar seu índice em uma variável local. Este path é então usado para armazenar um mp_grid_path() gerado path que, se conseguir encontrar seu caminho até o alvo, é então atribuído ao path indexado em "mypath". Finalmente, o "t_caminho" é excluído.
