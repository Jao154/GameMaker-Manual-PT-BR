path_flip

Esta função pega todos os pontos path e os desloca ao longo do eixo horizontal. Esta função muda o ativo real path , e assim afetará permanentemente a forma como o path é usado por todas as instâncias do jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função no asset duplicado (não esqueça de ligar para path_delete() no asset quando não for mais necessário).

Sintaxe:

path_flip(index);

ArgumentoTipo Descrição
index The index of the path to flip.

Devoluções:

Exemplo:

path_flip(mypath);

Isto inverteria o path indexado na variável "mypath" ao longo do eixo horizontal.
