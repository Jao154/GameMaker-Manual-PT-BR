caminho_espelho

Esta função leva todos os pontos path e os espelha ao longo do eixo vertical. Esta função muda o ativo path real, e assim afetará permanentemente a forma como o path é usado por todas as instâncias do jogo a partir do momento em que a função é usado até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, então ligue para esta função no duplicado asset (não esqueça de ligar para path_delete() no site asset quando não for mais necessário).

Sintaxe:

path_mirror(index);

ArgumentoTipo Descrição
index The index of the path to mirror.

Devoluções:

Exemplo:

path_mirror(mypath);

Isto espelharia o path indexado na variável "mypath" ao longo do eixo vertical.
