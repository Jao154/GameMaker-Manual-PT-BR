path_assign

Com esta função você pode copiar os dados path de um path para outro. O path a ser copiado será limpo primeiro (caso tenha algum ponto path ) e será completamente sobrescrito pelo path sendo copiado de. Nem path é excluído no processo e o resultado é dois paths, com dois índices diferentes, mas exatamente a mesma forma e pontos. Em geral, você gostaria de usar esta função em um path criado usando path_add(), uma vez que se você utilizá-lo em um path asset, afetará permanentemente o path para todas as instâncias do jogo desde o momento em que a função é utilizada até o final do jogo.

Sintaxe:

path_assign(index, path);

ArgumentoTipo Descrição
index The index of the path to be overwritten.
path The index of the path that will overwrite 'index'.

Devoluções:

Exemplo:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

O código acima criará um novo código vazio path indexado na variável "mypath" e depois copiará sobre os dados path de um dos três recursos path disponíveis.
