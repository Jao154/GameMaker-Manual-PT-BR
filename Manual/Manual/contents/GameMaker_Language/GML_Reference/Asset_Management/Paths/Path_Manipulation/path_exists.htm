caminho_existe

Esta função retorna se existe ou não um path com o índice dado. Note que se você verificar a existência de um path através de uma variável que ainda não foi declarada, isto irá lançar um erro.

Sintaxe:

path_exists(index);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to check for.

Devoluções:

Boolean

Exemplo:

if (path_exists(my_path))
{
    path_delete(my_path);
}

Este código verifica se a variável em questão armazena um índice path e se o faz, então o path é excluído da memória.
