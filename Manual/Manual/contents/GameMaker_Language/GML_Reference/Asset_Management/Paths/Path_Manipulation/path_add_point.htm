caminho_ponto_add

Com esta função, você pode adicionar um ponto ao path especificado e definir seu fator de velocidade. Este ponto é adicionado ao final do path, independentemente da posição do ponto, e o fator é igual à porcentagem de path_speed que a seguinte instância se aplica quando chega a este ponto no path. Se você deseja colocar um ponto path em alguma outra posição que não seja o fim, você deve usar path_insert_point().

Se usado em um path do navegador Asset, observe que a função mudará o ativo real, e assim afetará permanentemente como o path é usado por todas as instâncias no jogo desde o momento em que a função é usada até o final do jogo. Se isto não for o que você precisa, então você deve usar uma função como path_duplicate() para criar uma cópia do path primeiro, depois chamar esta função no asset duplicado (não se esqueça de chamar path_delete() no asset quando não for mais necessário).

Sintaxe:

path_add_point(index, x, y, speed);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to add the point to.
x Real The x coordinate of the new point.
y Real The y coordinate of the new point.
speed Real The speed factor of the point (default value is 100).

Devoluções:

N/A

Exemplo:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

O código acima criará uma circular path em torno da posição x/y da instância que está executando o código.
