caminho_escolher_pontar_velocidade

Com esta função você pode obter a velocidade do ponto (conforme definido no Path Editor ou quando você adiciona dinamicamente um ponto path usando path_add_point()) expresso como uma porcentagem. Portanto, se você tiver um ponto path ajustado para 50 no editor path, esta função retornaria 50 quando utilizada.

Sintaxe:

path_get_point_speed(index, n);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to check.
n Real The point number to check.

Devoluções:

Real

Exemplo:

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

O código acima verifica o path ponto "0" do path indexado por "pth_Patrol" para ver se a velocidade não é igual a 100. Se não for, então ele define a velocidade desse ponto para 100.
