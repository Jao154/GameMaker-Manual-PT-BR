path_get_name

Esta função retornará o nome do path que é referenciado como string. O nome é o que foi usado para definir o path no editor ou (se o path foi criado através de uma função de código) ele retornará um string com o formato " _newpathXX" onde "XX" é o número do path gerado, começando em 0 e incrementando por um cada vez que um novo path é criado. Observe que este é apenas um string e não pode ser usado para fazer referência direta ao path - para isso você precisaria da identificação do índice do caminho. Você pode, entretanto, usar este string para obter o ID do índice do caminho usando o retorno string ao longo com a função asset_get_index().

Sintaxe:

path_get_name(index);

ArgumentoTipo Descrição
index The index of the path to check.

Devoluções:

Exemplo:

path_name = path_get_name(path_array[0]);

Isto definirá "path_name" para o nome do path indexado no array dado na posição 0.
