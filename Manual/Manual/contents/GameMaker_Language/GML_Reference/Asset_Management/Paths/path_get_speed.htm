path_get_speed

Esta função retorna o fator de velocidade de qualquer posição em um path. A posição deve ter um valor entre 0 e 1 (você pode usar path_positionpor exemplo, se você precisar da posição atual da instância), sendo 0 a posição inicial e 1 a final. O valor retornado será o fator pelo qual o atual path_speed está sendo definido. Por exemplo, em um path com apenas dois pontos, 0 e 1, onde o fator de velocidade ponto 0 é 100 e o fator de velocidade ponto 1 é 0, a função path_get_speed(path, 0.5) retornará 50.

Sintaxe:

path_get_speed(ind, pos);

Argumento Tipo Descrição
index Ativo do caminho The index of the path to check.
pos Real How far through the path to check. Between 0 (start) and 1 (end), can be decimal.

Devoluções:

Real

Exemplo:

speed = path_get_speed(pth_Patrol, 0.5) / 20;

Isto ajustará a velocidade da instância de chamada para o fator de velocidade do ponto exatamente na metade do dado path, dividido por 20. Portanto, se o fator de velocidade encontrado for o padrão 100, ele ajustará a velocidade da instância de chamada para 5.
