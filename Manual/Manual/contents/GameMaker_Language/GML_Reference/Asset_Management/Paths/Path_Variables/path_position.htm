posição_do_caminho

Esta função pode ser usada para obter ou definir a posição de uma instância ao longo de um path. O valor é normalizado de 0 a 1, portanto, se você defini-lo para, por exemplo, 0,5, a instância será movida exatamente para o meio do path.

Sintaxe:

path_position;

Devoluções:

Exemplo:

path_position = random(1);

O código acima coloca a instância em uma posição aleatória em qualquer lugar no atual path.
