caminho_velocidade

Você pode usar esta função para obter ou para definir a velocidade de um path após ter sido iniciado, usando a função path_start(). Você pode usar valores negativos para significar que a instância deve seguir o path no verso.

Sintaxe:

path_speed;

Devoluções:

Exemplo:

path_speed = -1 + random(2);

O código acima estabelecerá a instância para percorrer o path a uma velocidade aleatória entre -1 e 1.
