timeline_loop

Esta variável retornará se a linha de tempo está em looping (verdadeiro) ou não (falso). Você pode mudar esta variável para switch looping on ou off e funciona com uma velocidade negativa da linha do tempo (se a posição da linha do tempo for inferior a 0, ela começará novamente no último momento definido).

Sintaxe:

timeline_loop;

Devoluções:

Exemplo:

if !timeline_loop
{
    timeline_loop = true;
}

O código acima verifica se a linha de tempo atualmente atribuída a uma instância será loop ou não e, se não o fizer, o define como verdadeiro.
