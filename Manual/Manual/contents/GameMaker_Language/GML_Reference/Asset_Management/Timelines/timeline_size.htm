tamanho_da_linha do tempo

Com esta função você pode obter o número total de momentos ativos para um timeline (um momento "ativo" é aquele que tem código ou GML Visual adicionado a ele). Isto pode ser útil ao criar momentos dinâmicos timelines usando o timeline_moment_add_script() e timeline_moment_clear() como você pode verificar se um determinado timeline tem o número correto de momentos ativos ou não tem nenhum.

Sintaxe:

timeline_size(ind);

ArgumentoTipo Descrição
ind The index of the timeline get the information from.

Devoluções:

Exemplo:

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

O código acima verifica o tamanho timeline dado, e se ele retornar 0 (ou seja: o timeline não tem momentos ativos) ele adicionará uma função script para ser usada em um momento aleatório dentro do timeline indexado na variável global "tl".
