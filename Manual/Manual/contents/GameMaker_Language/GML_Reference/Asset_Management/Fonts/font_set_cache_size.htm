font_set_cache_size_size

Ao realizar a mistura e coloração de imagens, o HTML5 não pode fazê-lo dinamicamente da mesma forma que um executável poderia ser executado. Portanto, o GameMaker tem que salvar temporariamente uma cópia misturada das imagens e carregar quando necessário. Esta função define quantas cópias mescladas do dado font podem ser colocadas em cache antes que as antigas sejam sobrescritas. O valor padrão é 4.

NOTA para sprite fonts você deve estar usando a função equivalente para sprites, sprite_set_cache_size().

Sintaxe:

font_set_cache_size(ind, max);

ArgumentoTipo Descrição
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

Devoluções:

Exemplo:

font_set_cache_size(fnt_MainMenu, 2);

Isto colocará o font cache do font indexado na variável "fnt_MainMenu" em 2 cópias.
