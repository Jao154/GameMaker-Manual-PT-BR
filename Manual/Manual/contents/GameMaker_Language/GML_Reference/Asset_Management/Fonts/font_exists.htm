font_existe

Esta função retorna se existe ou não um font com o índice especificado. Você pode verificar font índices como definido no Asset Browser, ou fonts que foram adicionados usando funções como font_add().

Sintaxe:

font_exists(ind);

ArgumentoTipo Descrição
ind Index of the font to check.

Devoluções:

Exemplo:

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

Isto irá definir o desenho ativo font para fnt_Main se ele existir.
