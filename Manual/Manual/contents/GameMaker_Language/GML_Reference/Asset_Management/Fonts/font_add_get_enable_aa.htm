font_addd_get_enable_aaa

Esta função pode ser usada para verificar se o anti-aliasing (AA) está habilitado para fonts adicionado usando font_add(). A função retornará true se AA estiver habilitada, e false se não estiver. Note que AA está habilitado por padrão, mas você pode alterar o estado de AA para adicionar fonts usando a função font_add_enable_aa()desde que seja chamado antes de adicionar o font.

Sintaxe:

font_add_get_enable_aa();

Devoluções:

Exemplo:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

O código acima verifica o status do anti-aliasing para adicionar fonts e, se não estiver ativado, então o habilitamos.
