font_add_enable_aaa

Esta função pode ser usada para ativar ou desativar o anti-aliasing (AA) para fonts adicionado usando font_add(). Esta função precisa ser chamada antes de adicionar qualquer fonts e configurá-la para true irá habilitar AA, e configurá-la para false irá desabilitá-la. Por padrão, AA está habilitado. Note que isto não terá efeito sobre fonts que foram adicionados antes da função ser chamada, e a função só precisa ser chamada uma vez quando o font for adicionado, e não a cada sorteio/passo que o font estiver sendo usado.

Sintaxe:

font_add_enable_aa(enable);

ArgumentoTipo Descrição
enable Whether to enable (true) ou desabilitar (false) AA para adicionar fonts.

Devoluções:

Exemplo:

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

O código acima verifica o status do anti-aliasing para adicionar fonts e, se não estiver ativado, então o habilitamos.
