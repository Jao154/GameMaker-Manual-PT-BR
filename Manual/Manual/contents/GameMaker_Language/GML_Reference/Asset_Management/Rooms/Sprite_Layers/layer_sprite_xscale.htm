camada_escrita_escala_x

Usando esta função você pode mudar se o elemento sprite dado em uma camada deve ou não ser escalado ao longo do eixo x. Você dá o ID do elemento sprite (que você obtém ao criar um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o valor da escala. Uma escala de 1 indica ausência de escala (1:1), valores menores serão escalados para baixo (0,5, por exemplo, será metade da largura do sprite usado), valores maiores serão escalados para cima, e valores negativos virarão o sprite e o escalarão a menos que o valor usado seja exatamente -1 (nesse caso o sprite usado é apenas virado da direita para a esquerda sobre sua posição de origem sem escala).

Sintaxe:

layer_sprite_xscale(sprite_element_id, xscale);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to change
xscale The xscale value (default is 1)

Devoluções:

Exemplo:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

O código acima verificará o elemento sprite atribuído à camada em que a instância que está executando o código está e, se for escalado em qualquer direção, definirá tanto a escala do eixo x quanto a escala do eixo y para 1.
