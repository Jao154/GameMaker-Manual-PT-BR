layer_sprite_alpha

Esta função controla o alfa (transparência) do sprite na camada asset. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e então definir o valor alfa a ser usado. O alfa pode ser entre 0 (totalmente transparente) e 1 (totalmente opaco) com o valor alfa padrão para o elemento sprite sendo 1. Nota que se a camada à qual o elemento sprite foi atribuído não for visível - ou o próprio elemento tiver sido tornado invisível - você não verá nenhuma diferença com esta função até que a camada ou elemento tenha sido tornado visível novamente.

Sintaxe:

layer_sprite_alpha(sprite_element_id, alpha);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

Devoluções:

Exemplo:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

O código acima recebe o valor de identificação do sprite asset chamado "Clouds" atribuído à camada "Asset_sky" e então define seu alfa para um valor aleatório entre 0 e 1.
