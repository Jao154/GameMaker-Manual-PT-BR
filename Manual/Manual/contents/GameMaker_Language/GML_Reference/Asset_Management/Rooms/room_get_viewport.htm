room_get_viewport

Com esta função você pode recuperar os detalhes de uma porta de visualização em um room diferente do atual. Você dá o ID room e o índice da porta de visualização para recuperar (de 0 a 7) e a função retornará uma matriz de 5 índices, onde:

  • [0] = visível (Booleano: true / false)
  • [1] = x posição (real)
  • [2] = posição y (real)
  • [3] = largura (real)
  • [4] = altura (real)

Sintaxe:

room_get_viewport(rm, vind);

Argumento Tipo Descrição
rm Ativos da sala The index of the room to get viewport data from
vind Real The index of the view port to get

Devoluções:

Array (5 elements: visible, x, y, width, height)

Exemplo:

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

O código acima recupera os dados da porta de visualização para o dado room e depois verifica se a porta está marcada como visível. Se não estiver, a visualização dos dados da porta é definida para torná-la visível.
