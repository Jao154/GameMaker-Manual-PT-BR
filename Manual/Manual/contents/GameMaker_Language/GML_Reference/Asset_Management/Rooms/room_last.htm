quarto_last

Esta variável lida apenas retorna o índice do último room no jogo (isto é definido pela ordem em que o rooms aparece no Gerente de Sala e não pela ordem em que eles foram criados). Note que esta variável não reconhecerá ou levará em consideração rooms que tenham sido adicionados dinamicamente usando room_add() ou room_duplicate().

Sintaxe:

room_last;

Devoluções:

Room Asset

Exemplo:

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

O código acima verifica se uma tecla foi pressionada e, em caso afirmativo, vai para o último room no jogo.
