nome_do_quarto_get_name

Esta função pode ser usada para retornar o nome do room especificado como um string. Note que este é apenas um string e não pode ser usado para fazer referência direta ao room - para isso você precisaria do índice da sala. Você pode, entretanto, usar este string para obter o índice da sala usando o string retornado junto com a função asset_get_index().

Sintaxe:

room_get_name(index);

ArgumentoTipo Descrição
index The index of the room to check the name of.

Devoluções:

Exemplo:

var roomname = room_get_name(room);

draw_text(32, 32, roomname);

O código acima receberá o nome do atual room e o desenhará na tela.
