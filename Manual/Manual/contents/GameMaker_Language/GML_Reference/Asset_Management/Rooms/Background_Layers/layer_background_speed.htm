fundo_da_camada_velocidade

Esta função pode ser usada para definir a velocidade de um animador sprite que foi atribuído a um elemento de fundo. Você dá a identificação do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e então definir a velocidade da animação. Se o IDE definiu a velocidade de animação para o sprite costumava ser calculado como Frames Por Frame de Jogo, então o valor da velocidade dada é um multiplicador, sendo 1 o valor padrão, e definindo-o como 0,5 será metade da velocidade da animação, enquanto que definindo-o como 2 será o dobro. Entretanto, se o IDE definiu a velocidade da animação para o sprite costumava ser calculado como Frames Por Segundo, então o valor da velocidade dada é um valor literal de frames por segundo, então definindo este valor para 5, por exemplo, definirá a animação para loop 5 frames a cada segundo que o jogo for executado. Se o elemento de fundo não tiver nenhum sprite anexado ou o sprite usado não tiver sub-imagens, esta função não terá efeito.

Sintaxe:

layer_background_speed(background_element_id, speed);

Argumento Tipo Descrição
background_element_id Identificação do elemento de fundo The unique ID value of the background element to change
speed Real The speed multiplier/value to use for the element

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

O código acima obterá a identificação da camada chamada "Background_water" e depois a utilizará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para mudar a velocidade da imagem do elemento.
