fx_get_parameter

Esta função é utilizada para recuperar o valor de um parâmetro a partir de um FX Struct. Você especifica o FX struct para ler (como retornado de fx_create() ou layer_get_fx()) e o nome do parâmetro como string, e a função retorna seu valor atual. Este pode ser um valor real, ou um array se o parâmetro for um vetor (que é um grupo de valores, como um vec4 que armazena uma cor RGBA).

Encontre os nomes dos parâmetros para um filtro/efeito nesta página: Tipos e parâmetros de FX.

Sintaxe:

fx_get_parameter(filter_or_effect, parameter_name);

Argumento Tipo Descrição
filter_or_effect Estruturas FX The FX struct to get the parameter from
parameter_name Cordão The name of the parameter to get the value of (as a string)

Devoluções:

Real or Array

Exemplo:

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

O código acima recupera o FX struct de uma camada que se supõe ter o filtro "Color Tint" aplicado a ele, e obtém o valor de seu parâmetro "g_TintCol ", que retorna um array contendo os 4 valores para sua cor de tonalidade (vermelho, verde, azul e alfa). Ele então imprime esse valor para o log de saída.
