fx_get_single_layer

Esta função retorna se o filtro/efeito fornecido está usando o modo de camada única (true) ou não (false). Ver fx_set_single_layer() para informações sobre o modo Camada Única.

Você especifica um FX struct (como retornado de fx_create() ou layer_get_fx()) e a função retorna um booleano que informa se o modo Camada Única está ativado ou não para o FX.

Sintaxe:

fx_get_single_layer(filter_or_effect);

ArgumentoTipo Descrição
filter_or_effect The FX struct to read

Devoluções:

Boolean

Exemplo:

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

O código acima recebe o FX struct de uma camada chamada "Distort " e verifica se o modo Camada Única está habilitado nele. Se não estiver, habilita o modo de Camada Única para esse FX.
