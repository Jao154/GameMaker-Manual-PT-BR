fx_create

Esta função é utilizada para criar um novo FX Struct para um determinado tipo de efeito, que pode então ser aplicado a uma camada.

Os nomes dos filtros/efeitos que você pode passar para esta função estão listados nesta página: Todos os tipos de filtros/efeitos.

Utilização

Você pode ler os parâmetros para o efeito criado, chamando fx_get_parameter/s sobre o retorno struct, e atribuir-lhes novos valores, ligando para fx_set_parameter/s.

Note que o struct retornado por esta função não conterá membros visíveis e só pode ser modificado através de uma das funções Filtro/efeito.

Falhas funcionais

A função retornará -1 se o struct falhou em criar por qualquer razão; isto pode acontecer se o nome FX especificado estava errado ou se o FX não foi carregado no pacote do jogo.

Para garantir que esta função funcione corretamente, certifique-se de que a FX de que você precisa esteja carregada em seu pacote de jogo, conforme explicado na página Funções de Filtro e Efeito:

Se estas condições não forem cumpridas, ou seja, você não está passando em um string literal e o filtro/efeito não está presente em nenhum rooms, então a função não será capaz de carregar esse filtro/efeito e retornará -1.

Sintaxe:

fx_create(filter_or_effect_name);

Argumento Tipo Descrição
filter_or_effect_name Cordão The name of the filter/effect to create (get them from All Filter/Effect Types).

Devoluções:

FX Struct or -1 (on failure)

Exemplo:

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

O código acima cria um novo FX struct a partir do efeito "_filter_tintfilter ", que é o filtro "Colour Tint" encontrado no editor Room. Ele atribui um valor a seu parâmetro "g_TintCol " que é a cor da tonalidade, e como é um vec4 internamente, ele toma uma matriz contendo quatro valores (correspondentes a seus valores vermelho, verde, azul e alfa). O FX struct para a tonalidade é então aplicado a uma camada FX.
