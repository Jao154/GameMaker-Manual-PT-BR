fx_get_parameters

Esta função é utilizada para recuperar uma estrutura contendo todos os parâmetros para o FX struct. Você especifica um FX struct (como retornado de fx_create() ou layer_get_fx()) e esta função retorna um struct contendo variáveis para todos os parâmetros aplicáveis àquele filtro/efeito. Os parâmetros que utilizam vetores (ou seja, grupos de valores) estarão presentes como arrays.

Você pode obter os nomes de todos os parâmetros em uma matriz separada ligando para fx_get_parameter_names(), ou encontrá-los nesta página: Tipos e parâmetros de FX.

A mudança de valores neste struct não aplicará essas mudanças no filtro/efeito; você precisará ligar para fx_set_parameters() para aplicar o struct modificado de volta ao filtro/efeito para que qualquer mudança tenha efeito.

Sintaxe:

fx_get_parameters(filter_or_effect);

Argumento Tipo Descrição
filter_or_effect Estruturas FX The FX struct for which the parameters will be returned

Devoluções:

Struct (or -1 on failure)

Exemplo:

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

O código acima obtém o FX struct para uma camada que se supõe ter o filtro "Color Tint" aplicado a ele, e recupera seu parâmetro struct ligando para fx_get_parameters(). Depois disso, ele cria um valor oscilante usando sin() e current_time, que é então usado para os valores RGBA para o efeito tint, para que ele continue animando. O array RGBA é atribuído à variável g_TintCol no parâmetro struct, e o struct é então aplicado de volta ao FX struct, chamando fx_set_parameters().
