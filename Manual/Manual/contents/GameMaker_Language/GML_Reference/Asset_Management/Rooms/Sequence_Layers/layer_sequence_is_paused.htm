layer_sequence_is_paused

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e verificará se o sequence está ou não em pausa, retornando true se estiver em pausa, e false se não estiver.

Sintaxe:

layer_sequence_is_paused(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Exemplo:

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

O código acima verifica se o ID do elemento sequence armazenado na variável "title_sequence" está em pausa, e se é ele que começa a tocar.
