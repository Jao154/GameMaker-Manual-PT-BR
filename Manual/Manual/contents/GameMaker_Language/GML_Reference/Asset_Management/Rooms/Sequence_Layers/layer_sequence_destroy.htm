layer_sequence_destroy

Com esta função você pode destruir (remover do room) um elemento sequence. Você fornece o ID do elemento sequence conforme retornado por layer_sequence_create() ou por uma das funções dos elementos da camada e o sequence será destruído.

Sintaxe:

layer_sequence_destroy(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Exemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_destroy(a[i])
    }
}

O código acima obtém as identificações de todos os elementos atribuídos à camada da instância que está executando o código. O código então verifica se algum dos elementos retornados é sequence assets e se eles são destruídos.
