layer_create

Esta função é utilizada para criar uma nova camada dentro da atual room. Você fornece a profundidade que a nova camada deve ocupar e a função retorna o valor único de identificação da camada que pode então ser usado para acessar a camada em outras funções. Você também pode fornecer um argumento opcional de "nome", que lhe permitirá dar à camada um nome específico (como um string), e em vez de usar o ID da camada para acessar a camada, você pode usar este nome string em vez disso, embora deva ser notado que usar o nome strings terá um impacto maior no desempenho do que usar apenas o valor de ID. Se você não especificar o nome da camada, então será atribuído à camada criada um nome com o formato "_layer_XXX", onde "XXX" é um valor hexadecimal usado para dar à camada um nome único.

Ao dar um valor de profundidade, este será maior quanto mais "longe" da câmera a camada está e menor quanto mais "perto" da câmera a camada está, portanto, se você tiver três camadas com profundidades -100, 0, 100, as camadas desenharão na ordem 100, 0, -100, de modo que a camada "superior" (ou seja, a mais próxima da visão da câmera e assim desenhada sobre tudo mais) será a camada com a profundidade -100.

A imagem a seguir mostra um esquema de como funciona a profundidade para camadas:

Vale notar que embora você possa atribuir mais de uma camada a uma única profundidade, não há como garantir que camadas na mesma profundidade serão desenhadas em qualquer ordem e, como tal, é melhor atribuir apenas uma camada a uma determinada profundidade. Observe também que as profundidades mínima e máxima das camadas são de -16000 a 16000, e qualquer coisa fora dessas profundidades não será renderizada. Se você precisar de uma profundidade fora dessa faixa, então você precisará usar a função layer_force_draw_depth().

Sintaxe:

layer_create(depth, [name])

Argumento Tipo Descrição
depth Real The depth that the layer has to be created at.
name Cordão OPTIONAL O nome para dar a camada (a string).

Devoluções:

Layer ID

Exemplo:

global.Bullet_Layer = layer_create(-100);

O código acima criará uma nova camada com um valor de profundidade de -100 e atribuirá seu valor de identificação a uma variável global.
