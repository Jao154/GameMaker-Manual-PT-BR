camada_add_instância

Esta função pode ser usada para mover uma determinada instância da camada em que se encontra atualmente para outra camada. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a identificação da instância a ser movida entre camadas. Por exemplo, digamos que seu jogador está sobre uma camada que está a uma profundidade inferior a outra camada e você quer que ela apareça atrás das camadas entre as duas. Você pode usar esta função para switch o jogador desde a camada de primeiro plano até a camada de fundo e fazê-la aparecer atrás das outras camadas que estão sendo desenhadas.

NOTA Esta função não pode ser usada para adicionar uma nova instância a uma camada. Você deve ter criado a instância anteriormente e armazenado sua identificação em uma variável.

NOTA: Esta função só é válida dentro do escopo do atual room e não pode ser usada quando o escopo tiver sido alterado usando a função layer_set_target_room().

Sintaxe:

layer_add_instance(layer_id, instance_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the instance layer to target (or the layer name as a string)
instance_id Identificação da instância The unique instance ID value of the instance to move over to the target layer

Devoluções:

N/A

Exemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

O código acima obterá primeiro o índice da instância mais próxima à posição x/y dada e o armazenará em uma variável local. Em seguida, ele obtém a identificação única da instância da camada chamada "Instances Front", e move a instância encontrada para essa camada.
