camada_destroy_instances

Esta função pode ser usada para destruir todas as instâncias atribuídas à camada em questão. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isto terá um impacto no desempenho), e então todas as instâncias que estão na camada serão removidas do jogo, desencadeando seus eventos Destroy and Clean Up.

Sintaxe:

layer_destroy_instances(layer_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer (or the layer name as a string)

Devoluções:

N/A

Exemplo:

if global.game_over
{
    layer_destroy_instances(layer);
}

O código acima verificará uma variável global e se for true então todas as instâncias que estão na camada da instância de chamada serão destruídas (incluindo a instância de chamada).
