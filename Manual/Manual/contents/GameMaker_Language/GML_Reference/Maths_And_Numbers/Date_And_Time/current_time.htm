tempo_actual

Esta variável lida apenas retornará o número de milissegundos que passaram desde que o jogo foi iniciado.

Sintaxe:

current_time;

Devoluções:

Exemplo:

if (current_time > 600000)
{
    msg = show_question_async("Would you like to rate?");
}

O código acima verifica se passaram mais de 10 minutos antes de fazer uma pergunta ao usuário.
