data_get_timezone

Esta função faz com que o fuso horário base seja utilizado para todas as demais funções de data e hora. Este fuso horário pode ser local (conforme definido pelo sistema) ou UTC, e a função retornará uma das seguintes constantes:

Fuso Horário Constante
Constante Descrição
timezone_local usar o fuso horário local conforme definido pelo sistema
timezone_utc uso Tempo Universal Coordenado

Sintaxe:

date_get_timezone();

Devoluções:

Time Zone Constant

Exemplo:

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

Este código verifica a configuração do fuso horário base do jogo e, se não for UTC, ele o altera.
