data_data_hora_string

Com esta função você pode criar um string contendo a data/hora dada, formatada para o sistema ou dispositivo que está executando o jogo quando a função é chamada.

Sintaxe:

date_datetime_string(date);

ArgumentoTipo Descrição
date The date to use.

Devoluções:

Exemplo:

str = date_datetime_string(date_current_datetime());

Isto definiria "str" para algo como "15 de setembro de 2011, 11:33.00", dependendo das configurações do sistema para exibição de data/hora e da data e hora atuais.
