matemático_set_epsilon

Epsilon é um valor utilizado para determinar se dois números sujeitos a erro de arredondamento estão suficientemente próximos para serem considerados "iguais". É útil quando se trata de matemática de ponto flutuante, pois pode reduzir os "erros de arredondamento" que fazem com que certas operações retornem valores que parecem incorretos ou contrários ao que se espera. Por exemplo, podemos ter adicionado um valor ao índice de imagem de um sprite e esperar que o resultado seja um único inteiro, mas devido à natureza da matemática de ponto flutuante, o valor final real acaba sendo algo como 5,0000002, portanto, quando temos a seguinte verificação:

if image_index == 5
{
    //do something
}

O código não se comporta como esperado e retorna false. Entretanto, se definirmos o valor epsilon para 0,000001, o valor image_index será arredondado para o número real mais próximo que é +/- 0,000001 do valor original, tornando o retorno da comparação acima verdadeiro.

O valor do epsilon será usado para todos os seguintes operadores inteiros:

  • < : Menos de
  • > : Maior do que
  • == : Igual a
  • <= : Menor ou igual a
  • >= : Maior ou igual a
  • != : Não igual a

Observe que a definição de um valor epsilon de 0 desativará todo o arredondamento, e usar um valor de 1 dará um erro.

Sintaxe:

math_set_epsilon(epsilon);

Argumento Tipo Descrição
epsilon Real The new epsilon value (from 0 to 0.999999999).

Devoluções:

Real

Exemplo:

math_set_epsilon(0.0001);

Isto definirá o valor do epsilon para todas as operações adicionais de ponto flutuante.
