mediana

Esta função retorna a mediana dos valores de entrada, ou seja, o valor médio. Quando o número de argumentos é igual, o menor dos dois valores médios é retornado e a função pode ter tantos argumentos quantos forem necessários (note que mais argumentos significará que a função será mais lenta de analisar) que devem ser todos valores reais. Isto significa que, por exemplo, median(43, 12, 25, 19, 6) retornaria 19, pois é o valor médio entre todos os outros.

Sintaxe:

median(val1, val2, ... max_val);

ArgumentoTipo Descrição
val0 ... max_val The values to compare.

Devoluções:

Exemplo:

x = median( 0, x, room_width );
y = median( 0, y, room_height );

Isto impedirá que o jogador saia de qualquer lado do room, usando a mediana como grampo. Se o jogador, por exemplo, se mover para a esquerda do limite room, seu x será menor que 0. Isto significará o número do meio da primeira das medianas será 0, então o jogador será pulado para (0,y).
