aleatório

Esta função é boa para probabilidades onde não é necessário devolver um número inteiro (número inteiro). Por exemplo, random(100) devolverá um valor de 0 a 99, mas esse valor pode ser 22,56473! Você também pode usar números reais e não números inteiros nesta função - random(0.5), que retornará um valor entre 0 e 0,499999999.

NOTA: Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo.

Sintaxe:

random(n);

ArgumentoTipo Descrição
n The upper range from which the random number will be selected.

Devoluções:

Exemplo:

if (random(10) >= 9)
{
    score += 100;
}

Isto produzirá aproximadamente uma em cada dez chances de adicionar 100 à pontuação.
