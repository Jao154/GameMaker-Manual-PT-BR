log10

Esta função é similar à função de log2(n) somente que log10(n) obtém o número de 10's que precisamos multiplicar juntos para obter n. Então log10(100) retorna 2, já que multiplicar 10 por si só dá 100, log10(1000) retorna 3 como 10x10x10=1000 etc... O número não precisa ser divisível por dez, no entanto, mas isso ainda retorna um valor apropriado, por exemplo, se tivermos log10(101) recebemos 2.00432137... que é quantas vezes você poderia dividir 101 por 10.

Sintaxe:

log10(n);

ArgumentoTipo Descrição
n The input value.

Devoluções:

Exemplo:

logval = log10(num);

O código acima recebe o log10 do valor armazenado em "num".
