andar

Retorna o piso de n, ou seja, n arredondado para um número inteiro. Isto é semelhante ao round() mas só arredonda para baixo, não importa qual seja o valor decimal, portanto floor(5.99999) retornará 5, assim como floor(5.2), floor(5.6457) etc...

Sintaxe:

floor(n);

ArgumentoTipo Descrição
n The number to floor.

Devoluções:

Exemplo:

val = floor( 3.9 );

Isto colocará Val a 3.
