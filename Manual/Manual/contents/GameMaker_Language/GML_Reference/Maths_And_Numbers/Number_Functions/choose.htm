escolha

s vezes você quer especificar algo diferente dos números para uma seleção aleatória, ou os números que você quer não estão em nenhuma ordem real ou dentro de qualquer faixa definida. Nestes casos, você usaria choose() para gerar um resultado aleatório. Por exemplo, digamos que você queira criar um object com um sprite aleatório no início, então você poderia usar esta função para definir o índice sprite para um de um conjunto de sprites dado. Note que você pode ter tantos quantos você precisar (note que mais argumentos significarão que a função será mais lenta de analisar).

Esta função retornará o mesmo valor toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo.

Sintaxe:

choose(val0, val1, val2... max_val);

Argumento Tipo Descrição
val0... max_val Variável Any type of value(s).

Devoluções:

Variable (One of the given arguments)

Exemplo:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

O código acima usa para definir uma série de propriedades para o exemplo.
