is_bool

Esta função retorna se uma determinada variável é um booleano (true i ou false) ou não. Em alguns casos você quer verificar e ver se uma variável no GameMaker possui um valor booleano, e é quando você usaria esta função.

Sintaxe:

is_bool(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

O código acima verifica a variável "val" para ver se é um número real e se é um número global.
