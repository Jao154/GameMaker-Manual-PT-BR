nomes_de_instância_variável_get_names

Com esta função você pode recuperar uma matriz preenchida com os nomes das variáveis de instância para uma instância, ou as variáveis globais para um jogo. Quando você passa em um valor de ID de instância, cada entrada no array será um string do nome da variável que corresponde a uma variável de escopo de instância que foi criada na instância. Entretanto, se você passar na palavra-chave global, cada entrada no array será um string do nome da variável que corresponde a uma variável de escopo global.

Sintaxe:

variable_instance_get_names(instance_id/global);

Argumento Tipo Descrição
instance_id/global Identificação da instância ou global The unique ID value of the instance to check or the keyword global

Devoluções:

Array (each entry is a string)

Exemplo:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

O código acima recuperará um array de todas as variáveis de escopo da instância executando o bloco de código e então exibirá estes junto com seus valores na saída de depuração.
