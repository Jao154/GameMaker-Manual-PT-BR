is_string

Esta função retorna se uma determinada variável é um número string ou não. Em alguns casos você quer verificar e ver se uma variável no GameMaker possui um string e não um real e é quando você usaria esta função. Ela não retorna o string mas sim true ou false, então um valor de, por exemplo, "peixe" para n retornará true, mas um valor de 200 para n retornará false.

Sintaxe:

is_string(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if is_string(val)
{
    name = "Player: " + val;
}

O código acima verifica a variável "val" para ver se ela contém um string e se a adiciona em outro string que é então atribuído à variável "name".
