is_matrix

Esta função retorna se uma determinada variável se refere a uma matriz ou não. Em alguns casos, você deseja verificar e ver que tipo de dados uma variável contém no GameMaker e é quando você usaria esta função. Ela retorna true ou false, dependendo se o valor é ou não uma matriz.

Sintaxe:

is_matrix(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if !is_matrix(val)
{
    show_debug_message("Not a valid matrix!");
}

O código acima verifica a variável "val" para ver se ela contém uma matriz e se não contém, então ela mostra uma mensagem no console de depuração.
