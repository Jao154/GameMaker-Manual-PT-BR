nome_da_estrutura_variável_get_names

Com esta função você pode recuperar um array preenchido com os nomes das variáveis de um struct. Você passa na referência struct para verificar, e cada entrada no array será uma seqüência dos nomes das variáveis que o struct contém.

Sintaxe:

variable_struct_get_names(struct);

ArgumentoTipo Descrição
struct The struct reference to check.

Devoluções:

(each entry is a string)

Exemplo:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
{
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
}

O código acima recuperará um array dos nomes das variáveis para o dado struct e então exibirá estes junto com seus valores na saída de depuração.
