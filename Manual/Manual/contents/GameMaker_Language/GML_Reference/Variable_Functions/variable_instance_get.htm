variable_instance_get

Com esta função você pode obter o valor de uma determinada variável nomeada. Você fornece o valor único de identificação da instância (que pode ser encontrado nas propriedades da instância no editor room, ou é retornado quando você chama a função instance_create_layer()) assim como o nome da variável para obter o valor de uma string (veja o código de exemplo abaixo). A função retornará o valor mantido pela variável, ou undefined se a variável não existir.

IMPORTANTE! Se a variável que você está recebendo não existir, então a função retornará a palavra-chave undefined e você poderá receber erros que impedirão o funcionamento do jogo, portanto, em caso de dúvida, use a função variable_instance_exists primeiro.

Sintaxe:

variable_instance_get(instance_id, name);

Argumento Tipo Descrição
instance_id Identificação da instância The unique ID value of the instance to use
name Cordão The name of the variable to get (as a string)

Devoluções:

Variable (any data type) or undefined (if the named variable does not exist)

Exemplo:

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

O código acima verificará se uma variável existe e se ela existe, então o valor que possui é recuperado e armazenado em uma variável local. Se não existir, a variável local é definida como -1.
