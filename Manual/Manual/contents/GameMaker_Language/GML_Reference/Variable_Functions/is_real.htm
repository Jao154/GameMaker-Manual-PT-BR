é_real

Esta função retorna se uma determinada variável é um número real (simples, duplo ou inteiro) ou não. Em alguns casos você quer verificar e ver se uma variável possui um número real, e é quando você usaria esta função. Ela não retorna o número real, mas sim true ou false, então um valor de, por exemplo, "peixe" retornaria false, porém um valor de 200 retornaria true.

NOTA: Esta função retornará false para qualquer enum como são armazenados como int64s. A função correta para verificar esse tipo de dados é is_int64().

Sintaxe:

is_real(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if is_real(val)
{
    score += val;
}

O código acima verifica a variável "val" para ver se é um número real e se é, então, adiciona-a à pontuação.
