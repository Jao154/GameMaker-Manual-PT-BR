is_nan

Esta função retorna se uma determinada variável é NaN (não um número) ou não, retornando true se for, e false se não for.

Sintaxe:

is_nan(n);

Argumento Tipo Descrição
n Variável The argument to check.

Devoluções:

Boolean

Exemplo:

if is_nan(global.value)
{
    show_debug_message("Value is not a number");
}

O código acima verifica o "valor" da variável global para ver se é um número ou não e mostra uma mensagem de debug se não for.
