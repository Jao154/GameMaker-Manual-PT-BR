array_resize

Com esta função você pode redimensionar uma dimensão de matriz existente para um novo tamanho. Você fornece a matriz a ser redimensionada assim como o novo número de índices para a matriz, e a função redimensionará essa matriz. Note que esta função foi projetada para redimensionar uma matriz para um comprimento menor, já que você pode redimensionar para cima simplesmente definindo um novo índice na matriz. Dito isto, se você a usar para dimensionar um array, qualquer novo índice será definido para o valor padrão de 0.

Esta função também pode ser usada para matrizes multidimensionais, desde que você especifique qual dimensão deseja redimensionar ao fornecer o índice da matriz, seguindo este padrão:

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

O código declara que apenas o primeiro slot da segunda dimensão é redimensionado, uma vez que os slots em qualquer dimensão da matriz não estão interligados e podem ter tamanhos diferentes; por exemplo:

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

No código acima, o comprimento da segunda dimensão da matriz é 10 na primeira fenda, mas 22 na segunda fenda.

Sintaxe:

array_resize(array_index, new_size);

ArgumentoTipo Descrição
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

Devoluções:

Exemplo:

if array_length(a) > 10
{
    array_resize(a, 10);
}

O código acima verifica o comprimento de uma matriz e, se ela tiver mais de 10 índices, é redimensionada.
