bool

Esta função tentará converter um determinado valor em um tipo de dado booleano, onde o valor será retornado como true se for maior ou igual a 0,5, e false caso contrário.

Sintaxe:

bool(n);

Argumento Tipo Descrição
n Real The value to convert.

Devoluções:

Boolean

Exemplo:

if !is_bool(val)
{
    val = bool(val);
}

O código acima verifica a variável "val" para ver se é um booleano e se não é, então é convertido em um.
