int64

Esta função tentará converter um determinado valor em um inteiro de 64 bits assinado, onde o valor deve ser ou um real, um string, um int64, um int32, ou um ptr. Qualquer outra coisa fará com que o jogo trave com uma mensagem de erro. Você pode verificar se uma variável contém uma int64 usando a função is_int64().

Sintaxe:

int64(val);

Argumento Tipo Descrição
val Real, Cordãoou Pointer The value to convert.

Devoluções:

int64 (signed 64-bit integer)

Exemplo:

steam_handle = int64(global.fileReadString);

O código acima converte o valor mantido na variável global para um inteiro de 64bit.
