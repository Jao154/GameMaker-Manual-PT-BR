Facebook

O Facebook é um dos sites de mídia social mais importantes da Internet e muitos jogos e desenvolvedores atualmente têm suas próprias páginas para postar jogos e mídia. O GameMaker tem um extension que permite que você se conecte com o Facebook e execute certas ações e tarefas que sua API permite. Recomenda-se que você visite a página dos desenvolvedores do Facebook antes de começar a usar o extension como um firme entendimento de como o Gráfico do Facebook funciona e as chamadas e permissões disponíveis é essencial para que você obtenha o máximo dessas funções.

Você pode descobrir como configurar e usar o Facebook extension para todas as plataformas disponíveis no link a seguir:

  • Android, iOS e HTML5: Integrando o Facebook

As seguintes são funções obsoletas que só eram válidas para o antigo API do Facebook. Como tal, você não pode mais usar estas funções em seus projetos e elas são listadas aqui caso você tenha importado um projeto antigo com elas. Para a funcionalidade atual do Facebook você precisa ter ativado a API do Facebook nas Opções de Jogo apropriadas para a plataforma alvo, o que o levará a baixar e instalar um extension. O extension vem com um "mini-manual" no formato *.pdf explicando como utilizá-lo, além de mostrar todas as definições e exemplos de funções. Todas as referências às antigas funções facebook_*() precisarão ser substituídas (e atualizadas) para usar os equivalentes do extension fb_*().

As funções listadas abaixo foram para integrar seu jogo com o Facebook:

  • facebook_init
  • facebook_login
  • facebook_request_publish_permissions
  • facebook_request_read_permissions
  • facebook_check_permission
  • facebook_status
  • facebook_accesstoken
  • facebook_user_id
  • facebook_graph_request
  • facebook_dialog
  • facebook_send_invite
  • facebook_post_message
  • facebook_logout
  • facebook_launch_offerwall
