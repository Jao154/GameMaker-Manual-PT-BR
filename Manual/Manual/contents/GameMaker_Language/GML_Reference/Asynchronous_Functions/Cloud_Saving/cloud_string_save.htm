cloud_string_save

Esta função comprometerá um string ao serviço de nuvem escolhido para armazenamento. A função retornará um valor de identificação único que deverá ser usado no evento assíncrono apropriado para identificar o mapa DS que é retornado como uma "chamada de retorno" do serviço de nuvem. O string deve conter todas as informações que você precisa salvar para seu jogo, pois você só pode armazenar um único "blob de dados" para a nuvem, e executar esta função novamente irá sobrescrever quaisquer valores previamente armazenados (assim como usar a função cloud_file_save() função). A descrição deve ser uma breve string de informações que descrevam a gravação, por exemplo: "Level2, Stage2".

Para maiores informações sobre os dados assíncronos retornados, por favor, veja a função cloud_synchronise().

Sintaxe:

cloud_string_save(string, description);

ArgumentoTipo Descrição
string The data string to be uploaded.
description A brief description of the data being stored.

Devoluções:

Async Request ID

Exemplo:

var t_str = "";
for (var i = 0; i < 10; i++;)
{
    t_str += string(global.Highscore[i]) + "|"
}
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

O código acima cria um string a partir dos valores armazenados na matriz global "Highscores" e depois escreve este string para o serviço de nuvem para armazenamento, assim como grava-o em um arquivo para armazenamento local.
