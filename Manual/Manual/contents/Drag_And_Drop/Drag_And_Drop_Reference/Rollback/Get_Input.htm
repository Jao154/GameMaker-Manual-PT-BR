Obter Input (Rollback)

Esta ação retorna um struct contendo os valores de entrada para um jogador. É necessário um argumento opcional especificando o ID do jogador para o qual serão devolvidos os valores de entrada.

É recomendado o uso de If Key Down (Rollback), If Key Pressed (Rollback) e If Key Up (Rollback) para verificação de entradas em multiplayer.

Quando usado em uma instância de jogador que foi criada pela Define Player (Rollback), você não precisa especificar um player ID, então você pode apenas especificar player_id.

Quando usado em quaisquer outras instâncias, ou instâncias de jogador que foram criadas manualmente, você deve especificar uma identificação de jogador.

Para informações sobre quais entradas são devolvidas e como podem ser alteradas, consulte Definindo entradas.

Argumentos:
