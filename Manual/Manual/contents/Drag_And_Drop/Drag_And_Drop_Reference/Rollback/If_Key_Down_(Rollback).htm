Se a chave estiver em baixo (Rollback)

Esta ação verifica se a entrada dada é realizada para o jogador atual.

Ele permite anexar ações à direita, que só funcionam quando a entrada dada é encontrada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Input O nome da entrada a ser verificada
Not Permitir que isto verifique se a entrada não é mantida

Exemplo:

As ações acima movem o jogador para a esquerda ou direita, dependendo da entrada.
