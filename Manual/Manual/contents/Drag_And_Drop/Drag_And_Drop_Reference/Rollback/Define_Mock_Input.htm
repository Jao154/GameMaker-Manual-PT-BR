Definir entrada simulada (Rollback)

Esta ação pode ser usada durante o Sync Test para atribuir uma entrada "mock" temporária ao jogador especificado. Isto substitui os controles definidos para o jogador e qualquer entrada aleatória do Sync Test.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Player ID O número do jogador que recebe este conjunto de entradas simuladas
Input O nome da entrada para definir
Value A constante de entrada a ser atribuída (ver Controles personalizados)

Exemplo:

A ação acima define as entradas simuladas de fogo, esquerda, direita, para cima e para baixo para o jogador 1.
