Definir Entrada (Rollback)

Esta função é utilizada para definir entradas personalizadas para todos os jogadores. Ela toma um Par de Entrada/Valor, onde a Entrada é o nome da entrada, e o Valor é a constante de entrada atribuída a ela.

Para saber como atribuir entradas personalizadas, consulte Controles personalizados.

Se utilizada, esta função deve ser executada antes da ação de junção/criação.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Input O nome da entrada para definir
Value A constante de entrada a ser atribuída (ver Controles personalizados)

A ação acima define o fogo de entrada, esquerda, direita, para cima e para baixo para todos os jogadores.
