Iniciar o Rollback Game

Esta ação força o jogo multijogador atual a começar, antes que todos os jogadores tenham se juntado. Primeiro é preciso criar um jogo multiplayer usando o Create Rollback Game.

O sistema normalmente espera que todos os jogadores adiram antes de iniciar automaticamente o jogo. Esta ação pode ser usada para iniciar o jogo mais cedo, antes que todos os jogadores tenham aderido.

Você pode ligar para o Manual de Utilização para fazer o Rollback esperar por esta chamada mesmo depois que todos os jogadores tenham aderido.

A execução desta ação desencadeará o evento Rollback Start.

Sintaxe de ação:

Voltar: Ações Multiplayer Rollback
A seguir: Deixe o Rollback Game
