Sequência de jogo

Com esta ação você pode pausar uma peça de teatro sequence. Você fornece o ID do elemento sequence (como retornado pela ação Create Sequence), e a ação fará uma pausa na jogada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Variable A identificação do elemento sequence para fazer uma pausa.

Exemplo:

O código de bloco de ação acima verifica se uma variável global é true. Se for então o elemento sequence referenciado na variável "my_seq" é pausado, caso contrário é definido para jogar.
