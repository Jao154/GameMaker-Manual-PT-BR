Criar Sequência

Com esta ação você pode criar um elemento sequence de qualquer sequence a partir do navegador Asset. Você dá o nome do sequence e uma posição x/y dentro do room onde ele deve ser criado, bem como a camada em que ele deve ser criado (esta camada deve ser uma camada patrimonial). Você pode marcar o "relativo" flags para posicionar o sequence em relação à instância que está executando os blocos de código de ação. Sempre que você criar um sequence no jogo desta forma, ele receberá um valor de identificação do elemento único com o qual você poderá então identificar o sequence em outras ações, portanto esta ação permite que você defina uma variável "alvo" para manter este valor de identificação (como mostrado no exemplo abaixo). Note que o alvo pode ser marcado como uma variável temporária (local), que será criada para a ação e pode ser usada em todas as ações subseqüentes para o evento em que a ação estiver.
