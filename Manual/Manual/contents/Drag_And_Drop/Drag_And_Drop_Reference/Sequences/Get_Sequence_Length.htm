Obter Sequence Comprimento

Com esta ação você pode recuperar o comprimento (em quadros) do elemento dado sequence no room. Você fornece o ID do elemento sequence (como retornado pela ação Criar Sequência), e esta ação retornará seu comprimento na variável alvo. Este será o número de quadros para os quais o Sequence funciona.

Observe que a variável alvo pode ser marcada como temporária (local), que será criada para a ação e pode ser usada em todas as ações subseqüentes para o evento em que a ação estiver.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Variable A identificação do elemento sequence para obter o comprimento de.
Target A variável alvo para segurar o comprimento.

Exemplo:

O bloco de ação acima recupera o comprimento do ID do elemento sequence armazenado na variável "seq_id". Esse valor é então usado para ajustar a posição da cabeça do mesmo elemento sequence para 10 quadros antes de seu fim.
