Se Sequence existir

Esta ação pode ser usada para verificar se o elemento dado sequence existe no jogo room. Você dá o elemento ID, como retornado pela ação Criar Sequência, e se o elemento sequence existe no room então a ação retornará true caso contrário retornará false.

Se você flag o argumento "Não", então a ação verificará se não existe tal elemento sequence e se nenhum for encontrado, ele retornará true.
