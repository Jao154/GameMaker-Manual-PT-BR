Fonte do conjunto

Com esta ação você pode configurar o font para usar em todas as ações de texto de sorteio subseqüentes. Clicando no diálogo abrirá a seguinte janela onde você poderá então selecionar o font que deseja utilizar:

Note que isto irá definir o font para usar em todas as chamadas de texto de sorteio subseqüentes, e portanto pode ser chamado uma vez no início do jogo - em qualquer caso, não apenas no evento do sorteio - e todas as ações de texto do sorteio usarão o conjunto font. No entanto, se você precisar de múltiplos fonts em seu jogo, então você precisará chamar esta ação antes de sortear. qualquer texto com o necessário font.
