Rectângulo de Gradiente de Desenho

Esta ação desenhará um retângulo em uma determinada posição dentro do room, usando um conjunto de cores de mistura para criar um gradiente. Você dá a posição superior esquerda e a posição inferior direita e o retângulo será desenhado entre eles, e a posição pode seja uma posição absoluta dentro do room, ou uma posição relativa à instância que chama a ação. Em seguida, você define as cores para misturar de cada um dos cantos e também pode definir se o retângulo pode ser preenchido ou delineado, verificando o preencha a caixa no fundo.

NOTA: Esta ação é apenas para uso nos diversos eventos de sorteio, e não desenhará nada se for usada em outro lugar.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Left A posição esquerda para começar a desenhar a partir de
Top A posição superior para começar a desenhar a partir de
Right A posição certa para terminar o desenho a partir de
Bottom A posição inferior para o desenho final a partir de
Top Left A cor para misturar a partir do canto superior esquerdo
Top Right A cor para misturar a partir do canto superior direito
Bottom Left A cor para misturar a partir do canto inferior esquerdo
Bottom Right A cor para misturar a partir do canto inferior direito
