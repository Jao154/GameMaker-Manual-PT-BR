Desenho Sprite Transformado

Esta função irá desenhar o dado sprite semelhante à ação Draw Sprite mas com opções adicionais para alterar a escala, mistura, rotação e moldura do sprite que está sendo desenhado. A alteração destes valores não modifica o recurso de qualquer forma (apenas como ele é desenhado), e você pode usar qualquer uma das variáveis de propriedade sprite disponíveis em vez de valores diretos para todos os argumentos na função. A imagem abaixo ilustra como valores diferentes afetam o desenho do duende:

NOTA A mistura de cores só é recomendada para o alvo HTML5 quando o WebGL está habilitado, embora você ainda possa definir a cor da mistura se ela não estiver habilitada e ela irá misturar o sprite como normal. Entretanto, toda mistura desta forma cria uma duplicata sprite que é então armazenado no site cache e utilizado quando necessário. Isto está longe de ser ideal e se você usar mudanças múltiplas de cor, isso diminuirá o desempenho de seus jogos a menos que você ative o WebGL. se você não quiser usar o WebGL, então você pode definir o tamanho font cache para tentar limitar isso, caso seja necessário, utilizando a função sprite_set_cache_size().

NOTA: Esta ação é apenas para uso nos diversos eventos de sorteio, e não desenhará nada se for usada em outro lugar.

Argumentos:

Argumento Descrição
Sprite O sprite para desenhar
Frame O quadro sprite para desenhar (use a variável embutida image_index para o quadro atual)
X A posição x a ser desenhada dentro da sala
Y A posição y para desenhar dentro da sala
Scale X O fator de escala para o eixo X (use a variável incorporada image_xscale para a escala X atual)
Scale Y O fator de escala para o eixo Y (use a variável incorporada image_yscale para a escala Y atual)
Rotation O ângulo de tração para o sprite (de 0 a 360, girando no sentido anti-horário onde 0 é para a direita - use a variável embutida image_angle para a rotação atual)
Colour A cor a ser misturada com o sprite (o padrão é branco, e você pode usar a variável embutida image_blend para a cor atual)
