Sprite de sorteio

Esta ação desenhará um sprite em algum lugar no room. Você especifica o sprite, a posição (ou uma posição absoluta dentro do room, ou uma posição relativa à instância que faz o desenho) e o quadro de animação do sprite, se ele tiver múltiplos quadros (observe que os quadros de imagem são numerados a partir de 0 para cima). Se você tiver um sprite animado e quiser que ele desenhe os quadros como se tivesse sido atribuído ao object, então use -1 para o índice de quadros ou a variável construída em instância image_index, mas observe que se a instância não tiver um sprite atribuído, então ela não animará, e se o sprite atribuído tiver um número de quadros diferente daquele que está sendo desenhado, então o sprite será desenhado com quadros extras ou quadros perdidos.

IMPORTANTE: Esta ação não atrairá o sprite transformado, mesmo que você tenha mudado a mistura ou a escala da imagem. Para isso, veja a ação Desenhe Sprite transformado.

NOTA: Esta ação é apenas para uso nos diversos eventos de sorteio, e não desenhará nada se for usada em outro lugar.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Sprite O sprite para desenhar
x A posição x para desenhar em
y A posição y para desenhar em
Frame Index O índice da moldura a desenhar
