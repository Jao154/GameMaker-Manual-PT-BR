Declare uma nova função

Esta ação pode ser usada para criar uma nova função definida pelo usuário, que pode ser chamada posteriormente em qualquer momento em seu código. Uma função definida pelo usuário é aquela que você define para fazer uma tarefa que não é coberta pelas ações visuais incorporadas GML ou pelas funções de corredor GML, e pode ser global, por exemplo ou local no escopo (veja aqui para mais informações sobre escopo); elas também podem ser usadas como construtoras para criar estruturas. Para saber mais sobre o uso desta ação para criar funções globais, consulte a seção sobre Criação de funções de ação.

Para usar esta ação, você pode adicioná-la a um evento ou a um roteiro e dar um nome à nova função. Em seguida, você fornece os argumentos/parâmetros necessários que a função terá, e os valores padrão para quaisquer argumentos opcionais.

Se a função deve ser usada como construtor (que é usada para criar um struct), então você precisa verificar a opção Construtor e posteriormente usar a Nova ação para criar um struct a partir da função. Você também pode flag a função como sendo estática, o que significa que ela só será definida uma vez, não importa quantas vezes esta ação for executada. Finalmente, se a função só vai ser usada localmente no mesmo evento e em nenhum outro lugar, então você pode habilitar a opção Temp .

Sintaxe de ação:

Argumentos:

Argumento Descrição
Name O nome da nova função.
Argument [+] Os diferentes argumentos que podem ser passados para a função, e seus valores padrão, se opcional.
Constructor Se esta função é um construtor.
Static Se esta função é estática.
Temp Se a função é temporária (o que significa que só é utilizada dentro do evento atual).
