Novas estruturas

Esta ação é utilizada apenas em ao criar structs usando uma função que foi marcada como uma função construtora (ver Declarar uma nova função para mais informações). Esta ação adota o método de função para uma função previamente definida, bem como uma série de argumentos, que você pode expandir para adicionar mais, se necessário, usando o ícone , à esquerda. Estes argumentos devem corresponder às entradas exigidas pela função, e serão usados para preencher o struct que está sendo criado. O struct será retornado à variável alvo, que pode ser marcada como uma variável local temporária ou não.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Function O nome do script ou função definida pelo usuário a ser chamada.
Argument0 ... Argument3 Os diferentes argumentos (valores) que devem ser passados ao script ou função (argumentos não utilizados podem ser deixados em branco)
Target O nome da variável que deve ser alvo de qualquer valor retornado (pode ser deixado em branco)
