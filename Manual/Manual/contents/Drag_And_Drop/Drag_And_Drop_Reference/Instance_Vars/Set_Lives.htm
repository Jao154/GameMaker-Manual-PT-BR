Conjunto de Vidas

Com esta ação você pode criar ou definir uma variável de instância para vidas. Você fornece o valor para definir a variável vidas ou para adicionar à variável vidas (se você marcar a caixa de seleção relativa, então o valor será adicionado à "vidas" atual. valor). Note que ao contrário das variáveis de instância regulares, o nome da variável para "vidas" é armazenado internamente, de modo que para recuperar o valor da pontuação você precisa usar a ação Get Lives.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Lives A quantidade para definir as vidas (ou para adicionar às vidas se parente for verificado)

O código de bloco de ação acima verificará se o room é o primeiro no jogo e se é ele estabelece as variáveis "pontuação", "vidas" e "saúde", por exemplo.
