Se a instância existir

Esta ação pode ser usada para verificar se existe uma instância específica object no jogo room, ou se existe uma instância específica em si. Você dá o object do explorador asset, e se houver alguma instância no room, então a ação retornará true caso contrário, ela retornará false. Note que você pode usar uma variável aqui com o ID único para uma instância específica (por exemplo, como retornado pela ação Criar Object Instância) ou mesmo o ID da instância conforme retornado pelo editor room quando você adicionou a instância.

Se você flag o argumento "Não", então a ação verificará se não existe nenhuma instância e se nenhuma for encontrada ela retornará true e se mesmo uma existir ela retornará false.
