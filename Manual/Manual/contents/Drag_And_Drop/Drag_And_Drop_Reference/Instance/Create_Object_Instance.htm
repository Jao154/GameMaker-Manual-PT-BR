Criar Object Instância

Com esta ação, você pode criar uma instância de qualquer object a partir do navegador Asset. Você dá o nome do object e uma posição x/y dentro do room onde deve ser criada, bem como o nome da camada em que deve ser criada a instância (deve ser um Camada de Instância). Você pode marcar o "relativo" flags para posicionar a instância em relação à instância que está executando os blocos de código de ação. Sempre que você cria uma instância no jogo, ela recebe um valor de identificação único com o qual você pode então identificar a instância em outras ações, portanto, esta ação permite definir uma variável "alvo" para manter este valor de identificação (como mostrado no exemplo abaixo). Note que o alvo pode ser marcado como uma variável temporária (local), que ser criado para a ação e pode ser usado em todas as ações subseqüentes.

Argumentos:

Argumento Descrição
Object O nome do object para criar uma instância de.
x A posição x no room para criar a instância (pode ser relativa à instância de chamada).
y A posição y no room para criar a instância (pode ser relativa à instância de chamada).
Layer O nome (a string) da camada de instância para criar a instância em.
Target A variável a ser alvo para o valor de retorno.
