Linha do tempo da instância definida

Esta ação atribui um timeline do Asset Browser à instância. Você dá o timeline para atribuir, e então você pode escolher ter o timeline loop quando ele chegar ao último momento (o próximo quadro após o último momento será loop de volta para o primeiro momento do timeline novamente), bem como se o timeline deve ser iniciado imediatamente ou não. Se você não iniciar o timeline imediatamente, poderá usar o conjunto de ações Timeline para iniciar mais tarde em outro evento ou script.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Timeline O recurso timeline para atribuir a uma instância
Loop Bandeira para ter o timeline loop após o último momento (a bandeira padrão está sinalizada)
Start Timeline Bandeira para que o timeline comece imediatamente ou não (a bandeira padrão está sinalizada)

O código de bloco de ação acima atribuirá um timeline à chamada de instância as ações assim como definir o momento inicial para 10 e depois a velocidade timeline para 2.
