Conjunto Timeline Momento

Esta ação será usada para pular um timeline do momento atual para um novo. Timelines avançar através de "momentos" diferentes, e cada momento normalmente ocupa um quadro de jogo (como o Step Event de um object). Uma vez que um timeline tenha começado progredirá através dos momentos até chegar ao final, altura em que loop voltará ao primeiro momento no timeline ou parará completamente. Entretanto, esta abordagem linear de timelines pode ser modificada usando esta ação para definir um a qualquer momento, permitindo que você crie seu próprio timeline loops, ou pular seções de um timeline sob certas circunstâncias, etc... Com esta ação você simplesmente define o momento para pular para dentro do timeline e o próximo quadro de jogo correrá naquele momento (note que você precisa ter atribuído um timeline à instância utilizando a ação Set Instance Timeline).
