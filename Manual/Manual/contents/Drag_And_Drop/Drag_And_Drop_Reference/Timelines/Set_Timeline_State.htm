Conjunto Timeline Estado

Esta ação pode ser usada para iniciar, parar, pausar ou retomar um timeline. O timeline deve primeiro ter sido definido para a instância usando a ação Definir Linha do Tempo da Instância, e depois você pode usar esta ação a qualquer momento para mudar seu estado de funcionamento a partir das seguintes opções

Opção Descrição
Início Iniciar o timeline designado para a instância (se o timeline foi pausado, isto colocará a posição inicial no momento 0 novamente).
Currículo Retomar uma pausa timeline (se o timeline foi interrompido e não pausado, isto não fará nada).
Pausado Pausa o timeline designado para a instância (enquanto pausa os momentos em um timeline não avançará com os quadros do jogo).
Parada Pare o timeline designado para a instância.

Argumentos:

Argumento Descrição
State O estado para colocar o timeline (veja a tabela acima)

O código de bloco de ação acima verificará o valor mantido em uma variável global e se for true então o timeline é pausado, e se for false então o timeline será retomado.
