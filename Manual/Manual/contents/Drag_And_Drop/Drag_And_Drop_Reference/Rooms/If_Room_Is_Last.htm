Se Room for o último

Esta ação verificará se o atual room é o último room, conforme listado no Room Manager. A ação retornará true se o atual room for o último no Gerente Room e false se não for. Você pode flag a caixa de verificação " não" para transformar isto em "Se o Room NÃO for o último".

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Essas ações serão executadas agora se o "se" for avaliado para true, enquanto quaisquer ações descartado em outro lugar será realizado após o bloco "se".
