Se [Fonte de tempo] Estado

Esta ação lhe permite controlar outras ações com base no estado da Fonte de Tempo dada.

Ele verifica se o Fonte de tempo Estado Constante dado em "Estado" é (ou não é) o estado da Fonte de Tempo dado em "Fonte de Tempo".

Argumentos:

Not Permitir que isso inverta a condição
State O Fonte de tempo Estado Constante para verificar contra

Exemplo:


Este bloco de código seria idealmente executado em um evento da Key Press.

Ele verifica se o estado de uma Fonte de Tempo está "ativo", e nesse caso, ele faz uma pausa nessa Fonte de Tempo.

Entretanto, se o estado for "pausado", ele inicia novamente a Fonte de Tempo.
