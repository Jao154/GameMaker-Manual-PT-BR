Criar fonte de tempo

Esta ação cria uma nova Fonte de Tempo, e a armazena na variável Alvo. Leia Time Sources para obter uma visão geral.

Isto é baseado no time_source_create() função. Leia sua página para obter informações detalhadas sobre os argumentos.

Você precisa usar a Start Time Source para ativar uma Fonte de Tempo após sua criação.

Você deve destruir uma Fonte de Tempo usando Destroy Time Source quando não precisar mais dela.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Target A variável onde a nova Fonte de Tempo será armazenada
Parent A fonte de tempo pai: ou uma fonte de tempo embutida ou uma fonte personalizada
Period A duração do período da Fonte de Tempo, quanto tempo leva para expirar
Units As unidades em que o período é expresso (segundos ou quadros)
Callback O method ou função de roteiro para chamar quando a Fonte de Tempo expirar
Arguments OPCIONAL Uma matriz contendo os argumentos para passar para a função de retorno de chamada
Repetitions OPTATIVA Quantas vezes a Fonte de Tempo deve funcionar no total, ou -1 para repetição indefinida
Expiry Type OPCIONAL O tipo de expiração para a Fonte de Tempo

Exemplo:

Este bloco de código faz o seguinte:

  • Ele cria uma nova função, com a opção "Temp" ativada. Isto cria uma variável de método local.
    • Este método leva um argumento, msg, que ele imprime para o Diário de Saída.
  • Em seguida, cria uma nova Fonte de Tempo com um período de 1 segundo de duração.
    • Ela especifica a variável callback_method como a "Callback", e um array com um argumento para a função.
    • Esta Fonte de Tempo é definida para repetir indefinidamente, como -1 é especificado em "Repetições".
  • A Fonte de Tempo é então iniciada.

Esta Fonte de Tempo executará seu retorno a cada segundo, imprimindo "Um segundo passou!" para o Diário de Saída.
