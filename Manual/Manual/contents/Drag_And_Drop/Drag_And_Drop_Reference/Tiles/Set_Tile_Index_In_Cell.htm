Conjunto Tile Índice Na Célula

Com esta ação você pode definir um tile em uma determinada camada para um novo índice. A camada é especificada a partir do nome da camada dada (um string, como usado para nomear a camada no editor Room ), e então você dá a linha e a coluna da célula dentro da camada para obter o tile a definir. Finalmente você fornece o índice tile para definir o tile para (tiles são indexados da esquerda para a direita de cima para baixo, com 0 na parte superior esquerda). Note que você sempre pode definir o índice tile com 0 para limpar o tile.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada para colocar o tile em
Column A coluna (da esquerda para a direita) da célula
Row A fila (de cima para baixo) da célula
Tile O índice tile para definir a célula para

Se não for o índice tile para essa célula está em 0 e a variável temporária (local) é incrementada antes que o loop faça outra conjunto de verificações.
