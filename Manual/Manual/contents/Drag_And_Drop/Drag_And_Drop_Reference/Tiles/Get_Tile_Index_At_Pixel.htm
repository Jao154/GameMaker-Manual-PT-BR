Obter Tile Índice no Pixel

Esta ação é utilizada para obter um índice tile em uma determinada camada. A camada é especificada a partir do nome da camada dada (um string, como usado para nomear a camada no editor Room ), e então você dá uma posição x e y para encontrar o tile para obter o índice de. O azulejo será aquela em que a posição dada se enquadra no room, e o índice será devolvido à variável alvo, onde 0 não é tile, e um inteiro positivo é o índice do tile. A posição x e y pode ser assinalada como sendo relativa a a posição da instância de chamada, e a variável alvo pode ser marcada como uma variável temporária (local) que pode ser usada até o final do evento.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada para colocar o tile em
x A posição x para verificar
y A posição y para verificar
Target A variável-alvo para armazenar o índice tile

Exemplo:

O código de bloco de ação acima verifica se há um botão esquerdo do mouse e, se for detectado, obtém-se o índice da célula tile em que a posição do mouse se encaixa. Este índice de tile é então verificado e se não for 0, é ajustado para 0.
