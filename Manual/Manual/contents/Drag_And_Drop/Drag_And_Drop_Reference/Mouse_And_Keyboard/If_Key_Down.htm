Se a Key Down

Com esta ação você pode pesquisar o teclado para ver se alguma tecla está sendo pressionada no momento. A ação retornará true a cada passo que a tecla do teclado estiver sendo mantida pressionada, ou false caso contrário, embora se você clicar no modificador not você poderá então verificar se a tecla não está sendo ajudada, ou seja: a ação retornará true enquanto nenhuma tecla estiver pressionada e false se houver. Se você só precisa verificar se há uma única tecla pressionada, então use a ação Se a tecla for pressionada.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Qualquer ação abandonada em outro lugar será executada após o "se". bloco.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Key O nome da tecla do teclado a ser verificado.
Not Negar o cheque (o verdadeiro torna-se falso e vice-versa)

Exemplo:

O código de bloco de ação acima pesquisa o estado do teclado a cada passo e se uma tecla está sendo mantido pressionado, verifica se a tecla inicial está pressionada. Se a verificação for true na prensa inicial, um som é tocado, então, enquanto a tecla é mantida pressionada, a mistura de sprite é mudada para vermelho. Se a tecla não estiver sendo mantida pressionada, uma verificação é feito na liberação da chave para repor a cor da mistura para branco.
