Se o mouse for pressionado

Esta ação verifica se o botão inicial do mouse está pressionado. Ela retornará true na etapa de jogo único (frame) em que o botão é inicialmente pressionado e false em todos os outros momentos. Se você ativar o modificador not, esta ação será revertida, e verificará se o botão não foi pressionado, retornando false no passo do jogo (quadro) que foi pressionado e true em todos os outros momentos... Se você precisar verificar se um botão foi pressionado e não apenas o evento inicial de pressão, então use a ação If Mouse Down.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão executadas agora se o "se" for avaliado para true, enquanto quaisquer ações descartadas em outro lugar serão executadas após o bloco "se".

Exemplo:

O código de bloqueio de ação acima faz com que o botão do mouse se movimente a cada passo e, se estiver sendo mantido pressionado, ele verifica se o mouse está pressionado inicialmente. Se a verificação for true na descida inicial pressione a velocidade da instância é definida, então, enquanto o botão está pressionado, a mistura sprite é mudada para vermelho. Se o mouse não estiver sendo pressionado, uma verificação é feita na liberação do mouse para redefinir a cor da mistura para branco e ajustar a velocidade para 0.
