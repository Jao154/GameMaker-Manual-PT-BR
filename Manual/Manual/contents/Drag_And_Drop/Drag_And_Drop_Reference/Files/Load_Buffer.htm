Tampão de carga

Com esta ação, você pode carregar um buffer que foi salvo anteriormente com a ação Save Buffer. A ação criará um novo buffer, carregará o arquivo de dados nele, e retornará o valor de identificação único para identificar o buffer para o variável. Você fornece o nome do arquivo a ser carregado (como um string) e uma variável para armazenar o ID buffer retornado. Observe que se você flag a variável como uma variável local temporária, você precisa ter certeza de excluir o buffer que esta função cria antes o fim da ação script ou evento, caso contrário você perderá a referência a ele e obterá um vazamento de memória quando a variável for descartada.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Filename O nome (como um string) do arquivo a ser aberto
Target A variável alvo para manter o valor de identificação único para o buffer criado em carga

Exemplo:

O código de bloco de ação acima criará uma variável para conter dados buffer e então verificará se um buffer salvar arquivo já foi feito anteriormente. Se o arquivo existe, então ele é carregado na variável buffer que criamos, mas se não existir, ele é criado, escrito e então salvo pronto para ser carregado na próxima vez.
