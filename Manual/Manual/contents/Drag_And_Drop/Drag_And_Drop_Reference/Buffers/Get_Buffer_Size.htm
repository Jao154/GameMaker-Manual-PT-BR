Obter Buffer Tamanho

Esta ação pode ser usada para recuperar o tamanho de um buffer, em bytes. Você fornece o valor único de ID buffer como retornado pela ação Criar Tampão e então dá uma variável alvo para retornar o valor do tamanho. A meta pode ser marcada como sendo uma variável local temporária, caso em que será criada para você e removida da memória novamente no final da ação atual script ou evento.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Buffer O índice buffer (armazenado em uma variável)
Target A variável alvo para manter o valor retornado

Exemplo:

O código de bloco de ação acima obtém o tamanho do buffer e então usa esse valor para definir o posição de leitura/escrita até o final dos dados atualmente armazenados.
