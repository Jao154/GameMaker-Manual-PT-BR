Obter Buffer Posição

Esta ação pode ser usada para recuperar a posição atual para leitura/escrita no buffer. Quando você cria um buffer, a posição de leitura/escrita é definida em 0 bytes, e à medida que você adiciona dados ao buffer esta posição se moverá sequencialmente, o que significa que você pode usar esta ação para obter o número de bytes no buffer que está definido a qualquer momento. Você fornece o valor único de ID buffer conforme retornado pela ação Criar Buffer e depois dar uma variável alvo para retornar o valor da posição para. A variável alvo pode ser sinalizada como sendo uma variável local temporária, caso em que será criada para você e removida da memória novamente ao final da ação atual script ou evento.

Argumentos:

Argumento Descrição
Buffer O índice buffer (armazenado em uma variável)
Position A posição de leitura/escrita (em bytes) dentro do buffer
