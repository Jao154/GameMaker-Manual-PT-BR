Procure um tampão

Esta ação pode ser usada para mover (procurar) para uma posição dentro de um buffer. Você fornece o valor de identificação único buffer como retornado pela ação Criar Tampão e depois o valor de compensação em bytes para se mover. O valor de offset será ser aplicado desde o início do buffer, portanto, chamando esta função moverá a posição de leitura/escrita para o início do buffer e, em seguida, moverá o número de bytes dados e posicionará a nova posição de leitura/escrita lá. Entretanto, se você sinalizar o offset como relativo, ele moverá a posição de leitura/escrita para frente (ou para trás se o offset for um valor negativo) a partir da posição atual.

Argumentos:

Argumento Descrição
Buffer O índice buffer (armazenado em uma variável)
offset A compensação (em bytes) em relação à posição atual de busca
