Leia Buffer

Esta ação pode ser usada para ler dados de um buffer previamente criado. Você fornece o valor único de ID buffer (como retornado pela ação Criar Tampão) e dá uma variável para armazenar o valor retornado. Se você flag a variável como uma variável local temporária, então ela será criada para você e descartada novamente no final do evento ou ação script. Todos os dados são devolvidos como uma string, portanto, se você tiver armazenado algum número real, você terá que usar uma das Ações do Tipo de Dados para convertê-lo do string para o valor. Tenha em mente também que buffers é lido e escrito sequencialmente e assim a leitura de um buffer moverá a posição "procurar" para a próxima entrada, o que significa que para ler vários pedaços de dados você simplesmente chama esta ação de número de vezes necessário e armazena cada dado retornado em uma variável.

Argumentos:
