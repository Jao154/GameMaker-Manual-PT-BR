Buffer de escrita

Esta ação pode ser usada para escrever dados em um buffer. Você fornece o ID único buffer para escrever (como retornado pela ação Criar Tampão) e depois o valor para escrever, que pode ser um real ou um string, mas note que todos os valores são armazenados como strings para que quando você ler a partir do buffer (usando o Read Buffer) você possa precisar usar uma das ações do tipo de dados para convertê-lo de uma string de volta ao valor original. Tenha em mente também que buffers é lido e escrito para seqüencialmente e assim escrever para um buffer irá mover a posição "buscar" ao longo de uma, significando que para escrever vários dados você simplesmente chama esta ação o número de vezes necessário.

Argumentos:

Argumento Descrição
