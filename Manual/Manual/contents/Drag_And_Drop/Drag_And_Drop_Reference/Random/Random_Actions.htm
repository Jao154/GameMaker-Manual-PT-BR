Vale notar que o GameMaker utilizará a mesma semente ao gerar números aleatórios cada vez que você executar o jogo a partir do IDE. Isto é para facilitar debugging e significa que você obterá os mesmos resultados iniciais ao gerar valores aleatórios, mas isto não ocorrerá quando você tiver compilado o jogo em um executável (uma nova semente será gerada cada vez que o jogo for executado). Se você não desejar este comportamento ao testar, então você deve chamar a ação Randomise uma vez no início do jogo, antes de gerar quaisquer valores aleatórios.
