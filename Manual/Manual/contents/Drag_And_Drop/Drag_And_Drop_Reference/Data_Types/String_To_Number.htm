String To Number

Esta ação pode converter um string em um número. Você fornece o string para mudar e depois dá uma variável alvo para retornar o número real (esta variável pode ser marcada como uma variável local temporária). Observe que se o string tiver algum caractere além de números e um ponto decimal, não será convertido, embora se o string começar com um número, estes serão convertidos e o restante do string será excluído. É importante notar que ao passar decimais para este ação, o retorno string será arredondado para apenas duas casas decimais. Abaixo está uma tabela de valores de entrada e o que você esperaria que fosse retornado

Cordão Valor retornado
"123" 123
"123 Hello World" 123
"Hello World" 0 (por padrão, se o string não puder ser convertido, 0 será devolvido)
"Hello World 123" 0
"123.45678" 123.46

Exemplo:

O código de bloco de ação acima inicia uma variável global e depois carrega um arquivo com um buffer guardado nele. Dois strings são lidos do buffer e armazenados em uma variável local temporária antes que uma delas seja convertida em um número real e armazenada em outra variável local temporária. Estas são então atribuídas a variáveis globais, e o buffer é excluído.
