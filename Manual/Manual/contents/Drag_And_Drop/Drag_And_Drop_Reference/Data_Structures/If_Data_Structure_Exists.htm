Se a estrutura de dados existir

Com esta ação você pode verificar uma variável para ver se ela indexa ou não uma estrutura de dados. Primeiro você precisa escolher o tipo de estrutura de dados a ser verificada (uma Lista, um Mapa, uma Grade ou uma Pilha) e então você fornece a variável que contém a estrutura de dados (como retornado pelas ações Criar lista, Criar mapa, Criar grade e Criar pilha). A ação retornará true se a estrutura de dados existir, e false se não existir. Observe que você pode flag "não" na ação para verificar se uma variável não possui uma estrutura de índice de dados.
