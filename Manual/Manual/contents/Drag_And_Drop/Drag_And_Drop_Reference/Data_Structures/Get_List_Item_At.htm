Obter Item da Lista em

Com esta ação, você pode recuperar a posição indexada de um item dentro da lista fornecida. Você fornece a variável que armazena o índice da lista (conforme retornado pela ação Criar Lista) e a posição do índice dentro da lista para obter o valor do item, assim como uma variável alvo para armazenar o valor do item devolvido (que pode ser marcada como uma variável local temporária a ser usada até o final do script ou evento).

OBSERVAÇÃO: Se você der uma posição fora do tamanho da lista dada (ou seja: posição 11 em uma lista de 10 valores), então a ação retornará undefined. Você pode verificar isso usando a ação If Undefined (Se Indefinido ).

Sintaxe de ação:

Argumentos:

Argumento Descrição
List O índice (armazenado em uma variável) da lista para remover de
Index A posição do índice dentro da lista para obter o item de
Target A variável alvo para armazenar o valor retornado

Exemplo:

O código de bloco de ação acima obtém o tamanho da lista referenciada na variável global e então usa um para loop para loop através dele e destruir as instâncias com as identificações armazenadas na lista. Após a conclusão do loop, a lista é liberada da memória.
