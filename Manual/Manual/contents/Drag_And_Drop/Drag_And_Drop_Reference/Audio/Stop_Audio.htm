Parar áudio

Esta ação pode ser usada para impedir um som de tocar. Você seleciona o som do explorador asset e todas as instâncias desse som deixarão de tocar (portanto, se você definiu o mesmo som para tocar em três eventos diferentes, todos eles irão parar).

Sintaxe de ação:

Argumentos:

Argumento Descrição
Sound O bom recurso para parar

Exemplo:

O código de bloqueio de ação acima verifica se o botão do mouse está pressionado e, se for detectado, verifica um variável global para ver se ela não é verdadeira. Se não for verdade, então um som é tocado (e loopado) e a variável global é definida para true, caso contrário o som é parado e a variável global é definida para false.
