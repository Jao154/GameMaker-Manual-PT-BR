Obter volume de áudio

Esta ação devolve o volume do som dado. Você escolhe o som do explorador asset e dá uma variável alvo para retornar o valor do volume para (você pode flag isto como sendo uma variável local temporária). O valor retornado estará entre 0 e 1, onde 0 é silencioso e 1 é volume total.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Sound O recurso sólido para obter o volume de
Target A variável alvo para armazenar o volume retornado

Exemplo:

O código de bloco de ação acima verifica uma variável global para ver se ela é true ou falsa. Se for true, então o volume do som dado é recuperado e armazenado em uma variável local temporária. O valor é então verificado para ver se ele não é igual a 0,5, e se não for, então o volume do som é definido como 0,5. Se a variável global é avaliada para false, então o som tem seu volume ajustado para 1.
