Se o botão Gamepad fosse pressionado

Com esta ação, você pode pesquisar o gamepad para ver se algum botão está sendo pressionado no momento. A ação retornará true a cada passo que o botão do gamepad estiver sendo pressionado (e mantido pressionado) para baixo, ou false caso contrário, embora se você clicar no botão not modificador você pode então verificar se o botão não está sendo pressionado, ou seja: a ação retornará true enquanto nenhum botão for pressionado e false se houver. Se você só precisa verificar se há um único botão, pressione então use a ação se o botão do gamepad estiver desligado.

Se o botão que está sendo verificado for um botão analógico, então a pressão não será detectada até que o valor do botão tenha ultrapassado o limite definido usando a ação Set Gamepad Button Threshold.

Note que para adicionar ações no bloco "se", elas devem ser descartadas ao lado da ação, como mostrado na imagem abaixo:

Estas ações serão agora executadas se o O "se" avalia para true, enquanto qualquer ação descartada em outro lugar será executada após o bloco "se".

Sintaxe de ação:

Argumentos:

Argumento Descrição
Gamepad O índice gamepad.
Button O nome do botão a ser verificado.
Not Negar o cheque (o verdadeiro torna-se falso e vice-versa)

Exemplo:

O código de bloco de ação acima pesquisa o estado do botão do gamepad 0 a cada passo e se um botão estiver sendo mantido pressionado, ele verifica se o botão inicial está pressionado. Se a verificação for true na pressão inicial para baixo, um som é reproduzido, então, enquanto o botão é mantido pressionado para baixo, a velocidade da instância é ajustada para 10. Se o botão não estiver sendo pressionado, a velocidade é ajustada para 10. mantido pressionado, é feita uma verificação no botão de liberação para repor a velocidade em 0 novamente.
