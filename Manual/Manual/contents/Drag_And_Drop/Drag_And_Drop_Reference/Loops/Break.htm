Intervalo

Esta ação só pode ser usada dentro de uma ação loop (por exemplo, Repetir ou Enquanto), e chamá-la será liberada do loop naquele momento. Normalmente, um loop funcionará ou um número fixo de vezes ou até uma condição específica é cumprida, mas você pode usar a ação de Intervalo para terminar prematuramente um loop a qualquer momento (ações dentro do loop após o Intervalo não serão executadas).
