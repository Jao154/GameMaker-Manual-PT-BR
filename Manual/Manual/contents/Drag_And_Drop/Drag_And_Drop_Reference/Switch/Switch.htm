Troca

Esta ação criará uma nova declaração Switch à qual você poderá então acrescentar outros casos para avaliação. Você dá o valor que será usado para comparação com os casos no Switch e então adiciona as diferentes ações de casos, muito como você acrescentaria uma ação a um "se", ou seja: deixá-la cair para o lado da ação em vez de sob ela:

Note que você não pode adicionar qualquer outro tipo de ação em um switch sem antes ter adicionado pelo menos um Estojo para contê-lo.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Value O valor a ser usado pelo switch para avaliação

Exemplo:

O código de bloco de ação acima recebe o valor armazenado em uma variável e depois o compara com os diferentes possíveis casos. Se qualquer um dos casos for igual ao valor, então a ação sob esse caso será executada, caso contrário, o próximo caso será avaliado. Após o switch ter sido executado, um alarme é ajustado para 30.
