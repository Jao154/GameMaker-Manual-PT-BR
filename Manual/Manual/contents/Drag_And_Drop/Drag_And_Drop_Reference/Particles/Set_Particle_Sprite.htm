Set Particle Sprite

Esta ação pode ser usada para definir o sprite que uma partícula pode usar ao desenhar. Você fornece o valor de identificação único para o tipo para definir o sprite de (como retornado quando você criou o tipo com a ação Criar tipo de partícula) e o recurso sprite a ser utilizado. Se o sprite tiver sub-imagens, então a partícula animará através dos quadros na velocidade definida no Editor de Imagens para o sprite. Observe também que se você definir um valor de cor para a partícula (usando a ação Set Particle Colour ), então esta cor será misturada com o sprite usado, assim como definir a mistura da imagem em uma instância irá misturar uma cor com a instância sprite (use branco para não misturar).

NOTA: Se você usar esta ação, então não deve usar a ação Set Particle Shape, pois você só pode ter um sprite ou uma forma, mas não ambos.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Type O valor de identificação para o tipo a definir o sprite de
Sprite O sprite para atribuir ao tipo de partícula

Exemplo:

O código de bloco de ação acima criará um novo tipo de partícula e atribuirá seu exclusivo valor de identificação para uma variável global. Em seguida, ele passa a definir todas as propriedades para o tipo de partícula.
