Preferências doCódigo GML

As Preferências doCódigo GML são usadas para definir como algumas das características de linguagem para codificação serão exibidas no IDE do GameMaker. As opções para o Código GML são as seguintes:

  • Mostrar os tipos de argumentos na barra de status: Esta opção refere-se a quando você está usando seu próprio costume scripts e usou a notação JSDoc para identificar os diferentes argumentos, etc... (veja aqui para mais informações). Quando esta opção for marcada, e você tiver especificado os tipos de argumentos no script, eles serão mostrados na barra de status na parte inferior do editor, incluída em {}. Isto está desativado por padrão.
  • Exibir descrições de argumentos na barra de status: Se isso for verificado, então qualquer descrição que você tenha dado a scripts personalizada usando a notação JSDoc será mostrada na barra de status na parte inferior da janela do editor. Isto está desativado por padrão.
  • Mostrar a ortografia americana para as funções do sistema: Por padrão o GameMaker usa a ortografia inglesa GB para funções, por exemplo draw_text_colour(), porém isto pode não ser exatamente o que você está acostumado e assim você pode switch para ortografias dos EUA usando esta opção, de modo que as funções se tornem draw_text_color(), por exemplo. Isto está desativado por padrão.
  • Número máximo de erros de sintaxe a exibir: Este valor define o número máximo de erros de sintaxe que serão exibidos a qualquer momento. Se você achar que o IDE está atrasado, então você pode tentar reduzir este valor. Este valor é definido como 50 por padrão.
  • Habilitar avisos de referência variáveis: Quando ativado, isto irá flag qualquer problema potencial com variáveis, como quando uma foi definida em algum lugar, mas não é realmente utilizada no código. Quando desativado, estas advertências não serão mais mostradas. O padrão é que isto esteja habilitado.
  • Habilitar declaração dobrável como regiões: Quando ativado, parênteses em seu código são capazes de ser dobrados como regiões. Isto é ativado por padrão.
