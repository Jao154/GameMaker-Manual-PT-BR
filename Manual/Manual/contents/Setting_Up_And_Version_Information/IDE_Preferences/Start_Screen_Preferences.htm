Preferências da tela inicial

As Preferências da tela inicial permitem mudar o comportamento da página inicial. As principais opções são:

  • Turnê do show: Isto permite selecionar se a turnê introdutória é mostrada quando o IDE é iniciado. Isto é automaticamente desativado na primeira vez em que você pular a turnê.
  • Mostrar Assistente de Configuração: Isto permite que você selecione se o Assistente de Configuração é mostrado quando o IDE é iniciado. O Assistente de Configuração leva-o essencialmente através dos passos necessários para criar um novo projeto.
  • Estilo de Lista de Projetos Recentes: Isto permite mudar a exibição da lista de Projetos Recentes na página inicial entre "Lista" e "Grade".
