Preferências do editor Sprite

As Preferências do Editor Sprite são usadas para definir a aparência e o funcionamento do Editor Sprite. Existem as seguintes opções:

  • Padrão sprite origem: Esta opção permite que você defina a posição de origem padrão para qualquer novo recurso sprite que você criar. O valor padrão é o Top Left.
  • Padrão sprite tipo máscara de colisão: Esta opção permite definir o tipo padrão de máscara de colisão a ser usada para o novo sprites. As escolhas são retângulo, retângulo girado, elipse, diamante ou preciso / preciso por moldura. O retângulo padrão é o retângulo.
  • Padrão sprite modo de animação: Esta opção permite definir o modo de animação padrão para sprites no editor sprite. Você pode escolher entre " Frames por segundo" ou " Frames por frame de jogo", com a configuração padrão sendo "Quadros por segundo".
  • Velocidade de animação padrão sprite : Essa opção permite que você defina o valor padrão da velocidade de animação para quando você criar um novo sprite asset. Note que esse valor está ligado ao modo de animação e - por exemplo - 3 quadros por segundo. não é o mesmo que 30 quadros por quadro de jogo, portanto, certifique-se de definir este valor corretamente em referência ao modo preferido. O valor padrão é 30.
  • Largura padrão sprite : Esta opção permite definir a largura padrão para todos os novos sprites criados. O valor padrão é 64.
  • Padrão sprite altura: Esta opção permite que você defina a altura padrão para todos os novos sprites. O valor padrão é 64.
  • Aplicar redimensionamento e escala na origem sprite: Quando marcada esta opção mudará a origem do sprite quando o sprite for redimensionado ou escalonado, com a nova origem sendo movida/escalonada para combinar com os novos valores para o sprite. Note que isto só é aplicável para as origens personalizadas sprite que não estão bloqueadas no editor sprite. Se a origem estiver bloqueada ou o sprite usa uma das posições padrão - por exemplo: "Center" ou "Top Left", etc... - então esta opção ser cinzento e indisponível para mudança. O padrão está habilitado.

A seguinte sub-categoria também existe para o editor Sprite:

Confirmar Diálogos

As opções disponíveis aqui controlam as mensagens que aparecem quando são necessárias para confirmar uma ação. Você pode alterar estas configurações para mostrar sempre a mensagem ou rejeitá-la com uma ação específica (Sim ou Não) :

  • Resposta automática à importação de arquivos: Quando você importa uma nova imagem e uma já existe, por padrão você será solicitado a continuar importando e sobrescrever a imagem existente ou cancelar a operação. Esta opção permite que você suprima esta mensagem e sempre cancelar a importação selecionando "Não", ou sobrescrever o atual sprite selecionando "Sim". O valor padrão é "Mostrar mensagem".
  • Resposta automática à falha de carga de imagem: Quando você importa uma nova imagem e a importação falha, por padrão será exibida uma mensagem de aviso informando que houve uma falha. Esta opção lhe permite suprimir esta mensagem selecionando "Ok". O valor padrão é "Mostrar mensagem".
