Seqüências Preferências

As Preferências de Seqüências são usadas para definir certas propriedades para o funcionamento do Editor de Seqüências e seus elementos. As principais opções são:

  • Mostrar moldura de lona em seqüências recém-criadas: Quando ativado, o guia inteligente da moldura da tela será mostrado na tela do editor Sequence para todos os novos sequences. Esta opção também pode ser habilitada/desabilitada conforme necessário a partir das propriedades da tela no próprio editor. O valor padrão é ativado.
  • Largura da moldura de lona em seqüências recém-criadas: Aqui você pode definir a largura padrão da moldura para todos os novos sequences. O valor padrão é 1366px.
  • Altura da estrutura da tela em seqüências recém-criadas: Aqui você pode definir a altura padrão da moldura para todos os novos sequences. O valor padrão é 768px.
  • Pausa ao clicar sobre a tela: Quando esta opção for marcada, clicando em qualquer lugar na tela do editor Sequence, será feita uma pausa na reprodução do sequence. Isto é ativado por padrão.
  • Atraso na seleção da sobreposição em milissegundos: Isto controla quanto tempo você precisa esperar antes que a Item Toolbox apareça para os itens sobrepostos na tela do editor Sequence. O valor padrão é de 1000ms.
  • Tradução path margem de inserção de pontos: Na tela do editor Sequence, quando você tiver adicionado um asset e depois criado uma faixa de parâmetros de posição, haverá linhas pontilhadas desenhadas para demonstrar o path que o asset levará ao mover-se. Se você colocar seu mouse próximo a esta linha, então você pode adicionar um novo ponto de parâmetro de faixa a ela, e esta opção define a distância da linha de tradução que o mouse precisa estar antes que o marcador de ponto seja mostrado. O valor padrão é 6px.
  • Criar automaticamente faixas de parâmetros padrão: Quando ativada, esta opção adicionará automaticamente faixas de parâmetros para posição, escala, rotação e origem a quaisquer novas faixas criadas no Painel de Pistas do sequence. O padrão está desativado.
  • Esticar automaticamente os ativos recém-criados: Esta opção esticará todas as teclas recém-criadas asset para corresponder ao comprimento do sequence. O padrão está desativado.
  • Altura da curva do animal na folha de endoscopia: Esta opção pode ser usada para definir quantas linhas de pista parametrização de pistas que utilizam curvas irão ocupar a folha de dopw quando na visualização de keyframe. O padrão é 2, mas você pode selecionar 1 ou 3 também.
  • Texto padrão para nova faixa de texto: O texto string usado quando você cria uma nova trilha de texto em um Sequence.
  • Cor de seleção do inspetor: A cor utilizada para destacar a faixa selecionada na lista Inspector Multi-Select.

Confirmar Diálogos

As preferências do Confirm Dialogs são usadas para alterar as mensagens que são mostradas no editor sequence quando você executa certas ações. As mensagens que você pode alterar são:

  • Escolha automática para incorporar curva editada: Se você tentar embutir uma animation curve que você editou, será mostrada uma mensagem por padrão para confirmar a ação. No entanto, a configuração desta opção para "Sim" não mostrará a mensagem e incorporará toda vez. Definindo-a como "Não", impedirá a incorporação toda vez.
  • Escolha automática para converter curva em chaves: Se você tentar converter uma animation curve para keyframes, será exibida uma mensagem por padrão perguntando se você deseja continuar. No entanto, a configuração desta opção para "Ok" não mostrará a mensagem e simplesmente não permitirá que você converta a curva.

Guias inteligentes

Aqui você pode selecionar o comportamento dos guias inteligentes ao trabalhar no Editor Sequences. As opções disponíveis são:

  • Habilitar guias inteligentes: Aqui você pode habilitar ou desabilitar os guias inteligentes na tela do editor Sequences. Isto é habilitado por padrão.
  • Habilitar as Réguas: Aqui você pode ativar ou desativar os guias inteligentes na tela do editor Sequences. Isto é habilitado por padrão.
  • Réguas de corte para bordas: Esta opção permitirá que as réguas "estalem" até as bordas de assets na tela sequences (e vice-versa). O padrão está habilitado.
  • Distância instantânea: Esta é a distância (em pixels) que um item precisa ser para uma régua (ou uma régua para um item) para que ele "estale" para a posição. O valor padrão é 6px.
