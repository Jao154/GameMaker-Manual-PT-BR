Preferências do editor Spline

São utilizadas as Preferências do Editor Spline principalmente para definir o aspecto suave do paths no Path Editor, embora eles afetem certos outros aspectos da IDE (como a forma física Editor). Existem as seguintes opções:

  • Inserir path precisão do ponto (px): Isto define o valor de precisão (em pixels) para inserção de pontos em um path, ou seja: o mouse deve estar dentro deste intervalo para inserir o ponto em vez de adicionar um novo. O valor padrão é 20px.
  • Largura dos pontos de ancoragem: Defina a largura dos pontos de ancoragem, em pixels. O padrão é 8px;
  • Altura do ponto de ancoragem: Definir a altura dos pontos de ancoragem, em pixels. O padrão é 8px;
  • Cor do caminho: A cor a ser usada para mostrar o path enquanto você o edita. A cor padrão é ( ARGB) #FFFFFFFFFFFF.
  • Em destaque path cor do segmento: A cor a ser usada para destacar qualquer segmento path selecionado. A cor padrão é (ARGB) #FFFFFFFFFFFF.
  • Inválido path cor: A cor a ser usada para destacar um path que não é válido. A cor padrão é (ARGB) #FF0000FF.
  • Cor dos pontos de ancoragem: A cor a ser utilizada para desenhar os pontos de ancoragem path. O padrão é (ARGB) #FFC0C0C0.
  • Cor dos pontos de ancoragem inválidos: A cor a ser utilizada para desenhar os pontos de ancoragem path. O padrão é (ARGB) #FF800000.
  • Cor do ponto de ancoragem selecionado: A cor a ser usada para desenhar os pontos de ancoragem path que foram destacados (selecionados). A cor padrão é (ARGB) #FF00A4FF.
  • Cor selecionada do ponto de ancoragem inválido: A cor a ser usada para desenhar os pontos de ancoragem que são destacados, mas inválidos. A cor padrão é (ARGB) #FFA400FF.
  • Cor do ponto de ancoragem do mouse: A cor a ser usada para desenhar os pontos de ancoragem que estão ativos. O padrão é (ARGB) #FFFFFFFFFFFF.
  • Cor do ponto de ancoragem inválido do mouse: A cor a ser usada para desenhar os pontos de ancoragem que estão ativos, mas inválidos. O padrão é (ARGB) #FF0000FF.
  • Cor da sobreposição da máscara: Defina a cor da sobreposição para as máscaras se elas estiverem habilitadas (ver acima). O padrão é (ARGB) #FF404040.
