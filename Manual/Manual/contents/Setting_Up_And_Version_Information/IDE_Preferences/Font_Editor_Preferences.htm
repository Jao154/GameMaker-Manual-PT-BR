Preferências do editor de fontes

As Preferências do Editor de Fontes são usadas para definir como o Editor de Fontes se comporta. As diferentes opções disponíveis estão listadas abaixo, juntamente com uma descrição do que cada uma faz.

  • Atualização Fonts importado de projetos GameMaker 1: Quando isto for verificado, o GameMaker tentará atualizar fonts e regenerar seu texture. Isto só acontecerá se:
    • Esta preferência está habilitada
    • O font está disponível na máquina
    • GameMaker pode localizar um rosto correspondente na família font (esta é a causa mais provável de incoerência na importação)
    Note que isso também acontecerá se o GameMaker 1 font não tiver um cache font texture. Se um legado font não puder ser atualizado, então o GameMaker usará o cache como padrão. font texture e o editor Font não mais permitirá que você edite o font, mostrando uma opção de backup e atualização para outro:

    Clicando nisso, você poderá editar o font, ou adicionando o font que falta ou definindo outro, e o O backup font é armazenado na pasta font asset, como <font name>.yy.gms1 e <font name>.png.gms1. Para restaurar estes arquivos, apagar manualmente os arquivos <font name>.yy e substituí-los por estes, removendo o .gms1 extension dos nomes dos arquivos. Importante! Todos os assets armazenam seus nomes no arquivo .yy, portanto é necessário editar estes e colocar o nome correto do recurso também no campo <name> (se diferente).
  • Escolha automática para o diálogo de regeneração de conteúdo falhado: Quando você vai para regenerar um font asset texture , é-lhe mostrado um aviso. A configuração desta opção para "OK" significará que o font é regenerado sem que o aviso seja mostrado.
  • Escolha automática para o diálogo de regeneração falhado font: Quando a regeneração de um font asset texture falha, é-lhe mostrado um aviso. A configuração desta opção para "OK" saltará mostrando a mensagem.
  • Escolha automática para atualização de fontes legadas importadas: Ao importar um projeto legado, você pode receber um aviso sobre o legado fonts e a oferta de atualização. A configuração desta opção para "OK" dará direito à atualização automática sem mostrando esta mensagem.
