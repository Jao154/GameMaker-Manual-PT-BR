Preferências de economia de energia

As preferências de economia de energia aqui afetarão quanta energia de CPU e GPU será usada para gerar o GameMaker IDE (essencialmente isto diminuirá o quadro de base no qual o programa roda e, portanto, o consumo de energia).

As opções aqui são:

  • Quadro de potência AC (%): O modificador de framerate de base IDE quando conectado a uma entrada de energia. O padrão é 100%.
  • Bateria de potência (%): O modificador de frame de base IDE ao usar a bateria para alimentar a máquina. O padrão é de 35%.
  • Quadro de fundo (%): A base IDE framerate modifier quando o GameMaker não está em foco (independentemente do fornecimento de energia). O padrão é 15%.
