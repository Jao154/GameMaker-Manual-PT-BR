Preferências da janela de saída

O Janela de saída As preferências controlam as diferentes opções de cor e formatação para as várias janelas de saída diferentes no IDE. Cada uma das diferentes janelas tem a mesma conjunto de opções, que estão listadas abaixo:

  • Cor: A cor padrão para o texto que está sendo escrito na janela.
  • Nome da fonte: O nome do font a ser usado (disponível fonts dependerá daqueles instalados em seu sistema).
  • Estilo: O estilo font a ser usado, como negrito ou itálico (note que nem todas as janelas permitirão os mesmos estilos e também dependerá do font usado).
  • Tamanho: O tamanho para definir o font para a janela de saída dada.
  • Anti-Aliasing: Aqui você pode definir o modo anti-aliasing para as janelas de saída font renderização. Você pode escolher deixar o IDE decidir selecionando Use Default, ou você pode forçá-lo a On ou Off , selecionando uma dessas opções.
