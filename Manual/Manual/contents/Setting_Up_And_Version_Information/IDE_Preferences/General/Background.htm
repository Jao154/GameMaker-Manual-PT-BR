Preferências de fundo

As preferências de fundo tratam do fundo Workspace e o que ele exibirá, bem como como como será exibido. As opções são:

  • Imagem de fundo: Este é o arquivo de imagem que será usado como fundo para os diferentes Workspaces. Há uma imagem padrão fornecida e ela é ativada por padrão, mas você pode selecionar sua própria imagem aqui. Os tipos de imagem que você pode selecionar são *.png, *.gif, *.jpg, *.jpeg, *.tif, *.tiff, *.ico, *.bmp, *.2bp ou *.mdi, mas observe que independentemente do formato inicial da imagem, todos os arquivos serão salvos para uso como arquivos *.png, Assim, os arquivos animados *.gif, por exemplo, só serão apresentados como um único quadro inicial.
  • Escala da imagem: Isto controla como a imagem de fundo será escalonada dentro dos diferentes workspaces. As escolhas são:
    • Stretch - A imagem será esticada para caber no workspace, mudando de escala dependendo das docas visíveis
    • Azulejo - A imagem será azulejada para cobrir o espaço de trabalho
    • Best Fit - A imagem será escalonada para se ajustar ao workspace proporcionalmente, com barras sendo adicionadas ao redor da borda se não for um ajuste 1:1
    • Desktop Stretch - O fundo cobrirá todo o workspace, e se moverá em vez de mudar de escala quando as docas forem abertas/fechadas
    • Esticamento da tela - O fundo cobrirá uma área do tamanho da janela e será cortado/extendido para mostrar mais ou menos da imagem à medida que as docas são abertas/fechadas.
  • Mostrar imagens de fundo nos espaços de trabalho: Esta opção controla se a imagem de fundo selecionada deve ser mostrada ou não. Por padrão, esta opção é verificada como ligada.
