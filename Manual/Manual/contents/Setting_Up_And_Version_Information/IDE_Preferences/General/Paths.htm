Preferências de caminho

As preferências do Path controlam os diferentes arquivos paths exigidos pelos elementos no GameMaker IDE, bem como para qualquer editor externo que você necessite. As seguintes opções existem para a IDE:

  • Diretório Temp: O local para salvar todas as pastas temporárias. Por padrão em Windows isto é

    \Users\<Username>\AppData\Local\Temp\GameMaker

    E em macOS é:

    /var/folders/<Path_Hash>/GameMaker

  • Diretório IDE cache: O local para salvar o IDE cache. Por padrão em Windows este é

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    E em macOS é:

    /Users/<Username>/.config/GameMaker/Cache

  • Ativo cache diretório: O local para salvar o asset cache para cada projeto. Por padrão no Windows este é

    \Users\<Username>\AppData\Roaming\GameMaker\Cache

    E em macOS é:

    /Users/<Username>/.config/GameMaker/Cache

  • Localização de "Meus Projetos": O local onde o GameMaker irá inicialmente criar novos projetos. Por padrão em Windows isto é

    \Users\<Username>\Documents\GameMaker

    E em macOS é

    /Users/<Username>/GameMaker

  • Apagar automaticamente o diretório temporário no fechamento: Ao verificar isso forçará o GameMaker a apagar automaticamente a pasta Temp que ele cria por projeto para compilação etc... Esta configuração está ativada por padrão, e desmarcá-la irá switch desligá-la (o que significa que você terá que remover manualmente quaisquer arquivos temporários mais tarde).
  • Apagar automaticamente asset cache no fechamento: Se isto for verificado, a pasta asset do compilador cache será removida quando você sair do GameMaker. Isto está desativado por padrão, e ativá-lo significará que toda vez que você carregar e executar qualquer projeto, o cache precisará ser reconstruído (o que pode levar tempo, dependendo do tamanho do jogo).
  • Apagar automaticamente IDE cache no fechamento: Se isto for verificado, a pasta IDE do compilador cache será removida quando você sair do GameMaker. Isto está desativado por padrão, e ativá-lo significará que toda vez que você iniciar o GameMaker, o cache precisará ser reconstruído.
  • Apagar Pasta Temp: Clicando neste botão, a pasta temp para o projeto será apagada.
  • Excluir Asset Cache: Clicando neste botão, o compilador asset cache do projeto será excluído.
  • Excluir IDE Cache: Clicando neste botão, o cache IDE será apagado.

Existem as seguintes opções para definir paths para editores externos:

  • Caminho para o editor/visualizador externo de arquivos SWF: Se você estiver trabalhando com o formato SWF sprites, você pode configurá-lo para o path de seu visualizador/editor preferido e ao clicar no botão Editar Imagem no Editor Sprite, ele abrirá o programa em questão. O valor padrão aqui é não ter nenhum path.
  • Caminho para o editor/visualizador externo para arquivos Spine: Se você estiver trabalhando com o formato Spine sprites , você pode configurá-lo para o path de seu visualizador/editor preferido e ao clicar no botão Editar Imagem no editor Sprite, ele abrirá o programa em questão. O valor padrão aqui é não ter nenhum path.
