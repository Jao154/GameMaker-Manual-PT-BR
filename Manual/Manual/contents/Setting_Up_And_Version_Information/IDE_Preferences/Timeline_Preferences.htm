Preferências de linha do tempo

As Preferências de Linha do Tempo são usadas para definir certas propriedades para quando você estiver trabalhando com o Editor de Linha do Tempo. As opções disponíveis são:

  • Resposta automática para GML Conversão visual para a linguagem GameMaker: Quando você usa o botão direito do mouse em um momento timeline para converter GML Ações visuais em script (código), você receberá uma mensagem perguntando se deseja continuar, já que esta é uma conversão de uma só maneira. A mudança desta opção para "Ok" suprimirá a mensagem e prosseguirá com a conversão como se você tivesse clicado em "Ok" no diálogo. A configuração padrão é "Mostrar mensagem".
  • Uma resposta utomática a um aviso de ação permanente: Certas ações dentro de um timeline não podem ser desfeitas, e quando isso acontece você receberá uma mensagem de aviso perguntando se deseja continuar. A mudança desta opção para "Ok" suprimirá a mensagem e prosseguirá com a ação como se você tivesse clicado em "Ok" no diálogo. A configuração padrão é "Mostrar mensagem".
  • Abrir novos momentos para editar na criação: Normalmente quando você adiciona um novo momento a um timeline, o GameMaker abrirá automaticamente o editor para que você possa adicionar seu código ou GML Visual a ele. Entretanto, se você desmarcar esta opção, o momento será adicionado sem abrir o editor (você pode abri-lo clicando duas vezes no momento). Por padrão, esta opção está ativada.
