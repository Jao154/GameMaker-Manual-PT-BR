Preferências do Navegador de Ativos

As Preferências do Navegador de Ativos são usadas para definir certas propriedades para o funcionamento do Navegador de Ativos e seus elementos. As principais opções são:

  • Mostrar Acesso Rápido: Quando ativado, esta opção mostrará a seção "Acesso Rápido" na parte superior do navegador Asset. Esta opção é ativada por padrão.
  • Criar pastas padrão em um novo projeto: Quando ativada, esta opção terá todos os novos projetos começando com as pastas padrão asset já criadas, e quando desativada, apenas uma pasta room será criada com a pasta padrão room exigida por todos os projetos. Esta opção é ativada por padrão.
  • Número máximo de assets para abrir simultaneamente sem aviso prévio: Você pode selecionar vários assets para abrir de cada vez usando / + para selecioná-los e depois clicar com o botão direito do mouse em e escolher abrir todos. Se você selecionou mais do que o número especificado nesta opção, então lhe será mostrado um aviso, uma vez que ter muitos assets abertos ao mesmo tempo pode causar problemas com o desempenho em máquinas de extremidade inferior. O valor padrão é 10.
  • Máximo de itens editados recentemente: Esta configuração rege o número de assets listados na seção "Recent" do Asset Browser. O valor padrão é 10.
  • Tamanho da paleta de cores: Aqui você pode definir o número de cores disponíveis na paleta do navegador asset para grupos asset. O valor padrão é 16, com um valor mínimo de 16 e um valor máximo de 100.
  • Aplique asset cores em Acesso Rápido: Quando marcada, esta opção utilizará as cores definidas para qualquer asset tipo para colorir o asset na seção Acesso Rápido do navegador. Ao desmarcar esta opção não será utilizada a cor asset. Isto é ativado por padrão.
  • As cores do grupo substituem as cores infantis: Quando verificado, isto utilizará a cor do grupo de nível superior para colorir qualquer assets dentro do grupo, ignorando qualquer cor que tenha sido definida para o indivíduo asset. Isto é desativado por padrão.
  • Criar asset estilo de diálogo: Ao clicar no botão "Adicionar Ativo" no navegador Asset, você será presenteado com uma janela mostrando o diferente assets que pode ser adicionado ao seu projeto. Esta janela pode ser apresentada em uma visualização de lista ou em uma visualização em grade, e esta configuração rege o que é mostrado por padrão. O valor padrão é grid.
  • Ordem de classificação por padrão: Isto pode ser usado para definir a ordem de classificação padrão para assets no navegador, seja por ordem alfabética de A a Z ou de Z a A. A ordem padrão é de A a Z.
  • Pastas de classificação por padrão primeiro: Quando isto estiver ativado, o navegador asset ordenará primeiro as pastas, depois qualquer "solta" assets segundo, enquanto que se isto estiver desativado, então todas as pastas assets e as pastas serão ordenadas juntas. Isto é habilitado por padrão.
  • Sprite duplo clique abre o editor de imagem: Por padrão, o duplo clique em qualquer asset abrirá esse asset para edição, mas muitas vezes quando se lida com sprites, após a configuração inicial quando você cria o novo asset, você o abre para editar a imagem em vez das propriedades sprite. Selecionando esta opção, será aberto o editor de imagem para o sprite asset em vez do próprio sprite asset. Você ainda pode abrir o editor sprite clicando com o botão direito do mouse em e selecionando Editar. Esta opção está desligada por padrão.
  • O objeto duplo clique abre todos os eventos object: Por padrão, um duplo clique em em qualquer asset abrirá esse asset para edição, mas muitas vezes quando se trata de objects, após a configuração inicial quando se cria o novo asset, você o abre para editar apenas os eventos, em vez das propriedades object. Selecionando esta opção, serão abertos os diferentes eventos object em um novo workspace, sendo que cada evento receberá sua própria guia dentro do novo workspace. Você ainda pode abrir o editor object clicando com o botão direito do mouse em e selecionando Editar. Esta opção está desligada por padrão.
  • Ogrupo vazio é removido por ordem de quarto: Quando ativado, o desagrupamento de todos os rooms que pertencem a uma pasta de grupo removerá também a pasta do grupo. Quando desativado, a mesma ação irá desagrupar o rooms mas deixará a pasta de grupo vazia no navegador asset. Isto é habilitado por padrão.
  • Zoom padrão (%): Isto define a porcentagem padrão de zoom para o navegador Asset quando você inicia um novo projeto. O valor padrão é 100%.
  • Zoom mínimo (%): Isto define o valor mínimo de zoom permitido para o navegador Asset. O valor padrão é 50%.
  • Zoom máximo (%): Isto define o valor máximo de zoom permitido para o navegador Asset. O valor padrão é de 250%.
  • Mostrar linhas de árvores: Aqui você pode ativar/desativar as linhas das árvores que são mostradas ao lado de assets no Asset Browser. Isto é verificado por padrão para mostrar as linhas das árvores.
  • Expandir árvore na troca de filtro: Esta opção forçará o navegador Asset a expandir quaisquer nós de árvore colapsados quando você mudar as opções de filtro. A desativação da opção impedirá isto. A opção padrão está ativada.

Confirmar Diálogos

Aqui você pode selecionar o comportamento dos diálogos de confirmação ao trabalhar com um bem do Browser Asset. As opções disponíveis são:

  • Resposta automática à eliminação de asset: Ao excluir um asset do navegador Asset, você normalmente recebe um aviso perguntando se você tem certeza de que deseja prosseguir. Ao definir esta opção para "OK", a mensagem será suprimida. como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem".
  • Resposta automática a Room Eliminação do grupo de pedidos: Quando você exclui uma pasta de grupo contendo rooms do navegador Asset você normalmente recebe um aviso perguntando se você tem certeza e quer fazer uma procissão (que lista todos os rooms do grupo que será apagado). Ao configurar esta opção para "OK", a mensagem será suprimida como se você tivesse clicado no botão OK para prosseguir e excluir todo o rooms do grupo, bem como a própria pasta do grupo. O valor padrão é "Show Mensagem".
  • Resposta automática à abertura de múltiplos ativos: Como mencionado acima nas preferências gerais do navegador Asset, quando você abre múltiplos assets de uma vez a partir do navegador Asset, você pode receber um aviso se o número exceder o número especificado. Ao definir esta opção para "OK", o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem".
  • Resposta automática para GML Conversão visual para a linguagem GameMaker: Ao selecionar a opção de converter um nó visual GML para GML, você será solicitado a confirmar a mudança, pois esta não pode ser desfeita. Ao definir esta opção como "OK". o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem".
  • Resposta automática à configuração de um pai room via arrasto do nó: Na seção "Herança" da janela Room Manager, você pode arrastar um room para outro room para configurá-lo como um filho do room para o qual está sendo arrastado. Ao fazer isto você receberá um aviso perguntando se você tem certeza e se quer fazer um prognóstico. Ao configurar esta opção para "OK", a mensagem será suprimida como se você tivesse clicado no botão OK para prosseguir. O valor padrão é "Mostrar mensagem".
  • Resposta automática a erro de nome de grupo duplicado: Ao criar uma pasta de grupo para organizar seu assets e nomeá-la como outra pasta no navegador, será exibida uma mensagem perguntando se você deseja continuar e ter duas pastas com o mesmo nome. Ao definir esta opção para "OK" o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem".
  • Resposta automática à configuração de um pai room via arrasto do nó: Quando você arrasta um room sobre outro room no Gerente Room e depois o libera, será exibida uma mensagem perguntando se você deseja fazer do room um "filho" do room sendo largado. Ao definir esta opção para "OK" o aviso será suprimido como se você tivesse clicado no botão "OK" para prosseguir. O valor padrão é "Mostrar mensagem".
