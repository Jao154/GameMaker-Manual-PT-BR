Guia de Instalação

GameMaker é projetado para uso em computadores desktop MacOS e Windows com as especificações mínimas mencionadas aqui.

Você pode baixar o instalador GameMaker da página de sua conta YoYo no site da YoYo Games. Uma vez instalado, a primeira vez que você executar o GameMaker você poderá ser solicitado a atualizar antes de poder usar o produto. Após qualquer atualização ter sido instalada, você será solicitado a fornecer os detalhes da licença de sua conta YoYo Games (veja aqui para mais informações) antes de poder usar o produto, significando que a primeira vez que você usar o GameMaker você deverá ter uma conexão com a internet. Após o login inicial, você deverá ser capaz de usar o produto offline, no entanto, periodicamente será necessário que você se conecte para verificar a conta de login.

É importante notar que - dependendo do tipo de licença que você possui - você pode ser solicitado a instalar componentes adicionais ou seguir os passos de adição para configurar e usar o GameMaker para plataformas específicas. Para informações completas sobre o que cada plataforma requer, consulte este artigo no site YoYo Games Helpdesk: SDKs necessários parao GameMaker.

Versões Estável e Beta

A versão que você baixar da página YoYo Games deve ser a versão estável atual do GameMaker, mas também pode haver outra versão " Beta" disponível que é usada para mostrar ou testar novas funcionalidades antes de entrar em uma construção estável final. Os detalhes completos da versão são os seguintes:

  • Estável: Esta versão é a que você estaria usando para desenvolvimento geral e é recomendada para uso com qualquer produto comercial. Pode não ter certas características disponíveis nas versões Beta, mas isto se deve ao fato de não terem sido totalmente testadas ou exigirem trabalho adicional.
  • Beta: A versão Beta do GameMaker é para aqueles que desejam experimentar novas funcionalidades antes de serem adicionados ao Stable build, ou para aqueles que necessitam de uma correção para um problema. Não é recomendado que você use o Beta build para a produção final do jogo a menos que ele contenha uma correção importante para um bug que está atrasando seu projeto. Você pode obter o Beta build do GameMaker em sua página YoYo Conta "Downloads", e ele pode ser instalado e executado lado a lado com o Stable build. Observe que se você estiver executando o GameMaker através do Steam , não terá acesso à construção Beta do produto usando o próprio Steam, embora você ainda possa baixar e usar a versão desktop.
